# 공정거래위원회 표준약관 가이드

## 개요

공정거래위원회(공정위)는 불공정 약관을 방지하기 위해 업종별 표준약관을 제정·권고합니다.
사업자는 표준약관을 그대로 사용하거나 이를 기반으로 자체 약관을 작성할 수 있습니다.

## 주요 표준약관 유형

### 전자상거래 (가장 빈번)
| 항목 | 내용 |
|------|------|
| 적용 대상 | 온라인 쇼핑몰, 오픈마켓, 소셜커머스 |
| 필수 조항 | 청약 철회(7일), 환불 절차, 개인정보 처리, 분쟁 해결 |
| 금지 조항 | 일방적 해제권, 과도한 위약금, 소비자 입증 전가 |

### 소프트웨어/SaaS
| 항목 | 내용 |
|------|------|
| 적용 대상 | SaaS, 클라우드 서비스, 소프트웨어 라이선스 |
| 필수 조항 | 서비스 범위, SLA, 데이터 소유권, 해지 시 데이터 반환 |
| 금지 조항 | 일방적 서비스 변경/중단, 무제한 면책, 자동 갱신 강제 |

### 용역/컨설팅
| 항목 | 내용 |
|------|------|
| 적용 대상 | 프리랜서 계약, 컨설팅, 외주 개발 |
| 필수 조항 | 업무 범위, 납기, 검수 절차, 지식재산권 귀속 |
| 금지 조항 | 추가 작업 무상 요구, 일방적 계약 해지, 대금 지급 지연 |

### 프랜차이즈
| 항목 | 내용 |
|------|------|
| 적용 대상 | 프랜차이즈 가맹계약 |
| 필수 조항 | 가맹비/로열티 산정 기준, 영업지역 보장, 계약 해지 사유 |
| 금지 조항 | 부당한 위약금, 경쟁 제한, 일방적 공급가격 인상 |

## 불공정 약관 판단 기준

공정위 기준으로 아래 조항은 무효 처리될 수 있습니다:

### 약관규제법 주요 무효 사유
1. **신의성실 원칙 위반** (제6조): 고객에게 부당하게 불리한 조항
2. **면책 조항** (제7조): 사업자의 고의/과실 면책
3. **손해배상 제한** (제8조): 부당한 배상액 예정/제한
4. **해제·해지** (제9조): 사업자만 해지권 보유
5. **의사표시 의제** (제10조): 침묵 = 동의 간주

## 자체 약관 작성 시 체크리스트

- [ ] 공정위 표준약관과 대비하여 불리한 조항 없는지 확인
- [ ] 청약 철회 기간 7일 이상 보장 (전자상거래)
- [ ] 개인정보 처리방침 별도 링크 제공
- [ ] 분쟁 해결 절차 명시 (소비자원 조정, 법원 관할)
- [ ] 약관 변경 시 7일 전 사전 고지 의무 준수
- [ ] 중요 내용 굵은 글씨 또는 별도 확인 절차 (약관규제법 제3조)
