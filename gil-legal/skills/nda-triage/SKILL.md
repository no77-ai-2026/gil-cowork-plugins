# NDA 트리아지 (NDA Triage)

## 개요

비밀유지계약서(NDA: Non-Disclosure Agreement)를 빠르게 검토하여 핵심 조항의 이상 여부, 불리한 조건, 법적 리스크를 신속히 파악하고 수정 권고안을 제시한다. 한국 민법·상법·영업비밀보호법 기준을 적용하며, 단방향/양방향 NDA, 기술협력·투자·채용 맥락별 맞춤 검토를 제공한다.

**면책 고지: AI 보조 분석이며 전문 법률 자문을 대체하지 않습니다**

## 트리거 키워드

NDA, 비밀유지계약, 기밀유지계약, 비밀유지협약, NDA 검토, 계약서 검토, 기밀 조항, 정보보호 계약, 영업비밀, 비밀보장

## 워크플로우

### 1단계: NDA 유형 및 맥락 확인

**계약 당사자 역할 파악:**
- **단방향(Unilateral)**: 한쪽만 비밀 공개 (스타트업이 투자자에게 공개 등)
- **양방향(Mutual/Bilateral)**: 양측 모두 비밀 공유

**계약 맥락 분류:**
- **투자·M&A**: 재무 정보, 사업 계획, 기술 자산 보호
- **기술 협력·파트너십**: 소스코드, 알고리즘, 노하우
- **채용·고용**: 직원의 대외비 유지 의무
- **외주·위탁 개발**: 고객사 데이터, 제품 설계

### 2단계: 핵심 조항 체크리스트 검토

**① 비밀정보의 정의 (Definition of Confidential Information)**
- [ ] 비밀정보의 범위가 지나치게 광범위하지 않은가?
- [ ] 구두 공개 정보의 포함 여부 및 서면화 기한 명시 여부
- [ ] 예외 조항 적절성: 공지 정보, 독자 개발, 제3자 취득, 법령 공개

**② 의무 존속 기간 (Duration)**
- [ ] 비밀유지 의무 기간이 적절한가? (일반: 2-5년, 영업비밀: 영구도 가능)
- [ ] 계약 종료 후 의무 지속 여부 명시

**③ 비밀정보 사용 목적 제한**
- [ ] 허용 목적이 명확하게 한정되어 있는가?
- [ ] 제3자 제공 금지 및 예외 조건

**④ 정보 반환·삭제 의무**
- [ ] 계약 종료 시 자료 반환 또는 폐기 방법 명시
- [ ] 전자 파일 삭제 포함 여부

**⑤ 위반 시 구제 수단 (Remedies)**
- [ ] 손해배상 조항: 실손해 vs 약정 손해배상
- [ ] 금지명령(가처분) 신청 권리 명시 여부
- [ ] 관할 법원 및 준거법 (한국법 적용 명시 권고)

**⑥ 기타 리스크 조항**
- [ ] 역설계(Reverse Engineering) 금지 조항
- [ ] 직원 채용 금지(Non-Solicitation) 포함 여부
- [ ] 경업 금지(Non-Compete) 조항: 지나치게 광범위한 경우 한국법상 무효 가능

### 3단계: 한국법 특이사항 검토

**영업비밀보호법 관련:**
- 영업비밀 요건: 비공지성 + 경제적 유용성 + 비밀관리성 (3요소 충족 필요)
- 영업비밀 침해 시 형사 처벌 가능 (부정경쟁방지법 제18조)
- 전직 직원의 영업비밀 유출은 형사 처벌 대상

**개인정보보호법 연계 (2026년 9월 11일 시행 개정 반영):**
- NDA에 개인정보가 포함된 경우 별도 개인정보 처리 동의 필요
- 유출 가능성 인지 시점부터 통지 의무 발생 (2026 개정)
- 위반 시 최대 매출액 10% 과징금 (반복·중대 위반)

### 4단계: 트리아지 평가 결과 생성

```
# NDA 트리아지 평가서

## 종합 판정
- 위험도: 높음 / 중간 / 낮음
- 즉시 수정 권고 항목: X건
- 협상 권고 항목: X건
- 수용 가능 조항: X건

## 조항별 상세 평가
| 조항 | 현재 내용 | 위험도 | 권고 수정안 |

## 즉시 수정 권고 사항

## 협상 포인트

## 최종 의견
```

## 사용 예시

**예시 1**: "투자사에서 보내온 NDA 검토해줘. 우리 스타트업이 비밀 공개하는 쪽이야. [NDA 내용]"
→ 단방향 NDA 체크 → 범위·기간 리스크 → 수정 권고안 → 협상 포인트

**예시 2**: "개발 외주 업체와 체결할 양방향 NDA 초안 만들어줘"
→ 소스코드·기술 정보 보호 포커스 → 양방향 NDA 초안 → 핵심 조항 설명

**예시 3**: "경쟁사와 협력 논의 중인데 NDA 없이 정보 공개해도 될까?"
→ 리스크 설명 → NDA 체결 권고 → 표준 NDA 초안 제공

## 산출물

- NDA 트리아지 평가서 (위험도 분류 포함)
- 조항별 위험도 표 및 수정 권고안
- 협상 포인트 우선순위 목록
- 한국법 기준 특이사항 해설
- 표준 NDA 초안 (단방향·양방향)

## 주의사항

**면책 고지: AI 보조 분석이며 전문 법률 자문을 대체하지 않습니다**

**관련 한국 법령:**
- 민법 제664조 이하 (비밀유지 관련)
- 부정경쟁방지 및 영업비밀보호에 관한 법률
- 개인정보 보호법 (2026년 9월 11일 개정 시행)
- 근로기준법 (직원 NDA 관련)

**NDA 검토 시 주의사항:**
- 경업금지 조항이 지나치게 광범위하면 한국 법원에서 무효 판단 가능
- 무기한 비밀유지 의무는 한국 법원에서 축소 해석될 수 있음
- 손해배상 예정액이 과도하면 법원이 감액 가능 (민법 제398조 제2항)

## 문제 해결

| 상황 | 대응 |
|------|------|
| NDA 원문이 없는 경우 | 계약 맥락·당사자 역할 확인 후 표준 NDA 초안 제공 |
| 영문 NDA인 경우 | 한국법 관점으로 번역·검토. 필요 시 핵심 조항 한국어 병기 권고 |
| 경업금지·채용금지 범위가 넓은 경우 | 한국 법원 판례 기준 설명 후 축소 수정안 제안 |

**AI 보조 분석이며 전문 법률 자문을 대체하지 않습니다**
