# 우즈베키스탄 벤더 검증 가이드

> gil-operations v0.1.0 | UZ 사업자 검증, INN 조회, 계약 관리

## 개요

UZ에서 신규 거래처·벤더와 거래 시작 전 사업자 검증, 세무 신고 정상 여부, 외국인 투자 등록 등을 확인하는 표준 절차입니다.

---

## 1. UZ 사업자 검증 핵심 (Due Diligence)

### 단계 1: INN/STIR 검증 (soliq.uz)

[soliq.uz](https://soliq.uz) 국세청 검색 도구로 INN 조회:

- 사업자명·대표자·등록 주소 확인
- 활동 분류 (OKED) 확인
- 세무 등록 상태 (Active / Suspended / Closed)
- 체납 여부 (Налоговая задолженность)

URL: https://soliq.uz/services/info/inn-search

### 단계 2: 사업자 등록 확인 (stat.uz / my.gov.uz)

- 통계청 사업체 등록부 검색
- 등록일 확인 (신생 회사 주의)
- 자본금 정보
- 임원·주주 변동 (최근 6개월 대규모 변동 = 사기 신호)

### 단계 3: 라이선스 검증 (해당 산업)

| 산업 | 검증 기관 |
|------|----------|
| 식품 | 표준 위원회 + Sanitary 위원회 |
| 의약품 | 보건부 |
| 교육 | 교육부 |
| 금융 | 중앙은행 (cbu.uz) |
| IT 서비스 | 정보통신부 (조건부) |
| 광업 | 지질자원부 |

### 단계 4: 외국인 투자 등록 확인

- 외국인 자본 30%+: invest.gov.uz 등록 여부
- IT Park 거주자: it-park.uz 거주자 디렉토리

### 단계 5: 평판·이력 조사

- 2GIS Tashkent 후기·리뷰
- LinkedIn 임원 정보
- 한국 회사 거래 이력 (KOTRA Tashkent에 문의)
- 법원 분쟁 기록 (supremecourt.uz)

## 2. 검증 체크리스트

신규 거래처·벤더 거래 전:

```
[기본 검증]
[ ] 회사명·INN 일치 (soliq.uz)
[ ] 사업자 등록 상태: Active
[ ] 등록 주소·실제 영업 주소 일치
[ ] 활동 분류 (OKED) 일치
[ ] 세무 신고 정상 (체납 없음)
[ ] 임원 변동 6개월 이내 정상

[산업별 검증]
[ ] 라이선스 유효 (식품·의약·교육·금융 등)
[ ] 인증서 (ISO·우즈벡 표준)

[재무 안정성]
[ ] 자본금 명시 (정관)
[ ] 매출 규모 (가능하면)
[ ] 신용 보증 또는 보증금

[외국인 투자]
[ ] 외국인 자본 30%+ 시 등록 확인
[ ] IT Park / SEZ 거주자 자격 (해당 시)

[평판·이력]
[ ] 한국 회사 거래 경험
[ ] 분쟁·소송 기록 (없음)
[ ] 2GIS·LinkedIn 평판
```

## 3. 한국 vs UZ 벤더 검증 비교

| 항목 | 한국 | UZ |
|------|------|-----|
| 사업자번호 검증 | 홈택스 즉시 | soliq.uz INN 검색 |
| 등록 정보 | 통합 (홈택스) | 분산 (soliq + stat + my.gov) |
| 라이선스 검증 | 산업별 부처 | 산업별 부처 (UZ도 분산) |
| 외국인 투자 | 비교적 쉬움 | invest.gov.uz 추가 검증 |
| 분쟁 기록 | 대법원 판례검색 | supremecourt.uz |
| 신용 평가 | NICE·KCB 등 | 부재 (부족) |

UZ는 한국 대비 통합 정보 시스템이 약하므로 **다중 소스 종합 검증** 필수.

## 5. UZ 벤더 계약 표준

### 계약 형식
- 우즈벡어 또는 러시아어 본문 (의무)
- 한국어 또는 영어 병기 (한국 회사 시)
- 양 당사자 INN 표기 의무

### 결제 통화
- USD: 외환 송금 자유 (수출 매출 시)
- UZS: 현지 매입 (인플레이션 헤지 어려움)
- KRW: 일반적이지 않음 (환전 비용)

### 인보이스 (Счёт-фактура)
- soliq.uz 전자 인보이스 의무 (2022~)
- NDS 12% 명시 (해당 시)
- 5년 보관

### 분쟁 관할
- UZ 법원 (UZ 거래)
- 한국 법원 (한국 본사 직거래)
- 국제 중재 (대형 거래, SIAC·LCIA)

상세는 `gil-legal:contract-review/references/uz-civil-commercial-law.md` 참조.

## 6. UZ 벤더 리스크 모니터링

### 분기별 검증 (정기)
- soliq.uz 사업자 등록 상태 (Active 유지 확인)
- 세무 신고 정상 (체납 발생 시 즉시 통보)
- 라이선스 갱신 (만료 6개월 전)

### 즉시 검증 (이상 신호)
- 임원 갑작스러운 변동
- 본사 주소 이전
- 결제 지연
- 2GIS·LinkedIn 평판 악화

### 부도·폐업 대응
- 대체 벤더 사전 확보 (3곳 이상)
- 미수금 즉시 회수 (UZ 법원 빠른 절차)
- 진행 중 발주 즉시 중단

## 7. 한국 본사 ↔ UZ 자회사 인터컴퍼니 거래

### 이전가격 문서화 (Transfer Pricing)
- 한국·UZ 양국 세무 적용
- 시장 가격 기준 (Arm's Length)
- 비교 가능 거래 자료 보관

### 한·우 DTA 활용
- 배당: 5% 원천세 (지분 25%+)
- 로열티: 5~15%
- 사업 이익 (PE): 양국 과세

### 인터컴퍼니 거래 검증
- 매월 내부 인보이스
- 양국 회계 동일 시점 인식
- 세무 감사 대비 문서화

---

## 8. KOTRA Tashkent 활용

한국 회사가 UZ 신규 벤더 발굴 시:

- KOTRA Tashkent 신뢰 벤더 리스트 요청
- 현지 컨설턴트 추천 (변호사·회계사)
- 입찰·계약 통역 지원
- 분쟁 발생 시 자문 (1차)

## 10. 검증 자동화 (스크립트 예시)

```bash
# UZ 벤더 검증 자동화 스크립트 (예시)

# 1. INN 검증 (Web 크롤링 또는 수동)
INN="123456789"
echo "Checking INN: $INN at soliq.uz"
# WebSearch + soliq.uz 결과 파싱

# 2. 사업자 등록 (stat.uz)
echo "Checking registration at stat.uz"

# 3. 라이선스 (해당 시)
echo "Checking license at relevant ministry"

# 4. 평판 (2GIS, LinkedIn)
echo "Checking reputation"

# 5. 종합 점수 계산 + 보고서 생성
```

> 실제 검증은 양국 법무·회계 전문가 또는 KOTRA Tashkent 협업 권장.

---

> **주의**: UZ 벤더 검증은 다중 소스 + 현지 컨설턴트 조합이 필수. AI 자동 검증만으로 의존하지 말고, 중요 거래 시 현지 변호사·회계사 검증 병행.
