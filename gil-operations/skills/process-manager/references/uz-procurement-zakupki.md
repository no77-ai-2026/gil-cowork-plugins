# 우즈베키스탄 공공조달 (zakupki.uz) 가이드

> gil-operations v0.1.0 | UZ 공공조달 시스템·입찰 참가·한국 기업 진출

## 개요

UZ 공공조달은 [zakupki.uz](https://www.xarid.uzex.uz)를 통해 진행됩니다. 한국 나라장터(G2B)와 유사하지만 절차·언어·문화 차이가 있습니다. 한국 진출 기업이 UZ 정부·공공기관 입찰에 참가하는 표준 절차를 정리합니다.

---

## 1. UZ 공공조달 시스템 개요

### zakupki.uz (공식)

- 정부·공공기관 모든 조달 처리
- 운영: 우즈베키스탄 상품원자재거래소 (UZEX)
- 약 GDP 5~10% 규모 (연간)
- 외국인 사업자도 참가 가능 (등록 후)

### 조달 유형

| 유형 | 설명 | 예시 |
|------|------|------|
| 일반 입찰 (Тендер) | 공개 경쟁 | 일반 물품·서비스 |
| 제한 경쟁 | 자격 제한 | 특수 기술·실적 요구 |
| 단일 공급 | 수의계약 | 긴급·특수 |
| 전자 매장 (e-Shop) | 소규모 즉시 구매 | $500 미만 |
| 프레임워크 계약 | 장기 공급 | 정부 IT 등 |

---

## 2. 한국 나라장터 vs UZ zakupki.uz 비교

| 항목 | 한국 나라장터 | UZ zakupki.uz |
|------|------|------|
| 인증 방식 | 공인인증서 + ActiveX | 전자서명 + 모바일 앱 |
| 입찰 보증금 | 입찰 금액의 5% | 입찰 금액의 1~3% |
| 계약 보증금 | 계약 금액의 10% | 계약 금액의 5~10% |
| 소수 우대 | 중소기업 가점 | 현지 기업 우대 (외국인 차별 X) |
| 언어 | 한국어 | 우즈벡어 + 러시아어 |
| 참가 자격 | 사업자 등록 + 납세 | UZ 등록 법인 (외국인 100% OK) |
| 결제 통화 | KRW | UZS 또는 USD |
| 평균 처리 | 30일 (마감→낙찰) | 14~30일 |

## 3. UZ zakupki.uz 등록 절차

### Step 1: UZ 사업자 등록

외국인 사업자는 UZ 법인 (ИЧТ/ООО) 또는 지사 (Филиал) 형태로 등록 필요.

상세는 `gil-business:strategy-planner/references/uz-market-entry.md` 참조.

### Step 2: zakupki.uz 회원 가입

1. zakupki.uz 또는 xarid.uzex.uz 접속
2. "Регистрация" (회원 가입)
3. INN/STIR 입력
4. 사업자 정보 등록
5. 전자서명 등록 (UZ 인증기관 발급)
6. 활동 분류 (OKED) 선택

### Step 3: 입찰 참가 자격 등록

- 사업자 등록증
- 정관 (러시아어 또는 우즈벡어)
- 납세 증명 (체납 없음)
- 라이선스 (해당 산업)
- 외국인 투자 등록 (외국인 자본 30%+)

## 4. UZ 입찰 참가 절차

### Step 1: 공고 검색

- zakupki.uz 메인 → 입찰 검색
- 필터: 산업·지역·금액·마감일
- 한국·외국 기업 참가 허용 공고만 필터링

### Step 2: 공고 분석

- 입찰 자료 다운로드 (러시아어 또는 우즈벡어)
- 요구사항·평가 기준·일정 확인
- **언어**: 응찰 자료는 러시아어 또는 우즈벡어 의무 (외국 회사도)

### Step 3: 응찰 자료 준비

- 가격 제안 (UZS 또는 USD 명시)
- 기술 제안 (러시아어/우즈벡어)
- 자격 증빙 (사업자등록·납세·라이선스)
- 입찰 보증금 납부 (UZ 은행 보증서 또는 현금)

### Step 4: 전자 입찰 제출

- 마감 시각 30분 전 권장
- 전자서명으로 제출
- 시스템 다운 사례 多 → 사전 제출 권장

### Step 5: 개찰 + 결과

- 공개 개찰 (Tender Opening)
- 평가위원회 심사 (1~2주)
- 낙찰 통보

### Step 6: 계약 체결

- 5일 내 계약 서류 제출
- 계약 보증금 납부
- 전자계약 또는 종이 계약
- 납품 → 검수 → 대금 청구

## 5. 한국 본사 + UZ 자회사 입찰 전략

### 모델 A: UZ 자회사 단독 입찰
- UZ 등록 법인이 직접 입찰
- 외국인 투자 등록만 필요
- 한국 본사 노하우·실적은 자회사 실적으로

### 모델 B: 한·우 합작 입찰 (컨소시엄)
- 한국 본사 + UZ 자회사 컨소시엄
- 기술·인력 한국 본사 + 현지 운영 UZ 자회사
- 정부 입찰에서 컨소시엄 우대 가능

### 모델 C: UZ 현지 파트너 + 한국 기술
- UZ 현지 회사가 주 입찰자
- 한국 본사가 기술 협력 파트너 (하청)
- 외국인 등록 절차 회피·간소화

### 권장
- **소규모 ($10K~$100K)**: 모델 A
- **중대형 ($100K+)**: 모델 B
- **정치적 민감 산업**: 모델 C

## 6. KOTRA Tashkent 활용

[KOTRA Tashkent 무역관](https://www.kotra.or.kr/KBC/tashkent)은 한국 기업 UZ 입찰 진출 지원:

- 입찰 정보 안내 (정기 발간)
- 통역·번역 지원
- 현지 법무·회계 컨설팅 추천
- 입찰 컨소시엄 매칭

## 7. 한국 회사 진출 사례 (참고)

### 성공 사례
- **삼성·LG**: 정부 IT 인프라 입찰
- **한국전력**: UZ 발전소 EPC
- **현대건설**: 인프라·플랜트
- **KT·SKT**: 통신 인프라 컨설팅

### 주의 사례
- 우즈벡어/러시아어 자료 부정확 → 탈락
- 한국 본사 단독 입찰 (UZ 등록 X) → 자격 미달
- 입찰 보증금 납부 지연 → 자동 탈락

## 8. 양국어 SOP 작성 가이드

UZ 입찰 참가 회사 내부 SOP:

```
구매·조달 SOP — 양국어 버전

1. 신청 (Application)
   - 한국어: 구매 신청서 작성
   - Russian: Заявка на закупку

2. 승인 (Approval)
   - 한국 결재 라인: 팀장 → 임원
   - UZ 결재 라인: Менеджер → Директор

3. 공급업체 선정 (Vendor Selection)
   - 한국: 나라장터 또는 자체 공모
   - UZ: zakupki.uz 또는 자체 RFQ

4. 계약 체결 (Contract)
   - 양국어 본문 (러시아어 + 한국어)
   - Choice of Law 명시

5. 검수·지급 (Inspection & Payment)
   - 한국: 세금계산서 발행
   - UZ: 인보이스 (Счёт-фактура) 발행
```
