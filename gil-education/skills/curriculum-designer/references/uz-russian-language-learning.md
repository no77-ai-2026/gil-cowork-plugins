# UZ 러시아어 학습 가이드

> gil-education v0.1.0 | 한국어 화자용 러시아어 학습 (UZ 진출 표준)

## 개요

UZ 진출 시 러시아어는 비즈니스·교육·정부 표준. 한국어 화자에게 우즈벡어보다 학습 부담 ↑이지만 활용 범위 매우 넓음 (UZ + 러시아 + CIS 9개국).

---

## 1. 왜 러시아어?

| 활용 | UZ에서 |
|------|--------|
| 비즈니스 | 표준 (계약·미팅·이메일) |
| 교육 | 대학 강의 (대부분) |
| 정부 | 우즈벡어와 병행 |
| 미디어 | 신문·TV·SNS |
| 일상 | 도시 거주자 다수 |
| 국제 비즈니스 | CIS 9개국 |

UZ에서 러시아어만으로도 비즈니스 가능. 우즈벡어는 보조.

## 2. CEFR 단계 (러시아어)

| 단계 | TORFL | 학습 시간 (한국어 화자) |
|------|-------|---------------------|
| A1 (Базовый - элементарный) | TBU | 100~150 |
| A2 (Базовый - базовый) | TBU+ | 200~300 |
| B1 (Первый сертификационный) | TRKI-1 | 400~600 |
| B2 (Второй сертификационный) | TRKI-2 | 600~900 |
| C1 (Третий сертификационный) | TRKI-3 | 900~1,200 |
| C2 (Четвёртый сертификационный) | TRKI-4 | 1,500+ |

UZ 비즈니스 권장: **B1~B2** (TRKI-1~2).

---

## 3. 한국어 화자 학습 난이도

### 어려운 점

- 키릴 알파벳 (33자) — 영어 알파벳 + 변형
- 격 체계 (6격) — 명사·형용사·대명사 모두 변형
- 동사 (완료·불완료, 운동 동사)
- 강세·발음 (영어보다 어려움)
- 어순 (자유로움 — 한국어 SOV vs 러시아어 자유)

### 쉬운 점

- 한국어 차용어 (동지·인터넷 등 일부)
- 영어 학습 경험 활용 (라틴 문자 → 키릴)
- UZ에서 몰입 학습 환경

---

## 4. 단계별 학습 계획

### A1 (입문, 100~150시간)

```
[Week 1-2] 알파벳·발음
[Week 3-4] 인사·자기 소개·숫자
[Week 5-6] 가족·일상 사물
[Week 7-8] 단순 동사·현재 시제
[Week 9-10] 기초 격 (주격·목적격·전치격)
[Week 11-12] 종합 회화 50개
```

### A2 (기초, 200~300시간)

- 6격 모두
- 과거·미래 시제
- 형용사·대명사
- 일상 회화 200개
- TBU+ 시험 가능

### B1 (실용, 400~600시간)

- 완료·불완료 동사
- 운동 동사
- 비즈니스 어휘 (500개)
- TRKI-1 통과 가능
- **UZ 비즈니스 시작 가능**

### B2 (전문, 600~900시간)

- 추상 주제 (정치·경제)
- 학술·뉴스 이해
- 비즈니스 협상
- TRKI-2
- **UZ 비즈니스 자유로움**

## 5. 핵심 격 체계

### 6격

| 격 | 이름 | 용도 | 어미 (남성) |
|----|------|------|-----------|
| 1 | 주격 (Им.) | 주어 | -∅ |
| 2 | 생격 (Род.) | 소유·부정 | -а / -я |
| 3 | 여격 (Дат.) | 간접 목적 | -у / -ю |
| 4 | 대격 (Вин.) | 직접 목적 | -∅ (무생물) / -а (생물) |
| 5 | 조격 (Тв.) | 도구·동반 | -ом / -ем |
| 6 | 전치격 (Пр.) | 위치·주제 | -е |

예시 (стол - 책상):
- стол (책상이)
- стола (책상의)
- столу (책상에게)
- стол (책상을)
- столом (책상으로)
- (о) столе (책상에 대해)

## 6. 동사 (완료·불완료)

러시아어 동사 가장 어려운 부분:

| 측면 | 의미 | 예시 |
|------|------|------|
| 불완료 (Несов.) | 과정·반복 | читать (읽다) |
| 완료 (Сов.) | 결과·1회 | прочитать (다 읽다) |

→ 한국어 화자가 가장 헷갈림. 학습 시간 비례 (300시간+).

## 7. UZ 비즈니스 회화 핵심

### 인사·자기 소개

```
Здравствуйте! - 안녕하세요!
Меня зовут Ким. - 제 이름은 김입니다.
Я из Кореи. - 저는 한국에서 왔습니다.
Очень приятно. - 만나서 반갑습니다.
```

### 비즈니스 미팅

```
Спасибо за встречу. - 미팅 감사합니다.
Давайте обсудим. - 논의해 봅시다.
Какая ваша цена? - 가격이 얼마입니까?
Мы согласны. - 우리는 동의합니다.
До свидания! - 안녕히 가세요!
```

### 식사·접대

```
Приятного аппетита! - 맛있게 드세요!
Очень вкусно. - 매우 맛있어요.
Спасибо за угощение. - 식사 대접 감사합니다.
```

```

- 15분: 듣기 (Podcast·YouTube)

```

주말 1~2시간 회화 (튜터·HelloTalk).

## 13. 흔한 실수 (한국어 화자)

1. **격 무시**: 영어식 어순으로 → 의미 X
2. **완료·불완료 혼동**: 동의어 처리 ❌
3. **강세 무시**: 의미 다름 (замок 자물쇠 vs замок 성)
4. **운동 동사 회피**: идти/ходить, ехать/ездить 학습 필수
5. **TY (sen) vs ВЫ (Siz)**: 비즈니스에서 항상 ВЫ
