# 외국어 학습 지도 (56-language-tutor)

> MoAI-Cowork v1.0.0 하네스 레퍼런스

## 개요

수준 진단, 커리큘럼 설계, 강의, 퀴즈, 복습 관리를 통해 외국어 학습을 지원하는 에이전트 팀 협업 하네스입니다.

## 전문가 역할

- **커리큘럼 설계자**: 수준 진단 결과에 기반하여 학습 목표, 단계별 학습 계획, 교재 선정, 마일스톤을 포함한 맞춤형 커리큘럼을 설계합니다.
  - 목표 설정: 학습자의 목표와 필요에 맞는 명확한 학습 목표
  - 커리큘럼 구조: 학습 단계와 진도 계획
  - 콘텐츠 선정: 수준과 관심에 맞는 교재와 자료
  - 기술 통합: 문법, 어휘, 청취, 말하기, 읽기, 쓰기의 통합
  - 적응적 계획: 학습 진도에 따른 맞춤형 조정

- **강의 튜터**: 커리큘럼에 따라 문법 설명, 어휘 학습, 회화 연습, 읽기/쓰기 과제를 포함한 구체적인 강의 자료를 생성합니다.
  - 문법 지도: 명확한 문법 설명과 예시
  - 어휘 구축: 체계적인 어휘 학습과 복습 전략
  - 회화 연습: 실생활 회화 상황과 상호작용
  - 읽기/쓰기 과제: 수준에 맞는 읽기 자료와 쓰기 연습
  - 상호작용식 연습: 즉각적인 피드백을 제공하는 연습

- **수준 진단 전문가**: CEFR 기반의 진단 시험을 설계·실시하여 학습자의 현재 수준과 기술별 강점·약점을 정확하게 평가합니다.
  - 진단 시험 설계: 신뢰성 높은 CEFR 기반 시험
  - 수준 결정: A1~C2 중 현재 수준 파악
  - 기술별 갭 분석: 청취, 말하기, 읽기, 쓰기 영역별 강약
  - 학습 준비도 평가: 학습 동기, 학습 스타일, 자율성 평가
  - 기초선 문서화: 추후 진도 추적을 위한 기초선 기록

- **퀴즈 마스터**: 학습 내용에 기반하여 다양한 유형의 퀴즈를 설계하고 난이도를 조정하며 정확한 채점 기준과 피드백을 제공합니다.
  - 퀴즈 설계: 학습 내용과 목표에 맞는 퀴즈 문항
  - 난이도 조정: 학습자 수준에 맞는 점진적 난이도 상향
  - 채점 기준: 명확한 정답과 부분 점수 기준
  - 상세 피드백: 오답의 이유와 학습 방향 제시
  - 진도 측정: 퀴즈 점수를 통한 학습 진도 추적

- **복습 코치**: Ebbinghaus 망각곡선에 기반한 분산 반복 학습을 설계하고 약점 보강 계획을 수립하며 전체 학습 진도를 관리합니다.
  - 분산 반복 설계: SM-2 알고리즘 기반 복습 일정
  - 약점 보강: 약점 영역 집중 학습
  - 진도 추적: 전체 학습 진도 모니터링
  - 복습 자료 큐레이션: 효과적인 복습 자료 선정
  - 학습 계획 조정: 진도에 따른 계획 수정

## 워크플로우

### 단계 1: 준비 (오케스트레이터 직접 수행)

1. 사용자 입력에서 추출:
   - **목표 언어**: 학습할 외국어
   - **모국어**: 학습자의 모국어 (기본값: 영어)
   - **학습 경험**: 선행 학습 경험과 자가 진단 수준
   - **학습 목표**: 여행/업무/시험/이민/취미 등
   - **가용 시간**: 주간 학습 가능 시간
   - **기존 자료** (선택): 선행 학습 자료, 성적표 등

2. 프로젝트 루트에 `_workspace/` 디렉토리 생성
3. 입력을 정리하여 `_workspace/00_input.md`로 저장

### 단계 2: 팀 구성 및 실행

| 순서 | 작업 | 에이전트 | 의존성 | 산출물 |
|------|------|--------|--------|--------|
| 1 | 수준 진단 | 진단 전문가 | 없음 | `01_level_assessment.md` |
| 2 | 커리큘럼 설계 | 설계자 | 작업 1 | `02_curriculum.md` |
| 3 | 첫 번째 강의 | 튜터 | 작업 2 | `03_lesson_01.md` |
| 4 | 첫 번째 퀴즈 | 마스터 | 작업 3 | `04_quiz_01.md` |
| 5 | 복습 계획 | 코치 | 작업 2, 3, 4 | `05_review_plan.md` |

**에이전트 간 커뮤니케이션 흐름:**
- 진단 전문가 완료 → 기술별 수준, 강점·약점, 학습 목표, 가용 시간을 설계자에게 전달
- 설계자 완료 → 1주차 주제, 문법, 어휘를 튜터에게, 마일스톤 기준을 마스터에게 전달
- 튜터 완료 → 핵심 학습 항목을 마스터에게, 학습 항목을 코치에게 전달
- 마스터 결과 → 코치에게 전달 → 약점 보강 + 복습 일정 생성

**지속적 학습 주기:**
학습자가 추가 강의를 요청하면:
1. 튜터가 다음 강의 생성 (`03_lesson_02.md`, `03_lesson_03.md` 등)
2. 마스터가 해당 강의용 퀴즈 생성 (`04_quiz_02.md` 등)
3. 코치가 복습 계획 갱신 + 진도 보고서 생성

### 단계 3: 진도 보고

코치가 주기적으로 진도 보고서 생성:
- `_workspace/06_progress_report.md` — 전체 진도, 기술별 성장, 퀴즈 추이, 학습 패턴

## 산출물

- `00_input.md` — 정리된 사용자 입력
- `01_level_assessment.md` — 수준 진단 결과
- `02_curriculum.md` — 학습 커리큘럼
- `03_lessons.md` — 강의 자료
- `04_quizzes.md` — 퀴즈
- `05_review_plan.md` — 복습 계획
- `06_review_report.md` — 복습 보고서

## 확장 스킬

- **spaced-repetition**: SM-2 알고리즘, Ebbinghaus 망각곡선, 복습 일정 설계
- **cefr-assessment**: CEFR 6단계 설명자, 적응적 진단 시험, CEFR-시험 매핑

## 오류 처리

| 오류 유형 | 대응 전략 |
|---------|---------|
| 목표 언어 미지정 | 영어로 기본값 설정 후 확인 |
| 수준 진단 거부 | 자가 진단으로 임시 수준 설정, 첫 강의에서 조정 |
| 에이전트 실패 | 1회 재시도 → 해당 산출물 없이 진행 |
| 난이도 불일치 | 학습자 피드백에 따라 즉시 조정, 커리큘럼 갱신 |
| 드문 언어 지원 부족 | 지원 범위 사용자 안내, 주요 언어(영어, 일본어, 중국어, 스페인어, 프랑스어, 독일어) 중심 |
