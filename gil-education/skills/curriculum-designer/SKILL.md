# 커리큘럼 디자이너 (Curriculum Designer)

## 개요

교육 과정 설계 및 학습 콘텐츠 제작을 지원한다. 온라인 강의 커리큘럼, 직무 역량 모델, 외국어 학습 전략 수립, Bloom's Taxonomy 기반 학습 목표 정의, 2026년 에듀테크 트렌드(초개인화 학습, AI 코스웨어)를 반영한 교육 설계를 제공한다.

> **한국 표준 + UZ 듀얼 컨텍스트** — 한국 교육과정·NCS가 기본이며, UZ 교육체계·러시아어 학습 전략은 [`references/uz-education-system.md`](references/uz-education-system.md)·[`references/uz-russian-language-learning.md`](references/uz-russian-language-learning.md) 참조.

## 트리거 키워드

강의, 커리큘럼, 온라인 교육, 학습 목표, 역량 모델, 스킬 갭, 외국어, 어학, 언어 학습, 교육 과정, 강의 계획, 직무 교육, 역량 개발

## 워크플로우

### 1단계: 학습 목적 파악

**대상 학습자:**
- 연령, 직무, 현재 수준
- 학습 경험 (초급·중급·고급)
- 학습 스타일 (시각·청각·실습형)

**교육 목표:**
- 기간 (1회성, 4주, 8주, 16주)
- 형식 (온라인, 오프라인, 블렌디드)
- 성취 목표 (수료 후 할 수 있는 것)

### 2단계: 학습 목표 정의

**Bloom's Taxonomy 6단계 적용:**
| 인지 수준 | 설명 | 학습 목표 예시 |
|-----------|------|---------------|
| 기억(Remember) | 사실 암기 | 기본 개념, 용어, 원리 설명 |
| 이해(Understand) | 개념 설명 | 자신만의 언어로 재정의 |
| 적용(Apply) | 실제 사용 | 실습 문제, 시뮬레이션 |
| 분석(Analyze) | 구조 파악 | 비교, 분류, 원인 분석 |
| 평가(Evaluate) | 판단·비판 | 비판적 사고, 타당성 평가 |
| 창조(Create) | 새로운 산출 | 프로젝트, 포트폴리오 |

**구체적 학습 목표 작성:**
- "할 수 있다" 형태 (예: Python 기초 문법을 사용하여 간단한 프로그램을 작성할 수 있다)
- 측정 가능한 행동 동사 사용 (설명한다, 작성한다, 분석한다)
- 성취 기준 명시 (예: 5분 이내에 문제를 해결할 수 있다)

### 3단계: 커리큘럼 구조 설계

**주차별/단원별 목차 구성:**
- 1주차: 오리엔테이션, 기본 개념
- 2주차: 핵심 원리 이해
- 3주차: 실습 및 응용
- 4주차: 프로젝트 및 종합 평가

**각 차시 학습 내용:**
- 학습 목표 (2-3개)
- 핵심 개념
- 학습 활동 (강의, 실습, 토론)
- 평가 방법 (퀴즈, 과제)

**학습 활동 방식:**
- 동영상 강의
- 실습 과제
- 토론·피어 리뷰
- 프로젝트 기반 학습(PBL)
- 플립러닝 (사전 학습 + 수업 중 적용)

### 4단계: 평가 방법 결정

**평가 방식:**
- **퀴즈**: 매주 또는 매단원 (기억·이해 수준)
- **과제**: 실습 문제, 보고서 (적용·분석 수준)
- **프로젝트**: 팀 또는 개별 (창조 수준)
- **발표**: 발표, 피드백 (평가 수준)

**평가 비중:**
- 퀴즈 30%, 과제 30%, 프로젝트 40%
- 형성평가(과정) 50%, 총괄평가(결과) 50%

### 5단계: 결과물 정리

**학습 목표 명세서:**
- 전체 학습 목표 (3-5개)
- 주차별 학습 목표
- 성취 기준

**주차별 커리큘럼 표:**
| 주차 | 학습 목표 | 학습 내용 | 학습 활동 | 평가 |
|------|----------|----------|----------|------|

**강사 가이드 초안:**
- 각 차시 운영 팁
- 예상 질문·답변
- 추가 자료

## 2026년 한국 교육 트렌드

에듀모닝 2026-2027 교육산업 전망 보고서 및 에듀테크 코리아 페어 동향 반영:

### 핵심 키워드

- **초개인화 학습(Hyper-Personalized Learning)**: PwC 삼일 Paradigm Shift Vol.6 — 에듀테크의 핵심 가치는 학습자 개별 분석을 통한 1인 맞춤형 학습
- **AI 코스웨어**: 디지털 교과서 + AI 결합. 자동채점, 오답 클리닉, 학습 분석 대시보드 통합
- **실감화(VR/AR)**: 가상·증강현실 기반 몰입형 학습 콘텐츠 확산
- **연결화**: 학습자-교사-학부모-플랫폼 간 데이터 연결 생태계
- **융합화**: 교과 경계 초월 프로젝트 기반 학습(PBL) 확산

### 한국 커리큘럼 설계 시 고려사항

| 영역 | 2026 트렌드 |
|------|------------|
| 공교육 | AI 디지털 교과서 전면 도입 대비, 교사 역할 재정의 |
| 사교육 | AI 튜터 경쟁력 강화, 인간 강사의 고차원 사고 지도 차별화 |
| 기업 교육 | AX(AI Transformation) 역량 개발, 직무별 AI 활용 교육 |
| 어학 교육 | AI 발음 교정, 실시간 번역, 원어민 AI 대화 파트너 |

### Bloom's Taxonomy 적용 커리큘럼 설계

커리큘럼 목표를 Bloom's 6단계에 맞춰 구조화:
1. 기억 → 이해: 기본 개념 학습 (AI 자동화 영역)
2. 적용 → 분석: 실습·프로젝트 (교사 지도 영역)
3. 평가 → 창조: 비판적 사고·창작 (고차원 목표)

## 사용 예시

**예시 1**: "Python 입문 온라인 강의 8주 커리큘럼을 만들어줘"
→ 학습자 수준 확인 → Bloom's 수준 배분 → 주차별 목차 → 평가 방법

**예시 2**: "영업직 신입사원 역량 모델을 설계해줘"
→ 직무 분석 → 핵심 역량 도출 → 현재 수준 vs 목표 수준 → 개발 계획

**예시 3**: "TOEIC 900점을 위한 3개월 학습 계획을 세워줘"
→ 현재 점수 확인 → 약점 분석 → 주차별 학습 계획 → 모의고사 스케줄

**예시 4**: "사내 AI 활용 교육 프로그램을 기획해줘"
→ 대상 직군 확인 → 학습 목표 정의 → 커리큘럼 설계 → 평가 방법

## 분야별 설계 예시 (예시 다양화)

| 유형 | 대상·기간 | 목표(Bloom) | 구조 핵심 |
|---|---|---|---|
| 어학(영어 회화) | 직장인·8주 | 적용: 비즈니스 상황 대화 수행 | 상황별 시나리오 → 롤플레이 → 피드백 |
| 자격증(정보처리기사) | 취준생·12주 | 분석: 기출 유형 분해·약점 보강 | 과목별 이론 → 기출 → 모의(assessment-creator 연계) |
| 직무역량(데이터 리터러시) | 사내·4주 | 평가: 지표 타당성 판단 | NCS 역량 매핑 → 실습 과제 → 사내 데이터 PBL |
| 학위(대학원 세미나) | 석사·16주 | 창조: 연구계획서 작성 | 문헌 → 토론 → research-assistant 연계 |

각 예시는 ① 학습목표(할 수 있다 형태) ② 주차 구조 ③ 평가 정렬을 함께 설계하며, 교재는 `textbook-builder`, 평가는 `assessment-creator`로 체이닝.

- 강사 가이드 초안

## 주의사항

**교육 자료 접근:**
- 교육 자료 접근 불가 시 학습자 수준, 목표 역량, 교육 기간을 알려주시면 구조만으로 설계 가능

**학습 목표 불명확 시:**
- "이 교육을 마친 학습자가 무엇을 할 수 있어야 하나요?"를 함께 논의

**기존 커리큘럼 개선 요청 시:**
- 현행 목차나 강의계획서를 공유해 주시면 개선안 제안

**특수 교육 대상 (장애, 고령층 등):**
- 접근성 가이드라인과 유니버설 디자인 원칙 적용

## 백워드 설계·역량 매핑 심화

- **백워드 설계(Wiggins & McTighe) 3단계** — ① 기대 결과(목표·핵심 질문) → ② 수용 가능한 증거(평가) → ③ 학습 경험·수업 계획. 평가를 먼저 설계.
- **건설적 정렬(constructive alignment)** — 학습목표 ↔ 학습활동 ↔ 평가의 일관성. Bloom 동사로 목표 진술.
- **역량 매핑** — 직무 역량은 NCS, 어학은 CEFR(A1~C2)로 정렬. 역량 갭 분석 → 모듈 배치.
- **계열화(sequencing)** — 선수학습 → 나선형 심화. 마이크로러닝·초개인화(2026 트렌드) 반영.
- **산출 연계** — 설계 후 `gil-education:textbook-builder`(교재)·`assessment-creator`(평가)로 체이닝.
