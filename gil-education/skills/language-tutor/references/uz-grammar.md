# 우즈벡어 기초 문법

> gil-education | language-tutor 참조 | UZ 팩

## 1. 알파벳 (라틴, 28자)

```
A B D E F G H I J K L M N O P Q R S T U V X Y Z
+ Ch Sh Ng O' G' '
```

특수:
- O' (모디파이어 letter prime + o) — 어두운 'o' (ɔ)
- G' — 후두 'g' (ɣ)
- ' (apostrophe, ʼ) — 발음 분리
- Ch, Sh, Ng — 디그래프

## 2. SOV 어순 (한국어와 동일)

```
한국어: 나는 책을 읽는다.
우즈벡어: Men kitobni o'qiyman.
        (Men [I-NOM] kitobni [book-ACC] o'qiyman [read-1SG])
```

## 3. 격 체계 (6격)

| 격 | 어미 | 의미 | 예시 |
|----|------|------|------|
| 주격 (Bosh) | -∅ | 주어 | kitob (책) |
| 소유격 (Qaratqich) | -ning | ~의 | kitobning (책의) |
| 목적격 (Tushum) | -ni | ~을/를 | kitobni (책을) |
| 여격 (Jo'nalish) | -ga | ~에/로 | kitobga (책에) |
| 처격 (O'rin) | -da | ~에서/에 | kitobda (책에서) |
| 탈격 (Chiqish) | -dan | ~에서/부터 | kitobdan (책에서) |

### 모음 조화 (약함)

- 어간 끝 자음에 따라 -ga vs -ka 등 변형 일부

## 4. 동사

### 사전형: -moq

```
o'qimoq - 읽다
yozmoq - 쓰다
yemoq - 먹다
ichmoq - 마시다
bormoq - 가다
kelmoq - 오다
```

### 현재 시제 (1인칭 단수 -man)

| 인칭 | 어미 | 예 (o'qi-) |
|------|------|----------|
| Men (I) | -yman | o'qiyman |
| Sen (you 반말) | -ysan | o'qiysan |
| U (he/she) | -ydi | o'qiydi |
| Biz (we) | -ymiz | o'qiymiz |
| Siz (you 정중) | -ysiz | o'qiysiz |
| Ular (they) | -ydilar | o'qiydilar |

### 과거 시제

```
o'qidim - 나는 읽었다
o'qiding - 너는 읽었다
o'qidi - 그는 읽었다
o'qidik - 우리는 읽었다
o'qidingiz - 당신은 읽었다
o'qidilar - 그들은 읽었다
```

### 미래 시제 (현재형으로 표현 가능)

```
Ertaga o'qiyman. - 내일 읽을 것이다.
```

또는 가능 미래:
```
O'qiy olaman. - 읽을 수 있다.
O'qimoqchiman. - 읽고 싶다.
```

## 5. 부정형

```
긍정: O'qiyman. (읽는다)
부정: O'qimaymanaman. (읽지 않는다)
   → -ma- 삽입
```

## 6. 의문문

### Yes/No

```
Siz koreyalikmisiz? - 당신은 한국인입니까?
Ha. - 네.
Yo'q. - 아니요.
```

→ 끝에 -mi 추가.

### Wh-

```
Kim? - 누구?
Nima? - 무엇?
Qachon? - 언제?
Qayerda? - 어디서?
Nega? - 왜?
Qanday? - 어떻게?
```

## 7. 명사·형용사

### 형용사

명사 앞 (한국어와 동일):
```
yangi kitob - 새 책
yaxshi odam - 좋은 사람
katta uy - 큰 집
```

형용사 비교·최상:
```
yaxshi (좋은) - yaxshiroq (더 좋은) - eng yaxshi (가장 좋은)
katta (큰) - kattaroq - eng katta
```

## 9. 정중함 (Sen vs Siz)

| Sen | Siz |
|-----|-----|
| 친구·가족·아이 | 정중·낯선·연장자·비즈니스 |
| 반말 | 존대 |
| 동사 -san | 동사 -siz |

→ 한국어 존대법과 유사. 비즈니스에서는 항상 Siz.

## 10. 자주 쓰는 표현

```
Men koreyalikman. - 저는 한국인입니다.
Men Toshkentda yashayman. - 저는 타슈켄트에 삽니다.
Men o'zbek tilini o'rganaman. - 저는 우즈벡어를 배웁니다.
Bu juda qiziq. - 이것은 매우 흥미롭습니다.
Iltimos, sekinroq gapiring. - 천천히 말씀해 주세요.
Tushunmadim. - 이해하지 못했습니다.
Qaytarib aytsangiz mumkinmi? - 다시 말씀해 주실 수 있나요?
```

## 11. 한국어 화자 학습 팁

1. **어순 동일 → 빠른 시작**
2. **격 어미 → 한국어 조사처럼 학습**
3. **동사 사전형 -moq → 한국어 -다처럼**
4. **모음 (o vs o') → 발음 구별 연습**
5. **q (까) → 한국어 ㄲ과 유사**
6. **Sen vs Siz → 반말·존대법처럼**
