# EN 팩 — 소리블록 스피킹·발음 (교재 기반)

> gil-education | language-tutor 참조 | `--target en --register life` · drill · conversation · pronunciation | 사용자 제공 교재 기반(개인 사용, 자유 반영)

"블록(chunk) 단위로 통문장을 즉시 산출"하는 스피킹 훈련 + 발음 자가점검 자산. 문법 규칙 암기가 아니라 **자주 쓰는 덩어리(블록)를 입에 붙이는** 방식.

## 3단 훈련법 (모드 매핑)

| 훈련 | 방식 | 튜터 모드 |
|---|---|---|
| **순한맛** | 빈칸 채우기(힌트 단어 제공) | `--mode drill` 1단계 |
| **매운맛** | 한글 문장 → 영어로 산출(무힌트) | `--mode drill` 2단계 / `--mode conversation` |
| **쉐도잉** | 답안 보고 원어민 소리 듣고 따라하기 | `--mode review` + 외부 음원 안내 |

레슨 흐름: 블록 제시 → 순한맛(빈칸) → 매운맛(한→영) → 쉐도잉 안내 → 누적 복습.

## 블록 인덱스 (1단계, 50 PART)

**어순&어법 개념**: Be동사 / Be+ing·Be+pp / 일반동사 / 조동사

**시작블록 (즉시 산출 덩어리)**
- be동사: I'm from·on my way·in the middle of·about to / good at·ready for·glad to·here for / afraid that·sorry for·sure of·nervous about / Be able to·likely to·aware of·prone to
- It's: It's just·time to·no wonder that·like / Is it possible to·It is important to·It's easy to·Is it true
- There is/are
- be ~ving: I'm getting·just trying to·thinking of·talking about / dying to·telling·looking forward to·I was wondering if
- be pp: surprised by·excited about·disappointed with/in·bored with·frightened / satisfied with·worried about·used to·supposed to
- 의지·요청: I wanna / I want you to / I just wanted to / I like / I think / I've got / I have to(need·must·have got to) / I know
- 의문·제안: Why don't / what / how(형용사·부사) / Can I / Could / may·might / should / would / would like to
- 미래: be ~ing(가까운 미래)

**코어블록 (기본동사)**: have · make · get · take · do · go · keep
**디테일블록 (정보 확장)**: 방법·장소·빈도·시간·목적·조건·동행·기간 / Location / Frequency / Time

## 발음 자가점검 — 소리튠 발음기호 대조표

음성 출력이 없으므로 IPA + 한국어 근사 + 사전 대조로 자가점검(실제 청취는 쉐도잉·외부 음원).

| 소리튠 | IPA | 예 |
|---|---|---|
| father | ɑː | car ɑːr |
| bus | ʌ | bus |
| apple | æ | apple |
| he | iː | he |
| him | ɪ | him (he iː ↔ him ɪ 구분) |
| moon | uː | moon |
| book | ʊ | book |
| bird | ɜːr/ər | bird |
| oh | oʊ | oh |
| now | aʊ | now |
| say | eɪ | say |
| boy | ɔɪ | boy |
| schwa | ə | (약모음, 강세 없는 음절) |

> /ɔː/는 미국식에서 /ɑː/로 내면 됨(영·미 차이). 사전마다 표기 다름 — 대조표로 통일.

## 안티패턴

- 블록을 단어로 쪼개 문법 분석 → 덩어리 통째 암기·산출.
- 순한맛만 반복(빈칸 의존) → 매운맛(한→영 무힌트)으로 산출력 강화.
- 발음 "완벽 교정" 표방 → 대조표 자가점검 + 쉐도잉·튜터로 보완.
