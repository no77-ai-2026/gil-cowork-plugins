## 한국어 차용 (매우 적음)

UZ 한국어 차용은 거의 없음. 한국 기업 진출로 일부 (Samsung, LG 등 브랜드명).

## 영어 차용 (현대 일상)

```
internet, kompyuter, telefon, mobil, biznes, ofis, kafe, klub, restoran...
```

## 러시아어 차용 (많음)

```
mashina (машина) - 자동차
samolyot (самолёт) - 비행기
universitet (университет) - 대학
direktor (директор) - 이사
biznes (бизнес) - 비즈니스
```

UZ 비즈니스 어휘는 러시아어 차용 다수. 러시아어 학습이 우즈벡어 어휘 학습에도 도움.

## 학습 순서 (한국어 화자)

```
1주차: 인사·예의 (50개)
2~3주차: 숫자·시간·요일·월 (50개)
4~5주차: 가족·집·일상 (50개)
6~8주차: 음식·시장·쇼핑 (150개)
9~10주차: 교통·여행 (50개)
11~12주차: 의료·긴급 (30개)
13~16주차: 기본 동사·형용사 (200개)
17~24주차: 비즈니스 (150개)
25~36주차: 추상·정치·사회 (100~200개)
```

→ 6개월 (총 800~1,000개) 학습.

## 일일 학습 추천

- 새 단어 5~10개/일
- 어제 단어 복습
- Anki·Memrise 활용
- 실생활 사용 (매일 1~2개 표현 시도)
