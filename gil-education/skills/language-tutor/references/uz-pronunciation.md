# 우즈벡어 발음 가이드

> gil-education | language-tutor 참조 | UZ 팩

## 1. 모음 (6개)

| 라틴 | IPA | 한국어 비슷 | 영어 비슷 |
|------|-----|------------|----------|
| a | a | 아 | car |
| e | e | 에 | bed |
| i | i | 이 | see |
| o | o (밝은) | 오 | (없음 정확히) |
| o' | ɔ (어두운) | 오 (어두운) | hot |
| u | u | 우 | food |

### 모음 길이

우즈벡어는 모음 길이 구별 X (한국어와 동일).

### 한국어 화자 주의

- **o vs o'**: 둘 다 '오'로 들림 → 발음 구별 연습 필수
  - bola (아이) vs bo'la (할 것이다 — 다른 의미!)

## 2. 자음

### 일반 자음

```
b - ㅂ
d - ㄷ
f - ㅍ (영어 f)
g - ㄱ
h - ㅎ
j - ㅈ
k - ㅋ
l - ㄹ
m - ㅁ
n - ㄴ
p - ㅍ
r - ㄹ (떠는 발음, 한국어보다 강함)
s - ㅅ
t - ㅌ
v - ㅂ (영어 v, 한국어에 X)
y - ㅑ·ㅛ·ㅠ (반모음)
z - ㅈ (영어 z)
```

### 특수 자음

| 라틴 | IPA | 발음 | 한국어 비슷 |
|------|-----|------|------------|
| q | q | 후두 k (목구멍) | ㄲ (강하게) |
| g' | ɣ | 후두 g | (없음, 가글 소리) |
| x | x | 독일어 'ch' | (없음, 으흐) |
| ch | tʃ | 영어 'ch' | ㅊ |
| sh | ʃ | 영어 'sh' | ㅅ + ㅑ·ㅛ |
| ng | ŋ | 영어 'ng' | ㅇ (받침) |
| ' (apostrophe) | ʔ | 성문 폐쇄 | (강한 분리) |

### 한국어 화자 어려움 자음

1. **q** vs **k**:
   - q = 후두 (목구멍에서)
   - k = 일반 (한국어 ㅋ)
   - 차이 알아야 의미 구별: qari (늙은) vs kari (없는 단어)

2. **g'** = 가글 소리 (한국어 X)
   - g'oz (거위), o'g'il (아들)

3. **x** = 독일어 'ch'
   - xush (좋은), Xayr (안녕)

4. **r** = 강하게 떠는 발음 (한국어 ㄹ보다 강함)
   - rahmat (감사)

5. **v** = 영어 v (한국어 ㅂ과 구별 약함)

## 3. 강세

- 일반적으로 **마지막 음절** 강세
- 차용어 (러시아어·아랍어) 다름
- 한국어와 유사 (강세 약함)

```
salóm (안녕)
rahmát (감사)
shifokór (의사)
universitét (대학)
```

## 4. 모음 조화 (약함)

- 어간 끝 모음에 따라 어미 변형 일부
- 한국어 모음 조화 (양·음성 모음)와 유사
- 우즈벡어는 약함 (러시아어 영향)

```
xona-da (방에서) — a 어간 → -da
shahar-da (도시에서) — a 어간 → -da
ish-da (일에서) — i 어간 → -da
```

## 5. 발음 연습

### 최소 대립쌍 (Minimal Pairs)

```
qari vs kari (q vs k)
o vs o' (bola vs bo'la)
```

### 한국어 화자 추천 연습

1. **q 발음**: 목구멍 깊이 (가글하듯)
   - qovun (멜론), qimmat (비싸다), qishloq (마을)

2. **g' 발음**: 가글 소리
   - g'alaba (승리), bog' (정원), oltin (금)

3. **x 발음**: 독일어 ch
   - xona (방), xush (좋은), xizmat (서비스)

4. **r 발음**: 떠는 r
   - rahmat, rang (색), Rossiya (러시아)

## 6. 인토네이션

### 평서문
- 끝 음절 약간 하강

### 의문문 (Yes/No)
- 끝 음절 상승
- 또는 -mi 어미

### Wh-의문문
- 일반적 (강세는 의문사에)

```
Siz koreyalikmisiz? ↗ (의문 상승)
Bu yaxshi. → (평서 평탄)
Kim u? ↗ (Wh-: kim 강조)
```

## 7. 키릴 vs 라틴

라틴 (현재 표준):
```
Salom, dunyo!
```

키릴 (구식, 농촌):
```
Салом, дунё!
```

라틴 학습 권장 (정부·도시·청년 표준).

## 10. 흔한 한국어 화자 실수

1. **q → ㅋ로 발음**: 의미 다름 (qari ≠ kari)
2. **g'를 g로**: 차용어로 들림
3. **x를 h로**: 비슷하지만 다름
4. **r 약하게**: 영어 r처럼 발음 X
5. **o vs o' 무시**: 의미 헷갈림
6. **v를 ㅂ로**: 영어 v 연습
