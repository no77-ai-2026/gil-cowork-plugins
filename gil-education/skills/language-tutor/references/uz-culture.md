# 우즈벡 문화·예의 가이드

> gil-education | language-tutor 참조 | UZ 팩

## 1. 핵심 문화 코드

### 손님 환대 (Mehmondoʻstlik)

UZ 문화 1순위:
- 손님은 신성한 존재
- 차·플로프 (Plov) 환대
- 갑작스러운 방문도 환영
- 거절 어려움 → 정중한 거절 학습 필요

### 가족·공동체

- 가족 1순위
- 부모·노인 공경 (Yaxshi qarash)
- 형·누나 호칭 (aka, opa) 친구에게도 사용
- Mahalla (이웃 공동체) — 사회 기본 단위

### 종교 (이슬람)

- 무슬림 약 88% (Sunni)
- 일상 영향:
  - 술·돼지 금기 (할랄)
  - 기도 시간 (5회/일)
  - Ramadan (단식월)
  - 금요일 (Juma) 모스크 기도

## 2. 일상 예의

### 인사

- 어른: 오른손을 가슴에 대고 살짝 인사
- 또래·후배: 악수 (남성)
- 여성 간: 포옹·볼 인사
- 남성 ↔ 여성: 보수적 (악수 자제, 가슴에 손)

### 호칭

- 어른: -aka (남성), -opa (여성), -amaki (삼촌급)
- 직장: Xon, Janob, Xonim (미스터·미스)
- 친구: 이름 또는 -jon (애칭)
- 비즈니스: Siz (정중)

### 식사 예의

- 손 씻기 먼저 (특히 식전)
- 주인 권유 시 시작
- 오른손 사용 (왼손 X — 이슬람)
- Plov는 큰 접시에서 나눠 먹기 (전통)
- 칭찬: "Juda mazali!" (매우 맛있어요)
- 떠나기 전: "Rahmat" (감사) + "Olloh xayr bersin" (신의 은혜를)

### 선물

- 차·과자·과일 무난
- 가족 방문 시 작은 선물 권장
- 술·돼지 가공품 X
- 포장 (간단해도 OK)

## 3. 비즈니스 예의

### 첫 미팅

- 시간 엄수 (단, 우즈벡 시간 + 10~30분 유연)
- 정장 (남성), 보수적 정장 (여성)
- 가벼운 인사: 가족·건강 안부
- 명함: 양손 교환

### 미팅

- Siz 항상 사용
- 직급 높은 사람 우선 발언
- 조급함 회피 (관계 형성 시간 길게)
- 차 권유 받음 (거절 X — 분위기 깸)

### 식사 접대

- Plov로 환영
- 술 권유 X (이슬람)
- 식사 시간 길게 (1~2시간 일반)
- 다음 미팅으로 식사 시간 = 비즈니스 진전 신호

### 협상

- 가격 협상 일반 (시장처럼)
- 첫 제안은 높게 (협상 여지)
- 인내·관계 우선
- 한국식 빠른 결정 X (관계 → 신뢰 → 결정)

### 명함 정보

```
Korea 측:
- 한국어 + 영어
- 회사명·직급·연락처

UZ 측:
- 우즈벡 라틴 + 러시아어
- 회사명·INN·직급·연락처
- 가능하면 한국어 (한국 회사 거래 시 우대)
```

## 4. 종교 코드 (필수)

### 식사
- 할랄 (Halol) — 이슬람 율법 적합
- 술 (alkogol) ❌
- 돼지 (cho'chqa go'shti) ❌
- 식당 메뉴 확인 필수

### 옷차림 (여성)
- 어깨·팔 노출 회피 (특히 농촌)
- 무릎 이하 치마
- 머리 가림 (선택, 도시는 자율)
- 모스크 방문 시 머리 가림 + 신발 벗기

### 옷차림 (남성)
- 정장 권장 (비즈니스)
- 짧은 반바지 X (공식 자리)

### 기도 시간

- 하루 5회 (Fajr, Dhuhr, Asr, Maghrib, Isha)
- 비즈니스 미팅 중 휴식 시간 가능
- 금요일 12~14시 모스크 기도 (점심 미팅 회피)

### Ramadan (단식월)

- 한 달간 일출~일몰 단식
- 비즈니스 점심 미팅 회피
- 일몰 후 Iftar (단식 종료 식사) 초대 영광

## 5. 사회·정치 민감 주제 (회피)

- 정치 비판 (특히 정부)
- 종교 비판
- 다른 민족 비교 (러시아·한국 vs 우즈벡)
- 옛 소련 시대 평가 (다양한 의견)
- 개인 재산·소득 직접 묻기

→ 안전 주제: 가족·날씨·음식·여행·우즈벡 문화·역사·자연.

## 8. 한국 진출 회사 적응 팁

### 한국 직원

```
✅ Siz 사용 (정중)
✅ 가족 안부 인사
✅ 차 거절 X
✅ 미팅 시간 여유
✅ 할랄 식당 미리 확인
✅ 기도 시간 인지
✅ Ramadan 배려

❌ 술·돼지 권유
❌ 정치·종교 비판
❌ 시간 압박
❌ 직설적 거절
❌ 신체 접촉 (이성)
```

### UZ 직원

- 한국 기업 문화 차이 인식
- 빠른 의사결정 적응
- 호칭 (-님, -씨)
- 회식 (음주 자제 가능)
- KOTRA·KOICA 한국 문화 강좌

## 9. 한·UZ 양국 운영 시 문화 다리

### 양국 회식

- 한국 식당 (UZ에 다수) — 할랄 인증 확인
- UZ 식당 (한국에 일부)
- 트릴링구얼 메뉴

### 명절 동시 축하

- 한국: 설·추석
- UZ: Navruz, Eid

### 양국 비즈니스 캘린더

- 한국 공휴일 + UZ 공휴일 동시 고려
- Ramadan 1개월 운영 조정

## 10. 자주 하는 실수

1. **악수 (남성→여성)**: 보수적 가정·시골 X
2. **술 권유**: 이슬람 금기
3. **돼지 식당**: 할랄 확인
4. **시간 강박**: UZ는 관계 우선
5. **정치 농담**: 위험
6. **왼손 사용**: 식사·악수 X
7. **사진 (모스크 내부)**: 사전 허락
8. **여성 모델 광고**: 보수적 표현
