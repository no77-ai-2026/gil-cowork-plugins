# 한국어 화자 우즈벡어 학습 로드맵

> gil-education | language-tutor 참조 | UZ 팩 | 한·러·우즈벡 대조·한국인 특화 경로

본 스킬은 이미 UZ 특화(문화·문법·발음·어휘 reference 보유). 본 파일은 한국어 화자 특화 학습 경로를 정리.

## 1. 한국어↔우즈벡어 대조 (학습 난이도 포인트)

- **어순**: 둘 다 SOV로 유사 → 한국인에게 유리.
- **격(case)**: 우즈벡어 6격(bosh·qaratqich·tushum 등). 한국어 조사와 대응 학습 효과적.
- **동사 활용**: 인칭·시제 어미 → 한국어 화자에 비교적 친숙.
- **모음조화**: 우즈벡어에도 존재 → 한국어 경험 전이.
- **차이**: 관사 없음, 후치사 발달, 키릴/라틴 이중 표기.

## 2. CEFR 단계별 경로 (한국인 기준)

- A1~A2: 인사·자기소개·격 기초 + 라틴 표기 우선.
- B1~B2: 일상·비즈니스 회화, 키릴 병행 도입.
- C1~C2: 행정·전문 문서, 러시아어 차용어 구분.

## 3. 트릴링구얼(한·러·우즈벡) 전략

- 러시아어를 매개로 한 우즈벡어 학습(고려인·러시아어 능통자) 트랙 별도.
- 3언어 대역 어휘카드로 전이 학습.

## 4. 안티패턴

- 영어 교재 기반 학습법 그대로 적용(어순·격 장점 미활용).
- 라틴/키릴 중 하나만 학습 → 실생활 문서 독해 한계.
