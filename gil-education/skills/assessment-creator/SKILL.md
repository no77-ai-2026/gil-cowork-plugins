# 평가 도구 제작 (Assessment Creator)

## 개요

시험·평가 문제 출제 및 학습 평가 설계를 지원한다. 수능·공무원·자격증 시험 기출 분석, 모의고사 제작, Bloom's Taxonomy 기반 문항 설계, 오답 분석표 생성, 2026년 에듀테크 동향(AI 자동 채점·오답 클리닉)을 반영한 학습 평가 시스템을 제공한다.

> **한국 표준 + UZ 듀얼 컨텍스트** — 한국 시험(수능·공무원·자격증)이 기본이며, UZ·CIS 국제시험(DTM·IELTS·CEFR)은 [`references/uz-international-exams.md`](references/uz-international-exams.md) 참조.

## 트리거 키워드

시험, 기출, 자격증, 수능, 문제 출제, 평가, 퀴즈, 학습 평가, 모의고사, TOEIC, TOEFL, 공무원 시험, 내신, 형성평가, 총괄평가, Bloom's Taxonomy

## 워크플로우

### 1단계: 평가 목적 파악

**평가 유형:**
- **형성평가**: 학습 중 이해도 확인 (퀴즈, 단답형)
- **총괄평가**: 학습 후 성취도 측정 (시험, 자격증 모의고사)

**대상 시험 종류:**
- 수능 (대학수학능력시험)
- 공무원 시험 (행정·기술직)
- 자격증 시험 (정보처리기사, 공인중개사 등)
- 어학 시험 (TOEIC, TOEFL, OPIC)
- 내신 (학교 성취도 평가)

### 2단계: 출제 범위 설정

**학습 목표 설정:**
- 단원, 주제, 핵심 개념 정의
- 난이도 비율 결정 (상/중/하)
- Bloom's Taxonomy 수준별 배분

**Bloom's Taxonomy 기반 문항 설계:**
| 인지 수준 | 설명 | 문항 유형 예시 |
|-----------|------|---------------|
| 기억(Remember) | 사실 암기 | 용어 정의, 연도 매칭 |
| 이해(Understand) | 개념 설명 | 요약, 비교, 분류 |
| 적용(Apply) | 실제 사용 | 문제 해결, 계산 |
| 분석(Analyze) | 구조 파악 | 원인 분석, 관계 파악 |
| 평가(Evaluate) | 판단·비판 | 타당성 평가, 비판 |
| 창조(Create) | 새로운 산출 | 설계, 작문, 프로젝트 |

### 3단계: 문항 유형 선택

**선택형:**
- 4지선다형 (일반)
- 5지선다형 (공무원, 수능)

**단답형·서술형:**
- 단답형 (키워드)
- 단순 서술형 (2-3줄)
- 논술형 (장문)

**OX형·빈칸 채우기:**
- OX 퀴즈
- 빈칸 채우기

**혼합 구성:**
- 선택형 70% + 단답형 20% + 서술형 10%

### 4단계: 문항 작성

**문항 작성 원칙:**
- 학습 목표와 직접 연결
- 명확하고 모호하지 않은 문장
- 적정 난이도 (50-70% 정답률 목표)
- 오답 선택지는 논리적 오류 유형

**정답 및 해설 포함:**
- 정답 표시
- 해설 (오답 선택지 why wrong 포함)

### 5단계: 검토 및 오답 분석표

**문항별 핵심 개념 연결:**
- 해당 문항이 평가하는 핵심 개념
- 관련 단원·주제

**오답 시 보충 학습 방향:**
- 오답 선택지별 오류 유형 분석
- 보충 학습 링크·자료 추천

## 2026년 한국 평가 트렌드

### AI 기반 평가 자동화

에듀테크 코리아 페어 및 AI코스웨어 동향 반영:

- **자동 채점**: AI가 단답형·서술형 문항 자동 채점 및 피드백 제공
- **문항 자동 생성**: AI가 교육 목표에 맞는 문항 생성 (단순 기억 → 고차원 사고 문항 포함)
- **개인별 오답 클리닉**: 학습자별 오답 패턴 분석 → 맞춤형 보충 문제 추천
- **역량 맵핑**: 문항별 Bloom's Taxonomy 수준 레이블링으로 학습 깊이 시각화

### 한국 주요 시험별 특성

| 시험 | 특징 | AI 활용 포인트 |
|------|------|---------------|
| 수능 | 국·수·영·탐구 선택형+서술형 혼합 | 기출 패턴 분석, 약점 진단 |
| 공무원 시험 | 5지선다, 빠른 정확도 | 반복 암기 자동화, 모의고사 |
| TOEIC/TOEFL | 리딩·리스닝·스피킹·라이팅 | AI 발음 교정, 에세이 피드백 |
| 자격증 | 종목별 상이, 실기 포함 | 이론 문제 자동 생성, 실기 준비 체크리스트 |
| 내신 | 학교별 출제 기준 다양 | 교과서 기반 예상 문제, 수행평가 준비 |

## 문항 예시·이원목적분류표

### 이원목적분류표 (Test Blueprint) 양식

출제 전 "영역 × 인지수준 × 문항수·배점"을 표로 고정해 내용타당도를 확보합니다.

| 영역 | 기억 | 이해 | 적용 | 분석+ | 문항수 | 배점 |
|---|---|---|---|---|---|---|
| 단원1 | 2 | 2 | 1 | 1 | 6 | 18 |
| 단원2 | 1 | 2 | 2 | 1 | 6 | 18 |
| … | | | | | | |
| 합계 | 5 | 7 | 6 | 2 | 20 | 100 |

### 문항 유형별 작성 예시

**선택형(5지선다, 적용)**
```
문) 다음 중 함수의 시간복잡도가 O(n log n)인 정렬은?
① 버블 정렬  ② 선택 정렬  ③ 병합 정렬  ④ 삽입 정렬  ⑤ 계수 정렬
정답: ③  | 난이도: 중 | 인지수준: 적용
오답 매력도: ①②④는 O(n²) 혼동 유도, ⑤는 비교정렬 아님(범위 제약)
```

**서답형(단답, 이해)**: "정규화의 목적을 한 문장으로 쓰시오." (모범답안·부분점수 기준 함께 제시)

**수행평가(분석/창조)**: "주어진 데이터셋으로 가설→분석→결론을 1쪽 보고서로." → 분석적 루브릭 4차원(문제정의·방법·해석·표현) 적용.

### 오답 분석표 양식

| 문항 | 정답률 | 변별도 | 최다 오답 | 추정 원인 | 조치 |
|---|---|---|---|---|---|
| 3 | 41% | 0.18 | ① | 복잡도 개념 혼동 | 보충·문항 재검토 |

## 사용 예시

**예시 1**: "정보처리기사 실기 모의문제 10개를 만들어줘"
→ 단원 선택 → 난이도 배분 → 문항 생성 → 정답 및 해설

**예시 2**: "이 단원에서 나올 수 있는 내신 예상 문제를 출제해줘"
→ 학습 목표 확인 → Bloom's 수준 배분 → 혼합 문항 출제

**예시 3**: "TOEIC 700점 목표 4주 학습 계획을 세워줘"
→ 현재 점수 확인 → 약점 분석 → 주차별 학습 계획 → 모의고사 스케줄

**예시 4**: "수강생 이해도를 평가할 형성평가 문항을 설계해줘"
→ 학습 목표 확인 → 퀴즈 문항 생성 → 오답 분석표

## 문항 설계·타당도 심화

좋은 평가는 "측정하려는 것을 정확히·일관되게" 재야 합니다.

- **문항 유형 선택** — 선택형(지식·이해 효율), 서답형(적용·분석), 수행평가(종합·창의). 목표 인지수준에 맞춰 혼합.
- **루브릭** — 분석적(차원별 점수, 피드백 풍부) vs 총체적(전체 인상, 채점 빠름). 수행평가엔 분석적 권장.
- **타당도** — 내용타당도(출제 영역 대표성·이원목적분류표), 구인타당도(측정 개념 일치).
- **신뢰도** — 문항 난이도(P)·변별도(D), 내적 일관성(Cronbach α). 변별도 낮은 문항 교체.
- **Bloom 정렬** — 기억→이해→적용→분석→평가→창조 분포를 의도적으로 설계.
- **보편적 학습설계(UDL)·편의제공** — 시간 연장·대체 양식 등 접근성 반영.
