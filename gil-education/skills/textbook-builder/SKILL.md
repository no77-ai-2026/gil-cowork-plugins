# 교재 제작 (Textbook Builder)

> gil-education | 학습자 최적화 교재 자동 제작

> **한국 표준 + UZ 듀얼 컨텍스트** — 한국 교육과정 기반 교재가 기본이며, 트릴링구얼(한·러·우즈벡) 교재·키릴/라틴 병기·UZ 교육과정·고려인 학습자 대상 설계는 [`references/uz-textbook.md`](references/uz-textbook.md) 참조. (UZ 트리거: "트릴링구얼 교재", "우즈벡어 입문 교재", "고려인 학습자 교재")

## 역할

학습자 수준·목표·시간에 최적화된 교재 제작. 단원·차시·평가·교사 지도서·시각 자료까지. 분야 무관 (어학·수학·과학·사회·인문·자격증·실용·우즈벡어).

## 트리거 키워드

교재, 교과서, 학습 자료, 워크북, 코스북, textbook, workbook, 교사 지도서,
학습 가이드, 자기 학습 교재, 어학 교재, 자격증 교재, 사내 교육 교재

## 워크플로우

### Step 1: 학습자 분석 (필수)

```
"학습자 정보를 알려주세요:"

- 대상자 (연령·직군·수준)
- 사전 지식 (전무·기초·중급·고급)
- 학습 목표 (자격증·일상·취미·승진·시험·직무)
- 학습 시간 (시간/주, 총 기간)
- 학습 환경 (온라인·오프라인·자기 학습·강사 지도)
- 동기·관심사
- 언어 (한국어·러시아어·우즈벡어·영어)
- 시장 (한국 / UZ / 양국)
```

### Step 2: 교재 구조 설계

#### 표준 단원 구조

```
[단원 1: 도입]
- 단원 목표 (Bloom)
- 사전 평가 (진단)
- 핵심 어휘·개념
- 학습 로드맵

[단원 2~N: 본문]
각 단원별:
- 학습 목표
- 도입 (관심 유발·실생활 연결)
- 개념 설명
- 예제 (단계별)
- 연습 문제 (난이도 점진)
- 심화 학습 (선택)
- 단원 평가
- 정리·요약

[단원 N+1: 종합]
- 종합 평가
- 부록 (해답·용어 사전·참고)
```

#### 단원 수 가이드

| 학습 시간 | 단원 수 | 단원당 시간 |
|----------|--------|-------------|
| 10시간 | 5 | 2시간 |
| 30시간 | 8~12 | 2.5~3시간 |
| 60시간 | 12~16 | 3~5시간 |
| 100시간+ | 20~30 | 3~5시간 |
| 학기제 (16주) | 16 | 1주 |

### Step 3: 학습 목표 (Bloom 분류법)

각 단원·차시별:

| Bloom 수준 | 동사 예시 | 활동 |
|----------|---------|------|
| 기억 | 정의·나열·이름 | 플래시카드·퀴즈 |
| 이해 | 설명·요약·비교 | 챗봇·요약 작성 |
| 적용 | 사용·실행·구현 | 실습·시뮬레이션 |
| 분석 | 분류·구분·비교 | 데이터 분석·도표 |
| 평가 | 판단·비판·정당화 | 토론·피어 리뷰 |
| 창조 | 설계·창작·생성 | 프로젝트·포트폴리오 |

### Step 4: 콘텐츠 작성

#### 개념 설명 (Conceptual)

- 정의 (간결)
- 예시 (구체)
- 비유·은유 (이해 ↑)
- 시각 자료 (그림·도표·플로우차트)
- FAQ (오개념 사전 해소)

#### 예제 (Worked Examples)

- 단계별 풀이
- 핵심 사고 과정 명시
- 변형 가능 (어떤 변화 시 어떻게 적응)

#### 연습 문제 (Practice)

- 난이도 점진 (기초·중급·심화)
- 오답 해설 (학습 강화)
- 실생활 적용 문제

#### 심화 학습 (Enrichment)

- 추가 자료
- 외부 링크
- 연구 과제·프로젝트

### Step 5: 평가 문제

`assessment-creator` 연계 또는 자체:
- 단원별 형성평가
- 종합평가
- 자기 평가 체크리스트
- 피어 평가 양식

### Step 6: 시각 자료

- 그림·도표·플로우차트
- 인포그래픽
- 표·매트릭스
- AI 이미지 생성 (`gil-media:nano-banana` 또는 `ideogram`)
- 비디오 스크립트 (`gil-content:media-production`)

### Step 7: 교사용 지도서

```
- 단원별 차시 계획
- 핵심 학습 목표
- 교수 방법 (강의·토론·실습·플립러닝)
- 보조 자료
- 평가 기준 (rubric)
- 학습자 오개념·해결
- 시간 배분
- 답안·해설
- 활용 팁
```

### Step 8: 형식·출력

- DOCX (`gil-office:docx-generator`)
- PPTX (`gil-office:pptx-designer`)
- HWPX (`gil-office:hwpx-writer` — 한국 전용)
- PDF (DOCX 변환)
- HTML (온라인 코스)

## 분야별 교재 템플릿

### 어학 (영어·일본어·중국어·러시아어·우즈벡어·한국어)

```
[Lesson 1: Greetings]
- 학습 목표 (CEFR A1)
- 도입 (실생활 시나리오)
- 핵심 어휘 (10~15개)
- 핵심 표현 (5~8개)
- 발음·억양 (오디오 가이드)
- 문법 포인트 (1~2개)
- 예시 대화 (3~5개)
- 연습 (말하기·듣기·읽기·쓰기 4영역)
- 문화 코드 (인사·예의)
- 단원 평가
```

### 수학·과학·이공계

```
[Chapter 1: Introduction]
- 학습 목표 (구체·측정 가능)
- 사전 진단 (선수 학습 점검)
- 핵심 개념 (정의·공식)
- 예제 (단계별 풀이)
- 연습 (기초·중급·심화)
- 실생활 응용
- 시각 자료 (그래프·도식)
- 단원 평가 (객관식·주관식·서술)
```

### 자격증 (정보처리·공무원·국가시험)

```
[Section 1: 출제 영역 X]
- 출제 범위·비중
- 핵심 이론 (압축)
- 기출 문제 (5~10개)
- 예상 문제 (5~10개)
- 오답 분석
- 핵심 요약 (시험 전 5분 복습)
- 모의고사 (단원 종합)
```

### 인문·사회

```
[Chapter 1: Topic]
- 도입 (사례·질문)
- 핵심 개념·이론
- 사례 분석
- 비판적 사고 질문
- 토론 주제
- 추가 읽기 (주요 저작)
- 에세이 과제 (Bloom 5~6단계)
```

### 실용·취미·자기 계발

```
[Module 1: Foundation]
- Why this matters (동기)
- Core skill
- Practice (5~10분 일일)
- Real-life application
- Self-assessment
- Next steps
```

## 학습자 유형별 최적화

- 짧은 단위 (10~20분)
- 시각 자료 ↑
- 게임·놀이 통합
- 반복·복습
- 칭찬·보상 시스템

### 청소년 (13~18세)

- 실생활 연결

- 시험 대비 명시

### 성인 (19+)

- 시간 효율 (응용 우선)
- 직무 연결
- 자기 평가 도구
- 배경 지식 활용

### 시니어 (60+)

- 큰 글자·여백
- 단계별 천천히
- 일상 연결
- 사회적 활동 통합

## 학습자 차이 (한국 vs UZ)

### 한국 학습자
- 시험·자격증 중심
- 정확한 정답 선호
- 문법·구조 강세

### UZ 학습자
- 다국어 환경 (러시아어·우즈벡어·영어)

### UZ에서 한국어 학습 (TOPIK)

- TOPIK I·II 단계별

## 산출 핸드오프 (docx/pdf)·교사용 지도서 양식

본 스킬은 교재 **구조·콘텐츠**를 설계하고, 실제 파일은 office 스킬로 핸드오프합니다.

| 산출물 | 핸드오프 대상 | 비고 |
|---|---|---|
| 학습자용 교재(.docx) | `gil-office:docx-generator` | 단원 구조·표·삽화 자리 표시 |
| 트릴링구얼 교재(.pdf) | `gil-office:pdf-writer` | 키릴/라틴·CJK 폰트 임베딩 |
| 슬라이드(강의용) | `gil-office:notebooklm-slide-prompt` | 교재≠슬라이드, 별도 산출 |

핸드오프 시 넘기는 구조화 입력(예): `{단원: [{제목, 목표(Bloom), 본문, 예제, 연습문제, 정리}], 부록: [...]}`.

### 교사용 지도서(Teacher's Guide) 양식

```
단원 N 지도안
- 학습 목표 / 선수 학습 / 소요 차시
- 도입(동기유발) · 전개(핵심활동·발문) · 정리(형성평가)
- 예상 오개념 & 처치
- 평가 연계 (assessment-creator 문항 매핑)
- 차등 지원(빠른/느린 학습자)
```
