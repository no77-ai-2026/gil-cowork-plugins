# 분야별 교재 템플릿

> gil-education | textbook-builder 참조

## 1. 어학 (영어·일본어·중국어·러시아어·우즈벡어·한국어)

### 단원 구조 (Lesson)

```
[Lesson X: Title]
Theme: [주제]
CEFR Level: [A1·A2·B1·...]
Time: [X 시간]

1. Lesson Objectives
   - 학습 목표 (Bloom)
   - Can-do statements

2. Warm-up (5분)
   - 사전 지식 활성화
   - 관심 유발

3. Vocabulary (15분)
   - 핵심 어휘 10~15개
   - 발음·예시·Translation

4. Key Expressions (10분)
   - 핵심 표현 5~8개
   - 상황별 활용

5. Grammar Point (15분)
   - 1~2개 문법 포인트
   - 규칙·예시·연습

6. Dialogue / Reading (15분)
   - 대화 또는 읽기
   - 듣기 (오디오)
   - 이해 확인

7. Practice (20분)
   - 듣기·말하기·읽기·쓰기 4영역
   - 짝 활동·그룹 활동

8. Cultural Note (5분)
   - 문화 코드
   - (UZ) 이슬람·예의

9. Summary (5분)
   - 핵심 정리

10. Assessment
    - 단원 평가 (10~20문항)
```

### 어학 교재 표준 분량

| CEFR | 단원 수 | 페이지 |
|------|--------|--------|
| A1 | 15~20 | 150~200 |
| A2 | 18~25 | 200~280 |
| B1 | 20~30 | 250~350 |
| B2 | 25~35 | 300~450 |

## 2. 수학·이공계

### 단원 구조 (Chapter)

```
[Chapter X: Title]

1. Learning Objectives

2. Pre-Assessment
   - 선수 학습 점검

3. Concept Introduction
   - 정의
   - 예시 (구체)
   - 시각 자료

4. Worked Examples (3~5개)
   - 단계별 풀이
   - 핵심 사고
   - 변형

5. Practice
   - 기초 (10개)
   - 중급 (10개)
   - 심화 (5개)

6. Real-World Application
   - 실생활 연결
   - 사례

7. Summary
   - 핵심 공식·개념

8. Chapter Test
   - 객관식 + 서술 (15~25문항)
   - 정답·해설
```

## 3. 자격증 (정처기·공무원·CFA·CISA 등)

### 단원 구조 (Section)

```
[Section X: 출제 영역 X]

1. 출제 비중·중요도
   - 회당 X문항
   - 난이도

2. 핵심 이론 (압축)
   - 필수 개념
   - 공식·법칙
   - 시각 자료

3. 기출 패턴 분석
   - 자주 출제 유형
   - 출제 트렌드

4. 기출문제 (5~10개)
   - 정답·해설

5. 예상문제 (10~15개)
   - 난이도 일치
   - 정답·해설

6. 핵심 요약 (시험 전 5분 복습)

7. 약점 진단·보충 학습 가이드

8. 모의고사 (단원 종합)
```

## 4. 인문·사회

### 단원 구조 (Chapter)

```
[Chapter X: Topic]

1. Learning Objectives

2. Introduction
   - 사례·질문·통계 (관심 유발)

3. Theoretical Background
   - 핵심 개념·이론
   - 주요 학자

4. Case Studies (2~3개)
   - 사례 분석
   - 비판적 사고 질문

5. Discussion Questions
   - 토론 주제
   - 다양한 관점

6. Critical Thinking
   - 비판·반론
   - 대안

7. Further Reading
   - 주요 저작 추천
   - 관련 학술지

8. Essay Prompts
   - 에세이 과제 (Bloom 5~6)
```

## 5. 실용·취미·자기 계발

### 단원 구조 (Module)

```
[Module X: Foundation]

1. Why this matters
   - 동기·이익
   - 시간 ROI

2. Core Skill
   - 핵심 기술 1개
   - 단계별

3. Daily Practice (5~10분)
   - 일일 루틴
   - 습관 형성

4. Real-Life Application
   - 실생활 적용
   - 사례

5. Self-Assessment
   - 진도 체크
   - 다음 단계

6. Community
   - 연결·공유
```

## 6. 어린이 (5~12세)

### 단원 구조 (Mini-Lesson)

```
[Lesson X: Theme]

1. Story / Picture (스토리·그림)

2. Words (5~7개 큰 글자)

3. Activity 1 (5분)
   - 게임·놀이

4. Practice (5분)
   - 색칠·그리기·짝 맞추기

5. Activity 2 (5분)
   - 노래·동작

6. Review (자기 평가 — 스티커)

7. 부모용 가이드
```

## 9. 평가 형식별

### 객관식 (4지·5지)
- 정답 1개 명확
- 오답 정교 (혼동)
- 부정형 회피

### 단답형
- 한 단어·한 문장 답
- 명확한 채점 기준

### 서술형
- 채점 rubric 필수
- 모범 답안 + 부분 점수

### 수행 평가
- 프로젝트·발표
- Rubric 다차원 (내용·구조·발표 등)

## 10. 교재 검수 단계

```
1. 자체 검수 (저자)
2. 동료 검수 (다른 강사)
3. 학습자 시범 (5~10명 베타)
4. 피드백 반영
5. 교사 훈련
6. 출판·배포
```
