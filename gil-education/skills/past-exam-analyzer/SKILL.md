# 기출 분석 + 예상문제 (Past Exam Analyzer)

> gil-education | 기출 패턴 분석·다음 회차 예측·예상문제 자동 생성

> **한국 표준 + UZ 듀얼 컨텍스트** — 수능·공무원·자격증·TOEIC 등 한국 시험이 기본이며, UZ·CIS 시험(DTM 국가시험·UZ 대학입시·IELTS/CEFR·러시아어 시험) 기출 패턴은 [`references/uz-exams.md`](references/uz-exams.md) 참조. (UZ 트리거: "DTM 기출 분석", "UZ 대학입시 패턴")

## 역할

기출문제 입력 → 패턴 분석 → 다음 회차 출제 예측 → 예상문제 자동 생성. 학습자 약점·오답 패턴 분석. 한국·UZ·국제 시험 모두 지원.

## 트리거 키워드

기출, 기출문제, 기출 분석, 출제 패턴, 다음 회차 예상, 예상문제, 예상 문제,
출제 트렌드, 약점 분석, 오답 분석, 시험 트렌드, 출제 영역

## 워크플로우

### Step 1: 시험 정보 수집

```
"분석할 시험은?"

[시험명·회차·연도]
- 한국: 수능, 공무원, 자격증 (정처기·CISA·...), CSAT
- UZ: DTM
- 국제: TOEIC, TOEFL, IELTS, TOPIK, TORFL, SAT, GRE, HSK

[기출 데이터]
- 연도·회차·문항·정답
- 텍스트 입력 또는 파일
- 또는 출제 영역만 (구조 분석)

[목표]
- 출제 패턴만 vs 예상문제 생성도 vs 학습자 약점 진단도
```

### Step 2: 출제 패턴 분석

#### 영역별 출제 빈도

```
[수능 국어 5개년]

| 영역 | 2020 | 2021 | 2022 | 2023 | 2024 | 평균 |
|------|------|------|------|------|------|------|
| 화법·작문 | 11 | 11 | 11 | 11 | 11 | 11.0 |
| 언어 | 11 | 11 | 5 | 5 | 5 | 7.4 |
| 독서 | 17 | 17 | 17 | 17 | 17 | 17.0 |
| 문학 | 17 | 17 | 17 | 17 | 17 | 17.0 |
| 선택 | - | - | 11 | 11 | 11 | 11.0 |
```

#### 난이도 분포

```
[난이도]
- 상: XX% (시간 ↑·정확도 ↓)
- 중: XX% (균형)
- 하: XX% (시간 ↓·정확도 ↑)

[유형]
- 사실적 이해: XX%
- 추론적 이해: XX%
- 비판적 이해: XX%
- 창의적 사고: XX%
```

### Step 3: 트렌드 분석 (시간선)

```
[연도별 변화]
2020: 새 유형 출현 (XX)
2021: 난이도 ↑
2022: 영역 개편 (XX 신설)
2023: AI·디지털 주제 ↑
2024: 융합 문항 ↑
```

### Step 4: 다음 회차 예상

```
[2026 출제 예상]

영역별 출제 확률:
- 영역 A: ★★★★★ (5년 연속, 비중 ↑)
- 영역 B: ★★★★☆ (4년 연속)
- 영역 C: ★★☆☆☆ (격년 출제, 2024 미출 → 2025 가능 ↑)

신유형 가능성:
- 융합형 (영역 A + B): ★★★★☆
- AI·디지털 주제: ★★★★★

예상 난이도:
- 평년 수준 또는 ↑ (변별력 강화)

예상 시간 배분:
- 영역 A: XX분
- 영역 B: XX분
```

### Step 5: 예상문제 자동 생성 (NEW 핵심)

#### 생성 원칙

- 기출 난이도·유형 일치
- 출제 영역 비중 따름
- 신유형 가능성 반영 (10~20%)
- 정답·오답 모두 정교
- 해설 충실

#### 생성 옵션

```
"예상문제 몇 개 생성?"

○ 10개 (소형 모의고사 1회)
○ 30개 (중형 1회)
○ 50개 (대형 1회)
○ 100개 (학습용 풀)
○ 200개+ (장기 학습)
+ Other

"난이도 비율?"
- 상:중:하 = 2:5:3 (평년)
- 또는 사용자 지정
```

#### 출력 형식

```
[예상문제 1]
[유형: 사실적 이해]
[난이도: 중]
[예상 영역: 독서]
[예상 출제 확률: 85%]

[지문]
... (기출 스타일·길이 일치)

[문제]
다음 중 ~에 해당하는 것은?

① 선택지 1
② 선택지 2 (정답)
③ 선택지 3
④ 선택지 4
⑤ 선택지 5

[정답] ②

[해설]
② 가 정답인 이유:
- 핵심 근거 (지문 X 줄)
- 키워드 매칭

다른 선택지 오답 이유:
① ... (혼동 포인트)
③ ... 
④ ...
⑤ ...

[학습 포인트]
- 핵심 개념: ...
- 비슷한 기출: 2023학년도 X번
- 약점 시 보충: ...
```

### Step 6: 학습자 약점 분석 (입력 시)

```
[학습자 모의고사 결과 입력]
정답: 80/100
오답 패턴:
- 영역 A: 5/20 오답 (25% 오답률 - 약점)
- 영역 B: 1/30 오답 (3% - 강점)
- 영역 C: 14/50 오답 (28% - 약점)

[진단]
1. 영역 A·C가 우선 보강 필요
2. 오답 유형: 추론적 이해 (개념 정확하나 적용 약점)
3. 시간 관리: 영역 A에 시간 초과

[처방]
- 영역 A 집중 학습 (10시간)
- 추론적 이해 유형 30개 추가 풀이
- 모의고사 시 영역 A 시간 제한
```

## 입력 모드 (데이터 유무 분기)

기출 원문 확보 여부에 따라 산출 깊이가 달라집니다. 시작 시 모드를 확정하세요.

| 모드 | 입력 | 산출 | 한계 |
|---|---|---|---|
| A. 데이터 기반 | 연도·회차·문항·정답 텍스트/파일 | 영역별 빈도·난이도 추세·정량 예측·예상문제 | 표본 부족 시 신뢰구간 명시 |
| B. 구조 분석(데이터 無) | 시험명·출제 영역·공식 시험요강만 | 출제 영역 구조·유형 분류·일반 전략·학습 로드맵 | 정량 빈도·구체 예측 불가(추정 명시) |
| C. 부분 데이터 | 일부 회차만 | A+B 혼합, 가용 구간만 정량 | 외삽 주의 |

**원칙**: 데이터가 없으면 정량 수치를 만들어내지 말고 **B 모드로 명시**한다(허위 통계 금지). 저작권상 기출 원문 전체 복제 대신 사용자 보유 데이터 입력을 받는다.

## 시험별 분석 가이드

### 한국 수능

- 5개년 기출 (2020~2024)
- 교과서·EBS 연계율
- 변별력 문항 패턴
- 신유형 (2024 통합사회 등)

### 한국 공무원

- 9·7급 기출 5~10개년
- 직렬별 (행정·교정·세무 등)
- 영역별 (국어·영어·한국사·전공 4과목)
- 난이도 평준화 트렌드

### 한국 자격증

- 종목별 (정처기·CISA·CFA·약사·간호사 등)
- 회차별 출제 영역
- 합격률 트렌드
- 필기 + 실기

### TOEIC·TOEFL·IELTS

- 영역별 (Reading·Listening·Speaking·Writing)
- 연도별 출제 트렌드
- 핵심 어휘 (자주 출제)
- 시간 관리 패턴

### TOPIK (한국어)

- I (1·2급) + II (3·4·5·6급)
- 영역별 (듣기·읽기·쓰기)
- 토픽별 출제 빈도

### TORFL (러시아어)

- 6단계 (Базовый ~ ТЭУ)
- 영역별 (Лексика·Грамматика·Чтение·Аудирование·Письмо·Говорение)

### DTM (UZ 대학 입시)

- 분야별 (수학·러시아어·역사·...)
- 다국어 출제 (러시아어·우즈벡어·영어 옵션)

### SAT·GRE·HSK

- 표준화 시험 패턴
- AI·자동 생성 강세

## 출제 예측 모델

### 통계 기반

- 영역별 5년 출제 빈도 평균
- 표준편차
- 격년 출제 패턴 (2년 미출 → 출제 확률 ↑)
- 시기별 트렌드 (계절·정책)

### 정성 기반

- 시험 출제 위원회 정책
- 사회·시사 이슈 (수능)
- 교육과정 개편 (2025·2027 등)
- AI·디지털 주제 ↑

### AI 보조

- 텍스트 분석 (출제 키워드)
- 문항 클러스터링
- 난이도 자동 분류 (Bloom)
