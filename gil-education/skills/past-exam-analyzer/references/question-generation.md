# 예상문제 자동 생성 가이드

> gil-education | past-exam-analyzer 참조

## 생성 원칙

### 1. 기출 일치
- 출제 영역 비중 일치
- 난이도 분포 일치
- 유형 (객관식·서술) 일치
- 시간 (문항당 풀이 시간) 일치

### 2. 신유형 (10~20%)
- 출제 트렌드 반영
- 정책 변화 반영
- 융합 가능성

### 3. 품질
- 정답 명확 (논란 X)
- 오답 정교 (혼동 유발)
- 해설 충실
- 학습 가치 (개념 강화)

## 객관식 문항 작성

### 좋은 객관식 5요소

1. **줄기 (Stem)**: 명확·구체적
   - "다음 중 ~로 옳은 것은?" (긍정형 권장)
   - 부정형 ("~이 아닌 것은?") 회피 또는 명시

2. **정답 1개**: 명확

3. **오답 (3~4개)**: 정교
   - 정답과 혼동 가능
   - 문법·길이 균일
   - "모두" "절대" 회피

4. **분포**: 정답 1·2·3·4·5번 균등

5. **언어**: 학습자 수준 적합

### 예시 (수능 국어)

```
[지문]
... [기출 스타일 본문]

[문제]
다음 중 글의 중심 내용으로 가장 적절한 것은?

① ... (오답 1: 너무 좁음)
② ... (오답 2: 너무 넓음)
③ ... (정답: 핵심 압축)
④ ... (오답 3: 부정확)
⑤ ... (오답 4: 부분만)

[정답] ③

[해설]
③ "..."는 글 전체 핵심을 요약.
지문 X 단락 (X행)에서 ... 으로 명시.

[오답 분석]
① 부분만 (단락 2)
② 너무 넓어 글 범위 초과
④ 사실 틀림 (단락 3 참조)
⑤ 일부 사례만

[학습 포인트]
- 중심 내용 찾기 = 글 전체 vs 부분 구분
- 비슷한 기출: 2023학년도 X번
```

## 서술형 문항 작성

### 좋은 서술형

```
[질문]
[조건] (분량·관점·근거)

[모범 답안]
[채점 rubric]
```

### 예시

```
[질문]
인공지능이 교육에 미치는 영향을 학습자·교사·정책 관점에서 각 1단락(약 200자)으로 서술하시오.

[모범 답안]
[학습자] AI는 개인 맞춤 학습을 가능케 한다. 학습자의 수준과 진도에 맞춰 ...

[교사] AI는 반복적 업무를 자동화하여 교사가 ...

[정책] AI 도입은 교육 격차 해소와 디지털 윤리 ...

[채점 rubric]
- 학습자 관점 (5점)
- 교사 관점 (5점)
- 정책 관점 (5점)
- 논리·구체성 (3점)
- 분량·형식 (2점)
총 20점

[감점 사항]
- 분량 미달 (-3점)
- 관점 누락 (-5점/관점)
- 사실 틀림 (-5점)
```

## 시험별 생성 패턴

### 수능 국어

- 지문 + 문제 5~6개
- 비문학 / 문학 / 화법·작문
- 지문 길이 1,000~2,000자

### TOEIC

- Part 1·2·3·4 (LC) / Part 5·6·7 (RC)
- Part 7: 단일 / 이중 / 삼중 지문
- 비즈니스 시나리오

### TOPIK 쓰기

- 51-52: 짧은 글 (각 200~300자)
- 53: 그래프 분석 200~300자
- 54: 논술 600~700자

### IELTS Writing

- Task 1: 그래프·차트 150단어
- Task 2: 에세이 250단어

### 자격증 (정처기 실기)

- 코드 작성
- DB 설계
- 시나리오 분석

## 자동 생성 워크플로우

```
1. 기출 분석 (exam-pattern-analysis 활용)
2. 영역별 출제 비중 매트릭스
3. 난이도 분포 결정
4. 유형 결정
5. 각 문항 생성:
   - 지문/배경 (필요 시)
   - 줄기
   - 정답 + 오답
   - 해설
6. 자체 점검 (정답 명확·오답 정교)
7. 학습자 시범 (베타)
8. 출판·활용
```

## 문항 품질 체크

```
[ ] 정답 1개 명확
[ ] 오답 정교 (혼동 유발 + 학습 가치)
[ ] 줄기 명확·구체
[ ] 부정형 회피 (또는 명시)
[ ] 문법·길이 균일
[ ] 학습자 수준 적합
[ ] 해설 충실
[ ] 학습 포인트 제공
[ ] 비슷한 기출 연결
[ ] 저작권 무관 (새 생성)
```
