# UZ·CIS 시험 기출 컨텍스트

> gil-education | past-exam-analyzer 참조 | DTM·UZ 대학입시·IELTS·러시아어 시험

한국 시험(수능·공무원·자격증·TOEIC)은 본문 워크플로우 그대로. UZ·CIS 시험에만 본 보강 적용.

## 1. 주요 UZ 시험 체계

- **DTM(국가시험센터, Davlat Test Markazi)** — UZ 대학입시·공무원·자격 표준화 시험 주관.
- **대학입시** — 블록(blok)별 과목 시험, 과목 조합이 학과별로 지정. 객관식 중심.
- **국제 시험** — IELTS·CEFR(우즈벡 정부 외국어 인증과 연동), SAT(사립·국제대).
- **러시아어권** — ЕГЭ(러시아 통합국가시험) 참고 가능(고려인·러시아어 학교).

## 2. 기출 분석 시 UZ 특이점

- 출제 언어: 우즈벡어(라틴/키릴) + 러시아어 병행 → 언어별 난이도·번역 정합성 점검.
- 과목 블록 구조 → "다음 회차 예측"은 블록 조합·과목 비중 변화를 추적.
- 공개 기출 접근성이 한국보다 제한적 → 표본 부족 시 신뢰구간 명시.

## 3. 예상문제 생성 시

- 우즈벡어/러시아어 문항은 생성 후 현지 검수 1회 필수(직역 금지).
- CEFR 레벨 태깅으로 난이도 정렬.

## 4. 안티패턴

- 수능식 분석틀을 DTM 블록 구조에 그대로 → 과목 비중 오판.
- 기계번역 문항을 검수 없이 사용.
