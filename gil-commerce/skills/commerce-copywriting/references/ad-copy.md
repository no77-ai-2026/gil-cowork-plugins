# 광고 카피 가이드 (네이버·구글·Meta·카카오)

## 1. 네이버 검색광고

### 형식
- **타이틀**: 15자 이내 (모바일에서 잘림 위험)
- **설명**: 45자 이내
- 표시 URL: 도메인만

### 좋은 패턴
```
타이틀: "X200 무선이어폰 30시간 배터리"
설명: "ANC + 통화 노이즈 제거. 직장인 베스트셀러. 무료배송 + 30일 환불"
```

### 핵심 룰
- 키워드 매칭 점수가 노출에 결정적
- 타이틀 첫 단어가 매칭 키워드와 일치하면 우대
- 숫자 포함 (배터리·시간·할인율)
- 구체적 혜택 (무료배송·환불 보장)

### 금지
- "최저가", "1위", "최고" (객관 근거 없을 시 거절)
- 경쟁사 브랜드명
- 특수문자 남용

## 2. 구글 검색광고 (Responsive Search Ads)

### 형식
- **헤드라인**: 30자 × 최대 15개 (구글이 자동 조합)
- **설명**: 90자 × 최대 4개

### 좋은 패턴
```
헤드라인 1: "30시간 배터리 무선이어폰"
헤드라인 2: "직장인 베스트셀러 X200"
헤드라인 3: "ANC + 통화 노이즈 제거"
헤드라인 4: "무료배송 + 30일 환불 보장"
헤드라인 5: "정가 14만 9천 → 9만 9천 (한정)"

설명 1: "통근 1주일을 한 번 충전으로. ANC와 IPX5 방수까지. 직장인 인생템 X200."
설명 2: "30일 환불 보장 + 무료배송. 4.7점 평점 8천 후기. 오늘만 정가 -30%."
```

### 핵심 룰
- 헤드라인 다양화 (구글이 A/B 테스트)
- 키워드 + USP + 혜택 + 신뢰 요소 다 포함
- "Pin" 기능: 특정 헤드라인을 첫 번째에 고정 가능

## 3. Meta 광고 (Facebook·Instagram)

### 형식
- **메인 텍스트**: 125자 이내 (모바일에서 더보기로 잘림)
- **헤드라인**: 40자 이내
- **설명**: 30자 이내 (선택)

### 좋은 패턴 (피드 광고)
```
메인 텍스트:
"30시간 배터리, ANC 탑재. 출근길 1주일을 한 번 충전으로 끝낸 X200.
정가 -30% 한정 행사 — 오늘 자정까지 ⏰"

헤드라인: "X200 무선이어폰 한정 -30%"
설명: "오늘 자정까지"
```

### 핵심 룰
- **시각이 90%, 카피는 보조**
- 첫 줄에 후크 (스크롤 멈춤)
- 이모지 1-2개 (과하면 광고 같아 보임)
- 모바일 우선 (90% 사용자가 모바일)
- "더 보기" 클릭률 < 20% — 핵심 메시지는 첫 125자 안에

### 광고 형식별 차이
| 형식 | 헤드라인 노출 | 본문 노출 |
|------|-----------|--------|
| 피드 | 작게 | 첫 125자 |
| 스토리 | 안 보임 (이미지 위 텍스트) | - |
| 릴스 | 작게 | 자막 권장 |
| 카탈로그 | 자동 (제품명) | 자동 (가격·할인) |

## 4. 카카오 모먼트 (카카오톡 광고)

### 형식
- **타이틀**: 20자 이내
- **본문**: 60자 이내
- 모바일만 노출 (카카오톡 채널)

### 좋은 패턴
```
타이틀: "30시간 무선이어폰 -30%"
본문: "직장인 베스트셀러 X200, 오늘 자정까지 정가 -30%. 무료배송."
```

### 핵심 룰
- 카카오톡 사용자 톤 (친근, 적당히 캐주얼)
- 광고 같지 않게 (네이티브 톤)
- 카카오 채널 친구 추가 유도 가능

## 5. 유튜브·인스타 릴스 (동영상 광고)

### 형식
- 길이: 6초 / 15초 / 30초 / 60초
- 사운드는 Off 기본 — 자막 필수
- 첫 3초가 결정적 (스킵 방지)

### 좋은 패턴 (15초 광고)
```
0:00~0:03 | 후크 영상 + 자막 "충전 깜빡한 출근길"
0:03~0:09 | 솔루션 시연 + 자막 "30시간 배터리 ANC X200"
0:09~0:13 | 혜택 + 자막 "-30% 한정 + 무료배송"
0:13~0:15 | CTA + 자막 "지금 보러 가기"
```

### 핵심 룰
- 자막은 큼직하게 + 컬러 대비 (모바일 시청)
- 브랜드 로고는 마지막 1-2초만
- 시청 완료율보다 첫 5초 시청 유지율이 더 중요

## 6. A/B 테스트 운영

### 채널별 권장 변형 수
- 네이버 검색: 타이틀 3-5개, 설명 3-5개 (자동 로테이션)
- 구글 RSA: 헤드라인 10-15개, 설명 4개 (구글 자동 최적화)
- Meta: 광고 세트당 헤드라인+이미지 5-10개 변형
- 카카오: 광고 세트당 카피 3-5개

### 측정
- 1주 단위로 ROAS·CTR·CPM 비교
- 최저 성과 카피 제거 후 새 변형 추가
- 4주 후 우승 카피 stabilize

## 7. 흔한 실수

- 모든 채널에 동일 카피 사용 (채널별 톤 무시)
- 너무 긴 카피 (모바일 잘림)
- 이모지 남발
- 절대 표현 ("최저가", "1위")
- 헤드라인에 브랜드명만 (검색 키워드 없음)
- 시즌 키워드 누락 (블프·설·추석 등)
