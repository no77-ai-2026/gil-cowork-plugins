# 톡톡·CS 응답 카피 가이드

## 1. 자동응답 메시지 (스마트스토어 톡톡 / 채널톡)

### 영업시간 외
```
안녕하세요, [브랜드명] 고객센터입니다.

지금은 영업시간(평일 10:00 ~ 18:00) 외 시간이라 즉시 답변이 어렵습니다.
영업일 시작 시 가장 먼저 답변 드릴게요!

급하시다면 자주 묻는 질문도 함께 살펴봐 주세요:
- 배송: 평균 1~3일, 도서산간 1~2일 추가
- 환불: 수령 후 7일 이내 무료 환불
- 교환: 사이즈·컬러 교환 가능 (왕복 배송비 부담)

감사합니다.
```

### 새 고객 환영
```
[브랜드명]을 만나주셔서 감사해요!

처음이신가요? 첫 구매 5,000원 적립금이 자동 지급되어 있습니다.
회원가입 후 [내 적립금]에서 확인하실 수 있어요.

상품 문의는 언제든 톡톡으로 보내주세요. 평일 10~18시에 답변 드립니다.
```

### FAQ 자동 안내
```
자주 묻는 질문 중 답이 있나요?

📦 배송: 평균 1~3일
💸 환불: 수령 후 7일 이내, 무료 (단순 변심 시 왕복 배송비)
🔄 교환: 사이즈·컬러 가능, 왕복 배송비 4,000원
🎁 적립: 첫 구매 5%, 후기 5,000원

원하는 답이 없으면 메시지 보내주세요. 영업일 6시간 이내 답변 드릴게요.
```

## 2. 상품 문의 응답

### 사이즈 문의
```
안녕하세요. [브랜드명] 입니다.

[상품명] 사이즈 추천드릴게요:
- 평소 키 168cm + 55사이즈 입으시면 → S 추천
- 키 173cm 이상 + 66사이즈 → M 추천
- 박시핏 선호하시면 한 단계 큰 사이즈 추천드려요

상품페이지 [사이즈 가이드]에 모델 착용샷이 있으니 참고 부탁드려요.
추가 문의는 편하게 보내주세요.
```

### 재입고 문의
```
[상품명] 관심 가져주셔서 감사해요.

해당 상품은 현재 품절 상태이며, 재입고 일정은 [날짜] 예정입니다.
재입고 알림 신청해주시면 입고 즉시 알림톡으로 안내 드릴게요.

📲 [재입고 알림 신청] 링크: [URL]
```

## 3. 클레임 응답 (1점 후기 등)

### 후기 평점 1-2점 응답
```
안녕하세요, [브랜드명]입니다. 먼저 불편을 드려 진심으로 죄송합니다.

[고객님이 언급한 문제]에 대해 정확히 파악하고 빠르게 해결해드리고 싶어요.
번거로우시겠지만 [톡톡 또는 1:1문의]로 주문번호와 함께 메시지 주시겠어요?
교환·환불·재발송 중 가장 만족하실 방법으로 도와드리겠습니다.

이번 일을 발판으로 더 나은 제품과 서비스로 보답하겠습니다. 다시 한 번 죄송합니다.
```

**핵심**: 사과 + 책임 인정 + 해결 옵션 제시 + 개선 약속.
**금지**: 변명, 책임 회피, 고객 잘못 지적, 형식적 사과.

### 환불 요청
```
안녕하세요. 환불 신청 도와드릴게요.

주문번호 [번호] 확인했습니다.
✅ 단순 변심: 수령 후 7일 이내 무료 환불 가능합니다. 왕복 배송비 4,000원 차감됩니다.
✅ 상품 불량: 사진 첨부해주시면 즉시 무료 환불 + 배송비 면제로 처리해드려요.

회수는 평일 출고 시 신청 부탁드리며, 회수 후 영업일 3~5일 내 환불 완료됩니다.
불편 드려 죄송합니다.
```

### 배송 지연 사과
```
[고객님 성함] 고객님, 배송 지연으로 불편을 드려 죄송합니다.

확인해보니 [택배사 사정 / 재고 부족 / 명절 물량] 사유로 평소보다 1~2일 지연되었습니다.
오늘 출고되어 [예상 도착일]에 도착 예정입니다.

지연으로 마음 상하셨을 텐데 [3,000원 적립금 또는 다음 구매 5% 쿠폰] 보내드릴게요.
다시 한 번 죄송합니다.
```

## 4. 톡톡 톤 가이드

### Do
- "고객님" 호칭
- 존댓말 + 약간의 친근함
- 이모지 1-2개 (과하지 않게)
- 구체적 답변 (모호한 안내는 신뢰 ↓)
- 24시간 이내 응답

### Don't
- 반말
- 기계적 자동응답만
- "저희" 자주 (주체 모호) → "[브랜드명]" 명시
- 책임 회피·변명
- 길게 늘어지는 답변 (모바일 시청)

## 5. 알림톡 vs 친구톡 (카카오)

| 구분 | 알림톡 | 친구톡 |
|------|-------|-------|
| 발송 조건 | 가입 회원에게 거래·정보 | 카카오 채널 친구 추가 + 광고 동의 |
| 비용 | 건당 8-10원 | 건당 12-15원 |
| 광고성 | 거래 알림은 광고 X | 광고 가능 |
| 광고 동의 | 불필요 (거래) | 필수 |

### 알림톡 카피 (거래)
```
[주문 확인]
김OO 고객님, 주문이 완료되었습니다.
주문번호: 20260501-1234
상품: X200 무선이어폰 1개
결제: 89,000원
배송: 평일 출고 (1~3일 소요)

📦 배송조회: [URL]
```

### 친구톡 카피 (광고, 동의 필요)
```
🎁 단 하루, X200 무선이어폰 -30%

평소 가격 14만 9천 → 오늘만 9만 9천 + 무료배송
재고 50개 한정 — 매진 임박

▶ 지금 보러 가기: [URL]

(광고) 수신거부: 채널 차단
```

## 6. 자동응답 운영 팁

- 카페24·아임웹·채널톡 모두 시간대별 자동응답 설정 가능
- 영업시간 + 점심시간 + 휴일 분리
- FAQ 키워드 매칭 (사이즈·환불·배송 자동 분기)
- 봇 답변 후 "사람과 대화" 옵션 항상 제공
