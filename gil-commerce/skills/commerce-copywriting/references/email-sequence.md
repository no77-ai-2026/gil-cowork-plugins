## 5. 알림톡 vs 이메일 선택 가이드

| 상황 | 권장 채널 |
|------|---------|
| 거래 알림 (주문·배송) | 알림톡 (즉시성) |
| 마케팅 (광고 동의 회원) | 이메일 (긴 콘텐츠) + 친구톡 (긴급) |
| 짧은 후크·CTA | 알림톡·푸시 |
| 스토리·신상 소개 | 이메일 |
| 시즌 캠페인 시작 | 이메일 (사전) + 푸시 (당일) |
| 카트 이탈 | 이메일 (1·24h) → 알림톡 (72h) |
| 후기 요청 | 이메일 (Day 7) → 알림톡 (Day 14) |

## 6. 이메일 디자인 팁

- 모바일 우선 (60%+ 모바일 오픈)
- 한 줄 폭 600px
- 이미지 < 30% (텍스트 차단 우회)
- CTA 버튼 큼직 + 명확
- 발신자 이름은 [브랜드명] (회사명 X)
- 발신자 이메일은 noreply@ 보다 hello@·marketing@ (개인 같은 톤)

## 7. 자동화 도구

- **자사몰 자체**: 카페24·아임웹 기본 자동화 (제한적)
- **외부 마케팅 자동화**: MAILY, Stibee, Mailchimp (이메일)
- **알림톡**: 알리고, 굿스플로, 카카오 비즈메시지
- **통합 CRM**: Braze, OneSignal (대형), 채널톡 (중소)
