# 푸시 알림 카피 가이드

## 1. 푸시 알림 형식

| 부분 | 길이 | 비고 |
|------|------|------|
| 제목 | 30자 (모바일 잘림 한계) | 핵심 후크 |
| 본문 | 120자 (확장 시) | 추가 정보 |
| 액션 버튼 | 10자 | "보러 가기", "쿠폰 받기" |

## 2. 카테고리별 카피 패턴

### 신상품 출시
```
제목: 🎉 [브랜드명] 신상품 단독 발매!
본문: 30시간 배터리 무선이어폰 X200 — 회원가 특가 + 30일 환불 보장
버튼: 지금 보러 가기
```

### 프로모션 시작
```
제목: ⏰ 24시간 한정 -30% 시작
본문: 베스트셀러 X200 무선이어폰, 평소 가격 14만 9천 → 9만 9천. 무료배송.
버튼: 지금 사러 가기
```

### 프로모션 종료 임박
```
제목: ⌛ 6시간 남았어요 -30%
본문: 오늘 자정 종료. X200 무선이어폰 한정 9만 9천원. 재고 12개 남음.
버튼: 마지막 기회
```

### 카트 회복
```
제목: 🛒 장바구니 살펴보고 가셨나요?
본문: X200 무선이어폰 + 케이스 — 추가 5% 할인 쿠폰을 보내드렸어요. 24시간 유효.
버튼: 결제하기
```

### 재구매 유도 (60일 미구매)
```
제목: 💝 [고객님 이름]님, 오랜만이에요!
본문: 10% 재구매 쿠폰 + 무료배송. 즐겨찾기 상품 X200, 새 색상 출시했어요.
버튼: 둘러보기
```

### 후기 요청
```
제목: ✍️ 후기 한 줄 부탁드려요
본문: 받으신 X200 어떠셨나요? 후기 작성 시 5,000원 적립 — 다음 구매에 사용 가능.
버튼: 후기 쓰기
```

### 배송 안내
```
제목: 📦 X200 출고됐어요
본문: 김OO님, 주문하신 X200 무선이어폰이 출고됐어요. 내일 도착 예정. 운송장 [번호]
버튼: 배송 조회
```

## 3. 푸시 카피 톤

### Do
- 짧고 강하게
- 첫 단어가 중요 (이모지 1개 OK)
- 숫자·구체성 (시간·할인율·재고)
- 호칭 (가능하면 이름)
- 액션 버튼 명확

### Don't
- "안녕하세요" 같은 인사 (낭비)
- 모호한 표현 ("좋은 소식 있어요" — 뭔지 알 수 없음)
- 이모지 3개 이상
- 긴 문장 (모바일 잘림)
- 전체 발송 (개인화 부재)

## 4. 발송 빈도 가이드

| 빈도 | 권장 |
|------|------|
| 매일 | NO (피로도 누적, 차단율 ↑) |
| 주 2-3회 | OK (활성 회원) |
| 주 1회 | OK (안전, 일반 회원) |
| 월 1-2회 | OK (비활성 회원 회복) |
| 시즌·이벤트 한정 | 추가 OK |

## 5. 발송 시점

### 좋은 시점
- 평일 오전 9-10시 (출근길 폰 확인)
- 평일 점심 12-13시 (점심 시간)
- 평일 저녁 19-21시 (퇴근·여가)
- 주말 오전 10-11시 / 오후 15-17시

### 나쁜 시점
- 평일 새벽·심야 (00:00-07:00) — 한국 정통망법 위반 가능
- 평일 출근 직전 (08:00-08:30) — 무시당함
- 주말 새벽 (00:00-09:00) — 휴식 방해

## 6. 한국 정통망법 (영리목적 광고성 정보)

- 야간(21:00-익일 08:00) 광고성 푸시·문자 발송 금지
- 거래·배송·CS 알림은 시간 무관
- 마케팅 푸시는 동의 받은 회원에게만

## 7. 세그먼트별 푸시 전략

| 세그먼트 | 메시지 톤 |
|---------|---------|
| 신규 회원 (가입 7일 내) | 환영·할인 적립금 안내 |
| 첫 구매 후 30일 내 | 사용법·후기·재구매 |
| 충성 고객 (3회+ 구매) | 신상 우선 안내·VIP 혜택 |
| 비활성 회원 (60일+ 미구매) | 회복 쿠폰·조심스러운 톤 |
| 카트 이탈 | 즉각 회복 (1시간-24시간 내) |
| 시즌 캠페인 | 전체 발송 OK |

## 8. A/B 테스트

- 제목 변형 2-3개
- 발송 시점 변형 (오전 vs 저녁)
- 이모지 유무
- 호칭 유무 (이름 vs 일반)

오픈율·클릭율로 측정 후 우승안 적용.

## 9. 자사몰 빌더별 푸시 도구

- **카페24**: 모바일 앱 푸시 (앱 사용자만), 알림톡 (카카오)
- **아임웹**: 알림톡 + 이메일
- **외부**: Braze, OneSignal (앱 푸시 전문), 채널톡 (채팅·푸시)

## 10. 흔한 실수

- 모든 회원에게 동일 메시지 (개인화 부재)
- 빈도 과다 (주 5회+ → 차단율 ↑)
- 야간 발송 (정통망법 위반)
- 모호한 제목 ("좋은 소식")
- 액션 버튼 없음 (어디 클릭?)
