# 이커머스 카피라이팅 (광고·톡톡·푸시·이메일)

## 개요

이커머스 운영 시 매장 페이지(상세페이지) 외에도 광고·CS 응답·알림 메시지·이메일 마케팅에서 카피가 끊임없이 필요합니다.
이 스킬은 채널·형식별 길이·톤 제약을 지키면서 전환·만족·리텐션을 높이는 카피를 산출합니다.

## 트리거 키워드

이커머스 카피, 광고 카피, 광고 헤드라인, Meta 광고, 페이스북 광고, 인스타 광고,
네이버 검색광고, 카카오 광고, 톡톡 카피, 톡톡 응답, 카카오 알림톡, 푸시 알림,
이메일 마케팅, 이메일 시퀀스, 카트 이탈, 재구매 메시지

## 카피 카테고리

### 1. 광고 카피
- 검색광고 (네이버·구글)
- 디스플레이·소셜 (Meta·카카오 모먼트)
- 동영상 광고 (유튜브·인스타 릴스)

### 2. 톡톡·CS 응답
- 자동응답 메시지
- FAQ 응답 템플릿
- 1점 후기 사과 응답
- 환불·교환 응대

### 3. 푸시 알림
- 신상품 출시
- 프로모션 시작·종료 임박
- 카트 회복
- 재구매 유도

### 4. 이메일·알림톡
- 주문 확인서
- 배송 안내
- 후기 요청
- 재구매 쿠폰
- 신상품 뉴스레터
- 카트 이탈 회복

## 워크플로우

### 1단계: 카피 카테고리 + 채널 식별

사용자가 어떤 카피를 원하는지 명확히. 채널마다 길이·톤이 매우 다름.

### 2단계: 컨텍스트 수집

- 상품/브랜드 정보
- 타겟 독자
- 메시지 목적 (전환·정보·리텐션)
- 톤 (정중·캐주얼·긴급·따뜻)
- 길이 제한 (채널별)

### 3단계: 채널별 카피 산출

상세 가이드는 references 참조:
- `ad-copy.md` — Meta·네이버·구글·카카오 광고
- `talk-copy.md` — 톡톡·CS 응대
- `push-copy.md` — 푸시 알림 (앱·웹)
- `email-sequence.md` — 이메일·알림톡 시퀀스

### 4단계: A/B 옵션 제공

각 카피마다 2-3개 변형 제시:
- A: 직설적·기능 중심
- B: 감성적·라이프스타일
- C: 긴급성·스토리

### 5단계: ai-slop-reviewer 자동 체이닝

출력 직전 `gil:ai-slop-reviewer`를 자동 호출하여 AI 패턴 정제.

## 채널별 길이 제약 (cheat sheet)

| 채널 | 헤드라인 | 본문 | 비고 |
|------|--------|------|------|
| 네이버 검색광고 | 15자 | 45자 | 키워드 매칭 중요 |
| 구글 검색광고 | 30자 × 3 | 90자 × 2 | RSA 형식 |
| Meta 광고 | 40자 (모바일) | 125자 (잘림 위험) | 시각 우선 |
| 카카오 모먼트 | 20자 | 60자 | 모바일만 |
| 카카오 알림톡 | 16자 (header) | 1,000자 (body) | 광고 별도 동의 |
| 톡톡 자동응답 | 제한 없음 | 1,000자 | 정중 톤 |
| 푸시 알림 | 30자 (제목) | 120자 (본문) | 짧게 강하게 |
| 이메일 제목 | 50자 | - | 25자가 모바일 잘림 한계 |

    },
    {

    {

## 사용 예시

- "Meta 광고 헤드라인 3개 만들어줘 — 무선이어폰"
- "네이버 검색광고 카피 — 비건 화장품"
- "톡톡 자동응답 템플릿 (영업시간 외)"
- "카트 이탈 회복 이메일 3개 짜줘"
- "재구매 유도 알림톡 카피"
- "신상품 출시 푸시 알림"

## 관련 스킬

- `gil-commerce:detail-page-copy` — 상세페이지 카피 (페이지 내 본문)
- `gil-commerce:commerce-strategy` — 마케팅 전략 (어떤 메시지를 언제 보낼지)
- `gil-content:copywriting` — 일반 광고 카피 (이커머스 외)
- `gil-marketing:email-sequence` — 이메일 마케팅 시퀀스 (일반)
- `gil:ai-slop-reviewer` — AI 패턴 검수 (자동 체이닝)

## 이 스킬을 사용하지 말아야 할 때

- 상세페이지 13섹션 카피: `detail-page-copy`
- 일반 광고 카피 (이커머스 외): `gil-content:copywriting`
- 블로그 콘텐츠: `gil-content:blog`

## 주의사항

- 한국 카카오 알림톡: **광고성 메시지는 별도 동의 필수**. 거래 알림은 광고 X.
- 이메일 광고: 정통망법 + 정보통신망법 — 발신자 정보·수신거부 링크 필수.
- "최저가" 등 절대 표현은 광고심의 위반 가능 (표시광고법).
- 푸시 알림은 사용자가 OS 설정에서 차단 가능 → 거슬리지 않게 빈도 제한 (주 2-3회 이하).
