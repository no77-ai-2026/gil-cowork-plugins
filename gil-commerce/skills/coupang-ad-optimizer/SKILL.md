# 쿠팡 광고 풀세트 최적화 (Coupang Ad Optimizer)

## 개요

쿠팡 셀러의 광고 운영을 자연어 한 줄 입력으로 분석·최적화하는 전담 스킬. 월매출 7천 → 6개월 만에 3.6억(광고비 매출 30% → 11.2%)을 달성한 한국 셀러 실전 노하우를 wrapper 스킬화.

**책임 한 줄**: 쿠팡 광고보고서 입력 → 3 캠페인 유형 자동 분류 + 검색/비검색영역 매출 분리 + 엔드 ROAS 자동 계산 + 상품별 의사결정 분기.

## 쿠팡 광고의 특수성

| 영역 | 검색영역 | 비검색영역 |
|------|---------|-----------|
| **노출 위치** | 키워드 검색 결과(중간/상단/배너) | 메인·상품·장바구니·주문완료 페이지 추천 |
| **CPM** | 약 1만원 | 약 60원 |
| **CPC** | 비쌈 | 매우 쌈 |
| **클릭률·전환율** | 높음 | 현저히 낮음 |
| **노출량** | 적음 | 어마어마하게 많음 |

> 핵심 통찰: 검색영역과 비검색영역은 **CPM이 167배 차이**. 같은 상품도 어느 영역에서 잘 팔리는지 분리 분석해야 함.

## 3가지 캠페인 유형

| 캠페인 | 용도 | 적용 시점 | 핵심 설정 |
|--------|------|----------|-----------|
| **AI스마트광고** | 잘 팔리는 상품 찾기 | 신규 셀러·신상품 출시 | 모든 상품 옵션 1 캠페인 + AI 자동 분배, 일예산 최소 3만원 |
| **매출최적화 캠페인** | 비검색영역 잘 팔리는 상품 별도 분리 | AI스마트광고 후 데이터 확보 | 광고 운영 방식: 자동운영-매출최적화 |
| **수동키워드 캠페인** | 검색영역 잘 팔리는 상품 키워드 입력 | AI스마트광고 후 검색 키워드 전환 확인 | 광고 운영 방식: 직접입력 |

**원칙**: 캠페인 = 정리함. **상품 1개당 캠페인 1개**가 가장 깨끗 (시스템이 헷갈리지 않음).

## 자동규칙 3종

### 규칙 1: 쿠팡 골든타임 예산부족 광고중지 예방
- 오후 6시 이전에 일예산 70% 이상 사용 시, 일예산 40% 자동 증액
- **권장**: ROAS 들쭉날쭉할 때만 사용 (예산 변동은 AI 학습에 영향)
- **꿀팁**: 예산 1만원 + 일예산 500% 증액 → 하루 종일 1만원, 6시 이후 5만원

### 규칙 2: 광고수익률 유지하면서 예산 늘려 매출규모 키우기
- ROAS 350% 이상일 때 일예산 40% 자동 증액
- **권장**: ROAS 좋은데 예산 올리기 애매할 때만

### 규칙 3: 광고운영 최소 기준 (광고비 매출보다 과다 시 알림/중지)
- 최근 7일 평균 ROAS 100% 미만 시 알림 또는 광고 중지
- **권장**: 항상 켜두기 (광고비 > 매출 방지 가드레일)

## 목표 ROAS(=목표 광고수익률)의 진짜 의미

목표 ROAS는 **입찰가 조절 역할**이지 수익률 보장이 아닙니다.

| 설정 | AI 동작 | 결과 |
|------|---------|------|
| 목표 1000% (높음) | 비싼 자리 회피, 100원짜리 싼 클릭만 노림 | CPC 낮음, 효율 좋아 보이나 노출 자체가 안 됨 |
| 목표 400% (쿠팡 추천) | 균형 잡힌 입찰 | 시작 권장값 |
| 목표 200% (낮음) | 1000원짜리 비싼 자리도 과감하게 입찰 | 노출·클릭 많지만 광고비 폭증 |

**원리**: AI는 객단가·전환율을 못 바꿈. CPC만 조절 가능.

## 엔드 ROAS (본전 ROAS) 계산

적자가 나지 않는 최소 광고수익률.

```
엔드 ROAS = (판매가 ÷ 마진) × 100
        = 광고비를 매출의 몇 배로 만들어야 본전인가
        ※ 페어 스킬 commerce-margin-calculator와 동일 공식 (부가세·수수료 반영 마진 사용)
```

**예시**: 1만원 상품, 원가+배송비 등 8천원 → 마진 2천원 → 광고비 최대 2천원 → **엔드 ROAS 500%** (광고비 2천원으로 1만원 매출 = 500%)

> **쿠팡 ROAS 보정**: 쿠팡 화면 ROAS는 부가세 빠진 금액 → 실제는 **10% 더 낮음**. ROAS 200% 표기 = 실제 180%.

## 광고 효율 = CPC × 전환율 × 객단가

| 레버 | 개선 방법 |
|------|----------|
| **CPC 낮추기** | 키워드 제외, 목표 ROAS 조정 |
| **전환율 높이기** | 상세페이지·리뷰·가격 (`detail-page-copy`·`commerce-jtbd-persona`로 연계) |
| **객단가 높이기** | 번들 구성, 객단가 높은 상품 취급 |

**원칙**: 하나라도 크게 개선하면 광고 효율 확 달라짐. 전환율 1% → 2%면 매출 2배.

## 워크플로우 — 상품별 의사결정 분기

광고보고서 분석 후 상품마다 다음 분기를 적용:

```
[Step 1] 엔드 ROAS 계산 (모든 상품)
       ↓
[Step 2] 실제 ROAS와 엔드 ROAS 비교
       ↓
   ┌───────────────────────────────────────────────────────┐
   │ A. ROAS > 엔드 ROAS × 1.5    → 별도 캠페인 + 예산 증액   │
   │                                (예: 4번 상품 ROAS 652%)  │
   │ B. ROAS < 엔드 ROAS          → 살릴 수 있는지 진단      │
   │     ├─ CTR 높음 + CVR 낮음   → 상세페이지·리뷰·가격 개선 │
   │     ├─ CTR 낮음 + CVR 높음   → 썸네일·로켓배송·무료배송 │
   │     └─ 둘 다 낮음            → 광고 제외 (다른 상품 집중) │
   └───────────────────────────────────────────────────────┘
       ↓
[Step 3] 검색영역 vs 비검색영역 매출 분리
       ↓
   ┌───────────────────────────────────────────────────────┐
   │ A. 비검색영역 매출이 더 큼   → 매출최적화 캠페인          │
   │ B. 검색영역 매출이 더 큼     → 수동키워드 캠페인          │
   │                                (잘 팔린 키워드 모아서)    │
   └───────────────────────────────────────────────────────┘
       ↓
[Step 4] 캠페인 분리 시 사전 작업: 리뷰 최소 작업
        - 사진 후기 1개 + 글 후기 1개라도 (전환율에 큰 영향)
```

## 추가 설정 (쿠팡 광고 풀세트)

| 설정 | 권장값 | 이유 |
|------|--------|------|
| **할인쿠폰 자동 적용** | ON | 본인부담 0원 (쿠팡 100% 부담). 목표 ROAS 낮을수록 쿠폰 붙을 확률↑ |
| **키워드 제외** | 광고비만 잡아먹는 키워드 500개까지 등록 | AI 학습 방향 바로잡기 |
| **오디언스 플러스** | OFF | 쿠팡 외 SNS·포털 노출은 효율 낮음. AI스마트광고는 자동 ON → 캠페인 생성 후 수정 버튼으로 OFF 필수 |
| **일예산** | 최소 3만원 | 3만원이 1만원보다 노출수 훨씬 많음 (1만원은 10만 미만, 3만원은 30만+) |
| **캠페인 이름** | `1. AI스마트광고 400%` | 다른 캠페인과 구분 + 목표 ROAS 명시 |

## 입력·출력

**입력**:
- 쿠팡 광고보고서 (엑셀, 매출 성장 광고 보고서, 일별 + 캠페인>광고그룹>상품>키워드)
- 또는 시크릿팡 호환 엑셀 (마진계산기 결과 포함)
- 상품 원가·배송비·수수료 정보 (엔드 ROAS 계산용)

**출력**:
1. 상품별 ROAS·CTR·CVR·엔드 ROAS 표
2. 검색영역/비검색영역 매출 분리 표
3. 상품별 의사결정 (별도 캠페인·진단·제외)
4. 추천 캠페인 분리 안 (매출최적화·수동키워드)
5. 자동규칙 3종 적용 권장 ON/OFF
6. 키워드 제외 후보 리스트
