# 크라우드펀딩 가이드 (와디즈·텀블벅)

## 개요

크라우드펀딩은 **사전 판매로 수요 검증 + 초기 자금 확보 + 팬 커뮤니티 형성** 세 가지를 동시에 잡는 런칭 방식입니다.
이 스킬은 한국 양대 플랫폼(와디즈 + 텀블벅)의 프로젝트 기획·심사·운영·정산을 통합 가이드합니다.

## 트리거 키워드

크라우드펀딩, 와디즈, wadiz, 텀블벅, tumblbug, 펀딩 프로젝트, 리워드 펀딩,
선주문, 사전판매, 메이커, 펀딩 후기, 해피빈 펀딩, 크라우드 펀딩 카피

## 채널 비교

| 항목 | 와디즈 (Wadiz) | 텀블벅 (Tumblbug) |
|------|--------------|----------------|
| 주력 카테고리 | 가전·테크·뷰티·식품·생활 | 출판·창작·문화·예술·게임 |
| 사용자 성향 | 얼리어답터·신상품 선호 | 창작 후원·팬덤 기반 |
| 평균 펀딩액 | 수천만 원-수십억 (대형도 가능) | 수백만 원-수억 (창작 중심) |
| 심사 강도 | 중-높음 (제품 안전성 검증) | 중 (창작·아이디어 중심) |
| 수수료 | 9% (성공 시) + PG 3.6% | 8% (성공 시) + PG 3.6% |
| All-or-Nothing | Yes (목표 달성 시에만 결제) | Yes |
| 투자형 펀딩 | 있음 (와디즈 펀딩) | 없음 |
| 적합 단계 | 양산 직전·MVP 검증 | 창작자·소량 한정 |

## 워크플로우

### 1단계: 채널 선택

```
가전·테크·뷰티·식품·생활용품  → 와디즈
출판·게임·아트·창작·팬굿즈      → 텀블벅
공익·기부형                     → 해피빈펀딩
```

### 2단계: 프로젝트 기획

#### 핵심 요소 (공통)
- **메이커 스토리**: 왜 이 프로젝트를 시작했는지
- **제품·창작 콘셉트**: 무엇을 만드는가
- **차별점**: 기존 시장과 어떻게 다른가
- **목표 금액**: 양산 가능한 최소액 (현실적으로)
- **리워드 구성**: 가격대별 옵션
- **타임라인**: 펀딩 기간 + 제작 + 배송

#### 와디즈 추가 요건
- 안전 인증 (KC·식약처 등 카테고리별)
- 제품 시연 영상 (3-5분 권장)
- 양산 계획 + 위험 요소 명시

#### 텀블벅 추가 요건
- 창작자 프로필
- 창작 과정·진행도
- 한정·특별 리워드 차별화

### 3단계: 페이지 카피 (스토리텔링)

크라우드펀딩 페이지는 **상세페이지보다 스토리 비중이 큽니다**.

#### 권장 구성
1. **Hook 헤드라인** — 펀딩 클릭 유도
2. **메이커 스토리** — "왜 시작했나?"
3. **문제 정의** — 시장에 없는 것
4. **솔루션 — 우리 제품** — 어떻게 다른가
5. **시연 영상 + 사진** — 시각적 증거
6. **리워드 옵션** — 가격대별
7. **타임라인** — 양산·배송 일정
8. **메이커 약속** — 책임·보증
9. **위험 요소 + 대응** — 솔직함이 신뢰
10. **후원자 혜택** — 펀딩만의 특전

### 4단계: 영상 시놉시스 (필수, 와디즈)

3-5분 영상이 펀딩 성패를 60% 결정합니다.

#### 권장 구조
- 0:00-0:20 — 후크 (시각적 임팩트)
- 0:20-1:00 — 문제 (사용자 페인)
- 1:00-3:00 — 솔루션 + 시연
- 3:00-4:00 — 메이커 인터뷰
- 4:00-4:30 — 리워드 + CTA

### 5단계: 리워드 가격 구성

#### 와디즈 표준 패턴
- **얼리버드** (정가 30-40% 할인, 한정 100-500개)
- **슈퍼얼리버드** (정가 40-50% 할인, 한정 50-200개)
- **일반 리워드** (정가 20-30% 할인, 무제한)
- **번들 리워드** (2개·3개 묶음, 추가 할인)
- **VIP 리워드** (사인본·한정 컬러 등 차별화)

가격대 다양화 > 가성비 단일 가격.

### 6단계: 심사 + 등록

#### 와디즈
1. 메이커 가입 (https://www.wadiz.kr)
2. 사업자 등록증 + 신분증
3. 프로젝트 등록 → 와디즈 PD 컨설팅 (무료)
4. 콘텐츠 심사 (1-2주)
5. 안전 검수 (카테고리별)
6. 오픈 일정 확정

#### 텀블벅
1. 창작자 가입
2. 프로젝트 작성
3. 가이드 스태프 검토 (1주)
4. 오픈 일정 확정

### 7단계: 펀딩 운영

- 새소식 (업데이트) 주 1-2회
- Q&A 24시간 이내 응답
- SNS 콘텐츠로 외부 트래픽 유도
- 인스타그램·블로그·카카오톡 단톡방

### 8단계: 펀딩 종료 후

- 약속 일정 준수가 가장 중요
- 지연 시 새소식으로 즉시 공지 + 사과
- 양산·QC → 배송
- 후속 후기 콘텐츠 → 다음 펀딩 또는 자사몰 연결

## 주의사항

- 크라우드펀딩은 **약속 이행**이 가장 중요. 지연·미이행 시 신뢰 회복 매우 어려움.
- 와디즈는 안전 인증 미통과 시 오픈 거절. 사전 준비 필수.
- 펀딩은 광고가 아닌 사전판매. 부풀린 약속 금지 (소비자 분쟁).
- 펀딩 종료 후 일반 판매 가격은 펀딩가 이상이어야 (펀딩 후원자 보호).
