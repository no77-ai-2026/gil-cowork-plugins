# 와디즈 (Wadiz) 운영 가이드

> 한국 1위 크라우드펀딩 플랫폼. 가전·테크·뷰티·식품·생활 카테고리 강세.

- 사업자등록증 필수 (개인사업자도 가능)
- 메이커 가입: https://www.wadiz.kr/web/maker
- 신분증, 통장사본, 사업자등록증 업로드

## 2. 프로젝트 유형

| 유형 | 정의 | 적합 |
|------|------|------|
| **리워드 펀딩** | 후원자에게 제품·서비스 제공 | 신상품 사전판매 |
| **투자형 펀딩** | 후원자에게 지분·수익 분배 | 스타트업 자본 모집 |

리워드가 일반적. 본 가이드는 리워드 기준.

## 3. 심사 + 안전 검수

### 1차: 콘텐츠 심사 (1주)
- 카피·이미지·영상 검토
- 과장·허위 표현 수정 요청
- 와디즈 PD 컨설팅 (무료, 매우 도움 됨)

### 2차: 안전 검수 (카테고리별)
- 가전·전자: KC 인증, 전자파 적합성
- 식품: 식약처 인증, 영양성분 표기
- 화장품: 식약처 등록, 전성분 표기
- 의료기기: 식약처 의료기기 허가 (3·4등급은 펀딩 불가)
- 건강기능식품: 식약처 등록 (질병 효능 표현 금지)

미인증 시 펀딩 오픈 거절 또는 수정 요구.

## 4. 영상 가이드

### 형식
- 길이: 3-5분 (짧을수록 시청 완료율 높음)
- 해상도: 1080p 이상
- 가로형 16:9
- 한국어 자막 필수

### 구조 (권장)
| 시간 | 내용 |
|------|------|
| 0:00-0:20 | 후크 (시각적 임팩트, 결과 미리보기) |
| 0:20-1:00 | 페인 정의 (사용자 고통) |
| 1:00-3:00 | 솔루션 + 시연 (영상의 핵심) |
| 3:00-4:00 | 메이커 인터뷰 (스토리) |
| 4:00-4:30 | 리워드 옵션 + CTA |

### 권장 사항
- 자막은 큼직하게 (모바일 시청 다수)
- BGM은 저작권 자유 (Epidemic Sound·Artlist 등)
- 시연 장면이 가장 중요 → 해당 부분에 시간 배분

## 5. 페이지 콘텐츠

### 표지 이미지
- 정사각 1:1 (썸네일용)
- 핵심 후크 카피 (10자 이내)
- 영상 첫 프레임으로 사용 가능

### 본문 이미지
- 너비 700px+ (모바일 우선)
- 세로 길이 무제한
- 1080×12720 합성본 그대로 활용 가능

### 카피 톤
- 일반 광고보다 솔직함이 신뢰
- 메이커가 직접 쓴 듯한 톤
- "우리는 ~하고 싶었습니다" 1인칭 권장
- 위험 요소도 솔직히 (지연 가능성, 양산 어려움 등)

## 6. 리워드 가격 구성

### 권장 구조 (5단계)
1. **슈퍼얼리버드** (정가 -45-50%, 한정 50-200) — 첫 구매 유도
2. **얼리버드** (정가 -35-40%, 한정 200-500) — 중간 유입
3. **일반** (정가 -20-25%) — 무제한
4. **2-3개 묶음** (추가 -5-10%) — 객단가 상승
5. **VIP·한정판** (사인본·한정 컬러) — 차별화

### 가격 룰
- 정가는 펀딩 후 일반 판매가 (편딩가 < 정가 필수)
- 펀딩가는 펀딩 후 절대 인하 금지 (후원자 보호)

## 7. 펀딩 운영 (오픈 후)

### 새소식 (Update)
- 주 1-2회
- 양산 진행도, 외부 노출, 메이커 인터뷰
- 후원자 코멘트에 반드시 답글

### Q&A
- 24시간 이내 응답 필수
- 부정적 질문도 솔직히 답변
- 답변 톤: 정중하고 책임감 있게

### 외부 트래픽
- 인스타그램·블로그 콘텐츠 → 페이지 유입
- 카카오톡 단톡방·디스코드 운영
- 인플루언서·커뮤니티 협업

## 8. 정산 + 수수료

- 와디즈 수수료: 9% (성공 시)
- PG 수수료: 3.6% (카드 + 간편결제 평균)
- **All-or-Nothing**: 목표 미달 시 결제 자동 취소, 환불
- 정산: 펀딩 종료 후 14-21일 (안전 검수 완료 후)

## 9. 펀딩 종료 후

### 양산·QC
- 약속 일정 준수가 가장 중요
- 지연 시 새소식으로 즉시 공지 + 사과 + 새 일정

### 배송
- 후원자별 송장 자동 등록 (와디즈 시스템)
- 배송 중 손상 시 재배송 보장

### 후속 후기
- 펀딩 후기 콘텐츠 → 다음 펀딩 또는 자사몰 트래픽
- 와디즈 자체 후기·평점 시스템
