# 텀블벅 (Tumblbug) 운영 가이드

> 한국 창작 크라우드펀딩 플랫폼. 출판·게임·아트·문화·창작 카테고리 강세.

## 1. 가입 + 창작자 자격

- 개인 가능 (사업자 미보유 OK)
- 본인 인증
- 창작자 가입: https://tumblbug.com

## 2. 적합 카테고리

| 카테고리 | 설명 | 평균 펀딩액 |
|---------|------|----------|
| 출판 | 독립 출판, 사진집, 일러스트북 | 500만-3천만 |
| 게임 | 보드게임, 인디 게임 | 1천만-수억 |
| 디자인 | 굿즈, 패션, 액세서리 | 500만-2천만 |
| 음악 | 앨범, 콘서트, 음반 | 500만-2천만 |
| 영화·영상 | 다큐, 인디 영화 | 1천만-5천만 |
| 패션 | 핸드메이드 의류·액세서리 | 500만-2천만 |
| 푸드 | 수제 식품 (와디즈와 다른 톤) | 500만-1천만 |
| 미술·전시 | 작가 작품·전시회 후원 | 200만-1천만 |

## 3. 와디즈와의 차이

| 항목 | 와디즈 | 텀블벅 |
|------|-------|--------|
| 톤 | 제품·기술·혁신 | 창작·문화·팬덤 |
| 후원자 의도 | 신상품 사전판매 | 창작 응원 + 굿즈 수령 |
| 안전 검수 | 매우 엄격 | 가벼움 (창작 중심) |
| 영상 비중 | 60% | 30% (이미지·텍스트 비중 큼) |
| 평균 후원자 수 | 수천 명 | 수십-수백 명 |
| 평균 후원액 | 5만-15만원 | 3만-7만원 |

## 4. 프로젝트 구성

### 표지
- 정사각 1:1
- 핵심 후크 + 시각적 임팩트
- 텍스트 없이 이미지만으로도 OK (창작 중심)

### 본문 (스토리텔링)
1. **창작자 소개** — 누가 만드는가
2. **창작 의도** — 왜 이것을 만드는가
3. **결과물 미리보기** — 무엇이 나오는가
4. **창작 과정** — 어떻게 만드는가 (스케치·드래프트 공개)
5. **리워드 구성** — 가격대별
6. **타임라인**
7. **창작자의 약속**

### 카피 톤
- 창작자의 1인칭 시점
- 솔직하고 따뜻한 어조
- 팬과 대화하듯
- 과장 없는 정직한 표현

## 5. 리워드 구성

### 권장 (출판 예시)
1. **단순 응원** (3,000-10,000원) — 디지털 답례
2. **기본 굿즈** (정가 가격대, 한정 200-500개) — 책 1권
3. **번들** (책 + 굿즈) — 객단가 상승
4. **한정판** (사인본·일러스트 추가) — 차별화
5. **컬렉터** (모든 작품 + 액자 등) — 고가 한정

### 가격대 폭
- 텀블벅은 **3,000원부터 30만원까지** 폭 넓게 (와디즈는 3만원부터가 일반)
- 소액 후원으로 펀딩 참여율 높임

## 6. 운영

### 업데이트
- 주 1회 권장
- 창작 과정 공개 (스케치, 드래프트, 작업실 사진)
- 후원자에게 "함께 만든다"는 느낌 전달

### 코멘트
- 모든 코멘트에 답글 (커뮤니티 형성)
- 창작자 본인이 직접 답변

### 외부 홍보
- 인스타그램 (창작 과정 + 작품 미리보기)
- X(트위터) (창작 커뮤니티 활발)
- 디스코드 (팬덤 단위)

## 7. 정산 + 수수료

- 텀블벅 수수료: 8% (성공 시)
- PG 수수료: 3.6%
- All-or-Nothing
- 정산: 펀딩 종료 후 5-10일

## 8. 종료 후

- 약속 일정 준수가 핵심
- 지연 시 즉시 공지 + 새 일정
- 디지털 답례부터 우선 발송 (수령 인증)
- 굿즈 배송 → 후속 콘텐츠

## 9. 적합 시나리오

### 텀블벅 추천
- 창작자 본인 아이덴티티가 차별화 요소
- 팬덤·커뮤니티 기반
- 한정 수량 + 차별화 굿즈
- 초기 자금 + 작품 사전판매

### 비추천 (와디즈 적합)
- 가전·테크·뷰티 (안전 인증 강함)
- 양산형 제품
- 일반 소비재
