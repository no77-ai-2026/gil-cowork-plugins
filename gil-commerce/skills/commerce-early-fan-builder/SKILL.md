# 충성 100명 부트스트랩 (Commerce Early Fan Builder)

## 개요

신생 D2C 브랜드가 "잘 만든 충성 고객 100명, 1만 고객 부럽지 않다" 원칙으로 부트스트랩하는 전담 스킬. **광고 0원 + 손편지·1:1 메시지 위주** 한국 케이스 실전 매뉴얼.

**책임 한 줄**: 카테고리 + 브랜드 단계 + 현재 회원 수 + 가용 자원 입력 → 충성 100명 부트스트랩 30일 로드맵 + 100명→1만 전환 시나리오 자동 생성.

## 부트스트랩 원칙 (블랭크·강아지 가방 케이스)

### 원칙 1: 첫 100명은 광고로 사지 않는다

광고 = CAC 발생 + 익명 고객. 부트스트랩 시기 광고는 LTV 낮은 익명 트래픽만 들어옴.

**대신**:
- 지인 네트워크 (창업자 인스타·페이스북·카톡)
- 카테고리 커뮤니티 (디시인사이드·네이버 카페·당근마켓 동네)
- 콘텐츠 (블로그·유튜브 후기) — 광고 0원

### 원칙 2: 100명 각자에 1:1 손편지

진심 손편지 또는 1:1 메시지가 가장 강한 충성 트리거. 자동화 ❌, 진짜 손글씨 또는 개인화 메시지 ✅.

**한국 케이스**:
- 블랭크: 첫 100명에게 창업자 손편지 + 무료 추가 제품
- 강아지 가방 브랜드: 첫 50명 강아지 사진 받아 캐릭터 카드 제작 후 동봉

### 원칙 3: 후기·UGC 100% 활용

100명 = 100개의 진짜 후기. 단순 별점이 아닌 **사진·영상 후기**가 다음 1,000명을 데려옴.

**후기 트리거 방법**:
- 후기 작성 시 10,000원 적립 (적립 ROI가 광고 ROI보다 100배 높음)
- "후기 100자 + 사진 1장" 조건
- 베스트 후기 매월 1명 선정 + 평생 VIP 부여

### 원칙 4: 100명을 위한 비공개 채널

100명만 들어올 수 있는 카톡 오픈 채팅 또는 디스코드. "당신은 1%" 정체성 부여.

### 원칙 5: 100명에서 1,000명으로의 전환

100명이 각자 10명을 데려오면 1,000명. 추천 보상 시스템 필수.

## 30일 부트스트랩 로드맵

```
[Day 1~7] 시드 50명
   - 창업자 SNS 공지 + 1:1 DM 50개 보내기
   - 무료 체험단 30명 모집 (제품 무상 + 후기 의무)
   - 디시인사이드·네이버 카페 노출 (브랜드명 검색)

[Day 8~14] 시드 50→100명
   - 후기 작성 +10,000원 적립 (광고 효과 100배)
   - 시드 50명에게 손편지 발송 (창업자 직접 작성)
   - 인스타·블로그 후기 리그램 (UGC 가이드 준수)

[Day 15~21] 100명 락인
   - 100명 비공개 채널 오픈 (카톡 오픈 채팅)
   - 100명만 살 수 있는 VIP 제품 출시 (희소성)
   - 월 1회 100명 라이브 (제품 기획·신상 발표 우선 공개)

[Day 22~30] 100명 → 200명 (추천)
   - 추천 코드 발급 (추천 시 양쪽 모두 적립)
   - 100명 라이브에서 친구 초대 이벤트
   - 인스타 라이브 + UGC 캠페인
```

## 100명 → 1만명 전환 시나리오

```
100명 (시드)
   ↓ 평균 10명 추천 (Day 60~90)
1,000명 (성장 진입)
   ↓ 광고 ROI 명확 (M+3 코호트 검증)
10,000명 (정상 성장)
   ↓ LTV/CAC ratio 3+ 안정
지속 성장
```

**핵심 지표**:
- 100명 충성도: 재구매율 80%+
- 100명 추천: 평균 10명 (=> 1,000명)
- 1,000명 → 1만명: LTV/CAC ratio 3+ 안정 후 광고 확대


## 후속 체인 (텍스트 산출물 검수)

본 스킬의 1:1 손편지 초안·DM 메시지·비공개 채널 안내·100명 라이브 스크립트 등 모든 진심 메시지 텍스트는 `gil:ai-slop-reviewer`로 자동 후처리 체이닝됩니다 (CLAUDE.local.md §3-2 HARD). "잘 만든 충성 고객 100명" 원칙은 진정성에 좌우되므로 AI 패턴(과한 형용사·번역체·hype 어휘) 제거가 특히 중요. 손편지는 ai-slop 검수 후 손글씨로 옮기는 흐름 권장.

```text

```
