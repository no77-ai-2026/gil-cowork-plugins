# 카카오 메이커스 (Kakao Makers) 가이드

> 주문생산 + 한정 + 카카오톡 노출의 결합. 재고 부담 없이 트래픽 폭발이 강점.

## 1. 정체성

- **주문생산 모델**: 주문량 확정 후 양산 → 재고 부담 ↓
- **한정 발매**: 보통 7-21일 한정 판매
- **카카오톡 노출**: 카카오톡 친구탭·선물하기 노출 → 일평균 수십만 도달

## 2. 적합 카테고리

| 카테고리 | 비중 |
|---------|------|
| 가전·라이프 | 40% (소형 가전, 주방용품) |
| 식품·음료 | 25% (수제·프리미엄 식품) |
| 뷰티 | 15% |
| 패션·잡화 | 10% (한정 굿즈) |
| 홈리빙 | 10% |

## 3. 입점 절차

### 1단계: 메이커 가입
- https://makers.kakao.com → 메이커 신청
- 사업자등록증, 신분증, 통장사본
- 카카오 채널 (선택, 권장)

### 2단계: 프로젝트 제안
- 콘셉트 문서 (제품 + 차별점 + 한정 요소)
- 제품 사진 + 시연 영상 (선택)
- 정가 + 메이커스가 + MOQ
- 일정 (제안 → 오픈 → 양산 → 배송)

### 3단계: MD 검토
- 1-2주 검토
- 콘셉트 수정 요청 가능
- 카카오 메이커스 내부 패널 선정 (랜덤)

### 4단계: 오픈
- 보통 평일 오전 10시 오픈
- 7-21일 한정 판매
- 카카오톡 친구탭 노출

## 4. 콘셉트 작성 (가장 중요)

### 좋은 콘셉트 요소
- **한정성**: "이번에만, 1주일만"
- **특별함**: 일반 마켓에 없는 디자인·기능
- **스토리**: 메이커의 의도·과정
- **공동구매 톤**: "함께 주문하면 가능해진다"

### 좋은 예시
- "○○ 작가가 직접 그린 한정판 도자기 — 작가가 만들 수 있는 양만큼만"
- "30년 경력 한식 명인 김장 키트 — 11월 한정 100세트"

### 나쁜 예시
- "기존 제품을 할인가에 판매" (일반 마켓 톤)
- "신상품 출시 기념 세일"

## 5. 페이지 구성

- 표지: 정사각, 강한 후크
- 본문: 스토리텔링 (창작자 1인칭, 따뜻한 톤)
- 시연 사진·영상 (있으면 좋음)
- 리워드: 보통 단일 가격 + 옵션 (수량별)

## 6. 가격·수수료

- 메이커스가는 정가 대비 -10-30% 권장
- 카카오 메이커스 수수료: 15-25% (협상)
- PG 수수료: 별도 (통상 메이커스가 부담)
- 정산: 양산·배송 완료 후

## 7. 운영

- 새소식 주 1-2회 (양산 진행도)
- 주문자 코멘트 답글
- 카카오 메이커스 자체 톡 알림

## 8. 적합 시나리오

### 추천
- 소량·한정·고감도 제품
- 재고 부담 회피
- 카카오톡 트래픽 활용
- 펀딩 후 일반 판매 전환

### 비추천
- 양산형 일반 소비재 (일반 마켓 적합)
- 즉시 발송 필요한 상품 (메이커스는 양산 시간 필요)
