# 무신사 (Musinsa) 입점 가이드

> 한국 1위 패션·뷰티 셀렉션 채널. 10-30대 패션 코어 사용자 다수.

## 1. 정체성

- 패션·스니커즈·뷰티·라이프스타일 셀렉션
- 매장 위탁 (재고는 무신사 물류센터)
- 한정·콜라보 자주 진행
- 매거진 콘텐츠 협업으로 브랜드 노출

## 2. 적합 카테고리

| 카테고리 | 비중 |
|---------|------|
| 의류 | 50% (스트리트·캐주얼·신진 디자이너) |
| 스니커즈 | 20% (한정·콜라보 강함) |
| 뷰티 | 15% (남성·캐주얼 뷰티 강세) |
| 액세서리 | 10% |
| 라이프스타일 | 5% |

## 3. 입점 절차

### 1단계: 입점 신청
- 무신사스토어 입점 신청 (https://my.musinsa.com)
- 회원가입 → 입점 신청 메뉴
- 브랜드 정보 + 상품 정보 + 매출 자료 제출

### 2단계: 1차 서류 심사 (1-2주)
- 브랜드 콘셉트
- 디자인 톤
- 가격대
- 기존 유통 채널 (자사몰·기타 마켓 매출)

### 3단계: 2차 미팅 (선정 시)
- 무신사 본사 (서울) 또는 화상 미팅
- 시즌 라인업 발표
- 가격·MOQ·한정 옵션 협상

### 4단계: 계약 + 입점
- 위탁 계약 (재고 무신사 보관)
- 매장 페이지 셋업
- 첫 발매

## 4. 심사 기준

- **브랜드 인지도**: SNS 팔로워, 미디어 노출, 자사몰 매출
- **디자인 차별화**: 무신사 톤과의 핏 (스트리트·미니멀·신진)
- **가격대**: 적정 가격 (너무 저렴하거나 너무 비싸면 거절)
- **운영 능력**: MOQ 대응 가능, 시즌 신상 출시 가능

## 5. 입점 후 운영

### 매장 페이지
- 무신사 자체 디자인 가이드 (스트리트 톤)
- 1080×1080+ 정사각 + 라이프스타일 컷
- 일반 상세페이지 톤과 다름 (감도 우선)

### 무신사 매거진 협업
- 신상 발매 시 매거진 콘텐츠
- 디자이너 인터뷰
- 룩북 제작

### 한정·콜라보
- 무신사 단독 컬러·사이즈
- 다른 브랜드와 콜라보
- 한정 수량 발매로 화제성

## 6. 가격·수수료

- 위탁 수수료: 25-33% (카테고리·브랜드별)
- MOQ: 카테고리별 다름 (의류 100-500벌 일반)
- 정산: 월 1-2회

## 7. 무신사 자체 광고

- 매장 페이지 메인 노출 (광고비 별도)
- 카테고리 베스트 노출 (판매·평점 기반)
- 매거진 광고 콘텐츠 (PPL 형식)

## 8. 적합 시나리오

### 추천
- 패션·스니커즈·캐주얼 뷰티 브랜드
- 10-30대 타겟
- 자사몰·SNS 팔로워 1만+ 보유
- 시즌 신상 출시 가능

### 비추천
- 30-40대 럭셔리·올드한 톤
- 식품·가전·홈리빙 (29CM가 적합)
- MOQ 대응 어려움 (재고 부담)
