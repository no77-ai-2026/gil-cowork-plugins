# 3대 프로모션 기획법 (Commerce Promotion Planner)

## 개요

스몰 한국 D2C 브랜드가 단일 프로모션 기획서를 산출하는 전담 스킬입니다. 3가지 프로모션 타입(이슈화·얼리버드·한정) 중 브랜드 단계·목표에 맞는 타입을 자동 추천하고, 명목·스토리·혜택 3종 세트 + 벤치마킹 포인트 + 실무 체크리스트 + 노션 템플릿까지 한 번에 출력합니다.

**책임 한 줄**: 브랜드 단계 + 목표 + 가용 할인율 + 시즌 입력 → 프로모션 1건 풀세트 기획서 자동 생성.

**vs commerce-integrated-strategy**: 페어 스킬은 전체 매출 향상 1장 종합. 본 스킬은 **단일 프로모션 기획서** 1건만 깊이 있게 산출.

**vault 근거**: '듣보잡' 스몰 D2C 브랜드가 매출 12배 달성한 한국 케이스 실전 매뉴얼 (비플레인 사례 포함).

## 3대 프로모션 타입 매트릭스

| 타입 | 목표 | 핵심 메커니즘 | 적합 브랜드 단계 | 한국 D2C 케이스 |
|------|------|--------------|-----------------|----------------|
| **① 이슈화** | 브랜드 인지도 폭발 | 사회적 화제·언론 노출·SNS 바이럴 | 신생·스몰 | 비플레인 12배 매출 / 사운드짐 IPO 화제성 |
| **② 얼리버드** | 충성 고객 확보 | 사전 등록·한정 인원·VIP 권한 | 스몰·중대형 | 와디즈 펀딩·텀블벅 후원·29CM 멤버 사전 |
| **③ 한정** | 즉각 매출 | 시간·재고·인원 한정 | 모든 단계 | 쿠팡 골든타임·에이블리 1시간 한정·무신사 라이브 |

### 타입 선택 의사결정 트리

```
[브랜드 단계 = 신생]
   └─ 목표 = 인지도 → ① 이슈화 권장
   └─ 목표 = 즉각 매출 → ③ 한정 (스몰 캠페인부터)

[브랜드 단계 = 스몰]
   └─ 목표 = 충성 고객 → ② 얼리버드
   └─ 목표 = 인지도 + 즉각 매출 → ① 이슈화 + ③ 한정 콤보

[브랜드 단계 = 중대형]
   └─ 목표 = VIP 유지 → ② 얼리버드 (VIP 등급 차등)
   └─ 목표 = 시즌 매출 → ③ 한정 (시즌 캘린더 활용)
```

## 명목·스토리·혜택 3종 세트 (HARD)

프로모션 1건 기획서는 반드시 다음 3종 세트를 포함:

### 1. 명목 (Reason to Buy Today)

"왜 오늘 사야 하는가"의 명분. 단순 할인 ≠ 명목. 명목은 고객이 친구에게 설명할 수 있어야 함:

| 좋은 명목 | 나쁜 명목 |
|----------|----------|
| "1주년 감사 — 첫 고객 50명께 진심" | "오늘만 30% 할인" (이유 없음) |
| "신상품 출시 기념 사전 예약" | "여름맞이 세일" (식상함) |
| "비건 인증 1주년 — 동물 살린 12,400마리" | "추석 특가" (특이성 없음) |
| "겨울 한정 라벤더 향 — 11월만 생산" | "재고 정리" (부정적 인상) |

### 2. 스토리 (Why Should I Care)

"왜 우리 브랜드인가"의 서사. 명목과 별개의 브랜드 메시지:

```
✅ 좋은 스토리 구조:
[기존 시장의 문제] → [우리의 해결책] → [지금 함께해야 하는 이유]

예: "비건 시장은 비싸고 효과가 약했다 → 한국 농가 직거래로 거품 빼고 임상 인증 받았다
    → 1주년 + 임상 결과 발표 동시 — 첫 1,000명께 진짜 가격으로"
```

### 3. 혜택 (What Do I Get)

"내가 받는 것"의 구체성. 할인 외 다양한 혜택 조합:

| 혜택 카테고리 | 예시 |
|--------------|------|
| **가격** | 30% 할인, 1+1, 무료 배송 |
| **수량** | 한정 100개, 첫 50명, 일 100개 |
| **시간** | 24시간 한정, 골든타임 12-13시, 오늘만 |
| **권한** | VIP 등급 부여, 사전 체험, 후기 우선 노출 |
| **사은품** | 시즌 한정 굿즈, 샘플 패키지, 무료 체험권 |
| **서비스** | 1:1 컨설팅, 무제한 교환, 평생 A/S |

> **3종 세트 검증**: 친구에게 "왜 샀어?"라고 물었을 때 답할 수 있는 게 명목 + 스토리. "뭐 받았어?"에 답할 수 있는 게 혜택.

## 벤치마킹 포인트 3개

타 브랜드 성공 케이스에서 **명목·스토리·혜택** 각 1개씩 차용:

| 차용 출처 | 차용 요소 | 예시 |
|----------|----------|------|
| 케이스 A | 명목 (브랜드 화제성) | 비플레인 "듣보잡 브랜드 12배" 명목 차용 |
| 케이스 B | 스토리 (서사 구조) | 29CM "셀렉트 큐레이션" 서사 차용 |
| 케이스 C | 혜택 (구체 조합) | 쿠팡 "로켓와우 회원 추가 10%" 혜택 구조 차용 |

> **차용 규칙**: 그대로 베끼지 말고 우리 브랜드 톤에 맞게 재구성. 명목은 우리 브랜드 고유성과 결합.

## 실무 체크리스트 (6항목)

프로모션 발송 직전 확인:

- [ ] **명목·스토리·혜택 3종 모두 명시** (1개라도 빠지면 BLOCK)
- [ ] **타겟 세그먼트 명확** (전체 vs 코호트 vs 1:1)
- [ ] **벤치마킹 케이스 3개 인용** (명목·스토리·혜택 각 1개)
- [ ] **랜딩 페이지·상세 페이지 일관성** (메시지 동일성 검증)
- [ ] **법규 게이트 통과** (`commerce-marketing-compliance-kr` 체이닝)
- [ ] **후속 측정 KPI 사전 정의** (전환율·매출·재구매율 목표)

## 노션 템플릿 페이지 구조

자동 생성되는 노션 페이지 구조:

```markdown
# {프로모션 명목 한 줄}

## 1. Why Now (명목)
- 무엇을 기념하는가
- 왜 오늘이어야 하는가

## 2. Why Us (스토리)
- 시장의 문제
- 우리의 해결책
- 지금 함께해야 하는 이유

## 3. What You Get (혜택)
- 가격·수량·시간·권한·사은품·서비스 조합
- 일반 vs 멤버 vs VIP 차등

## 4. 타겟 세그먼트
- 1차 타겟 (메인)
- 2차 타겟 (보조)
- 3차 타겟 (확장)

## 5. 채널 믹스
- 메타 광고
- 카카오 친구톡
- 이메일 시퀀스
- 앱 푸시
- SNS 콘텐츠

## 6. 일정
- D-7: 예고
- D-3: 사전 등록 시작
- D-Day: 본 발송
- D+1: 후속 메시지

## 7. KPI 목표
- 전환율: __%
- 매출: __원
- 재구매율 (60일 후): __%

## 8. 법규 체크리스트 (정통망법)
- (광고) 표기, 무료 수신거부, 발신자 정보
```

## 워크플로우

### 입력 슬롯

| 항목 | 필수 | 예시 |
|------|------|------|
| 브랜드 단계 | 필수 | `신생` (1년 미만) / `스몰` (1-3년) / `중대형` (3년+) |
| 주 목표 | 필수 | `인지도` / `충성고객` / `즉각매출` (1개 또는 복합) |
| 가용 할인율 | 필수 | 30% (마진 잠식 한계 확인) |
| 시즌·시기 | 권장 | `봄 시즌` / `1주년` / `연말` / `블프` 등 |
| 카테고리 | 권장 | 스킨케어·식품·패션·반려동물 등 |
| 현재 매출 규모 | 권장 | 월 1천만원·1억·10억 등 (벤치마크 비교용) |

### 산출 순서

```
[Step 1] 브랜드 단계 + 목표 → 3대 타입 추천 (1~2개)

[Step 2] 명목 5안 생성 → 가장 강한 1안 선정
   - 기념일·이정표·인증·시즌·신상품 5축에서 1안씩

[Step 3] 스토리 구조 작성
   - 시장 문제 → 해결책 → 지금 이유 3단계

[Step 4] 혜택 조합
   - 가격·수량·시간·권한·사은품·서비스 6축 중 3~4개 조합

[Step 5] 벤치마킹 케이스 3개 매핑
   - 한국 D2C 사례에서 명목·스토리·혜택 차용

[Step 6] 실무 체크리스트 6항목 검증

[Step 7] 노션 템플릿 페이지 자동 생성
```

### ai-slop-reviewer 자동 체이닝

기획서 산출 직후 `gil:ai-slop-reviewer` 자동 호출:
- AI 클리셰 제거 ("혁신적인"·"놀라운"·"특별한")
- 명목·스토리의 식상함 제거 ("여름맞이 세일" 등)
- 한국 D2C 자연스러운 표현 조정

## 사용 예시

```
"/commerce-promotion-planner — 신생 D2C, 인지도 목표, 가용 할인 30%, 1주년"
→ 추천: ① 이슈화
   명목: "1주년 감사 — 첫 고객 1,000명께 진심"
   스토리: 비건 시장 문제 → 우리 해결책 → 1주년 + 임상 결과
   혜택: 30% + 1주년 한정 굿즈 + VIP 등급 평생 부여
   벤치마킹: 비플레인 명목·29CM 스토리·쿠팡 혜택 차용
   노션 템플릿 페이지 구조 출력

"/commerce-promotion-planner — 스몰 D2C, 즉각매출, 50%, 연말"
→ 추천: ① 이슈화 + ③ 한정 콤보
   명목: "올해 마지막 — 12월 31일 자정 마감"
   ...

"/commerce-promotion-planner — 비플레인 케이스 그대로 적용해줘"
→ 듣보잡 12배 매출 케이스 명목·스토리·혜택 3종 그대로 + 우리 브랜드 변형 지침
```

## 합격 기준

- 3대 프로모션 타입 중 1-2개 추천 (브랜드 단계·목표 매칭)
- 명목·스토리·혜택 3종 세트 모두 명시
- 벤치마킹 케이스 3개 (명목·스토리·혜택 각 1개)
- 실무 체크리스트 6항목 모두 PASS
- 노션 템플릿 페이지 구조 (1-8 섹션) 자동 생성
- ai-slop-reviewer 검수 흔적

## 관련 스킬

체이닝 순서:
`commerce-promotion-planner` (프로모션 기획) → `commerce-channel-message` (메시지 카피) → `commerce-marketing-compliance-kr` (법규 게이트) → 실제 발송

- `commerce-channel-message` — 프로모션 채널 메시지 15종 작성
- `commerce-marketing-compliance-kr` — 발송 전 정통망법 게이트 (필수)
- `commerce-push-planner` — 앱 푸시 채널 카피
- `commerce-integrated-strategy` — 매출 향상 1장에 본 프로모션 포함
- `commerce-jtbd-persona` — 타겟 세그먼트 페르소나

## 이 스킬을 사용하지 말아야 할 때

- **거시 매출 전략 (분기·연간)**: `commerce-integrated-strategy` 사용
- **메시지 카피 작성**: `commerce-channel-message` 사용
- **광고 캠페인 기획 (광고 운영)**: `campaign-planner` 사용
- **이메일 시퀀스 자체**: `email-sequence` 사용
