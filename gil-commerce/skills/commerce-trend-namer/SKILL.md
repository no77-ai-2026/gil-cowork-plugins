# 트렌드 키워드 → 상품명·해시태그 (Commerce Trend Namer)

## 개요

네이버 데이터랩의 급상승 검색어·시즌 트렌드를 상품명·해시태그·블로그 제목으로 자동 변환하는 전담 스킬입니다.

**책임 한 줄**: 카테고리 + 기간(주간/월간/시즌) 입력 → 데이터랩 트렌드 키워드 10개 → 상품명 변형 3안 + 해시태그 세트 + 블로그 제목 3안 자동 생성.

## 변환 워크플로우

```
[Step 1] 네이버 데이터랩 트렌드 키워드 수집
   - 카테고리별 급상승 검색어 (주간·월간)
   - 시즌 키워드 (블프·솽스이·명절)
   - 연관 키워드 확장

[Step 2] 상품명 변형 3안
   - 트렌드 + 기본 상품명 결합
   - 검색량 상위 키워드 우선
   - 25자 이내 최적화

[Step 3] 해시태그 세트
   - 인스타 30개 한정
   - 카테고리 5개 + 트렌드 10개 + 브랜드 5개 + 일반 10개

[Step 4] 블로그 제목 3안
   - SEO 최적화 (제목 25자 이내)
   - 트렌드 키워드 1~2개 포함
   - C-Rank 친화 패턴
```

## 사용 예시

```
"/commerce-trend-namer — 비건 스킨케어, 이번주 트렌드"
→ 데이터랩 급상승 10개 → 상품명 3안 + 해시태그 30개 + 블로그 제목 3안
```

## 후속 체인 (텍스트 산출물 검수)

본 스킬의 상품명 변형 3안·해시태그 세트 30개·블로그 제목 3안은 모두 검색 노출·CTR에 직접 영향을 주는 텍스트 산출물이므로 `gil:ai-slop-reviewer`로 자동 후처리 체이닝됩니다 (CLAUDE.local.md §3-2 HARD). 트렌드 키워드를 그대로 차용한 부자연스러운 결합은 검수에서 정정됩니다.

```text
commerce-trend-namer → ai-slop-reviewer → (선택) commerce-product-naming (검증)
```

## 관련 스킬

- `commerce-product-naming` — 표준 상품명 3안 (페어)
- `commerce-season-calendar` — 연간 시즌 캘린더
- `ai-slop-reviewer` — 상품명·해시태그·블로그 제목 AI 슬롭 검수 (자동 체인, HARD)
