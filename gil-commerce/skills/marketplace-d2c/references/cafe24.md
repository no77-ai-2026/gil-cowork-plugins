# 카페24 (Cafe24) 운영 가이드

> 한국 자사몰 점유율 1위. 안정성·확장성·생태계 강점.

## 1. 가입 + 쇼핑몰 개설

- 가입: 사업자등록증 첨부 또는 개인 (개인 시 정산 한도 제한)
- 무료 플랜 (수수료만 차감) + 유료 플랜 (월 9,900원부터)
- 모바일·PC 동시 자동 생성

## 2. 도메인

- 카페24 자체 도메인 구매 (가비아 연동)
- 또는 외부 도메인 연결 (DNS A 레코드 변경)
- HTTPS 자동 적용

## 3. 디자인 (스킨)

### 무료 스킨
- 기본 제공 10여종

### 유료 스킨
- 카페24 스킨 마켓 (https://design.cafe24.com)
- 5-50만원, 일회성 구매
- 카테고리: 패션·뷰티·식품·홈리빙·전자

### 커스터마이징
- HTML/CSS 직접 수정 가능
- 자체 개발자 보유 시 자유도 매우 높음
- 또는 카페24 디자인 파트너 위탁

## 4. 결제 (PG)

- KG이니시스 (가장 보편)
- 토스페이먼츠 (최근 점유율 상승)
- 네이버페이 (필수, 전환율 높음)
- 카카오페이 (필수)
- 페이코, 삼성페이, 애플페이

가입 시 사업자 등록증 + 통신판매업 신고증 + 통장사본 필요.

## 5. 배송

연동 가능:
- CJ대한통운 (가장 보편, 자동 운송장)
- 한진, 롯데, 우체국
- 합포장·합배송 자동 계산

배송비 정책:
- 5만원 이상 무료 (관행)
- 도서산간 추가 3,000-5,000원

## 6. 마케팅 도구

### 카페24 자체
- 적립금·쿠폰
- 회원등급 (구매액별)
- 게시판 (Q&A·후기)

### 외부 연동
- 네이버 쇼핑 광고 (네이버페이 의무)
- 구글 애즈 + Google Merchant Center
- Meta 픽셀 + 카탈로그
- 카카오 픽셀 + 모먼트

## 7. 자동화 + 앱

카페24 앱스토어 (https://store.cafe24.com)
- 이메일·SMS 자동화 (마케팅 팝업·발송)
- 챗봇 (채널톡, 톡플러스)
- 리뷰 위젯 (스토어인사이드 등)
- 재고 관리, 사은품, 정기배송

## 8. 정산 + 수수료

- PG 수수료: 카드 2.9% + 부가세 / 간편결제 2-3.5%
- 카페24 자체 수수료: 무료 플랜은 결제건당 100원
- 정산 주기: 카드 영업일+5일, 간편결제 +2-3일

## 9. 통신판매업 신고

- 자사몰 운영 의무
- 관할 시·군·구청 또는 정부24 (https://www.gov.kr) 온라인 신고
- 사업자 등록증 + 호스팅 증명 + 인증·보증서

## 10. 적합 시나리오

### 카페24 추천
- 매출 5천만-수십억 안정 운영
- 자체 개발자 또는 디자인 파트너 보유
- 다양한 PG·앱 연동 필요
- 해외 결제·배송 (Cafe24 글로벌)

### 비추천
- "오늘 만들어 내일 런칭" 시급함 (스킨 디자인 시간 필요)
- 디자인이 매우 차별화되어야 (아임웹이 더 적합)

## 11. 자주 묻는 질문

- **무료 플랜으로 충분한가?**: 가능. 단, 일부 앱·고급 기능 제한.
- **카페24 vs 자사 개발?**: 매출 100억+ 또는 매우 특수한 요구 아닌 이상 카페24가 비용·운영 효율적.
- **영문몰?**: Cafe24 글로벌 (별도 가입). 영문·중문·일문 다국어.
