# D2C 자사몰 운영 전략

> 마켓플레이스(쿠팡·네이버) 의존에서 자사몰 중심으로 전환할 때의 전략 핵심.

## 1. 자사몰의 가치

| 측면 | 마켓플레이스 | 자사몰 |
|------|------------|--------|
| 수수료 | 8-13% | 0% (PG 2-3%만) |
| 트래픽 확보 | 마켓이 제공 | 직접 확보 (광고 비용) |
| 데이터 | 제한적 (이메일·연락처 X) | 100% 보유 |
| 브랜드 통제 | 제한 | 100% 자유 |
| 가격 통제 | 마켓 가격 압박 | 자유 |
| 재구매 유도 | 어려움 | 이메일·알림톡 자유 |

## 2. 자사몰 + 마켓 동시 운영 (권장)

신생 브랜드 1년차:
- **70% 마켓** (트래픽·매출 기반)
- **30% 자사몰** (데이터 축적·로열티 기반)

성장기 (2-3년차):
- **40% 마켓**
- **60% 자사몰**

브랜드 정착기:
- **20% 마켓** (탐색·신규 확보)
- **80% 자사몰** (LTV 기반)

## 3. 자사몰 첫 트래픽 확보

### 광고 (가장 빠름)
- 네이버 검색광고: 브랜드 키워드 + 카테고리 키워드
- Meta 광고: 핀터레스트·인스타그램 룩북 → 전환
- 구글 디스플레이: 리타겟팅 위주

ROAS 목표: 첫 3개월 200%+, 6개월차 300%+, 안정기 400%+

### 콘텐츠 (느리지만 무료)
- 블로그 SEO (네이버 블로그 + 자사몰 블로그)
- 인스타그램·유튜브 (브랜드 콘텐츠)
- 카카오뷰·1boon (네이버 콘텐츠 채널)

## 4. 첫 구매 전환

자사몰 신규 방문자 → 첫 구매 전환율 1-2% 일반.

전환 부스트:
- 첫 구매 쿠폰 5,000-10,000원
- 무료배송 5만원 → 3만원 임시 인하
- 회원가입 시 즉시 할인
- 신규 회원 첫 구매 후기 작성 시 +적립

## 5. 재구매 유도 (LTV)

### 자동 트리거
- 구매 후 7일: 사용 가이드 이메일
- 14일: 후기 요청 + 적립 인센티브
- 30일: 재구매 쿠폰
- 60일: "이런 상품 어때요?" 추천

### 등급제
- 일반 → 실버 → 골드 → VIP
- 누적 구매액 기준
- 등급별 무료배송·전용 쿠폰·생일 선물

## 6. 데이터 활용

### 핵심 지표
- 신규 가입율
- 첫 구매 전환율
- 재구매율 (30일/60일/90일)
- AOV (평균 객단가)
- LTV (Customer Lifetime Value)

### 도구
- Google Analytics 4
- 네이버 애널리틱스
- 자사몰 빌더 자체 통계 (카페24·아임웹 모두 제공)

## 7. 광고 채널별 특성 (자사몰 기준)

| 채널 | 강점 | 약점 | 적합 단계 |
|------|------|------|---------|
| 네이버 검색 | 의도 높음, 전환율 우수 | 단가 상승 | 안정 운영 |
| 구글 검색 | 글로벌·고객 의도 | 한국 검색은 네이버에 밀림 | 보조 |
| Meta (FB·IG) | 비주얼 강점, 프리미엄 | 플랫폼 의존 | 패션·뷰티 핵심 |
| 카카오 모먼트 | 한국 도달율 1위 | 광고 단가 비쌈 | 브랜드 확장기 |
| 유튜브 | 동영상·바이럴 | 제작 비용 | 콘텐츠 마케팅 강점 |

## 8. 자사몰 KPI 대시보드

매주 점검:
- [ ] 신규 가입 수
- [ ] 신규 구매자 수 + 전환율
- [ ] 재구매 횟수 + 재구매율
- [ ] AOV (객단가)
- [ ] 광고비 + ROAS (채널별)
- [ ] 후기 작성률

매월 점검:
- [ ] LTV (90일 기준)
- [ ] 등급별 회원 분포
- [ ] 이메일 오픈/클릭률
- [ ] 알림톡 발송 효율

## 9. 흔한 실수

- 자사몰만 운영 (트래픽 부족)
- 광고비 ROAS 100% 미만 지속
- 카탈로그·이메일 미설정
- 통신판매업 미신고 (법적 리스크)
- 환불·반품 정책 모호 (분쟁 다발)

## 10. 1년 운영 로드맵

**1-2개월**: 셋업 (도메인·디자인·결제·배송·기본 마케팅)
**3-4개월**: 첫 광고 운영, 데이터 축적, ROAS 최적화
**5-6개월**: 등급제·이메일 자동화·재구매 유도
**7-9개월**: 채널 믹스 확장 (Meta + 카카오 + 유튜브)
**10-12개월**: 브랜드 캠페인, 충성고객 프로그램
