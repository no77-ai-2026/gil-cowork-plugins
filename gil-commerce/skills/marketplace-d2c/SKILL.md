# 자사몰 (D2C) 운영 가이드

## 개요

자사몰(Direct-to-Consumer)은 마켓플레이스(쿠팡·네이버) 대비 **수수료 부담 없음 + 브랜드 통제권 100%**가 강점인 운영 방식입니다.
이 스킬은 한국 D2C 빌더 3종(카페24 / 아임웹 / 메이크샵)의 정책·SEO·광고 연동·결제 옵션을 통합 가이드합니다.

## 트리거 키워드

자사몰, D2C, 카페24, cafe24, 아임웹, imweb, 메이크샵, MakeShop, 자사 쇼핑몰,
브랜드 쇼핑몰, 독립몰, 자체 쇼핑몰, 자체몰, 노코드 쇼핑몰

## 채널 비교

| 항목 | 카페24 | 아임웹 | 메이크샵 |
|------|--------|--------|---------|
| 한국 점유율 | 1위 (최대 50%) | 디자인·신생 브랜드 강점 | 중견 (NHN커머스) |
| 노코드 친화 | 중 (스킨 편집 필요) | 매우 높음 (드래그앤드롭) | 중 |
| 디자인 자유도 | 높음 (HTML/CSS 제어 가능) | 높음 (디자인 우선) | 중 |
| 무료 플랜 | 있음 (수수료 일부) | 있음 (제한 있음) | 있음 |
| 결제 옵션 | KG이니시스/토스/네이버페이/카카오페이 등 폭 넓음 | 토스/이니시스/네이버페이 | KG이니시스/토스 |
| 해외 결제 | 지원 | 지원 (제한적) | 지원 |
| 적합 단계 | 매출 5천-수십억 안정 운영 | 신규·디자인 중심 브랜드 | 중간 규모 |

## 워크플로우

### 1단계: 채널 선택

| 추천 시나리오 | 채널 |
|----------|------|
| 신규 브랜드, 디자인이 핵심, 빠른 런칭 | **아임웹** |
| 한국 표준 운영, 안정성, 큰 규모 확장 | **카페24** |
| NHN 생태계 활용, 페이코 연동 | **메이크샵** |

### 2단계: 도메인 + 호스팅 설정

- 자체 도메인 구매 (가비아, AWS Route53, 후이즈)
- DNS A 레코드 / CNAME 설정
- HTTPS (대부분 빌더 기본 제공)

### 3단계: 디자인 (스킨/테마)

- 카페24: 스킨 마켓에서 구매·커스터마이징 (5-50만원)
- 아임웹: 무료 템플릿 + 드래그앤드롭, 디자인 자유도 가장 높음
- 메이크샵: 표준 스킨 + 커스터마이징

상세페이지 1080×12720 합성본 그대로 업로드 가능 (모든 빌더 지원).

### 4단계: 결제 + 배송

#### 결제 (한국 PG)
- KG이니시스 (가장 보편)
- 토스페이먼츠 (UX 좋음, 최근 점유율 상승)
- 네이버페이·카카오페이 (간편결제 필수)
- 페이코 (NHN 생태계)

#### 배송
- CJ대한통운, 한진, 롯데택배, 우체국
- 자사몰 배송비 정책 명확화 (무료 기준, 도서산간 추가)
- 운송장 자동 등록 (스마트택배·굿스플로 연동)

### 5단계: 검색·광고 연동

#### 네이버 검색 등록
- 네이버 서치어드바이저 사이트 등록
- 사이트맵 sitemap.xml 제출
- 모바일 호환성 검사

#### 구글 SEO
- Search Console 사이트 등록
- robots.txt + sitemap.xml
- meta description 명확화
- 구조화 데이터 (Product schema.org)

#### 광고 채널
- 네이버 검색광고 (브랜드 키워드 + 카테고리 키워드)
- 구글 검색·디스플레이
- 페이스북·인스타그램 (Meta Business Suite)
- 카카오 모먼트 (카카오톡 광고)

### 6단계: 회원·CRM

- 회원가입 정책 (간편가입 + 마케팅 동의 분리)
- 이메일·SMS 마케팅 (24-48시간 룰)
- 적립금/등급제 운영
- 첫 구매 쿠폰 → 재구매 유도 → 충성고객 등급

### 7단계: 운영 KPI

| KPI | 권장 목표 |
|-----|--------|
| 광고 ROAS | 300%+ (운영 비용 회수) |
| 신규 구매 전환율 | 1-2% |
| 재구매율 | 30%+ (브랜드 충성도) |
| 객단가 | 카테고리별 다름 |
| 장바구니 이탈율 | 60% 이하 |

## 주의사항

- 빌더별 정책은 수시 변경. 실제 등록 전 공식 사이트 확인 필수.
- PG 가입은 사업자등록 필수. 통신판매업 신고도 의무 (관할 시·군·구).
- 자사몰 운영 초기 1-3개월은 마켓플레이스 동시 운영 권장 (트래픽 확보).
