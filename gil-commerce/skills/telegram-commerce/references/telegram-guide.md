## Telegram 판매 모델 3가지

### 1. Telegram Channel (방송)

```
[운영]
- 채널 생성 (예: @MyShop_uz)
- 상품 사진·가격·설명 포스팅
- 구독자 → 댓글 또는 DM 주문
- 수동 결제 (계좌 송금)

[규모]
구독자 1,000~100,000 가능
주문 처리: 수동·반자동
```

### 2. Telegram Bot (자동화)

```
[Bot 기능]
- 카탈로그 표시
- 장바구니
- 자동 결제 (Click·Payme·Uzum Pay 연동)
- 주문 추적

[개발]
- BotFather로 생성
- Python·Node.js로 backend
- @ShopBot 같은 외주 가능
```

### 3. Telegram Shop (Bot Payments)

```
[공식]
- Telegram Payments 2.0
- 자동 결제 가능
- UZ 결제 게이트웨이 연동 (Payme·Click·Uzum)

[수수료]
- Telegram: 무료
- 결제 게이트웨이: 1~2.5%
```

## 운영 워크플로우

```
1. 채널·Bot 생성
2. 카탈로그 (제품 사진·가격·설명) — 트라이링구얼 권장
3. 광고 (Telegram Ads·인플루언서 채널)
4. 주문·결제·배송
5. 후기·재구매 유도
```
