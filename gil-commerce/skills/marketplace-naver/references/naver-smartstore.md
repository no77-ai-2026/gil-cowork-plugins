# 네이버 스마트스토어 상세 가이드

> 본 문서는 일반적 기준이며 실제 등록 전 sell.smartstore.naver.com 최신 공지 확인 필수.

## 1. 이미지 규격

### 대표(메인) 이미지
- 정사각형 1000×1000 권장 (최소 640×640)
- JPG / PNG / GIF
- 5MB 이하
- 흰색 배경 권장

### 추가 이미지 (최대 9장)
- 동일 규격
- 다양한 각도·디테일

### 상세 이미지 (페이지 본문)
- 너비 860-1000px 권장
- 세로 길이 무제한
- 1080×12720 합성본 그대로 업로드 가능
- 이미지 1장 < 10MB

## 2. 카테고리 분류 (매우 중요)

네이버 쇼핑은 카테고리 정확도가 검색 점수에 큰 영향을 줍니다.

전략:
1. 네이버 쇼핑에서 동일 상품 검색
2. 베스트 셀러 3개의 카테고리 확인
3. 가장 빈번한 카테고리 선택

잘못된 카테고리 등록 시:
- 검색 노출 감소
- 카테고리 위반으로 등록 취소 가능

## 3. 상품명 작성

### 패턴
```
[브랜드] [상품명] [핵심속성1] [핵심속성2] [용도/타겟] [용량]
```

### 좋은 예
- ✅ `"닥터자비 X200 무선이어폰 ANC 30시간 직장인 통화이어폰"`
- ✅ `"이마트 비건 비누 무향 100g 4개입 민감피부"`

### 금지
- ❌ 특수문자 남용 (★, ♥, [, ])
- ❌ 동일 단어 반복
- ❌ 경쟁사 브랜드명
- ❌ "최저가", "1위" 등

50자 이내. 첫 20자가 검색 매칭 가중치 가장 높음.

## 4. 태그 (해시태그)

10-20개 권장. 카테고리/타겟/용도/시즌으로 다양화.

좋은 예:
```
#무선이어폰 #ANC이어폰 #직장인이어폰 #출퇴근이어폰 #블루투스이어폰
#운동이어폰 #방수이어폰 #배터리오래가는이어폰 #통화이어폰 #에어팟대안
```

## 5. 톡톡 운영

### 자동응답 필수 항목
1. 영업시간 안내
2. 배송 정책 (출고일, 도서산간)
3. 반품 정책 (조건, 기간)
4. A/S 정책

### 응답 SLA
- 신규 문의: 24시간 이내
- 클레임: 12시간 이내
- 평균 응답 속도가 검색 점수에 반영됨

### 톡톡 카피 톤
- "고객님" 호칭 권장
- 존댓말 + 친근함
- 이모지는 1-2개 정도만
- 과도한 친근함은 피함

## 6. 가격·옵션

### 옵션 등록
- 옵션명: 한글, 명확
- 옵션값: 가격 추가/감면 명시
- 재고: 옵션별 정확

### 가격 전략
- "권장소비자가" 입력 시 검색 우대
- 무료배송 조건은 명확히
- 즉시할인 vs 쿠폰할인 구분

## 7. 후기 운영

### 후기 작성 유도
- 일반 후기: 100원 적립
- 포토 후기: 300-500원 적립
- 한달 후 후기: 추가 적립

### 후기 응답 (Owner Reply)
- 모든 후기에 응답 권장 (특히 4점 이하)
- 부정 후기에 방어적·공격적 응답 금지
- 사실 확인 후 사과 + 개선 약속

### 후기 조작 금지
- 가족·지인·직원 후기 작성 금지
- 후기 대가성 거래 금지
- 적발 시 영구 정지 + 환불

## 8. 검색 노출 우대 조건 (네이버 쇼핑)

| 항목 | 우대 조건 |
|------|---------|
| 후기 평점 | 4.5+ |
| 후기 건수 | 30+ |
| 1점 후기율 | 5% 이하 |
| 톡톡 응답 속도 | 2시간 이내 |
| 출고 정확도 | 95%+ |
| 반품률 | 5% 이하 |
| 카테고리 정확도 | 검증 통과 |

## 9. 시즌 활용

### 주요 시즌
- 1월: 새해 + 신년 선물
- 2-3월: 졸업·입학
- 5월: 가정의달 (어버이날·어린이날)
- 8월: 여름휴가
- 9-10월: 추석
- 11월: 빼빼로데이 + 광군제
- 12월: 크리스마스 + 연말선물

시즌 키워드를 상품명·태그에 일주일 전부터 반영.

| 위반 | 처분 |
|------|------|

| 카테고리 위반 | 등록 취소 |
| 금지 문구 | 등록 거절 / 1주 정지 |
