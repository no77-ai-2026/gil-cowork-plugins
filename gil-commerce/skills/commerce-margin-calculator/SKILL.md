# 마진·엔드 ROAS 자동 계산기 (Commerce Margin Calculator)

## 개요

이커머스 셀러가 상품별 마진·엔드 ROAS(본전 ROAS)·손익분기 광고비를 **자연어 한 줄 입력**으로 자동 계산하는 스킬. 시크릿팡 마진계산기의 핵심 로직을 차용하되, AI 챗봇 통합으로 엑셀 업로드 없이 바로 답을 받을 수 있도록 차별화.

**책임 한 줄**: 상품 원가·판매가·수수료 입력 → 마진·엔드 ROAS·손익분기 광고비 자동 산출.

## 트리거 키워드

마진, 마진율, 엔드 ROAS, 본전 ROAS, 최소 ROAS, 손익분기, 역마진, 수수료 반영, 부가세 반영, 광고비 한도, 본전 광고비, 마진 계산, 마진 계산기, 광고 효율, ROAS 목표

## 입력·출력

### 입력 슬롯

| 항목 | 필수 | 예시 |
|------|------|------|
| 판매가 | 필수 | 19,800원 |
| 원가 | 필수 | 6,000원 |
| 배송비 | 필수 | 3,000원 (셀러 부담 또는 0원) |
| 채널 | 필수 | 스마트스토어 / 쿠팡 / 카페24 / 아임웹 |
| 수수료율 | 자동 | 채널별 자동 적용 (수동 입력 가능) |
| 부가세 | 자동 | 10% (영세사업자 / 면세사업자 다르게 적용) |
| 쿠폰·할인 | 선택 | 5% / 10% / 1+1 |
| 결제 수수료 | 선택 | 토스·카카오페이 0.5-1.5% |

|------|---------------------|------|
| 스마트스토어 (네이버) | 5.94% (네이버페이 + 입금 수수료) | 결제 수단별 추가 |
| 쿠팡 일반 | 10.6% (카테고리별 8-13%) | 로켓배송 = 셀러 비용 차이 |

## 엔드 ROAS (본전 ROAS) 공식

```
판매가 - 부가세 = 부가세 빠진 판매가
부가세 빠진 판매가 × (1 - 수수료율) = 실제 입금 매출
실제 입금 매출 - 원가 - 배송비 - 쿠폰 = 마진
마진 = 광고비 최대 한도

엔드 ROAS = (판매가 ÷ 마진) × 100  (부가세 포함 판매가 기준)
        = 광고비를 매출의 몇 배로 만들어야 본전인가
```

**예시 계산** (쿠팡 무선이어폰):
- 판매가: 19,800원 / 원가: 6,000원 / 배송: 3,000원 / 쿠팡 수수료 10.7%
- 부가세 빠진 가격: 19,800 ÷ 1.10 = 18,000원
- 플랫폼 수수료: 18,000 × 0.107 = 1,926원
- 실제 입금: 18,000 - 1,926 = 16,074원
- 총 원가: 6,000 + 3,000 = 9,000원
- 마진: 16,074 - 9,000 = **7,074원**
- 엔드 ROAS: 19,800 ÷ 7,074 × 100 ≈ **280%**

> 즉 이 상품은 광고비를 19,800원 매출의 1/2.8 (= 7,074원) 이하로 써야 적자가 안 남. 이게 광고수익률 280% = 본전 ROAS.

## 워크플로우

```
[Step 1] 자연어 한 줄 입력 또는 JSON 입력 슬롯 채우기
   ↓ 채널 자동 인식 (스마트스토어/쿠팡/카페24/아임웹/11번가/G마켓/와디즈)
[Step 2] 부가세·플랫폼 수수료·결제 수수료·쿠폰 자동 차감
   ↓ 실제 입금 매출 산출
[Step 3] 마진 = 실제 입금 매출 - 원가 - 배송비
   ↓
[Step 4] 엔드 ROAS = (판매가 ÷ 마진) × 100 산출
   ↓ 권장 목표 ROAS = 엔드 ROAS × 1.3~1.5
[Step 5] 권장 ROAS·쿠폰 전략·쿠팡 부가세 보정 경고 출력
   ↓ (선택) coupang-ad-optimizer 로 체이닝
```

## 사용 예시

```
"무선이어폰 19,800원, 원가 6,000원, 배송 3,000원, 쿠팡으로 팔 때 엔드 ROAS 알려줘"
→ 엔드 ROAS: 280%, 마진: 7,074원, 권장 목표 ROAS: 400%

"비건 스킨케어 12,900원, 원가 4,000원, 배송 0원, 스마트스토어, 5% 쿠폰 적용"
→ 엔드 ROAS 계산 + 쿠폰 비용 반영

"카페24 자사몰 + 토스페이 결제 수수료 1%까지 정확히 마진 알려줘"
→ 결제 수수료 추가 반영 후 정확한 마진
```

## 페어 분리 (책임 경계)

| 페어 스킬 | 차이 |
|----------|------|
| `coupang-ad-optimizer` | 쿠팡 광고 캠페인 분류·의사결정 (본 스킬: 마진·ROAS 계산만) |
| `commerce-strategy` | 채널 믹스·가격·프로모션 통합 전략 (본 스킬: 단일 상품 마진) |
