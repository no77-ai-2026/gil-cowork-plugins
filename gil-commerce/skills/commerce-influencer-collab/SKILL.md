# 인플루언서·UGC 협업 가이드 (Commerce Influencer Collaboration)

## 개요

한국 D2C 셀러가 인플루언서 협업·UGC(User-Generated Content) 리그램·굿즈 기획을 **표시광고법·공정위 가이드 준수** 하에 진행하는 전담 스킬입니다.

**책임 한 줄**: 브랜드 단계 + 예산 + 카테고리 + 협업 목적 입력 → 인플루언서 티어 추천 + 협업 가이드 + 뒷광고 회피 체크리스트 + 굿즈 기획 (선택) 자동 생성.

**과태료 위험**: 뒷광고 (광고 표기 누락) 시 표시광고법 위반 — 과태료 5,000만원 + 시정명령 + SNS 노출 손실.

## 5 인플루언서 티어 매트릭스

| 티어 | 팔로워 | 협업 비용 | 전환율 | 적합 브랜드 단계 |
|------|--------|----------|--------|----------------|
| **메가** | 100만+ | 1,000만-5,000만 | 0.5-1.5% | 중대형 / 대량 인지도 |
| **매크로** | 10만-100만 | 200만-1,000만 | 1-3% | 스몰-중대형 |
| **마이크로** | 1만-10만 | 30만-200만 | 3-8% | 스몰 / 가성비 우수 |
| **나노** | 1천-1만 | 5만-30만 또는 무료 체험단 | 5-15% | 신생 / ROI 최고 |
| **메가나노** | 최대 1천 | 무료 / 제품만 | 2-10% | 신생 / 인지도 시드 |

> **권장**: 신생·스몰 브랜드는 마이크로-나노 위주. 메가·매크로는 단발성 매출보다 인지도 빠른 확보 목적.

## 뒷광고 회피 체크리스트 (HARD)

표시광고법 + 공정위 「추천·보증 등에 관한 표시·광고 심사지침」 준수:

- [ ] **"#광고" 또는 "유료광고 포함" 명시** (해시태그 또는 1줄 문구)
- [ ] **표기 위치**: 본문 첫 3줄 또는 영상 첫 3초
- [ ] **숨김 표기 금지** (해시태그 더보기 안에 가리기 ❌)
- [ ] **금전·물품 제공 명시** ("제품 무상 제공" 또는 "협찬")
- [ ] **표기 글자 크기**: 본문 글자 크기 이상 (작게 숨기기 ❌)
- [ ] **반복 노출**: 영상의 경우 시작·중간·끝에 모두 표기

## UGC 리그램 가이드

고객 후기·인스타 게시물을 브랜드 계정에서 리그램(repost)할 때:

- [ ] **사전 동의 받기** (DM 또는 댓글로 명시 동의)
- [ ] **출처 명확 표기** (@사용자명 태그)
- [ ] **사진·영상 무단 편집 금지** (원본 그대로)
- [ ] **상업적 활용 시 추가 동의** (광고·상세페이지 활용 등)

## 굿즈 기획법

브랜드 정체성 강화 + 충성 고객 락인 굿즈 5축:

| 굿즈 타입 | 목적 | 원가 비율 | 예시 |
|----------|------|---------|------|
| **시즌 한정 굿즈** | 발매 화제성 | 5-10% | 콜린스 1주년 핸드크림 미니어처 |
| **얼리버드 사은품** | 충성 100명 확보 | 10-15% | 비플레인 첫 100명 한정 굿즈 |
| **VIP 멤버 굿즈** | 락인 강화 | 5-10% | 29CM VIP 한정 토트백 |
| **이벤트 인증샷용** | 바이럴 유도 | 3-5% | 인스타 인증샷용 패키지 |
| **콜라보레이션** | 브랜드 확장 | 협상 | 디저트×향수·아티스트 콜라보 |

## 사용 예시

```
"/commerce-influencer-collab — 비건 스킨케어, 예산 100만원, 신생 D2C, 인스타 위주"
→ 마이크로 5명 + 나노 10명 추천 + 뒷광고 회피 체크리스트 + 굿즈 안 1개
```

## 후속 체인 (텍스트 산출물 검수)

본 스킬의 협업 제안서·뒷광고 회피 안내·UGC 리그램 동의 요청 메시지·굿즈 기획서 텍스트는 `gil:ai-slop-reviewer`로 자동 후처리 체이닝됩니다 (CLAUDE.local.md §3-2 HARD). 인플루언서 제안 카피는 외부 발송 텍스트이므로 검수 필수.

```text
commerce-influencer-collab → ai-slop-reviewer → (선택) sns-content
```

## 관련 스킬

- `campaign-planner` — 전체 캠페인 기획 (페어)
- `commerce-marketing-compliance-kr` — 정통망법 (표시광고법 별개)
- `sns-content` — 인플루언서용 콘텐츠
- `ai-slop-reviewer` — 협업 제안 카피 AI 슬롭 검수 (자동 체인, HARD)
