# 가격 전략 (Pricing)

## 1. 3단계 가격 구조

| 단계 | 가격 | 노출 |
|------|------|------|
| **정가** (List Price) | 100% | 자사몰 일반·후기 작성 후 노출용 |
| **할인가** (Discount Price) | 80-85% | 자사몰 + 일반 마켓 평상시 |
| **프로모션가** (Promotion Price) | 70-75% | 시즌 캠페인·블프·기념일 |

**이유**: 정가는 "이게 원래 얼마인지" 앵커. 할인가는 일상 거래가. 프로모션가는 트래픽 폭발 시점.

## 2. 채널별 가격 정책

### 권장 가격 정합성 룰

```
자사몰 ≤ 마켓플레이스 ≤ 큐레이션·크라우드펀딩
```

이유:
- 자사몰이 가장 저렴해야 직거래 유도 (수수료 절감)
- 단 마켓 vs 자사몰 가격차는 **5% 이내**가 안전 (마켓 정책 위반 X)
- 큐레이션·크라우드펀딩은 한정 가치로 정가 가능

### 채널별 권장가 예시 (정가 149,000원 기준)

| 채널 | 가격 | 차이 | 사유 |
|------|------|------|------|
| 자사몰 (회원) | 119,000 + 적립 5% | 정가 -25% | 회원 LTV 우대 |
| 자사몰 (일반) | 129,000 | 정가 -13% | 표준 |
| 쿠팡 | 124,900 | 정가 -16% | 마켓 수수료 반영 + 경쟁가 |
| 스마트스토어 | 129,000 | 정가 -13% | 자사몰 동일 |
| 와디즈 (펀딩) | 89,000-119,000 | 정가 -20-40% | 한정 사전판매 |
| 무신사·29CM | 149,000 | 정가 그대로 | 큐레이션 가치 |

## 3. 가격 결정 요소

### 원가 + 마진 (출발점)
```
원가율: 30~40% (제조)
배송비: 2,000~3,000원
PG 수수료: 2~3%
마켓 수수료: 8~33%
광고비 평균: 매출 15~25%
운영 인건비: 매출 10~15%
순이익: 5~15%
```

### 경쟁사 분석
- 다나와·네이버 쇼핑 검색 → 카테고리 동급 가격대 분포
- 1분위(저가) / 2-3분위(중가) / 4분위(프리미엄) 중 어디 포지셔닝?

### 브랜드 포지셔닝
- mass: 가격 경쟁 (Mass)
- premium_indie: 가성비 + 디자인 (디자이너 브랜드)
- luxury: 가격은 중요하지 않음 (브랜드 가치)

## 4. 할인의 함정

### 단기 효과 vs 장기 비용

- **단발성 프로모션 (시즌·블프)**: OK. 트래픽 폭발 + 신규 고객 확보.
- **상시 할인**: NO. 고객이 정가를 신뢰하지 않게 됨. 브랜드 가치 훼손.

### 제대로 된 할인 운영

```
프로모션 캘린더에 명시된 시점에만 할인
↓
"오늘만", "한정 수량" 명확히 (소비자 보호법 준수)
↓
프로모션 후 정가 복귀 (즉시)
```

## 5. 가격 표시 법규

한국 표시광고법:
- 이중가격 표시 시 **"최근 30일 최저가"** 명시 필수
- "정가 ₩149,000" 표기는 실제 30일 내 정가 판매 이력 있어야
- "최저가" 단정 표현 금지 (객관 근거 없으면 거절)
- 무료배송 조건 명확화 ("5만원 이상" 등)

## 6. 묶음·패키지 가격

### 객단가 상승 전략
- 2개 묶음 (정가 -5%) + 1개 추가 시 -10%
- "함께 사면 좋은 상품" (cross-sell)
- 사은품·증정 (브랜드 가치 유지)

### 정기배송 가격
- 일회성 대비 -15-20%
- 자사몰 LTV 핵심 도구
- 식품·건기식·생활용품에 효과적

## 7. 가격 변경 신호

- 원가 상승 5% 이상 → 가격 인상 검토
- 경쟁사 신상품 출시 → 포지셔닝 재점검
- 후기 평점 4.7+ + 매출 정체 → 가격 인상 가능 (브랜드 가치 ↑)
- ROAS 200% 미만 + 매출 정체 → 가격 인하 또는 광고비 축소
