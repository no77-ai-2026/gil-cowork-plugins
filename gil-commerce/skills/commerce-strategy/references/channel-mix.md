# 채널 믹스 (Channel Mix)

## 단계별 권장 채널 비중

### 런칭기 (0-6개월)

목표: **빠른 트래픽 확보 + 첫 매출**

```
70% 마켓플레이스 (쿠팡 + 스마트스토어)
20% 자사몰 (브랜드 페이지 운영)
10% 크라우드펀딩 (와디즈/텀블벅 — 신상 검증)
```

이유: 자사몰만 운영하면 광고비 폭증 + 매출 0. 마켓플레이스가 유입을 보장.

### 성장기 (6-24개월)

목표: **데이터 축적 + 채널 다각화 + LTV 시작**

```
50% 마켓플레이스 (쿠팡·스마트스토어 + 11번가/G마켓)
40% 자사몰 (광고 ROAS 200~300%)
10% 큐레이션·크라우드펀딩 (신상 발매·콜라보)
```

이유: 자사몰 데이터·재구매·이메일 자산이 쌓이기 시작. 단 마켓 수수료 대비 자사몰 광고 ROI 확인 필수.

### 안정기 (24개월+)

목표: **LTV 극대화 + 브랜드 자산**

```
30% 마켓플레이스 (탐색·신규 확보 채널)
60% 자사몰 (충성 고객·재구매·이메일·알림톡)
10% 큐레이션·콜라보 (브랜드 가치 강화)
```

이유: 자사몰 LTV 기반이 안정적이면 마켓 의존도 낮춰 수수료 절감.

## 카테고리별 채널 가중치

### 패션·뷰티
- 무신사·29CM 큐레이션 입점 시 채널 가치 높음
- 자사몰 비중 일찍부터 높일 수 있음
- 인스타·유튜브 SNS 트래픽 강함

### 가전·테크
- 와디즈 사전판매 → 일반 마켓 → 자사몰 흐름 권장
- 후기·리뷰가 핵심 → 마켓 후기 30건+ 누적 우선

### 식품·건기식
- 카카오 메이커스 + 쿠팡로켓프레시 + 자사몰
- 정기배송 자사몰 핵심 (LTV 가장 큼)

### 라이프스타일·홈리빙
- 29CM 입점이 브랜드 가치
- 자사몰 + 인스타·블로그 콘텐츠

### 핸드메이드·아트
- 텀블벅 + 자사몰 + 인스타
- 마켓플레이스 비중 낮아도 됨 (소량·고감도)

## 채널 추가·제거 의사결정

### 추가 신호
- 현재 채널 매출이 정체 (3개월 연속 매출 평행)
- 새 채널의 ROAS 시뮬레이션이 200%+ 예상
- 인력·재고가 다채널 운영 가능 수준

### 제거 신호
- 채널 ROAS 6개월 연속 100% 미만
- 재고 분산으로 SKU 관리 비용 증가
- 마켓 수수료보다 자사몰 광고 비용이 더 효율적
