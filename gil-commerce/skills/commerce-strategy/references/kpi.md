# KPI 대시보드 (Commerce KPIs)

## 1. 매출 KPI

| 지표 | 정의 | 권장 빈도 |
|------|------|---------|
| 일 매출 | 자사몰 + 마켓 합산 | 매일 |
| 채널별 매출 비중 | 마켓·자사몰·기타 | 매주 |
| 신규 고객 매출 | 첫 구매 회원 | 매주 |
| 재구매 매출 | 2회 이상 구매 회원 | 매주 |
| AOV (Average Order Value) | 평균 객단가 | 매주 |

## 2. 트래픽·전환 KPI

| 지표 | 권장 목표 |
|------|---------|
| 자사몰 일 방문자 | 카테고리별 |
| 신규 vs 재방문 비율 | 7:3 (런칭) → 4:6 (안정) |
| 전환율 (구매/방문) | 1-2% (전체) / 3-5% (자사몰 회원) |
| 카트 추가율 | 5-10% |
| 카트 이탈률 | 60-70% (낮을수록 좋음) |
| 결제 완료율 | 30-40% (카트 → 결제) |

## 3. 광고 KPI

| 지표 | 정의 | 목표 |
|------|------|------|
| ROAS | 광고매출 / 광고비 | 200%+ (런칭) → 400%+ (안정) |
| CAC | 신규 고객 1명 획득 비용 | 카테고리 평균 이하 |
| CPM | 1,000회 노출 비용 | 채널별 다름 |
| CTR | 클릭률 | 1-3% (Meta) / 5-10% (네이버 검색) |

## 4. 리텐션·LTV KPI

| 지표 | 정의 | 목표 |
|------|------|------|
| 재구매율 (30일) | 첫 구매 후 30일 내 재구매 | 25%+ |
| 재구매율 (90일) | 90일 누적 | 40%+ |
| 평균 구매 횟수 | 회원당 누적 | 1.5회+ |
| LTV (90일) | 90일 누적 매출 / 회원 수 | AOV × 1.5+ |
| LTV (365일) | 1년 누적 | AOV × 3+ |

## 5. 만족도·운영 KPI

| 지표 | 목표 |
|------|------|
| 평균 평점 | 4.5+ |
| 후기 작성률 | 30%+ |
| 1점 후기율 | 5% 이하 |
| 반품률 | 5% 이하 |
| 분쟁 발생률 | 1% 이하 |
| CS 응답 시간 | 24시간 이내 |
| NPS (Net Promoter Score) | 50+ |

## 6. 재무 KPI

| 지표 | 정의 |
|------|------|
| 매출총이익률 (Gross Margin) | (매출 - 원가) / 매출 |
| 영업이익률 | (매출 - 원가 - 광고비 - 인건비 - 운영비) / 매출 |
| 손익분기점 (BEP) | 고정비 / (단위 매출 - 변동비) |
| 현금흐름 | 정산일 기준 (마켓별 다름) |

## 7. 대시보드 템플릿

### 일일 (매일 점검)
```
[ ] 일 매출 (자사몰 + 채널별)
[ ] 일 광고비 + ROAS
[ ] 신규 가입 수
[ ] 신규 구매자 수 + 전환율
```

### 주간 (매주 점검)
```
[ ] 주 매출 추이 (전주 대비)
[ ] 채널별 매출 비중
[ ] AOV 추이
[ ] 재구매 횟수
[ ] 광고 채널별 ROAS
[ ] 카트 이탈률
[ ] 신규 후기 + 평점
```

### 월간 (매월 점검)
```
[ ] LTV (30·90·365일)
[ ] 등급별 회원 분포 + 변동
[ ] 이메일 오픈/클릭률
[ ] 알림톡 효율
[ ] 반품률 + 분쟁
[ ] 광고 채널별 CAC
[ ] 매출총이익률
```

### 분기 (매분기 점검)
```
[ ] 채널별 매출 기여도 + 트렌드
[ ] 시즌 캠페인 ROI
[ ] NPS·고객 만족도
[ ] 재고 회전율
[ ] 손익 (영업이익 + 영업이익률)
[ ] 전략 회고·다음 분기 목표
```

## 8. 도구

### 기본 분석
- Google Analytics 4 (GA4)
- 네이버 애널리틱스 (Naver Premium Log Analytics)
- Meta 픽셀 + 광고관리자
- 카페24·아임웹 자체 통계

### 광고 분석
- 네이버 광고관리시스템
- Meta Business Suite
- 카카오 모먼트
- 구글 애즈

### 후기·만족도
- 카페24·아임웹 자체 후기
- 채널톡 (CSAT 측정)
- Typeform·Google Forms (NPS 조사)

### 통합 대시보드
- Google Looker Studio (무료)
- Notion + Charts (라이트)
- 자체 BI (월 매출 1억+ 시 권장)

## 9. KPI 시각화 권장

- 시계열 그래프 (일·주·월)
- 채널별 비교 막대그래프
- 코호트 분석 (동기 가입 회원의 LTV 추이)
- 깔때기 (방문 → 카트 → 결제 → 재구매)
