# 리텐션·LTV (Customer Retention)

## 1. 리텐션이 왜 중요한가

- 신규 고객 획득 비용(CAC) > 재구매 유도 비용 5-10배
- LTV (Lifetime Value) = 평균 구매액 × 구매 빈도 × 고객 유지 기간
- 자사몰 사업의 안정성·수익성은 LTV가 결정

## 2. 자동화 트리거 (이메일·알림톡)

### 첫 구매 후 시퀀스

| 시점 | 채널 | 콘텐츠 | 목적 |
|------|------|------|------|
| 구매 직후 | 이메일 | 주문 확인서 + 배송 안내 + 사용법 미리 보기 | 신뢰 |
| +3일 (배송 완료) | 알림톡 | 배송 완료 + 사용법 단계별 안내 | 만족도 |
| +7일 | 이메일 | 첫 사용 후기 (가이드) + Q&A | 만족도 + 후기 |
| +14일 | 알림톡 | 후기 작성 시 적립금 5,000원 | 후기 |
| +30일 | 이메일 | 재구매 쿠폰 (10-20% 할인) | 재구매 |
| +60일 | 알림톡 | 추천 상품 (cross-sell) | 객단가 |
| +90일 | 이메일 | 회원 등급 업그레이드 안내 | 충성도 |

### 카트 이탈 회복

| 시점 | 채널 | 콘텐츠 |
|------|------|------|
| 이탈 +1시간 | 이메일 | "장바구니에 두고 가셨네요" |
| 이탈 +24시간 | 이메일 | 장바구니 5% 추가 할인 코드 |
| 이탈 +72시간 | 알림톡 | 마지막 안내 + 무료배송 코드 |

### 비활성 회원 회복

| 마지막 구매 | 콘텐츠 |
|-----------|--------|
| 60일 미구매 | 신상품 안내 + 5천원 적립 |
| 90일 미구매 | 재구매 한정 쿠폰 (15%) |
| 180일 미구매 | "오랜만이에요" + 명품 사은품 |
| 365일 미구매 | 회원 등급 강등 안내 (회복 동기) |

## 3. 등급제 (Tier Program)

```
일반 (구매액 0~10만)
   ↓
실버 (10~50만): 적립 2배
   ↓
골드 (50~150만): 무료배송 + 5% 할인
   ↓
VIP (150만+): 전용 쿠폰 + 신상 우선 + 생일 선물
```

### 등급 운영 핵심
- 등급 진입·승격 메시지 명확 (축하·기대감)
- 등급별 차별화된 혜택 가시성
- 등급 유지 조건 (1년간 X원 이상)

## 4. 적립금·쿠폰 운영

### 적립금
- 신규 가입 즉시 5,000원
- 첫 구매 후 5% 적립
- 후기 작성 시 1,000-5,000원
- 친구 추천 시 양쪽 5,000원

### 쿠폰 종류
- 신규 회원 쿠폰 (5-10%)
- 첫 구매 쿠폰 (5,000원)
- 재구매 쿠폰 (10-15%)
- 등급별 쿠폰 (VIP 한정 20%)
- 시즌 쿠폰 (블프 30%)
- 카트 회복 쿠폰 (5%)

## 5. 정기배송 (구독)

LTV 가장 큰 도구. 식품·건기식·생활용품에 효과적.

| 항목 | 권장 |
|------|------|
| 할인폭 | 일회 대비 -15-20% |
| 배송 주기 | 30/45/60일 선택 |
| 이탈 방지 | 다음 회차 전 알림 + 일시 중단 옵션 |
| 보너스 | 3회 연속 구독 시 사은품 |

## 6. 후기·리뷰 활용

- 포토 후기 적극 유도 (적립 5,000원)
- 후기 활용 콘텐츠 (인스타·매거진)
- 부정 후기 응답 (사과 + 개선 약속)
- 후기 점수 4.5+ 유지가 검색 노출에 결정적

## 7. CRM 도구

- **카페24**: 자체 적립·쿠폰·등급 (기본)
- **아임웹**: 자체 + 채널톡 연동
- **외부 도구**: 채널톡, 굿스플로, MAILY (이메일), 알리고/문자나라 (SMS·알림톡)

## 8. KPI 대시보드 (리텐션 지표)

매월 점검:
- [ ] 30일 재구매율 (목표 25%+)
- [ ] 90일 재구매율 (목표 40%+)
- [ ] 평균 구매 횟수 (목표 1.5회+)
- [ ] LTV 90일 / LTV 365일
- [ ] 등급별 회원 분포
- [ ] 이메일 오픈율 (목표 25%+)
- [ ] 알림톡 클릭율 (목표 5%+)
- [ ] 카트 이탈 회복율 (목표 5%+)

## 9. 흔한 실수

- 쿠폰 남발 (브랜드 가치 훼손)
- 모든 회원에게 동일한 메시지 (개인화 부재)
- 첫 구매 직후 추가 판매 압박 (신뢰 훼손)
- 등급제 혜택 애매 (동기 부재)
- 비활성 회원 방치 (LTV 감소)
