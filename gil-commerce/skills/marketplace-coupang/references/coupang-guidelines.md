# 쿠팡 상세페이지 정책 상세 가이드

> 본 문서는 일반적 가이드이며 쿠팡 정책은 수시로 변경됩니다. 실제 등록 전 wing.coupang.com 최신 공지 확인 필수.

## 1. 이미지 규격

### 메인(대표) 이미지
- 크기: 1000×1000 픽셀 정사각형 권장 (최소 500×500)
- 형식: JPG / PNG
- 파일 용량: 5MB 이하
- 배경: 가능하면 흰색 또는 투명
- 워터마크/로고: 5% 이하 크기 권장
- 모델·인물: 카테고리별 가이드 적용

### 상세 이미지
- 너비: 780px 이상 (1080px 권장)
- 세로 길이: 무제한 (단, 1장 이미지 < 5MB)
- 동일 이미지 반복 X
- GIF 사용 시 < 8초

## 2. 카테고리별 추가 정책

### 식품
- 식품유형 명시 (식약처 분류)
- 알레르기 유발 성분 표기
- 보관·유통기한 명시
- "건강에 좋다", "효능" 등 의학적 표현 금지

### 화장품
- 화장품 유형 표시
- 전성분 표기
- 사용기한 명시
- "주름개선", "미백" 등은 식약처 인증 시에만 표기 가능

### 건강기능식품
- 식약처 건강기능식품 마크 필수
- 일일 섭취량 명시
- 질병 치료·예방 단정 표현 금지

### 의류
- 사이즈표 필수 (cm 단위)
- 소재 함량 표기 (%)
- 세탁법 표기

### 디지털/가전
- KC 인증번호 표기 필수 (전기용품)
- A/S 정책 명시
- 정격 전압·소비전력

## 3. 금지 문구 카탈로그

### 절대 표현 (객관 근거 없을 시 거절)
- "최고", "최저", "1위", "유일", "독보적"
- "최초", "원조" (역사적 입증 시 한정 허용)
- "완벽", "100%", "절대"

### 보장성 표현
- "효과 보장", "결과 보장", "환불 보장 100%"
- "확실한 효과", "반드시"

### 의학적 단정 (식품/화장품)
- "치료", "예방", "완치"
- 특정 질병명 + 효능 연결
- "임상 결과 입증" (출처 미기재 시)

### 경쟁사·우회 표현
- 경쟁사 브랜드명 직접 비교
- 외부 사이트 URL/연락처
- "직거래", "현금결제 할인", "택배비 별도"

## 4. 검색 최적화 (쿠팡 SEO)

### 상품명 패턴
```
[브랜드] [상품명] [핵심스펙] [용도] [용량/사이즈]
```

50자 이내. 첫 20자에 핵심 키워드 집중.

### 좋은 예시
- ✅ `"닥터자비 무선이어폰 X200 ANC 30시간 직장인 통화 블루투스"`
- ✅ `"이마트 비건 비누 무향 100g 4개입 민감피부 약산성"`

### 나쁜 예시
- ❌ `"X200 무선이어폰"` (단순)
- ❌ `"#무선이어폰 #ANC #블루투스 #직장인 #출퇴근..."` (해시태그)
- ❌ `"X200 ★★★ 무선이어폰 ★★★ 1+1 이벤트"` (특수문자 남용)

### 검색태그 (최대 20개)
- 카테고리 메인 5개
- 사용 상황 5개 (출퇴근, 운동, 미팅 등)
- 타겟 5개 (직장인, 학생, 운동인 등)
- 시즌/트렌드 5개 (선물, 베스트, 신상품 등)

## 5. 우수상품 기준

### 배송
- 출고 정확도 95%+
- 약속 배송일 준수 95%+

### 후기
- 평균 평점 4.5+
- 최소 30건 이상
- 1점 후기율 < 5%

### 반품
- 반품율 5% 이하
- CS 응답 24시간 이내

### 상세페이지 품질
- 이미지 5장 이상
- 옵션 명확
- A/S 정책 명시

## 6. 로켓배송 vs 판매자배송

| 항목 | 로켓배송 | 판매자배송 | 로켓그로스 |
|------|---------|----------|----------|
| 배송 주체 | 쿠팡 풀필먼트 | 판매자 직접 | 쿠팡 풀필먼트 (3PL) |
| 입고 방식 | 사전 입고 | 주문 후 출고 | 사전 입고 |
| 배송 속도 | 익일 보장 | 2-3일 | 익일 보장 |
| 수수료 | 카테고리별 8-13% | 8-13% + 별도 풀필먼트 비용 X | 8-13% + 풀필먼트 비용 |
| 노출 우대 | ★★★ | ★ | ★★ |
| 적합 카테고리 | 회전율 높은 일상품 | 신상·테스트·비싼 상품 | 중간 |

### 로켓배송 등록 절차
1. 카테고리·상품 등록
2. 입고 신청
3. 쿠팡 물류센터 입고
4. 검수 + 활성화 (평균 3-5일)
5. 판매 시작

## 7. 클레임·환불 정책

- 단순 변심 환불: 7일 이내
- 상품 불량 환불: 30일 이내
- 부분 환불 사유 명시 필수
- 사진 증빙 요구 가능
