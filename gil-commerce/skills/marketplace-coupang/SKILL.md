# 쿠팡 상세페이지 가이드 (Coupang Marketplace)

## 개요

한국 최대 이커머스 채널 쿠팡(Coupang)의 상세페이지 정책·검색최적화·우수상품 기준에 맞춰
상품 등록을 최적화하는 가이드 스킬입니다.

## 트리거 키워드

쿠팡, 쿠팡 상세페이지, 쿠팡 등록, 쿠팡 가이드, 로켓배송, 판매자배송, 쿠팡 SEO,
쿠팡 검색 키워드, 쿠팡 우수상품, 쿠팡 정책, 쿠팡 금지문구

## 워크플로우

### 1단계: 채널 옵션 확인

다음을 확인합니다:
- 로켓배송 vs 판매자배송 vs 로켓그로스 (Rocket Growth)
- 쿠팡플레이/쿠팡이츠 등 부가 채널 여부
- 카테고리 (식품/생활/패션/뷰티/디지털 등)

### 2단계: 정책 체크리스트 적용

상세 가이드는 `references/coupang-guidelines.md` 참조. 핵심 체크:

#### 이미지 규격
- 메인 이미지: 1000×1000px 정사각형 (JPG/PNG)
- 상세 이미지: 너비 780px 이상 권장, 세로 길이 무제한
- 1080×12720 합성본은 그대로 업로드 가능 (자동 압축)
- 이미지 파일 크기 < 5MB
- 모자이크/필터 과도한 사용 금지

#### 금지 문구 (등록 거절·계정 정지 가능)
- "최저가", "최고", "1위" 등 절대 표현 (객관 근거 없을 시)
- "100% 효과", "완벽한", "확실한" 등 보장성 표현
- 의학적 효능 단정 (식품/화장품)
- 경쟁사 비방
- 외부 사이트/연락처 노출
- "직거래", "현금" 등 우회 결제 유도 표현

#### 우수상품 기준 (선정 시 노출 우대)
- 배송 정확도 95%+
- 반품율 5% 이하
- 후기 평점 4.5+ (최소 30건)
- 상세페이지 충실도 (이미지 5장+ 권장)

### 3단계: 검색 키워드 최적화 (쿠팡 SEO)

쿠팡은 **상품명 + 검색태그**를 검색 점수에 반영합니다.

상품명 작성 패턴:
```
[브랜드] [상품명] [핵심스펙] [용도/타겟] [용량/사이즈]
예: "닥터자비 무선이어폰 X200 ANC 30시간 직장인 출퇴근 통화이어폰"
```

- 50자 이내 (모바일 노출 영역)
- 핵심 키워드 앞쪽 배치
- 띄어쓰기로 형태소 분리 명확화
- 시즌·트렌드 키워드 활용 (예: "선물용", "올해", "베스트")

### 4단계: 검색 태그 (#태그)

최대 20개. 추천 구성:
- 카테고리 메인 키워드 (5)
- 사용 상황·타겟 (5)
- 경쟁 제품·대체제 키워드 (5)
- 시즌·트렌드 (5)

### 5단계: 등록 전 최종 점검

```
[ ] 메인 이미지 1000×1000 정사각형
[ ] 상세 이미지 780px+ 너비
[ ] 금지문구 0건
[ ] 상품명 50자 이내, 핵심 키워드 앞쪽
[ ] 검색태그 20개 충실
[ ] 옵션·가격·재고 정확
[ ] A/S 정책 명시
[ ] 로켓배송이면 입고 일정 확인
```

## 관련 스킬

- `gil-commerce:detail-page-copy` — 카피 생성
- `gil-commerce:detail-page-image` — 이미지 합성
- `gil-commerce:marketplace-naver` — 네이버/오픈마켓 비교
- `gil-marketing:seo-audit` — 채널 SEO 일반

## 이 스킬을 사용하지 말아야 할 때

- 네이버 스마트스토어 가이드: `marketplace-naver` 사용
- 자사몰(Cafe24/Shopify) 등록: 별도 가이드 필요
- 일반 마케팅 콘텐츠: `gil-marketing` 플러그인

## 주의사항

쿠팡 정책은 수시로 업데이트됩니다. 본 가이드는 일반적 기준이며, 실제 등록 전
[쿠팡 판매자센터](https://wing.coupang.com)의 최신 공지를 확인하세요.
특히 식품·화장품·건강기능식품은 식약처 광고심의 통과가 별도로 필요합니다.
