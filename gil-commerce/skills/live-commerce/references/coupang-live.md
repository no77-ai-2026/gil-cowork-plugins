# 쿠팡 라이브 가이드

> 쿠팡 셀러 직접 운영형 라이브. 로켓배송 셀러·30-50대 사용자 강점.

## 1. 가입 + 권한

- 쿠팡 셀러 자격 (로켓배송 셀러 우대)
- 쿠팡 라이브 운영 신청 (별도 심사)
- 셀러 등급에 따라 차등 권한

## 2. 쿠팡 라이브 특성

- **로켓배송 연동**: 라이브 중 즉시 구매 → 익일 도착
- **검색·노출 강점**: 쿠팡 자체 검색에 라이브가 우선 노출
- **30-50대 사용자**: 신뢰·정보 톤 적합
- **수수료 일반 마켓 대비 낮음**: 5-10%

## 3. 송출 + 장비

- 쿠팡 라이브 셋업 도구 (자체 앱 또는 OBS)
- 일반 쇼핑라이브와 비슷한 장비
- 로켓 물류 셋업 우선 점검 (라이브 중 즉시 출고 약속)

## 4. 라이브 한정 혜택

- 라이브 한정 가격 (정가·평소 가격보다 저렴)
- 로켓배송 무료배송 강조
- 쿠팡 와우 회원 추가 할인

## 5. 노출

### 메인 페이지
- 라이브 카드 노출
- 카테고리별 라이브 탭

### 검색
- 검색어 입력 시 "지금 라이브 중" 표시
- 라이브 종료 후 다시보기 검색 노출

### 푸시
- 알림 신청자에게 시작 푸시
- 와우 회원 우선 알림

## 6. 진행 톤

- 정보·신뢰 우선 (네이버 쇼핑라이브와 비슷)
- 로켓배송 강조 ("내일 도착")
- 와우 회원 혜택 명시
- 30-50대 타겟이라 차분·전문 톤

## 7. 수수료

- 쿠팡 라이브 수수료: 5-10% (카테고리·셀러 등급별)
- 로켓배송 수수료 별도 (8-13%)
- 중복 발생 시 셀러 부담

## 8. KPI

| 지표 | 권장 |
|------|------|
| 동시 시청자 | 100-3,000 |
| 평균 시청 시간 | 5-10분 |
| 전환율 | 5-12% (로켓배송 강점) |
| 객단가 | 카테고리별 |

## 9. 적합 시나리오

### 추천
- 로켓배송 운영 셀러
- 30-50대 타겟 (가전·생활·식품)
- 정보·신뢰 톤
- 즉시 구매·익일 배송 강점

### 비추천
- 판매자배송 단독 (로켓배송 강점 활용 X)
- 10-20대 타겟 엔터 톤 (그립 적합)
- 셀러 등급 낮음 (라이브 권한 제한)
