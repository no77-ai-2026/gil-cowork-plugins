# 그립 (Grip) 가이드

> 인플루언서·셀럽 중심 라이브 커머스. 10-20대 코어. 엔터·재미 톤.

## 1. 가입 + 권한

- 셀러 직접 운영보다 **인플루언서 호스트 매칭**이 일반적
- 그립 셀러 신청 (https://www.grip.show/)
- 인플루언서 운영자 신청 (그립 자체 시스템)

## 2. 그립의 특성

- **인플루언서 중심**: 그립의 호스트가 주인공. 셀러는 상품 공급자.
- **엔터·재미 톤**: 단순 상품 소개보다 인플루언서의 캐릭터·재미가 비중 큼
- **10-20대 코어**: 패션·뷰티·잡화 강세. 신뢰·정보 중심 카테고리는 약함
- **수수료 높음**: 12-15% (인플루언서 비용 포함)

## 3. 운영 방식

### 패턴 1: 그립 호스트 매칭
1. 셀러가 상품 등록
2. 그립 호스트가 라이브 신청
3. 셀러 동의 후 라이브 진행
4. 매출 기준 수수료 정산

### 패턴 2: 자체 호스트
1. 셀러가 직접 호스트 (또는 자체 인플루언서)
2. 그립 운영 신청 + 심사
3. 라이브 진행
4. 수수료 12-15% 그립 부담

## 4. 적합 카테고리

| 카테고리 | 적합도 |
|---------|------|
| 패션·잡화 | ★★★ |
| 뷰티 | ★★★ |
| 식품 (간식·디저트) | ★★ |
| 가전·테크 | ★ (10-20대 구매력 낮음) |
| 홈리빙 | ★ |
| 명품·럭셔리 | ★ |

## 5. 호스트 매칭 팁

### 고려 요소
- 호스트 팔로워·시청자 수
- 호스트 캐릭터 ↔ 브랜드 매칭
- 호스트 단가 (매출 % 또는 정액)
- 라이브 빈도 (주 1회 이상 운영자가 매출 안정)

### 인기 호스트 카테고리
- 패션 인플루언서
- 뷰티 유튜버
- 먹방 크리에이터
- 운동·라이프 인플루언서

## 6. 라이브 톤

네이버·카카오·쿠팡과 매우 다름:
- 엔터·재미 우선 (단순 상품 소개로는 시청자 이탈)
- 호스트의 캐릭터·일화·솔직한 후기
- 채팅 활발 (호스트가 시청자 호명·대화)
- 즉흥성·라이브 특유 텐션

## 7. 수수료·정산

- 라이브 수수료: 12-15% (호스트·플랫폼 통합)
- 인플루언서 별도 비용 (계약별 다름)
- 정산: 그립 시스템 통해 월 1-2회

## 8. KPI

| 지표 | 권장 |
|------|------|
| 동시 시청자 | 인플루언서 팔로워의 5-15% |
| 평균 시청 시간 | 10-30분 (엔터 특성) |
| 전환율 | 5-15% (충성 팬덤 효과) |
| 객단가 | 1만-3만원 (10-20대 구매력) |

## 9. 적합 시나리오

### 추천
- 패션·뷰티·잡화 (10-20대 타겟)
- 인플루언서 매칭 가능 또는 자체 운영자 보유
- 엔터·재미 톤 운영 가능
- 마진율 높아 12-15% 수수료 흡수 가능

### 비추천
- 신뢰·정보 중심 카테고리 (가전·건기식·B2B)
- 30-50대 타겟 (그립 사용자 부재)
- 마진율 낮음
