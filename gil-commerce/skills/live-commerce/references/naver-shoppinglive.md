# 네이버 쇼핑라이브 가이드

> 한국 라이브 커머스 1위. 검색 노출 + 30-50대 사용자 강점.

## 1. 가입 + 권한

- 스마트스토어 셀러 자격 필수
- 검색 노출 우대를 받기 위한 추가 조건:
  - 후기 30건+ / 평점 4.5+
  - 출고 정확도 95%+
- 라이브 셋업: https://sell.smartstore.naver.com → 쇼핑라이브 메뉴

## 2. 라이브 형식

### 송출 도구
- 모바일 앱 (네이버 쇼핑라이브 앱)
- PC OBS (OBS Studio + 송출 키)
- 스마트폰 + 짐벌 (가장 대중적)

### 장비 권장
- 카메라: 스마트폰 4K + 짐벌
- 조명: 링라이트 + 키라이트
- 마이크: 라발리에 또는 핀마이크
- 인터넷: 유선 또는 5GHz Wi-Fi 안정

## 3. 송출 권한 등급

| 등급 | 자격 | 노출 우대 |
|------|------|---------|
| 일반 | 스마트스토어 셀러 | 일반 |
| 우수 셀러 | 위 조건 + 매출·후기 | 검색 우대 |
| 라이브 우수 | 라이브 횟수 + 평점 + 시청 시간 | 메인 노출 |

## 4. 사전 준비

### D-3
- 라이브 등록 (제목·썸네일·일정·진행자)
- 상품 라이브 등록 (라이브 한정 가격 설정 가능)
- 알림 신청 받기 (배너 노출)

### D-1
- 카카오톡 단톡방·인스타·블로그에 사전 안내
- 이메일·알림톡 (회원에게)

### D-Day
- 1시간 전 셋업·테스트 송출
- 5분 전 라이브 전환

## 5. 라이브 한정 혜택

라이브에서만 가능한 가격·혜택:
- 라이브 한정 가격 (정가·평소 마켓가보다 저렴)
- 라이브 쿠폰 (코드 입력)
- 1+1·증정 (한정 수량)
- 무료배송 (평상시 5만원 → 1만원)

이런 혜택이 시청 동기·구매 전환의 핵심.

## 6. 노출 메커니즘

### 검색 노출
- "네이버 쇼핑라이브" 검색 시 노출 (상위 우대)
- 카테고리별 라이브 탭
- 키워드 매칭 (제목·태그)

### 푸시·배너
- 알림 신청자에게 시작 푸시
- 메인 페이지 배너 (제휴·우수 셀러)

### 다시보기
- 라이브 종료 후 1주-1달간 자동 저장
- 다시보기 검색 노출도 가능

## 7. 수수료·정산

- 라이브 수수료: 5-6% (스마트스토어 동일)
- 라이브 한정 가격 + 쿠폰은 셀러 부담
- 네이버페이 수수료 별도

## 8. 라이브 진행 팁

### 첫 30초 — 가장 중요
- 진행자 인사
- 오늘 무엇을 보여주는가 (한 문장)
- 라이브 한정 혜택 미리 강조 ("자정까지 -30%")

### 5분마다 다시 안내
- 새 시청자 입장 다수
- 핵심 혜택·상품·기간 반복

### 채팅 응대
- 시청자 이름 호명 ("OOO님 안녕하세요")
- 질문에 즉시 답변
- 부정적 댓글에 침착·정중하게

### 마감 5분 전
- 카운트다운 ("3분 남았어요")
- 재고 확인 ("12개 남았습니다")
- "오늘 자정까지" 명확히

## 9. KPI

| 지표 | 권장 |
|------|------|
| 동시 시청자 | 100-5,000+ (단계별) |
| 총 시청자 | 동시의 5-10배 |
| 평균 시청 시간 | 5-10분 |
| 라이브 전환율 | 3-10% (일반 페이지보다 높음) |
| 다시보기 시청 | 라이브의 30-50% |
