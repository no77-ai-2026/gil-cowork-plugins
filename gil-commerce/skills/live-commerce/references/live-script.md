# 라이브 진행 스크립트 작성 가이드

## 1. 라이브 흐름 (60분 기준)

### 0:00-0:30 — 오프닝 (가장 중요)

처음 30초가 시청자 유지율을 결정합니다.

**좋은 예시**:
```
"안녕하세요! 오늘은 [브랜드명]의 X200 무선이어폰을 보여드릴 거예요.
아마 아직 X200 모르시는 분도 계실 거고, 이미 사용해보신 분도 계실 텐데요,
오늘 60분 동안 X200 정가 14만 9천을 단 9만 9천원으로,
거기에 무료배송까지 — 이 라이브에서만 가능합니다.
들어오신 김에 끝까지 보시면 절대 후회 없을 거예요!"
```

**나쁜 예시**:
- "안녕하세요... 오늘은... 음... X200을... 보여드리려고..."
- 길게 자기소개부터 시작 (시청자 이탈)
- 혜택을 마지막에 안내 (시청자가 이미 떠남)

## 2. 채팅 응대 패턴

### 호명
- 시청자 닉네임을 직접 부르며 답변
- "OOO님 안녕하세요!", "OOO님 좋은 질문이에요"

### 긍정 응대
- "맞아요!", "그럴 수 있죠", "정말 좋은 포인트예요"

### 부정 응대 (조심)
- "그럴 수도 있는데, 사실은..."
- "이런 경우라면 ~로 해결돼요"
- 절대 시청자 비난·반박 X

## 3. 호스트가 절대 하면 안 되는 것

- 화면을 너무 가까이·너무 멀리 (얼굴 위주, 상품은 보조 — 잘못된 비중)
- 카메라 안 보고 말함 (시청자와 눈맞춤 부재)
- 채팅 무시 (일방향 방송)
- 상품 사용해본 적 없음 (질문에 답 못함)
- 다른 채널과 가격 비교 비방 ("쿠팡보다 싸요" → 표시광고법 위반 가능)

## 4. 라이브 효율 측정

| 지표 | 좋은 라이브 |
|------|---------|
| 시청자 유지율 (10분차) | 60%+ |
| 평균 시청 시간 | 8-15분 |
| 전환율 | 5-12% |
| 객단가 | 일반 페이지 대비 1.5-2배 (묶음·라이브 한정) |
| 채팅수 | 분당 5-20개 |

## 5. 라이브 후 활용

- 다시보기 (각 플랫폼 자동 저장 1주-1달)
- 짧은 클립 재가공 (틱톡·릴스 30-60초)
- 후기·인사 메시지 (이메일·알림톡)
- 다음 라이브 일정 사전 안내
