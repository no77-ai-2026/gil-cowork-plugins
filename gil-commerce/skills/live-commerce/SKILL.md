# 라이브 커머스 가이드 + 스크립트 작성

## 개요

라이브 커머스는 실시간 영상 + 채팅 + 즉시 구매가 결합된 채널로,
한국 이커머스에서 가장 빠르게 성장 중인 영역입니다(2026년 시장 규모 10조 원+).
이 스킬은 4대 채널(네이버·카카오·그립·쿠팡) 정책 + 30/60분 라이브 진행 스크립트를 통합 안내합니다.

## 트리거 키워드

라이브 커머스, 쇼핑라이브, 네이버 쇼핑라이브, 카카오 쇼핑라이브, 그립, Grip,
쿠팡 라이브, 라이브 방송, 라이브 스크립트, 라이브 진행, 라이브 호스트

## 채널 비교

| 항목 | 네이버 쇼핑라이브 | 카카오 쇼핑라이브 | 그립 (Grip) | 쿠팡 라이브 |
|------|---------------|---------------|----------|----------|
| 사용자 | 30-50대 다수 | 20-40대 균등 | 10-20대 코어 | 30-50대 |
| 강점 | 검색·노출 강함 | 카카오톡 도달 | 인플루언서·셀럽 | 로켓 연동 |
| 콘텐츠 톤 | 정보·신뢰 | 친근·일상 | 엔터·재미 | 정보·할인 |
| 수수료 | 5-6% | 5-6% | 12-15% | 5-10% |
| 호스트 | 셀러 직접 또는 외부 | 셀러 또는 인플루언서 | 인플루언서 중심 | 셀러 직접 |
| 진입 난이도 | 중 (상품 등록만 되면) | 중 | 높음 (인플루언서 매칭 필요) | 중 |

## 워크플로우

### 1단계: 채널 선택

```
30~50대 + 정보·신뢰 톤  → 네이버 쇼핑라이브
20~40대 + 친근 톤       → 카카오 쇼핑라이브
10~20대 + 엔터·셀럽     → 그립
로켓배송 셀러 + 즉시구매 → 쿠팡 라이브
```

### 2단계: 라이브 기획

#### 길이 (권장)
- **30분**: 단일 상품 집중 (1-2개 SKU)
- **60분**: 라인업 소개 (3-5개 SKU)
- **90분+**: 종합 행사 (대형 시즌·신상 발매)

#### 구성 (60분 기준)
| 시간 | 단계 | 콘텐츠 |
|------|-----|------|
| 0-5분 | 오프닝 | 인사·소개·라이브 혜택 안내 |
| 5-15분 | 상품 1 | 시연·기능·가격 |
| 15-25분 | 상품 2 | 시연·기능·가격 |
| 25-35분 | 실시간 응대 | 채팅 질문 답변 |
| 35-45분 | 상품 3 | 시연·기능·가격 |
| 45-55분 | 종합 추천 | 묶음·번들·연관 |
| 55-60분 | 마감 CTA | 한정·재고·종료 카운트다운 |

### 3단계: 스크립트 작성

상세 가이드는 `references/live-script.md` 참조. 핵심:
- 오프닝 첫 30초가 시청 유지율 결정
- 5분마다 신규 시청자에게 다시 안내 (시청 도중 입장 다수)
- 채팅 호출 + 호명으로 참여 유도
- 마감 5분 전 카운트다운으로 긴급성 폭발

### 4단계: 라이브 운영

#### 사전
- 네이버·카카오·그립·쿠팡 라이브 셋업
- 카메라·조명·마이크
- 진행자(호스트) 외 채팅·운영 1명 (필요 시)
- 사전 광고 (D-3, D-1, D-Day 알림)

#### 본방
- 시청자수 모니터링 (모든 플랫폼 실시간)
- 채팅 실시간 응대
- 한정 수량 카운트다운
- 댓글 이벤트 (선착순 5명 추가 할인 등)

#### 사후
- 라이브 다시보기 (네이버·카카오·쿠팡 자동 저장, 그립은 짧음)
- 재생용 짧은 클립 (틱톡·릴스로 재가공)
- 데이터 분석 (시청자수·체류시간·전환율)

## 주의사항

- 라이브는 **사전 준비**가 가장 중요. 호스트·시연·기술 셋업 1주 전 리허설 권장.
- 첫 라이브는 시청자 100명 이하가 일반. 광고비 투입해도 폭발적 성장은 3-5회차부터.
- 한정 수량·카운트다운은 사실대로. 부풀리면 신뢰 훼손 + 표시광고법 위반 가능.
- 인플루언서 호스팅 시 광고 표시 의무 (#광고 또는 음성 안내).
