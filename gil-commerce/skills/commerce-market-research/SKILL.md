# 카테고리 시장조사 (Commerce Market Research)

## 개요

이커머스 셀러가 특정 카테고리에 진입하거나 상품을 기획할 때 필요한 거시·경쟁·검색 3축 시장조사 1장 리포트를 자동 생성하는 스킬입니다.

**V6 ↔ MCP 매핑** (SPEC-COMMERCE-V6-003 §5.3):
- `trend_search` — 거시 트렌드 데이터
- `trend_shopping_categories` — 쇼핑 카테고리 성장률
- `trend_shopping_keywords` — 카테고리 핵심 키워드 트렌드
- `keyword_for_product_idea` — 상품 기획 키워드 아이디어

**MCP 백엔드**: 본 스킬은 MoAI-Commerce MCP Phase 1 (SPEC-COMMERCE-MCP-002)의 위 4종 도구를 호출합니다. MCP 미출시 시점에는 강사 본인 워크스페이스 사전 녹화 영상 5분으로 시연 대체 (PDF §4 운영 노트 §S4 인용).

**Day 2 시연 시점**: 1교시 10:35–10:55

## 트리거 키워드

시장조사, 카테고리 분석, 경쟁사 동향, 트렌드 리서치, 쇼핑 트렌드, 카테고리 진입, 시장 규모, 키워드 기회, 거시 분석, 경쟁 강도, 상품 기획 리서치, 카테고리 성장률

## 워크플로우

### 입력 슬롯

| 항목 | 필수 여부 | 예시 |
|------|----------|------|
| 카테고리 | 필수 | "비건 스킨케어", "반려견 간식", "남성 운동복" |
| 가격대 | 권장 | "1-3만원", "5만원 이상" |
| 채널 | 권장 | 스마트스토어, 쿠팡, 자사몰 |
| 기간 | 선택 | "최근 3개월", "1년" (기본값: 최근 3개월) |

### MCP 호출 순서

```
1. trend_search(category, period)
   → 거시 트렌드: 검색량 증감률, 시즌성 패턴

2. trend_shopping_categories(category)
   → 쇼핑 카테고리 성장률, 경쟁 밀도

3. trend_shopping_keywords(category, top_n=20)
   → 상위 20 키워드 + 트렌드 방향

4. keyword_for_product_idea(category, price_range, channel)
   → 상품 기획 기회 키워드 10개
```

## 포지셔닝 5축 분석 (v2.4.0 신규)

기존 카테고리 진입 검토 시, 5축 중 어느 위치에 놓을지 결정:

| 축 | 핵심 메시지 | 광고 전략 | 적합 상품 예시 |
|----|------------|----------|---------------|
| **품질·프리미엄** | "이 카테고리에서 가장 좋은 것" | 기능 설명보다 경험·감성 | 고급 시계·럭셔리 가전 |
| **가성비** | "이 품질을 이 가격에" | 수치·비교·합리성 | 노브랜드·다이소 |
| **전문성** | "이 분야의 전문가만을 위한" | 특정 집단의 정체성 구매 독점 | B2B·의료기기 |
| **편의성** | "가장 쉽고 빠르게" | 시간 절약·간편함 | 새벽배송·간편식 |
| **가치관** | "우리는 이런 것을 믿는다" | 같은 가치관 타겟의 정체성 구매 | 비건·친환경·동물복지 |

### 포지셔닝 문장 만들기

```
[타겟]을 위한, [카테고리]에서, [경쟁자와 다른 점] 때문에 [핵심 혜택]을 주는 브랜드
```

예시: "**20-30대 직장인 여성**을 위한, **다이어트 보조제** 카테고리에서, **국내 임상시험·제조허가 인증** 때문에 **간편하면서도 안전한 체중 관리**를 주는 브랜드"

이 한 문장을 쓸 수 없으면 포지셔닝이 아직 없는 것.

## 새 카테고리 창출 vs 기존 카테고리 경쟁

| 전략 | 강점 | 약점 | 예시 |
|------|------|------|------|
| **기존 카테고리 내 경쟁** (약함) | 시장 검증됨 | 비교당함 + 가격 경쟁 | "더 좋은 에너지 드링크" → 레드불/몬스터와 비교 |
| **새 카테고리 창출** (강함) | 카테고리 1위 자동 획득 + 비교 대상 없음 | 시장 인지 비용 | "건강한 에너지 음료" → 새 카테고리 1위 |

> **권장**: 시장조사 결과 기존 카테고리에서 경쟁사 ROAS·CTR 평균 대비 본인 상품이 30% 이상 낮으면, **새 카테고리 창출** 검토. 새 카테고리는 키워드 검색량이 적어도 비교 대상이 없어 객단가·마진 모두 유리.

## 6대 영역 진단 통합 (v2.4.0 신규)

시장조사 시 단일 상품·카테고리뿐 아니라 셀러의 6대 운영 영역까지 함께 진단:

| 영역 | 시장조사 시 추가 점검 |
|------|---------------------|
| A 상품 운영 | 상품 등록 자동화 가능성·이미지 품질 표준 |
| B 가격 & 프로모션 | 경쟁사 가격 모니터링 자동화 가능성 |
| C 주문 & 정산 | 주문량 증가 시 운영 인력 확장성 |
| D 재고 & 물류 | 재고 회전율·창고비·배송사 협상력 |
| E 마케팅 & 고객 | 광고비 매출 비율·CRM·재구매율 |
| F 데이터 & 경영 | KPI 대시보드·BI 자동화 가능성 |

> **연계**: 상세 자동화 진단은 `commerce-automation-audit` 스킬에서 풀세트 진행.

## 시장 세분화 + USP 추출 프로세스 (v2.7.0 보강, MD 11년차 관점)

시장조사 보고서 1장에서 더 나아가 **시장 세분화(Segmentation) + USP(Unique Selling Proposition) 추출**을 표준화한 4단계 프로세스. 11년차 MD가 신상품 진입 전 의무 거치는 단계.

### Step 1: 시장 세분화 4축

같은 카테고리도 4축으로 쪼개면 진입 가능한 micro-segment가 보임:

| 축 | 질문 | 예시 (비건 스킨케어) |
|----|------|---------------------|
| **인구 통계** | 누가 사는가? | 25-35세 여성 / 직장인 / 1인 가구 |
| **심리 통계** | 어떤 가치관·라이프스타일? | 미닝아웃 / 클린뷰티 / 동물복지 / 윤리 소비 |
| **행동 패턴** | 어떻게 사는가? | 성분 검색형 / 후기 의존형 / 인플루언서 추종형 |
| **상황 맥락** | 언제 사는가? | 환절기 / 출퇴근길 점심 / 휴가 전 |

### Step 2: Segment 진입 우선순위 — 5축 평가

각 segment에 대해 다음 5축 평가:

```
[1] 시장 크기 (잠재 고객 수)
    ├─ 대(10만+) / 중(1~10만) / 소(~1만)

[2] 성장률 (네이버 데이터랩 트렌드)
    ├─ 빠름(+20%/년) / 안정(0~20%) / 정체(-/~0%)

[3] 경쟁 강도 (검색 결과 상위 10개 브랜드 수)
    ├─ 낮음(3개 이하) / 중(4~6개) / 높음(7개+)

[4] 진입 비용 (광고비·재고·MOQ)
    ├─ 낮음(1천만원) / 중(1~5천만원) / 높음(5천만원+)

[5] 우리 강점 적합성 (USP 활용 가능성)
    ├─ 강력(우리 USP가 segment에 직격) / 적합 / 부적합

→ 우선 진입 segment: [1] 중+ × [2] 안정+ × [3] 중-이하 × [4] 본인 자본 범위 × [5] 강력
```

### Step 3: USP 추출 — 3가지 차별 축

진입 segment 결정 후 **3가지 차별 축** 중 가장 강한 1-2개를 USP로 추출:

| 차별 축 | 본질 | 표현 패턴 | 한국 D2C 예시 |
|--------|------|----------|--------------|
| **무엇 (What)** | 제품 자체의 차이 | 성분·재료·기술 강조 | "100% 비건 인증", "닥터 처방" |
| **어떻게 (How)** | 제조·서비스 방식의 차이 | 공정·생산·운영 강조 | "당일 출고", "1:1 맞춤", "월구독" |
| **왜 (Why)** | 가치관·미션의 차이 | 정체성·신념 강조 | "동물 실험 0건", "한국 농가 직거래", "탄소 중립" |

> **USP 검증 질문**: "이 USP를 가진 경쟁사가 5개 이내인가?" 6개+ = USP 아님 (시장 표준 기능). 재추출 필요.

### Step 4: USP → 상품명·메시지·랜딩 일관성

USP가 명확해야 다운스트림 모든 스킬이 일관됨:

```
USP "100% 비건 + 한국 농가 직거래 + 30일 환불"
  ↓
[commerce-product-naming] "[비건 인증] 한국 농가 직거래 비건 세럼 [30일 환불]"
  ↓
[commerce-channel-message] 검색광고 "비건 세럼 한국 농가 직거래"
                             CRM "지연 회원님, 동물 실험 0건 + 30일 환불"
  ↓
[detail-page-copy] Hero "비건 인증 + 한국 농가" / Risk "30일 이유 불문 환불"
  ↓
[campaign-planner] AARRR Activation 메시지: 첫 구매 + 30일 환불 보장 강조
```
