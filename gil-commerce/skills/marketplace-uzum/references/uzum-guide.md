# Uzum Market 운영 가이드

## 개요

Uzum Market — UZ 최대 이커머스 (Uzum 그룹). 월 활성 사용자 5백만+ (2025).

```
[유형]
- Uzum Market (메인 마켓플레이스)
- Uzum Bank (결제·할부)
- Uzum Delivery (자체 배송)
- Uzum Tezkor (Q-commerce 빠른 배송)
```

## 입점 (한국 셀러)

```
[준비]
- UZ 법인 또는 한국 본사 + UZ 대리인
- UZ 세무 등록 (INN)
- UZ 은행 계좌 (Uzum Bank 또는 시중 은행)
- 상품 인증 (해당 시 — Halal·식약처·KC)

[등록 절차]
1. seller.uzum.uz 가입
2. KYC (법인·대리인 서류)
3. 카테고리·상품 등록
4. 가격 (UZS)·재고 입력
5. 배송 옵션 (Uzum Delivery 또는 자체)
6. 결제 연동 (Uzum Bank·시중 은행)
7. 운영 시작
```
