# 앱 푸시 메시지 기획 (Commerce Push Planner)

## 개요

한국 앱 셀러·서비스가 앱 푸시 메시지를 발송할 때 무작정 보내지 않고 **4원칙 + 3요소 + 카피 변형 3안**으로 기획하는 전담 스킬입니다.

**책임 한 줄**: 발송 목적·비즈니스 타입·사용자 세그먼트·발송 시점 입력 → 4원칙 매칭 + 3요소 자가 점검 + 30+ 한국 브랜드 레퍼런스 매핑 + 카피 변형 3안 + 클릭률 예측 가이드.

**vs commerce-channel-message**: 본 스킬은 **앱 푸시 전용**. 카톡/SMS/이메일은 페어 스킬 담당. 앱 푸시 특유의 잠금화면·요약 표시·게이미피케이션 톤앤매너를 깊이 다룸.

## 푸시 메시지 4원칙

모든 푸시 발송 전 4가지 질문에 답해야 합니다:

| 원칙 | 질문 | 점검 기준 |
|------|------|----------|
| **WHY** (왜 보내는가) | 발송 목적은 4분류 중 어디? | 서비스 노출 / 할인·이벤트 / 인지도 / 리텐션 |
| **WHEN** (언제 보내는가) | 사용자 행동 또는 시간 트리거? | 행동 후 3분/3시간/3일 / 데일리 골든타임 / 시즌·이슈 |
| **WHO** (누구에게 보내는가) | 세그먼트가 정의되어 있는가? | 전체 회원 / 코호트 / 1:1 개인화 |
| **HOW** (어떻게 보내는가) | 톤앤매너는 명확한가? | 숫자·게이미피케이션·브랜딩·이모지 사용 정책 |

> **4원칙 미충족 시 발송 금지**: 4원칙 중 하나라도 답하지 못하면 발송 차단. 무차별 푸시는 옵트아웃 비율(앱 알림 OFF)을 급격히 높여 장기 ROI 손실.

## 3요소 자가 점검 (Timely · Personal · Actionable)

발송 직전 다음 3요소가 모두 충족됐는지 점검:

### Timely (적시성)
- 사용자가 메시지를 받았을 때 **즉각 행동 가능한** 상황인가?
- 출근 시간(7-9시) 할인 푸시 → 클릭은 가능하나 구매까지는 잠시 미뤄짐
- 점심 시간(12-13시) 식당 추천 → 즉각 행동 적합
- 평일 22-24시 콘텐츠 추천 → 자기 전 소비 적합

### Personal (개인화)
- 받는 사람의 **세그먼트·이름·구매 이력**이 반영됐는가?
- 일반: "오늘만 20% 할인" (전체 회원)
- 세그먼트: "비건 카테고리 관심 회원님께 — 비건 신상 30%" (관심 세그먼트)
- 1:1: "지연 회원님, 지난번 구매하신 세럼 재구매 시기예요" (이름 + 구매 이력)

### Actionable (행동 유도)
- 메시지를 받고 **명확한 다음 액션**이 있는가?
- ❌ "신상품 입고" → 무엇을? 어떻게?
- ✅ "지금 탭 → 신상 30% 한정 진입 (10:00까지)" → 탭·시간·혜택 명확

> **3요소 모두 충족 시 클릭률 2-5배 상승**. 무차별 푸시 평균 1.5% vs 3요소 푸시 평균 4-8%.

## 발송 목적 4분류 매트릭스

| 목적 | 핵심 KPI | 발송 빈도 | 한국 베스트 사례 |
|------|---------|---------|-----------------|
| **① 서비스 노출** | DAU·앱 진입율 | 주 2-3회 | 배민 "오늘 점심 뭐 드세요?" / 카카오헤어샵 "이번 주 인기 헤어샵 TOP 5" |
| **② 할인·이벤트** | 즉각 매출·CTR | 캠페인 단위 | 쿠팡 "로켓와우 멤버 추가 10%" / 에이블리 "오늘만 20% 할인" / 지그재그 "퇴근 후 30분 한정" |
| **③ 인지도** | 브랜드 호감도·검색량 | 월 1-2회 | 토스 "송금하면 1억 추첨" / 오늘의집 "이번 주 인기 인테리어" / 클래스101 "신규 강의 50% 오픈" |
| **④ 리텐션** | 재방문·재구매·휴면 복귀 | 사용자 행동 기반 | 인프런 "마지막 학습 14일 전 — 강의 이어 듣기" / 야놀자 "찜한 숙소 5만원 떨어졌어요" / 퍼블리 "지난주 읽다 만 글 이어보기" |

## 비즈니스 타입 3분류 톤앤매너

| 타입 | 톤앤매너 특성 | 한국 사례 |
|------|--------------|----------|
| **D2C 브랜드** (스킨케어·패션·식품) | 정체성·라이프스타일·감성 | 29CM·아리따움·콜린스·사운드짐 |
| **마켓플레이스** (가격·다양성·즉시 구매) | 숫자·게이미피케이션·CTA 강함 | 쿠팡·올웨이즈·당근마켓·요기요·쿠팡이츠 |
| **결제/구독** (안정·신뢰·습관 형성) | 절제·명확·기능 강조 | 토스·리디·넷플릭스·피파온라인·듀오링고 |

## 카피 변형 3안 — 자동 생성 원칙

같은 발송 목적에서도 메시지 표현을 **3안으로 변형**하여 A/B 테스트:

### 변형 축 1: 오늘만 vs 매일

| 변형 | 패턴 | 클릭률 효과 |
|------|------|-----------|
| 오늘만 (긴급성) | "오늘만 20% — 자정 마감" | 즉각 클릭 ↑ but 신뢰 ↓ (반복 시) |
| 매일 (안심성) | "매일 갱신되는 추천 상품" | 클릭률 균일 + 옵트아웃률 ↓ |

### 변형 축 2: 누구나 vs 너에게만

| 변형 | 패턴 | 클릭률 효과 |
|------|------|-----------|
| 누구나 (일반) | "전 회원 20% 할인" | 베이스라인 1.5% |
| 너에게만 (개인화) | "지연 회원님 한정 — 어제 본 그 세럼 20%" | 베이스라인 4-8% (개인화 효과) |

### 변형 축 3: 숫자 vs 게이미피케이션 vs 브랜딩

| 변형 | 패턴 | 한국 사례 |
|------|------|----------|
| **숫자** | "오늘만 30% 할인 / 3시간 남음" | 쿠팡·올웨이즈 (마켓 스타일) |
| **게이미피케이션** | "🎮 출석 5일째 — 다음 보상은 ★★★" | 토스·리멤버·듀오링고 |
| **브랜딩** | "오늘 저녁, 우리 잔에는 어떤 향이 어울릴까요?" | 29CM·아리따움·사운드짐 (감성 톤) |

## 워크플로우

### 입력 슬롯

| 항목 | 필수 | 예시 |
|------|------|------|
| 발송 목적 | 필수 | `서비스 노출` / `할인·이벤트` / `인지도` / `리텐션` |
| 비즈니스 타입 | 필수 | `D2C` / `마켓` / `결제·구독` |
| 사용자 세그먼트 | 필수 | `전체 회원` / `최근 7일 미접속` / `장바구니 이탈 24h` / `구매 후 60일` |
| 발송 예정 시점 | 필수 | `2026-05-18 12:30` (점심 시간 골든타임) |
| 핵심 혜택·콘텐츠 | 필수 | `비건 세럼 20% 할인` / `신규 인테리어 가이드 공개` |
| 톤앤매너 선호 | 권장 | `숫자` / `게이미피케이션` / `브랜딩` |

### 점검 순서

```
[Step 1] 4원칙 매칭
   - WHY: 4분류 중 하나로 분류
   - WHEN: 행동 트리거 또는 시간 골든타임 명시
   - WHO: 세그먼트 정의 (전체 / 코호트 / 1:1)
   - HOW: 톤앤매너 선택

[Step 2] 3요소 자가 점검
   - Timely: 발송 시점이 사용자 즉각 행동 가능한가?
   - Personal: 세그먼트·이름·이력 반영?
   - Actionable: 다음 액션 명확?

[Step 3] 비즈니스 타입 톤앤매너 매핑
   - D2C → 정체성·감성
   - 마켓 → 숫자·CTA
   - 결제·구독 → 절제·기능

[Step 4] 카피 변형 3안 자동 생성
   - 변형 1: 오늘만 vs 매일
   - 변형 2: 누구나 vs 너에게만
   - 변형 3: 숫자·게이미피케이션·브랜딩

[Step 5] 클릭률 예측
   - 기본: 1.5%
   - 3요소 충족: +2.5~4.5%p
   - 개인화 추가: +1~3%p
   - 게이미피케이션 추가: +0.5~1.5%p
   - 야간 발송 (정통망법 동의 필수): -0.5~1%p (옵트아웃 증가)
```

## 한국 30+ 브랜드 푸시 레퍼런스

### D2C 브랜드 (정체성·감성)
- 29CM, 아리따움, 콜린스, 사운드짐
- 클래스101, 인프런, 퍼블리, 브런치
- 오프린트미, 리디

### 마켓플레이스 (숫자·CTA)
- 쿠팡, 쿠팡이츠, 요기요, 배민
- 올웨이즈, 당근마켓, 지그재그, 에이블리, 브랜디
- 야놀자

### 결제/구독 (절제·기능)
- 토스, 카카오페이
- 넷플릭스, 피파온라인
- 듀오링고, 리멤버

### 콘텐츠·기타 (게이미피케이션)
- 라운즈, 네이버웹툰, 카카오웹툰
- 인스타그램, 카카오톡

## 합격 기준

- 4원칙 4질문 모두 답변 명시
- 3요소 자가 점검 모두 PASS
- 카피 변형 3안 (각 변형 축 1개씩) 생성
- 한국 브랜드 베스트 사례 매핑
- 클릭률 예측 + 근거 명시
- ai-slop-reviewer 검수 흔적

```
"/commerce-push-planner — 할인 푸시, 마켓, 전 회원, 점심 12:30, 오늘만 30%"
→ 마켓 톤 (숫자 강함) + Actionable CTA + 클릭률 3~5%

"/commerce-push-planner — 인지도 푸시, 결제, 신규 가입 회원, 출근 8시, 송금 1억 추첨"
→ 토스 스타일 + 게이미피케이션 + 절제된 톤
```

## 관련 스킬

체이닝 권장: **commerce-push-planner** (카피 작성) → `commerce-marketing-compliance-kr` (정통망법 게이트) → 실제 발송

- `commerce-channel-message` — 카톡·SMS·이메일 카피 (앱 푸시는 본 스킬)
- `commerce-marketing-compliance-kr` — 발송 전 법규 게이트 (필수 체이닝)
- `commerce-jtbd-persona` — 세그먼트 정의용 페르소나

## 이 스킬을 사용하지 말아야 할 때

- **카톡·SMS·이메일 카피**: `commerce-channel-message` 사용
- **광고 캠페인 전체 기획**: `campaign-planner` 사용
- **이메일 시퀀스 자동화**: `email-sequence` 사용
- **인앱 메시지·인앱 배너**: 별도 인앱 콘텐츠 스킬 (앱 푸시는 잠금화면 전용)
