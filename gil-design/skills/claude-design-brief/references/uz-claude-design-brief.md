# Claude Design 브리프 — 한·UZ 듀얼

## UZ 한인사회 디자인 브리프 개관
- 우즈베키스탄 한인 인구 약 17만명 (CIS 최대 한인사회) — 타슈켄트·사마르칸트·페르가나 집중
- 트릴링구얼 환경: 한국어 + 러시아어(공용·디지털 1순위) + 우즈벡어(국가어, 라틴·키릴 병기)
- 모바일 우선: Android 점유율 ~92%, 결제 Uzum Bank·Click·Payme, 채널 Telegram 1위

## 6요소 브리프 작성 시 한·UZ 듀얼 체크
| 요소 | 한국 기준 | UZ 한인 추가 고려 |
|---|---|---|
| AUDIENCE | 한국 거주 사용자 | 고려인 1.5~4세대 + UZ 거주 한국인 + 친한파 UZ인 |
| TONE | 한국어 형용사 3~5개 | 러시아어 톤 동시 정의 (러시아어가 더 김 — 레이아웃 여유) |
| PAGES | 한국어 단일 | 트릴링구얼 토글(KO/RU/UZ) 페이지 구조 명시 |
| REFERENCE | 한국 사이트 | UZ 현지(Uzum·Click) + CIS 한인 채널 참고 |
| CONSTRAINTS | 데스크톱·반응형 | 모바일 우선 + 키릴·라틴 글리프 폰트 지원 명시 |
