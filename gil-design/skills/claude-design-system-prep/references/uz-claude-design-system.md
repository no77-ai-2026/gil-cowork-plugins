# 디자인 시스템 자산 합성 — 한·UZ 듀얼

## UZ 한인 브랜드 디자인 시스템 개관
- UZ 한인·CIS 시장 브랜드는 트릴링구얼(한·러·우즈벡) UI를 전제로 디자인 시스템을 설계
- 우즈벡어는 라틴 표기가 표준이나 키릴 병기 사례 다수 — 폰트 글리프 커버리지 필수
- 러시아어 텍스트는 한국어 대비 평균 1.3~1.6배 길어 컴포넌트 간격·줄바꿈 토큰에 여유 필요

## DESIGN.md 합성 시 한·UZ 듀얼 체크
| 섹션 | 한국 기준 | UZ 한인 추가 고려 |

| Typography | 한글 폰트 | 키릴·라틴 동시 지원 폰트 (Noto Sans·Inter 키릴 서브셋) |
