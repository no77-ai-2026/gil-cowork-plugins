# 재무제표 작성 (Financial Statements)

## 개요

K-IFRS(한국채택국제회계기준)에 따른 재무제표 작성을 지원한다. 2026년 조기 적용 가능한 K-IFRS 제1118호(재무제표의 표시와 공시) 변경사항을 반영하며, 재무상태표·손익계산서·현금흐름표·자본변동표·주석 등 완전한 재무제표 세트를 지원한다.

**면책 고지: 이 스킬은 AI 보조 분석이며 공인 회계사의 전문 자문을 대체하지 않습니다. 재무제표는 반드시 공인회계사의 검토 또는 감사를 받으시기 바랍니다.**

## 트리거 키워드

재무제표, K-IFRS, 손익계산서, 재무상태표, 현금흐름표, 재무 보고서, 회계 결산, 분기 보고서, 연간 보고서, 감사 보고서, 영업이익, EBITDA

## 워크플로우

### 1단계: 적용 기준 확인

**적용 회계기준 선택:**
- **K-IFRS**: 상장법인, 금융기관, 일정 규모 이상 법인 (의무)
- **일반기업회계기준(K-GAAP)**: 비상장 중소기업

**보고 기간 확인** (연간/반기/분기)

**K-IFRS 제1118호 조기 적용 여부 결정** (2026년 조기 적용 가능, 의무 시행 2027년)

### 2단계: 2026-2027년 K-IFRS 주요 변경사항 적용

**K-IFRS 제1118호 '재무제표의 표시와 공시' (신규 제정)**
- 기존 K-IFRS 제1001호를 대체
- 의무 적용: **2027년 1월 1일** 이후 개시 회계연도
- 조기 적용: **2026년 1월 1일** 이후 개시 회계연도 허용

**손익계산서 5개 범주 분류 (K-IFRS 제1118호)**
| 범주 | 포함 항목 |
|------|-----------|
| **영업** | 주요 사업 활동에서 발생한 수익·비용 (잔여 개념) |
| **투자** | 투자자산 관련 수익·비용 (지분법, 배당금 등) |
| **재무** | 자금 조달 관련 수익·비용 (이자비용, 환차손익 등) |
| **법인세** | 법인세 비용 |
| **중단영업** | 중단영업 손익 |

**MPM(경영진 성과측정치) 공시 의무화**
- 기업이 외부에 공시하는 자체 성과지표 (예: 조정 영업이익, Adjusted EBITDA)
- 산출근거와 K-IFRS 중간합계와의 조정내역(reconciliation) 주석 공시 필수
- 단일 주석(single note)으로 집약

**K-IFRS 제1117호 개정 (보험계약 - 2026.1.1. 시행)**
- 무·저해지 보험상품 추정기법 차이 주석 공시 강화

### 3단계: 재무제표 세트 작성

**재무상태표 (Statement of Financial Position)**
```
자산
  유동자산: 현금및현금성자산, 매출채권, 재고자산, 기타유동자산
  비유동자산: 유형자산, 무형자산, 투자자산, 기타비유동자산
  자산 합계

부채
  유동부채: 매입채무, 단기차입금, 기타유동부채
  비유동부채: 장기차입금, 퇴직급여충당부채, 기타비유동부채
  부채 합계

자본
  납입자본, 이익잉여금, 기타포괄손익누계액
  자본 합계
```

**손익계산서 (Statement of Profit or Loss) — K-IFRS 1118호 반영**
```
영업 범주:
  수익 (매출액)
  매출원가
  매출총이익
  판매비와관리비
  영업이익 (EBIT)

투자 범주:
  지분법 이익(손실)
  배당금수익
  투자자산 처분손익

재무 범주:
  이자비용
  이자수익
  외환손익

법인세비용차감전순이익
법인세비용
당기순이익
```

**현금흐름표 (Statement of Cash Flows)**
- 영업활동 현금흐름 (간접법 권장)
- 투자활동 현금흐름
- 재무활동 현금흐름

### 4단계: 주석 공시

**필수 주석 항목:**
1. 유의적인 회계정책 요약
2. 중요한 추정 및 가정
3. 자본 관리 목적 및 정책
4. 금융위험 관리 (시장위험, 신용위험, 유동성위험)
5. MPM 공시 (K-IFRS 1118호 조기 적용 시)
6. 특수관계자 거래
7. 우발부채 및 약정사항

### 5단계: 재무비율 분석

```
수익성: 매출총이익률, 영업이익률, 순이익률, ROE, ROA
안정성: 부채비율, 유동비율, 당좌비율, 이자보상배율
활동성: 매출채권 회전율, 재고자산 회전율, 총자산 회전율
성장성: 매출 성장률, 영업이익 성장률, 자산 성장률
```

## 사용 예시

**예시 1**: "K-IFRS 기준 재무상태표 작성해줘"
→ 자산·부채·자본 항목 → 잔액 입력 → 재무상태표 생성

**예시 2**: "손익계산서 5개 범주로 분석해줘"
→ 기존 손익계산서 → 5개 범주 재분류 → K-IFRS 1118호 적용

**예시 3**: "2026년 K-IFRS 제1118호 조기 적용 검토해줘"
→ 현재 재무제표 → 5개 범주 분석 → MPM 공시 여부 → 조기 적용 권고

## 문제 해결

- **세율 변경 확인 불가**: 법인세율·이연법인세 계산은 최신 세법 개정 내용을 반영해야 합니다. 국세천(nts.go.kr)에서 현행 세율을 확인하거나 세무사에게 의뢰하세요.
- **과거 데이터 미제공**: 전기 재무제표 없이는 비교 표시가 불가능합니다. 전기 재무제표(최소 1개년)를 함께 제공해 주세요.
- **K-IFRS 기준 불확실**: 특수 거래(금융상품, 리스, 수익 인식)의 회계처리가 불분명한 경우 한국회계기준원(kasb.or.kr)의 유권해석을 참조하거나 공인회계사에게 질의하세요.
- **외부 감사 대상 여부 확인**: 자산 120억 원 이상 또는 부채 70억 원 이상이면 외부 감사 의무 대상입니다. AI가 작성한 재무제표를 공식 제출용으로 사용하려면 반드시 외부 감사인의 검토를 받으세요.

## 이 스킬을 사용하지 말아야 할 때

- **월말·급여 결산 업무**: 계정 마감, 급여 정산, 4대보험 계산은 `close-management` 스킬이 더 적합합니다.
- **예산 대비 실적 분석**: 분산 분석과 KPI 추적은 `variance-analysis` 스킬을 사용하세요.
- **세금 신고·납부**: 부가가치세·법인세 신고 절차는 `tax-helper` 스킬을 먼저 확인하고 세무사에게 의뢰하세요.
- **투자 판단용 실사(DD)**: 인수합병, 투자 결정을 위한 재무 실사는 공인회계사나 회계법인의 공식 실사가 필요합니다.
