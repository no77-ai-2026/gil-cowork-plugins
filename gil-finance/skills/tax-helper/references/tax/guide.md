# 세무/회계 도우미 (Tax Helper & Bookkeeping Automation)

프리랜서(3.3% 원천징수), 개인사업자, 소상공인의 세무 업무를 돕습니다.
세금 계산, 경비 정리, 신고 준비를 자동화하고 절세 전략을 제안합니다.

## Overview

> **주의**: 이 스킬은 세무 정보 제공 및 문서 정리 도구입니다.
> 실제 세무 신고는 반드시 홈택스(hometax.go.kr) 또는 세무사를 통해 진행하세요.
> Claude는 세무사가 아니며, 법적 세무 조언을 제공하지 않습니다.

## 프리랜서 세무 가이드 (3.3%)

### 기본 구조
```
수입금액 (총 수령액 ÷ 0.967)
- 필요경비 (경비율 적용 또는 장부 기장)
= 소득금액
- 소득공제 (인적공제, 연금, 보험 등)
= 과세표준
× 세율
= 산출세액
- 세액공제/감면
- 기납부세액 (3.3% 원천징수분)
= 납부/환급세액
```

### 경비율 구분
| 구분 | 기준 | 적용 |
|------|------|------|
| 단순경비율 | 직전연도 수입 2,400만원 미만 | 높은 경비율, 장부 불필요 |
| 기준경비율 | 직전연도 수입 2,400만원 이상 | 낮은 경비율, 주요경비 증빙 필요 |

## 개인사업자 세무 가이드

### 부가세 신고 준비 체크리스트
```
□ 매출 자료 정리
  - 세금계산서 발행 내역 (전자세금계산서)
  - 카드 매출 (POS 데이터)
  - 현금영수증 발행 내역

□ 매입 자료 정리
  - 매입 세금계산서
  - 사업용 신용카드 사용 내역 (홈택스 자동 수집)
  - 현금영수증 (지출증빙)

□ 공제/불공제 구분
  - 공제 가능: 사업 직접 관련 비용
  - 불공제: 접대비(1만원 초과 간이영수증), 비영업용 차량 등

□ 서류 준비
  - 사업자등록증 사본
  - 통장거래내역
  - 임대차계약서 (해당 시)
```

### 간편장부 vs 복식부기
| 구분 | 간편장부 | 복식부기 |
|------|---------|---------|
| 대상 | 수입 7,500만원 미만 (신규 포함) | 수입 7,500만원 이상 |
| 난이도 | 쉬움 (가계부 수준) | 높음 (차변/대변) |
| 절세 혜택 | 기본 | 기장세액공제 20% (최대 100만원) |
| 추천 | 소규모 사업자 | 경비가 많은 사업자 |

## 절세 핵심 전략

### 프리랜서 절세 3가지
1. **적격증빙 챙기기**: 사업 관련 모든 지출에 대해 세금계산서, 카드 영수증, 현금영수증 확보
2. **사업용 카드 등록**: 홈택스에 사업용 신용카드 등록 → 부가세 매입공제 자동 반영
3. **기장 세액공제**: 간편장부 대상자도 복식부기로 신고하면 기장세액 20% 감면 (최대 100만원)

### 사업자 등록 검토 기준
- 연 수입 2,400만원 이상이면 사업자등록 검토
- 시설/장비 투입이 큰 업종은 매입세액 공제를 위해 사업자등록 유리
- 창업 감면 제도: 나이/지역 요건 충족 시 5년간 소득세 최대 100% 감면

## 제공 기능

### 1. 세금 시뮬레이션
사용자의 수입, 경비 정보를 입력받아 예상 세금을 계산합니다.

**입력:**
- 연 총수입
- 항목별 경비
  - 사무실 임차료
  - 통신료
  - 광고비
  - 교육비
  - 기타 경비
- 소득공제 (인적공제, 연금, 보험)

**출력:**
```
[2024년도 예상 세금]

총 수입: 50,000,000원
- 필요경비 (기준경비율): -12,000,000원
= 소득금액: 38,000,000원
- 소득공제: -3,000,000원
= 과세표준: 35,000,000원
× 세율 15%: 5,250,000원
- 누진공제: -126,000원
= 산출세액: 5,124,000원
- 기납부세액(3.3%): -1,650,000원
= 납부세액: 3,474,000원

또는 환급: OOO원
```

### 2. 매출/매입 정리 엑셀 생성

**매출 정리표 구조:**
| 날짜 | 거래처 | 품목 | 공급가액 | 부가세 | 합계 | 증빙유형 |
|------|--------|------|---------|--------|------|---------|

**매입 정리표 구조:**
| 날짜 | 거래처 | 품목 | 공급가액 | 부가세 | 합계 | 공제여부 | 증빙유형 |
|------|--------|------|---------|--------|------|---------|---------|

**경비 정리표 구조:**
| 날짜 | 항목 | 내용 | 금액 | 증빙유형 | 비고 |
|------|------|------|------|---------|------|

### 3. 신고 체크리스트 생성
신고 유형별 필요 서류와 절차를 체크리스트로 제공합니다.

**항목:**
- [ ] 사업자등록증 사본
- [ ] 통장거래내역 (전년도 및 신고연도)
- [ ] 세금계산서 (발행분, 수취분)
- [ ] 현금영수증 발행 내역
- [ ] 카드 매출 (POS, 신용카드 명세)
- [ ] 임대차계약서 (해당 시)
- [ ] 인감증명서
- [ ] 대리인 위임장 (세무사 대리 신고 시)

### 4. 절세 분석 리포트
수입/경비 데이터를 분석하여 절세 가능한 항목을 찾아줍니다.

**예시:**
```
[절세 기회 분석]

1. 📊 경비율 최적화
   현재: 기준경비율 30% 적용
   → 장부 기장 시 복식부기 세액공제 20% (최대 100만원)
   절세 효과: 약 50~100만원

2. 💳 사업용 카드 미등록
   → 홈택스에 신용카드 등록하면 자동 매입공제
   절세 효과: 약 200~300만원 (카드 사용액 기준)

3. 📚 교육비/도서비 미신고
   → 직무 관련 교육비는 비용 처리 가능
   잠재 절세: 약 OOO원

4. 🏠 홈오피스 비용
   → 방 면적 비율로 임차료, 공과금 배분 가능
   잠재 절세: 약 OOO원
```

## 트러블슈팅

| 증상 | 원인 | 해결 |
|------|------|------|
| 계산 결과 다름 | 세율 변경, 공제 누락 | 홈택스 공식 계산기와 비교 |
| 경비 범위 불확실 | 개인/사업 구분 애매 | 세무사 상담 (판단 필요) |
| 신고 기한 놓침 | 일정 확인 부족 | 홈택스 달력 확인, 알림 설정 |
| 증빙 부족 | 영수증 미보관 | 향후 카드/현금영수증 반드시 수집 |
| 역산입세액공제 불가 | 공급자가 세금계산서 미발행 | 공급자에게 재요청 |
