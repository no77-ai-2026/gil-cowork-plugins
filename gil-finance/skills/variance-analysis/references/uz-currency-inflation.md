# UZ 환율·인플레이션 영향 분리 분석 가이드

> gil-finance v0.1.0 | UZS 평가절하 + 인플레이션 영향 분리

## 개요

UZ 자회사 운영 시 분산 분석에서 **명목 변동 / 환율 영향 / 인플레이션 영향**을 분리하지 않으면 실제 사업 성과 판단이 왜곡됩니다. 이 가이드는 3가지 영향을 분리하는 표준 방법을 제시합니다.

---

## 1. 3가지 영향 분리 원칙

### 명목 변동 (Nominal Change)
- UZS 기준 실제 변동
- 예: UZS 매출이 전년 대비 +20%

### 인플레이션 영향 (Inflation Effect)
- UZ 일반 물가 상승률 (CPI)
- stat.uz 또는 cbu.uz 발표
- 2026년 추정: 8~12%/년

### 환율 영향 (FX Effect)
- UZS↔KRW 또는 UZS↔USD 환율 변동
- cbu.uz 환율 시계열
- UZS 평가절하: 연 8~12%/년

### 실질 변동 (Real Change)
```
명목 변동 - 인플레이션 - 환율 = 실질 변동
```

---

## 2. 분리 분석 표준 방법

### Step 1: 명목 변동 산출 (UZS 기준)

```
명목 매출 변동 = (당기 UZS 매출 - 전기 UZS 매출) / 전기 UZS 매출
```

### Step 2: 인플레이션 조정

```
실질 UZS 매출 = 당기 UZS 매출 / (1 + 인플레이션율)

실질 변동 (UZS) = (실질 UZS 매출 - 전기 UZS 매출) / 전기 UZS 매출
```

### Step 3: 환율 환산 (KRW 보고 시)

```
KRW 매출 (당기) = 당기 UZS 매출 × 평균환율 (UZS/KRW)
KRW 매출 (전기) = 전기 UZS 매출 × 전기 평균환율

KRW 변동 = 명목 UZS 변동 + 환율 영향
```

### Step 4: 환율 영향 분리

```
순수 환율 영향 = 전기 UZS 매출 × (당기 환율 - 전기 환율) / 전기 환율
순수 환율 효과 (KRW) = 전기 KRW 매출 × (당기 환율 - 전기 환율) / 전기 환율
```

### Step 5: 통합 분리표

```
| 항목 | 변동률 | 비고 |
|------|--------|------|
| 명목 UZS 변동 | +20% | UZS 기준 실제 |
| 인플레이션 | -10% | UZS 구매력 감소 |
| 실질 UZS 변동 | +10% | 인플레이션 조정 후 |
| UZS/KRW 환율 변동 | -8% | UZS 평가절하 |
| KRW 환산 변동 | +12% | 한국 본사 인식 |
| 실질 KRW 변동 | +2% | 인플레이션·환율 모두 조정 |
```

## 3. 실전 분석 예시

### 케이스: UZ 자회사 IT 서비스

**기초 데이터** (전기 vs 당기, 1년 차이):

| 항목 | 전기 | 당기 |
|------|------|------|
| UZS 매출 | 1,200,000,000 UZS | 1,440,000,000 UZS |
| UZS 인플레이션 (연) | - | 10% |
| 평균 환율 (UZS/KRW) | 9.0 | 8.3 |
| 평균 환율 (UZS/USD) | 12,200 | 12,750 |

**분석**:

```
1. 명목 UZS 변동
   = (1,440 - 1,200) / 1,200 = +20.0%

2. 인플레이션 조정 후 실질 UZS 매출
   = 1,440,000,000 / 1.10 = 1,309,090,909 UZS
   실질 UZS 변동 = (1,309 - 1,200) / 1,200 = +9.1%

3. KRW 환산 매출
   전기: 1,200,000,000 / 9.0 = 133,333,333 KRW
   당기: 1,440,000,000 / 8.3 = 173,493,976 KRW
   KRW 명목 변동 = +30.1%

4. 순수 환율 영향 (KRW 단독)
   환율 변동: (8.3 - 9.0) / 9.0 = -7.8% (UZS 평가절하)
   순수 환율 효과: 1,200,000,000 × ((1/8.3) - (1/9.0)) ≈ 11,244,980 KRW
   = 전기 매출의 +8.4% 효과 (UZS는 약화됐지만 KRW로 환산하면 +)

   잠깐, 위 계산 오류 수정:
   UZS/KRW = 9.0 → 8.3 (UZS 1단위당 받는 KRW 변동, 환율 -)
   1 UZS = 1/9.0 KRW = 0.1111 KRW (전기)
   1 UZS = 1/8.3 KRW = 0.1205 KRW (당기, KRW 평가절하 의미)
   →  정확히는 KRW가 약해진 것 (UZS는 강해짐)

5. UZS/USD 비교
   12,200 → 12,750 = UZS가 USD 대비 +4.5% 평가절하
   환율 표기 방향 명확화 필요

6. 실질 KRW 변동
   KRW 명목 +30.1%
   - 인플레이션 -10% (UZ 측)
   - 한국 인플레이션 -3% (참고)
   = 실질 KRW 변동 +17.1%
```

(주: 환율 표기 방향과 KRW vs USD 동향에 따라 결과가 달라지므로, 실제 분석 시 cbu.uz 일별 환율 시계열로 정확히 계산)

## 4. KPI별 적용

### 매출
- 명목 / 실질 / KRW 환산 / 실질 KRW 4가지 동시 표기
- 사업 성과 판정은 **실질 UZS 또는 실질 KRW** 기준

### 인건비
- UZ 인건비는 매년 인플레이션 +10% 압력
- 실질 인건비 변동 = 명목 - 인플레이션
- 실질 인건비가 +면 진짜 인건비 부담 증가

### 매출원가 (수입 원자재)
- 외화 표시 (USD) → UZS 환산
- 환율 변동에 따라 UZS 비용 변동
- KRW 환산 시 추가 변동

### 영업이익
- UZS 영업이익 → 인플레이션 조정 → 실질 UZS 영업이익
- KRW 환산 → 환율 영향 추가 분리

## 5. 환차 헤지 전략 (참고)

### 자연 헤지 (Natural Hedge)
- USD 매출 + USD 비용 매칭 (수출 IT 기업 유리)
- UZS 매출 + UZS 비용 매칭 (현지 비즈니스)

### 금융 헤지
- USD 선물환 계약 (cbu.uz·시중은행)
- USD 옵션
- 비용 큼 → 일반적으로 자연 헤지 우선

### IT Park 거주자 특례
- 수출 매출의 50%까지 자유 송금 (USD)
- 자연 헤지 용이

## 8. 주의사항

- **환율 시점 차이**: 평균환율 vs 결산환율 vs 거래시점 환율 명확히 구분
- **인플레이션 측정**: CPI vs PPI vs GDP 디플레이터 — 분석 목적별 선택
- **UZS 변동성**: 정책 변동·외환 통제 변경으로 환율이 단기간 큰 폭 변동 가능
- **회계기준 연동**: NAS는 환차손익을 손익계산서, K-IFRS는 기타포괄손익 (조건부) — 비교 시 일관성
