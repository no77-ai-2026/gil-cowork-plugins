# 전자세금계산서 EDI 가이드

## 개요

전자세금계산서는 국세청 홈택스를 통해 발행·수취·수정·취소를 처리하는 디지털 세금계산서입니다.
2024년부터 모든 법인사업자와 직전연도 공급가액 1억원 이상 개인사업자에게 의무화되었습니다.

## 발행 절차

### 1. 홈택스 직접 발행
1. 홈택스 로그인 → 전자세금계산서 → 발급
2. 공급자/공급받는자 사업자번호 입력
3. 품목, 수량, 단가, 공급가액, 세액 입력
4. 발급 → 국세청 전송 (자동)

### 2. ERP/회계 프로그램 연동 (EDI)
1. ERP에서 매출 전표 생성
2. 전자세금계산서 자동 생성 (API 연동)
3. 국세청 ASP 사업자를 통해 전송
4. 전송 결과 확인 (성공/실패)

### 주요 ASP 사업자
| 사업자 | 용도 |
|--------|------|
| 비즈니스온 | 중소기업 |
| 더존 | ERP 연동 |
| 한국정보인증 | 공인인증 기반 |
| 금융결제원 | 금융기관 연동 |

## 수정 세금계산서

수정이 필요한 경우 원본을 취소하고 재발행합니다.

| 사유 | 수정 유형 | 기한 |
|------|----------|------|
| 기재사항 착오 | 기재사항 착오 정정 | 확정신고 기한 |
| 공급가액 변동 | 공급가액 변동 | 변동 사유 발생일 |
| 환입 (반품) | 환입 | 환입일 |
| 계약 해제 | 계약의 해제 | 해제일 |
| 착오에 의한 이중 발급 | 착오에 의한 이중 발급 | 확정신고 기한 |

## 가산세 기준

| 위반 유형 | 가산세 |
|----------|--------|
| 미발급 | 공급가액의 2% |
| 지연 발급 (1개월 이내) | 공급가액의 1% |
| 전자 외 발급 (종이) | 공급가액의 1% |
| 미전송 (국세청) | 공급가액의 0.5% |

## 실무 체크리스트

- [ ] 매출 발생 시 익월 10일까지 발행 (지연 가산세 방지)
- [ ] 공급받는자 사업자번호 사전 확인 (휴폐업 조회)
- [ ] 수정 세금계산서는 원본 번호 참조하여 발행
- [ ] 월말 미전송 건 일괄 확인 (홈택스 → 전송 현황 조회)
- [ ] 부가세 신고 전 매출/매입 세금계산서 대사 완료
