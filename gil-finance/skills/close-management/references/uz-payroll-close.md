# 우즈베키스탄 급여·결산 가이드

> gil-finance v0.1.0 | UZ 직원 급여·NDFL·사회세·월간 마감

## 개요

UZ에서 직원 급여 정산, NDFL 원천징수, 사회세 신고, 월간·연간 결산을 수행하는 실무 가이드입니다.

---

## 1. UZ 급여 구조 핵심

### 직원 받는 순급여 계산

```
총 급여 (Gross Salary)
  - NDFL 12% (개인소득세, 직원 부담)
  - 사회세 12% (직원 부담, 사회보장기금)
  = 순 급여 (Net Salary, 직원 실수령)
```

### 사업주 부담 (총 인건비)

```
총 급여 (Gross Salary)
  + 사회세 25% (사업주 부담, IT Park 거주자 12.5%)
  = 총 인건비 (Total Labor Cost, 사업주 지출)
```

### 예시: 월 급여 10,000,000 UZS (Gross)

**직원 측**:
- Gross: 10,000,000 UZS
- NDFL (12%): -1,200,000
- 사회세 (12%): -1,200,000
- **Net (실수령)**: 7,600,000 UZS

**사업주 측**:
- Gross: 10,000,000
- 사회세 (25%): +2,500,000
- **총 인건비**: 12,500,000 UZS (IT Park 거주자: 11,250,000)

**KRW 환산** (1 USD = 12,750 UZS, 1 USD = 1,380 KRW 기준):
- Gross 10M UZS ≈ $784 ≈ KRW 1,082,000
- Net 7.6M UZS ≈ $596 ≈ KRW 822,000
- 총 인건비 12.5M UZS ≈ $980 ≈ KRW 1,353,000

## 2. 최저임금·최저생활비

### 최저임금 (Минимальная заработная плата)

2026년 기준 (대통령령으로 매년 조정):
- 약 1,155,000 UZS/월 (약 $90~$95)
- lex.uz 최신 대통령령 확인 필수

### 최저생활비 (Минимальный потребительский бюджет)

- 약 870,000 UZS/월 (참고용 통계 지표)

### 외국인 직원 최저 급여 기준
- 외국인 노동허가서 발급 시: 최저임금의 5배 이상 권장 (변동 가능)
- 일반적 외국인 IT 인력: 월 $2,000~$5,000 수준

## 3. NDFL (개인소득세) 원천징수

### 단일 세율: 12%

2019년 누진 폐지 후 단일.

### 비과세 항목
- 사회보장 급여 (정부 지급)
- 출장비 (영수증 증빙)
- 일부 식사·교통 지원 (한도)
- 자녀 1인당 4 BHM (Базовая величина) 공제 (자녀 수당)

### IT Park 거주자 직원 특례
- 일부 구간 NDFL 감면 (정책 변동, it-park.uz 확인)
- 일반적으로 IT Park 거주자 직원도 12% 적용 (특례 제한적)

### 신고 일정
- 매월 15일까지 (전월 분)
- soliq.uz 전자 신고

## 4. 사회세 (Social Tax)

### 요율
- **사업주 부담**: 25%
- **직원 부담**: 12% (NDFL과 별도 항목)

### IT Park 거주자 사업주 특례
- 사회세 50% 감면 → **12.5% 부담**

### 적립 항목
- Pension Fund (연금)
- 의무 의료보험
- 사회보험 (실업·산재)

### 신고
- 매월 25일까지
- soliq.uz 통합 신고 (NDS와 함께)

## 5. 월간 급여 정산 워크플로우

### 1주차 (1~7일): 전월 데이터 정리
- [ ] 전월 출근부 확정 (시간외근무·휴가 포함)
- [ ] 성과급·보너스 결정 (해당 시)
- [ ] 식대·교통비·복리후생 정산
- [ ] 출장비 영수증 정리

### 2주차 (8~15일): 급여 계산
- [ ] 직원별 Gross Salary 확정
- [ ] NDFL 12% 계산
- [ ] 사회세 (직원 12%) 계산
- [ ] 자녀 수당 등 비과세 공제 적용
- [ ] Net Salary 산출
- [ ] 급여 명세서 (Расчётный лист) 작성

### 3주차 (15~25일): 신고·납부
- [ ] **15일까지**: NDFL 신고 (soliq.uz)
- [ ] **25일까지**: 사회세 신고 (soliq.uz)
- [ ] **25일까지**: NDS 신고 (해당 시)
- [ ] 급여 이체 (직원 계좌)
- [ ] 사회세 사업주 부담분 납부

### 4주차 (25~말일): 결산 마감
- [ ] 인건비 회계 처리 (NAS 분개)
- [ ] 미지급 급여 (해당 시)
- [ ] 인건비 분산 분석 (예산 vs 실적)

## 6. 1С 우즈벡 활용

### 급여 모듈 설정
- 1С:Зарплата и управление персоналом для Узбекистана
- NDFL/사회세 자동 계산
- 신고서 자동 생성 (soliq.uz 연동)

### 표준 거래
```
인건비 분개 (예시):
Дт 20 (생산비) / 26 (관리비) / 44 (판매비)
Кт 70 (급여 미지급) — 총 급여
Кт 68 (NDFL 미지급) — NDFL 12%
Кт 69 (사회세 미지급) — 사회세 25% + 12%

급여 지급:
Дт 70 (급여 미지급)
Кт 51 (은행 계좌)
```

## 7. 외국인 직원 급여 처리

### 한국인 임원·직원 (UZ 거주자)

**거주자 판정**: UZ 체류 183일/연 이상

**거주자**:
- NDFL 12% (전세계 소득)
- 사회세 정상 (12% 직원 + 25% 사업주)
- 한·우 DTA 적용 → 한국 측 종합소득세 외국납부세액공제

**비거주자**:
- NDFL 12~20% (UZ 원천 소득만)
- 사회세 일부 면제 가능 (조건부)

### 노동허가서 (Разрешение на привлечение и использование иностранной рабочей силы)
- 외국인 직원 채용 시 의무
- IT Park 거주자: 패스트트랙 (간소화)
- 비IT: 정상 절차 (1~3개월)

### 비자 종류
- HQ Visa (IT Park 거주자, 3년)
- 일반 노동 비자 (1년)
- 비즈니스 비자 (단기)

## 8. 한국 본사 직원의 UZ 파견 (Detached)

### 임금 지급 방식
- 한국 본사가 KRW 지급 + UZ에서 추가 수당 (Allowance)
- UZ 자회사가 지급 (UZS) — UZ 노동허가 필요
- 혼합 방식 (양국 지급)

### 세무 처리
- 한국 측: 종합소득세 신고 (UZ 원천 소득은 외국납부세액공제)
- UZ 측: NDFL 신고 (UZ 거주자 판정 시)
- 한·우 DTA 적용

### 사회보험
- 한·우 사회보장협정 (있는 경우 적용)
- 일반적으로 UZ 사회세 적용 (이중 가입 회피 협의 필요)

## 9. 결산 시 주의사항

### 환차손익
- UZS 단일 통화 회계가 원칙이나 외화 거래 빈번
- 매월 cbu.uz 결산 환율 적용
- 환차손익 → 손익계산서 직접 (NAS는 재무 항목)

### 인플레이션 영향
- UZ 인플레이션 8~12% (cbu.uz 통계)
- 자산 재평가 의무 (정해진 카테고리·주기)
- 손익 인플레이션 조정 (해당 시, NAS 별도 규정)

### 급여 인상 압력
- 인플레이션 8~12% → 매년 급여 인상 압력
- 예산 편성 시 인건비 +10~12%/년 가정 권장
- 실적 분산 분석 시 인플레이션 분리 분석 (variance-analysis 스킬)

## 10. 검증 체크리스트

```
월간 급여 정산 완료 시 확인:

[ ] 직원별 Net Salary 정확
[ ] NDFL 신고서 (15일) 제출 완료
[ ] 사회세 신고서 (25일) 제출 완료
[ ] 사업주 부담 사회세 납부 완료
[ ] 급여 명세서 직원 발급 완료
[ ] 회계 분개 (1С 또는 한국 본사 ERP)
[ ] 한국 본사 보고용 KRW 환산표 송부

연간 결산 시 추가:
[ ] 직원별 연간 NDFL 정산
[ ] 외국인 직원 노동허가 갱신 확인
[ ] 한·우 DTA 적용 한국 측 신고
[ ] 사회보장협정 적용 (있는 경우)
```
