# 결산 관리 (Close Management)

## 개요

월간·분기·연간 회계 결산 업무를 체계적으로 지원한다. 결산 체크리스트 생성, 계정별 마감 작업 순서, 4대보험·급여 정산, K-IFRS 기준 결산 분개, 결산 보고서 작성까지 단계별 가이드를 제공한다. 2026년 최신 4대보험 요율, 최저임금, 노동법 변경사항을 반영한다.

**면책 고지: 이 스킬은 AI 보조 분석이며 공인 회계사·세무사의 전문 자문을 대체하지 않습니다.**

## 트리거 키워드

결산, 월말 마감, 분기 결산, 연간 결산, 회계 마감, 급여 정산, 4대보험, 재무 마감, 클로징, 결산 체크리스트

## 워크플로우

### 1단계: 결산 유형 선택

- **월간 결산** (매월 말일)
- **분기 결산** (3, 6, 9, 12월)
- **반기 결산** (6, 12월) — 상장법인 의무
- **연간 결산** (12월) — 법인세 신고 연계

### 2단계: 2026년 최신 기준 확인

**2026년 최저임금 및 4대보험 요율**
| 항목 | 2026년 기준 |
|------|-------------|
| 최저시급 | **10,320원** (2025년 10,030원 대비 +2.9%) |
| 월 환산 최저임금 | **2,156,880원** (주 40시간, 월 209시간 기준) |
| 국민연금 (근로자) | **4.75%** (전체 9.5% — 2026년 0.5%p 인상) |
| 국민연금 (사업주) | **4.75%** |
| 건강보험 (근로자) | **3.595%** (전체 7.19%) |
| 건강보험 (사업주) | **3.595%** |
| 장기요양보험 | 건강보험료의 **12.95%** (근로자·사업주 각 50%) |
| 고용보험 실업급여 (근로자) | **0.9%** (전체 1.8%) |
| 고용보험 실업급여 (사업주) | **0.9%** |
| 산재보험 | **사업주 100% 부담** (업종별 요율 상이) |

**2026년 주요 변경사항:**
- 국민연금: 1998년 이후 최초 요율 인상 (9% → 9.5%)
- 고용보험 적용 대상 확대: 플랫폼 노동자·프리랜서까지 확대 예정
- 노란봉투법 시행 (2026.3.10.): 하청 근로자 교섭권 확대 → 원청 사업주 리스크

### 3단계: 월간 결산 체크리스트

**1주차 (1-7일): 전월 거래 마감**
- [ ] 전월 매출 전표 확정 (세금계산서 발행 확인)
- [ ] 매입 세금계산서 수취 확인 및 회계 처리
- [ ] 카드 매출/지출 정산
- [ ] 경비 보고서 취합 및 승인

**2주차 (8-15일): 급여 정산**
- [ ] 급여 대장 작성 (최저임금 충족 여부 확인)
- [ ] 4대보험 공제액 계산 (2026년 요율 적용)
- [ ] 소득세·지방소득세 원천징수
- [ ] 급여 이체 및 급여명세서 발급
- [ ] 4대보험 신고 (EDI 또는 4대보험 정보연계센터)

**3주차 (15-20일): 계정 마감**
- [ ] 감가상각비 계상
- [ ] 선급비용·미지급비용 조정
- [ ] 재고자산 실사 (필요 시)
- [ ] 매출채권·매입채무 잔액 검토
- [ ] 외화 자산·부채 환산 (월말 기준환율 적용)

**4주차 (20일-말일): 결산 보고서**
- [ ] 시산표 검토 및 오류 수정
- [ ] 손익계산서 초안 작성
- [ ] 전월 대비 주요 지표 변동 분석
- [ ] 경영진 보고용 결산 요약 작성

### 4단계: 연간 결산 추가 작업

**법인세 신고 준비 (12월 결산법인 기준: 익년 3월 31일까지)**
- 세무조정 사항 검토 (접대비 한도, 감가상각 방법 등)
- 이월결손금 활용 검토
- 연구·인력개발비 세액공제 확인

**급여 연말정산 (1월 중)**
- 근로자 연말정산 서류 수취
- 소득공제·세액공제 항목 확인
- 원천징수 영수증 발급 (2월 말까지)

**K-IFRS 제1118호 조기 적용 검토 (2026)**
- 손익계산서 5개 범주 분류 준비
- MPM 공시 여부 결정

```

- 영업이익: xxx원 (영업이익률 xx%)
- 당기순이익: xxx원

## 주요 비용 분석

## 4대보험 납부 현황

## 다음 달 주요 일정
```

## 사용 예시

**예시 1**: "3월 월말 마감 체크리스트 만들어줘"
→ 결산 유형 확인 → 주차별 작업 → 4대보험 요율 적용 → 체크리스트 생성

**예시 2**: "우리 회사 2026년 급여 정산 방법 알려줘. 직원 5명이야"
→ 급여 대장 형식 → 4대보험 계산 → 원천징수 → 신고 절차

**예시 3**: "연간 결산 보고서 초안 작성해줘"
→ 전년 대비 분석 → 주요 지표 → 경영진 요약 → 개선 권고

## 문제 해결

- **세율·요율 변경 확인 불가**: 국세천(nts.go.kr) 또는 4대보험 정보연계센터(4insure.or.kr)에서 최신 요율을 직접 확인하세요. AI가 제시하는 수치는 참고용이며 실제 신고 시 반드시 공식 고시를 확인해야 합니다.
- **과거 회계 데이터 미제공**: 이전 기간 재무 데이터 없이는 전기 대비 분석이 불가능합니다. 분석 대상 기간의 손익계산서·총계정원장을 함께 제공해 주세요.
- **ERP 연동 오류**: SAP·더존·iCUBE 등 ERP에서 내보낸 CSV/Excel 파일을 직접 붙여넣으면 결산 체크리스트 자동화가 가능합니다.
- **외화 환산 기준일 불일치**: 한국은행 기준환율(ecos.bok.or.kr)에서 월말 고시 환율을 확인한 후 입력해 주세요.

## 이 스킬을 사용하지 말아야 할 때

- **실제 세금 신고·납부**: AI가 작성한 결산 초안을 그대로 세무서에 제출하지 마세요. 세무사에게 검토를 의뢰하세요.
- **재무제표 공식 작성**: 외부 감사 또는 공시용 재무제표가 필요하면 `financial-statements` 스킬과 공인회계사 검토를 함께 활용하세요.
- **예산 대비 실적 분석**: 전략적 원가·수익 분산 분석은 `variance-analysis` 스킬이 더 적합합니다.
- **세무 상담**: 종합소득세·부가가치세 신고 방법 상담은 `tax-helper` 스킬을 사용하세요.
