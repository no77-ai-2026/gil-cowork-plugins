# 캠페인 플래너 (Campaign Planner)

## 개요

마케팅 캠페인 기획 및 퍼포먼스 마케팅 전문 스킬입니다. A/B 테스트 설계, 그로스 해킹, 인플루언서 전략, CRM 전략, 고객 여정 맵, 영업 지원을 다룹니다.

> **v2.3.0 책임 정리**: 본 스킬에서 "이커머스 상세페이지 제작·AI 이미지 생성" 책임이 제거되었습니다(SPEC-CATALOG-CLEANUP-007 REQ-CLEANUP-007). 상세페이지·이미지가 필요한 경우 아래 "관련 스킬" 표를 참고하세요.

## 트리거 키워드

캠페인, 마케팅 캠페인, A/B 테스트, 그로스해킹, 인플루언서, 인플루언서 마케팅, CRM, 고객 여정, 고객 여정 맵, 퍼포먼스 마케팅, AARRR, 퍼널, 캠페인 KPI

## 워크플로우

### 1단계: 캠페인 목표 및 유형 확인

목표 설정(인지도/리드/전환/재구매), 채널 선택(유료 광고/SNS/이메일/인플루언서), 예산 규모 및 기간을 확인합니다.

### 2단계: 전략 수립

**캠페인 기획:** AARRR(Acquisition-Activation-Retention-Revenue-Referral) 퍼널 매핑, 채널별 예산 배분 및 KPI 설정, 크리에이티브 전략 방향을 수립합니다.

**A/B 테스트 설계:** 가설 수립 → 변수 정의 → 샘플 크기 계산, 통계적 유의성 기준 설정(p < 0.05), 결과 측정 지표 정의를 수행합니다.

**인플루언서 전략:** 메가/매크로/마이크로 인플루언서 티어 결정, 선정 기준(팔로워 수, 참여율, 타겟 부합도), 협찬 제안서 및 ROI 측정 방법을 수립합니다.

**CRM·고객 여정:** 고객 여정 맵(인지→고려→구매→리텐션→옹호) 매핑, 단계별 트리거·메시지 설계, 자동화 시퀀스 권고를 제공합니다.

### 3단계: 산출물 생성 및 검토

캠페인 전략 문서·A/B 테스트 설계서·인플루언서 제안서·고객 여정 맵·CRM 전략 문서 등 본 스킬 책임 범위의 산출물만 생성합니다.

## 사용 예시

**예시 1**: "신제품 런칭 캠페인 기획해줘. 예산 500만원, 타겟은 2030 여성"
→ 채널 믹스 → 예산 배분 → KPI 설정 → 크리에이티브 방향 → 타임라인

**예시 2**: "인플루언서 마케팅 전략 짜줘. 마이크로 인플루언서 위주로"
→ 인플루언서 티어 결정 → 선정 기준 → 협찬 제안서 → ROI 측정

**예시 3**: "이메일 시퀀스 A/B 테스트 설계해줘"
→ 가설 수립 → 변수 정의 → 샘플 크기 → 성공 지표 → 분석 방법

## 출력 형식

- 캠페인 전략 문서 (목표, 타겟, 채널, 예산, KPI)
- A/B 테스트 설계서 (가설, 변수, 샘플 크기, 성공 지표)
- 인플루언서 협찬 제안서
- 고객 여정 맵
- CRM 전략 문서

## 광고 심리학 완전판 (v2.4.0 신규)

### 성과 공식 (책 전체의 지도)
```
성과 = 타겟 × 동기 × 가치 × 방아쇠 × 구조 × 크리에이티브 × 타이밍 ÷ 경쟁강도
```

각 캠페인 기획 시 7요소 + 경쟁강도를 명시적으로 분석.

### 3가지 구매 동기

| 동기 | 정의 | 핵심 광고 |
|------|------|----------|
| **기능적 필요** | 명확한 문제 해결 | 검색광고·스펙 비교 |
| **심리적 필요** (가장 큰 시장) | 감정·보상·불안·지루함·사회적 자극 | SNS 피드·감성 영상·인플루언서·타이밍 |
| **정체성 구매** | "그 제품 쓰는 나"를 산다 | 브랜드 세계관·커뮤니티·라이프스타일 |

> **핵심**: '필요'는 만들 수 있다. 심리적 필요가 마케터가 직접 촉발할 수 있는 가장 큰 시장.

### 6가지 심리 방아쇠

| 방아쇠 | 작동 | 적용 |
|--------|------|------|
| **신뢰** (배수) | 0이면 가치 0 | 실명 후기·인증·환불 보증 |
| **손실 회피** | 손실 = 이득 × 2.5배 | 마감·재고·한정 |
| **사회적 증거** | 동기별 다름 | 대중 vs 전문가 |
| **인지적 쉬움** | 복잡하면 이탈 | 3클릭·간편결제 |
| **정체성** | "이걸 쓰는 나" | 라이프스타일 |
| **앵커링** | 처음 숫자 = 기준 | 가격 비교·타협 효과 |

### 9 인지 편향
프레이밍·후광·타협·현재·매몰비용·제로 리스크·선택 과부하·디폴트·단순 노출

> 페어 스킬 `gil-commerce:commerce-channel-message`의 9종 표와 동일 항목. Kahneman 행동경제학 표준 카탈로그 기반.

### PAS 카피 공식 + 혜택 언어 3단계

```
P (Problem) → A (Agitate) → S (Solution)
타겟 문제 그대로     문제 키우기           해결책 + 사회적 증거
                  (손실 회피 작동)
```

혜택 언어: 기능 결과 → 결과의 변화 → 변화의 감정 (정체성 구매 결정타)

### 6가지 후크 유형

| 후크 | 패턴 | 강한 동기 |
|------|------|----------|
| 공감형 | "아침마다 거울 보기 싫으신 분?" | 심리적·정체성 |
| 질문형 | "무릎 통증으로 계단 피하고 계세요?" | 기능적 필요 |
| 숫자·충격형 | "37일 만에 0 → 12,400명" | 신뢰 |
| 금지·역설형 | "다이어트 중이라면 보지 마세요" | 콜드 타겟 |
| 스토리형 | "저도 처음엔 반신반의했는데..." | 신뢰 (고관여) |
| 시연·결과형 | Before/After | 의심 해소 |

### 영상 광고 30초 구조

```
0-3초    후크만 (로고·브랜드 금지)
3-10초   공감 확장 (제품 얘기 아직 안 함)
10-20초  가치 제시 (기능 아닌 결과)
20-30초  신뢰·CTA (브랜드 로고 마지막)
```

### 타겟 온도 4단계 × 3 동기 매트릭스 (12조합)

타겟 온도(콜드/웜/핫/슈퍼) × 구매 동기(기능/심리/정체성)별 메시지 차별화. 본 스킬 캠페인 기획 시 12조합 매트릭스 명시.

### 채널별 심리 상태

| 채널 | 심리 | 주의 |
|------|------|------|
| 메타 | 수동적 발견 | 학습 기간 48-72시간 절대 건드리지 않음, CAPI 필수 |
| 구글 | 능동적 탐색 | 광고비 70%+ 구매 의도 키워드 |
| 네이버 | 정보 탐색 | 블로그·리뷰 생태계 먼저 |
| 카카오 | 90%+ 열람 | 기존 고객 유지에 ROI 압도적 |

### 1st Party 데이터 + 픽셀 + Lookalike

- **씨앗 품질이 전부**: VIP 구매자(상위 20%) Lookalike → 유사도 1-3%부터
- **픽셀 실수 3종**: 구매자 미제외 / 이벤트 파라미터 없음 / CAPI 미설치
- **CAC vs LTV**: LTV ÷ CAC ≥ 3 (지속 가능 구조)

### 단계별 예산 배분

| 단계 | 퍼포먼스:브랜딩 |
|------|--------------|
| 초기 (0-6개월) | 80:20 (PMF 검증) |
| 성장 (6-18개월) | 65:35 (브랜딩 효율 시작) |
| 성숙 (18개월+) | 50:50 (브랜드 직접 검색·팬덤) |
