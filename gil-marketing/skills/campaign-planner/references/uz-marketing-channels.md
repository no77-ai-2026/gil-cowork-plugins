# UZ 광고 채널 가이드

> gil-marketing v0.1.0 | UZ 광고 플랫폼 통합 가이드 (Telegram Ads·Yandex Direct·Mail.ru·인플루언서)

## 개요

우즈베키스탄(UZ) 광고 시장은 한국과 구조가 매우 다릅니다. **Telegram Ads가 사실상 1위 채널**이고, 검색 광고는 Yandex Direct가 Google보다 우위입니다. CPC·CPM 단가는 한국 대비 1/10~1/30 수준이며, ROI는 매우 높습니다.

---

## 1. UZ 광고 시장 점유율 (2026 추정)

| 채널 | 점유율 | 특징 | 한국 등가 |
|------|--------|------|----------|
| **Telegram Ads** | ~40% | 도달·로컬·저비용 | 카카오모먼트 |
| Yandex Direct | ~20% | 러시아어 검색 | 네이버 광고 |

## 2. Telegram Ads (UZ 1위 채널)

### 광고 형식

- **Sponsored Messages**: 공식 채널 노출 (CPM)
- **봇 통한 직접 광고**: 채널 운영자와 직거래 (가장 일반적)
- **인플루언서 게시**: 인플루언서 채널에 게시 (가장 효과적)

### 단가 (2026 기준)

| 형식 | CPM (USD) | 비고 |
|------|----------|------|
| 공식 Sponsored | $0.05 ~ $0.10 | 채널 사이즈 자동 매칭 |
| 채널 운영자 직거래 | $0.02 ~ $0.20 | 채널 규모·니치 따라 |
| 인플루언서 게시 (단일) | $50 ~ $5,000 | 1만~50만 구독 |

> 한국 카카오모먼트 CPM ₩2,000~4,000 ($1.5~$3) 대비 **1/15~1/60** 수준.

### 캠페인 설계

```
1. 채널 리서치
   - UZ Telegram 채널 검색 (telega.io 또는 직접)
   - 카테고리: 뉴스·라이프·뷰티·IT·비즈니스 등
   - 구독자 수·참여율·게시 빈도 확인

2. 단가 협상
   - 공식 광고: Telegram Ads 셀프 서비스
   - 직거래: 채널 운영자에게 DM (러시아어 또는 우즈벡어)
   - 인플루언서: 에이전시 또는 직접

3. 광고 소재
   - 텍스트 + 이미지 (1~2장)
   - 영상 클립 (15~60초)
   - CTA: 채널 가입 / 외부 링크 / 봇 명령어

4. 추적
   - UTM 파라미터
   - 봇 통한 직거래 시 자동 추적
   - 짧은 URL (bit.ly 등)으로 링크 클릭 추적
```

### 효율 좋은 카테고리

- **뷰티·코스메틱**: K-Beauty 매우 인기
- **전자제품**: 삼성·LG 신뢰 1위
- **식품**: K-Food 트렌디
- **교육·IT**: 영어·코딩·IT
- **여행**: UZ 내 여행

## 3. Yandex Direct (UZ 검색 광고)

### 광고 형식

- **검색 광고**: Yandex 검색 결과 노출
- **YAN (Yandex Advertising Network)**: 디스플레이·앱
- **Smart Banners**: 자동 생성 배너
- **Yandex.Market**: 이커머스 통합

### 단가

| 광고 | CPC (USD) | 비고 |
|------|----------|------|
| 검색 광고 (러시아어) | $0.05 ~ $0.30 | 키워드 따라 |
| 검색 광고 (우즈벡어) | $0.03 ~ $0.20 | 경쟁 적음 |
| YAN 디스플레이 | $0.02 ~ $0.10 | CPM 모델 가능 |

> 네이버 광고 CPC ₩100~500 ($0.08~$0.4) 대비 유사 또는 약간 저렴.

### 캠페인 설계

```
1. Yandex Direct 계정 개설
   - direct.yandex.com
   - UZ 비즈니스 계정 또는 글로벌

2. Yandex Wordstat로 키워드 조사
   - wordstat.yandex.com
   - 러시아어 키워드 검색량 확인
   - 우즈벡어는 일부 데이터만 (Google Keyword Planner 보조)

3. 광고 그룹 구성
   - 키워드 매칭 타입 (정확·구문·광범위)
   - 광고문 다국어 (러시아어 + 우즈벡어 별도)
   - 랜딩 페이지 (러시아어 우선)

4. 입찰 전략
   - 최대 클릭 (초기)
   - 전환 최적화 (Yandex Metrica 연동 후)
   - CPA 목표

5. 추적
   - Yandex Metrica (필수)
   - 전환 목표 설정
   - UTM 파라미터
```

### Yandex Wordstat 활용

- 월간 검색량
- 시즌성·트렌드
- 지역별 (UZ 도시 단위)
- 디바이스별 (모바일 70%+)

- 자체 매장·파트너십

## 9. UZ 광고법 점검

### «Закон о рекламе Республики Узбекистан»

- 사전 인증: 의약·금융·식품·교육·아동
- 비교 광고 일부 제한
- 우즈벡어 표기 의무 (러시아어 단독 X 일부 카테고리)
- 미성년자 대상 제한
- 술·담배 광고 금지

### 위반 시
- 행정 제재
- 광고 차단
- 외국 회사도 동일 적용

상세는 `gil-legal:compliance-check`의 UZ 가이드 참조.

## 11. 한국+UZ 통합 캠페인 모델

### 모델 A: 한국 본사 직접 운영
- 한국 본사 마케팅팀이 UZ 캠페인 직접 운영
- KOTRA Tashkent 자문

### 모델 B: 현지 에이전시 위탁
- UZ 현지 마케팅 에이전시에 위탁
- 한국 본사가 전략·예산 결정
- 매월 보고

### 모델 C: 하이브리드
- 디지털 (Telegram·Yandex)는 한국 본사 운영
- 인플루언서·현지 PR은 에이전시 위탁
- 가장 일반적 모델
