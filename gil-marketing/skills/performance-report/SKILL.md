# 퍼포먼스 리포트 (Performance Report)

## 개요

GA4, 네이버 광고, 메타(Meta) 광고, 카카오모먼트, 구글 광고 등 한국 시장 주요 채널의 성과 데이터를 통합 분석하여 마케팅 성과 보고서를 생성합니다.

데이터 해석과 인사이트 도출, 다음 액션 플랜까지 포함한 완성형 리포트를 제공합니다.

## 트리거 키워드

성과 보고서, KPI 리포트, ROAS 분석, 광고 성과, GA4 분석, 메타 광고 리포트, 네이버 광고 분석, 퍼포먼스 리포트, 마케팅 ROI, 전환율 분석, CTR, CPC, 광고비 분석

## 실전 그로스 마케팅 리포트 패턴 (한국 2026 쿠키리스 시대)

2026년 한국 마케팅 환경은 근본적 전환점입니다. iOS ATT(2021.04) 이후 한국 추적 동의율 23.66%, 구글 크롬 쿠키 단계적 종료, 평균 전환율 2024년 2.18% → 2025년 1.99% 하락. 단일 ROAS 측정은 더 이상 통하지 않습니다. **1st party data + CRM + MMM + 맥락 타깃팅의 멀티 레이어 전략**이 표준이 되었습니다.

### 7개 핵심 패턴 (2026 한국)

**① 그로스 마케팅 3대 요소**

- **데이터**: 고객 행동을 정량화·측정
- **고객 가설**: "고객 행동 B를 이끌어 내려면 행동 A가 필요"
- **실험**: 가설 검증으로 행동 변화 입증

리포트 = 위 3요소의 결과 + 다음 액션. **데이터만 있고 가설 없으면** 의미 없음.

**② "측정 → 관리 → 개선" 사이클**

측정 없으면 관리 불가, 관리 없으면 개선 불가. 모든 마케팅 액션은 메트릭 가능해야 — 어렵더라도 정량화 우선.

**③ 한국 D2C·이커머스 메트릭 표준식**

```
유저 유입 × CTR × CVR × AOV = 매출
매출 − 원가 − 배송비 − 광고비 = 이익
이익 × 재구매율 = 누적 이익 (LTV)
```

ROAS 단일 지표만 보지 말고 **누적 이익 = LTV × 재구매**까지 봐야 진짜 그로스.

**④ NSM(North Star Metric) 중심 운영 (2026 SaaS 표준)**

| 비즈니스 | NSM 예시 | 입력 지표 |
|---|---|---|
| SaaS B2B | Weekly Active Teams | 신규 팀 활성화율·기능 채택 폭·협업 깊이·유지율 |
| SaaS B2C | Weekly Active Engaged Users | 핵심 액션 완료율·세션 빈도·세션 깊이 |
| 마켓플레이스 | GMV는 X, **수수료 매출 + 재구매율** | 매도자·매수자 활성·매칭 성공률 |
| 딥테크·B2B | PoC 통과율 + 앵커 고객 수 | 영업 사이클 단축·기술 임계점 |

**LTV:CAC 비율 최소 3:1** 유지가 SaaS 기준선. CAC 정당화 = LTV가 3배 이상 도달해야 자금 소진 안 됨. **DAU·가입자 수는 허무 지표**, 고객 가치 실현 행동이 NSM의 본질.

**⑤ 채널별 진단 체크리스트 (2026 한국)**

| 채널 | 핵심 지표 | 2026 주의 |
|---|---|---|
| **네이버 광고** | 파워링크 vs 쇼핑 광고 ROAS, AI 브리핑 영향 | AI 브리핑 도입으로 SERP CTR 감소 추세 |
| **메타** | 릴스 vs 피드, 학습 기간 48-72시간 | iOS ATT 후 EMQ 점수 관리 + CAPI 필수 |
| **구글** | 품질점수, GA4 + Consent Mode v2 | 쿠키 종료 대비 1st party 결합 |
| **카카오** | 채널 알림·디스플레이 | 슈퍼 타겟 90%+ 열람 = CRM 채널 |
| **유튜브 쇼츠** | CPV·CTR·시청 완료율 | 후킹 0-3초 절대적, 풀퍼널 포맷 |

**⑥ 1st Party Data + MMM 멀티 레이어 (2026 쿠키리스 대응)**

| 레이어 | 역할 | 도구 |
|---|---|---|
| **1st Party Data 자산화** | 이메일·전화·구매 이력·동의 정보 직접 보유 | CDP, CRM, 자사몰 |
| **CRM 마케팅** | 동의 받은 고객 → 인앱·카톡·이메일 직접 도달 | Stibee, 채널톡, 카카오 비즈니스 |
| **MMP** | 모바일 측정 파트너 | 앱스플라이어, 에어브릿지, 애드저스트 |
| **MMM** | AI·ML 기반 통합 성과 측정 | 채널 통합 효과 분해 |
| **맥락 타깃팅** | 쿠키 없이 콘텐츠 맥락으로 도달 | 광고 노출 페이지 카테고리 매칭 |

**SKAdNetwork(SKAN) + Privacy Sandbox**: 애플·구글 대안 솔루션 — 기존 대비 정확도는 떨어지지만 의무 활용. 단일 솔루션 X, **멀티 레이어 결합이 표준**.

**⑦ MVT (Minimum Viable Test) — 완벽한 실험 X**

| 마인드 | 변화 |
|---|---|
| ❌ 한 번 실험으로 완벽한 결과 찾기 | ✅ 연속·일관된 여러 실험으로 답 |
| ❌ 통제 변수 100% 통제 | ✅ 최소 충족 요건 수준 검증 |
| ❌ 통계 유의미만 해석 | ✅ 작은 실마리도 활용 |

회사 환경 = 실험실 아님. **합리적 해결의 실마리**가 1차 목표.

| 변화 | 영향 | 대응 |
|---|---|---|

### NSM 운영 케이던스 (2026 SaaS 표준)

| 주기 | 액션 |
|---|---|
| 주간 | NSM·입력 지표 변동 리뷰, 담당자 병목 해소 |
| 월간 | 코호트 + Unit Economics (LTV/CAC, Payback) 심층 |
| 분기 | NSM 정의·목표·입력 세트 재평가, A/B + 홀드아웃 |

### 그로스 실험 변수 표준

| 변수 | 정의 | 예시 |
|---|---|---|
| **독립 변수** | 우리가 조작하는 것 (A) | 새 광고 소재 도입 |
| **종속 변수** | 측정하는 결과 (B) | CTR·CVR·매출 |
| **통제 변수** | 동일 유지 | 타깃·예산·기간 |

실험 그룹 (통제 + 독립) vs 통제 그룹 (통제만)의 종속 변수 차이 = 독립 변수의 영향.

### 한국 리포트 표준 포맷 (5분 안에 읽힘)

```markdown
# [기간] 마케팅 성과 리포트

## 경영진 요약 (3줄)
- 총 광고비 / 총 매출 / 전체 ROAS / NSM 변동
- 전기 대비 핵심 변화
- 다음 분기 우선순위 1줄

## NSM 트렌드
| NSM | 이번 주 | 4주 평균 | 목표 대비 |

## 채널별 성과
| 채널 | 광고비 | 매출 | ROAS | LTV/CAC | 전월 대비 |

## 핵심 발견 3가지 (인사이트 ≠ 팩트)

## 1st Party Data 자산 상태
- 이메일 구독자 수·CRM 동의율·재참여율

## 다음 액션 플랜
- [ ] 예산 재배분
- [ ] 신규 실험 (가설 + 변수 명시)
- [ ] 중단 채널·세그먼트
```

### 안티 패턴 (2026)

- 데이터 나열만 (시사점 없음)
- ROAS만 보고 LTV·CAC·NSM 무시
- 채널 간 비교 없이 단일 채널만
- "전월 대비 +5%" 같은 무의미한 비교 (계절·캠페인 차이 미고려)
- 실험 가설 없이 사후 해석
- 1st party data 자산화 무시 (쿠키리스 시대 치명적)
- 단일 측정 도구만 (멀티 레이어 X)

> 💡 2026 한국 그로스 리포트 진짜 가치: **NSM → 입력 지표 → 가설·실험 → 다음 액션** 사슬 + **1st party 자산화 진행률**. ROAS만 보고하는 리포트는 쿠키리스 시대 의사결정에 도움 안 됩니다.

## 워크플로우

### 1단계: 보고서 범위 설정

보고 기간 선택(주간/월간/분기/연간), 포함할 채널 선택(유료 광고, 유기적 채널, 분석 도구), 목표 KPI 확인을 수행합니다.

### 2단계: 핵심 지표 수집 가이드

사용자가 제공할 데이터 형식을 안내합니다:
- GA4 필수 데이터: 세션 수, 사용자 수, 전환수, 전환율, 채널별 획득 데이터, 이탈률, 세션 지속 시간
- 광고 플랫폼 데이터: ROAS, CPA, CTR, CPC, CVR, CPM

### 3단계: 채널별 분석

**네이버 광고:** 파워링크 vs 쇼핑 광고 성과 비교, 스마트플레이스 노출 현황, 네이버 쇼핑 ROAS, C-Rank 영향 유기 트래픽 변화를 분석합니다.

**메타(인스타그램/페이스북) 광고:** 2026년 알고리즘 변화 반영(릴스 vs 피드 광고 성과 비교, 딥 인게이지먼트 지표 확인), 오디언스 인사이트, 광고 소재 A/B 테스트 결과를 분석합니다.

**구글 광고:** 검색 캠페인 품질점수, 전환 추적(GA4 연동 확인), 스마트 캠페인 vs 수동 캠페인 성과를 분석합니다.

### 4단계: 보고서 생성 (표준 포맷)

경영진 요약(총 광고비, 총 매출, 전체 ROAS, 전기 대비, 핵심 인사이트), 채널별 성과표, 주요 발견 및 인사이트, 개선 기회, 다음 기간 액션 플랜을 포함한 보고서를 생성합니다.

### 5단계: 인사이트 도출 및 권고안

성과 부진 채널 원인 분석, 예산 재배분 시뮬레이션, 다음 기간 테스트 가설 제안을 수행합니다.

## 사용 예시

**예시 1**: "지난달 네이버+메타 광고 성과 통합 분석해줘. 데이터 여기 있어: [데이터]"
→ 데이터 파싱 → 채널별 KPI → 비교 분석 → 인사이트 → 다음 달 권고안

**예시 2**: "분기 마케팅 성과 경영진 보고서 만들어줘. 총 광고비 3천만원, ROAS 350%"
→ Executive Summary → 채널별 성과표 → 주요 발견 → 다음 분기 전략

## 출력 형식

- 채널 통합 성과 대시보드 (마크다운/표 형식)
- 경영진용 요약 리포트 (1페이지)
- 실무팀용 상세 분석 리포트
- 다음 기간 예산 배분 권고안
- A/B 테스트 결과 분석

## 주의사항

| 상황 | 대응 |
|------|------|
| 데이터가 없거나 불완전한 경우 | 보유한 데이터로 분석 가능한 범위 제시. 누락 데이터 수집 방법 안내 |
| 채널이 너무 많아 복잡한 경우 | 매출 기여도 상위 2-3개 채널 집중 분석 후 나머지 요약 처리 |
| 업종 벤치마크가 없는 경우 | 범용 업종 기준값 제시. 사용자 직접 측정 KPI 설정 권고 |
