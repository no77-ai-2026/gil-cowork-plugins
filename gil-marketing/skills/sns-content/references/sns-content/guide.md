# SNS/블로그 콘텐츠 생성기

소상공인과 1인기업의 디지털 마케팅을 위한 콘텐츠를 자동 생성합니다.
한국에서 가장 많이 사용하는 플랫폼(네이버, 인스타그램, 카카오, 유튜브)에 최적화합니다.

## 지원 플랫폼

| 플랫폼 | 콘텐츠 유형 | 최적 길이 | 특징 |
|--------|-----------|----------|------|
| 네이버 블로그 | 정보형/리뷰형 글 | 1,500-3,000자 | SEO 중요, 키워드 배치 |
| 인스타그램 | 피드/릴스 캡션 | 100-300자 | 해시태그 30개, 이모지 |
| 카카오 채널 | 소식/프로모션 | 200-500자 | CTA 버튼, 이미지 연동 |
| 유튜브 | 스크립트/설명란 | 1,000-5,000자 | 타임스탬프, 키워드 |
| 블로그 (일반) | 전문 콘텐츠 | 2,000-5,000자 | 전문성, 신뢰도 |

## 네이버 블로그 SEO 가이드

### 키워드 배치 전략
```
1. 제목: 핵심 키워드 포함 (30자 이내)
2. 소제목: 관련 키워드 자연 삽입
3. 본문 첫 문단: 핵심 키워드 1~2회
4. 본문 중간: 관련 키워드 자연 분포
5. 태그: 핵심 키워드 + 관련 키워드 (최대 10개)
```

### 네이버 블로그 포스팅 구조
```
[제목] (핵심 키워드 포함, 호기심 유발)

[도입부] (1~2문단)
- 독자의 관심/공감을 끄는 첫 문장
- 이 글에서 얻을 수 있는 정보 예고

[본문] (3~5개 소제목)
■ 소제목 1
  (본문 내용 - 구체적 정보, 경험, 데이터)
  [이미지/사진 삽입 위치]

■ 소제목 2
  (본문 내용)

■ 소제목 3
  (본문 내용)

[마무리] (1문단)
- 핵심 요약
- CTA (댓글/공감/이웃추가 유도)

[태그]
#핵심키워드 #관련키워드1 #관련키워드2 ...
```

## 인스타그램 콘텐츠

### 피드 캡션 구조
```
[첫 줄] 후킹 (가장 중요! 더보기 클릭 유도)
.
[본문] 2~4줄 핵심 내용
.
[CTA] 액션 유도 (댓글, 저장, 공유)
.
.
[해시태그] (최대 30개)
#대분류 #중분류 #소분류 #브랜드태그 #위치태그
```

### 해시태그 전략
| 유형 | 예시 | 비율 |
|------|------|------|
| 대형 (100만+) | #맛집 #일상 #서울 | 20% |
| 중형 (1만-100만) | #홍대맛집 #카페추천 | 50% |
| 소형 (약 1만) | #홍대파스타 #연남동브런치 | 20% |
| 브랜드 | #우리가게이름 | 10% |

## 카카오 채널 메시지

### 소식 발송 구조
```
[제목] (15자 이내, 임팩트)

[본문] (200자 이내)
핵심 메시지를 간결하게.
혜택/정보를 명확히.

[CTA 버튼]
▶ 자세히 보기 / 예약하기 / 쿠폰 받기
```

## 유튜브 스크립트

### 구조
```
[인트로] (15~30초)
- 후킹: 영상의 핵심 가치 전달
- "이 영상을 끝까지 보시면 OO를 알 수 있습니다"

[본문] (주제별 챕터)
■ 챕터 1: (소주제)
  - 핵심 포인트
  - 예시/근거

■ 챕터 2: (소주제)
  - 핵심 포인트

[아웃트로] (15~30초)
- 핵심 요약
- 구독/좋아요 요청
- 다음 영상 예고

[설명란]
타임스탬프:
00:00 인트로
01:23 챕터 1
...

관련 링크:
키워드 태그:
```

## 콘텐츠 시리즈 기획

여러 편의 콘텐츠를 시리즈로 기획할 수 있습니다:

| 시리즈 유형 | 예시 | 추천 편수 |
|-----------|------|----------|
| 튜토리얼 | "엑셀 기초 마스터" 1-10편 | 5-10편 |
| Q&A | "자주 묻는 질문 TOP10" | 5-10편 |
| 비교 리뷰 | "A vs B 비교" | 3-5편 |
| 시즌별 | "봄 맞이 인테리어" | 4편 (계절별) |

## 출력 형식

- 텍스트: 바로 복사/붙여넣기 가능 (기본)
- `.md`: Markdown (블로그용)
- `.html`: HTML (웹 게시용)
- `.docx`: Word (시리즈 기획서)

## 워크플로우

```
사용자 요청 → 플랫폼/유형 파악 → 주제/키워드 확인
→ SEO/해시태그 설계 → 콘텐츠 작성 → 출력
```

## 트러블슈팅

| 증상 | 원인 | 해결 |
|------|------|------|
| 스킬이 트리거되지 않음 | 키워드 미매칭 | MANDATORY TRIGGERS 키워드(SNS, 블로그, 인스타 등)를 포함하여 요청 |
| 해시태그 관련성 낮음 | 타겟 정보 부족 | 타겟 고객, 업종, 지역 정보를 함께 제공 |
| 글 길이 부적절 | 플랫폼별 최적 길이 미달성 | 플랫폼 특성에 맞춘 길이 조정 |
| SEO 순위 낮음 | 키워드 배치 부족 | 제목, 첫 문단, 태그에 키워드 강화 |
