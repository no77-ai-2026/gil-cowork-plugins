# SNS 콘텐츠 제작 (SNS Content) — 한국 + 글로벌 통합

## 개요

인스타그램·네이버 블로그·카카오 채널(한국 3채널) + 스레드·X·링크드인·유튜브 쇼츠(글로벌 4채널) SNS 최적화 콘텐츠를 제작합니다.

브랜드 아이덴티티, 브랜드 보이스 가이드, 콘텐츠 재활용(리퍼포징), 퍼스널 브랜딩 전략을 포함한 종합 SNS 콘텐츠 서비스를 제공합니다.

> **v2.3.0 업데이트**: `gil-content:social-media`가 본 스킬로 흡수되었습니다. 글로벌 4채널(스레드·X·링크드인·유튜브쇼츠) 모드가 추가되었으며, 한국 3채널 모드는 기본 유지됩니다.
>
> **v2.4.0 강화**: 채널별 심리 상태 매트릭스(메타·유튜브·쇼츠·네이버·카카오·스레드·X·LinkedIn·쿠팡 9채널) + 메타 학습 기간 가이드(48-72시간 절대 미수정) + iOS 14.5+ 이후 CAPI 필수·GA4 교차 검증이 통합되었습니다 (SPEC-CATALOG-CLEANUP-007 §v2.4.0 Amendment §B).

SNS, 인스타그램, 인스타 릴스, 네이버 블로그, 카카오 채널, 스레드, Threads, X, 트위터, 링크드인, LinkedIn, 유튜브 쇼츠, YouTube Shorts, 콘텐츠, 브랜딩, 퍼스널 브랜딩, 브랜드 보이스, 글로벌 SNS, 한국 SNS, 콘텐츠 달력, 리퍼포징, 해시태그 전략, 숏폼

## 모드 선택

### 모드 1: 한국 3채널 (기본 — 한국 셀러·1인 사업자용)

- 인스타그램 (피드·릴스·스토리)
- 네이버 블로그 (C-Rank 최적화)
- 카카오 채널

### 모드 2: 글로벌 4채널 (B2B·SaaS·해외 마케터용)

- 스레드 (Threads)
- X (구 Twitter)
- 링크드인 (LinkedIn)
- 유튜브 쇼츠 (YouTube Shorts)

### 모드 3: 통합 (한·글로벌 7채널 동시)

## 워크플로우

### 1단계: 모드·채널·콘텐츠 유형 확인

대상 모드 선택(한국/글로벌/통합) → 채널 선택(인스타·블로그·카톡·스레드·X·링크드인·쇼츠) → 콘텐츠 유형(피드/릴스/스토리/블로그/포스트/숏폼) → 브랜드 톤·타겟 오디언스 확인.

### 2단계: 채널별 최적화 작성

**[한국] 인스타그램**
- 캡션(150자 이내 핵심 + 콜투액션)
- 해시태그(3-5개, 2026년 알고리즘 — 대형 태그 효과 급감)
- 릴스(후킹 첫 3초 + 스크립트: Hook → Problem → Solution → CTA)
- 활동 피크: 평일 19-24시, 목요일 22시 (DM 최다)
- 2026 핵심 변화: 프라이빗 소셜 강화(DM·공유 5-10배 신호), 릴스 20분 확장, 관심사 기반 알고리즘

**[한국] 네이버 블로그 (C-Rank 최적화)**
- 제목(핵심 키워드 포함, 25자 이내)
- 서론-본론-결론 구조(2,000자 이상)
- 경험 기반 콘텐츠 강조(D.I.A. 기준)

**[한국] 카카오 채널**
- 짧고 명확한 메시지(카카오톡 특성)
- 이모지·구분선 활용
- CTA 버튼 문구

**[글로벌] 스레드 (Threads)**
- 짧은 문장, 대화 유도
- Meta 알고리즘 — 인스타와 시너지

**[글로벌] X (구 Twitter)**
- 1-2개 해시태그 + 트렌딩 태그
- 280자 제한 활용
- 스레드(thread) 연결 가능

**[글로벌] 링크드인 (LinkedIn)**
```
첫 줄: 스크롤 멈추는 첫 문장 (질문 또는 충격적 통계)
...더 보기 클릭 유도

본문: 3~5개 단락 (각 2~3문장)
- 개인 경험 + 인사이트
- 구체적 수치

결론: 배운 점 1가지

CTA: 의견을 댓글로 남겨주세요 / 공감하시면 리포스트
```
- 개인 브랜딩 중심: 창업자·임원의 개인 계정 활성화
- 산업 인사이트 + 개인 경험 결합
- 해시태그 3-5개 (관련 업종 태그)

**[글로벌] 유튜브 쇼츠 (YouTube Shorts)**
- 처음 3초: 핵심 후킹 문구 (시청 지속률 결정)
- 자막 필수 (무음 시청 비율 높음)
- 시리즈화: 관련 쇼츠 플레이리스트로 시청 시간 증대
- 장편 영상의 하이라이트 클립 재활용

### 3단계: 브랜드 전략 (선택)

브랜드 보이스 가이드(톤, 키워드, 금지어), 콘텐츠 달력(주제별 월간 계획), 퍼스널 브랜딩(개인 포지셔닝 + 콘텐츠 방향성)을 수립합니다.

> 본격적인 기업·제품 브랜드 전략은 `gil-marketing:brand-identity`, 개인 단독 전략은 `gil-marketing:personal-branding`을 사용하세요.

### 4단계: 리퍼포징 (선택)

블로그 → 인스타 카드뉴스 → 유튜브 쇼츠 → 뉴스레터 → 링크드인 포스트 변환을 수행합니다.

## 사용 예시

**예시 1 (한국 모드)**: "스킨케어 브랜드 인스타 피드 포스팅 5개 써줘. 타겟은 30대 직장 여성"
→ 타겟 분석 → 톤 설정 → 주제 5개 → 캡션 + 해시태그 세트

**예시 2 (글로벌 모드)**: "B2B SaaS 제품의 링크드인 시리즈 포스트 5개 + X 쓰레드 5개 만들어줘"
→ 타겟 분석 → 페인포인트 → 인사이트 5개 → 채널별 포맷 변환

**예시 3 (통합 리퍼포징)**: "내 유튜브 강의 내용을 인스타·블로그·뉴스레터·링크드인·X로 리퍼포징해줘"
→ 원본 분석 → 채널별 포맷 변환 → 각 채널 최적화 완성

## 출력 형식

- 인스타그램: 캡션, 해시태그, 릴스 스크립트 (JSON)
- 네이버 블로그: 제목, 본문, 해시태그
- 카카오 채널: 메시지, 이모지, CTA
- 스레드: 짧은 포스트, 해시태그
- X: 트윗/스레드, 해시태그
- 링크드인: 첫 줄, 본문 단락, 결론, CTA, 해시태그 (JSON)
- 유튜브 쇼츠: Hook, 자막, 스크립트
- 브랜드 보이스 가이드: 톤, 키워드, 금지어
- 콘텐츠 달력: 주제별 월간 계획

| 상황 | 대응 |
|------|------|

## 채널별 심리 상태 매트릭스 (v2.4.0 신규)

각 채널은 완전히 다른 심리 상태의 사람들이 있는 곳. 같은 콘텐츠라도 채널별로 다른 메시지·구조 적용.

| 채널 | 심리 상태 | 강한 동기 | 메시지·구조 전략 |
|------|----------|-----------|-----------------|
| **인스타·페이스북** (메타) | 수동적 발견 | 심리적 필요·정체성 | 후크·감정 촉발·시각적 임팩트. 영상 0-3초가 결정타 |
| **유튜브** | 콘텐츠 소비 | 심리적·정체성 | 영상 30초 구조 (후크→공감→가치→CTA) |
| **유튜브 쇼츠 / 인스타 릴스 / 틱톡** | 빠른 발견 | 심리적 필요 | 0-3초 강력 후크 + 60초 이내 완결 |
| **네이버 블로그** | 정보 탐색 시작 (한국) | 기능적 필요 | C-Rank 최적화 + 블로그 후기 신뢰 (광고비보다 블로그·리뷰 먼저) |
| **카카오 채널** | 슈퍼 타겟 (90%+ 열람) | 정체성·재구매 | 정체성·VIP 혜택. 신규 획득 ❌ 기존 고객 유지 ✅ |
| **스레드 (Threads)** | 캐주얼 대화 | 심리적·정체성 | 4-5개 연속 포스트, 500자 단위 대화체 |
| **X (Twitter)** | 빠른 의견 | 정체성·뉴스 | 280자 헤드라인 + 스레드(threadcrop) |
| **LinkedIn** | 전문 정체성 | 기능적+정체성 | 1300자 인사이트 + Hook + CTA, B2B 정체성 |
| **쿠팡** (참고) | 즉시 구매 의도 | 기능적 | `gil-commerce:coupang-ad-optimizer`로 연계 |

## 메타 학습 기간 가이드 (v2.4.0 신규)

광고 세트 활성화 후 **48-72시간은 절대 건드리지 않음**:

- ❌ 예산 변경 → 학습 리셋
- ❌ 타겟 수정 → 학습 리셋
- ❌ 광고 중단 후 재개 → 학습 리셋
- ❌ 크리에이티브 교체 → 학습 리셋

**원칙**: 의심스러운 ROAS여도 첫 72시간은 데이터 축적 단계. 4일째부터 의사결정.

### iOS 이후 주의

메타 광고 관리자의 전환 수가 실제보다 **30-50% 적게 보일 수 있음**:
- **CAPI (전환 API) 설치 필수**
- **GA4 교차 검증 필수**
- pixel-only는 iOS 14.5+ 이후 신뢰도 낮음
