# 타겟 스크립트 (Target Script)

## 개요

타겟 고객 맞춤 메시징 스크립트 생성 스킬입니다. 타겟 오디언스 분석, 페인포인트 도출, 핵심 메시지 생성, 채널별 변환, A/B 테스트 제안의 5단계 파이프라인을 통해 타겟 고객에게 최적화된 메시징 스크립트를 생성합니다.

단순한 카피라이팅을 넘어, 타겟 오디언스의 심리적 특성과 페인포인트를 분석하여 설득력 있는 메시징 전략을 수립합니다.

## 실전 카피라이팅·메시징 패턴 (한국 2026 시장)

AI가 콘텐츠 생산을 90%까지 점유한 시대, 평균 전환율은 오히려 2024년 2.18% → 2025년 1.99%로 하락했습니다. 같은 도구·같은 프롬프트가 구별 안 되는 카피의 홍수를 만든 결과입니다. 2026년 한국 시장에서 다시 부상한 것은 "**무엇을 말하느냐**가 아니라 **어떻게 말하느냐**" — 사람의 설득 언어입니다.

### 7개 핵심 패턴 (2026 한국 검증)

**① 구체성 = 카피 제1원칙**

추상 표현은 머릿속에 그림을 만들지 못합니다. 구체로 바꾸면 같은 메시지가 사진 한 장과 함께 전달됩니다.

| 추상 (흘러감) | 구체 (그림이 그려짐) |
|---|---|
| 빠른 배송 | 잠들기 전 주문 → 아침 배송 도착 |
| 안전한 결제 | KISA 인증·금융결제원 검증 통과 |
| 인기 제품 | 이번 달 1,238명이 재구매 |
| 가벼움 | 사과 한 알 무게(180g) |

한 줄에 **숫자 또는 비유 1개 이상**이 베이스. 추상 형용사("최고", "프리미엄", "혁신적")는 거의 모든 경쟁사가 사용 — 차별화 효과 0.

**② 낯설게·불편하게 조합 (편안한 카피는 흘러간다)**

익숙한 단어 조합은 졸음을 부르고, 낯선 조합은 멈춤을 만듭니다. 카피의 본질 = 단어 조립.

| 익숙 (흘러감) | 낯섦 (멈춤) |
|---|---|
| 신선한 식재료 | 어제 잡은 고등어 |
| 깨끗한 정수기 | 5초 전 만들어진 물 |
| 빠른 배송 | 새벽 3시 주문도 아침 7시 |
| 친환경 화장품 | 먹어도 되는 립밤 |

서로 멀리 있는 단어를 핀셋으로 뽑아 붙일 때 시선이 머뭅니다.

**③ 반복으로 임팩트 (동어 반복 ≠ 동의어 반복)**

같은 단어를 한 문장에 두 번 사용. **동의어가 아닌 같은 단어**여야 임팩트가 생깁니다.

- "내일 살 것을 오늘 산다"
- "한 번 산 사람이 한 번 더 산다"
- "보고 싶어서 본다, 보다 보니 보인다"

문장이 길면 헤드라인+서브로 분리 — 헤드라인은 5-10자 이내가 임팩트 최적.

**④ "찾는다", "만들지 않는다" (search vs make)**

어깨에 힘 들어간 표현은 폭투. 회의실에서 만든 "혁신적인 솔루션" 같은 표현보다, **이미 사람들이 쓰는 말 중 딱 맞는 것을 발견**하는 게 카피 작업의 본질입니다.

채집 채널:
- 영업 미팅 녹취 (고객이 결정 직전에 한 말)
- 후기·리뷰 (실제 사용자 표현)
- 고객센터 통화 (반복되는 질문·고민)
- SNS 댓글·DM (자발적 발화)

**⑤ 한자어·전문어 → 일상어 (초등학생 기준)**

| 어려운 표현 | 쉬운 표현 |
|---|---|
| 종합적 검토 후 결정 | 다 보고 정함 |
| 효율성 극대화 | 시간 안 들이고 끝남 |
| 전사적 자원 관리 | 회사 운영 시스템 |
| 사용자 친화적 인터페이스 | 누구나 쓸 수 있게 만든 화면 |
| 최적의 솔루션 제공 | 딱 맞는 답 드림 |

쉬운 말 = 광범위 도달. 어려운 말 = 같은 직종 안에서도 일부만 이해.

**⑥ 한국 광고 소재 4단 구조 (D2C·이커머스·숏폼 검증)**

| 단계 | 시간 | 역할 |
|---|---|---|
| 1. 후크 | 첫 3초 | 시각·청각 자극 — 비교·도발·놀라움·질문 |
| 2. 공감 페인 | 3-10초 | 타겟의 구체적 상황 묘사 |
| 3. 솔루션 | 10-20초 | 우리 제품 = 그 페인의 해답 (Before/After) |
| 4. 신뢰·CTA | 마지막 5초 | 후기·수치·한정성 + 클릭 |

영상 30초 기준. **숏폼·릴스는 후크 0-3초가 절대적** — 이 구간 못 잡으면 알고리즘이 노출 안 함.

**⑦ AI 시대 카피 차별화 5가지 (2026 한국)**

| # | 항목 | 설명 |
|---|---|---|
| 1 | **사람 손길 정서** | AI 초안 → 사람이 정서·뉘앙스·금기어 검수. 한국 정서·유머·공감은 여전히 사람 우위 |
| 2 | **다변량 테스트** | 1-2개 변형이 아닌 **10-20개 동시 노출**, 알고리즘이 위너 선택. AI는 변형 생산기 |
| 3 | **브랜드 고유 보이스** | 같은 AI 도구 + 같은 프롬프트 = 모두 비슷한 카피 → 브랜드 고유 표현·금기어가 유일한 차별화 |
| 4 | **'근본이즘' 신뢰** | 트렌드 코리아 2026 키워드 — AI 모조품 의심 속 '진짜·원조·본질'이 강한 카피 |
| 5 | **AI Qualified Click 대응** | AI 검색(ChatGPT·Perplexity·네이버 AI 브리핑)이 답변에 **인용 가능한 단정형 사실 명제** 생성. 클릭 없이 브랜드 인지 |

### AIDA / PAS / 4P 카피 공식 (한국 검증)

| 공식 | 4단 | 적합 |
|---|---|---|
| **AIDA** | Attention → Interest → Desire → Action | B2B·SaaS (학습형 구매) |
| **PAS** | Problem → Agitate → Solution | B2C·이커머스 (감정 전환) |
| **4P** | Picture → Promise → Prove → Push | 광고 영상 (시각 중심) |

### 한국 카피 안티 패턴 (2026)

- **AI 초안 그대로 발행** — 같은 도구 = 같은 카피 → 차별화 0
- **영문 직역** — 한국 정서·맥락 무시
- **추상 형용사 나열** — "최고의·프리미엄·혁신적" 의미 0
- **자랑 위주** — "우리 회사 강점" (고객 관심 X)
- **어려운 한자어·전문어** — 도달률 감소
- **CTA 부재** — 다음 행동 불명확
- **사실 명제 없음** — AI 검색 인용 불가능 (2026 GEO 시대)
- **다변량 테스트 X** — 1-2개 변형으로만 운영 → 위너 발견 못함

> 💡 2026 한국 카피 차별점: **AI는 초안 도구일 뿐**. 진짜 차별은 (1) 사람 손길 정서 검수 (2) 브랜드 고유 표현 (3) AI 검색 인용 가능한 사실 명제. 이 셋이 광고 효율 차이를 만듭니다. AI가 생산을 평준화한 시대일수록 카피라이팅의 전략적 가치가 더 올라갑니다.

## 워크플로우

### 단계 1: 타겟 분석 (Target Audience Analysis)

타겟 고객의 인구통계학적 특성, 심리적 특성, 행동 패턴을 분석하여 명확한 Buyer Persona를 구축합니다. 타겟 고객이 가진 문제(Pain Point)와 욕구(Desire)를 식별합니다.

**출력 항목:** Buyer Persona 프로파일, 심리적 특성, 구매 여정 단계(인지 → 고려 → 구매 → 유지 → 확산), 페인포인트 식별, 욕구 식별

### 단계 2: 페인포인트 도출 (Pain Point Extraction)

타겟 고객이 겪고 있는 구체적인 페인포인트를 3-5개 식별하고, 각 페인포인트의 심각성과 긴급성을 평가합니다. 페인포인트를 해결하는 것이 타겟 고객에게 가져다줄 가치(Value)를 명확히 합니다.

**출력 항목:** 핵심 페인포인트 3-5개, 페인포인트 심각성 평가, 페인포인트 해결 시 가치, 페인포인트별 메시지 후크

### 단계 3: 핵심 메시지 생성 (Core Message Generation)

페인포인트와 가치 제안을 바탕으로 핵심 메시지를 생성합니다. AIDA 모델(Attention-Interest-Desire-Action)과 PAS 모델(Problem-Agitate-Solution)을 활용하여 설득력 있는 메시지 구조를 설계합니다.

**출력 항목:** 핵심 가치 제안, AIDA 모델 메시지, PAS 모델 메시지, 메시지 톤앤매너

### 단계 4: 채널별 변환 (Channel-Specific Adaptation)

핵심 메시지를 각 채널(이메일, SNS, 웹, 광고)의 특성에 맞게 변환합니다. 각 채널의 길이 제한, 형식, 사용자 행동 패턴을 고려하여 최적화된 스크립트를 생성합니다.

**출력 항목:** 이메일 스크립트(콜드 이메일, 뉴스레터, 트랜잭션 이메일), SNS 스크립트(Instagram, LinkedIn, Twitter/X, TikTok), 웹 스크립트(랜딩 페이지, 제품 상세페이지, 블로그 포스트), 광고 스크립트(검색 광고, 디스플레이 광고, 동영상 광고)

### 단계 5: A/B 테스트 제안 (A/B Testing Proposal)

각 채널별로 2-3개의 변형(Variant)을 제안하고, A/B 테스트를 통해 최적의 메시지를 찾을 수 있는 실험 계획을 수립합니다.

**출력 항목:** A/B 테스트 변형(헤드라인, CTA 버튼, 메시지 톤, 길이), 테스트 가설, 성공 지표(CTR, 전환율, 오픈률, 참여율), 실험 기간

## 사용 예시

**예시 1**: B2B SaaS 제품 타겟 스크립트
- 타겟 고객: 중소 제조업체 경영진 (CEO, 생산관리자, 연 매출 10-100억원)

- 제품: 프리미엄 반려동물 유기농 사료 (100% 유기농, 무添加剂)
- 캠페인 목표: 첫 구매 유도 (첫 구매 20% 할인)
- 채널: Instagram, 이메일, 랜딩 페이지

## 주의사항

### 필수 입력값

다음 4가지 입력값 중 하나라도 누락되면 스크립트 품질이 현저히 저하됩니다:

4. 채널 (이메일, SNS, 웹, 광고 중 최소 1개 이상)

### AI 슬롭 후처리

생성된 타겟 스크립트는 반드시 `gil:ai-slop-reviewer` 스킬을 통해 후처리해야 합니다. AI 패턴(반복적 표현, 과장된 어조, 추상적 문장)을 제거하고 실전 즉시 활용 가능한 구체적이고 명확한 스크립트로 변환합니다.

## 관련 스킬

| 스킬 | 사용 시점 |
|------|----------|
| `gil-marketing:campaign-planner` | 선행 체이닝: 타겟 스크립트 생성 전, 전체 캠페인 전략이 필요한 경우 |
| `gil-marketing:sns-content` | 사후 체이닝: 생성된 SNS 스크립트를 바탕으로 실제 SNS 포스트를 생성 |
| `gil:ai-slop-reviewer` | 사후 체이닝(필수): 생성된 타겟 스크립트의 AI 패턴을 검수하고 수정 |
| `gil-content:copywriting` | 대안: 브랜드 중심의 카피라이팅이 필요한 경우 |
