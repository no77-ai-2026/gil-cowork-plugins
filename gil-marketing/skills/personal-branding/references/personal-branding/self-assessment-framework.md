# 자기 분석 및 포지셔닝 프레임워크

> personal-branding 레퍼런스 v1.0.0

## 1. 전문성 인벤토리

개인 브랜딩의 출발점은 자신이 가진 것을 정확히 파악하는 것이다. 아래 4개 카테고리로 분류하여 작성한다.

### 기술 매트릭스

```
[기술명] | [숙련도 1-5] | [경력 연수] | [증명 가능 여부]
---------+--------------+-------------+------------------
예시:
UX 디자인        | 5 | 8년 | 포트폴리오 20건
데이터 분석      | 4 | 5년 | SQL/Python 프로젝트 10건
프레젠테이션     | 4 | 7년 | 사내 발표 50회+
팀 리더십        | 3 | 3년 | 10인 팀 매니징
```

숙련도 기준:
- 1: 기초 지식 보유 (학습/수강 수준)
- 2: 실무 적용 가능 (가이드 참고하며 수행)
- 3: 독립 수행 가능 (혼자서 완결)
- 4: 타인 교육 가능 (멘토링/강의 수준)
- 5: 업계 상위 전문가 (강연/출판/자문 수준)

### 경험 인벤토리

```
[경험 유형] | [구체적 내용] | [성과/결과] | [배운 점]
------------+---------------+-------------+-----------
프로젝트     | A사 앱 리디자인      | 전환율 35% 증가    | 사용자 리서치 중요성
강연/발표    | 스타트업 컨퍼런스 발표 | 참석자 200명       | 스토리텔링 기법
출판/기고    | IT 매거진 칼럼 연재   | 12회 연재 완료     | 마감 관리 능력
수상/인증    | UX 어워드 수상       | 업계 인지도 확보    | 포트폴리오 가치
```

### 자격/인증 목록

보유 자격증, 수료 과정, 학위를 모두 나열한다. 브랜딩에 활용 가능한 것과 부차적인 것을 구분한다.

### 성과 매트릭스

숫자로 증명할 수 있는 성과를 정리한다. 개인 브랜딩에서 가장 강력한 무기는 정량적 성과이다.

```
[성과 영역]   | [수치]        | [맥락]                    | [기간]
--------------+---------------+---------------------------+---------
매출 기여      | 연 5억 원     | 신규 채널 개척            | 2024
비용 절감      | 30% 절감      | 프로세스 자동화           | 2023-2024
팀 성장        | 3명 -> 12명   | 팀 빌딩 및 채용 주도      | 2022-2024
콘텐츠 성과    | 월 10만 PV    | 기술 블로그 운영          | 2025
```

## 2. 핵심 가치 도출 (가치 카드 정렬법)

### 진행 방법

아래 24개 가치 키워드에서 자신에게 중요한 것을 선택한다.

**가치 키워드 풀:**
```
성장 | 자유 | 안정 | 도전 | 창의 | 성실
전문성 | 협업 | 독립 | 영향력 | 진정성 | 혁신
신뢰 | 효율 | 공정 | 배려 | 열정 | 균형
탁월함 | 봉사 | 리더십 | 학습 | 소통 | 책임
```

**1차 선별**: 24개 중 10개를 고른다
**2차 선별**: 10개 중 5개를 고른다
**최종 선별**: 5개 중 3개를 고른다 -- 이것이 핵심 가치이다

### 가치 정의문 작성

선정된 3개 가치 각각에 대해 다음을 작성한다:

```
가치: [키워드]
나에게 이 가치란: [1문장 정의]
이 가치가 내 일에 반영되는 방식: [구체적 행동/습관]
이 가치가 드러나는 성과 사례: [실제 경험]
```

```
3. ____________________________

1. ____________________________
2. ____________________________
```

### Competitor (경쟁 전문가) 분석

경쟁 전문가 3-5명을 선정하여 분석한다:

```
[전문가명] | [플랫폼] | [팔로워/구독자] | [강점] | [약점] | [차별화 포인트]
-----------+----------+-----------------+--------+--------+----------------
A          | 유튜브   | 10만             | 설명력 | 빈도낮음| 실무 사례 풍부
B          | 링크드인 | 5천              | 네트워크| 깊이부족| 업계 인맥
C          | 브런치   | 3천              | 글솜씨 | 일관성 | 감성적 접근
```

### Company (나) 분석 -- 교차점 도출

```

이것이 나의 핵심 차별화 요소이다.
```

## 4. 개인 가치 제안(PVP) 작성

### PVP 공식

```
[타겟]이 [문제]를 겪을 때,
나는 [방법]으로 [결과]를 제공하는 [직함]이다.
```

### 작성 예시

```
스타트업 PM이 데이터 기반 의사결정에 어려움을 겪을 때,
나는 SQL과 시각화 도구를 활용한 대시보드 설계로
의사결정 속도를 3배 높여주는 데이터 분석 컨설턴트이다.

중소기업 대표가 온라인 마케팅 성과 측정에 어려움을 겪을 때,
나는 GA4와 광고 플랫폼 데이터를 통합 분석하여
ROAS를 150% 개선하는 퍼포먼스 마케터이다.
```

### PVP 검증 체크리스트

- [ ] 타겟이 구체적인가 (모든 사람 X, 특정 직군/상황 O)
- [ ] 문제가 실제로 존재하는가 (타겟에게 물어본 적 있는가)
- [ ] 방법이 나만의 것인가 (아무나 할 수 있는 일이 아닌가)
- [ ] 결과가 측정 가능한가 (숫자나 변화로 표현 가능한가)
- [ ] 10초 안에 말할 수 있는가 (너무 길지 않은가)

## 5. 브랜드 키워드 3개 선정

### 선정 공식

```
키워드 1: [형용사] -- 나의 스타일/인상 (예: 논리적, 따뜻한, 날카로운)
키워드 2: [전문 분야] -- 나의 영역 (예: UX 디자인, 데이터 분석, B2B 마케팅)
키워드 3: [가치/결과] -- 내가 전달하는 것 (예: 성장, 효율, 혁신)
```

### 키워드 활용법

선정된 3개 키워드는 다음에 일관되게 반영한다:
- 프로필 자기소개 (모든 플랫폼)
- 콘텐츠 주제 선정 기준
- 시각적 톤 (색상, 이미지 스타일)
- 네트워킹 자기소개 멘트

## 6. 포지셔닝 맵 작성법

### 축 설정

```
                    전문적
                      |
                      |
        [영역 A]      |      [영역 B]
                      |
   감성적 ————————————+———————————— 논리적
                      |
        [영역 C]      |      [영역 D]
                      |
                      |
                    대중적
```

### 작성 절차

1. 경쟁 전문가 3-5명을 맵에 배치한다
2. 각 전문가의 콘텐츠 톤과 타겟 수준을 기준으로 위치를 결정한다
3. 빈 공간(비어있는 영역)을 파악한다
4. 빈 공간 중 자신의 강점과 맞는 위치를 선택한다
5. 선택한 위치가 타겟 오디언스의 니즈와 일치하는지 검증한다

### 포지셔닝 맵 해석

- 전문적+논리적 (영역 B): 데이터 기반 분석, 학술적 접근, B2B 전문가
- 전문적+감성적 (영역 A): 스토리텔링 전문가, 코칭, 멘토링
- 대중적+논리적 (영역 D): 쉬운 설명, 교육 콘텐츠, 입문 가이드
- 대중적+감성적 (영역 C): 라이프스타일, 동기부여, 인플루언서

## 7. 개인 브랜드 건강도 진단 (100점)

20개 항목, 각 5점 만점으로 자가 평가한다.

### 검색 가능성 (25점)

```
[ /5 ] 이름(또는 전문가명) 검색 시 1페이지 내 관련 결과가 노출된다
[ /5 ] 대표 전문 키워드로 검색 시 내 콘텐츠가 노출된다
[ /5 ] 포트폴리오 또는 대표 콘텐츠에 접근할 수 있는 링크가 존재한다
[ /5 ] 소셜 미디어 프로필이 검색 결과 상단에 노출된다
[ /5 ] 이름을 검색했을 때 부정적 정보 없이 전문성 관련 정보만 나온다
```

### 일관성 (25점)

```
[ /5 ] 모든 플랫폼에서 같은 프로필 사진을 사용한다
[ /5 ] 자기소개 메시지가 플랫폼 간 일관된 핵심 메시지를 전달한다
[ /5 ] 콘텐츠 주제가 핵심 전문 분야를 벗어나지 않는다
[ /5 ] 시각적 톤(색상, 디자인)이 통일되어 있다
[ /5 ] 어조와 말투가 채널 간 일관성을 유지한다
```

### 전문성 증거 (25점)

```
[ /5 ] 전문 분야 관련 정기적 콘텐츠를 발행한다 (월 2회 이상)
[ /5 ] 강연, 출판, 인터뷰 등 외부 활동 이력이 있다
[ /5 ] 포트폴리오 또는 사례 연구가 공개되어 있다
[ /5 ] 관련 자격증, 인증, 수상 이력을 보유한다
[ /5 ] 업계 행사나 커뮤니티에 정기적으로 참여한다
```

### 사회적 증명 (25점)

```
[ /5 ] 추천서, 리뷰, 감사 메시지를 보유한다 (링크드인 추천 등)
[ /5 ] 전문 분야 팔로워/구독자 수가 의미 있는 수준이다
[ /5 ] 댓글, DM 등을 통한 오디언스 상호작용이 활발하다
[ /5 ] 다른 전문가로부터 인용, 멘션, 협업 요청을 받는다
[ /5 ] 강연 섭외, 기고 요청, 자문 의뢰 등 인바운드 기회가 발생한다
```

### 점수 해석

| 점수 | 등급 | 진단 |
|------|------|------|
| 80-100 | A등급 | 개인 브랜드가 잘 구축됨. 심화 전략(사고 리더십, 수익화)에 집중 |
| 60-79 | B등급 | 기본 토대 존재. 일관성과 콘텐츠 빈도 강화 필요 |
| 40-59 | C등급 | 개인 브랜드 초기 단계. 핵심 채널 1개에 집중하여 기반 구축 |
| 20-39 | D등급 | 브랜드 부재. 자기 분석부터 시작하여 PVP 수립 필요 |
| 0-19 | F등급 | 온라인 존재감 없음. 프로필 생성과 첫 콘텐츠 발행부터 시작 |

## 8. 직장인/전문가 특화 가이드

### 겸양 문화와 전문성 어필의 균형

직장인 개인 브랜딩에서 가장 큰 장벽은 "자랑하는 것 같아서 부담스럽다"는 심리이다. 다음 원칙으로 균형을 잡는다:

**허용되는 어필 방식:**
- 팀 성과 속 개인 기여를 객관적으로 서술한다: "팀이 달성한 X에서 제가 맡은 부분은 Y이다"
- 배운 점 중심으로 경험을 공유한다: "이 프로젝트에서 배운 3가지"
- 질문을 통해 전문성을 드러낸다: "이런 상황에서 여러분은 어떻게 하시나요?"
- 타인의 성과를 소개하며 자연스럽게 관여를 밝힌다

**피해야 할 어필 방식:**
- "제가 모든 것을 이끌었습니다" 식의 독점적 공로 주장
- 연봉, 직급 등 위계적 지표를 직접 언급
- 경쟁자나 동료를 깎아내리는 비교

### 회사 소속과 개인 브랜드 병행 가이드

```
가능한 것:
- 업무 관련 일반적 전문 지식 공유 (회사 기밀 제외)
- 업계 트렌드에 대한 개인 의견 표명
- 회사 공식 채널과 별도의 개인 채널 운영
- "OO 회사 소속" 명시 후 "개인 의견" 면책 문구 추가

주의할 것:
- 회사 내부 정보, 미공개 프로젝트, 고객 데이터 절대 불가
- 회사 SNS 정책/가이드라인 반드시 사전 확인
- 근무 시간 중 개인 브랜드 활동 자제
- 회사 로고, 브랜드 자산 무단 사용 금지
```

### 이직 시장에서의 개인 브랜드 활용

```
단계별 전략:

[재직 중]
- 전문 분야 콘텐츠를 꾸준히 발행하여 전문성을 축적한다
- 링크드인 프로필을 항상 최신 상태로 유지한다
- 업계 네트워크를 꾸준히 확장한다

[이직 결심 시]
- 타겟 기업/직무에 맞춰 콘텐츠 방향을 조정한다
- 포트폴리오를 최신화한다
- 헤드헌터에게 노출될 수 있는 키워드를 프로필에 추가한다

[이직 활동 중]
- 면접관이 검색할 것을 전제로 온라인 프로필을 점검한다
- 최근 3개월 콘텐츠가 전문성을 증명하는지 확인한다
- 추천서(링크드인 Recommendations)를 3개 이상 확보한다

[입사 후]
- 새 직함과 소속을 즉시 업데이트한다
- 새 환경에서의 학습과 성장을 콘텐츠로 전환한다
```
