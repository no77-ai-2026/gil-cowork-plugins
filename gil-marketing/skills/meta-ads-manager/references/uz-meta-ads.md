# UZ 시장 Meta 광고 운영 가이드

> gil-marketing | meta-ads-manager 참조 | 우즈베키스탄 한인·고려인 타겟

UZ(우즈베키스탄) 한인·고려인 또는 현지 시장을 겨냥한 Meta(페이스북·인스타그램) 광고를 운영할 때만 로드합니다. 한국 광고 계정 운영은 본문 §3 워크플로우를 그대로 사용하세요. 모든 안전 가드(§4 기본 PAUSED·쓰기 승인)는 UZ에서도 동일하게 적용됩니다.

## 1. 도달 지역·언어 타겟팅

- **지역**: Uzbekistan 전체 또는 Tashkent·Samarkand 등 도시 단위. 한인 밀집지는 타슈켄트 중심.
- **언어**: 광고세트 언어 타겟을 Russian(고려인·CIS 공용) + Uzbek(현지 청년) + Korean(한인 1세)로 분기. 단일 소재에 3언어 혼합 금지 — 언어별 광고 분리.
- **연령·관심사**: 한인 커뮤니티·K-콘텐츠·교민 서비스 관심사 레이어를 활용.

## 2. 통화·예산·결제

- Meta 광고 결제 통화는 광고 계정 설정에 종속. UZS 직접 결제 제약 시 USD 카드 계정 권장.
- 예산 제안 시 UZS·USD 환산 병기 (현지 체감 단가가 한국과 크게 다름).
- 신규 캠페인은 본문 가드대로 **PAUSED**로 생성, 환율·결제 확인 후 사용자 승인하에 활성화.

## 3. 소재·카피 현지화

- 러시아어/우즈벡어 카피는 기계 번역 후 **현지 검수 1회 필수** (gil:humanize-korean은 한국어 전용).
- 표시광고·금지표현은 한국 표시광고법 외에 UZ 현지 광고 규정도 점검 (현지 파트너 확인 권장).

## 4. 체이닝

- UZ 캠페인 진단도 한국과 동일하게 `meta-ads-analyzer` 또는 `gil-ads-audit` MCP로 연결.
- 운영 결과 텍스트 정리는 `gil:ai-slop-reviewer` 자동 체이닝.
