# 부록 E — 5 자동 모드 판단 로직

SPEC-META-ADS-001 §3.5 + 작업명세서 부록 E.1-E.5 직접 인용.

---

## E.1 자동 모드 판단 로직 (SPEC §3.5 코드블록 직접 인용)

```
파일 1개 + 노출지면만          → 단일 캠페인 진단 모드 (기본)
파일 1개 + 노출지면 + 인구통계 → 통합 분석 모드

파일 2개 + 같은 기간 + 다른 차원 → 분리 보고서 통합 모드
  (노출지면 보고서 + 인구통계 보고서)

파일 2~6개 + 다른 기간         → 다중 월 비교 모드
  (추세 분석 + 소재 라이프사이클 활성화)

파일 2~6개 + 같은 기간 + 다른 캠페인 → 다중 캠페인 일괄 모드
  (대행사용)

그 외                          → 사용자 확인 요청
```

---

## E.2 단일 캠페인 진단 모드 (기본)

**판단 조건**: 파일 1개 + 노출지면(Placement) 컬럼만 존재 (연령·성별 없음)

**활성화 항목**:
- 모듈 1-5, 7-8 완전 실행
- 모듈 3: 플랫폼·지면·소재·컨셉 차원만 (인구통계 차원 비활성화)
- 모듈 4: 광고×지면 매트릭스만 (연령·성별 축 비활성화)
- 모듈 6: 비활성화 (단일 월이므로 라이프사이클 판단 불가)

**출력 시 명시**: "인구통계 컬럼이 없어 연령·성별 분석은 제외되었습니다."

---

## E.3 통합 분석 모드

**판단 조건**: 파일 1개 + 노출지면(Placement) + 연령(Age) + 성별(Gender) 컬럼 모두 존재

**활성화 항목**:
- 모든 9개 모듈 완전 실행 (모듈 6 제외 — 단일 월)
- 모듈 3: 6개 차원 모두 + 4D 교차 분석 완전 활성화
- 모듈 4: 모든 축 조합 가능 (광고×지면, 광고×연령, 지면×성별, 연령×성별)
- 페르소나 검증 모듈 활성화 (연령×성별 교차)

**이 모드의 특징**: 가장 정밀한 진단 가능. 인구통계 기반 페르소나 검증까지 지원.

---

## E.4 분리 보고서 통합 모드

**판단 조건**: 파일 2개 + 같은 기간 + 다른 차원 (노출지면 보고서 1개 + 인구통계 보고서 1개)

**자동 처리**:
- 두 보고서의 공통 키(광고 이름 + 보고 기간)로 매칭
- 매칭 성공 행: 두 차원 결합 분석
- 매칭 실패 행: 개별 차원으로만 분석 + 사용자에게 경고

**활성화 항목**:
- 모듈 3: 개별 차원 분석은 정확하게 수행 (노출지면 측 + 인구통계 측 각각)
- 4차원 교차: 제한적 (각 보고서의 차원 조합으로만 가능)
- 4차원 교차 한계 사용자 명시 필수

**출력 시 명시**: "분리 보고서이므로 4차원 교차 분석에 제한이 있습니다. 정확한 교차 분석은 통합 보고서를 권장합니다."

**분리 보고서 매칭 키 (SPEC §3.4 + 작업명세서 H.4)**:
- 광고 이름 (Ad name)
- 보고 기간 (Reporting period: starts + ends)
- 매칭 실패 시: 사용자 알림 + 가능한 분석만 수행

## E.5 다중 월 비교 모드

**판단 조건**: 파일 2-6개 + 각 파일의 보고 기간이 다름

**자동 처리**:
- 파일을 시간순으로 정렬
- 각 기간의 일수를 계산하여 일평균으로 정규화 (30일 환산값 제공)
- 추세 분석 활성화 (월별 KPI 꺾은선 차트)
- 소재 라이프사이클 모듈 6 활성화

**활성화 항목**:
- 모듈 2: 추세 + 증감률 + 변동성 추가 산출
- 모듈 6: 소재 라이프사이클 완전 활성화
- 전 모듈: 월별 비교 시각화 추가

**정규화 방식 (작업명세서 H.3)**:
- 각 보고 기간 일수 계산 (`Reporting ends` - `Reporting starts`)
- 일평균 = 해당 기간 지표 합계 ÷ 일수
- 30일 환산 = 일평균 × 30

---

## E.6 다중 캠페인 일괄 모드

**판단 조건**: 파일 2-6개 + 같은 기간 + 다른 캠페인 이름

**주요 활용**: 대행사 실무자가 여러 클라이언트 또는 캠페인을 한 번에 진단할 때

**자동 처리**:
- 캠페인별 분리 분석 (각 캠페인 개별 9 모듈 실행)
- 통합 인사이트 산출 (공통 패턴 및 차이점 비교)
- 캠페인 간 효율 비교 (ROAS·CTR·CPC 상대적 순위)

**산출**:
- 캠페인별 개별 보고서 (각 캠페인 독립 7-Level 출력)
- 통합 대시보드 (모든 캠페인 한눈에 비교)

---

## E.7 모호한 경우 처리

위 5가지 조건에 명확히 해당하지 않는 경우:

- 판단 불가 상황을 사용자에게 설명
- 가장 가까운 모드 후보 2개 제시 + 차이 설명
- 사용자가 직접 모드 선택
- 선택된 모드로 분석 진행

**사용자 확인 요청 예시 상황**:
- 파일 3개인데 기간이 일부 겹치는 경우
- 캠페인 이름이 동일한데 기간이 다른 경우
- 인구통계 컬럼이 일부 행에만 있는 경우
