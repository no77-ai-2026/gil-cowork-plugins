## K.2 보고서 → .xlsx 변환 안내

케어밀 보고서 DOCX를 분석 가능한 .xlsx로 변환하는 절차:

### 방법 1: 메타 광고관리자에서 직접 추출 (권장)

케어밀 실계정에 접근 가능한 경우:

1. 메타 광고관리자 → 보고서 탭 → 사용자 지정 보고서
2. 분석 기간: 케어밀 보고서와 동일 기간 설정
3. 차원 선택: 광고 + 노출 위치 + 연령 + 성별 (통합 보고서 권장)
4. .xlsx로 내보내기
5. 본 스킬에 업로드

### 방법 2: DOCX 데이터로 수동 fixture 생성

케어밀 DOCX의 표 데이터를 Excel로 이식:

```
케어밀 DOCX 추출 데이터 구조 (예시):
- 광고 이름 컬럼
- 기간 컬럼 (보고 시작/종료)
- 노출 위치 컬럼
- 지출 금액, 클릭, CTR, CPC, 구매, ROAS 컬럼
- (연령·성별: 인구통계 보고서가 별도인 경우 2번째 파일로)
```

## K.4 인구통계 통합 분석 모드 검증

케어밀 fixture로 다음 시나리오를 검증합니다:

### 시나리오 1: 통합 보고서 (연령·성별 포함)

- 파일 1개 업로드 + 연령·성별 컬럼 모두 포함
- 기대 동작: 통합 분석 모드 자동 선택
- 검증 항목:
  - 4D 교차 분석 (광고×지면×연령×성별) 활성화
  - 페르소나 검증 모듈 (연령×성별 교차) 활성화
  - 🔴 AN 지면 누수 발견 → 표준 템플릿 액션 권장안 생성

### 시나리오 2: 분리 보고서 (노출지면 + 인구통계 별도)

- 파일 2개 업로드: 노출지면 보고서 + 인구통계 보고서 (같은 기간)
- 기대 동작: 분리 보고서 통합 모드 자동 선택
- 검증 항목:
  - 공통 키(광고 이름 + 기간) 매칭 성공률 확인
  - 4차원 교차 제한 안내 출력

### 시나리오 3: 다중 월 비교 (1월 + 2월)

- 파일 2개 업로드 (다른 기간)
- 기대 동작: 다중 월 비교 모드 자동 선택
- 검증 항목:
  - 일평균 정규화 (2월 = 28일, 1월 = 31일)
  - 모듈 6 소재 라이프사이클 활성화
  - ROAS 추세 꺾은선 차트 생성
