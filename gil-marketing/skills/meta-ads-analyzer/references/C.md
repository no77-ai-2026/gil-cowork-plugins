## C.1 상품 카테고리 프리셋 (8종)

| 카테고리 | 기본 관여도 | 비고 |
|---------|-----------|------|
| 식품/음료 | 저-중 | 일반 식품, 충동 구매 가능 |
| 패션/뷰티 | 저-중 | 트렌드 의존, 계절성 있음 |
| 건강기능식품/의료 | 고 | 신뢰 중요, 비교 구매 긴 편 |
| IT/디지털 제품 | 중-고 | 스펙 비교 구매, 정보 탐색 긺 |
| 가정용품/생활 | 저-중 | 반복 구매 가능, 실용 판단 |
| 교육/지식 상품 | 고 | 정보 탐색 후 결정, 관여도 높음 |
| 서비스/B2B | 고 | 장기 검토, 다수 의사결정자 |
| 기타 | 사용자 선택 | 관여도를 사용자가 직접 선택 |

**관여도 자동 추정 알고리즘** (카테고리 + 단가 + 캠페인 목표 조합):
- 저관여: (식품·패션) + (저가-중가) + (구매·트래픽)
- 중관여: (가정용품·IT) + (중가-고가) + (구매·리드)
- 고관여: (건강기능식품·교육·B2B) + (고가-프리미엄) + (리드·인지도)

> **수동 변경 허용**: 자동 추정 결과가 실제와 다를 경우 사용자가 저/중/고 중 하나로 직접 변경 가능. 변경 시 이후 모든 모듈의 등급 기준에 반영됩니다.

## C.2 상품 단가 구간 (5종)

| 구간 | 범위 | 비고 |
|------|------|------|
| 저가 | 약 3만원 이하 | 충동 구매 가능 범위 |
| 중가 | 3-10만원 | 비교 후 구매 범위 |
| 고가 | 10-30만원 | 신중 구매, 관여도 높음 |
| 프리미엄 | 30만원 이상 | 매우 신중한 의사결정 |
| 직접 입력 | 사용자 입력 | 구체적 가격 입력 시 모듈 8 정밀도 향상 |

## C.3 관여도 3단계 설명

관여도는 소비자가 구매 결정을 내리기 위해 정보를 탐색·비교하는 정도입니다.

**저관여 (Low Involvement)**
- 특징: 충동 구매, 즉각적 반응, 짧은 의사결정
- 예: 식품·음료, 저가 패션 악세서리, 생활 소모품
- ROAS 기준 (모듈 4): S: ROAS 5+ / A: 3-5 / B: 1.5-3 / C: 1-1.5 / D: 0-1 / F: 구매 0
- 노후화 판정: 2개월 연속 ROAS 50% 하락 → 노후화

**중관여 (Mid Involvement)**
- 특징: 몇 가지 대안 비교 후 구매, 중간 정도의 탐색
- 예: 가전·IT 기기, 의류 주요 품목, 건강식품
- ROAS 기준: S: 4+ / A: 2.5-4 / B: 1.2-2.5 / C: 0.8-1.2 / D: 0-0.8 / F: 구매 0
- 노후화 판정: 3개월 연속 ROAS 40% 하락 → 노후화

**고관여 (High Involvement)**
- 특징: 오랜 탐색, 가족·지인 상담, 신중한 의사결정
- 예: 고가 의료기기, 교육 강좌, B2B 서비스
- ROAS 기준: S: 3+ / A: 2-3 / B: 0.8-2 / C: 0.5-0.8 / D: 0-0.5 / F: 구매 0
- 노후화 판정: 신중 판정 (어시스트 가능성 고려, 직접 전환 ROAS만으로 판단 어려움)

## C.4 캠페인 목표 5종

| 목표 | 설명 | 모듈 영향 |
|------|------|---------|
| **구매 (Sales)** | 직접 구매 전환 최대화 | 모듈 5 누수 분석 중점, 모듈 7 학습 단계 진입 기준 적용 |
| **리드 수집 (Lead)** | 잠재 고객 정보 수집 | KPI를 구매 대신 리드 수 기준으로 전환 |

| **앱 설치 (App Install)** | 모바일 앱 다운로드 유도 | CPI(Cost per Install) 지표 추가 |

## C.5 운영 기간 4단계

| 단계 | 기간 | 해석 방향 |
|------|------|---------|
| **신규** | 1개월 미만 | 데이터 수집 단계 — 판단 유보 권장 안내 |
| **초기** | 1-3개월 | 학습 단계 — 성급한 OFF 주의, 학습 완료 여부 확인 우선 |
| **안정** | 3-12개월 | 최적화 단계 — 데이터 충분, 적극 분석·개선 가능 |
| **장기** | 12개월 이상 | 노후화 관리 단계 — 소재 라이프사이클 모듈 6 집중 적용 |

## C.6 운영 철학 3종

### 자동 신뢰형

메타 알고리즘 우선, 최소 개입 선호.

- 권장안이 🟢 보수안 위주로 출력됩니다.
- 해석: "알고리즘이 최적화 중일 수 있음" 입장을 강조.
- 해당 사용자: 광고 세트 수가 적고 단순 구조를 선호하는 운영자.

### 적극 통제형

데이터 기반 수동 조정 선호.

- 권장안이 🟡🔴 중도안·적극안 위주로 출력됩니다.
- 해석: "데이터가 명확하면 수동 조정이 효율적" 입장을 강조.
- 해당 사용자: 광고 세트별 세부 관리, 수동 입찰을 선호하는 숙련 운영자.

### 양쪽 모두 제시 (기본값)

최종 판단은 본인이 내리는 방식.

- 🟢🟡🔴 모든 강도 옵션을 균형 있게 제시합니다.
- 운영 철학에 대한 강요 없이 데이터와 양면 해석만 제공.
- 권장 기본값: 처음 사용하거나 철학이 확정되지 않은 경우.
