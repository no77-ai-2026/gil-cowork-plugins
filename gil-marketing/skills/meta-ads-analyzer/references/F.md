# 부록 F — 액션 권장안 표준 템플릿

SPEC-META-ADS-001 §3.6 + 작업명세서 부록 F.1-F.3 직접 인용.

---

## F.1 표준 템플릿 (SPEC §3.6 직접 인용)

모든 권장안은 아래 5요소를 갖추어야 합니다:

```
### [발견] {제목}

**사실**: {데이터로 확인된 내용}

**해석 (양면)**:
- A 입장 (자동 신뢰): {해석}
- B 입장 (적극 통제): {해석}

**권장 옵션**:
🟢 보수안: {액션} | {효과} | {리스크}
🟡 중도안: {액션} | {효과} | {리스크}
🔴 적극안: {액션} | {효과} | {리스크}

**의사결정 가이드**: 예산 규모·운영 기간·운영 철학별 권장
```

---

## F.2 5요소 작성 가이드

### 사실 작성 원칙

- 데이터로 확인된 수치만 기재
- 기간·비교 기준을 명확히 명시
- 해석·판단 없이 수치 자체만 서술

예시:
```
**사실**: AN(Audience Network) 지면 2개월 누적 지출 180만원, 구매 전환 0건, ROAS 0.0
```

### 양면 해석 원칙

- A 입장 (자동 신뢰): 알고리즘·데이터를 믿고 현 상태를 유지하는 관점
- B 입장 (적극 통제): 데이터에 기반해 적극 개입하는 관점
- **편향 금지**: 두 입장을 동등하게 서술. 한쪽을 더 유리하게 표현하지 않음

예시:
```
**해석 (양면)**:
- A 입장 (자동 신뢰): 메타 알고리즘이 AN 지면을 통해 퍼널 상단 인지도를 쌓고
  있을 수 있음. 단기 전환 없이도 장기 효과가 있을 가능성 존재.
- B 입장 (적극 통제): 2개월 연속 구매 전환 0건은 직접 ROI 관점에서 명확한 누수.
  예산 재배분으로 더 효율적인 지면에 집중할 수 있음.
```

### 강도별 옵션 작성 원칙

| 강도 | 특징 | 적합 상황 |
|------|------|---------|
| 🟢 보수안 | 최소 변화, 리스크 낮음 | 예산 제한·운영 초기·알고리즘 신뢰 |
| 🟡 중도안 | 부분 조정, 데이터 모니터링 병행 | 안정 운영 중, 개선 여지 확인됨 |
| 🔴 적극안 | 적극 개입, 효과·리스크 모두 큼 | 데이터 충분·숙련 운영자·명확한 손실 |

각 옵션은 3요소로 구성: `{액션} | {예상 효과} | {수반 리스크}`

예시:
```
**권장 옵션**:
🟢 보수안: AN 지면 예산 비중 현행 유지, 1개월 더 모니터링 |
          알고리즘 학습 기회 보존 | 추가 손실 가능성 잔존
🟡 중도안: AN 지면 예산 비중 50% 축소, IG Feed로 재배분 |
          누수 절반 차단 + 알고리즘 신호 유지 | 전환 최적화 일부 저해 가능
🔴 적극안: AN 지면 노출 중단 검토, 전액 IG Feed·Reels로 재배분 |
          누수 180만원 차단, 전환 효율 개선 기대 | 도달 감소 가능, 재활성화 러닝 필요
```

### 의사결정 가이드 원칙

- 예산 규모·운영 기간·운영 철학 3가지 변수에 따른 권장 강도 제시
- 단정적 "-하라" 표현 금지 → "-검토 권장", "-고려" 수준으로 표현

예시:
```
**의사결정 가이드**:
- 월 예산 100만원 이하·소규모: 🟢 보수안 (변화 리스크 최소화)
- 월 예산 300만원 이상·대행사·안정 운영: 🟡 중도안 (데이터 기반 점진 조정)
- 데이터 충분·2개월 이상 동일 패턴·적극 통제형: 🔴 적극안 (명확한 누수 차단)
```

## F.3 표현 수준 가이드 (작업명세서 F.2 직접 인용)

### ✓ 포함할 표현

- "AN 지면 ROAS 0 → 노출 OFF 검토"
- "IG 스토리 슈퍼 조합 발견 → 예산 비중 확대 방향 검토"
- "30대 여성 타겟 효율 낮음 → 해당 인구통계 제외 검토"
- "소재 노후화 징후 → 변형 소재 신규 제작 권장"

### ✗ 배제할 표현

- "광고 관리자 → 광고 세트 → 편집 → 노출 위치 → ..." (메타 UI 조작 가이드)
- "체크박스 해제하세요" (구체적 UI 조작)
- "버튼을 누르세요" (단계별 조작 안내)
- "즉시 끊어라", "바로 OFF하라" (단정적 명령)

## F.4 그룹별 톤 조정 (작업명세서 F.3 직접 인용)

### 인하우스 톤

```
"데이터상 AN 지면 ROAS 0.0이 2개월 지속되었습니다.
다음 주간 회의에서 AN 지면 노출 중단 여부 논의 권장.
보고용 근거 자료는 부록 참조."
```

### 대행사 톤

```
"AN 누수 발견. 클라이언트 보고 시 강조 포인트 3개:
(1) 2개월 연속 ROAS 0.0
(2) AN 지출 비중 전체의 23%
(3) 즉시 비중 축소 시 월 NN만원 누수 차단 가능.
슬라이드 자동 생성됨."
```

### 소규모 사업자 톤

```
"AN이라는 지면에서 광고비가 새고 있어요.
2개월 동안 여기서 구매가 한 건도 없었거든요.
다음 액션은 3가지 선택지가 있는데,
예산이 빠듯하다면 🟢 보수안부터 시도해보세요."
```

## F.5 Quick Wins 분류 기준

claude-ads v1.5.1 차용 (MIT) — 한국 시장 적용.

**Quick Win 조건**: Critical/High 심각도 + 예상 조치 시간 15분 이하

> **시간 표현 예외 (SPEC §6.3)**: "15분"은 시간 예측이 아닌 Quick Win 분류 임계값으로 허용.

| 심각도 | 설명 | 예시 |
|--------|------|------|
| Critical | 즉시 수익에 영향 | 구매 0 지면 지출 차단 |
| High | 높은 개선 여지 | 상위 20% 조합 예산 재배분 |
| Medium | 중간 개선 여지 | 소재 노후화 변형 제작 |
| Low | 장기적 개선 | Lookalike 씨앗 품질 향상 |
