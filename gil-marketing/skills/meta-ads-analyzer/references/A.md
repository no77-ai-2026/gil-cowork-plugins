# 부록 A — 사용자 그룹 차별화 매트릭스

SPEC-META-ADS-001 §3.3 + 작업명세서 부록 A 직접 인용.

---

## A.1 그룹 정의

### 🏢 인하우스 실무자

- **정의**: 광고주 회사 내부의 퍼포먼스 마케팅 담당자
- **특징**: 단일 브랜드 깊이 운영 / 사내 임원 보고용 필요 / 다른 채널과 통합 관점
- **숙련도**: 중상-상
- **사용 빈도**: 주간-월간 정기
- **민감 정보**: 사내 기밀 주의

### 🤝 대행사 실무자

- **정의**: 광고 대행사 AE, 퍼포먼스 마케터
- **특징**: 다수 클라이언트 운영 / 다양한 업종 다룸 / 시간 효율성이 가장 큰 가치
- **숙련도**: 상
- **사용 빈도**: 일일-주간
- **민감 정보**: 클라이언트별 격벽 중요

### 👤 소규모 사업자

- **정의**: 1인 운영자, 작은 브랜드 대표
- **특징**: 광고를 본업과 병행 / 예산 제한적(월 100-500만원) / 메타 광고 지식 편차 큼
- **숙련도**: 초-중
- **사용 빈도**: 월간 비정기
- **민감 정보**: 본인 데이터

---

## A.2 부가 사용자

학습자·강의 운영자는 1차 그룹에 흡수됩니다:
- 강의 운영자 → **대행사** 그룹으로 흡수 (다수 케이스 다루는 시각)
- 광고 학습자 → **소규모** 그룹으로 흡수 (친절한 해설 우선)

---

## A.3 명시 입력 UX 가이드

**원칙**: 첫 호출 시 반드시 사용자가 명시 선택 — 자동 추정 금지.

스킬이 사용자 그룹을 자동으로 추정하면:
- 잘못된 톤으로 출력될 위험 (소규모 사업자에게 사내 비즈니스 톤 → 혼란)
- 액션 강조 방식 불일치 (대행사에게 🟢 위주 → 원하는 🟡 누락)

**입력 방식**: AskUserQuestion 또는 텍스트 프롬프트에서 명시 선택.

```
"사용자 그룹을 선택해주세요:
① 인하우스 실무자 (사내 마케터)
② 대행사 실무자 (대행사 AE)
③ 소규모 사업자 (1인 운영자·대표)"
```

## A.4 그룹별 출력 차별화 매트릭스 (SPEC §3.3 직접 인용)

| 항목 | 🏢 인하우스 | 🤝 대행사 | 👤 소규모 |
|------|-----------|---------|---------|
| **최우선 가치** | 정확성·일관성 | 효율성·시각화 | 친절함·실행성 |
| **기본 출력 형식** | DOCX | HTML + DOCX | HTML + MD |
| **분석 깊이** | 표준 | 표준-심층 | 요약 |
| **액션 강조** | 🟢🟡 | 🟡 | 🟢 |
| **용어 설명** | 최소 (숙련자 가정) | 최소 (숙련자 가정) | 상세 (초심자 친화) |
| **보고 톤** | 사내 비즈니스 | 클라이언트 톤 | 친근한 해설 |

## A.5 그룹별 톤 예시 (작업명세서 부록 F.3)

### 인하우스 톤

```
"데이터상 AN 지면 ROAS가 2개월 연속 1.0 미만입니다.
다음 주간 회의에서 AN 지면 노출 중단 여부 논의 권장.
보고용 근거 자료는 부록 참조."
```

### 대행사 톤

```
"AN 지면 누수 발견. 클라이언트 보고 시 강조 포인트 3개:
(1) 2개월 연속 ROAS 0.8 이하
(2) AN 지출 비중 전체의 23%
(3) 즉시 비중 축소 시 월 NN만원 누수 차단 가능.
슬라이드 자동 생성됨."
```

### 소규모 사업자 톤

```
"AN이라는 지면에서 광고비가 새고 있어요. ROAS가 0.8인데,
이건 100원 쓰면 80원 버는 상황이에요.
다음 액션은 3가지 선택지가 있는데,
예산이 빠듯하다면 🟢 보수안부터 시도해보세요."
```

## A.6 그룹 선택과 출력 변형 동작

1. **인하우스**: Level 3 KPI 차트 → 정밀 수치 테이블 우선. Level 6 액션 → 🟢🟡만 강조.
2. **대행사**: Level 2 핵심 발견 → 클라이언트 보고용 슬라이드 포인트 3개 형식. Level 6 액션 → 🟡 중도안 중심.
3. **소규모**: Level 1 요약 → 2문장 이내 쉬운 언어. 용어 첫 등장 시 괄호 설명 추가. Level 6 액션 → 🟢 보수안 강조 + 예산별 안내.
