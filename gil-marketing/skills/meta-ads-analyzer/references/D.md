# 부록 D — 9 분석 모듈 상세 명세

SPEC-META-ADS-001 §3.1 + 작업명세서 부록 D.1-D.9 직접 인용.

---

## D.1 모듈 1 — 퍼널 분해

**트리거**: 보고서 업로드 시 자동 실행

**입력**: 보고서 데이터 (노출·클릭·구매·매출)

**산출**:
- 노출 → 클릭 → 구매 깔때기 (각 단계 절대값 + 전환율)
- 단계별 전환율: 노출→클릭(CTR), 클릭→구매(CVR), 노출→구매(전체 전환율)
- 관여도별 벤치마크 비교 (참고용, 출처 명시 필수)

**시각화**: 깔때기 차트 (HTML), 단계별 카드 (DOCX/PPTX)

**벤치마크 수치 사용 주의**: 검증된 출처(SPEC §3.6 / 케어밀 reference / 공개 벤치마크)가 있는 수치만 인용. 출처 없는 벤치마크 사용 금지 (REQ-META-ADS-022).

## D.2 모듈 2 — KPI 분해

**트리거**: 보고서 업로드 시 자동 실행

**산출 지표**:

| 분류 | 지표 | 설명 |
|------|------|------|
| 절대값 | 지출 금액 | 총 광고비 |
| 절대값 | 노출 | 광고 노출 횟수 |
| 절대값 | 클릭 | 링크 클릭 수 |
| 절대값 | 구매 | 전환(구매) 수 |
| 절대값 | 매출 | 구매 전환값 합계 |
| 효율 지표 | CTR | 링크 클릭률 (클릭/노출) |
| 효율 지표 | CPC | 클릭당 비용 |
| 효율 지표 | CVR | 클릭→구매 전환율 |
| 효율 지표 | ROAS | 광고 투자 수익률 (매출/지출) |
| 효율 지표 | AOV | 평균 주문 금액 (매출/구매수) |
| 효율 지표 | CPA | 구매당 비용 (지출/구매수) |

**다중 월 시 추가 산출**:
- 월별 추세 (꺾은선 차트)
- 전월 대비 증감률 (%)
- 변동성 지수 (안정성 지표)

## D.3 모듈 3 — 차원별 분해 (v1.1 확장)

**트리거**: 보고서 업로드 시 자동 실행

### 단일 차원 분해

| 차원 | 분류 기준 | 비고 |
|------|---------|------|
| 플랫폼 | FB / IG / AN / Threads / Messenger | 컬럼 있을 경우 |
| 노출 위치 | Feed / Story / Reels / AN / Marketplace 등 | 12 필수 컬럼 내 포함 |
| 광고 소재 | 광고 이름 기반 그룹핑 | 광고 이름 패턴 자동 분류 |
| 메시지 컨셉 | 광고 이름 키워드 기반 자동 분류 | 할인/사회증거/기능 등 |
| 연령 | Age 컬럼 기반 | 인구통계 컬럼 있을 경우 |
| 성별 | Gender 컬럼 기반 | 인구통계 컬럼 있을 경우 |

### 교차 분석 (깊은 분석) — 4D 교차

| 교차 유형 | 설명 |
|---------|------|
| **광고 × 지면 × 연령 × 성별** | 4차원 교차 (통합 보고서 시) |
| 연령 × 성별 | 페르소나 검증 |
| 광고 × 연령 | 소재별 타겟 적합도 |
| 지면 × 성별 | 지면별 성별 반응 |
| 기타 사용자 선택 조합 | 모듈 4 범용 매트릭스와 연계 |

### 표본 부족 셀 처리

- 구매 N건 미만 셀은 "표본 부족"으로 표시하고 분석에서 제외
- N의 기본값은 별도 확정 전까지 **3건**으로 적용 (미결정 사항 §7.2)
- 마스킹된 셀 비율이 25% 이상이면 사용자에게 보고서 방식 변경 권고

### 시각화

차원별 차트 (막대/도넛), 교차 매트릭스, 히트맵, 페르소나 카드

## D.4 모듈 4 — 범용 매트릭스 (v1.1 확장)

**트리거**: 보고서 업로드 시 자동 실행

### 대상

지출 1만원 이상인 행만 포함.

### 축 조합 예시

| 행 축 | 열 축 | 기본 여부 |
|------|------|---------|
| 광고 소재 | 노출 위치 | ✅ 기본값 |
| 광고 소재 | 연령 | 선택 |
| 노출 위치 | 성별 | 선택 |
| 연령 | 성별 | 선택 |
| 기타 조합 | 기타 조합 | 사용자 선택 |

### 등급 기준 (저관여 기준, SPEC §3.1)

| 등급 | 저관여 ROAS | 설명 |
|------|-----------|------|
| **S** | 5.0 이상 | 최우수 — 예산 확대 적극 검토 |
| **A** | 3.0-5.0 | 우수 — 유지 또는 점진적 확대 |
| **B** | 1.5-3.0 | 양호 — 개선 여지 있음 |
| **C** | 1.0-1.5 | 주의 — 손익분기 근방 |
| **D** | 0-1.0 | 위험 — 적극 검토 필요 |
| **F** | 구매 0 | 누수 — 즉시 검토 |

**관여도 보정** (고관여: 기준 -30%):
- S: 3.5+, A: 2.1-3.5, B: 1.0-2.1, C: 0.7-1.0, D: 0-0.7, F: 구매 0

**표본 부족 보정**:
- 구매 3건 미만 셀: "평가 불가"로 표시 (D 등급으로 자동 분류하지 않음)
- 차원이 많을수록 셀 크기 감소 → 표본 부족 셀이 많아질수록 해석 주의 경고

## D.5 모듈 5 — 누수·파레토

**트리거**: 보고서 업로드 시 자동 실행

### 누수 분석

- 구매 0인 행(광고·지면·연령·성별 조합)의 지출 절대값 합산
- 전체 지출 대비 누수 지출 비중 (%)
- 누수 상위 조합 TOP N (지출 내림차순)

### 파레토 분석 (80/20 법칙)

- 매출의 80%를 기여하는 광고·지면·소재 조합 추출
- 누적 기여율 차트 (파레토 곡선)
- "20% 최상위 조합"과 "구매 0 조합" 간 지출 비중 대비

### 시각화

누수 막대 차트, 파레토 곡선, 누수 조합 테이블

## D.6 모듈 6 — 소재 라이프사이클

**조건**: 2개 이상 월 데이터 (다중 월 비교 모드 활성화 시)

**산출**:
- 소재 제작 시점 추출 (광고 이름 패턴 인식 — 날짜 포함 패턴 우선)
- 시간 경과 vs ROAS 추이 (월별 변화)
- 노후화 패턴 자동 진단
- 변형 소재 제작 권장 시점 제안 (방향성 제시, 조작 가이드 아님)

### 노후화 판정 기준 (관여도별)

| 관여도 | 판정 기준 | 비고 |
|--------|---------|------|
| 저관여 | 2개월 연속 ROAS 50% 이상 하락 | 충동 구매 소재는 빠르게 노후화 |
| 중관여 | 3개월 연속 ROAS 40% 이상 하락 | 비교 구매 소재는 조금 더 유지 |
| 고관여 | 신중 판정 (어시스트 가능성 고려) | 직접 전환 ROAS만으로 판단 어려움 |

> **시간 표현 예외 (SPEC §6.3)**: "2개월 연속", "3개월 연속"은 데이터 분석 기준값으로 시간 예측이 아님. REQ-META-ADS-021 예외 적용.

## D.7 모듈 7 — 학습 단계 진입 가능성

**트리거**: 자동 실행

**계산 방식**:

```
주간 전환 추정 = 일평균 구매 × 7
학습 진입 갭  = 50 - 주간 전환
캠페인당 주간 전환 = 주간 전환 ÷ 광고 세트 수
```

**산출**:
- 현재 학습 단계 진입 가능성 (주 50건 기준 달성률)
- 광고 세트 분리 vs 통합 권장 방향 (갭 해소를 위한 구조 제안)
- 진입 불가 시 원인 분석 (예산·캠페인 구조·타겟 크기)

> 메타 공식 학습 단계 기준인 "주 50건 전환"은 메타 공개 정책에 근거하며, 출처: Meta Business 공식 문서 (학습 단계 관련 설명).

## D.8 모듈 8 — 예산 적정성

**조건**: 사용자가 월 광고 예산 또는 단가를 입력한 경우 활성화

**계산 방식 (하이퍼컨버전 기준)**:

```
권장 일일 예산 = 제품 단가 × 3
현재 적정성   = 현재 일일 예산 ÷ 권장 일일 예산
```

**산출**:
- 권장 일일 예산 vs 현재 일일 예산 대비 비율
- 과소 예산 시: 학습 단계 진입 어려움 경고 + 🟢🟡🔴 예산 조정 옵션
- 과잉 예산 시: 소재 다각화 권고 + 예산 효율화 방향 제안

> 출처: 작업명세서 K.2 "제품가의 3배 일일 예산 권장 (하이퍼컨버전)" — 검증된 실무 원칙.

## D.9 모듈 9 — 시나리오 시뮬레이션 (선택)

**조건**: 사용자가 시뮬레이션을 선택한 경우 (Level 7, Optional)

**산출**:
- 각 액션별 예상 매출 변화
- 예상 ROAS 변화
- 신뢰 구간 3종:
  - 낙관: 상위 20% 케이스 기준
  - 현실: 중위값 기준
  - 비관: 하위 20% 케이스 기준
- 변화 관찰 권장 기간 (기간 예측이 아닌 관찰 기준 제안)

**시뮬레이션 기반 데이터**:
- 현재 보고서 데이터의 추세 + 모듈 1-8 분석 결과
- 과거 데이터가 풍부할수록 신뢰 구간이 좁아짐 (다중 월 시 정밀도 향상)

> **시간 표현 예외 (SPEC §6.3)**: "Quick Wins 15분" 분류 기준값은 예측이 아닌 분류 임계값으로 허용.
