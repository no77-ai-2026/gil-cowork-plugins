# 브랜드 아이덴티티 (Brand Identity)

## 개요

브랜드의 전략 수립부터 네이밍, 언어 아이덴티티, 비주얼 아이덴티티, 가이드라인 문서까지 체계적으로 설계합니다.

신규 브랜드 아이덴티티 전체 설계, 기존 브랜드 리브랜딩/리뉴얼, 브랜드 네이밍 단독 개발, 슬로건/태그라인 개발, 톤앤매너 가이드 설계, 비주얼 아이덴티티 설계, 브랜드 가이드라인 문서 생성을 지원합니다.

## 트리거 키워드

브랜드 만들어줘, 브랜드 아이덴티티, 브랜드 네이밍, 이름 짓기, 슬로건, 태그라인,
톤앤매너, 브랜드 가이드라인, 로고 컨셉, CI 설계, 브랜드 전략, 브랜드 포지셔닝,
리브랜딩, 브랜드 리뉴얼, 기업 브랜딩, 브랜드 스토리, 브랜드 컬러, 브랜드 보이스

## 실전 브랜드 네이밍·아이덴티티 패턴 (한국 시장)

좋은 브랜드 이름·슬로건은 **마케팅 광고비를 줄여줍니다**. 잘못 지은 이름은 평생 광고비 2배로 갚게 됩니다.

### 6개 핵심 패턴

**① 광고비 줄이는 네이밍 5원칙 (한국 가게·회사)**

| 원칙 | 안 좋은 예 | 좋은 예 |
|---|---|---|
| **검색 친화** | "행복 카페" (검색 불가) | "당근마켓" (고유 + 카테고리) |
| **발음 용이** | "프란치스카네스" | "토스" (2-4글자 baseline) |
| **표기 일관** | "E-mart / 이마트" 혼용 | "카카오" 일관 |
| **상표 검증** | 키프리스 미확인 → 분쟁 | 출원 전 검증 필수 |
| **확장 가능성** | "OO 김밥집" → 김밥만 | "OO" → 카테고리 확장 가능 |

**한국 특화 함정**: 외래어 표기법 미준수("커피숖" X), 동음이의어 (특히 욕설 연상), 한자 변환 시 의미 변질.

**② 슬로건·매니페스토·톤앤매너 구분**

| 구분 | 길이 | 사용 채널 | 한국 사례 |
|---|---|---|---|
| **슬로건** | 10자 이내 | TVCF·옥외·SNS | "방을 구할 땐 직방" / "금융을 쉽고 편하게 토스" |
| **태그라인** | 5-15자 | 로고 하단·서명 | "오늘은 무엇이든 가능합니다" |
| **매니페스토** | 200-500자 | About·홈·브랜드북 | 회사 신념 선언문 |
| **톤앤매너** | 가이드라인 | 모든 콘텐츠 | 어조·금기어·필수어 룰북 |

**③ 한국 시장 브랜드 컨텍스트 7요소 (작업 전 필수)**

작업 시작 전 다음 항목 수집:
- 사업 영역 (필수)
- 타깃 고객 (연령·성별·라이프스타일·핵심 니즈)
- 경쟁 브랜드 2-3개 (벤치마크)
- 브랜드 방향 (이미지 키워드 3개)
- 기존 확정 요소 (이미 정해진 이름·컬러·로고)
- 사업 단계 (스타트업·중기·대기업)
- 예산·인력 (네이밍만 vs 풀 패키지)

**④ 네이밍 SMILE 평가 5축**

| 축 | 평가 |
|---|---|
| **S**imple | 짧고 단순한가 |
| **M**eaningful | 의미가 명확한가 |
| **I**nteresting | 호기심·기억을 자극하는가 |
| **L**egal | 상표·도메인 충돌 없는가 |
| **E**ndurance | 5-10년 후에도 유효한가 |

각 5점 척도 → 25점 만점. 후보 **15개 이상 생성 → SMILE 평가 → TOP 3** 추천. 한국 사업자등록 명칭·키프리스 검색 필수.

**⑤ 카테고리별 톤앤매너 매트릭스 (한국)**

| 카테고리 | 톤 | 한국 사례 |
|---|---|---|
| 식품·F&B | 친근·정겨움·푸근함 | 본죽·삼립·이춘무 메밀국수 |
| 패션·뷰티 | 트렌디·세련·자유로움 | 무신사·올리브영·스타일난다 |
| IT·SaaS | 명료·전문·간결 | 토스·당근마켓·노션 |
| 교육 | 신뢰·따뜻·격려 | 콴다·산타토익·해커스 |
| B2B | 격식·정확·전문 | NHN·삼정 KPMG·삼성 SDS |
| 라이프스타일 | 진정성·차분 | 마켓컬리·오늘의집 |

카테고리 톤과 다른 방향 = 의도적 차별화 시그널 (예: 토스가 핀테크의 격식을 깸).

**⑥ 0원으로 시작하는 브랜딩 5단계 (초기 스타트업)**

1. **포지셔닝 한 줄 확정**: "OO하는 OO" 공식 (방 구할 땐 직방 / 금융을 쉽고 편하게 토스)
2. **인스타·블로그·X 무료 채널 일관 운영**: 톤·시각 동일
3. **고객 피드백 즉시 반영**: 페르소나 가설 → 검증 사이클
4. **PR·바이럴 우선**: 광고비 0원, 후기·리뷰·언론 기사 노출
5. **체화된 브랜드 표현**: 창업자 본인이 SNS·인터뷰로 직접 발화

### 한국 사업자 등록 시 추가 체크리스트

- [ ] 키프리스 상표 검색 (출원·등록 양쪽)
- [ ] 도메인 .com / .kr / .co.kr 확보
- [ ] 카카오톡 채널 ID 사용 가능 여부
- [ ] 인스타·X·유튜브 핸들 동일 확보
- [ ] 비속어·연관 검색어 (네이버·구글) 점검
- [ ] 다른 언어 (영어·중국어·일본어) 발음 시 문제 없는지

> 💡 한국 브랜딩 가장 자주 실패: **(1) 검색 안 되는 일반 명사 (2) 표기 혼용 (3) 상표 미검증 → 분쟁**. 이름 짓고 광고 2개월 후 상표 분쟁으로 전부 폐기하는 경우 흔합니다.

## 워크플로우

### 사전 수집: 브랜드 컨텍스트 체크리스트

작업 시작 전 다음 항목을 확인합니다. 필수 항목이 누락되면 3가지 방향을 제안하여 선택을 유도합니다.

| 항목 | 필수 | 설명 |
|------|-----|------|
| 사업 영역 | 필수 | 제품/서비스 종류, 업종 |
| 타깃 고객 | 권장 | 연령, 성별, 라이프스타일, 핵심 니즈 |
| 경쟁 브랜드 | 권장 | 벤치마크할 브랜드 2-3개 |
| 브랜드 방향 | 권장 | 원하는 이미지 키워드 |
| 기존 확정 요소 | 선택 | 이미 정해진 브랜드명, 컬러, 로고 등 |

### 1단계: 시장 분석 및 경쟁사 벤치마킹

해당 업종의 시장 현황을 분석하고 경쟁 브랜드 3개 이상을 벤치마킹합니다. 경쟁사 대비 차별화 기회를 도출합니다.

### 2단계: 브랜드 전략

포지셔닝맵 설계, 타깃 페르소나 개발, 아키타입 선택, 브랜드 에센스(미션, 비전, 핵심 가치) 정의를 수행합니다.

### 3단계: 네이밍

최소 15개 네이밍 후보를 생성하고 SMILE 평가로 정량 평가합니다. 도메인/상표 사전검증 안내를 제공하고 TOP 3를 추천합니다.

### 4단계: 언어 아이덴티티

슬로건/태그라인 개발(각 5개 이상), 브랜드 스토리 작성(300자/500자/1000자 버전), 톤앤매너 가이드 설계를 수행합니다.

### 5단계: 비주얼 아이덴티티

색상 시스템 설계(60-30-10 법칙, WCAG AA 준수), 타이포그래피 선정, 로고 컨셉 제안(2개 이상)을 수행합니다.

### 6단계: 브랜드 가이드라인 문서 생성

1-5단계 산출물을 통합하고 채널별 적용 가이드, 브랜드 용어집을 포함한 가이드라인 문서를 생성합니다.

## 사용 예시

| 요청 | 실행 범위 |
|------|----------|
| "친환경 생활용품 브랜드 만들어줘. 2030 여성 타깃, 프리미엄" | 풀 파이프라인 (1-6단계 전체) |
| "펫 케어 스타트업 브랜드 네이밍해줘" | 1-3단계 (전략+네이밍) |
| "브랜드명은 '그리너리'로 정했어. 슬로건이랑 톤앤매너 만들어줘" | 4단계 (언어 아이덴티티) |
| "프랜차이즈 카페 브랜드 리뉴얼 해줘. 기존명 '모닝빈'" | 풀 파이프라인 (기존 요소 반영) |
| "테크 스타트업 비주얼 아이덴티티 설계해줘" | 2+5단계 (전략+비주얼) |
| "패션 브랜드 가이드라인 문서로 정리해줘" | 6단계 (가이드라인 생성) |

## 주의사항

- 네이밍 후보는 반드시 15개 이상 생성하고 SMILE 매트릭스로 정량 평가합니다
- 아키타입은 주(70%) + 보조(30%) 조합으로 설계합니다
- 슬로건은 한국어/영문 각 5개 이상 제시합니다
- 색상 시스템은 60-30-10 법칙을 따르며 WCAG AA 접근성을 확보합니다
- 실제 로고 파일(.ai, .svg) 제작, 인쇄물 시안, 상표 출원 대행은 이 스킬의 범위가 아닙니다
- 도메인/상표 확인이 불가능한 경우 "[확인 필요]"로 표시하고 사용자에게 직접 확인을 권고합니다
