# UZ 브랜드 현지화 가이드

> gil-marketing v0.1.0 | UZ 시장 진출 브랜드 현지화 (트릴링구얼·문화·종교 코드)

## 개요

한국 브랜드가 우즈베키스탄(UZ) 시장에 진출 시 브랜드 아이덴티티 현지화는 단순 번역이 아닌 문화·종교·언어 환경 전체를 고려해야 합니다. 한국에서 자연스러운 브랜드명·슬로건·이미지가 UZ에서 부정적으로 받아들여질 수 있고, 그 반대도 마찬가지입니다.

---

## 1. UZ 시장 환경 (브랜드 관점)

### 인구 통계
- 인구 약 3,500만 명
- 평균 연령 28세 (젊은 시장)
- 도시화율 약 50% (타슈켄트 320만 명)
- 다언어: 우즈벡어·러시아어·카라칼팍어

### 종교
- 무슬림 약 88% (대부분 수니파)
- 이슬람 영향 일상 (식음료·의류·광고 코드)
- 정부는 세속주의 유지 (광고 가능 범위 넓음)

### 문화 코드
- 가족·공동체 중심
- 손님 환대 문화
- 어른 공경
- 한국 K-Drama·K-Pop·K-Beauty 매우 인기 → 한국산 브랜드는 프리미엄 인식

## 2. 트릴링구얼 네이밍 가이드

### 언어별 특징

| 언어 | 알파벳 | 사용 환경 | 주의 |
|------|-------|----------|------|
| 우즈벡어 (라틴) | Latin | 정부·일반 (점차 전환 중) | 일상·공식 |
| 우즈벡어 (키릴) | Cyrillic | 농촌·고령층 | 점차 사용 ↓ |
| 러시아어 | Cyrillic | 비즈니스·도시·교육 | 비즈니스 표준 |
| 영어 | Latin | IT·국제 비즈니스 | 한정적 |

### 네이밍 점검 체크리스트

```
[ ] 한국어 발음 → 러시아어 표기 부드러움 (음운 충돌 없음)
[ ] 한국어 → 우즈벡어 라틴 표기 가능성
[ ] 우즈벡어로 부정적 의미 X (현지인 검수 필수)
[ ] 러시아어로 부정적 의미 X
[ ] 이슬람 금기어 X (술·돼지·종교 모독)
[ ] .uz 도메인 가용성 확인
[ ] 우즈벡 상표 검증 (UZ 지식재산청)
[ ] Telegram 채널명 가용성
[ ] Yandex 검색 결과 확인
```

### 한국어 발음 → 러시아어 표기

| 한국어 음 | 러시아어 표기 | 예시 |
|----------|-------------|------|
| 김 | Ким | 김치 → Кимчи |
| 이 | Ли | 이씨 → Ли |
| 박 | Пак | 박사 → Пак |
| 사 | Са | 사과 → Сагва |
| ㅁ | м | 미소 → Мисо |
| ㄴ | н | 누나 → Нуна |

### 흔한 실수

- **'씨'·'쉬' 음** → 러시아어로 어색 ("Шипилла" 등)
- **이중모음** (예: ㅢ) → 러시아어 표기 어려움
- **자음 군집** → 우즈벡어로 부드럽게 변형 필요
- **유사 부정 단어 미점검** → "닭" → русск. созвучно "так" 등

## 3. 슬로건 현지화 가이드

### 길이 가이드

| 언어 | 권장 단어 수 | 권장 글자 수 |
|------|-------------|-------------|
| 한국어 | 5~7 | 15~25자 |
| 영어 | 5~7 | 30~45자 |
| 러시아어 | 4~6 | 30~45자 |
| 우즈벡어 (라틴) | 4~6 | 25~40자 |

### 트릴링구얼 슬로건 작성

원칙:
1. 한국어 마스터 슬로건 작성
2. 영어 번역 (의미·운율 보존)
3. 러시아어 번역 (UZ 비즈니스 표준)
4. 우즈벡어 번역 (UZ 정부·일반)
5. 현지인 검수 (의미·자연스러움·종교 코드)

### 예시 매핑

| 한국어 | 영어 | 러시아어 | 우즈벡어 |
|--------|------|---------|---------|
| 빛나는 매일 | Shine Every Day | Сияй каждый день | Har kuni porlang |
| 자연을 담다 | Nature in You | Природа внутри | Tabiat sizda |
| 첫 번째 선택 | Your First Choice | Ваш первый выбор | Sizning birinchi tanlovingiz |

### 회피 표현 (UZ)

- 술·와인·돼지 (이슬람 금기)
- 이성 간 노골적 표현
- 종교 풍자
- 정치적 메시지
- "최고"·"유일" 절대 표현 (UZ 광고법 제한)
- 비교 광고 (경쟁사 비방)

## 4. 색상·비주얼 코드

### 호감 색상 (UZ)

| 색상 | 의미 | 추천 용도 |
|------|------|----------|
| 청록색 | 우즈벡 전통 (도자기·돔) | 헤리티지·문화·럭셔리 |
| 녹색 | 이슬람·평화·자연 | 식품·웰빙·환경 |
| 흰색 | 순수·평화 | 럭셔리·뷰티 |
| 금색 | 부·럭셔리·태양 | 프리미엄 |
| 빨간색 | 활력·축제 | 캠페인·할인 |
| 보라색 | 럭셔리·영성 | 고급·명품 |
| 파란색 | 신뢰·하늘 | 금융·IT |

### 회피 색상

- **검정 단독**: 죽음 연상 (장례). 디자인 포인트는 OK
- **노랑 단독**: 질투·배신 인식 가능 (악센트는 OK)

### 시각 요소

- **사진**: 가족·공동체 장면 효과 (개인 장면은 한국적)
- **인물**: 우즈벡 외모 (백인계 + 아시아계 혼합) 또는 한국+UZ 혼합
- **의상**: 보수적 → 노출 X (특히 여성)
- **음식**: 할랄 인증 필요 (식품 카테고리)

## 6. 채널별 브랜딩 적용

### Telegram 채널 (UZ 1위 SNS)

- 채널명: 짧고 검색 가능 (예: @gil_uz)
- 채널 사진: 정사각 512x512 (브랜드 일관)
- 채널 설명: 우즈벡어 + 러시아어 병기
- 게시 빈도: 1일 1~3회
- 톤: 정중·친근·정보형

### Instagram (UZ)

- 한국 가이드 + 러시아어 캡션
- 해시태그: `#tashkent #uzbekistan #узбекистан #toshkent`
- 인플루언서: 미드 티어 (10~100K 팔로워)

### Uzum Market

- 제품명: 러시아어 + 우즈벡어
- 상세 설명: 러시아어 본문
- 한국산 명시 («Сделано в Корее») → 신뢰 ↑

## 7. 종교·문화 점검 체크리스트

```
[ ] 술·와인·돼지·돼지 가공품 없음
[ ] 음식의 경우 할랄 인증 (또는 문구)
[ ] 여성 노출 없음 (이슬람 보수)
[ ] 종교 상징·명칭 회피
[ ] 정치 메시지 없음
[ ] 우즈벡어 표기 의무 (광고법)
[ ] 우즈벡 어른 공경 메시지 활용 (선택)
[ ] 가족·손님 환대 가치 강조 (선택)
[ ] 한국산 프리미엄 강조 (가능 시)
```

## 8. 한국 회사 UZ 진출 단계

### 1단계: 시장 조사 (1~2개월)
- KOTRA Tashkent 사전 미팅
- 경쟁자 분석 (한국·UZ·러시아 브랜드)
- 가격대·소비자 인식 조사

### 2단계: 브랜드 현지화 (1~3개월)
- 트릴링구얼 네이밍·슬로건
- 비주얼 현지화
- 가이드라인 문서

### 3단계: 시범 운영 (3~6개월)
- Telegram 채널 개설
- 인플루언서 1~2명 협업
- 마이크로 캠페인 ($500~$2,000)

### 4단계: 본격 진출 (6~12개월)
- Uzum Market 입점
- Yandex Direct + Telegram Ads
- KOTRA 한국관 / 무역관 연계

## 9. 자주 하는 실수

1. **한국어 직역만**: 러시아어/우즈벡어 어색 → 현지인 검수 X
2. **문화 코드 무시**: 술·돼지 등 종교 금기 미점검
3. **단일 언어**: 우즈벡어만 또는 러시아어만 → 도달 제한
4. **한국 인플루언서 카피**: UZ 인플루언서 단가 1/5~1/10인데 한국 모델로
5. **고가 정책**: UZ 평균 소득 한국 1/5 미만 → 가격 재조정 필수
6. **결제 통합 부재**: 한국 결제만 → UZ Click·Payme 통합 필수
