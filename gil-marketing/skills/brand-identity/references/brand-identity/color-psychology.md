# 색상 심리학 (Color Psychology)

브랜드 컬러 시스템 설계를 위한 색상 심리학 방법론. 색상별 심리 반응, 한국 문화 특수 색상 의미, 팔레트 설계 프로세스, 접근성 기준을 포함한다.

## 왜 색상이 브랜드를 결정하는가

색상은 로고보다 먼저 인식된다. 소비자의 62-90%가 첫인상을 색상만으로 판단한다. 삼성의 파랑, 카카오의 노랑, 네이버의 초록 -- 색상이 곧 브랜드다. 색상 선택은 감정에 기반하되, 데이터로 검증해야 한다.

## 색상별 심리 반응 매트릭스

| 색상 | 긍정적 연상 | 부정적 연상 | 적합 업종 | 한국 브랜드 예시 |
|------|-----------|-----------|---------|---------------|
| **빨강** | 열정, 에너지, 긴급, 식욕 촉진 | 위험, 공격, 분노 | 식품, 미디어, 유통, 세일 | 롯데, SK, 교보문고, 요기요 |
| **파랑** | 신뢰, 안정, 전문성, 냉철 | 차가움, 보수적, 거리감 | 금융, IT, 의료, B2B | 삼성, 현대, 신한은행, 토스 |
| **초록** | 자연, 성장, 건강, 안전 | 미숙, 정체, 시기 | 친환경, 건강, 식품, 금융 | 네이버, 스타벅스코리아, 올리브영, 풀무원 |
| **노랑** | 낙관, 따뜻함, 창의, 주목 | 주의, 불안, 유치함 | 식품, 어린이, 플랫폼, 엔터 | 카카오, 이마트, 오뚜기, 바나나맛우유 |
| **보라** | 고급, 창의, 신비, 지혜 | 과장, 불안정, 허세 | 뷰티, 럭셔리, 테크 | 마켓컬리, 배민커넥트, 보라카이(여행) |
| **주황** | 친근, 활력, 재미, 행동 유도 | 싸구려, 경박 | 커머스, 식품, 스포츠, 통신 | 당근마켓, 쿠팡, SK텔레콤(T) |
| **검정** | 고급, 권위, 세련, 모던 | 무거움, 애도, 위압 | 럭셔리, 패션, 테크, 프리미엄 | 무신사, 29CM, 현대카드 |
| **흰색** | 순수, 미니멀, 깨끗, 공간감 | 공허, 차가움, 무미 | 의료, 테크, 뷰티, 프리미엄 | 애플코리아, 아모레퍼시픽, 이솝코리아 |

### 색상 온도와 브랜드 포지셔닝

```
따뜻함  <--------------------------------------->  차가움
빨강    주황    노랑    |    초록    파랑    보라
열정    친근    낙관    |    균형    신뢰    고급
충동적  <--------------------------------------->  이성적
```

따뜻한 색상은 감정적 구매(F&B, 패션)에, 차가운 색상은 이성적 판단(금융, 의료, 테크)에 적합하다.

## 한국 문화 특수 색상 의미

### 오방색 (전통 5색)

한국 문화에서 오방색은 우주의 질서를 상징하며, 전통 브랜딩에 깊은 의미를 부여한다.

| 오방색 | 색상 | 방위 | 전통 의미 | 현대 브랜딩 활용 |
|-------|------|------|---------|--------------|
| 청(靑) | 파랑/초록 | 동 | 봄, 생명, 시작 | 친환경, 성장, 혁신 |
| 백(白) | 흰색 | 서 | 순수, 결백, 진실 | 프리미엄, 미니멀, 의료 |
| 적(赤) | 빨강 | 남 | 정열, 생명, 태양 | 에너지, 열정, 축제 |
| 흑(黑) | 검정 | 북 | 지혜, 근원, 존엄 | 럭셔리, 권위, 전문성 |
| 황(黃) | 노랑 | 중앙 | 우주 중심, 귀함 | 프리미엄, 핵심 가치 |

### 한국인 색상 선호도

| 세대 | 선호 색상 | 비선호 색상 | 특징 |
|------|---------|-----------|------|
| MZ세대 (1990-2005) | 뉴트럴(베이지, 그레이), 파스텔, 블랙 | 원색 과다 사용 | 미니멀 선호, "힙한" 감성 |
| X세대 (1970-1989) | 네이비, 와인, 올리브 | 형광색, 과도한 파스텔 | 안정적이면서 세련된 색상 |
| 베이비붐 (1955-1969) | 금색, 진한 파랑, 진한 빨강 | 검정 단독 사용 | 신뢰감과 격조 중시 |

### 세대별 색상 감수성 설계 원칙

1. MZ세대 타깃: 채도를 낮추고 톤온톤 조합을 활용한다
2. 전 세대 공통: 원색보다 명도/채도를 조절한 변형색을 사용한다
3. 프리미엄 타깃: 골드/실버 계열 악센트를 활용한다
4. B2B 타깃: 네이비/그레이 기반으로 신뢰감을 구축한다

## 60-30-10 법칙

브랜드 컬러의 사용 비율을 체계적으로 관리하는 황금 비율.

| 비율 | 역할 | 사용처 | 선택 기준 |
|------|------|--------|---------|
| **60%** | 주 색상 (Primary) | 배경, 대면적, 헤더, 푸터 | 브랜드 핵심 감정을 대표 |
| **30%** | 보조 색상 (Secondary) | 섹션 구분, 강조 영역, 카드 배경 | 주 색상과 조화, 시각 변화 |
| **10%** | 강조 색상 (Accent) | CTA 버튼, 링크, 핵심 아이콘, 알림 | 시선 집중, 행동 유도 |

## 팔레트 유형 5가지

| 유형 | 색상 관계 | 시각적 느낌 | 적합 브랜드 | 한국 예시 |
|------|---------|-----------|-----------|---------|
| **단색 (Monochromatic)** | 한 색상의 명도/채도 변형 | 세련, 일관, 고급 | 미니멀, 럭셔리 | 현대카드 (블랙 계열) |
| **유사 (Analogous)** | 색상환 인접 3-4색 | 조화, 안정, 자연스러움 | 자연, 웰니스, 식품 | 올리브영 (초록+연두) |
| **보색 (Complementary)** | 색상환 반대편 | 강한 대비, 에너지 | 스포츠, 엔터, 식품 | 배달의민족 (민트+분홍) |
| **분할보색 (Split-Comp)** | 보색 양옆 2색 | 대비+안정의 균형 | 다용도, 교육 | -- |
| **삼색 (Triadic)** | 색상환 120도 간격 | 활발, 다채로움, 유쾌 | 어린이, 크리에이티브 | 카카오프렌즈 (노랑+파랑+분홍) |

## 6단계 팔레트 설계 프로세스

### 1단계: 감정 키워드 추출

브랜드 에센스(미션, 비전, 핵심 가치)에서 감정 키워드 3-5개를 추출한다.
- 예: "신뢰+혁신+따뜻함" -> 파랑 계열 주색 + 주황 계열 강조색

### 2단계: 경쟁사 색상 분석

- 경쟁사 3개 이상의 주 색상을 조사한다
- 동일 색상 회피 또는 의도적 차별화 전략을 선택한다
- 업종 내 색상 포화도를 확인한다 (예: 금융 = 파랑 포화)

### 3단계: 주 색상 결정

- 심리 반응 매트릭스에서 감정 키워드와 최적 매칭되는 색상을 선택한다
- 한국 문화 특수 의미를 교차 확인한다
- 타깃 세대의 색상 선호도를 반영한다

### 4단계: 팔레트 유형 결정

- 브랜드 성격에 맞는 팔레트 유형을 선택한다
- 60-30-10 비율로 Primary/Secondary/Accent를 배분한다
- Neutral 색상(흰색, 회색, 검정 계열)을 추가한다

### 5단계: 접근성 검증

- WCAG AA 이상 대비율을 확보한다 (아래 접근성 기준 참조)
- 색맹 시뮬레이션을 수행한다
- 그레이스케일 변환 시 구분 가능성을 확인한다

### 6단계: 적용 시뮬레이션

- 로고, 웹사이트, 앱, SNS, 인쇄물, 명함 등 모든 터치포인트에서 테스트 결과를 기술한다
- 다크모드 대응 색상을 정의한다
- 배경색 위 텍스트 가독성을 확인한다

## WCAG 접근성 대비율 기준

| 기준 | 본문 텍스트 | 대형 텍스트 (18pt+) | 비텍스트 UI |
|------|-----------|-------------------|-----------|
| **WCAG AA** (최소) | 4.5:1 | 3:1 | 3:1 |
| **WCAG AAA** (권장) | 7:1 | 4.5:1 | -- |

### 접근성 체크 항목

- [ ] 모든 텍스트-배경 조합이 AA 이상 대비율을 충족한다
- [ ] 적록 색맹(전체 남성 8%) 시뮬레이션을 통과한다
- [ ] 청황 색맹 시뮬레이션을 통과한다
- [ ] 그레이스케일 변환 시 모든 색상이 구분 가능하다
- [ ] 다크모드 팔레트의 대비율도 AA 이상을 충족한다
- [ ] CTA 버튼의 강조색이 배경과 충분히 구분된다

각 색상에 대해 다음 형식으로 명세한다.

```
## 브랜드 컬러 시스템

### Primary Color -- [브랜드 고유 이름]
- HEX: #XXXXXX
- RGB: R, G, B
- CMYK: C, M, Y, K
- Pantone: XXXX C (참고값)
- 사용처: 로고, 주요 배경, 헤더, 대표 그래픽
- 비율: 전체 면적의 60%

### Secondary Color -- [브랜드 고유 이름]
- HEX: #XXXXXX
- RGB: R, G, B
- CMYK: C, M, Y, K
- Pantone: XXXX C (참고값)
- 사용처: 섹션 배경, 카드, 보조 그래픽
- 비율: 전체 면적의 30%

### Accent Color -- [브랜드 고유 이름]
- HEX: #XXXXXX
- RGB: R, G, B
- CMYK: C, M, Y, K
- Pantone: XXXX C (참고값)
- 사용처: CTA 버튼, 링크, 알림 뱃지, 핵심 아이콘
- 비율: 전체 면적의 10%

### Neutral Colors
- White: #FFFFFF -- 기본 배경
- Light Gray: #F5F5F5 -- 섹션 분리
- Mid Gray: #9E9E9E -- 비활성 텍스트
- Dark Gray: #333333 -- 본문 텍스트
- Black: #000000 -- 제목, 강조

### 사용 금지 규칙
- 주 색상 위에 보조 색상 텍스트를 올리지 않는다
- 강조 색상은 전체 면적의 10%를 초과하지 않는다
- [특정 색상 조합]은 접근성 미달로 금지한다
- 브랜드 색상을 임의로 투명도 조절하여 새로운 색상을 만들지 않는다
- 그라데이션은 동일 색상 계열 내에서만 허용한다
```
