# 네이밍 방법론 (Naming Methodology)

브랜드명 개발을 위한 체계적 네이밍 방법론. 12가지 기법, SMILE 정량 평가, 한국어 음운론 분석, 도메인/상표 사전검증 절차를 포함한다.

## 왜 체계적 네이밍이 필요한가

좋은 이름은 우연히 떠오르지 않는다. "카카오"는 전 세계 누구나 발음할 수 있는 음식 이름에서, "토스"는 쉽게 던진다는 동작에서, "당근마켓"은 "당연히 근처"의 축약에서 탄생했다. 성공한 브랜드명 뒤에는 반드시 전략적 사고가 있다.

## 네이밍 기법 12가지

### 1. 서술형 (Descriptive)

- **방법**: 제품/서비스의 기능이나 특성을 직접 설명하는 이름
- **장점**: 즉시 이해 가능, 마케팅 비용 절감
- **단점**: 차별화 어려움, 상표 등록 난이도 높음, 사업 확장 시 제약
- **한국 예시**: 배달의민족 (배달 서비스), 야놀자 (놀자의 변형, 여행/숙박), 직방 (직접 찾는 방)
- **글로벌 예시**: YouTube (너의 TV), PayPal (지불 친구)

### 2. 환기형 (Evocative)

- **방법**: 브랜드 가치를 간접적으로 암시하거나 감정을 환기하는 이름
- **장점**: 감정적 연결 형성, 상표 등록 용이, 사업 확장에 유연
- **단점**: 의미 전달에 시간과 마케팅 투자 필요
- **한국 예시**: 쿠팡 (쿠킹+쇼핑의 결합 이미지), 토스 (쉽게 던지다 -> 간편 금융), 마켓컬리 (컬리 -> 신선하게 말리는 이미지)
- **글로벌 예시**: Amazon (거대함, 무한), Apple (단순함, 신선)

### 3. 신조어 (Coined / Invented)

- **방법**: 완전히 새로운 단어를 만들어 브랜드명으로 사용
- **장점**: 고유성 최대, 상표 등록 최적, 글로벌 확장 용이
- **단점**: 의미 학습 비용 높음, 초기 인지도 구축에 시간 소요
- **한국 예시**: 카카오 (코코아에서 변형), 네이버 (Navigate + er), 싸이월드 (사이버 + 월드)
- **글로벌 예시**: Kodak, Xerox, Spotify

### 4. 합성어 (Compound / Portmanteau)

- **방법**: 두 개 이상의 단어를 결합하여 새로운 의미를 만듦
- **장점**: 의미 전달 + 고유성의 균형, 스토리텔링 용이
- **단점**: 길어질 수 있음, 발음 어색할 위험
- **한국 예시**: 당근마켓 (당연히+근처+마켓), 오늘의집 (오늘+의+집), 여기어때 (여기+어때)
- **글로벌 예시**: Facebook (얼굴+책), Instagram (즉석+전보), Netflix (인터넷+영화)

### 5. 두문자 (Acronym)

- **방법**: 이니셜이나 축약어로 구성
- **장점**: 짧고 기억하기 쉬움, 글로벌 발음 동일
- **단점**: 의미 전달 약함, 이미 포화 상태, 초기 인지도 구축 어려움
- **한국 예시**: LG (Lucky-Goldstar), SK (Sunkyong), KT (Korea Telecom), CJ (Cheil Jedang)
- **글로벌 예시**: IBM, BMW, HP

### 6. 인명형 (Eponymous)

- **방법**: 창업자 이름, 인물, 또는 의미 있는 고유명사에서 차용
- **장점**: 신뢰감과 유산 전달, 스토리텔링 자산
- **단점**: 창업자 리스크, 글로벌 발음 문제 가능
- **한국 예시**: 삼성 (이병철 -- 세 개의 별), 현대 (정주영 -- 현재의 시대), 아모레퍼시픽 (아모레 -- 사랑)
- **글로벌 예시**: Chanel, Ford, Adidas (Adi Dassler)

### 7. 외국어형 (Foreign Language)

- **방법**: 다른 언어에서 의미 있는 단어를 차용하거나 변형
- **장점**: 이국적 느낌, 의미 레이어 추가, 프리미엄 이미지
- **단점**: 발음 어려울 수 있음, 원어 의미와 괴리 가능
- **한국 예시**: 무신사 (무진장 신발을 사다 -> 영어식 느낌), 마켓컬리 (Kurly -> 영어 curly 변형), 뷰노 (VUNO -> 라틴어 느낌)
- **글로벌 예시**: Volvo (라틴어 "굴러가다"), Audi (라틴어 "듣다"), Lego (덴마크어 "잘 놀다")

### 8. 의성어/의태어 (Onomatopoeia)

- **방법**: 소리나 움직임을 이름으로 표현
- **장점**: 감각적, 기억 용이, 브랜드 개성 강함
- **단점**: 진지한 업종에 부적합할 수 있음
- **한국 예시**: 뚜레쥬르 (프랑스어 "매일" + 뚜레 소리감), 빙그레 (빙그레 웃다), 새우깡 (깡 소리)
- **글로벌 예시**: Zoom (확대 소리), Snap (찰칵), Buzz (윙윙)

### 9. 은유형 (Metaphor)

- **방법**: 다른 영역의 이미지를 차용하여 브랜드 가치를 비유
- **장점**: 풍부한 연상 작용, 스토리텔링 강점
- **단점**: 은유가 통하지 않는 문화권 존재
- **한국 예시**: 쏘카 (So Car -- 그래서 자동차), 플립 (Flip -- 뒤집다, 관점 전환), 리디 (Ridi -- Read + i)
- **글로벌 예시**: Red Bull (에너지), Jaguar (속도/우아함), Amazon (광활한 강)

### 10. 접두/접미 변형 (Prefix/Suffix)

- **방법**: 기존 단어에 접두사/접미사를 붙여 새로운 브랜드명 생성
- **장점**: 의미 확장 + 고유성 확보, 패밀리 브랜드 확장 용이
- **단점**: 접미사 트렌드에 따라 진부해질 수 있음
- **한국 예시**: 카카오뱅크 (카카오+뱅크), 네이버페이 (네이버+페이), 쿠팡이츠 (쿠팡+이츠)
- **일반 패턴**: -ly, -ify, -io, -er, -lab, -hub, re-, un-, pre-

### 11. 절단형 (Truncation)

- **방법**: 단어를 줄이거나 철자를 의도적으로 변형
- **장점**: 현대적, 간결, 디지털 친화적
- **단점**: 원래 의미 손실 가능, 오타로 오해받을 수 있음
- **한국 예시**: 삼전 (삼성전자 축약, 비공식), 엘지 (LG의 한국어 발음), 현카 (현대카드 축약)
- **글로벌 예시**: Tumblr (tumbler), Flickr (flicker), Lyft (lift)

### 12. 영숫자형 (Alphanumeric)

- **방법**: 숫자, 기호를 포함한 브랜드명
- **장점**: 테크 느낌, 강한 차별화, 숫자의 상징적 의미 활용
- **단점**: 검색 어려움, 구두 전달 시 혼란
- **한국 예시**: T1 (SKT 게이밍), 29CM (29센치미터), 11번가 (11번째 거리)
- **글로벌 예시**: 7-Eleven, 3M, Y Combinator

## SMILE 평가 매트릭스

네이밍 후보를 정량적으로 평가하는 5기준 매트릭스. 각 기준 1-5점, 합격 기준 총점 20/25 이상.

| 기준 | 평가 질문 | 1점 | 3점 | 5점 |
|------|---------|-----|-----|-----|
| **S**imple (단순성) | 3초 안에 발음 가능한가? 철자를 물어볼 필요가 없는가? | 5음절 이상, 발음 어려움 | 3-4음절, 약간의 설명 필요 | 2음절, 즉시 발음 가능 |
| **M**emorable (기억성) | 한 번 듣고 다음 날 기억할 수 있는가? | 타 브랜드와 혼동 | 반복 노출 시 기억 | 한 번에 각인 |
| **I**mage-evoking (연상력) | 구체적 이미지나 감정이 떠오르는가? | 무미건조 | 약한 연상 | 즉각적 시각/감정 환기 |
| **L**egally available (법적 가용성) | 상표 등록 가능한가? 도메인 확보 가능한가? | 동일 상표 존재 | 유사 상표 존재, 협의 가능 | 완전 가용 |
| **E**xpandable (확장성) | 사업 확장 시 이름이 제한하지 않는가? | 특정 제품에 한정 | 관련 카테고리 확장 가능 | 모든 사업에 범용 |

### SMILE 평가 기록 양식

```
| 후보명 | S | M | I | L | E | 합계 | 판정 |
|--------|---|---|---|---|---|------|------|
| 후보 A | 4 | 5 | 4 | 4 | 5 | 22   | 합격 |
| 후보 B | 3 | 4 | 5 | 3 | 4 | 19   | 불합격 |
| 후보 C | 5 | 5 | 3 | 5 | 4 | 22   | 합격 |
```

## 한국어 음운론 분석

### 한글 초성별 인상

브랜드명의 첫 소리가 주는 심리적 인상을 이해하고 의도에 맞게 활용한다.

| 초성 그룹 | 음소 | 인상 | 적합 업종 | 예시 |
|----------|------|------|---------|------|
| 파열음 (예사소리) | ㄱ, ㄷ, ㅂ | 단단함, 결단력 | 테크, 건설, 금융 | 구글코리아, 다노, 배민 |
| 파열음 (거센소리) | ㅋ, ㅌ, ㅍ | 강렬함, 힘 | 스포츠, 에너지, 게임 | 카카오, 토스, 포스코 |
| 파열음 (된소리) | ㄲ, ㄸ, ㅃ | 강인함, 거침 | 캐주얼 식품, 엔터 | 빙그레, 뚜레쥬르 |
| 마찰음 | ㅅ, ㅆ, ㅎ | 부드러움, 정교함 | 뷰티, 패션, 라이프스타일 | 설화수, 쏘카, 한섬 |
| 파찰음 | ㅈ, ㅊ | 산뜻함, 경쾌함 | 식음료, 유통 | 제주항공, 차이 |
| 비음 | ㄴ, ㅁ | 따뜻함, 안정감 | 식품, 건강, 가족 | 네이버, 마켓컬리 |
| 유음 | ㄹ | 유연함, 세련됨 | 라이프스타일, 프리미엄 | 리디, 롯데 |

### 음절 수 권장

| 음절 수 | 평가 | 근거 | 예시 |
|---------|------|------|------|
| 2음절 | 최적 | 한국인 호칭 습관과 일치, 즉시 기억 | 삼성, 현대, 토스, 쿠팡 |
| 3음절 | 우수 | 리듬감 형성, 대부분의 성공 브랜드 | 카카오, 네이버, 무신사 |
| 4음절 | 양호 | 최대 한계, 의미 전달력 높음 | 배달의민족, 오늘의집, 마켓컬리 |
| 5음절+ | 주의 | 축약 가능성 확보 필수 | 배달의민족 -> 배민 |

### 한영 혼합 네이밍 규칙

1. 한글과 영문의 발음이 자연스럽게 연결되어야 한다
2. 한글 -> 영문 전환 시 로마자 표기법을 따르되, 발음 우선 원칙을 적용한다
3. 영문 브랜드명의 한글 표기가 직관적이어야 한다 (예: Coupang -> 쿠팡)
4. 한영 병기 시 어느 쪽이 주 표기인지 명확히 정한다
5. 도메인은 영문 기준으로 확보하되, 한글 도메인(.kr)도 고려한다

## 도메인/상표 사전검증 체크리스트

### 도메인 확인

| 우선순위 | 도메인 패턴 | 비고 |
|---------|-----------|------|
| 1순위 | brandname.com | 글로벌 표준, 가격 높을 수 있음 |
| 2순위 | brandname.co.kr | 국내 사업 기본 |
| 3순위 | brandname.kr | 국내 대안 |
| 4순위 | brandname.co / .io | 테크 브랜드 대안 |
| 5순위 | getbrandname.com / trybrandname.com | .com 불가 시 대안 |

### KIPRIS 상표 검색 절차

1. KIPRIS (한국특허정보원, kipris.or.kr) 접속
2. 상표 검색 -> 상표명 입력
3. 니스 분류(해당 업종 클래스) 선택
4. 동일/유사 상표 존재 여부 확인
5. 출원 중인 상표도 반드시 확인 (등록 전이라도 선출원 우선)
6. 유사 상표 발견 시 전문 변리사 상담 권고

### SNS 핸들 가용성

| 플랫폼 | 확인 사항 |
|--------|---------|
| Instagram | @brandname 가용 여부 |
| X (Twitter) | @brandname 가용 여부 |
| YouTube | youtube.com/@brandname 가용 여부 |
| 네이버 블로그 | blog.naver.com/brandname 가용 여부 |
| 카카오톡 채널 | 채널명 중복 여부 |

### 검색 결과 확인

1. 네이버 검색: 브랜드명 입력 -> 동일/유사 사업자 존재 여부
2. 구글 검색: 브랜드명 입력 -> 글로벌 동명 브랜드 여부
3. 부정적 연상 확인: 브랜드명이 다른 언어에서 부정적 의미를 갖지 않는지 확인 (최소 영/중/일)
4. 검색 결과 1페이지에 경쟁 정보가 없으면 SEO 관점에서 유리
