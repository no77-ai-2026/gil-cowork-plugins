# 브랜드 가이드라인 템플릿 (Brand Guide Template)

브랜드 아이덴티티 설계 결과물을 통합 문서로 정리하는 마크다운 템플릿. 각 섹션에 작성 지침과 fill-in-the-blank 형식을 포함한다.

## 템플릿 사용법

1. 아래 템플릿을 `_workspace/05_brand_guideline.md`에 복사한다
2. 대괄호 `[___]` 부분을 실제 내용으로 채운다
3. 선택 항목은 해당 없으면 삭제한다
4. 1-4단계 산출물의 내용을 통합하여 작성한다

---

## 템플릿 본문

```markdown
# [브랜드명] 브랜드 가이드라인

> 버전: 1.0 | 작성일: [YYYY-MM-DD] | 작성자: [작성자/팀명]
> 이 문서는 [브랜드명]의 모든 커뮤니케이션에 적용되는 공식 가이드라인이다.
```
