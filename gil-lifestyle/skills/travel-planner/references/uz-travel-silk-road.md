# UZ Silk Road 여행 가이드 (Tashkent → Samarqand → Buxoro → Khiva)

> 한국 여행객·UZ 거주자를 위한 Silk Road 종합 여행 가이드

## Silk Road 개요

UZ는 고대 실크로드의 중심. **Samarqand, Buxoro (Bukhara), Khiva** 3대 도시 + Tashkent (현대 수도) = UZ 여행 4대 거점.

```
[추천 동선]
Tashkent (도착) → Samarqand → Buxoro → Khiva → Tashkent (출국)
또는 역순
```

---

## 시즌

### 봄 (3~5월) ★★★★★

```
- 기온: 15~25도
- 사막 꽃·과일
- 사진·여행 최적
- 가격: 중간
- 추천: 강력
```

### 여름 (6~8월) ★★

```
- 기온: 35~45도 (사막 더 ↑)
- 매우 더움
- 호텔 에어컨 필수
- Khiva·Buxoro 사막 여행 어려움
- 추천: 불편
```

### 가을 (9~11월) ★★★★★

```
- 기온: 15~25도
- 수확 (멜론·수박·석류·포도)
- 음식 최고
- 가격: 중간
- 추천: 강력
```

### 겨울 (12~2월) ★★★

```
- 기온: -5~10도
- 한산·할인
- Tashkent·Samarqand 가능, Khiva 추움
- 추천: 예산·한산 선호
```

## 한국에서 UZ 여행

### 비자

```
[한국인 무비자]
- 30일 (한국·UZ 협정)
- 도착 시 입국 도장
- 연장: 한국 1회 가능 ($300~$500)

[필요 서류]
- 여권 (6개월 유효)
- 왕복 항공권
- 호텔 예약 (입국심사 일부)
```

### 항공

```
[직항]
- 인천 → Tashkent (8시간)
- 항공사: 아시아나·대한항공·Uzbekistan Airways
- 가격 왕복: 80~150만 원 (시즌)

[경유]
- 베이징·이스탄불·두바이 경유
- 가격: 60~120만 원
- 시간 ↑

[Tashkent → Samarqand·Buxoro·Khiva 국내선]
- Uzbekistan Airways
- $50~$150 편도
```

### 환전·결제

```
[UZ 도착 후 환전]
- USD → UZS (가장 효율)
- 공항·은행·일부 호텔
- 시내 환전소 더 유리

[ATM]
- 국제 카드 일부 가능
- Tashkent 多, 지방 적음
- 인출 한도 일부

[현금]
- UZS 현금 필수 (시장·소형)
- 큰 금액 USD 권장
```

### SIM 카드

```
[도착 시]
- 공항 SIM 부스 (Beeline·Ucell)
- 여권 등록
- 데이터 패키지: 1~7일 $5~$15

[준비]
- 한국 로밍 비싸 → UZ SIM 추천
- 여행자용 데이터 (Yandex Go·맵·번역)
```
