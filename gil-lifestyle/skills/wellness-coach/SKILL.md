## 독립 실행 워크플로우

전략 가이드 파일 없이도 다음 단계로 실행 가능합니다:

1. **요청 유형 파악**: 운동/식단/육아/시니어 케어 중 주 관심사 확인
2. **개인 조건 수집**: 나이, 건강 상태, 목표, 일정(시간 가용성), 선호도 파악
3. **맞춤 계획 설계**: 주차별 운동 루틴, 영양 균형 식단표, 아이 발달 활동 또는 복지 서비스 목록 작성
4. **주의 사항 안내**: 운동 시 부상 예방, 식단 시 영양 균형, 육아 시 발달 기준 편차 안내
5. **실천 지원**: 장보기 목록, 주간 체크리스트, 진도 모니터링 방법 제공

## 2026 한국 건강·식단 트렌드 컨텍스트

식품음료신문, 식품외식경제, 보그 코리아 2026 웰니스 트렌드 반영:

### 2026 식단 관리 핵심 트렌드

| 트렌드 | 설명 |
|--------|------|
| 건강 + 편리미엄 | 건강하면서 간편한 '한 그릇 음식' 주류화 |
| 자기만족 건강식 | 완벽한 건강식보다 죄책감 줄인 '나만의 건강식' |
| 단백질 + 식이섬유 | 2026 건강 식단의 기본 공식. 장 건강·혈당 조절·포만감 |
| 뇌 건강 식단 | MIND 식단 주목. 고령화 + 인지 기능 유지 수요 증가 |
| CGM 기반 식품 선택 | 연속혈당측정 데이터로 개인별 혈당 반응 최적 식품 선택 |
| 비빔밥·샐러드 성장 | 샐러드(+22.2%), 비빔밥(+13.7%), 덮밥(+8.2%) 급성장 |

### 2026 웰니스 트렌드 6가지 (보그 코리아)

1. **슬로우 웰니스**: 빠름 대신 지속 가능한 건강 루틴
2. **디지털 디톡스**: 스크린 피로 대응 오프라인 활동 증가
3. **수면 최적화**: 수면의 질 데이터 기반 관리
4. **마이크로바이옴**: 장 건강 = 전신 건강 인식 확산
5. **멘탈 웰니스**: 심리 건강 관리의 일상화
6. **하이브리드 피트니스**: 온라인+오프라인 혼합 운동

### 한국 특화 건강 정보

- **보건복지부 영양 기준**: 한국인 영양소 섭취 기준(KDRIs) 기반 식단 설계
- **체질량지수(BMI)**: 아시아인 기준 과체중 ≥23, 비만 ≥25 (서구 기준과 상이)
- **건강보험 활용**: 국가 건강검진(2년 1회), 암 검진 제도 안내 포함 권장

## 문제 해결

| 상황 | 해결 방법 |
|------|-----------|
| 예산 정보 미제공 | 식단은 1인 주간 식비 예산을 알려주시면 그에 맞는 장보기 목록을 작성합니다 |
| 일정 조건 부족 | 하루 운동 가능 시간(예: 30분)만 알려주셔도 맞춤 루틴을 설계합니다 |
| 기저 질환 있음 | 질환명을 알려주시면 주의 사항을 포함하되, 처방은 의료진에게 받으세요 |
| 아이 발달 편차 | 발달 기준은 참고용이며, 걱정되시면 소아청소년과 전문의와 상담하세요 |

## 이 스킬을 사용하지 말아야 할 때

- **의료 진단이나 처방이 필요한 경우** → 반드시 의료 전문가(의사, 영양사, 물리치료사)에게 상담하세요. 이 스킬은 일반적인 건강 정보를 제공하며 의료 행위를 대체하지 않습니다
- **정신건강 위기 또는 자해·자살 위험** → 자살예방상담전화 1393, 정신건강 위기상담전화 1577-0199로 즉시 연락하세요
- **영유아 응급 상황** → 119 또는 소아응급실로 즉시 이동하세요
- **노인 장기 요양 등급 신청** → 국민건강보험공단(1577-1000)에 직접 신청해야 합니다
