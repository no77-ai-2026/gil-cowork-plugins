## 의사결정 플로우차트

```
1. UZ 거주 기간?
   → 1~3년 (단기) → B 또는 E (한국 교과 유지)
   → 4~10년 (중기) → A (국제학교) + B (한글학교)
   → 10년+ (장기·영주) → C·D (UZ 통합) + B (한국어 유지)

2. 예산?
   → $25K+/년 → A (국제학교)
   → $5K~$15K → C (UZ 사립) + B (한국학원)
   → $3K 이하 → D (UZ 공립) + B (한글학교)

3. 진학 목표?
   → 미국·영국 대학 → A (IB·IGCSE)
   → 한국 대학 → B + 검정고시
   → UZ 대학·인하대 Tashkent → C·D + 영어 강화

4. 자녀 나이?
   → 0~6세 (유아) → 어떤 환경도 적응 빠름, 언어 다양 노출
   → 7~12세 (초등) → 한국어 기반 + 영어/러시아어 보충
   → 13~18세 (중·고) → 진학 목표 결정 후 선택
```
