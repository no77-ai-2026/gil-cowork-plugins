# UZ 약국·약품 (성분 기반 검색·구매)

> Tashkent·UZ 거주 한국인을 위한 약국 가이드

## 핵심 원칙

**한국 약품과 UZ 약품은 성분 (활성 물질, Active Ingredient)이 동일하면 동등 효과**. 브랜드명·포장이 달라도 성분이 같으면 동일 약. 한국 약국 처방을 UZ에서 찾으려면 **성분명 (영어·라틴어)**으로 검색.

## 성분 기반 검색 워크플로우

### Step 1: 한국 약품 → 성분 추출

한국 약 포장에서 **활성 성분**을 영어·라틴어로 확인.

```
[한국 타이레놀 500mg]
포장 표시:
- 성분명 (한국어): 아세트아미노펜 500mg
- 성분명 (영문): Acetaminophen 500mg / Paracetamol 500mg
- 함량: 500mg per tablet

→ UZ 검색 키워드: "Paracetamol 500mg"
```

```
[한국 게보린]
성분:
- Acetaminophen 300mg
- Isopropylantipyrine 150mg
- 무수카페인 50mg

→ UZ 단일 성분 약 없음 → 동등 약 (Citramon 등) 또는 개별 구매
```

### Step 4: 처방전 처리

#### 한국 처방전 → UZ 사용

```
[방법 1: UZ 의사 재처방]
1. UZ 병원 (Korean Medical Center, Akfa Medical 등) 방문
2. 한국 처방전·진단서 (영문) 제시
3. UZ 의사 동등 약 처방
4. UZ 약국에서 구매

소요: 1~2시간
비용: 진료비 100,000~500,000 UZS ($8~40)
```

```
[방법 2: OTC 약 직접 구매]
- 처방전 불필요 약은 약국 직접 구매
- 일부 항생제도 OTC로 판매 (UZ 규정 다름, 약사 재량)
```

```
[방법 3: 한국 직배송]
- 한국 약국에서 처방·해외 직배송
- 일부 약은 통관 제한 (마약·향정신성 불가)
- DHL·EMS 5~7일
- 비용: 약가 + 배송비 50,000~150,000원
```

### Step 5: 안전·주의

#### 위조약·품질

```
[안전 약국 선택]
✅ Oson, Sof Pharma, Vita, Dori-Darmon (체인)
✅ 영수증 발급
✅ 유효기간 확인
✅ 포장 손상 X

❌ 길거리·시장 판매 (위조 위험)
❌ 영수증 없음
❌ 비정상 가격 (너무 저렴)
```
