# 학술지 투고 체크리스트

> gil-research | journal-selection 참조

## 투고 전 (1주~1개월 전)

```
[ ] Author Guidelines 다운로드·정독 (2회+)
[ ] Aim & Scope 재확인
[ ] Article Type 선택 (Original·Review·Brief 등)
[ ] Word Limit 확인
[ ] Figure·Table 수 제한
[ ] Reference 수 제한
[ ] 사전 등록 (preregistration) 여부 확인
[ ] 보고 가이드라인 (CONSORT·STROBE·PRISMA 등) 확인
[ ] OA·APC 옵션 결정
[ ] APC 예산 확인 (waiver 적용 여부)
[ ] 공동 저자 동의·기여 (CRediT)
[ ] 저자 순서 합의
[ ] 이해 충돌 (COI) 검토
[ ] IRB·IACUC 승인 (해당 시)
[ ] Data Sharing 정책 결정
```

## 투고 직전 (1주)

```
[ ] 본문 최종 교정 (영어 교정 서비스 권장)
[ ] 인용 형식 일관성 확인 (APA·Vancouver·IEEE 등)
[ ] Reference list 정확성 (DOI 모두 확인)
[ ] 표·그림 캡션 self-contained
[ ] 표·그림 해상도 (300 dpi+, vector 권장)
[ ] 약자·전문 용어 정의
[ ] 통계 보고 정확 (효과 크기 + CI)
[ ] 한계 (limitations) 솔직
[ ] Cover Letter 작성
[ ] Reviewer 추천·회피 리스트 (요구 시)
[ ] 저자 정보 (이름·소속·이메일·ORCID)
[ ] 표절 검사 (iThenticate·Turnitin)
[ ] Devil's Review 시뮬레이션 (gil-research:devil-review)
```

## 투고 시

```
[ ] 학술지 사이트 회원 가입·로그인
[ ] 원고 PDF·DOCX 업로드
[ ] 표·그림 별도 업로드 (해상도)
[ ] Cover Letter 업로드
[ ] 저자 정보 입력 (모든 저자)
[ ] Funding 정보
[ ] COI 선언
[ ] 보고 가이드라인 준수 체크
[ ] Reviewer 추천 (3~5명)
[ ] Reviewer 회피 (필요 시, 1~2명)
[ ] OA·구독 옵션 선택
[ ] APC 결제 정보 (OA 시)
[ ] Submission 완료 확인 이메일
```

## 심사 중 (2주~6개월)

```
[ ] Submission ID 보관
[ ] 학술지 시스템 정기 확인 (1주)
[ ] 추가 심사자 요청 (해당 시)
[ ] 진행 상태 학술지 문의 (3개월+ 무응답 시)
```

## Decision 이후

### Accept (with edits)
```
[ ] 편집자 코멘트 반영
[ ] Galley proof 정독
[ ] OA 결제 (선택 시)
[ ] 게재 후 OA 자가 아카이빙 (preprint·repository)
[ ] SNS·웹사이트 공유
```

### Minor Revision
```
[ ] 모든 코멘트 응답 (Revise & Resubmit Template)
[ ] 변경 사항 명시 (page·line)
[ ] 원고 수정
[ ] 마감 준수 (보통 30~60일)
```

### Major Revision
```
[ ] 추가 분석·실험 (해당 시)
[ ] Reviewer 코멘트 모두 응답
[ ] 본문 대폭 수정
[ ] Devil's Review 재실행
[ ] 마감 준수 (보통 60~90일)
[ ] Resubmit + Cover Letter (변경 요약)
```

### Reject
```
[ ] 이유 분석 (Methods·Results·Fit·Quality)
[ ] Reviewer 코멘트 적용 (다음 학술지)
[ ] 다음 학술지 선정 (journal-selection 재호출)
[ ] 형식 변환 (journal-style-adapter)
[ ] 재투고
```

### Reject without review (Desk Reject)
```
[ ] Aim & Scope 재확인
[ ] 형식 적절성
[ ] 다른 학술지 즉시 재투고 가능
```

## Cover Letter 핵심 요소

```
[ ] 학술지명·편집자 정확
[ ] 논문 제목
[ ] Article Type
[ ] 핵심 발견 (1~2 문장)
[ ] 학술지 적합성 (1 문장)
[ ] 저자 확인 사항 (모두 동의·미게재·COI 없음)
[ ] Suggested reviewers
[ ] (해당 시) Excluded reviewers + 이유
[ ] Corresponding author 정보
```

## ORCID·CRediT

```
[ ] 모든 저자 ORCID
[ ] CRediT 기여 명시:
  - Conceptualization
  - Methodology
  - Software
  - Validation
  - Formal analysis
  - Investigation
  - Resources
  - Data Curation
  - Writing - Original Draft
  - Writing - Review & Editing
  - Visualization
  - Supervision
  - Project administration
  - Funding acquisition
```

## 이중 출판 회피

```
[ ] 동일 본문 다른 학술지 X
[ ] 데이터 분할 시 (다른 측면) 설명
[ ] 회의 발표 → 학술지 (학술지 정책 따름)
[ ] 프리프린트 → 학술지 (학술지 정책 따름)
[ ] 한·영·러 동시 게재 시 명확한 분리
```
