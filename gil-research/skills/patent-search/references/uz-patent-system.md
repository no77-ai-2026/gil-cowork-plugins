# UZ·CIS·국제 특허 시스템 가이드

> gil-research v0.1.0 | UZ Agency on IP·PCT·Madrid Protocol·EAPO

## 개요

UZ 진출 시 특허·상표 출원은 한국 KIPRIS와 별도 절차. UZ Agency on Intellectual Property + 양자 협정 + PCT (국제) + Madrid Protocol (상표) 활용.

## UZ 지식재산권 시스템

### Agency on Intellectual Property (UZ 특허청)

- URL: https://ima.uz
- 위치: Tashkent
- 관할: 특허·실용신안·디자인·상표·저작권

### 출원 절차 (특허)

```
1. 출원서 (러시아어 또는 우즈벡어)
2. 발명 설명서 (Описание изобретения)
3. 청구의 범위 (Формула изобретения)
4. 도면 (Чертежи)
5. 요약 (Реферат)
6. 출원료 납부

→ 형식 심사 (1~2개월)
→ 실체 심사 (12~18개월)
→ 등록 또는 거절
```

### 출원 비용 (2026 기준 추정)

- 출원료: 약 $200~500
- 심사료: 약 $300~800
- 등록료: 약 $200~400/년
- 현지 변리사 (외국인 출원 시 필수): $500~1,500
- **합계: ~$1,200~3,100**

> 한국 KIPO 대비 약 1/2 비용. 국제 출원 (PCT 진입) 추가 시 증가.

## CIS 통합 특허 (EAPO)

### Eurasian Patent Office (EAPO)

- 회원국: 러시아·카자흐스탄·UZ·아제르바이잔·아르메니아·키르기스스탄·타지키스탄·투르크메니스탄·벨라루스
- 1회 출원 → 9개국 동시 보호
- URL: https://eapo.org

### EAPO vs UZ 직접

| 옵션 | 장점 | 단점 |
|------|------|------|
| UZ 직접 | 비용 ↓, 단순 | UZ만 보호 |
| EAPO | 9개국 동시 | 비용 ↑ ($3,000~5,000) |

## PCT (Patent Cooperation Treaty) 국제 출원

### PCT 기본

- WIPO 운영
- 156개국 동시 보호
- 한국 KIPO·UZ Agency·EAPO 모두 회원
- 우선권 12개월

### PCT 흐름

```
T+0: 한국 KIPRIS 출원 (또는 UZ Agency·EAPO)

T+12개월: PCT 국제 출원
- 한국 또는 UZ 우선권 주장
- 모든 PCT 회원국 동시 출원 효과

T+18개월: 국제 공개

T+30개월: 각국 진입 (National Phase)
- 미국 USPTO
- 유럽 EPO
- 일본 JPO
- 중국 CNIPA
- 인도 IPO
- 호주 IP Australia
- 캐나다 CIPO
- UZ Agency 또는 EAPO

각국 진입 후 각국 특허청 심사 (1~3년)
```

### PCT 비용

- PCT 출원: 약 $4,000~6,000 (변리사 + WIPO 비용)
- 각국 진입: 국가당 $2,000~10,000
- 5개국 진출 → 약 $15,000~50,000

## Madrid Protocol (상표 국제 출원)

### Madrid 기본

- WIPO 운영
- 130+ 회원국
- 1회 출원 → 다수 국가 보호
- UZ는 Madrid 회원국 (가입 2006)

### Madrid 흐름

```
T+0: 한국 KIPO 또는 UZ Agency 기초 출원·등록

T+0~6개월: WIPO Madrid 국제 출원
- 한국 또는 UZ 기초 등록 활용
- 지정 국가 선택 (미국·유럽·중국·UZ 등)

T+12~18개월: 각 지정 국가 심사
- 거절 또는 보호

각 국가 보호 기간 10년 (갱신 가능)
```

### Madrid 비용 (5개 국가 예시)

- 기본료: CHF 653 (~$700)
- 지정국별 추가: 국가당 $100~500
- 변리사 수임: $500~1,500
- **합계: 5개국 ~$2,500~5,000**

> 한국 + 미국 + 유럽 + 중국 + UZ Madrid 출원 = 약 $3,500.

## 한·우즈벡 양국 동시 출원 전략

### 시나리오 1: 한국 본사 핵심 IP

```
T+0: 한국 KIPRIS 출원
T+12개월: PCT 국제 출원 (한국 우선권)
T+30개월: 각국 진입 (UZ 포함 또는 EAPO)
```

### 시나리오 2: UZ 자회사 현지 발명

```
T+0: UZ Agency on IP 출원 (또는 EAPO)
T+12개월: PCT 국제 출원 (UZ 우선권)
T+30개월: 한국 KIPRIS 진입 등
```

### 시나리오 3: 공동 개발

```
T+0: 한국 KIPRIS + UZ Agency 동시 출원
T+12개월: PCT 추가 (다른 국가)
- 양국 발명자 공동 명시
- IP 소유권 계약 (인터컴퍼니)
```

## 분야별 UZ IP 특이사항

### 의약품
- 우즈벡 보건부 추가 인증
- 임상시험 데이터 보호
- 생물학적 제제 별도 절차

### 농업·식품
- UZ 농식품부
- GI (Geographical Indication) — 우즈벡 와인·과일

### IT·소프트웨어
- 저작권 (Copyright) 보호 강함
- 특허 보호 제한적 (algorithm 자체 X)
- IT Park 거주자 우대

## 흔한 실수

1. **한국만 출원**: UZ 진출 시 UZ 특허 X → 모방 위험
2. **PCT 시기 놓침**: 12개월 우선권 만료
3. **현지 변리사 X**: UZ 출원은 외국인 경우 필수
4. **상표 vs 특허 혼동**: 별도 절차
5. **러시아어 비공증**: UZ 정부 제출 거절 가능
6. **EAPO vs UZ 단독**: 진출 국가 수에 따라 결정
