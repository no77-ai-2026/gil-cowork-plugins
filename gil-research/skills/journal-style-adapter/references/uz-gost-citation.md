# UZ·CIS 인용·형식 (ГОСТ) 컨텍스트

> gil-research | journal-style-adapter 참조 | ГОСТ 7.0.5·러시아어 논문 형식

APA·IEEE·Vancouver 등 국제 양식은 본문 그대로. UZ·CIS 권역 투고 시 ГОСТ 적용.

## 1. ГОСТ 7.0.5-2008 인용 (참고문헌)

- 본문 내 각괄호 번호 인용: `[1]`, `[2, с. 15]`(페이지 명시).
- 참고문헌 목록: 저자. 제목 : 부제 / 책임표시. — 판차. — 출판지 : 출판사, 연도. — 면수. (키릴 표기 규칙).
- УДК(전세계 십진분류)·ББК 분류 코드 표기.

## 2. 러시아어 논문 구조

- аннотация(초록) + ключевые слова(키워드) 러시아어 필수(+영어 병기 권장).
- 서론(Введение)·방법·결과·결론(Заключение)·литература 순.

## 3. 변환 워크플로우 추가

국제 양식 → ГОСТ 변환 시: ① 번호 인용 체계로 통일 ② 참고문헌 키릴 전사(transliteration) 규칙(ГОСТ 7.0.99) ③ УДК 부여.

## 4. 안티패턴

- APA 저자-연도를 ГОСТ 저널에 그대로 → 형식 반려.
- 키릴 전사 규칙 무시한 라틴 혼용.
