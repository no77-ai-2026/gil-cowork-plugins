# 학술지별 형식 템플릿

## Nature

```
[Title] 65 characters max

[Authors] First Last¹, First Last²
[Affiliations] ¹Department, University, City, Country

[Summary] 200 words, no subheadings

[Main Text] 1500-3000 words
- Introduction (no subheading)
- Results (subheadings allowed)
- Discussion

[Methods] After main text, 3000 words

[References] 50 max, Author-Year
- Smith, J. & Lee, K. Title. *J. Name* 15, 100-120 (2024).

[Figures] 4-6, vector PDF/EPS
[Tables] up to 4
[Extended Data] up to 10 figures + tables
```

## 변환 우선순위

```
1. 단어 수 조정 (Abstract·본문 limit)
2. 섹션 구조 재배열
3. 인용 형식 일괄 변환 (Zotero·EndNote)
4. Figure·Table 수·해상도
5. Reference 수 조정
6. 학술지별 특수 요구 (CONSORT·STROBE 등)
7. 보고 가이드라인 적용
8. Cover Letter 작성
```
