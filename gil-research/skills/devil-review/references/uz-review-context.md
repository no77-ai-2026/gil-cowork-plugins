# UZ·CIS 학술 심사 컨텍스트

> gil-research | devil-review 참조 | 러시아어 논문·УзВАК·ВАК 심사 관행

국제 저널(영어) 심사 기준은 본문 워크플로우를 그대로 쓰고, UZ·CIS 권역 투고 시에만 본 보강을 적용합니다.

## 1. 권역별 심사 체계

- **국제(영어)**: peer review 중심, IF·Scopus 지표. 본문 Reviewer 2 시뮬레이션 그대로 적용.
- **러시아 ВАК / 우즈벡 УзВАК**: 등재지(перечень) 요건·형식 심사 비중이 큼. 표절(Антиплагиат) 검사 통과율, 초록(аннотация) 러시아어 필수, УДК/ГОСТ 형식 점검이 사실상 게이트.

## 2. 러시아어 논문 Reviewer 관점 추가 점검

- 초록·키워드의 러시아어 적절성(직역투 감점).
- УДК 분류·참고문헌 ГОСТ 7.0.5 정합성.
- 방법론 서술이 소비에트 전통 서술 양식과 충돌하지 않는지.

## 3. 트릴링구얼 대응

한국어 초안 → 러시아어/영어 투고본 분기 시, 의미 보존·현지 검수 1회 필수. 한국어 휴머나이징은 `gil-content:humanize-korean`(한국어 전용)이며, 러시아어는 현지 검수로 대체.

## 4. 안티패턴

- 영어 저널 기준을 ВАК/УзВАК에 그대로 적용 → 형식 게이트 탈락.
- 기계 번역 초록 그대로 제출 → Антиплагиат·가독성 문제.
