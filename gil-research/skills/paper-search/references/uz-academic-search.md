# UZ·CIS 학술 검색 가이드

> gil-research v0.1.0 | UZ·CIS·러시아어 학술 검색

## 개요

UZ에서 학술 검색 시 영어권 DB(Scopus·WoS·PubMed) 외에 러시아어·CIS 전용 DB가 중요합니다. 우즈벡 학술계는 영어 + 러시아어 + 우즈벡어 트릴링구얼 환경.

### 양국 공동 연구
- KOTRA·KIST·UZ MID 협력
- 한국 NRF + UZ Innovation Foundation 매칭 펀드
- Inha Tashkent (한·UZ 합작 대학)

### 교류
- UZ 한국학 (Korean Studies in UZ)
- Tashkent State University of Oriental Studies — 한국학과
- 양국 학회 (한국우즈벡학회·UZ Korean Studies Association)

## 검색 워크플로우

### Step 1: 영어 우선
- Scopus·WoS·Google Scholar로 글로벌 선행 연구

### Step 2: 러시아어 추가
- eLIBRARY.RU·CyberLeninka 러시아어 키워드 검색
- РИНЦ로 CIS 인용 분석

### Step 3: UZ 특화
- Tashkent 대학 도서관·VAK 인덱스
- UZ 발간 학술지

### Step 4: 한국어 (한국 본사·UZ 한국학)
- KCI·RISS·DBpia
- 한국 NRF DB

## 인용 형식 (CIS 표준)

ГОСТ Р 7.0.5-2008:
```
Автор А.А. Название статьи // Название журнала. — Год. — Т. Х, № Y. — С. Z-W.

예: Иванов И.И. Применение машинного обучения в медицине //
Журнал РАН. — 2024. — Т. 25, № 3. — С. 100-120.
```
