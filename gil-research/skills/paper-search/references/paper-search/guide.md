# 논문 검색 가이드

## KCI API 상세

- 논문 검색: apiCode=articleSearch
- 참고문헌: apiCode=referenceSearch
- 응답 포맷: XML
- 발급 (경로 A): data.go.kr 활용신청 (자동승인, 간편)
- 발급 (경로 B): KCI 직접 신청 (kciadmin@nrf.re.kr 공문 필요)

## 검색 전략 패턴

1. 키워드 확장: 동의어, 영문 번역 포함
2. 연도 필터: 최근 5년 우선
3. 인용수 정렬: 영향력 높은 논문 우선
4. 학술지 등급: KCI 등재지 > 등재후보 > 기타
