# 트릴링구얼 학술 작성 가이드

> gil-research v0.1.0 | 한국어·영어·러시아어 트릴링구얼 학술 표기

## 개요

한국 본사·UZ 자회사 공동 연구 또는 한·우즈벡 학자 협력 시 다국어 학술 작성. 영어 (국제 학술지) + 한국어 (KCI) + 러시아어 (CIS·UZ VAK) 동시 투고 가능 옵션.

## 4가지 트릴링구얼 옵션

### 옵션 A: 단일 언어 (표준, 가장 일반)
- 영어로 국제 학술지
- 한국어로 KCI
- 러시아어로 CIS·UZ VAK 학술지
- 별도 출판 (이중 출판 X 주의)

### 옵션 B: 한국어 본문 + 영문 초록 (KCI 표준)
- 본문 한국어
- Abstract + Keywords 영어
- 가장 흔한 한국 학술지 양식

### 옵션 C: 영어 본문 + 다국어 초록
- 본문 영어 (국제 학술지)
- Abstract: 영어
- 추가 초록: 한국어 + 러시아어 (선택)
- 일부 국제 학술지 (Lancet 등) 다국어 초록 지원

### 옵션 D: 트릴링구얼 (한+영+러)
- KCI + CIS 학술지 동시 투고용 (단, 동일 학술지 X — 분리 학술지)
- 본문은 영어
- 초록 한·영·러 3개

## 학술지별 초록·키워드 요구

| 학술지 | 본문 | 초록 |
|--------|------|------|
| Nature | 영어 | 영어 (200) |
| Lancet | 영어 | Structured 영어 (300) |
| KCI 한국 | 한국어 또는 영어 | 한국어 (300자) + 영어 (200단어) |
| RSCI 러시아 | 러시아어 또는 영어 | 러시아어 + 영어 (200단어) |
| UZ VAK | 우즈벡어 또는 러시아어 또는 영어 | 우즈벡어 + 러시아어 + 영어 |

## 인용 형식 매핑 (다국어)

### 영어 학술지 → APA·Vancouver·IEEE·Chicago

```
Smith, J., Lee, K., & Kim, S. (2024). Title.
*Journal*, *15*(3), 100-120.
```

### 한국 KCI → KCI 표준

```
스미스, 이, 김 (2024). 제목. *학술지명*, 15(3), 100-120.
```

### CIS·UZ → ГОСТ Р 7.0.5-2008

```
Смит Дж., Ли К., Ким С. Название //
Журнал. — 2024. — Т. 15, № 3. — С. 100-120.
```

### 한국어 → 영어 → 러시아어 동일 인용

```
영어: Kim, S. & Lee, K. (2024). Title. *Journal*, *15*(3), 100-120.
한국어: 김, 이 (2024). 제목. *학술지명*, 15(3), 100-120.
러시아어: Ким С., Ли К. Название // Журнал. — 2024. — Т. 15, № 3. — С. 100-120.
```

## 트릴링구얼 초록 예시

### 영어 (국제 표준)
```
Background: Korean cosmetics market in Uzbekistan grew by 30% from 2020 to 2024.
Methods: Survey of 500 Uzbek consumers (aged 18-45) in Tashkent.
Results: 68% prefer Korean products for skin care.
Conclusions: K-Beauty has strong market potential in Uzbekistan.
Keywords: K-Beauty, Uzbekistan, consumer behavior, market analysis
```

### 한국어 (KCI 표준)
```
배경: 우즈베키스탄 한국 화장품 시장은 2020년 대비 2024년 30% 성장하였다.
방법: Tashkent 18~45세 소비자 500명 설문조사.
결과: 68%가 스킨 케어에서 한국 제품을 선호한다고 응답하였다.
결론: K-Beauty는 우즈베키스탄 시장에서 강력한 잠재력을 가진다.
키워드: K-Beauty, 우즈베키스탄, 소비자 행동, 시장 분석
```

### 러시아어 (RSCI·UZ VAK 표준)
```
Введение: Рынок корейской косметики в Узбекистане вырос на 30% с 2020 по 2024 год.
Методы: Опрос 500 узбекских потребителей (18-45 лет) в Ташкенте.
Результаты: 68% предпочитают корейские продукты для ухода за кожей.
Заключение: K-Beauty имеет значительный рыночный потенциал в Узбекистане.
Ключевые слова: K-Beauty, Узбекистан, поведение потребителей, анализ рынка
```

옵션:


## 학술 윤리 (다국어 출판)

### 이중 출판 (Duplicate Publication) 회피

- 동일 본문을 여러 학술지에 → ❌
- 동일 데이터·결과를 다른 언어로 → ⚠️ (학술지·기관 정책 따라)
- 명확한 분할 (예: 한국 시장 분석 + 우즈벡 시장 분석 별도) → OK

### 표절 (Plagiarism)

- 자기 표절 (self-plagiarism)도 ❌
- 본인 이전 발표는 인용 + 명확한 차별점

### 저자 표시 (Authorship)

- ICMJE 기준 (의학)
- CRediT (Contributor Roles Taxonomy)
- 기여자 모두 명시

## 한·우즈벡 공동 연구 출판 전략

### 권장 시나리오

1. **글로벌 영문 학술지 1편** (메인 발견)
2. **한국 KCI** (한국 시장 분석 분리, 한국어)
3. **UZ VAK 또는 RSCI** (UZ 시장 분석 분리, 러시아어)

### 권장 X
- 동일 데이터·동일 본문 3언어 동시 게재
- 동일 결과를 분량만 다르게

## 학술지 선정 (다국어)

`journal-selection` 호출:
- 영어 국제 학술지 1개
- KCI 한국 학술지 1개
- VAK·RSCI 러시아어 학술지 1개

## 흔한 실수

1. **AI 직역만**: 학술 용어 부정확 → 현지 학자 검수
2. **다른 학술지에 동일 본문**: 이중 출판
3. **저자 순서 분쟁**: 사전 명확
4. **인용 형식 혼용**: 학술지별 일치
5. **국가별 회계·이해 충돌**: COI 다국어 명시
6. **언어별 단위·날짜**: 영어 USD·한국 KRW·러시아 UZS·날짜 형식
