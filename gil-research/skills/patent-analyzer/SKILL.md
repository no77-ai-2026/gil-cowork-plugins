# 특허 분석 (Patent Analyzer)

## 개요

특허 데이터를 체계적으로 분석하여 기술 동향, 선행기술 보고서, FTO(Freedom-to-Operate) 분석, 특허 출원서 초안을 작성하는 전문 스킬입니다. patent-search 스킬로 검색한 특허 데이터를 바탕으로 심층 분석을 수행합니다.

## 트리거 키워드

- 특허 분석, 특허 동향, 특허 맵
- 선행기술 조사, prior art search
- FTO 분석, 침해 분석, 권리 범위 분석
- 특허 출원서, 명세서 작성
- 기술 로드맵, IPC 분석

## 워크플로우

### 1단계: 분석 목적 확인

AskUserQuestion으로 분석 목적을 선택받습니다:

| 분석 유형 | 설명 | 산출물 |
|----------|------|--------|
| 특허 동향 분석 | 기술별 출원 추이, 주요 출원인, IPC 분포 | 특허 동향 보고서 (차트 포함) |
| 선행기술 조사 | 발명 핵심 구성요소별 관련 특허 검색 및 분석 | 선행기술 조사 보고서 |
| FTO 분석 | 등록특허 청구항 대비 침해 가능성 판단 | FTO 분석 보고서 |
| 출원서 초안 | 발명 내용을 특허 명세서 형식으로 구성 | 특허 출원서 초안 |

### 2단계: 특허 데이터 수집 (필요시)

분석 대상 특허 데이터가 없는 경우 patent-search 스킬을 연계하여 검색합니다:
- 검색 키워드 및 IPC 분류코드
- 검색 범위 (국내/해외, 연도, 상태)
- 데이터 수집 목적 (동향/선행기술/FTO)

### 3단계: 분석 수행

**A. 특허 동향 분석 (Patent Landscape)**

1. 연도별 출원 추이 분석
   - 5-10년 단위 장기 추세
   - 기술 생명주기 단계 파악 (도입기/성장기/성숙기/쇠퇴기)

2. 출원인별 경쟁 현황
   - 상위 10개 출원인 출원 건수 및 점유율
   - 주요 출원인 기술 포트폴리오 분석

3. IPC/CPC 분류별 기술 맵
   - 대분류별 기술 분포
   - 세부 분야별 핵심 기술 식별

4. 시각화 (data-visualizer 스킬 연계)
   - 연도별 출원 추이 차트
   - 출원인별 점유율 파이 차트
   - IPC 분포 트리맵

**B. 선행기술 조사 (Prior Art Search)**

1. 발명의 핵심 구성요소 추출
   - 발명의 기술적 과제
   - 해결 수단별 핵심 구성요소
   - 구성요소별 키워드 및 IPC 매핑

2. 관련 특허 검색 (patent-search 스킬 연계)
   - 구성요소별 개별 검색
   - 조합 검색 (구성요소 2개 이상 AND)
   - 배제 검색 (차별점 강조)

3. 유사도 분석 및 차별점 도출
   - 청구항 대비 구성요소 매칭
   - 선행기술과의 차별점 식별
   - 신규성/진보성 평가

4. 선행기술 조사 보고서 작성
   - 검색 전략 (키워드, IPC, 검색식)
   - 관련 특허 리스트 (가장 유사한 5-10건)
   - 청구항 대비 구성요소 비교표
   - 차별점 및 신규성/진보성 의견

**C. FTO 분석 (Freedom-to-Operate)**

1. 분석 대상 등록특허 식별
   - 대상 기술 관련 등록특허 검색
   - 권리 범위 심사 관련 특허 필터링
   - 경쟁사 특허 우선 검토

2. 청구항 분석 (독립항 중심)
   - 독립항 구성요소별 요건 해석
   - 구성요소 대비 비교 (All Element Rule)
   - 균등論 적용 가능성 검토

3. 침해 가능성 판단
   - 직접 침해 (모든 구성요소 포함)
   - 간접 침해 (부품/소프트웨어 공급)
   - 공동 침해 (공동 실행)
   - 균등 침해 (균등론 적용 시)

4. 회피 설계 방향 제안
   - 비침해 대안 구성요소 제시
   - 무효 심판 가능성 평가
   - 라이선스 협상 시 참고사항

5. FTO 분석 보고서 작성
   - 분석 대상 특허 리스트
   - 청구항 대비 구성요소 비교표
   - 침해 가능성 종합 의견
   - 리스크等级 (High/Medium/Low)
   - 회피 설계 방안

**D. 출원서 초안 작성**

1. 발명의 명칭

2. 기술분야
   - 발명이 속하는 기술 분야

3. 배경기술
   - 기존 기술의 문제점
   - 해결하려는 과제

4. 발명의 내용
   - 과제 해결 수단 (핵심 구성요소)
   - 발명의 작용 효과
   - 실시예 (바람직한 형태)

5. 청구항 (Claims)
   - 독립항 (핵심 구성요소 필수)
   - 종속항 (부가 구성요소)
   - 구성요소별 기능적 표현

6. 요약
   - 발명의 요약 (200자 내외)

7. 도면 간단 설명 (도면 있을 시)

### 4단계: 보고서 작성

분석 목적별로 다음 보고서를 작성합니다:
- 특허 동향 분석 보고서 (차트 포함)
- 선행기술 조사 보고서
- FTO 분석 보고서
- 특허 출원서 초안

### 5단계: 검토 및 제안

- 분석 결과에 대한 종합 평가
- 후속 조건 제안 (출원/무효심판/라이선스 등)

## 사용 예시

```
"배터리 기술 특허 동향을 분석해줘. 최근 10년 기준으로."
"이 발명의 선행기술을 조사해줘. 핵심 구성요소는 A, B, C야."
"FTO 분석을 해줘. 대상 기술은 'AI 기반 이상 탐지'야."
"특허 출원서 초안을 작성해줘. 발명의 명칭은 '양자 암호 통신 방법'이야."
```

## 출력 형식

- 특허 동향 분석 보고서 (Markdown + 차트)
- 선행기술 조사 보고서 (Markdown + 비교표)
- FTO 분석 보고서 (Markdown + 리스크 등급)
- 특허 출원서 초안 (Markdown)

## 주의사항

- 선행기술 조사 및 FTO 분석은 변리사 자격이 있는 전문가의 검토를 거치시기 바랍니다.
- FTO 분석은 등록특허를 대상으로 하며, 출원 중인 특허는 등록 후 권리 범위가 확정됩니다.
- 출원서 초안은 참고용이며, 실제 출원 전 반드시 변리사의 검토를 거치시기 바랍니다.
- 특허 데이터가 부족한 경우 patent-search 스킬을 먼저 실행하여 데이터를 수집한 뒤 연계 분석하는 것을 권장합니다.
- 시각화가 필요한 경우 data-visualizer 스킬을 연계하여 차트를 생성할 수 있습니다.
