# UZ 연구 데이터·통계 컨텍스트

> gil-research | research-analysis 참조 | stat.uz·공공데이터·러시아어 데이터셋

범용 통계/ML 가이드는 본문 그대로. UZ 현지 데이터 활용 시 본 보강 적용.

## 1. 주요 UZ 데이터 출처

- **stat.uz** — 우즈벡 국가통계위원회(인구·경제·산업 공식 통계).
- **data.gov.uz** — 공공데이터 포털.
- **Lex.uz** — 법령(정책 연구).
- CIS 비교: Rosstat(러)·국가별 통계청.

## 2. 데이터 특성·주의

- 다수 데이터가 **러시아어/우즈벡어**로 제공 → 변수명·범주 라벨 번역·코딩 필요.
- 시계열 단절(통화 개혁·행정구역 개편) 주의.
- 표본·센서스 접근 제약 → 결측·대표성 점검 강화.

## 3. 분석 워크플로우 추가

stat.uz 원자료 → 정제(인코딩·라벨 번역) → 본문의 통계검정 선택 트리 적용. 공공데이터 직접 조회는 `gil-data:public-data`·`gil-business:real-estate-search` 등과 연계.

## 4. 안티패턴

- 러시아어 범주 라벨 자동번역만 믿고 코딩 → 의미 오류.
- 통화·행정 개편 구간 무시한 시계열 회귀.
