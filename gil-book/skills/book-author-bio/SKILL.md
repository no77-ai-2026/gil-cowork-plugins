# book-author-bio

> 저자가 타깃 페르소나에게 **"왜 이 사람이 이 책을 쓰는가"**를 1분 안에 설득하는 문서. 책의 신뢰도 50%는 본문 내용, 나머지 50%는 저자 약력·저자의 말에서 결정됩니다. book-target-reader의 페르소나·JTBD를 입력받아 4 장르별 신뢰 신호를 자동 분기. **짧은(50자)·중간(200자)·긴(500자) 3 길이 약력**과 **저자의 말(500-800자)**을 동시 산출합니다.

## 1. 개요

저자 약력은 단순한 자기소개가 아닙니다. 페르소나에게 다음 **3 신호**를 보내야 합니다.

| 신호 | 의미 | 작용 |
|------|------|------|
| **권위** | 이 분야에 대한 자격·경력·업적 | 책의 정확성·신뢰성 보장 |
| **공감** | 페르소나와 비슷한 경험·고민 | "이 사람도 나와 같았다" 동질감 |
| **변화** | 저자가 이뤄낸 구체적 결과 | "나도 책을 통해 같은 결과를 얻을 수 있다" |

3 신호가 균형있게 들어가야 합니다. 권위만 있으면 거리감, 공감만 있으면 신뢰 부족, 변화만 있으면 자랑.

저자 약력은 **3 길이**(50자·200자·500자)로 동시 작성:

| 길이 | 사용처 |
|------|--------|
| 50자 | 표지 띠지·SNS 프로필·서점 검색 결과 |
| 200자 | 책 표지 뒷날개·온라인 서점 저자 소개 |
| 500자 | 책 뒷부분 저자 소개·언론 보도자료·강연 약력 |

## 2. 트리거 키워드

## 3. 워크플로우 (4단계)

### Step 1: 저자 자료 수집 (5 영역)

AskUserQuestion으로 저자에 대한 다음 5 영역 자료 수집:

| 영역 | 질문 | 예시 |
|------|------|------|
| **학력·자격** | 학위·자격증·교육과정 | 서울대 경영학과·CFA·하버드 MBA |
| **실무 경력** | 회사·직책·연차·산업 | 삼성SDS 10년 / 컨설팅 5년 / 스타트업 창업 |
| **저술·강연** | 이전 저서·기고·강연 | 이전 저서 2권·매일경제 칼럼·삼성전자 사내 강연 |
| **수상·인증** | 수상 이력·언론 인용 | 한국마케팅대상 / 포브스 30 Under 30 |
| **팬덤·플랫폼** | SNS 팔로워·구독자·강의 누적 수강생 | 인스타 10만 / 유튜브 50만 / 클래스101 누적 5천명 |

> 💡 **5 영역 중 최소 2개 영역에서 강한 신호가 있어야 출판사 통과 가능**. 모든 영역이 약하면 자비 출판·인플루언서 협업 검토.

### Step 2: 신뢰 신호 우선순위 결정

5 영역 자료를 페르소나 관점에서 **우선순위**로 재정렬. 어떤 신호가 페르소나에게 가장 강력한가?

| 페르소나 유형 | 1순위 신호 | 2순위 신호 |
|--------------|-----------|-----------|
| 입문 학습자 (실용서·기술서) | 실무 경력 + 학력 | 강연·수강생 수 |
| 전문가 (기술 심화서) | 실무 경력 + 저술 | 수상·인증 |
| 공감 추구자 (인문·에세이) | 개인 경험·서사 | 팬덤·SNS |
| 장르 독자 (소설·문학) | 이전 작품·문학상 | 작가 활동 |

페르소나 1순위 신호를 **약력 첫 문장**에 배치.

### Step 3: 3 길이 약력 동시 작성

#### 50자 약력 (표지 띠지·SNS 프로필용)

**공식**: [정체성] + [핵심 신호] + [현재 활동]

예시:
- "10년 차 마케팅 매니저, 3권의 책을 쓴 저자, 인스타 10만 팔로워."
- "13년 차 백엔드 개발자, 한빛미디어 베스트셀러 작가, 풀스택 유튜버."
- "두 아이 엄마이자 워킹맘, 일과 가족 사이 30대를 기록하는 에세이스트."

50자 작성 원칙:
- 한 문장이 아닌 2-3 짧은 문장 (쉼표로 구분 OK)
- 직업·작품·플랫폼 3 요소 중 최소 2개 포함
- 추상어 ("열정적", "전문가") 금지, 구체적 숫자·이름 우선

#### 200자 약력 (책 뒷날개용)

**공식**: 50자 약력 + [페르소나 공감] + [핵심 변화·결과]

예시 구조 (200자):
```
[50자 약력 도입]
[페르소나와의 공감 — 비슷한 고민·시작점]
[저자가 이뤄낸 구체적 변화·결과 1~2개]
[책을 쓴 이유 한 줄]
```

200자 작성 원칙:
- 첫 문장에 1순위 신호 (Step 2 결정)
- 두 번째 문장에 페르소나 공감
- 세 번째 문장에 변화·결과 (숫자 포함)
- 마지막에 책을 쓴 이유 한 줄

#### 500자 약력 (긴 형식)

**공식**: 200자 약력 + [경력 타임라인] + [활동 영역] + [관심 분야]

예시 구조 (500자):
```
[200자 약력 도입]
[학력·자격 1~2줄]
[실무 경력 타임라인 2~3 회사]
[저술·강연·강의 활동 1~2 단락]
[현재 관심 분야·앞으로의 계획 1~2 문장]
[연락처·SNS·웹사이트 1줄]
```

500자 작성 원칙:
- 시간 순서 (오래된 순 → 최근) 또는 영역별 (학력 → 경력 → 저술)
- 구체적 회사명·작품명·수상명 명시
- 추상어 최소화

### Step 4: 저자의 말 작성 (500-800자)

저자의 말은 책의 **첫 페이지**에 들어가는 머리말. 4 단 구조:

| 단 | 분량 | 내용 |
|----|------|------|
| **1단 도입** | 100-150자 | 페르소나가 책을 펴는 순간의 상황 묘사 |
| **2단 공감** | 150-200자 | 저자도 그 자리에 있었다는 고백·서사 |
| **3단 약속** | 150-200자 | 이 책에서 무엇을 다룰지, 어떻게 도울지 |
| **4단 초대** | 100-150자 | 함께 읽기를 권하는 마무리 |

#### 저자의 말 작성 원칙

1. **2인칭 또는 1인칭** — "당신은 지금…", "저는 5년 전…"
2. **구체적 장면** — 시간·장소·감각이 그려지도록
3. **숫자·이름 인용** — 추상 대신 구체 (예: "3년 전 마포구 카페에서…")
4. **공감 우선, 권위 후행** — 공감 신호가 먼저, 권위는 뒤에 자연스럽게
5. **책의 약속을 한 문장으로** — 3단 약속을 짧고 명확하게

## 4. 5 장르 프리셋 (자동 분기)

### 4-3. 기술·전문서·IT 입문서

| 차원 | 분기 |

| 2순위 신호 | **이전 저서·기술 블로그·발표 이력** |

### 4-4. 소설·문학

| 차원 | 분기 |
|------|------|
| 1순위 신호 | **이전 작품·문학상** |
| 2순위 신호 | **작가 활동·신인공모 통과·문학 동인** |
| 50자 약력 강조 | 데뷔·문학상·작품집 (예: "2022년 ___신인상으로 데뷔") |
| 저자의 말 톤 | 문학적·은유적·서사 짧게 |
| 페르소나 공감 | 작품 세계관·정서로 이어주는 안내 |

### 4-5. 경험서·실무회고 (GIL 오리지널)

- **신뢰 신호 우선순위**: 실무 경력·현장 연차(**학력보다 우선**) > 강연·기고·뉴스레터 이력
- 저자의 말: "왜 내가 이 경험을 기록하는가" — 가르침이 아니라 공유의 태도

- **공감 신호 인식**: ✅ / ❌ (이 사람이 나와 비슷한가?)

## 6. 사용 예시

### 예시 1 — 실용서·자기계발 (부업서)

```
"30일 챌린지로 시작하는 부업" 책 저자 약력 작성해줘.
저자: 10년 차 마케팅 매니저, 3년간 부업으로 월 200만원 달성, 인스타 5만.
```

→ 1순위 신호: 실무 경력 + 결과(월 200만원) + 인스타 팬덤  
→ 50자: "10년 차 마케팅 매니저, 부업으로 월 200만원 달성한 N잡 워커, 인스타 5만 팔로워."  
→ 200자: 위 + "5년 전 야근에 지쳐 있을 때 부업을 시작했고, 시행착오 끝에 30일 만에 첫 매출을 만들었습니다…"  
→ 500자: 200자 + 학력·회사 타임라인 + 저술·강연 활동  
→ 저자의 말: 도입(퇴근 후 카페에서 책을 펼치는 장면) + 공감 + 약속 + 초대

### 예시 2 — IT 입문서

```
"Next.js 15 + shadcn/ui" 책 저자 약력 작성해줘.
저자: 13년 차 풀스택, Next.js 컨트리뷰터, 유튜브 5만 구독, 한빛미디어 이전 저서 2권.
```

→ 1순위 신호: 실무 + Next.js 컨트리뷰터(오픈소스 권위)  
→ 50자: "13년 차 풀스택 개발자, Next.js 컨트리뷰터, 한빛미디어 베스트셀러 작가, 유튜브 5만."  
→ 200자: + 페르소나(주니어 3년차)에 대한 공감 + 풀스택 전환기 서사  
→ 저자의 말: Next.js 13 → 15 변화 막막함에 공감 + 책의 약속 (실전 프로젝트 4개)

### 예시 3 — 인문·에세이 (40대 워킹맘)

```
"엄마이지만 나로 살기" 에세이집 저자 약력 작성. 
저자: 40대 워킹맘, 마케팅 업계 15년, 브런치·인스타 활동.
```

→ 인문 프리셋 → 정체성 강조 + 일상 관찰 서사  
→ 50자: "엄마이지만 나로 사는 40대 마케터, 브런치·인스타에서 일상을 기록한다."  
→ 저자의 말: 새벽 5시 카페에서 글을 쓰는 장면 + 워킹맘 공감 + 책의 약속

## 7. 한국 출판 컨텍스트

### 저자 약력 한국 관행

- **'~다'/'~습니다' 어미**: 출판사별 다름. 한빛미디어·인사이트 = '~다' / 민음사·문학동네 = 작가 자유
- **본명·필명**: 자비 출판·에세이는 필명 OK, 기술서·전문서는 본명 권장
- **국문·영문**: 한국 출판은 국문 우선, 영문 약력은 강연·해외 진출용
- **저자 사진**: 출판사 요구 시 별도 제공, 책에는 보통 단정한 정면 사진

### SNS 약력 한국 채널

| 채널 | 한도 | 특징 |
|------|------|------|
| 인스타그램 | 150자 | 줄바꿈·이모지 활용 가능 |
| 브런치 | 100자 (소개) + 자기소개 페이지 | 글로 자신을 표현 |
| 링크드인 | 영문 권장, 한국어 가능 | 비즈니스·강연 채널 |
| 유튜브 | 1000자 채널 설명 | SEO 키워드 포함 |
| X(트위터) | 160자 | 짧은 한 줄 정체성 |
| 카카오톡 채널·블로그 | 무제한 | 한국 중장년층 도달 채널 |

### 강연·강의 약력 한국 컨벤션

- 클래스101·인프런·유데미: 강사 소개 200자 + 강의 분야 키워드 5-10개
- 기업 사내 강연: 약력 500자 + 강의 주제 3-5개 + 사진
- 정부·공공기관 강연: 공식 이력서 형식 (학력·경력·자격증·수상)

## 8. 주의사항

### 약력 함정 회피

| 함정 | 피하는 법 |
|------|----------|
| **자랑 일변도** | 권위만 강조하면 거리감. 공감·변화 신호 추가 |
| **추상어 남용** | "열정적", "전문가" 같은 추상어 금지. 구체적 회사·작품·숫자 |
| **시간 순서 혼란** | 200자·500자 모두 시간 흐름 또는 영역 흐름 명확히 |
| **사적 정보 과다** | 가족·연애·종교 등 책과 무관한 사적 정보 금지 |
| **장르 미스매치** | 인문서에 IT 경력만, 기술서에 에세이만 자랑하지 않기 |
| **페르소나 무시** | 페르소나 1순위 신호를 약력 도입에 배치 |

### 저자의 말 함정

- **자기 변명·이력서식** 금지 — "이런 어려움이 있었지만…" 같은 변명 톤
- **책 내용 스포일러** 금지 — 책의 핵심 결론을 머리말에 다 풀어 놓지 않음
- **너무 긴 도입** — 도입이 200자 초과 시 핵심으로 빨리 진입
- **2인칭 갑작스러운 전환** — 1인칭·2인칭 시점 일관성 유지

### 출처 정책

- 학력·경력·수상 등 **사실 정확성** 필수 (출판 후 검증 가능)
- 부풀린 표현 금지 (예: "베스트셀러 작가" → 실제 베스트셀러 등극 시에만)
- vault 외부 참고 자료 직접 인용 금지
- SNS 팔로워·구독자 수는 작성 시점 기준 명시 (변동 가능)
