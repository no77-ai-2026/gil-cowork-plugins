# book-proposal-writer

> 한국 출판사 편집장이 **1분 안에 통과·반려를 판단**할 수 있는 투고용 제안서를 작성하는 스킬. 앞선 4 스킬(컨셉서·페르소나·목차·저자약력) 산출물을 결합해 출판기획서 + 샘플 챕터 + 마케팅 플랜의 **3 합 패키지**를 생성. 한국 주요 출판사별 양식 자동 분기 + 거절 신호 사전 검출. **출판사 통과율 = 컨셉서(USP) 40% + 샘플 챕터(필력) 30% + 마케팅 플랜(영업력) 30%**.

## 1. 개요

출판사 투고 제안서는 다음 **3 패키지**의 결합입니다.

| 패키지 | 분량 | 목적 |
|--------|------|------|
| **출판기획서** | A4 5-10장 | 책 전체 그림을 1분 안에 전달 |
| **샘플 챕터** | A4 10-30장 (1-2 챕터) | 저자의 필력·완성도 직접 검증 |
| **마케팅 플랜** | A4 2-3장 | 출판사 부담을 줄일 저자의 영업력 |

출판사 편집장은 보통 **5분 안에 통과·반려 결정**. 첫 1분 안에 컨셉서·USP가, 다음 2분 안에 샘플 챕터 첫 5장이, 마지막 2분 안에 마케팅 플랜이 판단됩니다.

## 3. 워크플로우 (4단계)

### Step 1: 출판기획서 구성 (5 섹션)

출판기획서는 다음 5 섹션으로 구성합니다.

#### 1-1. 표지 (1 페이지)

| 요소 | 내용 |
|------|------|
| 가제 | book-concept-planner의 가제 |
| 한 줄 카피 | book-concept-planner의 한 줄 요약 (15자) |
| 30자 카피 | book-concept-planner의 30자 카피 |
| 저자 | 본명·필명 + 짧은 정체성 (50자 약력) |
| 장르·분량 | 5 장르 중 1 + 예상 매수 |
| 제출일·연락처 | 작성일 + 이메일·전화번호 |

#### 1-2. 책 개요 (A4 1장)

| 항목 | 분량 | 내용 |
|------|------|------|
| 300자 시놉시스 | 300자 | book-concept-planner의 300자 시놉시스 |
| USP 3축 | 200자 | book-concept-planner의 USP |
| 한 문장 요약 | 50자 | "이 책은 ___를 위한 ___입니다." |

#### 1-3. 시장·독자 분석 (A4 1-2장)

| 항목 | 내용 |
|------|------|
| 시장 사이즈 | KPIPA·교보문고 데이터 인용 (장르별 시장 규모·성장률) |
| 타깃 페르소나 | book-target-reader의 200자 페르소나 카드 요약 |
| JTBD 3 차원 | book-target-reader의 JTBD 요약 |
| 페인포인트 | book-target-reader의 긴급·핵심 페인 5개 |
| 경쟁작 매트릭스 | book-concept-planner의 4-8권 매트릭스 |

#### 1-4. 목차 + 시놉시스 (A4 2-3장)

| 항목 | 내용 |
|------|------|
| 전체 목차 트리 | book-outline-designer의 부·장·꼭지 트리 |
| 분량 배분 | 부·장별 분량 (200자 원고지 매수) |
| 챕터 시놉시스 | book-outline-designer의 5요소 시놉시스 (장 단위) |
| 부록·자료 | 템플릿·체크리스트·GitHub 등 |

#### 1-5. 저자 약력 (A4 1장)

| 항목 | 내용 |
|------|------|
| 200자 약력 | book-author-bio의 200자 |
| 신뢰 신호 매트릭스 | 5 영역 (학력·실무·저술·수상·팬덤) 표 |
| 이전 활동 | 강연·저서·SNS 팔로워·구독자 수 (수치 포함) |
| SNS·연락처 | 인스타·유튜브·블로그·이메일 |

### Step 2: 샘플 챕터 작성 (1-2 챕터)

샘플 챕터는 **본문 첫 인상**. 다음 원칙 준수:

#### 샘플 챕터 선정

| 선택 | 이유 |
|------|------|
| **1장 또는 2장** | 도입부의 필력·구성 검증 |
| **USP가 가장 잘 드러나는 장** | 책의 차별점을 미리 보여줌 |
| **꼭지 3-5개** | 1장 분량 + 1꼭지 추가 권장 |

#### 샘플 챕터 작성 원칙

1. **분량**: 12-30매 (장르·출판사별 다름)
2. **완성도**: 본문과 동일한 수준 (요약·초안 아님)
3. **시각 자료**: 도표·코드·삽화 모두 포함 (실용서·기술서)
4. **인용·각주**: 정확하게 표기 (인문서·기술서)
5. **장르 톤**: 5 장르 프리셋의 톤·문체 일관성

> 💡 출판사 편집장은 샘플 챕터 첫 5쪽으로 필력·완성도 판단. 첫 5쪽이 가장 강력해야 함.

### Step 3: 마케팅 플랜 작성 (A4 2-3장)

저자가 직접 가능한 마케팅 활동을 출판사에 제시. 출판사 부담을 줄이는 만큼 통과 확률 ↑.

#### 마케팅 플랜 5 카테고리

| 카테고리 | 활동 | 측정 지표 |
|---------|------|----------|
| **SNS 활동** | 인스타·유튜브·브런치·블로그 시리즈 | 팔로워·구독자·도달수 |
| **강연·강의** | 사내 강연·온라인 클래스·북토크 | 강연 횟수·참석자 |
| **언론·미디어** | 칼럼 기고·인터뷰·팟캐스트 출연 | 매체 수·도달 |
| **북클럽·커뮤니티** | 트레바리·문토·사내 북클럽 추천 | 북클럽 수·구성원 |
| **출간 이벤트** | 출간 기념 강연·사인회·온라인 라이브 | 행사 횟수·참석자 |

#### 마케팅 플랜 작성 공식

```
출간 전 (D-90 ~ D-1)
- [SNS] 인스타 100개 포스트 시리즈 / 유튜브 5편 영상
- [강연] 사내 강연 2회 / 외부 강연 1회 시범
- [언론] 칼럼 3편 기고 (매일경제·한경·서울신문 중 1)

출간 직후 (D+0 ~ D+30)
- [출간 이벤트] 온라인 북토크 1회 (300명 목표)
- [SNS] 인스타 라이브 / 유튜브 책 소개 영상 / 브런치 매거진
- [언론] 신문 서평·잡지 인터뷰 의뢰

출간 후 (D+30 ~ D+90)
- [강연] 기업·기관 강연 5회 / 온라인 클래스 1개 오픈
- [북클럽] 트레바리·문토 5개 그룹 등록·참가
- [커뮤니티] 독자 인증 캠페인 / 리뷰 50개 모으기
```

#### 출판사가 좋아하는 마케팅 플랜 특징

1. **숫자가 구체적** — "많이" 대신 "100개 포스트·5회 강연·300명 참석"
2. **이미 진행 중인 활동** — 출간 후 시작이 아닌 출간 전부터 보유한 채널 활용
3. **예산 부담 최소** — 광고 의존 ↓ 저자 직접 영업 ↑
4. **출판사 협업 명시** — "출판사 보도자료 + 저자 채널 동시 발송" 같은 협업 안

## 4. 5 장르 프리셋 (자동 분기)

### 4-1. 실용서·자기계발 (웅진·다산북스·길벗 양식)

| 차원 | 분기 |
|------|------|
| 표지 한 줄 카피 | 베네핏 직접 노출 ("30일 만에…") |
| 시장 분석 | 시장 사이즈·성장률·트렌드 수치 |
| 샘플 챕터 | Q&A·실습 박스·체크리스트 포함 |
| 마케팅 플랜 | 강연·클래스·기업 사내 강연 강조 |
| 분량 | A4 12-18장 (전체 제안서) |

### 4-2. 인문·교양·에세이 (민음사·문학동네·창비 양식)

| 차원 | 분기 |
|------|------|
| 표지 한 줄 카피 | 정체성·관점·은유 |
| 시장 분석 | 비슷한 작가·작품 인용 (시장 수치보다 작품) |
| 샘플 챕터 | 서사·문체·인용·각주 |
| 마케팅 플랜 | 북클럽·문학 강연·작가와의 대화 |
| 분량 | A4 10-15장 |

### 4-3. 기술·전문서·IT 입문서 (한빛미디어·인사이트·길벗 IT 양식)

| 차원 | 분기 |
|------|------|
| 표지 한 줄 카피 | "X 책 1권으로 Y까지" 패턴, 버전 명시 |
| 시장 분석 | 기술 채용·연봉·시장 동향 (잡코리아·원티드 데이터) |
| 샘플 챕터 | 코드 예제·도표·따라하기 단계 |
| 마케팅 플랜 | GitHub·기술 블로그·컨퍼런스 발표·온라인 강의 |
| 분량 | A4 15-20장 (코드 포함) |

### 4-4. 소설·문학 (민음사·문학동네·창비·문학동네신인상 양식)

| 차원 | 분기 |
|------|------|
| 표지 한 줄 카피 | 인물·갈등·세계관 압축 |
| 시장 분석 | 비슷한 작품·문학상 동향 |
| 샘플 챕터 | 완결된 1장 (단편소설은 전체 분량) |
| 인물 카드 | 주요 인물 3-7명 별도 |
| 마케팅 플랜 | 문학 행사·낭독회·작가와의 만남 |
| 분량 | A4 20-50장 (단편 전체 또는 장편 1-2장) |

### 4-5. 경험서·실무회고 (GIL 오리지널)

- 장르 목록 4→5에 경험서 추가. **양식은 실용서 양식(웅진·다산북스·길벗) 재사용**.
- 셀링 포인트: 저자의 현장 경험 고유성 + "판단 기준" 차별화.

## 6. 사용 예시

### 예시 1 — 실용서 (다산북스 투고)

```
"30일 챌린지로 시작하는 부업" 다산북스 투고 제안서 작성해줘.
```

## 8. 주의사항

### 거절 신호 사전 검출

다음 신호가 있으면 출판사 통과 어려움. 사전 보완 필수:

| 거절 신호 | 보완 방법 |
|----------|----------|
| **USP 약함** | book-concept-planner Step 3 재방문, 경쟁작 매트릭스 강화 |
| **타깃 모호** | book-target-reader Step 1-3 재방문, 페르소나 1명 구체화 |
| **목차 분량 불균형** | book-outline-designer Step 2·4 검증 통과 |
| **저자 신뢰 신호 부족** | book-author-bio 5 영역 중 2개 이상 강화 (강연·기고·SNS 등) |
| **샘플 챕터 미완성** | 본문 수준으로 완성 후 첫 5쪽 강화 |
| **마케팅 플랜 추상** | 숫자·기간·채널 모두 구체화 |
| **출판사 양식 무시** | 투고 출판사의 신간 5권 분석 후 양식 맞춤 |

### 투고 함정 회피

| 함정 | 피하는 법 |
|------|----------|
| **이력서식 제안서** | 책의 가치(USP·페르소나·시장)를 중심으로, 저자 약력은 후반 |
| **너무 긴 제안서** | A4 20장 초과 시 핵심 요약 추가, 부록 분리 |
| **숫자 부풀리기** | SNS 팔로워·강연 횟수 등 사실 정확성 필수 |
| **출판사 일괄 메일** | 받는 사람·인사말 일괄 복사 금지, 출판사별 맞춤 |
| **답변 재촉** | 답변 대기 2-6주 표준, 1개월 후 정중한 후속 메일 OK |
| **사적 정보 과다** | 가족·종교·정치 등 책과 무관한 사적 정보 금지 |

### 출처 정책

- 시장 데이터·인용은 **출처 명시** (KPIPA·교보문고·잡코리아 등)
- SNS 수치·강연 횟수 등 **사실 정확성** 필수
- vault 외부 참고 자료 직접 인용 금지
