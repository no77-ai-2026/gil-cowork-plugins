# book-publisher-matcher

> book-proposal-writer로 완성된 제안서를 **어느 출판사에 보낼지** 결정하는 스킬. 4 차원 평가(장르 적합도·규모·계약 조건·투고 채널)로 Top 5 출판사를 우선순위와 함께 추천. 30+ 한국 출판사 라이브러리에서 자동 매칭 + 거절 후 차순위 시나리오 동시 산출. **잘 만든 책도 잘못된 출판사에 보내면 거절**, 잘못 만든 책도 맞는 출판사에는 통과 가능성 있음.

## 1. 개요

한국 출판 시장에는 **약 3,500개 출판사**가 활동(KPIPA 통계 기준)하지만, 실제 신간 출간이 활발한 출판사는 **약 300-500개**. 책의 장르·규모·저자 신뢰 신호에 따라 매칭이 다릅니다.

| 매칭 기준 | 평가 차원 |
|----------|----------|
| **장르 적합도** | 출판사의 주력 장르와 책의 장르 일치 (가장 중요) |
| **규모** | 대형(연 200권+) / 중형(연 50-200권) / 소형(연 10-50권) / 1인 출판사 |
| **계약 조건** | 인세율·선인세·계약 기간·전자책 권리 |
| **투고 채널** | 이메일·홈페이지·신인공모·에이전시·소개 |

4 차원 평가로 Top 5를 선정하고 우선순위 적용. 1순위 거절 시 2-5순위로 순차 투고.

## 2. 트리거 키워드

## 3. 워크플로우 (4단계)

### Step 1: 매칭 입력 수집

다음 5 정보가 모두 필요합니다 (선행 스킬 산출물 인용):

| 입력 | 출처 |
|------|------|
| 책 장르 | book-concept-planner (5 장르 중 1) |
| 페르소나 | book-target-reader (200자 페르소나 카드) |
| 분량 | book-outline-designer (200자 원고지 기준 매수) |
| 저자 신뢰 신호 | book-author-bio (5 영역 매트릭스) |
| 저자 선호 | AskUserQuestion (자비 vs 출판사·대형 vs 소형 등) |

### Step 2: 4 차원 출판사 평가

30+ 출판사 라이브러리(§7)에서 다음 4 차원으로 점수화 (0-5 각):

#### 2-1. 장르 적합도 (가중치 40%)

| 점수 | 의미 |
|------|------|
| 5 | 책의 장르가 출판사 주력 장르 (예: IT 책 → 한빛미디어) |
| 3 | 출판사가 가끔 다루는 장르 |
| 1 | 출판사가 거의 다루지 않는 장르 |

#### 2-2. 규모 적합도 (가중치 25%)

| 책 규모 | 적합 출판사 규모 |
|--------|----------------|
| 대중적·시장 큼 (베스트셀러 가능) | 대형 출판사 |
| 중간 시장·특정 독자 | 중형 출판사 |
| 매니악·전문 | 소형·1인 출판사 |
| 실험적·표현 자유 | 자비 출판·독립 출판 |

#### 2-3. 계약 조건 (가중치 20%)

| 항목 | 한국 표준 |
|------|----------|
| 인세율 | 정가의 8-10% (대형) / 5-8% (소형) / 100% (자비 출판) |
| 선인세 | 500만원-3000만원 (베스트셀러 작가) / 0-100만원 (신인) |
| 계약 기간 | 5-10년 또는 무기한 |
| 전자책 권리 | 종이책과 함께 / 별도 협상 |
| 해외 판권 | 출판사 통합 / 저자 보유 협상 가능 |

#### 2-4. 투고 채널 (가중치 15%)

| 채널 | 통과 확률 |
|------|----------|
| 신인공모전 | 데뷔 작가 권장 (당선 시 계약 자동) |
| 에이전시 추천 | 통과율 높음 (5%~) |
| 출판사 편집자 소개 | 통과율 높음 |
| 이메일 일반 투고 | 통과율 낮음 (1-3%) |
| 자비 출판 대행 | 100% (저자 비용 부담) |

#### 가중 평균 계산

```
출판사 점수 = (장르적합도 × 0.4) + (규모적합도 × 0.25) + (계약조건 × 0.2) + (투고채널 × 0.15)
```

상위 5개 출판사 = Top 5.

### Step 3: Top 5 출판사 추천 + 투고 우선순위

Top 5에 대해 다음 정보 정리:

| 항목 | 내용 |
|------|------|
| **출판사명** | 본명 + 임프린트 (예: 웅진지식하우스 → 웅진 그룹) |
| **점수** | 4 차원 점수 + 가중 평균 |
| **투고 채널** | 이메일·홈페이지·신인공모 |
| **첫 미팅 대상** | 편집장·기획자·외부 에이전시 |
| **협상 포인트** | 인세·선인세·전자책·해외 판권 등 |
| **거절 후 대안** | 같은 출판사 다른 임프린트 / 차순위 출판사 |

### Step 4: 거절 후 차순위 시나리오

1순위 거절 시 자동 진행 시나리오:

```
1순위 → 2순위 (2~6주 대기 후 결과 확인)
2순위 거절 → 3순위·4순위 동시 투고 (시간 단축)
모두 거절 → 자비 출판 또는 독립 출판 펀딩 (부크크·텀블벅)
모두 거절 + 자비 부담 어려움 → 6개월 후 재투고 또는 작품 수정
```

차순위 진행 시 **제안서 수정 권장**: 1순위 거절 이유 추측 → 약점 보완 → 2순위 투고.

## 8. 주의사항

### 매칭 함정 회피

| 함정 | 피하는 법 |
|------|----------|
| **대형 출판사만 노림** | 작은 출판사가 더 정성스럽게 다룰 수 있음. Top 5에 중·소형 포함 |
| **장르 부적합 출판사 무리** | 장르 적합도 5점 우선, 3점 미만 출판사는 Top 5 제외 |
| **계약 조건 무시** | 인세·계약 기간·전자책·해외 판권 협상 포인트 사전 검토 |
| **신인공모 제외** | 신인은 신인공모전 통과율이 일반 투고보다 높음 (특히 문학) |
| **자비 출판 즉시 결정** | 1순위 거절 후 2-5순위 모두 시도, 자비는 최후 |
| **출판사명 오타** | "민음서"·"한빛 미디어"·"문학동내" 같은 오타 사전 검증 |

### 한국 출판 시장 현실

- **2025-2026 시장 동향**: 종이책 약 80%·전자책 약 20%, 오디오북 성장 중
- **신간 평균 발행 부수**: 신인 1,000-3,000부 / 베스트셀러 작가 10,000부+
- **인세 한국 표준**: 정가의 8-10% (대형 출판사 기준)
- **계약 기간**: 5-10년 또는 무기한 (전자책은 별도 협상 권장)
- **답변 대기**: 2-6주 (대형 출판사는 1-3개월)

### 출처 정책

- 출판사 정보·인세·연 출간 권수 등 사실 정확성 (출판사 홈페이지·KPIPA 자료 참조)
- vault 외부 참고 자료 직접 인용 금지
- 출판사 인세·연 출간 권수는 작성 시점 기준 명시 (변동 가능)
