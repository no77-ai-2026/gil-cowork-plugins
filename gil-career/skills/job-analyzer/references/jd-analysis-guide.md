# 채용공고(JD) 분석 가이드

> 2026년 채용 시장 기준

## JD 읽는 법

채용공고는 기업이 원하는 인재를 설명하는 문서입니다. 행간을 읽는 능력이 합격의 열쇠입니다.

### 구조 해독

| 섹션 | 읽는 포인트 |
|------|------------|
| 직무 소개 | 이 포지션이 팀에서 어떤 위치인지, 신설 vs 충원 |
| 담당 업무 | 첫 3개 항목이 핵심 업무, 나머지는 부수 업무 |
| 필수 자격 | 없으면 서류 통과 불가. 100% 충족 아니어도 80% 이상이면 지원 |
| 우대 사항 | 차별화 요소. 1-2개만 충족해도 경쟁력 |
| 근무 조건 | "협의 가능"은 협상 여지, "경력 무관"은 신입 환영 신호 |

### 숨겨진 의미 해독

```
"빠르게 성장하는 환경"  → 인력 부족, 멀티태스킹 필요
"자율적인 업무 환경"    → 스스로 찾아서 해야 함
"수평적 조직문화"       → 직급 구분 적음, 빠른 의사결정
"글로벌 환경"           → 영어 커뮤니케이션 필수
"데이터 기반 의사결정"  → SQL/분석 역량 필요
"최신 기술 스택"        → 학습 의지 중요, 레거시 적음
"팀과 함께 성장"        → 협업 중시, 개인 플레이 지양
```

## 필수/우대 조건 분석 프레임워크

### 역량 분류 매트릭스

```
              필수 조건          우대 조건
하드 스킬    [자격 요건]        [차별화 스킬]
소프트 스킬  [기본 역량]        [리더십/추가 역량]
경험         [최소 연차/경험]   [추가 경험]
자격         [필수 자격증]      [가산점 자격증]
```

### 역량 매칭 템플릿

| JD 요구사항 | 분류 | 내 보유 수준 | 근거(경험) | 자소서 활용 |
|------------|------|-------------|-----------|------------|
| Python 3년 | 하드/필수 | 2년 (프로젝트 3건) | 인턴+개인 | Action에 기술 |
| 팀 리더십 | 소프트/우대 | 동아리 회장 1년 | 리더십 경험 | 별도 항목 |

## 기업 리서치 체크리스트

### 1. 기본 정보

- [ ] 설립연도, 대표자, 직원 수, 매출 규모
- [ ] 사업 영역, 주요 제품/서비스
- [ ] 본사 위치, 지사/해외법인
- [ ] 최근 투자 유치 또는 IPO 계획

### 2. 재무 건전성

- [ ] 최근 3년 매출 추이 (DART 전자공시)
- [ ] 영업이익률, 순이익률
- [ ] 부채비율, 현금 흐름
- [ ] 업종 평균 대비 재무 상태

### 3. 조직 문화

- [ ] 잡플래닛/블라인드 리뷰 (최근 6개월)
- [ ] 채용 페이지의 복지/문화 소개
- [ ] CEO/임원 인터뷰 기사
- [ ] 기업 블로그/기술 블로그 콘텐츠

### 4. 산업 동향

- [ ] 시장 규모, 성장률
- [ ] 주요 경쟁사 3-5개
- [ ] 최근 M&A, 신사업 진출
- [ ] 규제 환경 변화

### 5. 채용 패턴

- [ ] 공채/수시 채용 주기
- [ ] 과거 채용공고와 비교 (요구사항 변화)
- [ ] 면접 후기 (잡플래닛, 블라인드)
- [ ] 인재상, 핵심 가치 키워드

## 2026년 주요 채용 트렌드 참고

**스킬 기반 채용(Skills First):**
실무 테스트와 포트폴리오 평가 비중이 전체의 60%를 넘었습니다. JD에서 "관련 경험"을 요구할 때, 학위나 자격증보다 프로젝트 경험과 실무 성과가 더 높게 평가됩니다.

**AI 역량 필수화:**
직무 전문성(64.7%) 다음으로 중요한 역량이 AI 도구 활용입니다. JD에 "AI 활용 경험 우대"가 없더라도, 면접에서 AI 활용 경험을 물어보는 비율이 높아졌습니다.

**팀핏(Team Fit) 평가:**
"팀 분위기와 팀원 간 정서적 연결"을 중시하는 기업이 증가했습니다. JD에서 "협업 능력", "커뮤니케이션"을 강조하는 경우 팀 면접이 포함될 가능성이 높습니다.
