## 5. 채용 프로세스 (UZ)

### 일반 (UZ 현지 기업)

```
1. 공고 (HeadHunter·Telegram)
2. CV·이력서 제출
3. 1차 면접 (HR, 30분)
4. 2차 면접 (실무·기술, 1~2시간)
5. 3차 면접 (이사·경영진, 30~60분)
6. Offer 발급
```

### 한국 진출 외국 기업

```
1. 공고 (LinkedIn·KOTRA·본사 사이트)
2. CV (영어·러시아어, 한국어 옵션)
3. 1차 (HR, 영어/러시아어)
4. 2차 (실무, 영어/러시아어)
5. 3차 (한국 본사·이사, 영어 또는 한국어)
6. Background check
7. Offer
```

### Big Tech·외국계 (영어권)

```
1. LinkedIn·Recruiter
2. Phone screen (영어)
3. Technical (Coding·System Design — IT)
4. Behavioral (영어 STAR)
5. Onsite·Final (HQ 또는 원격)
6. Offer
```

#### 옵션 2: 외국계 (영어권)
