# 면접 유형별 대비 가이드

> 2026년 채용 시장 기준

## AI 면접 (AI 역량검사)

뇌신경과학 기반으로 지원자의 사고·행동 패턴을 분석하여 기업 문화 및 직무 적합성을 평가합니다. 2026년 기준 대기업 320개 이상 도입.

### 진행 단계 (총 1시간 30분-2시간)

| 단계 | 시간 | 평가 요소 |
|------|------|----------|
| 1. 자기소개 | 1분 | 첫인상, 자신감, 요약력 |
| 2. 기본 질문 | 10분 | 지원동기, 직무 이해도 |
| 3. 성향 파악 | 15분 | 선택형 문항, 일관성 |
| 4. 상황 대처 | 20분 | 갈등 해결, 윤리적 판단 |
| 5. 보상 선호 | 10분 | 동기 부여 요인 분석 |
| 6. 전략 게임 | 20분 | 인지력, 판단력, 전략적 사고 |
| 7. 심층 대화 | 15분 | AI 꼬리질문, 논리적 사고 |

### AI 분석 대상

- **텍스트**: 답변 키워드 추출, 논리 구조 분석
- **표정**: 미세 표정 변화, 눈 맞춤(카메라 응시), 자연스러움
- **음성**: 발화 속도, 톤 변화, 자신감 수준, 머뭇거림

### 대비 팁

1. 조용한 환경, 밝은 조명, 카메라 정면 응시
2. 답변 중 시선을 카메라에서 벗어나지 않기
3. 일정한 발화 속도 유지 (분당 250-300자)
4. 전략 게임은 마이다스AI, 뷰인터HR 모의 테스트로 연습
5. 성향 문항에서 극단적 선택 회피, 일관성 유지

### 도입 기업 (2026년 기준)

기아, 농심, 롯데그룹, LG유플러스, SK하이닉스, KT그룹, KB국민은행, CJ그룹, 한화, 현대건설, GS리테일, 아모레퍼시픽 등

## 역량 면접 (BEI: Behavioral Event Interview)

과거 행동을 통해 미래 성과를 예측하는 구조화 면접입니다. 대기업 실무면접의 주류 방식입니다.

### 질문 패턴

```
"~한 경험을 구체적으로 말해주세요"
"그때 본인이 한 역할은 무엇이었나요?"
"왜 그런 선택을 했나요?"
"결과는 어땠나요? 다시 한다면 어떻게 하겠나요?"
```

### 평가 기준

| 기준 | 점수 | 설명 |
|------|------|------|
| 구체성 | 상/중/하 | 시간, 장소, 인원 등 디테일 |
| 본인 역할 | 상/중/하 | "우리 팀"이 아닌 "내가" 한 것 |
| 행동의 논리 | 상/중/하 | 왜 그 행동을 선택했는가 |
| 결과 정량화 | 상/중/하 | 수치, 지표, 변화 측정 |
| 성찰 수준 | 상/중/하 | 배운 점, 개선 방향 |

### 대비 팁

- 사전에 STAR 경험 카드 10장 준비
- 경험당 2분 이내로 요약 연습
- 꼬리질문 예상: "왜?", "구체적으로?", "다시 한다면?"

## PT 면접

주어진 주제에 대해 30-60분 준비 후 10-15분 발표, 이후 Q&A를 진행합니다.

### 주제 유형

| 유형 | 예시 |
|------|------|
| 산업 이슈 | "AI가 금융업에 미치는 영향과 대응 전략" |
| 기업 전략 | "당사의 해외 시장 진출 전략을 제안하시오" |
| 직무 과제 | "신규 서비스 론칭 마케팅 계획을 수립하시오" |
| 시사 이슈 | "ESG 경영의 실질적 효과와 한계" |

### 발표 구조 (10분 기준)

```
도입 (1분): 주제 정의 + 핵심 메시지
본론 (7분):
  1. 현황 분석 (2분)
  2. 핵심 제안 3가지 (4분)
  3. 기대 효과 (1분)
마무리 (2분): 요약 + 실행 로드맵
```

### 대비 팁

- 경제/산업 뉴스 매일 1개씩 요약 연습
- 3C(Company/Customer/Competitor) 프레임워크 활용
- 시간 내 완료 못하면 감점, 타이머 필수

## 토론 면접

4-6명이 찬반 또는 자유 토론을 진행합니다. 30-60분 소요됩니다.

### 유형

| 유형 | 설명 | 평가 포인트 |
|------|------|------------|
| 찬반 토론 | 주제에 찬성/반대 배정 | 논리적 근거, 반론 대응 |
| 자유 토론 | 열린 주제로 자유 논의 | 주도성, 경청, 합의 도출 |
| 과제 해결 | 그룹으로 문제 해결 | 협업, 역할 분담, 시간 관리 |

### 역할 전략

- **사회자/진행자**: 리더십 어필, 발언 기회 균등 배분
- **타임키퍼**: 책임감 어필, 시간 관리 능력
- **발표자**: 표현력 어필, 논의 결과 요약력

### 대비 팁

- 공격적 반박 지양, 건설적 반론 제시
- "~님의 의견에 추가하면" 식의 연결 화법
- 발언 비율: 전체의 20-30%가 적정 (너무 많아도 감점)

## 임원 면접

최종 관문으로 인성, 조직 적합성, 비전을 종합 평가합니다. 10-20분 소요됩니다.

### 빈출 질문

1. "우리 회사를 선택한 진짜 이유가 뭔가요?"
2. "본인의 가장 큰 실패 경험은?"
3. "5년 후 어떤 모습이고 싶나요?"
4. "우리 회사의 가장 큰 도전 과제는 뭐라고 생각하나요?"
5. "연봉 외에 회사에서 가장 중요하게 여기는 것은?"

### 대비 팁

- 기업 CEO 인터뷰, 최근 보도자료 필수 확인
- 진정성 있는 태도 (암기한 답변은 역효과)
- 역질문 1-2개 준비: "팀의 현재 가장 큰 과제는 무엇인가요?"
