# 면접 빈출 질문 30선

> 2026년 채용 시장 기준, 직무면접/인성면접/임원면접 통합

## 자기소개/지원동기 (5문항)

1. **1분 자기소개 해주세요.**
   - 프레임: 이름 → 핵심 역량 1가지 → 대표 경험 → 지원 연결
   - 시간: 45초-1분 (300자 내외)

2. **왜 이 직무에 지원했나요?**
   - 프레임: 직무 이해 → 내 역량 매칭 → 기여 포인트

3. **왜 우리 회사에 지원했나요?**
   - 프레임: 기업 강점(구체적) → 내 가치관 연결 → 기여 비전
   - 주의: "복지가 좋아서", "대기업이라서"는 감점 요인

4. **다른 회사도 지원했나요?**
   - 프레임: 솔직하게 인정 → 이 회사가 1순위인 이유 → 직무 일관성

5. **입사 후 5년 내 목표는?**
   - 프레임: 1-2년 실무 성장 → 3-5년 전문가/리더 성장 → 조직 기여

## 답변 프레임워크 (범용)

### 두괄식 STAR

```
[결론] "저의 강점은 ~입니다."
[S] "~때 ~한 상황이 있었습니다."
[T] "저는 ~를 맡아 ~해야 했습니다."
[A] "그래서 ~하고, ~했습니다."
[R] "그 결과 ~를 달성했고, ~를 배웠습니다."
```
