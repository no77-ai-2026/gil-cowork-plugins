# 트릴링구얼 이력서·CV 가이드

> gil-career v0.1.0 | 한국어·영어·러시아어·우즈벡어 이력서

## 개요

UZ 진출 한국인·UZ 인재 한국 진출·다국적 글로벌 진출 시 트릴링구얼 (또는 다국어) 이력서·CV. 언어별 형식·문화 코드 차이.

---

## 1. 4가지 트릴링구얼 옵션

### 옵션 A: 단일 언어 + 보조 (가장 일반)

- 메인 언어 1개 (지원 회사 표준)
- 핵심 정보 영어 보조
- 1~2 페이지

예:
- 한국 회사 → 한국어 메인 + 영어 보조
- UZ 자회사 → 러시아어 메인 + 한국어 보조
- 글로벌 → 영어 메인 + 한국어 보조

### 옵션 B: 양국어 페이지 분리

- 1페이지: 영어
- 2페이지: 한국어 또는 러시아어

→ Big Tech·다국적·KOTRA 권장.

### 옵션 C: 트릴링구얼 매트릭스 (단일 페이지)

- 1페이지에 한국어·영어·러시아어 동시
- 정보 압축 필수
- 트릴링구얼 인재 강조 효과

### 옵션 D: 다국어 사이트 (포트폴리오 통합)

- 개인 웹사이트
- /ko /en /ru /uz 분리
- LinkedIn 다국어 프로필

## 2. 언어별 이력서 표준

### 한국 (한국어)

```
[사진] (블라인드 시 X)

[기본 정보]
이름·생년월일·연락처·이메일·주소

[학력]
대학·전공·학번·졸업

[경력]
회사·직급·기간·담당 업무

[자격증·외국어]

[프로젝트·수상]

[자기 소개서] (별도)
```

특징:
- A4 1~2장
- 사진 (선택, 블라인드 시 X)
- 한글 (영문 보조)
- 격식 문체

### 영어 (Resume - 미국·국제 표준)

```
[NAME]
City, Country | Phone | Email | LinkedIn | GitHub

[SUMMARY] (3~5 lines)
[EXPERIENCE]
Company | Title | Dates
- Achievement (Action verb + quantified result)

[EDUCATION]
University | Degree | Dates

[SKILLS]
Languages, Frameworks, Tools

[PROJECTS]
[CERTIFICATIONS]
[LANGUAGES] - English (Native), Korean (Fluent), Russian (B2), Uzbek (A2)
```

특징:
- 1 page (신입·중급) / 2 pages (시니어)
- 사진 X (미국·EU 표준)
- Reverse chronological
- Action verbs (Led·Designed·Implemented)
- Quantified achievements

### 영어 (CV - 영국·학술)

Resume 대비 더 길게 (2~5 pages):
- 학력 상세 (논문·학회)
- 출판물·연구
- 강의·발표

### 러시아어 (Резюме)

```
[ФИО] - 성명
[Дата рождения] - 생년월일
[Контакты] - 연락처

[Желаемая должность] - 희망 직무

[Опыт работы] - 경력
Компания | Должность | Период
- 담당·성과

[Образование] - 학력

[Навыки] - 스킬
[Языки] - 언어 능력

[Дополнительная информация] - 추가
```

특징:
- 사진 (UZ·러시아 표준 — 포함)
- 러시아어 키릴
- 정중한 톤

### 우즈벡어 (Rezyume - 라틴)

```
[F.I.Sh.] - 성명 (Familiya·Ism·Sharif)

[Tug'ilgan sana] - 생년월일
[Aloqa ma'lumotlari] - 연락처

[Maqsadli lavozim] - 희망 직무

[Ish tajribasi] - 경력

[Ta'lim] - 학력

[Mahoratlar] - 스킬
[Tillar] - 언어
```

특징:
- 우즈벡 라틴 알파벳
- 사진 (UZ 표준)
- 정중한 톤

## 6. 번역 워크플로우

### 표준 절차

```
1. 마스터 작성 (한국어 또는 영어)

2. 번역 (전문 번역 또는 AI 초안)
   - 한국어 → 영어: DeepL·Google
   - 한국어 → 러시아어: DeepL·Yandex
   - 한국어 → 우즈벡어: Google (제한)

3. 현지인 검수 (필수)
   - 영어: Native speaker 또는 Grammarly
   - 러시아어: TORFL 강사·UZ 동료
   - 우즈벡어: KOTRA Tashkent 또는 현지

4. 형식 통일 (사진·연락처·날짜)

5. PDF 변환 (편집 잠금)
```

### AI 번역 한계

- 학술·전문 용어 부정확
- 문화 코드 무시
- 정중함·존대 미숙

→ 현지 검수 필수.

## 11. 흔한 실수

1. AI 번역만 → 어색함 → 현지 검수 필수
2. 사진 (영미 회사) → 차별 우려
3. 직급 직역 → 회사 문화 미반영
4. 트릴링구얼 매트릭스 너무 빽빽 → 가독성 ↓
5. 언어별 다른 정보 → 일관성 ↓
6. 한국어만 → 글로벌 도달 ↓

## 13. 추천 트랙별 옵션

| 트랙 | 추천 옵션 |
|------|---------|
| 한국 학생 → 한국 회사 | 한국어 단독 + LinkedIn 영어 |
| 한국 학생 → UZ 진출 | 한국어 마스터 + 영어·러시아어 보조 |
| 한국 학생 → Big Tech | 영어 단독 (Resume 1p) |
| UZ 학생 → 한국 진출 | 한국어 + 영어 (트릴링구얼 LinkedIn) |
| UZ 학생 → 글로벌 | 영어 + 러시아어 |
| 한국 직원 → UZ 자회사 파견 | 한국어 + 러시아어 (인터컴퍼니용) |
| 다국적 글로벌 | 영어 + LinkedIn 다국어 |
