# 우즈베키스탄 근로계약서 가이드

> gil-hr v0.1.0 | UZ 근로계약서 작성·외국인 노동허가·HQ Visa

## 개요

UZ 노동법 (Трудовой кодекс)에 따른 근로계약서 작성, 외국인 직원 노동허가, IT Park HQ Visa, 한국인 임원 UZ 파견 모델을 정리한 실무 가이드입니다.

---

## 1. UZ 근로계약서 (Трудовой договор) 핵심

### 법적 근거
- Трудовой кодекс Республики Узбекистан (UZ 노동법전)
- 외국인 노동허가 관련 대통령령
- IT Park HQ Visa 정책 (it-park.uz)

### 필수 포함 사항

```
ТРУДОВОЙ ДОГОВОР № [번호]
[도시명], [날짜]

1. СТОРОНЫ ДОГОВОРА (양 당사자)
- Работодатель (사업주): [회사명, 주소, INN]
- Работник (직원): [이름, 여권번호 또는 ID, 주소]

2. ДОЛЖНОСТЬ И МЕСТО РАБОТЫ (직무·근무지)
- Должность (직무): [직무명]
- Подразделение (부서): [부서명]
- Место работы (근무지): [주소]

3. СРОК ДЕЙСТВИЯ (계약 기간)
- 무기 (Бессрочный) 또는
- 기간제 (Срочный — 최대 5년)

4. РАБОЧЕЕ ВРЕМЯ (근무 시간)
- 주 40시간 (표준)
- 주 35시간 (야간 근무·청소년 직원)
- 휴게시간: 식사 시간 1시간

5. ОПЛАТА ТРУДА (임금)
- 기본급 (Должностной оклад): X UZS/월
- 추가 수당 (Доплаты)
- 상여금 (Премии)
- 지급일 (Срок выплаты): 매월 X일
- 통화: UZS (외환 옵션 가능)

6. ОТПУСКА (휴가)
- 연차 (Ежегодный отпуск): 최소 21일/년 (UZ 노동법)
- 추가 휴가 (Дополнительные отпуска): 야간 근무·위험 작업
- 출산 휴가 (Декретный отпуск): 70+56일 (출산 전후)

7. СОЦИАЛЬНОЕ СТРАХОВАНИЕ (사회보험)
- Социальный налог: 사업주 25% (IT Park 12.5%) + 직원 12%
- NDFL (개인소득세): 12% 원천징수

8. ОБЯЗАННОСТИ СТОРОН (양 당사자 의무)
- 직원 의무 (성실 근무, 비밀 유지)
- 사업주 의무 (임금 지급, 안전 환경)

9. ПРЕКРАЩЕНИЕ ДОГОВОРА (계약 해지)
- 양 당사자 합의
- 직원 통보 (최소 14일 전)
- 사업주 통보 (사유 있는 경우, 14일~2개월)

10. КОНФИДЕНЦИАЛЬНОСТЬ (기밀 유지)

11. ПРОЧИЕ УСЛОВИЯ (기타 조건)

12. РЕКВИЗИТЫ И ПОДПИСИ СТОРОН (양 당사자 정보 및 서명)
```

### 언어 요건
- **공식 언어**: 우즈벡어 (의무)
- **병기 표준**: 러시아어 (실무 표준)
- **외국인 직원**: 직원 모국어 추가 (한국어 등) 권장
- **분쟁 시**: 우즈벡어 원본이 우선

### 등록 의무
- 계약 체결 후 [mehnat.uz](https://mehnat.uz) 또는 [my.gov.uz](https://my.gov.uz)에 전자 등록 의무
- 미등록 시 사업주에 과태료

## 2. UZ 직원 임금 구조

### 임금 구성 표준

```
Заработная плата (총 급여)
  Должностной оклад (기본급) — 기본 임금
  Доплаты (수당)
    - За сложность (난이도 수당)
    - За интенсивность (집중도 수당)
    - За работу в выходные (휴일 근무 수당)
    - Сверхурочные (야근 수당, 표준 임금 1.5~2배)
  Премии (성과급)
  Компенсации (보상금 — 출장비, 식대 등)
```

### 차감 (직원 부담)

```
- NDFL 12%
- 사회세 12%
- 합계: 24%
- Net 급여 = Gross × 0.76
```

### 사업주 추가 부담

```
- 사회세 25% (IT Park 12.5%)
- 산재보험 (업종별 차등)
```

### 환율 기준 (참고, 2026년 4월 추정)

```
1 USD = 12,750 UZS
1 KRW = 9.2 UZS

월 급여 10,000,000 UZS (Gross)
- NDFL 1,200,000
- 사회세(직원) 1,200,000
- Net 7,600,000 UZS ≈ $596 ≈ KRW 825,000

총 인건비 (사업주 부담 포함):
- Gross 10,000,000
- + 사회세(사업주) 2,500,000
- = 12,500,000 UZS ≈ $980 ≈ KRW 1,355,000
- (IT Park: 11,250,000 UZS ≈ $882)
```

## 3. 외국인 노동허가 (Разрешение на привлечение иностранной рабочей силы)

### 일반 절차 (mehnat.uz)

1. **회사 자격 검토**: UZ 등록 법인 (ИЧТ/ООО)
2. **노동허가 신청**: mehnat.uz 또는 노동부 직접 (1~3개월 소요)
3. **할당 (Квота) 확인**: 산업별 외국인 채용 한도
4. **노동허가 취득**: 회사가 직원당 신청 (개인 신청 불가)
5. **근로계약 체결**: 노동허가 승인 후
6. **거주증 (RP) 신청**: 1년 이상 체류 시 (별도 절차)
7. **다중 입국 비자**: 기간 따라 종류 결정

### IT Park 거주자 특례 (HQ Visa)

[it-park.uz](https://it-park.uz) 거주자 회사의 외국인 임원·전문가는 HQ Visa로 패스트트랙:

- **유효 기간**: 3년 (자동 갱신 가능)
- **다중 입국**: 가능
- **가족 동반**: 배우자·자녀 동반 비자
- **노동허가 통합**: HQ Visa 발급 시 노동허가 자동 부여
- **신청 절차**:
  1. IT Park 거주자 회사가 임원 추천
  2. it-park.uz 시스템 신청
  3. 외무부·내무부 협력 심사 (2~4주)
  4. HQ Visa 발급

### SEZ 입주 기업 특례

자유경제지대 입주 기업의 외국인 임원도 간소화 절차 적용 (구체 조건은 SEZ별 상이).

## 4. 한국인 UZ 파견 듀얼 모델

### 옵션 A: 한국 본사 단독 고용 + UZ 파견

**구조**:
- 한국 본사가 KRW 급여 + 해외 수당 지급
- UZ 측 노동허가서 + 비자 별도 발급
- UZ 자회사는 임원 자리만 제공 (실고용 X)

**세무**:
- 한국 사회보험 유지 (한·우 사회보장협정 검토)
- 한국 종합소득세 (전세계 소득)
- UZ 거주자 (183일+) 시 NDFL 적용 → 한국 측 외국납부세액공제

**장점**: 단순 구조, 한국 인사 정책 그대로 적용
**단점**: UZ 자회사 거버넌스 약함 (외부 컨설턴트로 분류 위험)

### 옵션 B: 한국 본사 + UZ 자회사 듀얼 고용

**구조**:
- 양국 근로계약서 동시 체결
- 양국에서 각각 급여 지급 (한국 KRW + UZ UZS)
- 양국 사회보험 동시 가입

**세무**:
- 한국 4대보험 + UZ 사회세
- 한·우 DTA 적용 → 외국납부세액공제

**장점**: 거버넌스 명확, 양국 직원 신뢰
**단점**: 행정 복잡, 양국 세무·사회보험 이중 신고

### 옵션 C: 한국 본사 휴직 + UZ 자회사 단독 고용

**구조**:
- 한국 측 휴직 처리 (1~3년)
- UZ 자회사가 단독 고용 (UZS 급여)

**세무**:
- UZ NDFL + 사회세만
- 한국 측 비거주자 → 한국 종합소득세 면제 가능
- 외국인 임원 = HQ Visa (IT Park) 또는 일반 노동허가

**장점**: 깔끔한 단일 고용, UZ 측 절세
**단점**: 한국 본사 복귀 시 직급 협상 필요

### 권장 선택 가이드

```
조건                           | 권장 옵션
-----------------------------------|----------
1년 미만 단기 파견              | A
1~3년 중기 파견 + 한국 본사 유지 | B
3년+ UZ 정착 + IT Park 거주자  | C
한국 본사 거버넌스 강화 필요    | B
세제 최적화 우선               | C (IT Park) or B
```

## 5. UZ 직원 휴가·복지

### 의무 휴가
- 연차 (Основной отпуск): 21일/년 이상 (사용 의무)
- 추가 휴가 (Дополнительный отпуск): 야간·위험 작업, 18~21세 청소년
- 공휴일 (Праздничные дни): UZ 공식 공휴일 (Navruz, Eid 등)
- 출산 휴가 (Декретный отпуск): 70+56일 (출산 전 70일, 출산 후 56일)
- 육아 휴직 (Отпуск по уходу за ребенком): 자녀 만 2세까지

### 복지 의무
- 건강검진 (해당 산업)
- 안전 교육 (업무 시작 전)
- 작업복·보호구 (해당 직무)

## 6. UZ 직원 해고 (Увольнение)

### 합법적 해고 사유
- 양 당사자 합의 (소장 작성)
- 직원 자발적 사직 (14일 통보)
- 사업주 사유 — 14일~2개월 통보
  - 정리해고 (Сокращение): 2개월 통보 + 보상금
  - 부적합 (Несоответствие): 평가 결과 + 통보
  - 중대한 위반 (Грубое нарушение): 즉시 해고 가능

### 해고 보상
- 정리해고: 1~3개월 임금 + 잔여 연차
- 양 당사자 합의: 협상

### 분쟁 처리
- 노동 분쟁 위원회 (Комиссия по трудовым спорам)
- 법원 (Суд)

## 7. 검증 체크리스트

근로계약서 작성 후 확인:

- [ ] 우즈벡어 + 러시아어 양식 동시 작성
- [ ] 직원 모국어 (한국어 등) 추가 (외국인 시)
- [ ] mehnat.uz 또는 my.gov.uz 전자 등록 완료
- [ ] 외국인 노동허가서 또는 HQ Visa 사본 첨부
- [ ] 거주증 (RP) 사본 (1년 이상 체류 시)
- [ ] NDFL/사회세 신고 시작 (다음달 15일·25일)
- [ ] 4대보험·사회세 사업주 부담분 납부

한국인 파견 시 추가:

- [ ] 한국 본사 측 처리 (휴직 / 듀얼 고용 / 파견)
- [ ] 한·우 사회보장협정 적용 검토
- [ ] 한국 종합소득세 신고 계획 (이중 거주자 처리)
- [ ] DTA 외국납부세액공제 활용
