### UZ JD 작성 팁
- **임금 표시**: USD가 표준 (UZ 인플레이션 영향). UZS만 쓰면 매력도 ↓
- **IT Park 거주자 명시**: 0% 법인세 + 외국인 비자 패스트트랙은 인재 매력도 +
- **언어 요구**: 러시아어 필수, 영어 B2+ 권장 (글로벌 IT)
- **원격/하이브리드 허용**: 채용 풀 확대 (UZ 외 사마르칸트·페르가나 인재)
- **보너스 명시**: KPI/OKR 보너스, 13월급 (러시아·CIS 표준)

### 우즈벡어 JD (정부·관공서 대상)

UZ 정부와 거래하는 사업이거나 우즈벡 직원 우선이면 우즈벡어 JD 추가:
- 헤더만 우즈벡어, 본문은 러시아어로 가능
- 또는 양 언어 병기

## 3. UZ 면접 프로세스

### 표준 단계
1. **서류 검토**: hh.uz 응답 + 추천 인재 (1~2주)
2. **HR 인터뷰** (30분): 동기·기대 연봉·근무 조건
3. **기술 인터뷰** (1~2시간): 직무 역량 (코딩 테스트·실무 케이스)
4. **팀 인터뷰** (1시간): 협업·문화 적합성
5. **임원 인터뷰** (30~45분): 비전·장기 의지
6. **레퍼런스 체크** (선택): 이전 회사 동료
7. **오퍼**: 1주 검토 후 회신 표준

### UZ 면접 문화
- **언어 옵션**: 러시아어 우선, 영어 가능 (글로벌 IT), 우즈벡어 (현지 비즈니스)
- **협상**: 임금 협상은 표준 (5~15% 상향 흔함)
- **트렌드**: IT Park 거주자 회사 선호도 ↑ (세제 혜택 + 외국인 비자)
- **소요 시간**: 한국 대비 빠른 결정 (전체 2~4주)

## 4. 외국인 직원 채용 절차

### 일반 외국인 직원 (한국인 시니어 개발자 등)

```
Step 1: 회사 자격 확인
- UZ 법인 등록 (ИЧТ 또는 ООО)
- 사업자 INN/STIR 보유

Step 2: 노동허가 할당 (Квота) 확인
- 산업별 외국인 채용 한도
- mehnat.uz 또는 노동부 직접 문의

Step 3: 노동허가 신청 (회사가 신청)
- mehnat.uz 또는 my.gov.uz 온라인
- 직원 여권 사본·이력서·교육 증빙·고용 의도서
- 처리 기간: 1~3개월

Step 4: 비자 신청 (직원이 신청)
- E-Visa 또는 일반 노동 비자 (1년)
- 한국 외교부 또는 UZ 영사관

Step 5: UZ 입국 + 등록
- 입국 후 3일 이내 거주지 등록 (호텔이 자동 또는 본인 신청)
- 1년 이상 체류 시 거주증 (RP)

Step 6: 근로계약 체결 + mehnat.uz 등록

Step 7: 4대보험 (NDFL + 사회세) 등록
```

### IT Park 거주자 회사 외국인 임원 (HQ Visa 패스트트랙)

```
Step 1: IT Park 거주자 자격 확인 (회사)
- it-park.uz 거주자 라이선스 보유

Step 2: 임원 추천 신청
- IT Park 거주자 회사가 임원 추천
- it-park.uz 온라인 시스템

Step 3: 외무부·내무부 심사 (2~4주, 일반 노동허가의 1/3 시간)

Step 4: HQ Visa 발급
- 3년 유효, 자동 갱신
- 노동허가 + 비자 통합 (별도 신청 불필요)
- 가족 동반 비자

Step 5: 입국 + 거주지 등록

Step 6: 근로계약 체결
```

## 7. 한국 진출 기업 채용 베스트 프랙티스

### 한국어 가능 인재 우선
- IUT (Inha University Tashkent) 졸업생 (한국어 + IT)
- 한국 유학파 (KAIST·서울대 등)
- 재타슈켄트 한인 (한국어 + 러시아어)

### 한·우 듀얼 채용 모델
- **모델 1**: 한국인 임원 (HQ Visa) + 현지 매니저 + UZ 직원
- **모델 2**: 한국인 컨설턴트 (출장 비자) + 100% 현지 채용
- **모델 3**: 100% 현지 채용 + 본사 원격 협업 (한국어 가능 인재 핵심)

### 채용 후 정착 지원
- 한국식 복지 + UZ 표준 (양국 모두 만족)
- 식대: 한국 표준 (월 20만원 비과세) + UZ 점심 식대 (현지 표준)
- 교육비 지원: 영어·한국어 학습 보조
- 주거 지원: 외국인 임원 (Tashkent 외국인 거주지 임대료 지원)
