# speech-video — 토킹 헤드 비디오 생성 스킬

## 개요 (Overview)

텍스트를 읽는 "토킹 헤드" 비디오를 생성하는 전용 스킬입니다. **Higgsfield speech-to-video** 모델을 사용하여 브랜드 캐릭터가 자연스럽게 말하는 비디오를 만듭니다.

**핵심 특징:**
- **텍스트-투-스피치 비디오**: 입력된 텍스트를 캐릭터가 자연스럽게 읽는 비디오 생성
- **브랜드 캐릭터 참조**: 미리 생성된 브랜드 캐릭터를 일관되게 사용
- **다양한 보이스 스타일**: 친근한, 전문적인, 활기찬 등 다양한 톤 지원
- **세로형 비디오 지원**: 9:16 비율로 릴스, 쇼츠, 틱톡에 최적화

## 트리거 키워드 (Trigger Keywords)

이 스킬은 다음 상황에서 자동으로 호출됩니다:

- **토킹 헤드**: "말하는 비디오", "토킹 헤드", "사람이 말하는 영상"
- **캐릭터 스피치**: "캐릭터가 말하는 비디오", "마스코트가 제품 소개", "브랜드 캐릭터 스피치"
- **제품 설명**: "제품 설명 동영상", "상품 소개 비디오", "제품 홍보 영상"
- **교육 콘텐츠**: "교육용 강사 비디오", "가상 강사", "강의 영상"
- **SNS 콘텐츠**: "릴스용 토킹 헤드", "쇼츠용 말하는 영상", "틱톡 인물 영상"
- **스크립트 기반**: "이 스크립트로 비디오 만들어줘", "대사를 읽는 비디오"

## 워크플로우 (Workflow)

### 1단계: 사전 요구사항 확인

**필수 조건:**
- 브랜드 캐릭터가 먼저 생성되어 있어야 합니다.
- 캐릭터가 없으면 먼저 `/character-mgmt` 스킬을 실행하여 캐릭터를 생성하세요.

### 2단계: 스크립트 및 파라미터 확인

- **스크립트**: 비디오에서 읽을 텍스트 (한글 또는 영어)
- **캐릭터 ID**: `character-mgmt` 스킬로 생성된 캐릭터 ID
- **보이스 스타일**: 친근한(friendly), 전문적인(professional), 활기찬(energetic), 차분한(calm)
- **길이**: 비디오 길이 (초 단위, 기본값 5)
- **화면 비율**: 16:9 (가로), 9:16 (세로), 1:1 (정사각형)

### 3단계: 캐릭터 ID 확인 (character-mgmt 연동)

```
사용자가 캐릭터 ID를 모르면?
  → character-mgmt 스킬의 list 명령으로 캐릭터 목록 조회
  → 적절한 캐릭터 선택 후 ID 확보
```

### 4단계: Higgsfield speech-to-video 생성

**MCP 툴 파라미터:**
- `text`: 스크립트 내용 (필수)
- `character_id`: 브랜드 캐릭터 ID (필수)
- `voice_style`: 보이스 스타일 (선택, 기본값 "friendly")
- `duration`: 비디오 길이 (선택, 기본값 5)
- `aspect_ratio`: 화면 비율 (선택, 기본값 "16:9")
- `seed`: 재현성을 위한 시드 값 (선택)

### 5단계: 결과 전달

생성된 비디오 URL과 추가 작업 옵션을 제공합니다.

## 주의사항 (Cautions)

### 전제 조건: 캐릭터 생성 필수

이 스킬을 사용하려면 먼저 브랜드 캐릭터를 생성해야 합니다.

1. **캐릭터 생성**: `/character-mgmt` 스킬 실행
2. **캐릭터 ID 확보**: 생성된 캐릭터의 ID 저장
3. **비디오 생성**: 확보한 캐릭터 ID로 `speech-video` 스킬 실행

캐릭터가 없으면 비디오를 생성할 수 없습니다.

### API 키 필수

이 스킬을 사용하려면 다음 환경변수가 설정되어야 합니다:

- **Higgsfield**: `HIGGSFIELD_API_KEY`, `HIGGSFIELD_SECRET`
  - 발급처: Higgsfield 개발자 대시보드
  - MCP 설정: `gil-media/.mcp.json`에 `higgsfield` 서버로 등록

API 키 등록 절차: `gil-media/CONNECTORS.md` 참조

### 비용 참고

- **Speech-to-Video**: 영상당 ~$0.15-0.50 (길이/품질에 따라 변동)
- 긴 비디오(15초 이상)는 비용이 상대적으로 높습니다

### 제한사항

- **최대 길이**: 단일 비디오 최대 30초 권장 (초과 시 품질 저하 가능)
- **스크립트 언어**: 한국어, 영어 지원 (다른 언어는 번역 후 사용 권장)
- **캐릭터 의존성**: 캐릭터가 삭제되면 해당 캐릭터로 생성된 비디오도 재생 불가
- **보이스 스타일**: 지원되는 스타일만 사용 가능 (friendly, professional, energetic, calm)

### 보이스 스타일 가이드

| 스타일 | 특징 | 사용 사례 |
|--------|------|-----------|
| `friendly` | 친근하고 부드러운 톤 | 제품 소개, 브랜드 홍보, 일반 콘텐츠 |
| `professional` | 전문적이고 신뢰감 있는 톤 | 교육 자료, 비즈니스 프레젠테이션 |
| `energetic` | 활기차고 역동적인 톤 | 프로모션, 세일 안내, 짧은 SNS 콘텐츠 |
| `calm` | 차분하고 안정적인 톤 | 명상, 브랜드 스토리텔링, 럭셔리 브랜드 |

## 관련 스킬 (Related Skills)

- **character-mgmt**: 브랜드 캐릭터 생성 및 관리 (전제 조건)
- **video-gen**: 시네마틱 이미지-투-비디오, 텍스트-투-비디오 (Kling)
- **image-gen**: 캐릭터 이미지 생성용 소스 (fal.ai Imagen, Gemini 3)
- **audio-gen**: 배경 음악 생성 (ElevenLabs, fal.ai)

## 활용 시나리오

### 제품 런칭 캠페인

```
1단계: character-mgmt로 브랜드 마스코트 캐릭터 생성
2단계: speech-video로 제품 소개 토킹 헤드 비디오 3개 생성 (각각 10초)
3단계: video-gen으로 시네마틱 제품 샷 비디오 생성
4단계: SNS에 순차 게시 및 반응 모니터링
```

### 교육 콘텐츠 시리즈

```
1단계: 전문적인 톤의 가상 강사 캐릭터 생성
2단계: 각 주제별로 5-10분 분량의 스크립트 작성
3단계: speech-video로 30초 단위로 분할하여 비디오 시리즈 생성
4단계: 온라인 강의 플랫폼에 업로드
```

### SNS 광고 캠페인

```
1단계: 브랜드 캐릭터 생성 (여러 버전)
2단계: speech-video로 9:16 세로형 토킹 헤드 비디오 다수 생성
3단계: A/B 테스트를 위한 다양한 보이스 스타일과 스크립트 조합
4단계: 인스타 릴스, 틱톡, 유튜브 쇼츠에 동시 게시
```

## 기술 참고사항

- **MCP 툴**: `generate_speech_video` (Higgsfield)
- **지원 모델**: Higgsfield Speech-to-Video
- **최대 길이**: 30초 권장 (기술적 한계: 60초)
- **지원 화면 비율**: 16:9 (가로), 9:16 (세로), 1:1 (정사각형)
- **지원 언어**: 한국어, 영어 (우선 지원), 일본어, 중국어 (베타)
- **출력 포맷**: MP4 (H.264), 웹 최적화됨
