# media-moodboard — Day 3 S1 무드보드 자동 생성

## 개요

**책임 한 줄**: Day 3 S1 (10:00–10:50)에서 ⑦ 페르소나 3명 + ⑧ 7단계 카피 + 카테고리를 입력으로 받아 색 팔레트 3종 + 톤 키워드 5개 + 레퍼런스 이미지 5장을 산출하는 무드보드 스킬.

**V6 18 산출물 체인 위치**: 산출물 ⑫ (Day3 S1, 10:10–10:25 강사 시연) — ⑦ 페르소나·⑧ 7단계 카피 → **⑫ 무드보드** → ⑬ GPT Image 2 이미지 입력으로 흘러감.

### 입력 슬롯

|------|------|------|

| `category` | 수강생 입력 | 의류/패션·뷰티·건강식품·생활용품 중 택1 |

### 처리 흐름

**Step 1 — 카테고리별 색 방향 매핑**

카테고리에 따른 색상 방향 초기화:

| 카테고리 | 권장 색 방향 | 무드 |
|---------|------------|------|
| 의류/패션 | 뉴트럴·자연광 팔레트 | 룩북·에디토리얼 |
| 뷰티 | 클린·파스텔·클로즈업 | 스튜디오·순도 강조 |
| 건강식품 | 어스·그린·라이프스타일 | 자연주의·결과 시각화 |
| 생활용품 | 밝고 실용적 일상 | 화이트·그레이·사용 장면 |

**Step 2 — 페르소나 × 카피 분석**

- 페르소나 3명의 라이프스타일 키워드 추출
- 7단계 카피 중 1단계(문제인식)·2단계(공감)·3단계(핵심)에서 감성 방향 도출
- 타겟 감성 축 설정 (예: "신뢰감 + 자연스러움 + 결과 확신")

**Step 3 — 색 팔레트 3종 생성**

각 팔레트는 다음을 포함:
- 메인 컬러 (HEX + RGB)
- 서브 컬러 2종 (HEX + RGB)
- 텍스트 컬러 (HEX)
- 배경 컬러 (HEX)
- 팔레트 네이밍 (예: "민트 신뢰형", "워밍 어스형", "클린 화이트형")

**Step 4 — 톤 키워드 5개 선정**

페르소나·카피·카테고리 교집합에서 5개 선정. 예:
- "자연스러운 일상", "신뢰감 있는", "결과 중심", "감각적 단순", "따뜻한 전문성"

**Step 5 — 레퍼런스 이미지 5장 방향 제시**

Higgsfield Soul 또는 기존 이미지 라이브러리에서 검색 키워드 제공:

| # | 이미지 용도 | 검색 키워드 | 참고 구성 |
|---|-----------|-----------|---------|
| ①레퍼런스 | Hero 이미지 방향 | 카테고리 + 메인 팔레트 + 무드 | 제품 중심·배경 단순 |
| ②레퍼런스 | 인포/증명 이미지 방향 | 성분·데이터·클로즈업 | 텍스트 오버레이 가능 구도 |
| ③레퍼런스 | 라이프 이미지 1 방향 | 페르소나 라이프스타일 | 자연광·일상 |
| ④레퍼런스 | 라이프 이미지 2 방향 | 사용 후 결과 | 비포/애프터 무드 |
| ⑤레퍼런스 | CTA 배너 방향 | 간결·행동유도·가격·혜택 | 텍스트 비중 높음 |

### 합격 기준 (PDF §6.8 ⑫)

- 색 팔레트 3종 (메인+서브+텍스트+배경 HEX 포함)
- 톤 키워드 5개 (페르소나 근거 명시)
- 레퍼런스 이미지 방향 5장 (용도별 검색 키워드 포함)

## 사용 예시

### 예시 1: 뷰티 카테고리 무드보드

```
"Day3 S1 무드보드. 카테고리: 뷰티(마스크팩). 페르소나는 어제 만든 ⑦ 파일 참고. 7단계 카피는 ⑧ 파일."
→ 클린·파스텔 팔레트 3종 + "순도 높은", "임상 신뢰", "After 결과", "피부 밀착", "부드러운 전문성" 5개 키워드
→ Hero·클로즈업·라이프·비포애프터·CTA 레퍼런스 방향 5장
```

### 예시 2: 건강식품 카테고리 무드보드

```
"건강식품(콜라겐 음료) 무드보드 뽑아줘. 30대 여성 페르소나."
→ 어스·그린·크림 팔레트 3종 + "자연주의", "과학적 근거", "결과 숫자", "라이프스타일 흡수", "따뜻한 건강" 5개
→ 제품 중심·성분 클로즈업·일상·수치 오버레이·할인 CTA 방향 5장
```
