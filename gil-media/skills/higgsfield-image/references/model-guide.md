# Higgsfield 이미지 모델 가이드 — 11종 공식

[higgsfield.ai 공식 페이지](https://higgsfield.ai) 명시 기준 (2026-05). 모델은 Higgsfield 정책에 따라 추가·변경될 수 있습니다.

## Soul 계열 (Higgsfield 자체 모델) ★

### 1. Soul

**강점**:
- 시네마틱 그레이딩 1세대
- 한국어 프롬프트 이해도 좋음
- 인물 표정·기본 디테일

**약점**:
- Soul 2.0 출시 이후 신규 작업은 2.0 권장

**권장 시점**:
- Soul 2.0 미지원 환경 호환

### 2. Soul 2.0 ★

**강점**:
- Soul 1세대 개선판
- 인물 디테일·표현력 향상
- Soul Characters reference 지원 (시리즈 일관성)

**약점**:
- 글자 렌더링은 Nano Banana Pro보다 약함
- 추상적 일러스트는 Seedream 4.0이 더 적합

**권장 시점**:
- 인물 시리즈·브랜드 캐릭터
- 광고 키 비주얼
- 발표 슬라이드 표지

**예시 프롬프트**:
```
한국인 30대 여성 CEO가 회의실에서 노트북을 보고 있다.
부드러운 자연광, 시네마틱 그레이딩, 미니멀 인테리어, 16:9.
```

### 3. Soul Cinema

**강점**:
- 영화 필름 그레이딩 특화
- 35mm·LUT·컬러 사이언스
- 광고 키 비주얼 톤

**약점**:
- 일러스트·만화 톤 부적합
- 글자 정확도 떨어짐
- 크레딧 가장 비싼 편

**권장 시점**:
- 광고 키 비주얼
- 영상 썸네일 (Veo·Sora 영상 전 미리보기)
- 영화 포스터 톤

**예시 프롬프트**:
```
영화적 와이드샷, 35mm 필름 룩, 골든 아워 조명,
한국 도심 거리, 우산을 든 인물, 비 떨어지는 디테일.
```

## Nano Banana 계열 (Gemini 3 기반)

### 4. Nano Banana

**강점**:
- 글자 렌더링 정확도
- 빠른 생성·저비용
- 다양한 폰트·레이아웃

**약점**:
- 1K 기본 (2K 이상은 Pro)
- 글자 정확도는 GPT Image 2에 비해 떨어짐
- 인물 일관성 약함

**권장 시점**:
- 빠른 반복 시도
- 일반 SNS 이미지 (글자 비중 낮을 때)

### 5. Nano Banana Pro

**강점**:
- 2K 이상 해상도
- 빠른 속도·저비용
- 일반 카드뉴스 톤

**약점**:
- 글자 렌더링 정확도는 **GPT Image 2가 더 우수** (한글·영문 모두)
- 시네마틱 톤은 Soul 계열보다 약함

**권장 시점**:
- 일반 SNS 카드 (글자가 부차적일 때)
- 인쇄 보조용
- GPT Image 2 fallback

**예시 프롬프트**:
```
인스타 카드뉴스 1:1. 헤드라인: "5분 만에 끝나는 AI 영상 편집".
크림 베이지 배경, Anthropic Orange 포인트, Pretendard 굵은 헤딩.
```

### 6. GPT Image

**강점**:

- 범용·다양한 톤

**권장 시점**:
- 일반 콘텐츠
- GPT Image 2 미지원 시 fallback

### 7. GPT Image 2 ★ (글자 렌더링 1순위)

**강점**:
- **글자 렌더링 SOTA** — 한글·영문 모두 정확
- OpenAI 개선판, 사실성·정확성 향상
- 디테일 정교
- 카드뉴스·포스터·인쇄에 가장 적합

**약점**:
- 시네마틱 톤은 Soul Cinema가 더 적합
- 추상 아트는 Seedream 4.0이 더 적합

**권장 시점**:
- **카드뉴스 4-10장 (글자 핵심) — 1순위 ★**
- 포스터·배너·인포그래픽
- 사실적 일러스트
- 제품·풍경 (Flux Kontext 대안)

### Q. 마케팅 캠페인 비주얼 6종을 같은 인물로
