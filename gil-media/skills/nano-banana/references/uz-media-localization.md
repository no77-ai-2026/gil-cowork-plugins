# UZ 미디어 시각 현지화 가이드

> gil-media v0.1.0 | UZ 시각 코드 (이슬람·문화·도자기·인물·복장)

## 개요

한국 브랜드가 우즈베키스탄(UZ) 시장 대상 이미지·영상을 제작할 때 시각 코드 점검 필수. 한국에서 자연스러운 모델·복장·배경이 UZ 사용자에게 부정적으로 인식될 수 있고, 그 반대도 마찬가지입니다.

---

## 1. 시각 코드 우선순위

```
1순위: 종교 (이슬람 — 88% 인구)
2순위: 가족·공동체 가치
3순위: 우즈벡 전통 (도자기·건축·실크)
4순위: K-Drama·K-Beauty 인기 (한국적 요소 OK)
5순위: 글로벌 미니멀 (서양식 차분)
```

---

## 2. 인물 가이드

### UZ 외모 특징

- 백인계 (러시아·우크라이나·우즈벡 혼합)
- 아시아계 (우즈벡·카자흐 등 중앙아시아)
- 다양한 외모 공존
- 한국·일본 외모도 거부감 적음 (K-Drama 영향)

### 프롬프트 키워드

```
"Central Asian woman with warm skin tone"
"Uzbek woman in modest clothing"
"Mixed Korean and Uzbek family"
"Tashkent young professional"
```

### 인물 노출 가이드

**여성:**
- 어깨·팔뚝·무릎 노출 회피
- 머리 가리지 않아도 OK (보수 지역 외)
- 전통 복장 (Atlas 실크 드레스) 가능
- 비즈니스 정장·캐주얼 OK

**남성:**
- 보수적 정장 또는 캐주얼
- 근육 노출 광고 회피 (특정 카테고리만)

**가족:**
- 가족 단위 사진 효과적
- 어린이 + 부모 + 조부모 (3대 함께)
- 노인 공경 표현

### 회피 사항

- 노골적 신체 노출
- 이성 간 친밀한 신체 접촉 (광고 한정 — 부부도 신중)
- 술잔·돼지고기 든 사진
- 여성 단독 야외 (밤·외진 곳)

## 10. 카드뉴스·릴스 비주얼 체크리스트

```
[ ] 인물: UZ 외모 또는 한국+UZ 혼합
[ ] 복장: 보수적 (특히 여성)
[ ] 배경: UZ 친화 또는 미니멀
[ ] 색상: 청록·녹색·흰색·금색 우선
[ ] 식품 (해당 시): 할랄
[ ] 술·돼지 X
[ ] 종교 풍자 X
[ ] 정치 상징 X
[ ] 한국 전통과 조화 (있다면)
[ ] 가족·공동체 가치 (가능 시)
```

---

## 11. 자주 하는 실수

1. **한국 광고 그대로 사용**: 노출·이성 친밀도 → UZ 부정 인식
2. **한국 모델만**: UZ 사용자 거리감 → UZ 모델 또는 혼합 권장
3. **검정 단독**: 장례 연상 → 강조 포인트만
4. **돼지·술 무심결**: 식품·라이프 광고에 등장 → 즉시 제거
5. **종교 상징 가벼운 사용**: 신중 + 현지인 검수
6. **고가 정책**: UZ 평균 소득 한국 1/5 미만 → 가격·디자인 조정
