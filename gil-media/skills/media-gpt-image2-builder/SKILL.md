# media-gpt-image2-builder — Day 3 S2 한글 타이포 광고 이미지 5장

## 개요

**책임 한 줄**: Day 3 S2 (11:00–11:50)에서 수강생의 자연어 한 줄 입력을 받아 내부적으로 8단계 프롬프트를 자동 리라이팅하고, GPT Image 2로 한글 타이포그래피 광고 이미지 5장 세트를 생성하는 스킬.

**V6 18 산출물 체인 위치**: 산출물 ⑬ (Day3 S2, 11:08–11:18 강사 시연 → 11:18–11:45 수강생 실습) — ⑧ 7단계 카피 + ⑫ 무드보드 → **⑬ 이미지 5장** → ⑭ 합성 PNG 입력으로 흘러감.

> **수강생 비공개 룰 (REQ-MEDIA-013 HARD)**: 수강생은 자연어 한 줄만 입력합니다. 플러그인이 내부적으로 8단계 프롬프트를 자동 리라이팅하여 GPT Image 2에 전달합니다. 8단계 구조 원문은 수강생 응답에 노출하지 않습니다.

> **OpenAI 모델 제한 (REQ-MEDIA-014 HARD)**: GPT Image 2(이미지 전용)만 사용. GPT-5.5(LLM)·Sora(영상) 호출 금지.

### 본 스킬 내부 동작

> **수강생에게는 자연어 한 줄 입력만 노출합니다.** 아래 8단계 처리는 스킬 내부 동작이며, 수강생 응답 화면에 8단계 구조 원문을 표시하지 않습니다.

**Step 1 — 자연어 → 8단계 구조 자동 리라이팅 (내부)**

| 단계 | 내용 | 무드보드·카피 연동 |
|------|------|------------------|
| 1) 목적/용도 | 이미지가 사용될 채널·목표 | 채널 메시지 ⑩ 참고 |
| 2) 핵심 브리프 | 제품 USP + 타겟 감성 | 카피 ⑧ 3단계 핵심 메시지 |
| 3) 필수 요소 | 로고·텍스트·제품 배치 | 무드보드 ⑫ 톤 키워드 |
| 4) 맥락/환경 | 사용 상황·배경 | 페르소나 ⑦ 라이프스타일 |
| 5) 구도/공간 | 화면비·여백·초점 | 이미지 유형별 (Hero/인포/라이프/CTA) |
| 6) 빛/색/재질 | 조명 방향·색 팔레트·재질감 | 무드보드 ⑫ 색 팔레트 |
| 7) 제약/금지/고정 | 저작권·브랜드 가이드·금지 요소 | 카테고리별 규제 |
| 8) 출력/포맷 | 해상도·화면비·파일 형식 | 채널 규격 |

**Step 2 — 5장 이미지 유형 매핑**

| 이미지 | 역할 | 카피 ⑧ 매핑 | 화면비 |
|--------|------|------------|--------|
| Hero | 제품 중심·USP 전달 | 1단계(문제인식) + 3단계(핵심) | 1:1 또는 3:4 |
| 인포 | 성분·증명·데이터 텍스트 오버레이 | 4단계(증명) | 1:1 |
| 라이프1 | 사용 전 라이프스타일·감성 | 2단계(공감) | 3:4 |
| 라이프2 | 사용 후 결과·변화 | 5단계(사용법·혜택) | 3:4 |
| CTA 배너 | 할인·구매 행동 유도 | 7단계(CTA) | 9:16 또는 1:1 |

**Step 3 — GPT Image 2 호출 (OpenAI API)**

```
OpenAI API: POST /v1/images/generations
model: "gpt-image-2"
size: 이미지 유형별 (1024×1024 / 1024×1792 / 1792×1024)
quality: "hd"
n: 1 (이미지당 1회, 총 5회 호출)
```

> 동일 시드/스타일 일관성 유지: seed 고정 + 프롬프트 스타일 접두어 고정

**Step 4 — 한글 가독성 자동 보정 (REQ-MEDIA-010)**

GPT Image 2 한글 타이포 깨짐 감지 시:
1. 자동 재지시: "한글 폰트 두껍게", "자간 줄여" 등 자동 보정 1회 시도
2. 실패 시: 텍스트는 별도 합성 단계(⑭)에서 처리하도록 안내

**Step 5 — AI 표기 자동 체인 (REQ-MEDIA-002)**

5장 이미지 생성 완료 후 `media-ai-disclosure` 자동 체인하여 메타데이터 + AI 생성 표기 부착.

### 합격 기준 (PDF §6.8 ⑬)

- 같은 시드/룩 일관성 (5장 간 스타일 통일)
- 한글 가독성 (타이포 깨짐 없음)
- AI 표기 자동 메타데이터 부착

## 사용 예시

### 예시 1: 뷰티 마스크팩 광고 이미지

```
수강생 입력: "마스크팩 광고 이미지 5장 만들어줘"
→ (내부) 8단계 프롬프트 자동 리라이팅 + 무드보드⑫·카피⑧ 연동
→ 결과: Hero("피부가 바뀌는 순간") + 인포(성분 표 오버레이) + 라이프1(세안 후 루틴) + 라이프2(글로우 피부) + CTA(오늘만 30%↓)
→ media-ai-disclosure 자동 체인 → AI 생성 메타데이터 부착
```

### 예시 2: 건강식품 콜라겐 광고 이미지

```
수강생 입력: "콜라겐 음료 광고 이미지 세트 만들어줘"
→ Hero(숫자 강조 "98%") + 인포(원료 클로즈업) + 라이프1(일상 한 모금) + 라이프2(피부 결 변화) + CTA(첫 구매 특가)
→ AI 표기 자동 부착
```

### 예시 3: 한글 타이포 깨짐 자동 보정

```
수강생 입력: "생활용품 광고 이미지"
→ (내부) 생성 시도 → Hero 한글 깨짐 감지
→ 자동 재지시: "Bold 체 한글 타이포, 자간 -10%, 두꺼운 고딕" 보정 1회
→ 보정 성공: 5장 세트 반환 / 보정 실패: "⑭ 합성 단계에서 텍스트 추가 권장" 안내
```
