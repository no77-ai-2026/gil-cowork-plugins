# 부록 A — 8단계 프롬프트 구조

> **[강사·개발팀 내부용]** 수강생 비공개. 수강생 출력·응답에 이 파일 내용을 노출하지 않습니다 (REQ-MEDIA-013 HARD).
> 출처: SPEC-MEDIA-CAMP-004 §6.3 + PDF 부록 A

## 8단계 구조 상세

수강생의 자연어 한 줄 입력을 GPT Image 2에 전달하기 전 스킬 내부에서 아래 8단계로 자동 리라이팅합니다.

| 단계 | 항목 | 설명 | 입력 소스 |
|------|------|------|---------|
| 1 | 목적/용도 | 이미지가 사용될 채널·광고 목표 | 채널 메시지 ⑩, 수강생 자연어 |
| 2 | 핵심 브리프 | 제품 USP + 타겟 감성 1문장 | 카피 ⑧ 3단계 핵심 메시지 |
| 3 | 필수 요소 | 제품·로고·텍스트 배치 필수 항목 | 무드보드 ⑫ 톤 키워드 + 브랜드 가이드 |
| 4 | 맥락/환경 | 사용 상황·배경·시간대 | 페르소나 ⑦ 라이프스타일 |
| 5 | 구도/공간 | 화면비·여백·초점·피사계심도 | 이미지 유형 (Hero/인포/라이프/CTA) |
| 6 | 빛/색/재질 | 조명 방향·색 팔레트·재질감·분위기 | 무드보드 ⑫ 색 팔레트 |
| 7 | 제약/금지/고정 | 저작권·브랜드 금지 요소·규제 | 카테고리별 광고 규제 |
| 8 | 출력/포맷 | 해상도·화면비·파일 형식·품질 | 채널 규격 |

## 카테고리별 단계 적용 예시

### 뷰티 (마스크팩)

```
[자연어 입력] "마스크팩 광고 이미지 Hero 만들어줘"

[내부 8단계 리라이팅 — 수강생 미노출]
1. 목적: 인스타그램 피드 광고, 클릭률 최대화
2. 핵심 브리프: "7일 피부 변화" — 20~30대 여성, 자연주의 + 결과 확신
3. 필수 요소: 마스크팩 제품 정면, 브랜드명 상단, USP 1줄 텍스트
4. 맥락: 욕실 또는 미니멀 흰 배경, 저녁 스킨케어 루틴
5. 구도: 1:1, 제품 중앙 배치, 여백 30%, 부드러운 보케
6. 빛/색: 소프트박스 정면광, #F8F5F1 배경, 크림-페일핑크 팔레트
7. 제약: 모델 얼굴 없음, "효과 보장" 문구 금지 (광고심의)
8. 출력: 1024×1024, PNG, HD 품질

[GPT Image 2 전달 프롬프트 — 수강생 미노출]
"Product shot of a mask pack skincare product, centered composition,
minimal white background with cream tones (#F8F5F1), soft front lighting,
1:1 ratio, clean Korean typography '7일 피부 변화' in bold sans-serif,
high quality, photorealistic, no model face, beauty editorial style"
```

## 5장 이미지 유형별 리라이팅 패턴

| 이미지 | 5·6·8단계 핵심 파라미터 |
|--------|----------------------|
| Hero | 1:1 정방형, 제품 중심, 메인 USP 카피, 소프트 보케 |
| 인포 | 1:1, 텍스트 오버레이 여백 50%+, 데이터·수치 강조, 클린 배경 |
| 라이프1 | 3:4 세로, 자연광, 일상 상황, 감성적 보케 |
| 라이프2 | 3:4 세로, 결과 비포애프터, 밝고 따뜻한 톤 |
| CTA | 9:16 세로 또는 1:1, 텍스트 60%+, 할인/혜택 강조, 고대비 |

## 한글 타이포 깨짐 보정 프롬프트

```
"bold Korean gothic font, thick strokes, clear legibility,
-10% letter spacing, no blur on text, sharp typography"
```

2차 보정 실패 시:
```
→ 텍스트 제거 버전 생성 + 별도 텍스트 레이어 합성 (⑭ 단계에서 처리)
```
