# media-canva-magic-layer — Day 3 S7 Canva 매직 레이어 가이드

## 개요

**책임 한 줄**: Day 3 S7 (17:40–17:45) 강사 5분 시연에서 기존 합성 PNG(⑭)나 광고 이미지를 Canva 매직 레이어로 카피만 교체하여 시즌·캠페인 변경 시 GPT Image 2 재호출 대비 90% 비용 절감하는 재사용 가이드 스킬.

**V6 18 산출물 체인 위치**: Day3 S7 보너스 (17:40–17:45 강사 5분 시연) — ⑭ 합성 PNG 또는 ⑬ 광고 이미지 → 매직 레이어 카피 교체 → 신규 시즌 소재 (GPT Image 2 재호출 없이).

> **자동화 범위**: 본 스킬은 5단계 체크리스트와 시나리오 가이드를 출력합니다. 실제 Canva 조작은 수강생이 Canva 웹(canva.com)에서 직접 진행합니다. Canva API 자동 호출은 현재 미구현 (PDF Exclusions §9).

> **AI 표기 유지 (REQ-MEDIA-011 HARD)**: 매직 레이어로 텍스트를 수정해도 "AI 생성" 메타데이터 표기는 반드시 유지합니다. 수정 완료 후 media-ai-disclosure 재호출 권장.

> **디스코드 셀프 학습 안내**: 강의 내 시연 5분 외, D+7 디스코드 라이브에서 추가 실습 제공 (PDF §6.4 S7 운영 노트 + 부록 D).

### 5단계 Canva 매직 레이어 가이드 (부록 D 직접 인용)

**Step 1 — Canva 업로드 → 매직 레이어 클릭**

```
1. canva.com 접속 → [새 디자인 만들기]
2. [파일 업로드] → 합성 PNG(⑭) 또는 광고 이미지(⑬) 업로드
3. 업로드된 이미지를 캔버스에 배치
4. 이미지 선택 후 상단 메뉴 [편집] → [매직 레이어] 클릭
   (또는 이미지 더블클릭 → 레이어 분리 자동 제안)
```

**Step 2 — 텍스트 레이어 분리 확인**

```
매직 레이어 실행 후:
- 이미지 레이어 (배경·제품) ← 수정하지 않음
- 텍스트 레이어 ← 카피 교체 대상
텍스트 레이어가 보이지 않으면: [레이어] 패널 → 텍스트 레이어 활성화
```

> GPT Image 2 생성 이미지의 경우 텍스트가 이미지에 번인된 경우 있음 — 이 경우 별도 텍스트 박스 추가 권장.

**Step 3 — 카피만 교체**

```
텍스트 레이어 클릭 → 기존 카피 전체 선택 (Ctrl+A / Cmd+A)
→ 새 카피 입력 (시즌·캠페인 변경 내용)
폰트·크기·색상은 기존 유지 (디자인 일관성)
```

카피 교체 체크리스트:
- [ ] Hero 이미지: USP 메인 카피 교체
- [ ] CTA 배너: 할인율·혜택 텍스트 교체
- [ ] 인포 이미지: 수치·데이터 업데이트 (해당 시)

**Step 4 — 다운로드**

```
[공유] → [다운로드]
→ 파일 형식: PNG (고품질) 또는 JPG (파일 크기 작음)
→ 해상도: 최소 1080px 이상 (채널 규격 준수)
→ 다운로드
```

**Step 5 — 스마트스토어·카페24 즉시 업로드**

```
스마트스토어:
  판매자센터 → 상품관리 → 해당 상품 → 이미지 교체

카페24:
  쇼핑몰 관리자 → 상품 → 이미지 교체

Instagram/Meta:
  광고 관리자 → 소재 교체 (채널 패키지 ⑰ 참고)
```

### 시즌 재사용 시나리오

**기존 합성 PNG → 카피만 교체 → 새 시즌 소재**

| 시나리오 | 재사용 방법 | GPT Image 2 재호출 |
|---------|-----------|------------------|
| 여름 → 가을 시즌 변경 | 카피만 교체 ("시원한 여름" → "따뜻한 가을") | 불필요 |
| 기간 한정 이벤트 | 할인율·날짜만 교체 ("30%" → "50%", 날짜 변경) | 불필요 |
| 채널별 카피 최적화 | 메타용 카피 ↔ 네이버용 카피 교체 | 불필요 |
| 제품 라인업 확장 | 제품명·SKU만 교체 | 불필요 |

**비용 절감 효과**: GPT Image 2 재호출 대비 90% 비용 절감 (이미지 생성비 0원, Canva Free 플랜 가능).

```
매직 레이어 카피 교체 완료
→ [필수] media-ai-disclosure 재호출

```
