## 모델별 강점 및 Higgsfield API 파라미터

### Kling 3 — 인물·Hook 특화

```
강점: 인물 움직임 자연스러움, 감성 룩북, 의심차단형 Hook
주요 카테고리: 의류/패션(인물), 뷰티(Hook), 건강식품(라이프)

Higgsfield MCP 파라미터:
{
  "model": "kling-3",
  "aspect_ratio": "9:16",  # 릴스/세로 광고
  "duration": 5,           # 5~10초
  "prompt": "...",
  "negative_prompt": "blurry, watermark, text",
  "cfg_scale": 7.5
}
```

### Veo 3 — 클로즈업·성분·디테일 특화

```
강점: 극단 클로즈업 선명도, 뷰티 텍스처, 성분 시각화
주요 카테고리: 뷰티(메인), 건강식품(성분), 의류(소재 디테일)

Higgsfield MCP 파라미터:
{
  "model": "veo-3",
  "aspect_ratio": "1:1",   # 정방형 클로즈업
  "duration": 5,
  "prompt": "...",
  "camera_motion": "zoom_in",  # 클로즈업 강조
  "style": "cinematic"
}
```

### Seedance 2.0 — 무빙샷·일상·사용 장면 특화

```
강점: 자연스러운 동작 무빙, 일상 사용 장면, 생활용품 실용
주요 카테고리: 생활용품(메인), 의류(무빙), 건강식품(일상)

Higgsfield MCP 파라미터:
{
  "model": "seedance-2.0",
  "aspect_ratio": "9:16",
  "duration": 8,
  "prompt": "...",
  "motion_style": "natural",
  "camera": "handheld"    # 자연스러운 무빙
}
```
