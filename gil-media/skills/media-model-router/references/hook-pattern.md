### 공통 DNA 5가지 (부록 C 직접 인용)

1. **매크로/익스트림 클로즈업** — 제품·피부·소재를 극단 클로즈업하여 시선 잡기

4. **권위 인증 노출** — 식약처·임상·후기·인증 텍스트 오버레이 (7단계 4단계)
