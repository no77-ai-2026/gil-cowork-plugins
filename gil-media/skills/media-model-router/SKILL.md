# media-model-router — Day 3 S4 광고 영상 자동 모델 라우팅

## 개요

**책임 한 줄**: Day 3 S4 (14:00–14:50)에서 카테고리·컨셉 자연어 입력을 받아 내부 카테고리 매트릭스 기반으로 Higgsfield 영상 모델을 자동 선택하고, 의심차단형 후크 패턴이 적용된 메인 광고 영상(⑮)과 보조 영상 2컷(⑯)을 생성하는 스킬.

**V6 18 산출물 체인 위치**: 산출물 ⑮ 메인 영상 + ⑯ 보조 영상 (Day3 S4, 14:08–14:20 강사 시연 → 14:20–14:45 수강생 실습) — ⑭ 합성 PNG + ⑧ 7단계 카피 → **⑮⑯ 영상** → ⑰ 채널 패키징 입력.

> **수강생 자동 라우팅 (REQ-MEDIA-012 HARD)**: 수강생은 카테고리·컨셉 자연어 한 줄만 입력합니다. 모델 선택·비교는 플러그인이 자동 처리합니다. 카테고리 매트릭스 원문은 수강생에게 노출하지 않습니다.

> **큐잉 룰 (REQ-MEDIA-009 HARD)**: 4명×5조 시차 호출 (5분 간격). Higgsfield 동시 호출 비용 폭증 방지.

### 처리 흐름

**Step 1 — 카테고리 자동 감지**

자연어에서 카테고리 키워드 추출:
- 의류/패션: 옷, 패션, 코디, 의류, 룩북, 티셔츠, 아웃도어
- 뷰티: 마스크팩, 크림, 세럼, 선크림, 메이크업, 스킨케어, 화장품
- 건강식품: 콜라겐, 비타민, 보충제, 다이어트, 건강기능, 단백질
- 생활용품: 청소, 주방, 생활, 욕실, 인테리어, 수납, 가전

**Step 2 — 모델 자동 라우팅 (내부 카테고리 매트릭스 기반)**

> 수강생에게는 라우팅 결과(선택된 모델 이름)만 표시합니다.

| 카테고리 | 메인 영상 모델 | 보조 영상 모델 | Hook 톤 | 첫 3초 전략 |
|---------|-------------|-------------|---------|------------|
| 의류/패션 | Kling 3 (인물) | Seedance (무빙) | 핏룩북 | 착용 핏 매크로 클로즈업 |
| 뷰티 | Veo 3 (클로즈업) | Kling 3 (Hook) | Before-After | 극단 클로즈업·피부 텍스처 |
| 건강식품 | Kling 3 (라이프) | Veo 3 (성분) | 결과 숫자 | 숫자 강조·임상 데이터 |
| 생활용품 | Seedance (사용) | Kling 3 (Hook) | 전후 비교 | 사용 전/후 비교 전환 |

**Step 3 — 의심차단형 후크 패턴 적용 (PDF §6.6 + 부록 C, REQ-MEDIA-011)**

메인 영상 첫 3초 공통 DNA:
1. **매크로/익스트림 클로즈업** — 제품·피부·소재 극단 클로즈업으로 시선 잡기
2. **굵은 한글 자막 오버레이** — 음소거 시청자(70%+)를 위한 1줄 자막 필수
3. **의심 차단형 후크 메시지** — 7단계 ⑧ 1단계(문제인식) + 2단계(공감) 합본
4. **권위·인증 노출** — 7단계 4단계 텍스트 오버레이 (식약처·임상·후기)
5. **마지막 1초 = 할인·CTA 명시** — 7단계 7단계 (오늘 단 하루·단독특가)

**Step 4 — 시차 호출 큐잉 (REQ-MEDIA-009 HARD)**

Higgsfield 동시 호출 비용 폭증 방지 큐잉:
```
큐잉 토큰 발급 → 조별 순서 배정 (5분 간격)
호출 순서: 1조(즉시) → 2조(+5분) → 3조(+10분) → 4조(+15분) → 5조(+20분)
각 조 내 4명 병렬 호출 허용 (조간 시차만 강제)
```

> D-14 강사 워크스페이스 사전 비용 충전 필수 (PDF §6.9 리스크 + D-14 체크리스트).

**Step 5 — AI 표기 자동 체인 (REQ-MEDIA-002)**

영상 생성 완료 후 `media-ai-disclosure` 자동 체인:
- 영상 메타데이터: AI 생성 표기
- 첫 프레임·마지막 프레임: "AI 생성" 자막 오버레이

### 합격 기준 (PDF §6.8 ⑮·⑯)

**⑮ 메인 영상**:
- 의심차단형 후크 첫 3초 (매크로 클로즈업 + 굵은 한글 자막 + 문제인식·공감 카피)
- 7단계 ⑧ 1·2·7단계 반영
- AI 표기 (메타데이터 + 첫/마지막 프레임)
- 길이: 5~10초

**⑯ 보조 영상 2컷**:
- Veo 클로즈업 1컷 + Seedance 무빙 1컷
- 한글 자막 포함
- AI 표기

## 사용 예시

### 예시 1: 뷰티 카테고리 (강사 시연 케이스)

```
수강생 입력: "마스크팩 광고 영상 만들어줘"
→ (내부) 카테고리=뷰티 자동 감지 → Veo 3 클로즈업(메인) + Kling 3 Hook(보조) 자동 선택
→ 첫 3초: 피부 텍스처 익스트림 클로즈업 + "조작 아닌가?" 의심차단 자막
→ 큐잉 토큰 발급: "2조 시작 시간: 14:25" 안내
→ 결과: 메인 8초 영상 + 보조 2컷 + AI 표기 자동 부착
```

### 예시 2: 건강식품 카테고리

```
수강생 입력: "콜라겐 광고 영상 만들어줘, 숫자 강조"
→ 카테고리=건강식품 → Kling 3 라이프(메인) + Veo 3 성분(보조)
→ 첫 3초: "98% 흡수율" 숫자 오버레이 + 결과 숫자 Hook
→ 마지막 1초: "오늘만 특가" CTA 명시
```

### 예시 3: 의류/패션 카테고리

```
수강생 입력: "봄 원피스 광고 영상, 자연스러운 룩북 느낌"
→ 카테고리=의류 → Kling 3 인물(메인) + Seedance 무빙(보조)
→ 핏룩북 톤: 자연광 인물 착용 → 걷기 무빙샷
```
