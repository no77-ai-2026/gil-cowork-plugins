# media-channel-ad-packager — Day 3 S6 채널별 광고 소재 패키징

## 개요

**책임 한 줄**: Day 3 S6 (16:00–16:50)에서 메인 영상(⑮) + 보조 영상 2컷(⑯) + 채널 메시지 15종(⑩)을 입력으로 메타·네이버 GFA·카카오모먼트 3채널 규격에 맞게 자동 변환하고 .zip 패키지로 출력하는 스킬.

**V6 18 산출물 체인 위치**: 산출물 ⑰ (Day3 S6, 16:20–16:45 수강생 실습) — ⑮ 메인 영상 + ⑯ 보조 영상 + ⑩ 채널 메시지 → **⑰ 채널별 광고 소재 세트** → 수강생 직접 보관 (광고 실집행은 본 캠프 외 영역, PDF §1.3).

> **광고 실집행 제외**: 본 스킬은 소재 패키징만 담당합니다. 실제 메타·네이버·카카오 광고 매체 집행·운영은 본 캠프 범위 밖입니다 (PDF §1.3).

## 워크플로우

### 입력 슬롯

| 슬롯 | 출처 | 설명 |
|------|------|------|
| `main_video` | 산출물 ⑮ | 메인 광고 영상 1편 (5~10초) |
| `sub_videos` | 산출물 ⑯ | 보조 영상 2컷 (Veo 클로즈업 + Seedance 무빙) |
| `channel_messages` | 산출물 ⑩ | 채널 메시지 15종 (헤드라인·설명·CTA·해시태그 세트) |
| `image_set` | 산출물 ⑬ | 한글 타이포 이미지 5장 (선택 — 피드 이미지 소재) |

### 처리 흐름

**Step 1 — 채널별 규격 자동 변환**

#### 메타 (인스타그램·페이스북)

| 형식 | 규격 | 용도 |
|------|------|------|
| 피드 정방형 | 1:1 (1080×1080) | 인스타 피드·페이스북 피드 |
| 릴스/스토리 | 9:16 (1080×1920) | 인스타 릴스·스토리 |
| 가로형 | 16:9 (1920×1080) | 페이스북 뉴스피드 영상 |

메시지 매핑:
- 헤드라인: ⑩ 인스타 헤드라인 (40자 이내)
- 본문: ⑩ 인스타 본문 (150자 이내)
- CTA: ⑩ CTA 텍스트

#### 네이버 GFA (네이버 성과형 디스플레이)

| 형식 | 규격 | 길이 제한 |
|------|------|---------|
| 피드 정방형 영상 | 1:1 (1080×1080) | 30초 이내 |
| 동영상 광고 | 16:9 (1920×1080) | 60초 이내 |
| 이미지 배너 | 1:1 또는 16:9 | — |

메시지 매핑:
- 광고 제목: ⑩ 네이버 제목 (35자 이내)
- 설명: ⑩ 네이버 설명 (50자 이내)

#### 카카오모먼트 + 카카오 알림톡

**카카오모먼트**:
| 형식 | 규격 |
|------|------|
| 이미지 | 1:1 (800×800) 또는 16:9 |
| 영상 | 1:1 또는 16:9, 최대 30초 |

**카카오 알림톡 변수 처리**:
- `#{이름}` → 수신자 이름 변수 유지
- `#{상품명}` → 상품명 변수 유지
- `#{할인율}` → 할인율 변수 유지

메시지 매핑:
- 알림톡 본문: ⑩ 카카오 알림톡 텍스트 (변수 포함)
- 이미지 카피: ⑩ 카카오 헤드라인

**Step 2 — .zip 패키지 구조 생성**

```
channel-ad-package_{상품명}_{날짜}/
├── meta/
│   ├── feed_1x1/
│   │   ├── main_video_1x1.mp4
│   │   ├── sub_video_closeup_1x1.mp4
│   │   ├── hero_image_1x1.jpg
│   │   └── meta_copy.txt (헤드라인·본문·CTA)
│   └── reels_9x16/
│       ├── main_video_9x16.mp4
│       └── hero_image_9x16.jpg
├── naver-gfa/
│   ├── feed_1x1/
│   │   ├── main_video_1x1_30s.mp4
│   │   └── image_1x1.jpg
│   ├── banner_16x9/
│   │   ├── main_video_16x9_60s.mp4
│   │   └── image_16x9.jpg
│   └── naver_copy.txt (제목·설명)
├── kakao/
│   ├── moment_1x1/
│   │   ├── main_video_1x1.mp4
│   │   └── image_1x1.jpg
│   ├── moment_16x9/
│   │   └── main_video_16x9.mp4
│   └── alimtalk_template.txt (#{변수} 포함)
└── README.txt (채널별 업로드 안내)
```

**Step 3 — README.txt 자동 생성**

각 채널별 업로드 절차 안내:
- 메타 광고 관리자 업로드 경로
- 네이버 GFA 소재 등록 경로
- 카카오모먼트 소재 등록 경로
- 알림톡 변수 치환 안내

**Step 4 — AI 표기 확인**

패키징 전 모든 소재에 `media-ai-disclosure` 표기 부착 여부 확인. 미부착 소재 발견 시 자동 체인 재실행.

### 합격 기준 (PDF §6.8 ⑰)

- 메타·네이버 GFA·카카오모먼트 3채널 규격 변환 완료
- 채널별 메시지 15종 매핑 완료 (⑩ 기반)
- .zip 패키지 다운로드 가능

## 관련 스킬

- **이전 단계 ← media-model-router** (산출물 ⑮⑯ 영상 입력)
- **이전 단계 ← media-gpt-image2-builder** (산출물 ⑬ 이미지 5장 선택 입력)
