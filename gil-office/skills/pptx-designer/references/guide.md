# PPT 디자인 시스템 빌더 (PowerPoint Design System Builder)

일관되고 전문적인 파워포인트 프레젠테이션을 위한 **디자인 시스템**을 자동 생성합니다.
색상 팔레트, 폰트, 슬라이드 레이아웃, 마스터 정의를 통해 **브랜드 일관성**과 **제작 속도**를 동시에 실현합니다.

## Overview

**파워포인트 마스터 슬라이드**를 기준으로 다음을 제공합니다:

1. **색상 시스템** (Primary, Secondary, Neutral, Accent)
2. **타이포그래피** (제목, 본문, 강조 폰트)
3. **슬라이드 레이아웃 템플릿** (12가지)
4. **아이콘/요소 라이브러리** (반복 사용)
5. **마스터 슬라이드** (PPTX 파일)

---

## 지원 범위

### ✅ 생성 가능
```
- 기업/조직 프레젠테이션
- 제품 피치/데모 자료
- 교육/교과 자료
- 영업/제안 자료
- 내부 보고서
- 이벤트 프로그램
```

### ⚠️ 제약
```
- 복잡한 애니메이션 (PPT 기본 기능만)
- 비디오 임베딩 (링크 참조)
- 고급 상호작용 (특수 플러그인 필요)
```

## 디자인 시스템 구조

### 1. 색상 팔레트

#### Primary (주 색상)
```
브랜드를 대표하는 색상
- UI 요소, 강조, 링크
- 밝기: 주색 1개 + 라이트 / 다크 변형

예:
Primary: #0066CC (파란색)
Primary Light: #E6F0FF
Primary Dark: #003366
```

#### Secondary (보조 색상)
```
Primary를 보완하는 색상
- 강조, 배경, 구분선
- 보색 또는 유사색

예:
Secondary: #FF9900 (주황색)
```

#### Neutral (무채색)
```
텍스트, 배경, 구분선
- White: #FFFFFF
- Light Gray: #F5F5F5
- Medium Gray: #999999
- Dark Gray: #333333
- Black: #000000
```

#### Accent (강조 색상)
```
정보, 성공, 주의, 오류
- Info: #0099FF
- Success: #00CC66
- Warning: #FFCC00
- Error: #FF3333
```

### 2. 타이포그래피

#### 한글 폰트
```
제목: Pretendard Bold, 44pt
소제목: Pretendard SemiBold, 32pt
본문: Pretendard Regular, 18pt
캡션: Pretendard Regular, 14pt
```

#### 영문 폰트
```
제목: Inter Bold, 44pt
본문: Inter Regular, 18pt
```

#### 폰트 무게
```
Bold: 제목, 강조
SemiBold: 소제목, 레이블
Regular: 본문, 설명
```

### 3. 슬라이드 레이아웃 (12개 템플릿)

```
1. Title Slide (제목 슬라이드)
   - 큰 제목 + 부제 + 날짜/작성자

2. Title + Content (제목 + 본문)
   - 좌측 제목, 우측 텍스트/이미지

3. Two Content (2분할)
   - 좌우 동일 비율 콘텐츠

4. Comparison (비교)
   - "A vs B" 레이아웃

5. Title + Big Image (제목 + 큰 이미지)
   - 이미지 중심, 제목 상단

6. Statistics (통계)
   - 숫자 강조, 3~4개 지표

7. Quote (명언/인용)
   - 큰 따옴표 + 인용문 + 출처

8. Section Header (섹션 제목)
   - 섹션 시작을 알리는 슬라이드

9. Three Column (3단)
   - 3개 항목 병렬 배치

10. Timeline (타임라인)
    - 시간 흐름에 따른 단계

11. Process (프로세스)
    - 화살표로 연결된 단계

12. Closing (종료/마무리)
    - 감사 인사, CTA, 연락처
```

## 슬라이드 레이아웃 상세

### 1. Title Slide (제목 슬라이드)
```
┌─────────────────────────────────┐
│                                 │
│                                 │
│    [메인 제목]                   │
│    [부제목]                      │
│                                 │
│              2024년 4월          │
│              작성자명            │
│                                 │
└─────────────────────────────────┘

디자인:
- 배경: 그래디언트 또는 단색 (Primary)
- 제목: Primary Dark, 56pt Bold
- 부제: Medium Gray, 28pt Regular
- 정렬: 중앙
```

### 2. Title + Content
```
┌─────────────────────────────────┐
│ [제목]                          │
├─────────────────────────────────┤
│ ■ 항목 1                       │
│   - 설명 1                      │
│   - 설명 2                      │
│                                 │
│ ■ 항목 2                       │
│   - 설명 3                      │
│   - 설명 4                      │
└─────────────────────────────────┘

디자인:
- 제목: Primary Dark, 40pt Bold
- 글머리: Primary 색상
- 본문: Dark Gray, 18pt Regular
- 라인 스페이싱: 1.5
```

### 3. Two Content (비교/병렬)
```
┌─────────────────────────────────┐
│        [제목]                    │
├─────────────────────────────────┤
│        │                        │
│ [좌측]  │  [우측]               │
│        │                        │
│        │                        │
└─────────────────────────────────┘

디자인:
- 50:50 분할 (또는 40:60)
- 구분선: Light Gray
- 각 섹션 제목: Primary, 24pt SemiBold
```

### 4. Statistics (통계)
```
┌─────────────────────────────────┐
│        [제목]                    │
├─────────────────────────────────┤
│                                 │
│  [100만+]  [50%]  [3배]        │
│   사용자   성장률   증가         │
│                                 │
└─────────────────────────────────┘

디자인:
- 숫자: Primary, 72pt Bold
- 라벨: Dark Gray, 18pt Regular
- 아이콘: 선택 (각 숫자 옆)
- 배경: Light Gray 박스 또는 None
```

### 5. Quote (명언)
```
┌─────────────────────────────────┐
│                                 │
│                                 │
│      " [인용문]               "  │
│                                 │
│              - 출처             │
│                                 │
└─────────────────────────────────┘

디자인:
- 따옴표: Primary, 큰 크기
- 텍스트: Primary Dark, 40pt
- 정렬: 중앙
- 배경: Accent 색상 (옅은)
```

### 6. Section Header
```
┌─────────────────────────────────┐
│                                 │
│                                 │
│  Ⅱ. 섹션 제목                   │
│                                 │
│                                 │
└─────────────────────────────────┘

디자인:
- 배경: Primary 색상 + 그래디언트
- 텍스트: 흰색, 56pt Bold
- 로마 숫자 또는 숫자 추가
```

## 마스터 슬라이드 생성

### PowerPoint 마스터 구성
```
1. Master Slide (기본 마스터)
   - 배경 이미지/색상
   - 로고 위치 (우측 하단)
   - 페이지 번호 (우측 하단)
   - 기본 글꼴 설정

2. Title Master
   - 타이틀 슬라이드용 마스터
   - 로고 + 그래디언트

3. Layouts (12개)
   - 각 슬라이드 유형별 레이아웃
```

### 마스터 설정 예시
```
[뷰] → [슬라이드 마스터]

마스터 슬라이드:
- 배경색: White (#FFFFFF)
- 로고: [우측 하단 5mm×5mm]
- 페이지 번호: [우측 하단 12pt]
- 기본 글꼴: Pretendard

레이아웃 마스터 (타이틀):
- 배경: Primary 그래디언트
- 로고: 중앙 하단 (크게)
```

## 절대 규칙 (Hard Rules)

### 1. 색상 일관성
```
- Primary는 제목, 강조, 링크에만 사용
- Secondary는 배경, 구분선에만 사용
- 사용 색상은 12개 이하 (복잡도 관리)
- 명암 대비: 최소 4.5:1 (텍스트 가독성)
```

### 2. 타이포그래피
```
- 슬라이드당 최대 3가지 폰트만 사용
- 제목: Bold 또는 SemiBold
- 본문: Regular
- 한글/영문 폰트 통일
```

### 3. 여백
```
- 상하좌우 20mm 여백 (기본)
- 제목 상단: 10mm
- 본문 시작: 제목 아래 15mm
- 요소 간격: 10mm 이상
```

### 4. 레이아웃
```
- 동일 섹션의 슬라이드는 같은 레이아웃
- 화면 비율: 16:9 (표준) 또는 4:3 (구형)
- 텍스트 가독성: 본문 최소 18pt
```

### 5. 로고/브랜드
```
- 로고는 우측 하단 (또는 중앙 상단) 고정
- 크기: 최소 5mm×5mm
- 투명도: 100% (불투명)
- 슬라이드 모두 동일 위치/크기
```

## 출력 형식

### PowerPoint 파일
```
Brand_Design_System.pptx
(12개 레이아웃 + 마스터 포함)
```

```
Design_Guide.docx 또는 .pdf
- 색상 팔레트 (색상값 포함)
- 폰트 사용 규칙
- 여백/간격 기준
- 사용 예시
```

### 색상 팔레트 파일
```
Palette.ase (Adobe 형식)
또는
Palette.json (개발자용)
```

## 워크플로우

```
1. 브랜드 요소 확인
   - 기존 색상/로고/폰트
   - 회사 정체성

2. 색상 팔레트 설계
   - Primary 선정
   - Secondary, Neutral, Accent 정의

3. 타이포그래피 정의
   - 한글/영문 폰트 선정
   - 크기 및 무게 정의

4. 슬라이드 레이아웃 설계
   - 필요한 레이아웃 우선순위
   - 마스터 슬라이드 구성

5. PowerPoint 마스터 적용
   - 마스터 슬라이드 생성
   - 12개 레이아웃 작성

6. 가이드 문서 작성
   - 사용 규칙
   - 제작 예시

7. 최종 PPT 파일 제공
   - Brand_Design_System.pptx
   - Design_Guide.pdf
```

## 트러블슈팅

| 증상 | 원인 | 해결 |
|------|------|------|
| 마스터 적용 안 됨 | 슬라이드가 로컬 마스터 | [뷰] → [슬라이드 마스터] 재접근 |
| 로고 위치 변함 | 슬라이드별 마스터 다름 | 모든 레이아웃에서 로고 위치 확인 |
| 폰트 깨짐 | 다른 PC에 폰트 미설치 | 폰트 임베드 또는 대체 폰트 설정 |
| 색상 표현 다름 | 모니터/프린터 색감 차이 | sRGB 색상공간 사용 |
| 슬라이드 크기 변경 어려움 | 고정된 마스터 | [디자인] → [슬라이드 크기] 조정 |
