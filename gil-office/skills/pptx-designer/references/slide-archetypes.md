# 9가지 비즈니스 슬라이드 아키타입

각 아키타입은 발표의 특정 순간에 사용하는 와이어프레임. Claude 톤 디자인 시스템 + 큐레이션 팔레트 위에 적용.

---

## 1. Title (표지)

```
┌─────────────────────────────────────────────┐
│                                             │
│                                  [LOGO]     │
│                                             │
│                                             │
│   ▌ [큰 제목 56pt Bold]                     │
│     [부제 18pt SemiBold Mid Gray]           │
│                                             │
│                                             │
│                                             │
│  2026.05.20 · 발표자명 · 부서               │
└─────────────────────────────────────────────┘
```

- 좌측 4pt 강조선 Primary 색
- 제목: Pretendard 56pt Bold, Ink
- 부제: Pretendard 18pt SemiBold, Mid
- 날짜·발표자: 11pt Regular, Mid

## 2. Agenda (목차)

```
┌─────────────────────────────────────────────┐
│  Agenda                                     │  ← 18pt
├─────────────────────────────────────────────┤
│                                             │
│   01 ⬢  배경 & 문제                         │
│                                             │
│   02 ⬢  솔루션                              │
│                                             │
│   03 ⬢  주요 기능                           │
│                                             │
│   04 ⬢  시장 데이터                         │
│                                             │
│   05 ⬢  요청 & 다음 단계                    │
│                                             │
└─────────────────────────────────────────────┘
```

- 번호: Primary 색 22pt Bold
- 아이콘: Primary, 작은 도형
- 항목: 22pt SemiBold Ink
- 줄 간격: 50pt 이상 (시원하게)

## 3. Problem (문제 정의)

```
┌─────────────────────────────────────────────┐
│  Problem                                    │
├─────────────────────────────────────────────┤
│                                             │
│   현재 시장이 겪고 있는 문제              │
│                                             │
│   ┌──────┐ ┌──────┐ ┌──────┐               │
│   │ ⚠️   │ │ 📉   │ │ ⏱️   │               │
│   │ 비용 │ │ 효율 │ │ 시간 │               │
│   │ 증가 │ │ 저하 │ │ 낭비 │               │
│   └──────┘ └──────┘ └──────┘               │
│                                             │
│   [본문 12pt] 시장 조사에 따르면 ...        │
│                                             │
└─────────────────────────────────────────────┘
```

- 헤딩: 36pt Bold Ink
- 3 카드 (또는 1 큰 문제): Beige 배경 + Primary 아이콘
- 본문: 14-18pt Regular Ink

## 4. Solution (해법)

```
┌─────────────────────────────────────────────┐
│  Our Solution                               │
├─────────────────────────────────────────────┤
│                                             │
│   ▌ 한 줄로 정의하는 우리의 가치           │
│     [강조선 4pt Primary]                    │
│                                             │
│   [좌] 시각 다이어그램      [우] 핵심 설명  │
│   - 흐름도 또는 일러스트   - 1-2-3 단계     │
│   - 또는 큰 이미지          - 각 단계 한 줄  │
│                                             │
└─────────────────────────────────────────────┘
```

- 헤딩: 36pt Bold Ink
- 좌우 분할 50:50 또는 60:40
- 다이어그램: Primary·Secondary 색만
- 본문: 16-18pt Regular Ink

## 5. Features (기능·차별점)

```
┌─────────────────────────────────────────────┐
│  Features                                   │
├─────────────────────────────────────────────┤
│                                             │
│   ┌─────────┬─────────┬─────────┐          │
│   │ ⬢       │ ⬣       │ ⬡       │          │
│   │ 기능 1  │ 기능 2  │ 기능 3  │          │
│   │ 한 줄   │ 한 줄   │ 한 줄   │          │
│   ├─────────┼─────────┼─────────┤          │
│   │ ⬟       │ ⬠       │ ⬢       │          │
│   │ 기능 4  │ 기능 5  │ 기능 6  │          │
│   └─────────┴─────────┴─────────┘          │
│                                             │
└─────────────────────────────────────────────┘
```

- 3×2 또는 3×1 그리드
- 아이콘: Primary, 28-32pt
- 기능명: 20pt SemiBold Ink
- 설명: 12-14pt Regular Mid

```
│  By the Numbers                             │
├─────────────────────────────────────────────┤
│                                             │
│                                             │
│    ▌37%▐         ▌2.4배▐         ▌₩1.2억▐  │
│    매출 증가      ROI 개선         예상 절감 │
│                                             │
│    ▌145개사▐                                 │
│    신규 고객                                 │
│                                             │
│   [출처: Q1 2026 보고서, 캡션 9pt Mid]      │
│                                             │

```

- 숫자: Primary 80pt Bold (큼)
- 라벨: 16pt Regular Mid
- 3-4개 강조 (1개 단독도 가능 — 더 임팩트)
- 출처: 9pt Mid 하단

```
│   ┌────┬────┬────┬────┐                    │

│   │ 이름│ 이름│ 이름│ 이름│                  │
│   │ 직책│ 직책│ 직책│ 직책│                  │
```

- 사진: 원형 100×100, Beige 보더 2pt
- 이름: 18pt Bold Ink
- 직책: 12pt Regular Mid
- 회사·경력: 9pt Mid

## 8. CTA (행동 요청)

```
┌─────────────────────────────────────────────┐
│                                             │
│                                             │
│    "[한 문장 가치 제안]                    │
│      Pretendard 40pt SemiBold]"             │
│                                             │
│                                             │
│      ╔═════════════════════════╗            │
│      ║   [지금 시작하기]       ║            │
│      ╚═════════════════════════╝            │
│       (Primary 배경 + White 텍스트)         │
│                                             │
│    contact@example.com · 02-XXX-XXXX        │
│                                             │
└─────────────────────────────────────────────┘
```

- 가치 제안 문장: 40pt SemiBold Ink
- CTA 버튼: Primary 색 채워진 박스, 텍스트 24pt SemiBold White
- 연락처: 14pt Regular Mid 하단

## 9. Closing (마무리·Q&A)

```
┌─────────────────────────────────────────────┐
│                                             │
│                                             │
│                                             │
│                                             │
│     감사합니다                              │
│                                             │
│     Thank You                               │
│                                             │
│                                             │
│     Q & A                                   │
│                                             │
│                                             │
│   contact@example.com · 회사 URL · QR       │
│                                             │
└─────────────────────────────────────────────┘
```

- "감사합니다": 80pt Bold Ink
- "Thank You": 32pt Regular Mid
- "Q & A": 60pt SemiBold Primary
- 연락처: 12-14pt Regular Mid 하단
- QR (옵션): 우측 하단, 회사 URL·LinkedIn 등

## 시퀀스 조합 예시

### 3분 데모 (5장)
1. Title
2. Problem
3. Solution
4. Stats
5. Closing

### 10분 사내 발표 (12장)
1. Title · 2. Agenda
3. Problem · 4. Solution
5-7. Features (3개씩 또는 6 그리드)
8. Stats · 9. Stats (2 장)
10. CTA · 11. Closing · 12. Q&A

### 30분 외부 피치 (25장)
1. Title · 2. Agenda
3-5. Problem (3 측면)
6-9. Solution (4p — 흐름·기술·차별성·증명)
10-13. Features (4-5 카테고리)
14-16. Stats (트랙션·시장·재무)
17. Team
18-22. Case studies (사례 5)
23. CTA · 24. Closing · 25. Q&A

---

## 아키타입 적용 규칙

- 한 슬라이드에 한 아키타입만 (혼합 금지)
- Title·Closing은 발표당 1개씩
- Stats는 연속 2-3장 가능 (트랙션·시장·재무)
- Features는 그리드로 묶기 (페이지마다 1-2개 아키타입 흩어지지 않게)
- 같은 아키타입은 동일 레이아웃 유지 (위치·크기 변형 금지)
