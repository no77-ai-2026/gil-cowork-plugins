# Modern Templates — 6대 문서 유형별 구조

각 템플릿은 [modern-design-system.md](./modern-design-system.md)의 Claude 브랜드 톤 디자인 시스템을 따릅니다.

## 1. 한국 공문서 (Mono Strict)

```
[로고/기관명]                                                      [문서번호: 2026-001]
                                                                  [작성일: 2026-05-20]

수신: [수신 기관·부서]
참조: [공유 대상]
제목: [한 줄 요약 — Bold]

1. 관련: [근거 문서·법령·이전 공문]
2. 본 건은 [목적]에 관한 것으로 다음과 같이 [동작]하고자 하니
   협조해 주시기 바랍니다.
   가. [세부 사항 1]
   나. [세부 사항 2]
   다. [세부 사항 3]
3. 붙임: [첨부 목록]
   가. [붙임 1] 1부.
   나. [붙임 2] 1부.   끝.

                                              [발신 기관명]
                                              [기관장 직위 · 성명]
                                              담당: [부서·이름·연락처]
```

**팔레트**: Mono Strict (Dark `#141413` + White)
**폰트**: 굴림 12pt (전통) 또는 Pretendard 11pt (모던 공문서)
**여백**: 상하 30mm·좌우 25mm

## 2. 기업 보고서 (Claude Classic)

```
┌──────────────────────────────────────────┐
│                                          │
│  2026 Q1 사업 성과 보고서                │  ← 표지 제목 28pt Bold
│  Quarterly Business Report               │  ← 부제 14pt Mid
│                                          │
│  2026년 5월 20일                          │
│  ○○사업본부 · 발행                       │
│                                          │
│  [로고]                                   │
└──────────────────────────────────────────┘

[p.2 — 목차]
1. Executive Summary
2. 시장 현황
3. Q1 성과 분석
4. 주요 이슈와 대응
5. Q2 계획

[p.3 — Executive Summary]
┃ Executive Summary                       ← Orange 강조선
┃ ────────────────────────────────────
┃ Q1 매출은 전 분기 대비 23% 증가했고
┃ 신규 고객 145개사를 확보했습니다.
┃ 다음 분기는 마진 개선에 집중합니다.

[p.4-N — 본문]
1. 시장 현황                              ← H1 22pt
   1-1. 시장 규모                          ← H2 18pt
   1-2. 경쟁 환경                          ← H2 18pt
        주요 경쟁사 동향                   ← H3 14pt

본문 ... Pretendard Regular 11pt 1.5 줄간격

   [Stat Callout]
   ╔══════════════════════════════════════╗
   ║   [37%]      [2.4배]      [₩1.2억]   ║
   ║  매출 증가   ROI 개선      예상 절감  ║
   ╚══════════════════════════════════════╝

   [표 — Header beige + zebra]
   ┌──────────┬──────────┬──────────┐
   │ 분기     │ 매출     │ 성장률   │  ← Header Light Beige
   ├──────────┼──────────┼──────────┤
   │ 2025 Q4  │ 100억    │ +12%     │
   │ 2026 Q1  │ 123억    │ +23%     │  ← zebra White
   └──────────┴──────────┴──────────┘

   [본문]                                ← H3 다음 본문

[마지막 페이지 — 발행 정보]
─── ⬢ ────────────────────────────────
발행: ○○사업본부
작성: [담당자]
문의: [이메일]
```

**팔레트**: Claude Classic (Orange + Beige + Dark)
**폰트**: Pretendard 11pt 본문
**여백**: 상하 25mm·좌우 22mm

## 3. 계약서 (Mono Strict)

```
[표지]                                  계약 번호: 2026-XXX
                                       체결일: 2026년 5월 20일

         [계약명] 계약서

   "갑"                              "을"
   상호: ○○주식회사                   상호: △△주식회사
   대표: ○○○                         대표: △△△
   주소:                              주소:
   사업자: 123-45-67890               사업자: 098-76-54321

[본문]
제1조 (목적)
   본 계약은 "갑"과 "을" 사이에 [목적]을 정함을 목적으로 한다.

제2조 (계약 기간)
   1. 계약 기간은 2026년 5월 20일부터 2026년 12월 31일까지로 한다.
   2. 계약 종료 30일 전까지 양 당사자의 합의가 없을 시 계약은 자동 연장된다.

제3조 (대금 및 지급)
   ┌──────────────┬──────────────────┐
   │ 항목         │ 금액             │
   ├──────────────┼──────────────────┤
   │ 기본 계약금  │ ₩10,000,000      │
   │ 부가세 (10%) │ ₩1,000,000       │
   │ 합계         │ ₩11,000,000      │
   └──────────────┴──────────────────┘
   지급 일정: 계약 체결 시 30%, 중간 30%, 완료 시 40%

제4조 (지적재산권) ...
제5조 (비밀유지) ...
제6조 (계약 해지) ...
제7조 (분쟁 해결) — 관할 법원: 서울중앙지방법원

[서명란]
"갑"                                "을"
서명: __________  (인)             서명: __________  (인)
일자: 2026.05.20                    일자: 2026.05.20
```

**팔레트**: Mono Strict
**폰트**: 맑은 고딕 11pt (또는 Pretendard 11pt)
**여백**: 상하 25mm·좌우 22mm

## 4. 제안서 (Claude Coral)

```
[표지]
   [회사 로고]

   [고객사 이름]을 위한
   [핵심 가치 제안]

   ── Proposal ──
   제출: 2026.05.20
   유효: 2026.06.20

   ○○회사 | 담당: [이름] | [이메일]

[p.2 — 요약]
┃ Executive Summary                      ← Coral 강조
┃ ────────────────────────────────
┃ 4-5줄 핵심 — 무엇을, 왜, 어떻게, 얼마에

[p.3 — 고객 현황]
   현재 마주한 도전 과제                  ← H1
   ─ 도전 1: [...]
   ─ 도전 2: [...]
   ─ 도전 3: [...]

[p.4 — 우리의 제안]
   제안 개요                             ← H1

   [3-column Feature Grid]
   ┌─────────┬─────────┬─────────┐
   │ ⬢       │ ⬣       │ ⬡       │  ← 아이콘 (Coral)
   │ 기능 1  │ 기능 2  │ 기능 3  │
   │ 설명    │ 설명    │ 설명    │
   └─────────┴─────────┴─────────┘

[p.5 — 기대 효과]
   [Stat Callout]
   ╔══════════════════════════════╗
   ║   [40%]    [3개월]    [10배] ║
   ║   비용 절감 ROI 회수   효율   ║
   ╚══════════════════════════════╝

[p.6 — 일정 & 예산]
   ─── 단계별 일정 ───
   M1 [킥오프] ──── M2 [구현] ──── M3 [출시]

   [표]
   단계 | 기간 | 산출물 | 비용

[p.7 — 팀 & 사례]
   추진 팀
   - [팀원 1] · 역할 · 경력
   - [팀원 2] · 역할 · 경력

   유사 사례
   - [사례 1] · 결과 지표

[p.8 — 다음 단계]
   1. [날짜] 미팅 / 데모
   2. [날짜] 계약
   3. [날짜] 킥오프

   감사합니다.
   문의: [이메일·전화]
```

**팔레트**: Claude Coral (Crail `#c15f3c` + Pampas `#f4f3ee` + Dark)
**폰트**: Pretendard
**여백**: 상하 22mm·좌우 20mm
**특징**: Stat Callout · Feature Grid · 강한 시각적 임팩트

## 5. 기획서 (Claude Blue)

```
[표지]
   ── 2026 신규 사업 기획서 ──

   [핵심 메시지 한 줄]

   기획: [부서]
   2026.05.20

[p.2 — 한 페이지 요약]
   배경:    [...]
   목표:    [...]
   전략:    [...]
   실행:    [...]
   KPI:     [...]

[p.3 — 배경 & 문제 정의]
   왜 지금 이 기획이 필요한가              ← H1
   1. 시장 변화
   2. 내부 자원 변화
   3. 경쟁 변화

   [Pull Quote — Blue 좌측 보더]
   ┃ "고객의 N% 는 [문제]를 겪고 있으나
   ┃  적절한 해결책을 찾지 못하고 있다."
   ┃                       — 시장 조사 보고서 2026

[p.4 — 목표]
   ┌─── 정량 목표 ───┬─── 정성 목표 ───┐
   │ 매출 ○○억      │ 브랜드 인지     │
   │ MAU ○○만       │ 시장 포지셔닝   │
   │ 만족도 NPS 50  │ 팀 역량         │
   └────────────────┴────────────────┘

[p.5-N — 전략 & 실행]
   ── 3대 전략 축 ──
   전략 1. [...]
       세부 실행안 ...
   전략 2. [...]
   전략 3. [...]

   타임라인
   Q2 → Q3 → Q4 (간트 차트)

[p.last — 리스크 & Q&A]
   주요 리스크
   - 리스크 1 · 대응안
   - 리스크 2 · 대응안

   질문 & 답변
   - Q. [예상 질문]  A. [답변]
```

**팔레트**: Claude Blue (Anthropic Blue `#6a9bcc` + Beige + Dark)
**폰트**: Pretendard
**여백**: 상하 25mm·좌우 22mm
**특징**: Pull Quote · 단순 다이어그램 · 데이터 강조

## 6. 사업계획서 (Claude Bold)

```
[표지 — Full bleed Orange]
                                              [로고 — 우측 상단]

   ○○○○○○

   "한 문장으로 정의하는 우리의 가치"

   ── Business Plan 2026 ──

   2026.05.20 | 작성: [이름·직책]

[p.2 — 1-page Summary]
   회사:    [한 줄]
   문제:    [한 줄]
   해법:    [한 줄]
   시장:    [TAM/SAM/SOM 한 줄]
   비즈니스 모델: [한 줄]
   트랙션:  [지표 한 줄]
   요청:    [라운드 규모 + 사용처]

[p.3-N — 본문 14-20 페이지]
   1. 회사 & 비전 (2p)
   2. 문제 & 시장 (2p)
   3. 솔루션 & 차별성 (3p)
   4. 비즈니스 모델 & 시장 진입 (2p)
   5. 트랙션 & 데이터 (2p) — Stat Callout 활용
   6. 경쟁 분석 (2p) — Comparison Table
   7. 재무 계획 & 추정 (3p) — 표·차트 중심
   8. 팀 (1p)
   9. 요청 & 사용처 (1p)
   10. 부록 — 시장 자료·인터뷰·법적 정보

각 페이지 디자인:
- 좌측 22mm 여백에 작은 페이지 번호 + 섹션명 (Orange 작은 글자)
- 상단 H1 — Orange 좌측 강조선 4pt + Dark 22pt Bold
- 본문 — Pretendard 11pt 1.5 줄간격
- 표·차트는 Orange 강조 시 한 곳만, 나머지는 Mid Gray
```

**팔레트**: Claude Bold (Orange highlight + Dark + Light Beige)
**폰트**: Pretendard Bold 헤딩 + Pretendard Regular 본문
**여백**: 상하 22mm·좌우 20mm
**특징**: 표지 임팩트 · Stat Callout 적극 활용 · 챕터별 시각적 일관성

## 사용 가이드

1. 문서 유형을 결정 → 위 6개 중 1개 선택
2. 팔레트·폰트·여백은 해당 템플릿 따름
3. 본문 구조는 템플릿 골격 유지
4. 색·타이포 사용 규칙은 [modern-design-system.md](./modern-design-system.md) 따름
5. 결과는 [qa-checklist.md](./qa-checklist.md) 10단계 점검 후 출력
