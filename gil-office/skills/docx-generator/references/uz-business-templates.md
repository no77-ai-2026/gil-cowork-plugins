# UZ 비즈니스 문서 양식 가이드

> gil-office v0.1.0 | UZ 계약서·인보이스·MOU·견적서 양식

## 개요

UZ 비즈니스 문서 양식 모음. 한국 회사가 UZ 자회사·UZ 거래처와 거래 시 사용하는 표준 양식. 모든 양식은 러시아어 본문 + 한국어/영어 병기 옵션 + INN·UZS·NDS 12% 의무 표기.

---

## 1. 표준 비즈니스 계약서 (Договор)

### 10개 표준 조항

```
ДОГОВОР № [번호]
г. [도시], [DD.MM.YYYY]

[Сторона 1 정보 - 회사명·INN·주소·대표자]
[Сторона 2 정보 - 동일]

заключили настоящий договор о нижеследующем:

1. ПРЕДМЕТ ДОГОВОРА
   Сторона 1 обязуется [의무], а Сторона 2 обязуется [의무]

2. СТОИМОСТЬ И ПОРЯДОК ОПЛАТЫ
   2.1. Общая стоимость: [금액] UZS (включая НДС 12%)
   2.2. Порядок оплаты:
        - Предоплата [%] в течение [일수] дней
        - Окончательный расчёт по факту
   2.3. Валюта: UZS / USD (외화 거래 시)

3. СРОКИ ИСПОЛНЕНИЯ
   3.1. Начало: [DD.MM.YYYY]
   3.2. Окончание: [DD.MM.YYYY]
   3.3. Этапы [필요 시]

4. ПРАВА И ОБЯЗАННОСТИ СТОРОН
   4.1. Сторона 1 обязуется:
        - [의무 1]
        - [의무 2]
   4.2. Сторона 2 обязуется:
        - [의무 1]

5. ПОРЯДОК ПРИЁМКИ
   5.1. Акт сдачи-приёмки в течение [일수] дней

6. ОТВЕТСТВЕННОСТЬ СТОРОН
   6.1. Пеня за просрочку: [%] от суммы за каждый день
   6.2. Максимальная ответственность: [한도]

7. ФОРС-МАЖОР
   Стороны не несут ответственности при обстоятельствах непреодолимой силы

8. КОНФИДЕНЦИАЛЬНОСТЬ
   8.1. Стороны обязуются не разглашать конфиденциальную информацию
   8.2. Срок: [기간], даже после расторжения договора

9. РАЗРЕШЕНИЕ СПОРОВ
   9.1. Споры решаются путём переговоров
   9.2. При невозможности — Хозяйственный суд г. [도시]
   9.3. Применимое право: законодательство Республики Узбекистан

10. ЗАКЛЮЧИТЕЛЬНЫЕ ПОЛОЖЕНИЯ
    10.1. Договор составлен в [부수] экземплярах
    10.2. Языки: русский (основной) [+ корейский 인터컴퍼니]
    10.3. Изменения только в письменной форме

РЕКВИЗИТЫ И ПОДПИСИ СТОРОН:

Сторона 1:                                Сторона 2:
[회사 정보 풀]                              [회사 정보 풀]

_____________________ Ф.И.О.            _____________________ Ф.И.О.
              М.П.                                    М.П.

Дата: ___________                       Дата: ___________
```

### 한국 본사 ↔ UZ 자회사 인터컴퍼니 변형

추가 조항:
- 양국 법인 정보 (한국 본사 + UZ 자회사)
- 통화: USD 우선 (외환 송금 자유), UZS 보조
- 이전가격 (Transfer Pricing) 명시
- 양국 분쟁 관할 (한국 또는 UZ 또는 국제 중재)

## 2. Счёт-фактура (인보이스)

NDS 12% 의무 표기. soliq.uz 전자 인보이스 의무 (2022~).

### 양식

```
СЧЁТ-ФАКТУРА № [번호]
от [DD.MM.YYYY]

Поставщик:
Название: ООО "[회사명]"
ИНН: [9자리]
Адрес: [주소]
р/с: [계좌] в [은행명]

Покупатель:
Название: [회사명]
ИНН: [9자리]
Адрес: [주소]

Договор: № [계약번호] от [날짜]

| № | Наименование товара/услуги | Ед. изм. | Кол-во | Цена за ед. | Сумма | НДС 12% | Итого |
| 1 | [품목] | шт. | 1 | 100 000 | 100 000 | 12 000 | 112 000 |
| 2 | [품목] | кг | 5 | 20 000 | 100 000 | 12 000 | 112 000 |

Всего по счёту:
- Без НДС: 200 000 UZS
- НДС 12%: 24 000 UZS
- Итого с НДС: 224 000 UZS

Сумма прописью: двести двадцать четыре тысячи UZS

Поставщик [서명]                  Покупатель [서명]
       М.П.                              М.П.
```

### NDS 면제 카테고리

- 기본 의약품 (0%)
- 일부 농산물 (0%)
- 외환 거래 (0%)
- 수출 (0%)
- 교육·의료 일부 (0%)

상세는 `gil-finance:tax-helper`의 UZ 세제 가이드 참조.

## 5. 위임장 (Доверенность)

UZ에서 일부 행정 처리 시 필수 (계약 서명 대리·은행 거래·상표 등록 등).

### 양식

```
ДОВЕРЕННОСТЬ № [번호]

г. [도시], [DD.MM.YYYY]

ООО "[회사명]" (ИНН: [번호], адрес: [주소]),
в лице директора [이름], действующего на основании Устава,

настоящей доверенностью уполномочивает

[수임인 이름] (паспорт: [여권번호], адрес: [주소])

совершать от имени Доверителя следующие действия:

1. [권한 1]
2. [권한 2]
3. [권한 3]

Срок действия: [기간]

Подпись доверенного лица: _____________

Директор:
_____________________ [이름]
       М.П.

[공증 — 해당 시]
Удостоверено нотариусом [이름] [날짜]
```

> 일부 위임장은 공증 (нотариальное заверение) 필요. 타슈켄트 공증사무소 방문.

## 7. 표준 약자·용어

### 회사 형태

| 약자 | 풀 명칭 | 설명 |
|------|---------|------|
| ООО | Общество с ограниченной ответственностью | 유한책임회사 (한국 LLC) |
| АО | Акционерное общество | 주식회사 |
| ИП | Индивидуальный предприниматель | 개인사업자 |
| ИЧТ | (UZ-specific) | 외국인 100% 출자 LLC |

### 직급

| 러시아어 | 한국어 |
|---------|--------|
| Директор | 대표이사 |
| Генеральный директор | 사장 |
| Главный бухгалтер | 회계 책임자 |
| Менеджер | 매니저 |
| Заместитель директора | 부사장 |

### 일반 약자

| 약자 | 의미 |
|------|------|
| ИНН | 사업자등록번호 (한국 사업자번호 등가) |
| НДС | 부가가치세 (12%) |
| ЕСП | 사회세 (12% 또는 1%) |
| М.П. | 도장 자리 (Место Печати) |
| р/с | 은행 계좌 번호 |
| МФО | 은행 코드 |
| Ф.И.О. | 성명 (姓名 풀 표기) |
| г. | 도시 (город) |
| ОКЭД | UZ 업종 분류 |

## 8. 자주 하는 실수

1. **콤마로 천 단위 구분**: UZ 표준 공백 (`850 000 UZS` not `850,000 UZS`)
2. **NDS 12% 누락**: 의무 표기
3. **INN 누락 또는 형식 오류**: 9자리 숫자 정확
4. **날짜 미국식**: DD.MM.YYYY가 UZ 표준
5. **М.П. 자리 없음**: 도장 자리 명시 필수
6. **공증 필요 문서 미공증**: 일부 행정 거부
7. **러시아어 단독**: 정부 제출 시 우즈벡어 의무
8. **고가 인터컴퍼니**: 이전가격 미문서화 → 양국 세무 추징

## 9. python-docx 코드 (계약서 자동 생성)

```python
from docx import Document
from docx.shared import Pt, Cm

doc = Document()
section = doc.sections[0]
section.top_margin = Cm(2.5)
section.bottom_margin = Cm(2.5)
section.left_margin = Cm(2.5)
section.right_margin = Cm(2.5)

# 제목
title = doc.add_paragraph()
title.alignment = 1  # Center
run = title.add_run(f'ДОГОВОР № {contract_number}')
run.font.name = 'Inter'
run.font.size = Pt(14)
run.bold = True

# 날짜·도시
date_p = doc.add_paragraph()
date_p.alignment = 1
run = date_p.add_run(f'г. {city}, {date_str}')
run.font.size = Pt(11)

# 당사자 정보
party1 = doc.add_paragraph(
    f'ООО "{party1_name}", именуемое в дальнейшем "Сторона 1", '
    f'ИНН {party1_inn}, адрес: {party1_address}, '
    f'в лице директора {party1_director}, действующего на основании Устава, '
    'с одной стороны,'
)
party1.runs[0].font.size = Pt(11)

# (Сторона 2 동일)

# 조항 1: 예제
section1 = doc.add_paragraph('1. ПРЕДМЕТ ДОГОВОРА')
section1.runs[0].bold = True

doc.add_paragraph(
    f'1.1. Сторона 1 обязуется передать товар, '
    f'а Сторона 2 — оплатить стоимостью {price_uzs:,}'.replace(',', ' ')
    + ' UZS (включая НДС 12%).'
)

# 서명 표
table = doc.add_table(rows=2, cols=2)
table.cell(0, 0).text = 'Сторона 1:'
table.cell(0, 1).text = 'Сторона 2:'
table.cell(1, 0).text = '\n_______________ ' + party1_director + '\n      М.П.'
table.cell(1, 1).text = '\n_______________ ' + party2_director + '\n      М.П.'

doc.save(f'contract_{contract_number}.docx')
```

## 10. 검수 자원

| 자원 | 용도 |
|------|------|
| `gil-legal:contract-review` | 계약서 법률 검토 |
| `gil-legal:nda-triage` | NDA 빠른 검토 |
| `gil-legal:compliance-check` | 규제 준수 |
| `gil-finance:tax-helper` | NDS·세무 검토 |
| KOTRA Tashkent | 한국 회사 무료 자문 |
| 현지 변호사 | 법적 구속력 검수 (필수) |
| 공증사무소 (Нотариус) | 공증 (해당 시) |

---

> **주의**: UZ 비즈니스 양식은 «Гражданский кодекс РУ» 및 관련 법령 준수 필수. 본 가이드는 일반 양식이며, 카테고리별·금액별 추가 조항·공증 필요. 법적 구속력 있는 계약 발행 전 현지 변호사 검수 필수.
