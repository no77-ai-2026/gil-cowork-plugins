# UZ 정부·기업 문서 표준 가이드

> gil-office v0.1.0 | UZ 비즈니스 문서 양식·통화·INN·서명란

## 개요

UZ 비즈니스 문서는 한국·영미와 형식이 크게 다릅니다. 통화 표기 (공백 구분), INN(STIR) 의무 표기, 서명·도장 양식, 러시아어/우즈벡어 헤더 등 표준 점검이 필수입니다.

## 2. INN(STIR) 표기 의무

UZ 모든 비즈니스 문서에 INN 표기 (한국 사업자등록번호 등가).

### 형식

- 9자리 숫자
- 예: `123456789`
- 검증: https://soliq.uz (UZ 국세청)

### 비즈니스 문서 필수 항목

```
ИНН: 123456789
ОКЭД: 47.11.1 (예: 식품 소매)
р/с: 20208000900XXXXXXXX (계좌)
МФО: 00014 (은행 코드)
Банк: АКБ "Asaka Bank" г. Ташкент
```

## 3. 통화 표기 (UZS)

### 천 단위 구분

UZ 표준: **공백** (한국 콤마와 다름)

| 잘못 | 올바른 |
|------|--------|
| 1,000,000 UZS | 1 000 000 UZS |
| 850,000 UZS | 850 000 UZS |

### 통화 코드

- UZS (러시아어): сум
- UZS (우즈벡 라틴): so'm
- 약자: `сум`, `so'm`, `UZS`
- 국제 표기: ISO 4217 = `UZS`

### 환율 표기 (인터컴퍼니)

```
Стоимость:
- 850 000 UZS
- ≈ $68 USD (курс 12 500 UZS/USD на [날짜])
- ≈ 93 600 KRW (курс 1380 KRW/USD)
```

## 5. 서명란 양식

### 양국 서명 (한국+UZ)

```

        [도장]                                  М.П.

날짜: [YYYY-MM-DD]                     Дата: [DD.MM.YYYY]
```

## 6. 러시아어 vs 우즈벡어 라틴

### 비즈니스 문서 표준 언어

| 문서 유형 | 권장 |
|----------|------|
| 일반 비즈니스 계약 | 러시아어 (UZ 비즈니스 표준) |
| 정부 제출 | 우즈벡어 (라틴) 의무 |
| 외국 기업 거래 | 러시아어 + 영어 병기 가능 |
| 내부 문서 | 러시아어 또는 우즈벡어 |

### 우즈벡 라틴 특수 문자

- `oʻ` (o + 모디파이어 letter prime)
- `gʻ` (g + 모디파이어 letter prime)
- `sh`, `ch`, `ng` (디그래프)
- 숫자·문장부호는 영어와 동일

### 키릴 알파벳 (구식)

- 농촌·고령층용
- 점차 라틴으로 전환 중
- 비즈니스 문서는 라틴 권장

---

## 7. 폰트 가이드

### 러시아어 (키릴)

- **권장**: Inter Cyrillic, Roboto Cyrillic, PT Sans
- **전통**: Times New Roman, Arial
- **금지**: 키릴 미지원 폰트 (한국 전용 폰트는 깨짐)

### 우즈벡어 (라틴)

- **권장**: Inter, Roboto, Open Sans
- **전통**: Times New Roman, Arial
- **특수 문자**: `ʻ` (모디파이어 letter prime, U+02BB) 표시 정확

### 한국어 + 러시아어 + 우즈벡어 동시

- 단일 다국어 폰트: **Inter** (Cyrillic + Latin 모두 지원)
- 한국어 보조: Pretendard 또는 맑은 고딕

## 8. 문서 카테고리별 표준

### 계약서 (Договор)

```
ДОГОВОР № [번호]
г. [도시], [날짜 DD.MM.YYYY]

[10대 표준 조항 - 상세 uz-business-templates.md 참조]

РЕКВИЗИТЫ И ПОДПИСИ СТОРОН:
[갑·을 INN·주소·은행 정보·서명]
```

### 인보이스 (Счёт-фактура)

```
СЧЁТ-ФАКТУРА № [번호]
от [날짜]

[NDS 12% 의무 표기]
[INN 양국 표기]
[UZS 가격 + USD 환산]
```

### 협조 공문 (Сопроводительное письмо)

```
[보내는 회사 헤더]

Кому: [수신처]
Тема: [제목]

Уважаемый/ая [수신자],

[본문]

С уважением,
[보내는 사람]
[직급]
[날짜] [서명]
```

### 견적서 (Коммерческое предложение)

```
КОММЕРЧЕСКОЕ ПРЕДЛОЖЕНИЕ
[회사 헤더]

Дата: [DD.MM.YYYY]
Действительно до: [DD.MM.YYYY]

| № | Наименование | Кол-во | Цена | Сумма | НДС 12% |
| 1 | ... | 1 | 100 000 | 100 000 | 12 000 |

Итого: 112 000 UZS

Условия оплаты: [결제 조건]
Срок поставки: [납기]
```

### MOU (Меморандум о взаимопонимании)

```
МЕМОРАНДУМ О ВЗАИМОПОНИМАНИИ
между [Сторона 1] и [Сторона 2]

[목적·범위·기간·기밀유지·서명]
```

## 9. 인터컴퍼니 거래 문서 (한국 본사 ↔ UZ 자회사)

### 필수 표기

- 양국 회사명 + INN (한국 사업자등록번호 + UZ INN)
- 양국 통화 + USD 환산 (이전가격 문서화)
- 양국 법무 검토 도장
- 양국 서명 (한국 본사 대표이사 + UZ 자회사 Директор)

### 이전가격 (Transfer Pricing) 문서

UZ 세무 + 한국 세무 양국 적용:
- Arm's Length Principle (시장 가격 기준)
- 비교 가능 거래 자료
- 5년 보관 (양국 세무 감사 대비)

상세는 `gil-finance:financial-statements`의 인터컴퍼니 가이드 참조.

## 10. 정부 제출 문서

### 우즈벡어 라틴 의무

UZ 정부·공공기관 제출 시 우즈벡어 라틴 의무 (러시아어 단독 X).

### 공증 (Нотариальное заверение)

일부 문서는 공증 필요:
- 외국인 투자 등록
- 법인 등록
- 상표 등록
- 부동산 거래
- 위임장

타슈켄트 공증사무소 (Нотариальная контора) 방문 또는 공증인 출장.

### 아포스티유 (Apostille)

한국 본사 문서를 UZ 정부에 제출 시:
1. 한국 외교부 또는 법무부 아포스티유
2. 우즈벡어/러시아어 번역 (공증 번역사)
3. UZ 공증 추가 (해당 시)

## 12. python-docx 코드 예시 (UZ)

```python
from docx import Document
from docx.shared import Pt, Cm, RGBColor
from docx.enum.text import WD_ALIGN_PARAGRAPH

doc = Document()

# 페이지 설정
section = doc.sections[0]
section.page_height = Cm(29.7)
section.page_width = Cm(21.0)
section.top_margin = Cm(2.5)
section.bottom_margin = Cm(2.5)
section.left_margin = Cm(2.5)
section.right_margin = Cm(2.5)

# 헤더 (회사 정보)
header = doc.add_paragraph()
header.alignment = WD_ALIGN_PARAGRAPH.RIGHT
run = header.add_run('ООО "회사명"\n')
run.font.name = 'Inter'
run.font.size = Pt(11)
run.font.bold = True

run = header.add_run('Юридический адрес: [주소]\n')
run.font.size = Pt(10)

run = header.add_run('ИНН: 123456789 | ОКЭД: 47.11.1\n')
run.font.size = Pt(10)

# 제목
title = doc.add_paragraph()
title.alignment = WD_ALIGN_PARAGRAPH.CENTER
run = title.add_run('ДОГОВОР № 001/2026')
run.font.name = 'Inter'
run.font.size = Pt(14)
run.font.bold = True

# 날짜·도시
location = doc.add_paragraph()
location.alignment = WD_ALIGN_PARAGRAPH.CENTER
run = location.add_run('г. Ташкент, 29.04.2026 г.')
run.font.size = Pt(11)

# 본문 (Inter 폰트로 키릴·라틴 모두 지원)
body = doc.add_paragraph('Настоящий договор...')
body.runs[0].font.name = 'Inter'
body.runs[0].font.size = Pt(11)

# UZS 통화 (공백 구분)
price = doc.add_paragraph('Стоимость: 850 000 UZS (≈ $68 USD)')

doc.save('uz_contract.docx')
```
