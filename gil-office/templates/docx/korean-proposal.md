# 한국 비즈니스 제안서 DOCX 템플릿 명세

## 개요

한국 B2B 비즈니스 환경에 최적화된 제안서 DOCX 템플릿. 입찰 제안, 컨설팅 제안, 파트너십 제안 등 다목적 활용.

---

## 문서 구조

### 1. 표지

| 항목 | 설명 |
|------|------|
| 프로젝트명 | 제안 프로젝트/사업 명칭 (24pt, 굵게, 중앙) |
| 제안사 | 회사명 + 로고 삽입 공간 |
| 제안일 | YYYY년 MM월 DD일 |
| 수신처 | 발주처/고객사명 (선택) |
| 제안 번호 | 내부 관리용 제안 번호 |

### 2. 회사 소개

- 회사 개요 (설립연도, 인원, 주요 사업)
- 핵심 역량 및 강점 (3-5개 항목)
- 주요 실적 및 고객사 목록
- 인증/수상 이력 (해당 시)

### 3. 제안 배경 및 목적

- 고객사 현황 및 Pain Point 분석
- 제안 필요성 및 기대 효과 요약
- 제안 목표 (SMART 기준 기술 권장)

### 4. 제안 내용

#### 4.1 범위 (Scope of Work)
- 포함 업무 범위 명시
- 제외 범위 명시 (분쟁 예방)

#### 4.2 방법론
- 추진 방법론 및 프레임워크
- 단계별 접근 방식
- 품질 관리 방안

#### 4.3 일정 (Schedule)
- 마일스톤 기반 일정표 (표 형식)
- 주요 산출물 및 납기일
- 단계별 완료 조건

### 5. 추진 체계

#### 5.1 조직 구성
- 제안사 추진 조직도
- 고객사-제안사 협업 구조

#### 5.2 투입 인력
| 역할 | 성명 | 경력 | 투입 기간 |
|------|------|------|---------|
| PM | | | |
| 담당자 | | | |

### 6. 예산

| 항목 | 단가 | 수량 | 금액 |
|------|------|------|------|
| 인건비 | | | |
| 직접 경비 | | | |
| 기타 | | | |
| **합계** | | | |

- VAT 별도/포함 명시
- 결제 조건 (선금/중도금/잔금 비율)
- 유효 기간 (제안 유효 기간)

### 7. 기대 효과

- 정량적 효과 (수치 기반)
- 정성적 효과 (운영 개선, 역량 향상 등)
- ROI 추정 (가능한 경우)

### 8. 부록

- 포트폴리오 (유사 프로젝트 사례)
- 참조 사례 (레퍼런스)
- 회사 등록증/인증서 사본
- 담당자 프로필

## 페이지 설정

| 항목 | 값 |
|------|-----|
| 용지 | A4 (210mm × 297mm) |
| 여백 상 | 25mm |
| 여백 하 | 25mm |
| 여백 좌 | 30mm |
| 여백 우 | 25mm |
| 머리글 | 회사 로고 + 제안 번호 |
| 바닥글 | 기밀 표시 + 페이지 번호 |

---

## 스타일 정의

| 스타일명 | 폰트 | 크기 | 속성 |
|---------|------|------|------|
| 표지 제목 | 맑은 고딕 | 24pt | 굵게, 진한 남색 |
| 섹션 제목 | 맑은 고딕 | 16pt | 굵게, 브랜드 컬러 |
| 소제목 | 맑은 고딕 | 13pt | 굵게, 어두운 회색 |
| 본문 | Pretendard | 11pt | 보통, 줄간격 1.5 |
| 강조 박스 | Pretendard | 11pt | 배경색 연한 파랑, 테두리 |

---

## 기밀 등급 표시

- 표지 하단: "대외비" 또는 "기밀" 스탬프 스타일 텍스트 박스
- 모든 페이지 바닥글: "본 제안서는 제안 목적 외 무단 공개를 금합니다."

---

## 체크리스트

- [ ] 표지 항목 (프로젝트명, 제안사, 제안일) 모두 입력
- [ ] 회사 소개 최신 정보 반영
- [ ] 제안 범위 및 제외 범위 명확히 기술
- [ ] 일정표 마일스톤 확인
- [ ] 투입 인력 정보 최신화
- [ ] 예산 합계 및 VAT 처리 확인
- [ ] 기대 효과 정량 수치 포함
- [ ] 부록 포트폴리오 최신 사례 3개 이상
- [ ] 기밀 표시 적용
- [ ] 제안 유효 기간 명시
