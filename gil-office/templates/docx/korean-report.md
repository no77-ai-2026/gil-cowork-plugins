# 한국 비즈니스 보고서 DOCX 템플릿 명세

## 개요

한국 비즈니스 환경 표준에 맞는 공식 보고서 DOCX 템플릿. 정부 및 대기업 제출용 양식 기준 준수.

---

## 문서 구조

### 1. 표지

| 항목 | 설명 |
|------|------|
| 제목 | 보고서 메인 제목 (맑은 고딕, 24pt, 굵게, 중앙 정렬) |
| 부제 | 보조 제목 (맑은 고딕, 16pt, 중앙 정렬) |
| 작성일 | YYYY년 MM월 DD일 형식 |
| 작성자 | 직책 + 성명 |
| 부서 | 소속 부서명 (로고 삽입 공간 포함) |

### 2. 목차

- 자동 생성 (Word 목차 필드 활용)
- 들여쓰기 2단계: 제목1 → 제목2
- 점선 탭 리더 + 페이지 번호 우측 정렬

### 3. 요약 (Executive Summary)

- 전체 1페이지 이내
- 핵심 내용 3-5개 불릿 포인트
- 의사결정자 대상 요약 박스 스타일 적용

### 4. 본문

#### 1. 배경
- 보고서 작성 배경 및 필요성
- 관련 법규/정책 참조 (해당 시)

#### 2. 현황 분석
- 데이터 기반 현황 기술
- 표 또는 차트 삽입 공간 포함

#### 3. 주요 발견
- 분석 결과 및 인사이트
- 번호 목록 또는 강조 박스 활용

#### 4. 결론 및 제언
- 결론 요약
- 실행 가능한 제언 항목별 기술

### 5. 부록

- 참고 자료 목록
- 데이터 출처 및 방법론
- 추가 표/차트

---

## 페이지 설정

| 항목 | 값 |
|------|-----|
| 용지 | A4 (210mm × 297mm) |
| 여백 상 | 30mm |
| 여백 하 | 25mm |
| 여백 좌 | 30mm |
| 여백 우 | 25mm |
| 머리글 | 15mm (보고서 제목 + 부서명) |
| 바닥글 | 15mm (페이지 번호, 하단 중앙) |
| 페이지 번호 | 표지 제외, 목차부터 시작 |

---

## 스타일 정의

### 텍스트 스타일

| 스타일명 | 폰트 | 크기 | 속성 |
|---------|------|------|------|
| 제목1 | 맑은 고딕 | 18pt | 굵게, 진한 남색 (#003366) |
| 제목2 | 맑은 고딕 | 14pt | 굵게, 남색 (#1F4E79) |
| 제목3 | 맑은 고딕 | 12pt | 굵게, 어두운 회색 (#404040) |
| 본문 | Pretendard | 11pt | 보통, 검정 (#1A1A1A), 줄간격 1.5 |
| 인용 | Pretendard | 10pt | 이탤릭, 회색 (#666666), 들여쓰기 1cm |

### 표 스타일

- 헤더 행: 남색 배경 (#1F4E79), 흰색 텍스트, 굵게
- 홀수 행: 연한 파랑 (#EBF3FB)
- 짝수 행: 흰색
- 테두리: 연한 회색 (#CCCCCC), 0.5pt

## 폰트 요구사항

| 용도 | 폰트 | 대체 폰트 |
|------|------|----------|
| 제목 전체 | 맑은 고딕 | 나눔고딕 |
| 본문 텍스트 | Pretendard | 나눔바른고딕 |
| 영문/숫자 | Inter | Calibri |

> 폰트 미설치 환경 대비: 시스템 기본 한글 폰트(맑은 고딕) 폴백 적용

## 체크리스트

- [ ] 목차 자동 생성 필드 삽입 완료
- [ ] 요약 1페이지 이내 유지
- [ ] 본문 4개 섹션 구조 준수
- [ ] A4 여백 설정 적용
- [ ] 머리글/바닥글 설정 완료
- [ ] 스타일 일관성 확인
- [ ] 부록에 참고 자료 목록 포함
