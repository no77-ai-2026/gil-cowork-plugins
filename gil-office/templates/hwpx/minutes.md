# HWPX 회의록 양식 (minutes)

> gil-office v1.0.0 | 회의록 표준 스펙

## 개요

회의 정보, 참석자, 안건, 결정사항, 향후조치를 체계적으로 기록하는 HWPX 회의록 템플릿입니다.

---

## 1. 페이지 설정

`base.md` 상속, 아래 항목 덮어씀:

```yaml
여백:
  위: 25mm
  아래: 25mm
  왼쪽: 30mm
  오른쪽: 25mm
  
페이지_크기: A4 세로
```

## 3. 섹션별 상세 스펙

### 3.1 표지 정보 표

```yaml
회의정보_표:
  스타일: 헤더_배경 진남색(#1F3864) 헤더_폰트 흰색
  열너비: [레이블 20%, 내용 80%]
  셀높이: 8mm
  레이블_폰트: 10pt 굵게 중앙정렬
  내용_폰트: 10pt 왼쪽정렬
```

### 3.2 안건 섹션

```yaml
안건_블록:
  제목:
    형식: "[제 {n} 안건]  {{안건제목}}"
    폰트: 11pt 굵게
    배경: 연회색(#F0F0F0)
    여백: 위 12mm, 아래 4mm
    
  소목차:
    번호: "1. 논의내용 / 2. 결정사항 / 3. 첨부자료"
    들여쓰기: 10mm
    
  결정사항_박스:
    배경: 연파란(#EBF3FB)
    테두리: 왼쪽 3pt 파란(#2E5090)
    내부여백: 3mm
    폰트: 10pt 굵게
```

### 3.3 향후 조치 표

```yaml
향후조치_표:
  헤더: ["번호", "조치내용", "담당자/부서", "완료기한", "비고"]
  열너비: [5%, 40%, 20%, 15%, 20%]
  빈행_수: 5  # 기본 5행 제공
  스타일: base.md 기본 표 스타일 적용
```

### 3.4 서명란

```yaml
서명란:
  레이아웃: 수평 배치 (작성 / 검토 / 승인)
  각_칸:
    너비: 40mm
    높이: 20mm
    레이블: "작성 / 검토 / 승인"
    폰트: 9pt 중앙
  직인: 빈 공간 (수작업 날인)
```

## 4. 변수 플레이스홀더

| 변수 | 설명 | 필수 |
|------|------|------|
| `{{문서번호}}` | 회의록 일련번호 | - |
| `{{회의명}}` | 회의 이름 | ✓ |
| `{{일시}}` | 회의 날짜·시간 | ✓ |
| `{{장소}}` | 회의 장소 (오프라인/온라인) | ✓ |
| `{{주관부서}}` | 회의 주관 부서·팀 | ✓ |
| `{{참석자목록}}` | 참석자 이름·직책 목록 | ✓ |
| `{{불참자목록}}` | 불참자 목록 | - |
| `{{안건{n}제목}}` | n번 안건 제목 | ✓ |
| `{{안건{n}논의}}` | n번 안건 논의 내용 | ✓ |
| `{{안건{n}결정}}` | n번 안건 결정 사항 | ✓ |
| `{{작성자}}` | 회의록 작성자 | ✓ |
| `{{작성일}}` | 작성 날짜 | ✓ |
| `{{차기일시}}` | 다음 회의 일시 | - |
