# HWPX 보고서 양식 (report)

> gil-office v1.0.0 | 기업·기관 보고서 스펙

## 개요

표지, 목차, 개요, 본문, 결론 구조를 갖춘 표준 보고서 HWPX 템플릿입니다.
분량이 많은 정식 보고서, 기획서, 분석 보고서에 사용합니다.

---

## 1. 페이지 설정

`base.md` 설정 상속:

```yaml
여백:
  위: 30mm
  아래: 30mm
  왼쪽: 35mm
  오른쪽: 30mm

페이지_번호:
  형식: "- {n} -"
  위치: 하단 중앙
  시작_페이지: 표지 제외 (로마자 i부터 시작 후 본문부터 아라비아 숫자)
```

---

## 2. 보고서 구조

```
섹션 0: 표지 (cover)           → 별도 section0.xml
섹션 1: 목차 (toc)             → 자동 생성
섹션 2: 개요/요약 (executive_summary)
섹션 3: 본문 (body)            → 반복 가능
섹션 4: 결론 및 제언 (conclusion)
섹션 5: 참고문헌 (references)   → 선택
섹션 6: 붙임/부록 (appendix)   → 선택
```

---

## 3. 표지 (섹션 0)

```yaml
표지_레이아웃:
  상단_여백: 60mm
  
  분류_표시:
    위치: 우측 상단
    형식: "[{{보안등급}}]"  # 일반/대외비/비밀
    폰트: 10pt 빨강
    
  보고서_종류:
    위치: 중앙
    형식: "{{보고서종류}}"  # 예: 기획 보고서 / 현황 분석
    폰트: 14pt 굵게 회색
    여백_아래: 20mm
    
  제목:
    위치: 중앙
    폰트: 24pt 굵게 진남색(#1F3864)
    형식: "{{보고서제목}}"
    줄간격: 130%
    
  부제목:
    위치: 중앙
    폰트: 16pt 남색
    형식: "{{부제목}}"
    여백_위: 8mm
    
  구분선: 실선 2pt 진남색(#1F3864) 너비 80%
    
  작성정보:
    위치: 하단 중앙 (하단에서 80mm)
    형식: |
      {{기관명 또는 부서명}}
      {{작성자}}
      {{작성일}}
    폰트: 11pt
    줄간격: 180%
```

---

## 4. 목차 (섹션 1)

```yaml
목차:
  제목: "목 차"
  폰트: 16pt 굵게 중앙 정렬
  여백_아래: 20mm
  
  항목_스타일:
    1단계: 폰트 11pt 굵게, 탭 리더: 점선(……), 우측 페이지 번호
    2단계: 폰트 10pt 들여쓰기 10mm
    3단계: 폰트 10pt 들여쓰기 20mm
    
  자동생성: true  # 제목 스타일(H1/H2/H3) 기반 자동 구성
```

---

## 5. 개요/요약 (섹션 2)

```yaml
개요_섹션:
  제목: "요약 (Executive Summary)"
  스타일: style_h1
  
  권장_항목:
    - "보고서 목적"
    - "주요 발견사항 (3~5개)"
    - "핵심 제언"
    - "기대 효과"
    
  박스_스타일:
    테두리: 실선 1pt 진남색
    배경: 연파란색(#EEF4FB)
    내부_여백: 5mm
    폰트: 10pt
```

---

## 6. 본문 (섹션 3)

```yaml
본문_구조:
  장_번호체계:
    H1: "제{n}장 {{장제목}}"
    H2: "제{n}절 {{절제목}}"
    H3: "{n}.{m} {{소제목}}"
    
  단락:
    줄간격: 160%
    첫줄들여쓰기: 10mm
    
  그림_스타일:
    캡션: "[그림 {n}] {{캡션텍스트}}"
    위치: 그림 아래 중앙
    폰트: 9pt
    
  표_스타일:
    캡션: "[표 {n}] {{캡션텍스트}}"
    위치: 표 위 왼쪽
    폰트: 9pt 굵게
    
  각주:
    폰트: 8pt
    구분선: 실선 0.4pt 너비 30%
    번호체계: 1, 2, 3 (전체 문서)
```

---

## 7. 결론 (섹션 4)

```yaml
결론_섹션:
  필수_항목:
    - "요약 및 결론"
    - "제언 및 권고사항"
    - "향후 계획 (선택)"
    
  제언_박스:
    배경: 연노란색(#FFFDE7)
    테두리: 실선 1pt 주황(#F57C00)
    아이콘: "💡" (선택)
```

## 8. 변수 플레이스홀더

| 변수 | 설명 | 필수 |
|------|------|------|
| `{{보고서제목}}` | 보고서 전체 제목 | ✓ |
| `{{부제목}}` | 부제목 또는 기간 | - |
| `{{보고서종류}}` | 기획/현황/분석/결과 보고서 | - |
| `{{보안등급}}` | 일반/대외비/비밀 | ✓ |
| `{{기관명}}` | 발행 기관·기업명 | ✓ |
| `{{부서명}}` | 담당 부서 | ✓ |
| `{{작성자}}` | 작성자 이름 | ✓ |
| `{{작성일}}` | 작성 날짜 | ✓ |
| `{{검토자}}` | 검토자 이름 | - |
| `{{승인자}}` | 결재자 이름 | - |
| `{{버전}}` | 문서 버전 (예: v1.0) | - |
