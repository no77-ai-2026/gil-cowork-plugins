# HWPX 제안서 양식 (proposal)

> gil-office v1.0.0 | 비즈니스 제안서 스펙

## 개요

표지, 요약, 배경 및 현황, 제안 내용, 일정, 예산 구조를 갖춘 비즈니스 제안서 HWPX 템플릿입니다.
영업 제안, 프로젝트 제안, 서비스 도입 제안에 활용합니다.

---

## 1. 페이지 설정

`base.md` 상속:

```yaml
여백:
  위: 30mm
  아래: 30mm
  왼쪽: 35mm
  오른쪽: 30mm

컬러_테마:
  주색: "#1F3864"   # 진남색
  보조색: "#2E5090"  # 파란색
  강조색: "#E67E22"  # 주황
  배경색: "#F0F4F8"  # 연회색
```

---

## 2. 제안서 구조

```
섹션 0: 표지 (Cover Page)
섹션 1: 목차 (Table of Contents)
섹션 2: 제안 요약 (Executive Summary)
섹션 3: 배경 및 현황 (Background / As-Is)
섹션 4: 제안 내용 (Proposal / To-Be)
  4.1 제안 범위
  4.2 핵심 솔루션 / 서비스 소개
  4.3 차별화 포인트
섹션 5: 추진 일정 (Project Schedule)
섹션 6: 예산 계획 (Budget)
섹션 7: 수행 역량 / 레퍼런스 (선택)
섹션 8: 결론 및 기대효과
```

---

## 3. 표지 (섹션 0)

```yaml
표지:
  배경: 진남색 풀페이지 또는 상단 30% 컬러 밴드
  
  로고_영역:
    위치: 좌측 상단
    크기: 높이 20mm
    
  수신처:
    위치: 우측 상단
    형식: "수신: {{수신기관명}}"
    폰트: 12pt 흰색(배경 컬러 시)
    
  제목_블록:
    여백_위: 60mm
    제안유형: "{{제안유형}}"  # 예: 기술 제안서
    폰트: 13pt 회색
    여백_아래: 10mm
    
    메인제목: "{{제안서제목}}"
    폰트: 26pt 굵게 흰색
    줄간격: 120%
    
    부제목: "{{부제목}}"
    폰트: 15pt 연회색
    
  하단_정보:
    위치: 하단 30mm
    형식: |
      {{제안사명}}
      제안일: {{제안일}}
      유효기간: {{유효기간}}
    폰트: 11pt 흰색
    줄간격: 170%
    
  기밀표시:
    위치: 최하단 중앙
    형식: "CONFIDENTIAL — 본 제안서의 내용은 수신처 외 무단 공개를 금합니다"
    폰트: 8pt 연회색
```

---

## 4. 제안 요약 (섹션 2)

```yaml
요약_박스:
  테두리: 실선 1pt 주색
  배경: 배경색(#F0F4F8)
  내부여백: 5mm
  
  항목:
    - label: "제안 목적"
      변수: "{{제안목적}}"
    - label: "핵심 제안"
      변수: "{{핵심제안}}"
    - label: "기대 효과"
      변수: "{{기대효과}}"
    - label: "총 예산"
      변수: "{{총예산}}"
    - label: "추진 기간"
      변수: "{{추진기간}}"
```

---

## 5. 추진 일정 (섹션 5)

```yaml
일정_표:
  형식: 간트차트 또는 단계별 표
  
  표_구조:
    헤더: ["단계", "주요 활동", "담당", "기간", "산출물"]
    행수: 6  # 기본 6단계
    
  간트_차트:
    기간단위: 주(Week) 또는 월(Month)
    최대기간: 12주
    색상: 
      진행중: 주색(#1F3864)
      완료: 녹색(#27AE60)
      지연: 빨강(#E74C3C)
      
  마일스톤:
    기호: "▼"
    색상: 주황(#E67E22)
    폰트: 9pt 굵게
```

## 6. 예산 계획 (섹션 6)

```yaml
예산_표:
  헤더: ["구분", "항목", "단가(₩)", "수량", "금액(₩)", "비고"]
  
  소계행:
    배경: 연회색
    폰트: 굵게
    
  합계행:
    배경: 주색(#1F3864)
    폰트: 흰색 굵게
    
  부가세:
    형식: "공급가액: ₩{{공급가액}} / 부가세(10%): ₩{{부가세}} / 합계: ₩{{최종금액}}"
    정렬: 우측
    
  지불조건:
    형식: |
      계약금: 계약 시 {{계약금비율}}%
      중도금: {{중도금일정}}
      잔금: 완료 후 {{잔금비율}}%
```

## 7. 변수 플레이스홀더

| 변수 | 설명 | 필수 |
|------|------|------|
| `{{제안서제목}}` | 제안서 메인 제목 | ✓ |
| `{{부제목}}` | 부제목 | - |
| `{{제안유형}}` | 기술/영업/서비스 제안서 | - |
| `{{수신기관명}}` | 제안 대상 기관·기업 | ✓ |
| `{{제안사명}}` | 제안하는 기업·기관 | ✓ |
| `{{제안일}}` | 제안서 제출 날짜 | ✓ |
| `{{유효기간}}` | 제안 유효 기간 | - |
| `{{제안목적}}` | 제안 목적 (1-2문장) | ✓ |
| `{{핵심제안}}` | 핵심 제안 요약 | ✓ |
| `{{기대효과}}` | 도입 시 기대 효과 | ✓ |
| `{{총예산}}` | 총 예산 (원화) | ✓ |
| `{{추진기간}}` | 프로젝트 기간 | ✓ |
| `{{공급가액}}` | VAT 제외 금액 | ✓ |
| `{{부가세}}` | 부가가치세 10% | ✓ |
| `{{최종금액}}` | 부가세 포함 최종 금액 | ✓ |
| `{{담당자명}}` | 제안사 담당자 | ✓ |
| `{{연락처}}` | 담당자 연락처 | ✓ |

## 8. 연동 스크립트

```bash
python scripts/hwpx/create_hwpx.py \
  --template templates/hwpx/proposal.md \
  --values proposal_values.json \
  --output 제안서_모아이테크_20260409.hwpx \
  --toc \
  --color-theme navy

# 빠른 견적 제안서 생성
python scripts/hwpx/create_hwpx.py \
  --template templates/hwpx/proposal.md \
  --preset quote \  # 간소화 버전
  --output 견적제안서.hwpx
```
