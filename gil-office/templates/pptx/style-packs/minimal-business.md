# Minimal Business PPTX 스타일 팩

## 개요

군더더기 없는 미니멀 비즈니스 프레젠테이션 스타일 팩. 임원 보고, 투자자 프레젠테이션, B2B 제안 발표에 최적화.

## 레이아웃 패턴 (7종)

### 1. Title Slide (표지)
```
[배경: 진한 네이비 전체 채움]
[하단 1/3: 블루 강조 수평 바]
제목 텍스트: 화이트, 중앙 정렬
부제목: 화이트 60% 투명도
날짜/회사명: 좌하단 소형 텍스트
```

### 2. Section Divider (섹션 구분)
```
[배경: 블루 강조색 전체]
섹션 번호: 대형, 반투명 화이트
섹션 제목: 화이트, Bold
설명 한 줄: 화이트 70% 투명도
```

### 3. Content + Image (콘텐츠 + 이미지)
```
[좌측 55%: 텍스트 영역]
제목: 네이비
불릿 포인트: 3-4개 최대
[우측 45%: 이미지/차트 영역]
이미지: 라운드 코너 8px
```

### 4. Two Column (2단 콘텐츠)
```
[좌측 50% | 우측 50%]
각 컬럼: 제목 + 내용
구분선: 블루 수직선 1px
```

### 5. Stats Grid (지표 그리드)
```
[2×2 또는 3×1 그리드]
각 셀: 수치(대형 Bold) + 레이블(소형)
배경: 네이비 카드
강조 수치: 블루 또는 그린
```

### 6. Quote (인용/강조)
```
[배경: 연한 그레이 또는 네이비]
인용 부호: 블루, 대형
인용 텍스트: 중형, 이탤릭
출처: 소형, 우측 정렬
```

### 7. CTA (Call to Action)
```
[배경: 블루 강조색]
메인 메시지: 화이트, 대형 Bold
서브 텍스트: 화이트 80%
버튼/액션 아이템: 흰색 아웃라인 박스
```

## 디자인 규칙

### 텍스트 제한
- [ ] 슬라이드당 최대 6줄 텍스트
- [ ] 불릿 포인트당 최대 15단어
- [ ] 폰트 종류: 최대 2개 (Pretendard + Inter)
- [ ] 폰트 크기: 최소 18pt (본문 기준)

### 장식 제한
- [ ] 그라디언트 금지 (단색 또는 플랫 컬러만)
- [ ] 외곽선 그림자 최소화
- [ ] 클립아트 사용 금지
- [ ] 워드아트 사용 금지
- [ ] 애니메이션: 페이드 인만 허용 (0.3초 이하)

### 이미지 가이드
- [ ] 고해상도 (최소 1200px 이상)
- [ ] 비율 강제 변형 금지
- [ ] 라운드 코너 8px 일관 적용
- [ ] 이미지 밝기: 텍스트와 대비 확보

## 슬라이드 구성 권장 순서

1. Title Slide (1장)
2. Agenda / 목차 (1장)
3. Section Divider → Content Slides (섹션당 3-5장)
4. Stats Grid (핵심 지표, 1-2장)
5. CTA (마지막 1장)

> 전체 슬라이드 수: 10-20장 권장
