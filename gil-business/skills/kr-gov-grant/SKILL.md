# 대한민국 정부지원사업 통합 분석·신청 스킬

누구든 정부 지원을 받을 수 있도록 돕는 통합 도우미입니다. 창업자부터 소상공인, 연구자, 비영리단체까지 — 신청자 유형과 상황에 맞춰 가장 적합한 지원사업을 찾고, 고득점 신청서를 함께 만들어드립니다.

> 이 스킬은 범용 지원사업 전반을 다룹니다. **연구비 특화(NRF·IITP·KIAT 등 학술/R&D 과제)** 는 `gil-research:grant-writer` 를 사용하세요.

---

## 시작: 신청자 유형 파악

요청이 들어오면 먼저 신청자가 어떤 상황인지 **핵심 2가지만** 파악합니다. 이미 대화에서 알 수 있으면 묻지 않습니다.

**① 신청자 유형** (해당하는 것 하나)
- 예비창업자 (아직 사업자 없음)
- 초기창업자 (창업 3년 이내)
- 성장기 스타트업 (창업 3-7년)
- 소상공인·자영업자
- 중소·중견기업
- 연구자·연구기관
- 비영리단체·협동조합·사회적기업
- 개인 (프리랜서·청년·농업인 등)

**② 원하는 지원 목적** (해당하는 것 하나 이상)
- 창업 자금 / 사업화 / R&D 기술개발
- 시설·장비 / 운영자금
- 마케팅·수출 / 해외진출
- 인력 채용·교육 / 컨설팅
- 공간·플랫폼 / 콘텐츠 제작

> 추가 정보(업종, 지역, 보유 역량)가 있으면 더 정밀한 매칭이 가능하지만, 없어도 기본 추천은 제공합니다.

## MODE 1 — 지원사업 탐색·추천

**트리거:** "맞는 지원사업 찾아줘", "어떤 거 받을 수 있어", "지원금 뭐 있어" 등

### 탐색 절차

1. 신청자 유형·목적 파악 (위 항목 참고)
2. 아래 데이터베이스에서 후보 3-5개 선별
3. 변동 가능성이 높은 정보(마감일, 지원 규모, 선발 인원)는 웹 검색으로 최신 공고 확인
   - 검색: `"[사업명] 2026 공고"` / `site:k-startup.go.kr` / `site:bizinfo.go.kr`
4. 아래 출력 템플릿으로 결과 제공

### 출력 템플릿

```
## [신청자 상황] 맞춤 지원사업 추천

### 1순위: [사업명]
- 주관기관:
- 지원 규모:
- 지원 대상:
- 신청 가능 여부: 가능 / 조건 확인 필요 / 해당 없음
- 추천 이유: (신청자 상황과의 매칭 근거 2~3줄)
- 핵심 평가 지표:
- 경쟁 난이도: 낮음 / 보통 / 높음
- 공고 시기: (정확한 일정은 공식 사이트 확인 필수)
- 공식 정보: [URL]

[2순위, 3순위 동일 형식]

---
### 현재 조건으로 신청 불가한 사업 및 이유
### 다음 단계
1. 공고문 원문 확인: K-Startup(k-startup.go.kr) 또는 각 기관 사이트
2. 신청서 작성 필요 시: "○○ 신청서 써줘"라고 말씀해 주세요
```

### 신청자 유형별 핵심 지원사업

상세 목록 → `references/programs.md` 참고

**예비창업자·초기창업자**
예비창업패키지, 초기창업패키지, 창업도약패키지, 청년창업사관학교, 로컬크리에이터 활성화

**소상공인·자영업자**
소상공인 성장지원센터, 전통시장·상점가 활성화, 소상공인 디지털화, 소진공 정책자금

**중소·중견기업**
중소기업 기술개발(R&D), 수출바우처, 스마트공장, 청년일자리도약장려금, 중진공 정책자금

**연구자·연구기관**
NRF 기초·응용연구, IITP 정보통신방송기술, KIAT 산업기술혁신, 농진청 농식품 R&D
(학술/R&D 중심은 `gil-research:grant-writer` 병용 권장)

**비영리·사회적경제**
사회적기업 육성, 협동조합 활성화, 마을기업, 자활기업 지원

**문화·관광·콘텐츠**
관광벤처사업 공모전, 로컬콘텐츠 지원, 문화예술 지원, 콘텐츠 펀드

**농업·식품**
농식품 창업지원, 6차산업화, 농촌관광, 스마트팜 지원

## MODE 2 — 신청서·사업계획서 초안 작성

**트리거:** "신청서 써줘", "사업계획서 초안 만들어줘", "지원서 작성 도와줘" 등

### 필수 파악 정보

없는 항목은 `[추후 기입]`으로 표시하고 보강 포인트에서 안내합니다.

```
1. 어떤 사업에 신청하는가?
2. 사업 아이템 한 줄 소개 (무엇을 / 누구에게 / 어떻게)
3. 핵심 차별점 (왜 우리만 할 수 있는가)
4. 신청자·팀 역량 (대표 경력, 핵심 팀원)
5. 보유 성과 (매출, 특허, 수상, 투자, 고객 등)
```

### 작성 원칙

심사위원이 높은 점수를 줄 수 있는 서술을 만드는 것이 목표입니다.

- **수치로 말하기**: "큰 시장" → "국내 시장 규모 2조원, 연 15% 성장" 처럼 구체적 수치
- **차별성 선공**: 첫 문단에 경쟁 우위를 명확히
- **평가 항목 균형**: 사업성·기술성·팀역량·실현가능성이 골고루 커버되도록
- **약점 선제 대응**: 특허 없음·매출 없음 등 약점이 있으면 보완 전략 먼저 제시
- **사업별 언어 사용**: 창업 지원은 "혁신·시장 검증", R&D는 "TRL 단계·정량 목표", 전통시장은 "지역 상생·활성화"

### 항목 출력 형식

```
## [항목 번호]. [항목명]

### 작성 가이드
이 항목에서 심사위원이 보고 싶은 것: ...
주의할 점: ...

### 초안
[실제 신청서에 들어갈 내용]

### 보강 포인트
- 추가하면 점수 올라가는 내용: ...
- 현재 약점 및 대응 방안: ...
```

### 사업별 핵심 항목 구조

**예비창업패키지 / 초기창업패키지**
1. 창업 아이템 개요 (문제→솔루션→목표고객)
2. 차별성·혁신성
3. 시장 분석 (TAM/SAM/SOM)
4. 사업화 계획 (로드맵·수익모델·고객확보)
5. 대표자 및 팀 역량
6. 지원금 사용 계획

**TIPS**
1. 기술 혁신성 (원천기술·특허·논문)
2. 팀 역량 + 팁스운영사 연계
3. 글로벌 시장 진출 가능성
4. R&D 목표 및 개발 계획 (TRL 명시)
5. Exit 전략 (IPO/M&A)

**전통시장·소상공인 활성화**
1. 현황 및 문제점 (지역 상권 데이터 활용)
2. 사업 추진 방향 (상생·활성화 키워드)
3. 세부 추진 계획 및 일정
4. 기대 효과 (방문자 수·매출 증대·일자리 창출 수치)
5. 지속 가능성 방안

**중소기업 기술개발(R&D)**
1. 개발 목표 및 최종 결과물 규격 (정량 목표 필수)
2. 국내외 기술 수준 및 개발 필요성
3. 수행기관 역량·협력기관
4. 연차별 세부 개발 계획 (간트차트 권장)
5. 기술 사업화 방안 (매출 예측)
6. 연구개발비 산정 근거

**관광벤처·문화콘텐츠**
1. 사업 아이디어 및 관광·문화적 가치
2. 차별화된 체험·콘텐츠 요소
3. 타깃 고객 (내국인/외국인/MZ세대 등)
4. 운영 계획 및 수익 모델
5. 지역 연계 및 파급 효과

### 문서 파일 출력

신청서 초안 작성 후 파일로 원하면 아래 스킬을 체인합니다.
우선순위: Anthropic 공식 스킬 > `gil-office:*`.

- **Word(.docx)**: `anthropic-skills:docx` 우선, 없으면 `gil-office:docx-generator`
- **한글(.hwpx)**: `gil-office:hwpx-writer` (Anthropic 공식 미제공)
- **Excel(.xlsx)** (일정표·예산표): `anthropic-skills:xlsx` 우선, 없으면 `gil-office:xlsx-creator`

## MODE 3 — 서류 검토·피드백

**트리거:** "신청서 검토해줘", "사업계획서 봐줘", "심사 기준으로 보면 어때" 등

사용자가 신청서 내용을 붙여넣거나 파일로 제공하면 아래 기준으로 분석합니다.

### 심사 기준별 체크포인트

| 평가 항목 | 배점(일반) | 핵심 체크 포인트 |
|----------|----------|----------------|
| 사업성 | 30-40% | 시장 규모 수치, 수익 모델 구체성, 경쟁사 대비 차별점 |
| 기술성 | 20-30% | 기술 독창성, 구현 가능성 근거, 특허·논문 연계 |
| 팀 역량 | 20-25% | 관련 경력, 역할 분담 명확성, 외부 자문 |
| 실현가능성 | 15-20% | 일정 현실성, 예산 집행 계획, 리스크 대응 |
| 전달력 | 가산 | 핵심 메시지 위치, 수치·근거 활용, 오탈자·비문 |

### 출력 템플릿

```
## 종합 평가
전체 완성도: (5점 만점)

### 강점
- [구체적 강점]

### 반드시 보완 (TOP 3)
1. [항목] — 현재: "..." → 개선안: "..."
2. [항목] — 현재: "..." → 개선안: "..."
3. [항목] — 현재: "..." → 개선안: "..."

---
## 항목별 상세 피드백
### 사업성 / 기술성 / 팀 역량 / 실현가능성 / 전달력
[각 항목별 세부 의견 + 구체적 수정 방향]
```

## 마지막 단계 — AI 슬롭 검수

사용자에게 전달되는 **사업계획서·신청서·검토 피드백 본문**은 마지막에 AI 슬롭 검수를 거칩니다.
우선순위: `anthropic-skills:ai-slop-reviewer` > `moai:ai-slop-reviewer`.

- 대상: 사업계획서 본문, 신청서 초안, 검토 피드백 내러티브
- 제외: 공통 서류 체크리스트, 일정표, 평가 기준 표, 공고 링크 목록
- 출력: 진단 요약 → 수정 텍스트 → 주요 변경사항 3블록

## 웹 검색 활용 원칙

마감일·지원 규모·선발 인원 등 변동 정보는 **항상 최신 공고를 웹 검색으로 확인**합니다.

주요 공고 확인 사이트:
- 창업 지원: https://www.k-startup.go.kr
- 중소기업 지원: https://www.bizinfo.go.kr
- R&D 과제: https://www.iris.go.kr / https://www.nrf.re.kr
- 전통시장: https://www.semas.or.kr
- 문화·관광: https://www.mcst.go.kr / https://www.kto.or.kr
- 농식품: https://www.mafra.go.kr

출처 URL을 반드시 명시하고 "공고문 원문 확인을 권장합니다"라고 안내합니다.
