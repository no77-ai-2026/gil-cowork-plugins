# 우즈베키스탄 사업자 검증 + 상권 분석 가이드

> gil-business v0.1.0 | UZ 상권 분석 (소상공인365 PDF 부재 환경)

## 개요

UZ에는 한국의 소상공인365처럼 통합된 상권분석 PDF 시스템이 없습니다. 대신 stat.uz, soliq.uz, data.gov.uz, 민간 데이터(2GIS, OLX 등)를 조합해 상권을 분석합니다. 이 가이드는 그 분석 절차를 정리합니다.

---

## 1. 사업자 검증 (Due Diligence)

### 1-1. 사업자 등록 확인

**soliq.uz (국세청)**
- INN/STIR (납세자 식별번호) 조회
- 사업자명, 등록일, 활동 분류, 세무 상태
- URL: https://soliq.uz/services/info/inn-search

**stat.uz (통계청 사업체 등록부)**
- OKED 코드 (산업 분류)
- 종업원 수 구간
- 본점 주소

**my.gov.uz (전자정부)**
- 법인 등기 확인
- 정관, 주주, 임원 정보 (제한적)

### 1-2. 검증 체크리스트

거래·계약 전 사업자 검증:

```
[ ] INN/STIR 유효성 (soliq.uz 검색)
[ ] 등록 주소 일치 (실제 영업 주소와 일치 여부)
[ ] 활동 분류 일치 (광고된 사업과 등록 활동 코드 일치)
[ ] 세무 신고 정상 (체납·세무 분쟁 여부)
[ ] 임원 변동 (최근 6개월 변경 여부 — 사기 신호)
[ ] 자본금 입금 (신규 회사 검증)
[ ] 라이선스 (해당 산업 — 식품·의약·교육·금융 등)
```

### 1-3. 한국 거래 상대방 검증과의 차이점

| 한국 (사업자등록번호 검증) | UZ (INN 검증) |
|--------------------------|---------------|
| 국세청 홈택스 즉시 조회 | soliq.uz 또는 my.gov.uz 조회 |
| 사업자 정보 통합 | 정보 파편화 (여러 사이트 종합 필요) |
| 법인등기 인터넷 발급 | 일부 정보만 온라인, 상세는 방문 필요 |

## 2. 상권 분석 — UZ 방식

### 2-1. 분석 대상 상권 정의

UZ 상권 단위는 한국과 다릅니다:
- **마할라 (Mahalla)**: 우즈벡 고유 행정/공동체 단위 (약 9,000개 전국)
- **투만 (Tuman)**: 군 단위
- **시 (Shahar)**: 시 단위
- **메트로 인근**: Tashkent의 경우 지하철역 인근 상권 (Yunusabad, Chilonzor, Sergeli 등)
- **시장 (Bozor)**: Chorsu, Alay, Yangiabad 등 전통시장

상권 분석 시 위 단위 중 적합한 단위 선택. Tashkent라면 메트로역 단위가 가장 실용적.

### 2-2. 데이터 수집 절차

**Step 1: 인구·소득 (stat.uz)**
- 마할라/투만 단위 인구 (가능한 가장 작은 단위)
- 주거인구·연령 구성·성별 비율
- 1인당 평균 소득 (지역별 평균)

**Step 2: 사업체 분포 (data.gov.uz + 현장)**
- 동종 업종 등록 사업체 수
- 공식 통계 + 2GIS Tashkent 지도 검색 (현장 정보 보완)

**Step 3: 임대료 시세 (OLX.uz, Uybor.uz)**
- 상업 부동산 임대 매물 검색 (구역별)
- 평당/㎡당 월 임대료 시세 (USD 또는 UZS)
- 권리금·보증금 관행 (UZ는 한국과 다름 — 보증금 1~3개월, 권리금 일부 산업만)

**Step 4: 유동인구 (현장 조사 + 추정)**
- UZ는 공식 유동인구 통계 없음
- 대안:
  - 메트로역 일평균 승하차 (Tashkent Metro 발표 자료)
  - 주변 거주인구·직장인구 추정
  - 현장 직접 카운팅 (평일·주말 시간대별, 30분~1시간)

**Step 5: 경쟁 동종 업소 매출 (추정)**
- UZ는 매출 통계 부재
- 대안:
  - 동종 업소 객단가 × 일평균 손님 수 추정 (2GIS 별점·리뷰로 인기도 판단)
  - HeadHunter.uz 같은 산업 채용 공고로 인건비 역산 → 매출 규모 추정
  - 현지 컨설턴트 (KOTRA Tashkent에 의뢰)

### 2-3. 한국 vs UZ 분석 비교표

| 항목 | 한국 (소상공인365 기준) | UZ |
|------|----------------------|-----|
| 유동인구 | 통신사 기반 정확 | 메트로 데이터 + 현장 카운팅 |
| 매출 통계 | 카드 매출 정확 | 부재 (추정 기반) |
| 임대료 | 부동산 플랫폼 정확 | OLX/Uybor 시세 (변동 큼) |
| 동종 업소 수 | 사업자등록 정확 | soliq + 2GIS 종합 |
| 폐업률 | 통계청 정확 | 부재 (추정 또는 KOTRA 보고서) |

UZ 분석은 **공식 통계 50% + 현장 조사 30% + 컨설턴트/현지 정보 20%** 조합 권장.

## 4. UZ 창업 타당성 평가 (한국 4축 평가 적용)

기본 4축 (성장성·경쟁·수요·재무)을 UZ 컨텍스트에 맞춰 조정:

### 축 1: 상권 성장성 (30점)
- 인구 증가 vs 정체 (stat.uz 5년 추세)
- 신규 아파트 단지 입주 여부
- 메트로 신설/연장 계획
- 개발 계획 (마스터플랜)

### 축 2: 경쟁 적합도 (25점)
- 2GIS + soliq.uz로 동종 업소 수
- 평방미터당 동종 업소 수 (포화도)
- 신규 진입 사례 (최근 1년)

### 축 3: 수요 매칭도 (25점)
- 인구 구성 (연령·성별·소득)
- 외국인 거주 비율 (해당 업종 시)
- 시간대별 유동패턴 (메트로 데이터)

### 축 4: 재무 타당성 (20점)
- 임대료 수준 (USD 환산)
- 인건비 (HeadHunter.uz 시세)
- UZS 환차 리스크 (USD 매출 vs UZS 비용 비율)
- 회수 기간 (보수적 5년 시나리오)

총점 기준은 한국과 동일:
- 80점 이상: 창업 적합
- 60~79점: 조건부 적합
- 60점 미만: 재검토 필요

## 5. 보고서 추가 섹션 (UZ 전용)

UZ 상권 분석 보고서에는 한국 9섹션 외 추가:

### 10. 외국인 사업 가능성
- 외국인 100% 진출 가능 여부 (산업별)
- 라이선스 요건 (식품·교육·의료 등)
- 노동허가서 발급 가능성

### 11. 환율·인플레이션 리스크
- 임대료 USD 고정 가능 여부
- 인건비 UZS 변동
- 원재료 수입 비중 → 환차 영향

### 12. 한국 진출 사례 (해당 시)
- 인근 한국 사업체 (2GIS 검색)
- 한인회 정보
- KOTRA 진출 컨설팅 받은 사업 사례

## 6. UZ 상권 분석 체크리스트 요약

```
PRE-ANALYSIS
[ ] 분석 단위 결정 (마할라/투만/시/메트로역)
[ ] 분석 기간 정의 (최근 1~3년 데이터)

DATA COLLECTION
[ ] stat.uz 인구·소득 데이터
[ ] data.gov.uz 사업체 분포
[ ] soliq.uz 사업자 검증 (선택 동종 업소)
[ ] 2GIS Tashkent 동종 업소 매핑
[ ] OLX/Uybor 임대료 시세
[ ] HeadHunter.uz 인건비 시세
[ ] 현장 조사 (평일·주말 유동인구 카운팅)

ANALYSIS
[ ] 4축 평가 (성장성·경쟁·수요·재무) 점수 산정
[ ] 외국인 사업 가능성 검토
[ ] 환율·인플레이션 시나리오 (보수/기본/낙관)

REPORTING
[ ] DOCX 보고서 (한·러·우 중 선택)
[ ] 한국 분석 시 동일 9섹션 + UZ 전용 3섹션
[ ] AI 슬롭 검수 (gil:ai-slop-reviewer)
```
