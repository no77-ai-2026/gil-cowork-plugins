# 상권분석 수행 가이드

소상공인365 PDF 데이터를 바탕으로 5가지 분석을 수행하는 방법론입니다.

---

## 분석 시작 전 준비사항

1. **인터뷰 결과 확인**: 관심 업종, 창업 목적, 예산, 분석 중점 4가지
2. **PDF 데이터 추출 완료**: data-fields.md의 체크리스트 기준으로 확인
3. **분석 관점 설정**: 인터뷰 결과에 따라 분석의 초점을 맞춤

예시: 관심 업종이 "카페"라면 → 20-40대 여성 유동인구 비율에 특히 주목
예시: 목적이 "투자 판단"이라면 → 매출 성장성과 임대 수익 관점으로 분석

---

## 분석 1: 유동인구 분석

### 목적
- 상권의 기본 수요 규모 파악
- 관심 업종의 핵심 고객층이 얼마나 있는지 확인

### 수행 방법

**규모 평가**:
- 일평균 유동인구 < 1,000명: 소규모 골목 상권
- 1,000-5,000명: 준거리 상권 (동네 상권)
- 5,000-20,000명: 중형 상권
- 20,000명 이상: 대형 발달 상권

**핵심 고객층 비율 계산**:
```
핵심 고객층 비율 = (핵심 연령대 비율 × 해당 성별 비율) × 2
```

예시: 카페의 경우
- 핵심 고객층 = 20-30대 여성
- 20대 비율 28% + 30대 비율 24% = 52% (해당 연령대)
- 여성 비율 55%
- 핵심 고객층 = 52% × 55% ≈ 29%

**영업 전략 시사점 도출**:
- 매출 Top 시간대 = 핵심 영업 시간 → 인력 집중 배치 권고
- 유동인구 저조 시간대 = 휴식 시간 또는 사전 준비 시간으로 활용

## 분석 2: 매출 현황 분석

### 목적
- 창업 후 현실적 기대 매출 수준 파악
- 재무 타당성 평가의 기초 데이터 수집

### 수행 방법

**기대 매출 3단계 시나리오 계산**:
```
낙관 시나리오: 동종 업종 상권 평균 × 120%
(상위 업소 수준 — 경험자, 차별화된 콘셉트 가정)

기본 시나리오: 동종 업종 상권 평균 × 60%
(신규 오픈 업소 평균 — 가장 현실적인 추정)

보수 시나리오: 동종 업종 상권 평균 × 40%
(초기 6개월 안정화 기간 — 리스크 대비용)
```

**객단가 해석**:
- 건당 평균 매출 1만원 미만: 저단가 상권 (박리다매 전략 필요)
- 1-3만원: 일반 상권
- 3만원 이상: 고단가 상권 (프리미엄 포지셔닝 가능)

**손익분기점 간단 계산** (보수 시나리오 기준):
```
예상 고정비 = 임대료 + 인건비 + 기타(전기·통신 등)
              ≈ 월 예상 매출의 40~50% 추정
손익분기점 = 고정비 / 이익률(매출 대비 15~20% 가정)
```

## 분석 3: 경쟁 및 업종 현황 분석

### 목적
- 신규 진입 장벽과 시장 포화도 파악
- 기존 업소 생존율로 사업 난이도 파악

### 포화도 지수 계산

```
포화도 지수 = 일평균 유동인구 ÷ 동종 업종 수

해석:
> 1,000명: 수요 대비 공급 부족 → 진입 유리
700~999명: 양호
400~699명: 경쟁 보통
200~399명: 경쟁 심함
< 200명: 포화 상태
```

### 창폐업 분석

**생존율 지표**:
```
업력 5년 이상 비율이 30% 이상 = 생존력 있는 상권
업력 5년 이상 비율이 20% 미만 = 고위험 상권
```

**시장 활력 지표**:
- 창업률 > 폐업률: 시장 성장 중
- 창업률 ≈ 폐업률: 시장 정체
- 창업률 < 폐업률: 시장 쇠퇴 또는 위험 신호

**업종 다양성 해석**:
- 상위 3개 업종이 전체의 80% 이상: 단일화된 상권 (쏠림 위험)
- 상위 3개 업종이 50% 이하: 다양성 있는 상권 (틈새 기회 존재)

## 분석 4: 입지 환경 분석

### 목적
- 상권의 근본적 성격 파악 (주거형/직장형/유동형)
- 교통과 집객 시설이 업종에 유리한지 판단

### 상권 유형 분류

```
주거인구 ÷ (주거인구 + 직장인구)

> 70%: 주거형 상권
  → 저녁·주말 강세, 생필품·식당 유리
  → 카페는 주말 중심 전략

50~70%: 혼합형 상권
  → 균형 잡힌 수요, 대부분 업종 무난

< 50%: 직장형 상권
  → 평일 점심·오전 강세
  → 주말 전략 별도 필요, 배달 중요
```

### 집객 시설 해석

| 시설 | 창업에 미치는 영향 |
|------|-----------------|
| 대학교 인근 | 10-20대 고객 풍부, 가성비 중요 |
| 사무용 빌딩 | 직장인 평일 수요, 점심 특수 |
| 병원·의원 | 40-60대 비율 높음 |
| 대형마트 | 쇼핑 연계 유동인구, 주말 강세 |
| 지하철역 | 통행 유동인구, 테이크아웃 유리 |

---

## 분석 5: 배달 현황 분석

### 목적
- 홀 영업만으로 충분한지, 배달 채널 필요성 판단

### 배달 의존도 판단 기준

```
배달 매출 비율 기준:
< 20%: 홀 중심 상권 (배달 보조 수단)
20~40%: 홀+배달 균형 전략 권장
> 40%: 배달 중요 상권 (배달 초기 투자 필요)
```

### 업종별 배달 전략 시사점

| 상황 | 권고 전략 |
|------|---------|
| 배달 성장 + 해당 업종 배달 Top | 오픈과 동시에 배달앱 3사 입점 |
| 배달 성장 but 해당 업종 배달 낮음 | 배달 특화 메뉴 개발 후 진출 |
| 배달 정체·감소 | 홀 경험 중심 전략, 배달은 보조 |
| 배달 데이터 없음 | 인근 동종 업소 배달앱 현황 직접 조사 권고 |

## 분석 데이터 불충분 시 대처 방법

### 간단분석 PDF (항목 제한)
- 유동인구·매출 기본 수치만 있는 경우
- 포화도 지수, 창폐업률 등이 없는 경우
→ 해당 분석은 "간단분석 기준, 제한된 데이터" 명시 후 가용 데이터로 부분 분석

### KOSIS 공공데이터 보완 필요 시
gil-data:public-data 스킬 활용:
- 행정동별 인구 통계 (KOSIS)
- 사업체 수 통계 (소상공인 부문)
- 매출 및 카드 거래 데이터

### 현장 조사 권고 기준
다음 상황에서는 보고서에 "현장 확인 필수" 권고 포함:
- 유동인구 데이터가 6개월 이상 오래된 경우
- 창폐업률이 이상치를 보이는 경우
- 입지 관련 핵심 데이터 누락된 경우
