# 창업 타당성 평가 기준 및 점수 산정

소상공인365 PDF 데이터를 기반으로 창업 타당성을 4축으로 평가하는 채점 기준입니다.

---

## 종합 평가 체계

**총점 100점** | 판정 기준:
- 80점 이상 → **창업 적합 ✅** (적극 진행 권고)
- 60-79점 → **조건부 적합 ⚠️** (전략 보완 후 진행)
- 40-59점 → **재검토 필요 ❌** (대안 상권/업종 검토 권고)
- 39점 이하 → **진입 위험 🚫** (강력 재검토 권고)

---

## 축 1: 상권 성장성 (30점)

상권이 앞으로 성장할 가능성을 평가합니다.

### 유동인구 증감 추세 (15점)
| 상황 | 점수 |
|------|------|
| 전년 대비 10% 이상 증가 | 15점 |
| 전년 대비 5-10% 증가 | 12점 |
| 전년 대비 0-5% 증가 (정체) | 8점 |
| 전년 대비 감소 | 4점 |
| 데이터 없음 | 7점 (중간값 부여) |

### 매출 성장 추세 (15점)
| 상황 | 점수 |
|------|------|
| 전년 대비 매출 10% 이상 증가 | 15점 |
| 전년 대비 매출 5-10% 증가 | 12점 |
| 정체 (±5% 이내) | 8점 |
| 전년 대비 감소 | 4점 |
| 데이터 없음 | 7점 |

> 성장지수가 수치로 제공되는 경우: 성장(15점), 유지(9점), 쇠퇴(4점)

## 축 2: 경쟁 적합도 (25점)

시장 포화도와 생존율로 신규 진입 가능성을 평가합니다.

### 경쟁 포화도 지수 (15점)

**포화도 계산**: 일평균 유동인구 ÷ 동종 업종 수

| 포화도 구간 | 의미 | 점수 |
|------------|------|------|
| 1,000명 이상/업소 | 경쟁이 적고 수요가 충분 | 15점 |
| 700-999명/업소 | 양호한 수준 | 12점 |
| 400-699명/업소 | 경쟁 보통 | 8점 |
| 200-399명/업소 | 경쟁 심함 | 5점 |
| 200명 미만/업소 | 포화 상태 | 2점 |
| 동종 업소 수 없음 | 선발자 유리 | 13점 |

### 창폐업률 건전성 (10점)

**건전성 계산**: 창업률 - 폐업률 (순증감률)

| 상황 | 점수 |
|------|------|
| 순증가 (창업 > 폐업) + 폐업률 15% 미만 | 10점 |
| 순증가 but 폐업률 15-25% | 7점 |
| 균형 (창업 ≒ 폐업) | 6점 |
| 폐업 우위 (폐업률 > 창업률) | 3점 |
| 폐업률 30% 초과 | 1점 |
| 데이터 없음 | 5점 |

해당 업종의 핵심 고객층과 실제 유동인구 구성의 일치도를 평가합니다.

## 축 4: 재무 타당성 (20점)

예상 매출 대비 투자비 회수 가능성을 평가합니다.

### 투자 회수 기간 추정

**월 예상 순이익 계산** (보수적 추정):
- 동종 업종 월 평균 매출 × 60% = 실현 가능 매출 (신규 업소 할인)
- 실현 가능 매출 × 15% = 예상 순이익 (임대료·인건비·재료비 후)

**투자 회수 기간** = 창업 예산 ÷ 예상 월 순이익

| 투자 회수 기간 | 점수 |
|--------------|------|
| 24개월 이내 (2년) | 20점 |
| 25-36개월 (2-3년) | 15점 |
| 37-60개월 (3-5년) | 10점 |
| 61개월 이상 (5년+) | 4점 |
| 매출 데이터 없음 | 10점 |

> 예상 순이익 계산 시 보고서에 반드시 가정 사항 명시:
> "해당 수치는 소상공인365 매출 데이터 기반 추정치로, 실제 운영 결과와 차이가 있을 수 있습니다."

## 종합 판정 서술 가이드

### 창업 적합 (80점+)
> "[업종] 창업에 있어 [지역] 상권은 **우호적인 환경**을 갖추고 있습니다.
> [유동인구 특성]이 주요 고객층과 잘 맞으며, [경쟁 현황]으로 시장 진입 여지가 충분합니다.
> 다만 [1-2가지 주의사항]을 사전에 대비하는 전략이 필요합니다."

### 조건부 적합 (60-79점)
> "[지역] 상권에서 [업종] 창업은 **신중한 접근이 필요**합니다.
> [긍정 요소]는 유리하지만, [부정 요소]로 인해 리스크가 존재합니다.
> [3가지 보완 전략]을 갖춘다면 성공 가능성을 높일 수 있습니다."

### 재검토 필요 (40-59점)
> "[지역] 상권의 현재 데이터는 [업종] 창업에 **도전적인 환경**을 나타냅니다.
> 특히 [주요 문제 2가지]가 핵심 위험 요인입니다.
> 창업을 진행하려면 [대안 전략] 또는 [인근 대안 지역] 검토를 권장합니다."

### 진입 위험 (39점 이하)
> "현재 데이터 기준으로 [지역] 상권에서 [업종] 창업은 **높은 리스크**를 안고 있습니다.
> [구체적 문제]로 인해 현 시점 진입은 신중히 재고하시기 바랍니다.
> 대안으로 [인근 상권] 또는 [다른 업종] 검토를 강력히 권장합니다."

## 성공 조건 3가지 도출 가이드

종합 판정 후, 분석 데이터 기반으로 **해당 상권에서 성공을 위한 필수 조건 3가지**를 도출합니다.

예시:
1. **차별화된 타깃 설정**: 유동인구 30대 여성 비율이 높으므로, [구체적 차별화 포인트]
2. **영업시간 최적화**: 매출 집중 시간대인 [시간대]에 맞춘 운영 전략
3. **배달 채널 병행**: 배달 성장률 [X%]를 감안해 홀+배달 투트랙 운영
