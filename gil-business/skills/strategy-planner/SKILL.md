# 전략 플래너 (Strategy Planner)

사업계획서, 비즈니스 모델, 전략 분석, 신시장 진출 전략을 위한 종합 전략 기획 스킬입니다. 검증된 전략 프레임워크를 활용하여 구조화된 전략 문서를 생성합니다.

## 개요

이 스킬은 다양한 비즈니스 상황에 맞는 전략 프레임워크를 선택하고 적용하여 체계적인 전략 문서를 생성합니다. 창업 아이디어 검증부터 기업의 신사업 진출, 해외 시장 확장, 조직 목표 설정까지 폭넓게 활용 가능합니다.

## 트리거 키워드

- 사업계획서, 비즈니스 모델, 전략 기획, strategy
- 린 캔버스, BMC, 비즈니스 모델 캔버스
- SWOT, Porter's 5 Forces, 블루오션, Blue Ocean
- OKR, 목표 설정, 조관 목표
- 해외 진출, 신시장 진출, 시장 진입 전략
- 시나리오 플래닝, 미래 시나리오

## 실전 SWOT·전략 프레임워크 패턴 (한국 시장)

SWOT은 단순히 4칸을 채우는 도구가 아니라 **마케팅·사업 플랜의 첫 단추**입니다. SWOT 앞부분은 팩트·환경 분석, SWOT 뒷부분은 의도·방향성을 가진 전략 — 의사결정권자 설득이 SWOT에서 꼬이면 이후가 다 어려워집니다.

### 7개 압축 패턴

**① SWOT은 진단표가 아닌 의사결정 근거**
"강점이 무엇인가"가 아니라 **"어디서 이길지 + 무엇을 의도적으로 포기했는지"** 설명하는 도구. 슬라이드가 "그래서 우리는 무엇을 하겠다는 것인가"에 답하지 못하면 실패. 강점·기회는 설명으로 끝나면 안 되고, 반드시 다음 중 하나로 이어져야 함:
- 제품 로드맵·타깃 고객 설정
- 영업 전략·가격 정책
- 파트너십 전략

**② 흔한 실수 5가지 (한국 마케팅 실무)**
| 실수 | 증상 | 처방 |
|---|---|---|
| 글자 많이 채우려 함 | "조사 많이 했다" 자랑 | 핵심만 압축, 간결이 더 어려움 |
| 같은 포인트를 4칸에 반복 | "1위 점유"→강점, "벤치마킹 없음"→약점 | 각 박스에 고유 영역 1포인트 |
| 팩트만 나열, 인사이트 없음 | "20대 초반 비중 높음" | 시사점·사업 영향까지 (충성도 낮은 층) |
| 너무 뻔하거나 너무 생소함 | 의사결정권자가 알맹이 없다 평가 | 그들 표현·관심사 반영 |
| SWOT 자체 포커스 흐림 | 비즈니스 전략 자체에 포커스 없음 | 가장 중요한 이슈 1-2개로 좁힘 |

**③ 3단계 작성 순서**
1. **자료 수집** (강·약·기·위 판단 X, 중요한지만 판단). 비즈니스 프레임워크(3C·4P·PESTEL 등)로 핵심 이슈 빠짐없이 정리.
2. **4칸 채우기** (강점 → 약점 → 기회·위기 순). 강점은 쉬움, 기회가 가장 어려움. 약점은 "주요 이슈 인지하고 있다" 메시지가 핵심.
3. **인사이트 추출·다듬기** (전체 그림 보며 전략 방향과 논리적 연결). 기회 보강이 핵심.

**④ 의사결정권자 마음 읽기 (한국 조직 문화 특화)**
- **그들이 평소 쓰는 표현 그대로 쓰기**: "OO 노후화", "OO 비효율" 같은 그들의 단어가 SWOT에 등장하면 공감대 형성 즉시
- **반복 지적되는 이슈는 SWOT 재료**: 지겨워도 회의·1on1에서 반복된 이슈는 분명한 약점·위기 신호
- **회의 질문 받아적기**: 그들이 던진 질문 = 그들도 답 모름 → 우리가 답하면 SWOT의 가치 ↑
- **이메일·메신저 표현 차용**: 그들이 자주 쓰는 비유·단어를 SWOT에 활용

**⑤ SWOT 작성 체크리스트 6개**
- [ ] 한눈에 도드라지는 박스가 있는가 / 가장 약한 박스도 명확한가
- [ ] 1장만 봐도 향후 사업의 집중 방향이 보이는가
- [ ] 각 박스가 중요도 순으로 정리되어 있는가
- [ ] 사용한 용어가 구체적·명확하며 경영진이 자주 쓰는 용어인가
- [ ] 각 박스가 사업 주요 이슈를 골고루 커버하는가
- [ ] 다음 전략과 논리적 일관성이 있는가

**⑥ "선택과 포기" 전략 언어 변환 (As-Is → To-Be)**
| As-Is (추상적 진단) | To-Be (판단 드러내는 전략) |
|---|---|
| 기술력 우수 | 공정 안정성 확보 후 단계적 확장, 글로벌 진출은 의도적 후순위 |
| 시장 성장성 높음 | 1-49인 소규모 사업체 + 밀레니얼 세대 집중 공략 |
| 경쟁사 다수 존재 | 글로벌 표준 미적용 영역(로컬 상권·KR 임대 관행)을 우선 점유 |
| 자금 부족 | 앵커 고객 1-2개 확보 후 시리즈 A 진입, 마케팅비 후순위 |

추상적 형용사("우수", "다수", "부족") 대신 **현장 숫자**(리드타임, 재고비, 환율 민감도)와 **시간순 실행 계획**으로 표현.

**⑦ SWOT 외 한국 실무 프레임워크 5종**
- **Lean Canvas**: 9블록 BMC의 스타트업 압축 버전 → MVP 정의·핵심 가정 검증
- **3C (Customer/Company/Competitor)**: 패러다임 시프트 포착용 (나열 X)
- **4P (Product/Price/Place/Promotion)**: 마케팅 믹스
- **Porter's 5 Forces**: 산업 구조 분석 + 진입장벽 역이용
- **Blue Ocean (ERRC)**: 제거/감소/증가/창조 — 무경쟁 시장 설계

**OKR 한국 적용 시 주의**: Objective 너무 추상화하지 말 것. Key Result는 반드시 **숫자 + 마감 시점**. 분기 단위 운영 권장 (반기는 너무 길어 피드백 늦음).

> 💡 한국에서 가장 자주 실패하는 전략 발표: "SWOT은 그럴듯한데 다음 전략이 SWOT 결론과 따로 노는 경우". 전략은 SWOT의 인사이트에서 직접 도출되어야 합니다.

## 워크플로우

### 1단계: 요청 유형 파악

사용자 요청에서 목표, 산출물 유형, 대상 비즈니스 단계를 파악합니다.

- **목표 산출물**: 사업계획서 / 전략 프레임워크 분석 / 비즈니스 모델 / 시나리오 플래닝
- **비즈니스 단계**: 초기 아이디어 / 성장 단계 / 신사업 진출 / 해외 진출
- **가용 데이터**: 내부 데이터, 시장 조사 자료, 경쟁사 정보

### 2단계: 프레임워크 선택 및 적용

요청 유형에 따라 최적의 전략 프레임워크를 선택하고 적용합니다.

**스타트업/창업:**
- Lean Canvas → 핵심 가정 검증 → MVP 정의
- Problem-Solution Fit → Product-Market Fit 진단

**비즈니스 모델:**
- Business Model Canvas 9블록 분석
- 가치 제양 설계 → 수익 모델 정의

**전략 분석:**
- SWOT 분석 → SO/WO/ST/WT 전략 도출
- Porter's 5 Forces → 산업 구조 분석
- Blue Ocean Strategy → 무시장 전략 수립

**목표 설정:**
- OKR (Objectives and Key Results) 설정
- KPI 정의 및 측정 기준 마련

**신시장 진출:**
- 시장 매력도 분석 (TAM/SAM/SOM)
- 진입 장벽 평가
- 파트너 전략 수립

**시나리오 플래닝:**
- 핵심 불확실성 요인 식별
- 복수 시나리오 작성
- 대응 전략 마련

### 3단계: 구조화된 문서 생성

분석 결과를 바탕으로 전문적인 전략 문서를 생성합니다.

- **Executive Summary**: 경영진을 위한 1페이지 요약
- **현황 분석**: 데이터 기반 진단
- **전략 프레임워크 적용 결과**: 선택한 프레임워크 분석
- **전략 방향성**: 우선순위별 액션 아이템
- **실행 계획**: 단계별 로드맵과 성공 지표

### 4단계: 검토 및 정제

- 사용자 피드백 수렴
- 수치와 가정 구체화
- 리스크 요소 식별 및 대응 방안 수립

## 사용 예시

### 예시 1: 스타트업 사업계획서

**입력:**
```
온라인 영어 교육 플랫폼 창업 계획
타겟: 직장인 대상 1:1 화상 영어 회화
차별화: AI 기반 발음 평가 + 필리핀 튜터
```

**출력:**
```
## 온라인 영어 교육 플랫폼 사업계획서

### 1. Lean Canvas
- Problem: 직장인의 시간 부족, 비싼 원어민 강사
- Solution: AI 발음 평가 + 저렴한 필리핀 튜터
- UVP: 1/3 가격, 24/7 예약 가능, 실시간 피드백
- Revenue: 월 구독 19만원 (주 2회, 25분)
- Cost: 튜터 비용 60%, 플랫폼 운영 20%, 마케팅 20%

### 2. 시장 분석
- TAM: 국내 어학시장 5조원
- SAM: 온라인 영어 3,000억원
- SOM: 초기 1% = 30억원 목표

### 3. 실행 계획
- Q1: 플랫폼 개발, 튜터 10명 확보
- Q2: 베타 서비스, 가입자 100명 목표
- Q3: 정식 런칭, 가입자 500명 목표
- Q4: 시리즈 A 준비
```

### 예시 2: 기업 SWOT 분석 및 전략 수립

**입력:**
```
회사: 중견 제조업체 (자동차 부품)
현황: 전기차 전환으로 기존 부품 수요 감소
목표: 신사업 방향 설정
```

**출력:**
```
## SWOT 분석 및 전략 방향

### Strengths
- 30년 제조 노하우, 품질 경쟁력
- 완성차사 파트너십
- 현금 보유 100억원

### Weaknesses
- R&D 투자 부족 (매출 1% 미만)
- 전문 인력 부족
- 브랜드 인지도 낮음

### Opportunities
- 전기차 시장 연평균 30% 성장
- 스마트 팩토리 정부 지원
- 해외 진출 기회

### Threats
- 중국 저가 공세
- 완성차사 직접 생산
- 규제 변화

### 전략 방향 (SO/ST/WO/WT)
- SO: R&D 투자 확대 → 전기차 부품 개발
- ST: 품질 경쟁력으로 중국과 차별화
- WO: 정부 지원 활용, 스마트 팩토리 도입
- WT: 비용 구조조정, 핵심 역량 집중
```

## 출력 형식

### 필수 섹션

1. **Executive Summary**: 1페이지 요약
2. **현황 분석**: 데이터 기반 진단
3. **전략 프레임워크**: 채택한 프레임워크와 적용 결과
4. **전략 방향**: 우선순위별 전략과 액션 아이템
5. **실행 계획**: 단계별 로드맵과 성공 지표

### 선택적 섹션

- **리스크 분석**: 실행 리스크와 완화 방안
- **성공 사례**: 벤치마킹 사례
- **용어 설명**: 전문 용어 정의

## 주의사항

### 필수 입력값

- **비즈니스 목표**: 달성하려는 목표 명시
- **대상 산업/시장**: 타겟 시장 정의
- **현재 상황**: 비즈니스 단계, 보유 자원, 주요 이슈

### 데이터 기반 접근

- 주장은 데이터나 근거로 뒷받침
- 명확한 데이터가 없으면 합리적 가설 설정 후 검증 방법 제시
- 벤치마킹 활용: 유사 기업/업계 평균 비교

### 흔한 실수 피하기

- **과도한 분석**: 실행 가능한 인사이트 우선
- **일반적 전략**: "마케팅 강화" 같은 모호한 제안 지양
- **맥락 무시**: 업계 특성과 기업 상황 반영
- **비현실적 목표**: 3개월에 1년 치 과제 설정 등
