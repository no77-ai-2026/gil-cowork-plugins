# 데일리 브리핑 (Daily Briefing)

매일 아침 업무 시작 전 5분 만에 비즈니스 현황을 파악할 수 있는 구조화된 브리핑을 생성합니다. 국내외 업계 뉴스, 경쟁사 동향, 주요 지표, 오늘의 액션 아이템을 하나의 리포트로 통합합니다.

## 개요

이 스킬은 사용자 업종·관심 주제에 맞게 브리핑 항목을 맞춤 설정하여 구조화된 일일/주간 브리핑을 생성합니다. 업계 주요 뉴스, 경쟁사 모니터링, 규제·정책 업데이트, 시장 지표, 오늘의 액션 아이템을 포함합니다.

## 트리거 키워드

- 일일 브리핑, 데일리 리포트, 오늘의 이슈, 오늘 뉴스
- 아침 브리핑, 비즈니스 동향, 시장 동향, 뉴스 요약
- 경쟁사 모니터링, 주간 브리핑, 위클리 리포트

## 실전 비즈니스 브리핑 패턴 (한국 시장)

5분 안에 비즈니스 현황을 파악하지 못하면 데일리 브리핑은 의미가 없습니다. **시사점 없는 정보는 빼고, 액션으로 연결되는 항목만**.

### 6개 핵심 패턴

**① 한국 기획자의 자료 조사 사이트 매트릭스**

| 카테고리 | 핵심 사이트 | 용도 |
|---|---|---|
| 국내 정책·경제 | 정책브리핑(korea.kr), KDI | 정책·경제 동향 |
| 통계 | KOSIS 국가통계포털, 공공데이터포털 | 1차 통계 자료 |
| 기업 정보 (상장) | DART 공시, 네이버 증권 | 재무·사업·공시 |
| 기업 정보 (비상장·스타트업) | The VC, 혁신의 숲, CRETOP | 투자·MAU·객단가 |
| 산업 리포트 | 한경컨센서스, MK증권, 삼정 KPMG, 삼일 PWC, LG경영연구원 | 산업 분석 |
| 빅데이터 뉴스 | 빅카인즈 (54개 언론사) | 뉴스 트렌드 검증 |
| 직원 수 | 크레딧잡 (국민연금 연동) | 회사 규모 |
| 검색 트렌드 | 네이버 데이터랩, 구글 트렌드 | 키워드별 관심도 |
| Z세대 트렌드 | 캐릿 | MZ 줄임말·핫토픽 |
| 무역·수출 | KOTRA 해외시장뉴스, 한국무역협회 K-stat | 해외 시장 |

**한국 특화 뉴스레터**: 어피티(경제) · 뉴닉(전반) · 콘텐타(마케팅) · 캐릿(Z세대) · 어거스트(미디어) · 스타트업 위클리

**② 한 줄 헤드라인 + 시사점 구조**

원본 기사를 그대로 옮기지 말 것. 표준 포맷:

```
[헤드라인]
- 출처: [매체·발행일]
- 핵심: [3줄 요약]
- 시사점: [우리 비즈니스에 미치는 영향 1~2줄]
- 액션: [오늘 할 일 / 모니터링 / 무관]
```

**시사점이 없으면 그 기사는 브리핑에 넣지 말 것**.

**③ 빅카인즈 빅데이터로 트렌드 검증**

"공기청정기" 같은 키워드로 빅카인즈 분석 → 연관어·기간별 추이·언론사별 분포로 **시장 이슈의 진짜 비중** 검증. 뉴스 한 건이 아닌 N건 패턴이 진짜 트렌드. AI 활용도 가능 — 단순 검색을 넘어 인사이트 도출.

**④ 경쟁사 모니터링 4채널 (한국 B2B 표준)**

| 채널 | 무엇을 본다 | 추론할 수 있는 것 |
|---|---|---|
| 웹사이트·상세페이지 | 텍스트·카피·이미지·기능·정책·프로모션 변화 | "지금 겪는 문제 추론" (예: 할인 연장 = 모객 부진) |
| 채용 공고 | 신사업·확장 방향 | 어떤 직군 채용 = 어디 투자 |
| 보도자료 | 공식 제휴·자금조달·신제품 | 다음 분기 로드맵 신호 |
| SNS·블로그·고객 리뷰 | 마케팅 톤·고객 반응 | NPS·VOC 우회 측정 |

**1일 1방문 루틴**: 크롬 시작 페이지를 경쟁사 사이트로 설정 → 자연스럽게 루틴화. 슬랙 봇 알림도 효과적.

**⑤ 경쟁사 고객센터·세일즈 콜 (정보전)**

잠재 고객인 척 통화·문의로 **할인 기간·신청자 수·만족도·신상품 출시 시점·리드 타임** 등을 직접 확인. 웹사이트·보도자료만으로는 파악 안 되는 운영 디테일을 가장 빠르게 채집하는 방법. 한 번의 정보전 통화로 "월 평균 신청자 수·이번 달 캠페인 부진 여부" 같은 결정적 데이터를 얻고 우리 리타게팅·캠페인을 즉시 조정할 수 있음.

**전화·문의 시 주의**:
- 회사 유선 사용 X (발신번호 노출)
- 회사 이메일 도메인 X (등록 시 정체 드러남)
- 전문용어 사용 X (담당자가 동종 업계임을 감지)
- 아는 것도 잘 모르는 척, 자연스럽게
- 무례 X — **물밑 정보전**, 경쟁사도 똑같이 우리에게 함

**⑥ 규제·정책 모니터링 의무 채널 (한국)**

| 부처/위원회 | 모니터링 대상 |
|---|---|
| 금융위원회 | 핀테크, 가상자산, 대출, 보험 |
| 공정거래위원회 | B2C·이커머스·가격 표시·과대 광고 |
| 중소벤처기업부 | 스타트업·소상공인 지원사업·R&D |
| 과학기술정보통신부 | IT·SW·AI·통신·디지털 |
| 식품의약품안전처 | 식품·화장품·의약품·헬스케어·광고 심의 |
| 개인정보보호위원회 | 모든 IT 서비스의 개인정보 처리 |
| 환경부 | ESG·자원순환·탄소중립 |
| 정책브리핑(korea.kr) | 통합 검색 |

**규제 변화는 며칠 안에 사업 모델을 바꿀 수 있어** 일일 브리핑의 필수 섹션.

### 데일리 브리핑 표준 포맷 (5분 안에 읽힘)

```markdown
# [YYYY-MM-DD] 비즈니스 데일리 브리핑

## 헤드라인 3줄
1. [핵심 임팩트 사건]
2. [경쟁·시장 변화]
3. [규제·정책 변화]

## 국내 업계 뉴스
| 제목 | 출처 | 시사점 |

## 글로벌 동향 (TechCrunch·Bloomberg·WSJ)
| 헤드라인 | 시사점 |

## 경쟁사 동향
- A사: ...
- B사: ...

## 규제·정책 업데이트
- ...

## 시장 지표
- 코스피·코스닥·환율·금리·유가

## 오늘의 액션 아이템
1. [구체적 의사결정·검토 사항]
2. ...
```

### 주간·월간 리포트 추가 섹션

- **주간**: 이번 주 핵심 변곡점 + 다음 주 모니터링 포인트
- **월간**: 시장 점유율 변화·CAC·LTV 추이·경쟁 강도 변화

### 안티 패턴

- 시사점 없는 뉴스 단순 나열
- 액션 없는 정보만 가득
- 한 매체 의존 (편향 위험)
- 영문 매체만 보고 한국 시장 놓침 (또는 반대)
- 규제·정책 섹션 누락 → 사업 모델 갑작스러운 무력화 위험

> 💡 한국 사업가의 데일리 브리핑은 **국내 정책 + 글로벌 트렌드 + 규제 모니터링 + 경쟁사 + 키워드 검증** 5요소가 모두 있어야 의미가 있습니다. 한 요소라도 빠지면 의사결정 정보가 부족합니다.

## 워크플로우

### 1단계: 브리핑 범위 설정

- 대상 업종 및 키워드 확인 (`.gil/config.json`의 `industry` 설정 우선 사용)
- 브리핑 기간 설정 (일간/주간/월간)
- 모니터링 경쟁사 목록 확인

### 2단계: 정보 수집 및 분류

다음 섹션으로 구조화하여 수집:

**국내 업계 뉴스**
- 네이버 뉴스, 조선일보 경제, 한국경제, 파이낸셜뉴스
- 업종별 주요 매체 (IT: 블로터, ZDNet; 스타트업: 벤처스퀘어)

**글로벌 동향**
- WSJ, TechCrunch, Bloomberg 주요 헤드라인 요약
- 글로벌 트렌드 및 시사점

**경쟁사 모니터링**
- 경쟁사 채용 공고, 보도자료, 제품 업데이트
- SNS 활동, 마케팅 캠페인

**규제·정책 변화**
- 금융위, 공정거래위, 중기부, 과기정통부 공지
- 정책 브리핑(korea.kr)

**시장 지표**
- 코스피/코스닥, 환율, 유가, 금리 동향

### 3단계: 브리핑 리포트 생성

아래 표준 포맷으로 마크다운 리포트 생성:

```markdown
# [날짜] 데일리 브리핁

## 헤드라인 요약 (3줄 핵심)
1. ...
2. ...
3. ...

## 업계 주요 뉴스
| 제목 | 출처 | 시사점 |

## 경쟁사 동향

## 규제/정책 업데이트

## 시장 지표

## 오늘의 액션 아이템
- [ ] ...
```

### 4단계: 사용자 검토 및 저장

- 리포트를 사용자에게 제시
- 피드백 반영 후 `.gil/briefings/YYYYMMDD.md`에 저장 (옵션)

## 사용 예시

### 예시 1: SaaS 업계 데일리 브리핑

**입력:**
```
업종: SaaS
기간: 오늘
경쟁사: 워크립, 리브
```

**출력:**
```
# 2026년 4월 30일 데일리 브리핑

## 헤드라인 요약 (3줄 핵심)
1. 네이버, HR 테크 SaaS '워크립'에 500억원 추가 투자 발표
2. 정부, 중소기업 AI 도입 지원 확대 (내년부터 30% 세제 혜택)
3. 리브(채용플랫폼), 일본 시장 진출 발표

## 업계 주요 뉴스
| 제목 | 출처 | 시사점 |
|------|------|--------|
| 네이버, 워크립 500억원 추가 투자 | 한국경제 | HR 테크 시장 과열, 경쟁 심화 예상 |
| 정부, 중소기업 AI 도입 세제 혜택 | 정책브리핑 | B2B SaaS 타겟 기업의 구매력 증대 |
| 카카오엔터프라이즈, AI 협업툴 출시 예정 | 전자신문 | 협업툴 시장 신규 진입, 경쟁 확대 |

## 경쟁사 동향
**워크립**
- 대기업 타겟 '워크립 엔터프라이즈' 베타 서비스 시작
- 채용: AI 엔지니어 10명 급증 (연봉 1억원+)

**리브**
- 일본 'Recruit'와 제휴 발표 (일본 시장 진출)
- 리브 오픈 API 개발자 컨퍼런스 예정

## 규제/정책 업데이트
- **개인정보보호법**: AI 학습 데이터 위임 동의 가이드라인 발표 (행안부)
- **플랫폼 사업법**: SaaS 플랫폼은 적용 제외 (금융위)

## 시장 지표
- 코스닥: 875.42 (+0.8%)
- 원/달러 환율: 1,352원 (+0.3%)
- 기준금리: 3.50% (동결)

## 오늘의 액션 아이템
- [ ] 워크립 엔터프라이즈 베타 신청 후 기능 분석
- [ ] 정부 AI 지원 세제 혜택 요건 확인 및 활용 방안 검토
- [ ] 리브 일본 진출 관련 우리 서비스의 동남아 진입 가능성 검토
```

### 예시 2: 주간 핀테크 시장 동향

**입력:**
```
업종: 핀테크
기간: 이번 주
경쟁사: 카카오페이, 토스
```

**출력:**
```
# 2026년 4월 4주 주간 브리핑 (핀테크)

## 헤드라인 요약 (3줄 핵심)
1. 금융위, 마이데이터 사업 3단계 확대 발표 (은행권 반발)
2. 카카오페이, 미국 하와이서 QR 결제 서비스 시작
3. 토스, 청년층 타겟 '토스 체크카드' 출시 (연회비 무료)

## 업계 주요 뉴스
```

## 주의사항

### 정보 소스 신뢰성

- **경제/비즈니스**: 한국경제(hankyung.com), 매일경제(mk.co.kr), 파이낸셜뉴스(fnnews.com)
- **스타트업/테크**: 블로터(bloter.net), 벤처스퀘어(venturesquare.net), 이코노미스트
- **정책/규제**: 정책브리핑(korea.kr), 중기부 공식 채널
- **금융 지표**: 네이버 금융, KRX 한국거래소, 한국은행 ECOS

### 브리핑 자동화 팁

- `.gil/config.json`에 `industry`, `competitors`, `keywords` 설정 시 맞춤 브리핑 자동화
- 사용자가 "매일 아침 브리핑" 요청 시 이전 브리핑과 비교하여 변화 포인트를 강조
- TAM·SAM·SOM 기반 시장 동향 분석 가능 (market-analyst 스킬과 연동)

### TAM·SAM·SOM 시장 규모 분석 (투자자 브리핑 모드)

사업계획서 또는 IR 브리핑 요청 시 다음 공공 데이터 소스 활용:
- 통계청 KOSIS(kosis.kr)
- 한국무역협회 K-stat(stat.kita.net)
- 중소벤처기업부 K-스타트업(k-startup.go.kr) 통계

## 관련 스킬

- **market-analyst**: 심층 시장 분석이나 TAM/SAM/SOM 산출
- **investor-relations**: 투자자 보고서나 IR 덱 작성
