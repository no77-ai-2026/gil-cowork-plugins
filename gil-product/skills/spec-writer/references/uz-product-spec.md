# UZ 컨텍스트 — Product Spec 작성

## 우즈베키스탄 시장 특화 PRD 항목

### 타슈켄트 IT Park 스타트업 환경
- 거주자 자격 요건 (직원 5인 이상, 외화 수익 80% 이상)
- 세제 혜택: 법인세 0%, 사회세 3% (일반 12% 대비)
- VAT(NDS) 면제 (소프트웨어 수출)

### 트릴링구얼 사용자 인터페이스 스펙
- 한국어 / 러시아어 / 우즈벡어 (라틴) 동시 지원
- 키릴/라틴 전환 (우즈벡어는 2030년 라틴 완전 전환 정책)
- 화폐: KRW · UZS (1 USD ≈ 12,800 UZS, 2026년 5월 기준)

### UZ 모바일 환경
- Android 92% / iOS 6% / 기타 2% (한국 60/39 대비)
- 결제: Uzum Bank / Click / Payme / Humo / Uzcard
- 메신저: Telegram 1위 (50%+), WhatsApp 2위, Imo 3위

## PRD 섹션에 추가할 한·UZ 비교 표
- 사용자 페르소나: 한국 / UZ 각 2명 이상
