# CS 응답 톤 가이드

> gil-support | draft-response 참조 가이드 | 2026 기준

## 개요

고객 응대 품질은 어조(tone)에서 결정됩니다. 동일한 정보라도 어조에 따라 고객 만족도가 크게 달라집니다. 이 가이드는 한국 CS 환경에 최적화된 채널별 어조 기준을 제공하며, gil-hr의 `korean-tone-reviewer` 에이전트와의 연동 방법을 안내합니다.

---

## 1. 경어 체계

### 한국어 경어 수준 구분

| 수준 | 형태 | 사용 상황 |
|------|------|-----------|
| 격식체 최고 | ~합니다/~습니다 + 높임말 전체 | 공문서, 법적 서한, B2B 공식 답변 |
| 격식체 표준 | ~합니다/~습니다 | 이메일, 헬프데스크, 일반 CS |
| 비격식 높임 | ~해요/~세요 | 채팅, 카카오톡, 친근한 브랜드 |
| 반말 금지 | ~해/~야 | CS에서 절대 사용 금지 |

### 경어 사용 원칙

- 고객을 지칭할 때: "고객님" (이름 모를 때), "[성함]님" (이름 알 때)
- 회사를 지칭할 때: "저희" (주어) / "당사" (공문서)
- 사과 표현: "죄송합니다" > "미안합니다" (CS에서는 전자 사용)
- 부정 표현 변환: "안 됩니다" → "어려운 점이 있어, 대신 ○○을 도와드릴 수 있습니다"

## 2. 채널별 어조 차이

### 전화 응대

- 첫인사: "안녕하세요, [회사명] 고객센터 [담당자명]입니다."
- 속도: 명확하고 천천히, 핵심 먼저 말하기
- 반복 설명: 한 번 더 요청 시 "네, 다시 한번 말씀드리겠습니다" (짜증 표현 절대 금지)
- 마무리: "추가로 도움이 필요하시면 언제든지 연락 주세요."

### 이메일 응대

- 제목: "[문의 접수] 내용 요약" 형식으로 확인 가능하게
- 본문 구조: 공감 인사 → 문의 내용 확인 → 처리 결과 → 다음 단계 → 마무리 서명
- 길이: 200-400자 (너무 길면 핵심이 묻힘)
- 서명: 담당자명, 연락처, 처리 가능 시간 포함

### 채팅(카카오채널·앱 내)

- 응답 속도가 품질보다 중요한 채널
- 짧고 명확하게: 한 메시지당 1가지 내용
- 이모지: 브랜드 가이드에 따라 선택적 사용 (과도한 사용 지양)
- 자동응답 연계: 처리 시간이 필요할 경우 "확인 중입니다. 잠시만 기다려 주세요" 선발송

## 3. 어조 품질 기준

### 좋은 응답의 특징

- [ ] 고객의 감정에 먼저 공감하는 문장 포함
- [ ] 고객이 해야 할 다음 행동이 명확함
- [ ] 처리 기한 또는 기대 시간 명시
- [ ] 부정어 최소화, 가능한 것 먼저 안내
- [ ] 경어 수준이 처음부터 끝까지 일관됨

### 금지 표현 목록

| 금지 표현 | 대체 표현 |
|-----------|-----------|
| "규정상 안 됩니다" | "현재 기준으로는 어렵지만, [대안]을 제안드릴 수 있습니다" |
| "확인해 보겠습니다" (단독) | "확인 후 [시간] 내로 연락드리겠습니다" |
| "다음에 이용해 주세요" | 명확한 해결 방안 또는 일정 안내 |
| "저도 잘 모릅니다" | "담당 부서에 확인 후 정확히 안내드리겠습니다" |

## 4. korean-tone-reviewer 에이전트 연동

gil-hr의 `korean-tone-reviewer` 에이전트를 호출하여 경어 수준과 어조 적절성을 자동으로 검증할 수 있습니다.

### 호출 방법

```
korean-tone-reviewer에게 다음 응답의 경어 수준과 어조를 검토해달라고 요청하세요:

[초안 응답 내용]

검토 기준:
- 채널: [이메일/채팅/전화/공문서]
- 브랜드 어조: [공식/표준/친근]
- 고객 상황: [일반 문의/불만 고객/VIP]
```

### 검토 항목

- 경어 일관성 (격식체 수준 유지 여부)
- 공감 표현 포함 여부
- 부정 표현 사용 여부
- 채널 어조 적합성
- 다음 단계 명확성
