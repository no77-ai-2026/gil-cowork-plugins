# CS 응답 템플릿 모음

## 초기 접수 응답
"안녕하세요, {고객명}님. 문의 주셔서 감사합니다.
접수번호 {티켓번호}로 등록되었으며, 담당자가 확인 후 {응답시간} 이내에 답변드리겠습니다."

## 해결 완료 응답
"안녕하세요, {고객명}님. 문의하신 {이슈요약}에 대해 안내드립니다.
{해결 내용}
추가 문의사항이 있으시면 언제든 말씀해 주세요."

## 에스컬레이션 안내
"안녕하세요, {고객명}님. 해당 건은 전문 담당팀으로 이관하여 처리 중입니다.
{예상 처리일}까지 결과를 안내드리겠습니다."

## 환불/교환 안내
"안녕하세요, {고객명}님. {사유}에 따른 {환불/교환} 절차를 안내드립니다.
처리 기간: 영업일 기준 {N}일 이내
{추가 안내사항}"

## 불만 대응 (경어 강화)
"불편을 드려 대단히 죄송합니다, {고객명}님.
말씀하신 내용을 정확히 확인하여 {처리 방안}으로 조치하겠습니다.
재발 방지를 위해 내부적으로도 개선하겠습니다."
