## 2. WB 조달 표준

### Procurement Regulations (2016, 개정)

#### 컨설팅
- QCBS (Quality and Cost-Based Selection) — 80:20 일반
- QBS (Quality-Based) — 복잡·전문
- LCS (Least Cost) — 표준화
- CQS (Consultant's Qualification) — 소규모
- SSS (Single Source) — 예외

#### 시공·물품
- ICB (International Competitive Bidding)
- LIB (Limited International Bidding)
- NCB (National Competitive Bidding)
- Direct Selection (예외)

### STEP (Systematic Tracking)
WB 사업 입찰 추적 시스템.

### 절차
1. EOI (Expression of Interest) — 14일+
2. Shortlist (5~7개)
3. RFP 발송
4. 기술·가격 제안서 (45일+)
5. 평가 (기술 → 가격)
6. Negotiation
7. Contract

### Step 3: EOI 응답 (1차)


## 12. 입찰 평가 매트릭스 (예: WB QCBS 80:20)

### 기술 평가 (80점)

```
1. 컨설턴트 자격·경험 (10)
2. 사업 이해·접근 방법 (30)
3. 핵심 인력 (40)
4. 보고서·발표 (있으면)
```

### 가격 평가 (20점)

```
가장 낮은 가격 = 100% 점수
다른 가격 = (최저가 / 해당 가격) × 100%
```

### 합계 점수

```
Combined Score = (Technical Score × 0.8) + (Financial Score × 0.2)
```

## 13. 입찰 흔한 실수

1. **EOI 너무 많은 정보**: 간결·핵심
2. **CVs 표준 X**: WB·ADB는 표준 양식
3. **유사 사업 X**: 분야·국가·규모 매칭
4. **TOR 분석 부족**: 모든 요구 응답
5. **방법론 일반론**: 본 사업 특화
6. **Joint Venture 합의 모호**: 사전 명확
7. **가격 비현실**: 너무 낮 X·너무 높 X
8. **언어 (영어 vs 러시아어)**: WB·ADB 영어, UZ 러시아어
9. **마감 직전 제출**: 시스템 오류 위험
10. **외국 회사 단독 (UZ)**: 현지 파트너 권장
