# gil-oda

> 한·우즈벡 ODA·EDCF 사업 특화 플러그인 · v0.1.0

## 개요

대한민국 공적 개발 원조 (ODA) — 무상 협력 (KOICA), 유상 협력 (EDCF), 정책 자문 (KSP) — 의 우즈베키스탄 사업을 기획·신청·실행·평가하는 풀 라이프사이클 도구. UZ 정부·국제기구 (UN·World Bank·ADB·EBRD)와 협력하는 한국 기업·컨설턴트·NGO·연구기관·KOICA Tashkent·KOTRA Tashkent 관계자 활용.

## 풀 라이프사이클 워크플로우

```
1. 사업 발굴·기획 (oda-project-planner)
   - Concept Note + Log Frame
   - SDG·UZ 정부 우선순위 매핑

2. 타당성 조사 (edcf-feasibility)
   - Pre-F/S → F/S
   - 경제·재무·기술·환경·사회

3. 제안·승인 (oda-proposal-writer)
   - PCP·Pre-F/S·F/S 제출
   - KSP 정책 자문 별도 트랙

4. 입찰·계약 (oda-tendering-uz)
   - 컨설턴트·시공·물품 입찰
   - UZ + WB + ADB 표준

5. 실행·모니터링 (oda-monitoring-report)
   - 분기·반기·연간 보고
   - PDM·RBM 추적

6. 완료·평가 (oda-monitoring-report)
   - 사업 완료 보고
   - SROI·임팩트 평가

7. 거버넌스 (oda-stakeholder-engagement)
   - 전 단계 다자 협력
```

## 한·우즈벡 ODA 핵심

| 시기 | 주요 사업 |
|------|---------|
| 1992~2000 | 외교 수립·KOICA 시작 |
| 2000s | 직업 훈련·IT 협력 |
| 2010s | Tashkent 도시·e-Government·한국문화원 |
| 2020s | 보건·교육·디지털·기후 |
| 2024~ | UZ 개혁 + 한국 G2G 강화 |
