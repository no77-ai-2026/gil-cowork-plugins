# 링크드인 (LinkedIn) 2026 콘텐츠 가이드

## 규격 (2026 기준)

| 항목 | 규격 |
|------|------|
| 게시물 텍스트 | 최대 3,000자 |
| 이미지 | 1200×627px 권장 |
| 문서/PDF | 최대 300장 |
| 영상 | 최대 10분 (네이티브) |
| 링크 프리뷰 | 자동 생성 |
| 설문 | 최대 4개 선택지 |

---

## 플랫폼 특성 (2026)

- **B2B 전문 네트워크**: 산업 인사이트, 커리어, 기업 문화 콘텐츠 선호
- **한국 이용자 특성**: IT/스타트업/금융/컨설팅 종사자 비중 높음
- **기업 페이지 강화**: 직원 옹호 프로그램(Employee Advocacy) 확산
- **크리에이터 모드**: 팔로워 기반 도달률 확장 기능

---

## 알고리즘 핵심 원칙 (2026)

- **네이티브 콘텐츠 선호**: 외부 링크 포함 게시물 도달률 30% 이하 억제
- **문서 콘텐츠 우대**: PDF/슬라이드 형식 게시물 높은 체류 시간 유발
- **초기 반응 중요**: 게시 후 1-2시간 인터랙션이 전체 도달 결정
- **1촌 네트워크 댓글 증폭**: 1촌이 댓글 달면 그의 네트워크에 노출

---

## 최적 게시물 구조 (1,300자 기준)

```
[후크] 주목을 끄는 첫 문장 (통계, 질문, 반직관적 주장)
↓
[본문] 핵심 인사이트 3-5개 항목
  - 구체적 사례 또는 데이터 포함
  - 짧은 단락 (2-3줄 최대)
  - 공백 줄로 가독성 확보
↓
[CTA] 댓글, 공유, 팔로우 유도 질문으로 마무리
```

---

## 콘텐츠 유형별 전략

| 유형 | 도달률 | 사용 시기 |
|------|--------|---------|
| 네이티브 문서 (PDF) | 매우 높음 | 인사이트 리포트, 가이드 |
| 텍스트만 (1,300자) | 높음 | 경험 공유, 의견 피력 |
| 이미지 + 텍스트 | 중간 | 인포그래픽, 팀 소식 |
| 외부 링크 게시물 | 낮음 | 피하거나 댓글에 링크 |
| 영상 (네이티브) | 높음 | 제품 데모, 인터뷰 |

---

## 최적 포스팅 시간 (한국 기준)

- **평일**: 화-목 08:00-10:00, 12:00-13:00
- **피크**: 화요일 오전이 가장 높은 참여율
- **빈도**: 주 3-5회 권장

---

## 콘텐츠 체크리스트

- [ ] 첫 문장(후크)에 주목 요소 포함
- [ ] 후크-본문-CTA 3단 구조 준수
- [ ] 네이티브 업로드 우선 (외부 링크 최소화)
- [ ] 단락 간 공백 줄로 가독성 확보
- [ ] 해시태그 3-5개 (산업 관련 키워드)
- [ ] 태그: 언급 인물/기업 1-3개
- [ ] 게시 후 1시간 내 댓글 적극 답변
- [ ] PDF 문서 16:9 비율 슬라이드 형식
