## 카카오톡 채널 (2026)

### 친구 추가 유도 전략
- 웰컴 메시지: 채널 추가 즉시 자동 발송 (쿠폰/혜택 포함)
- 소식 알림: 주 1-2회 이상 발송 시 수신 거부율 상승
- 채널 홈 구성: 채널 소개, 주요 메뉴, 최근 게시물 최적화

### 메시지 유형

| 유형 | 특징 | 비용 |
|------|------|------|
| 친구톡 | 채널 친구에게만 발송 | 건당 과금 |
| 알림톡 | 거래 정보 전달 (예약확인, 배송) | 건당 과금 |
| 스마트채팅 | 자동응답 챗봇 | 월정액/건당 |

### 1:1 채팅 운영
- 응답 시간 목표: 평일 1시간 이내
- 자동응답 설정: FAQ 상위 10개 자동화
- 상담 시간 외 자동 안내 메시지 설정

## 카카오모먼트 (2026)

### 타겟 광고 유형

| 타겟 | 방법 |
|------|------|
| 행동 타겟 | 앱/웹 행동 기반 리타겟팅 |
| 관심사 타겟 | 카카오 서비스 사용 패턴 분석 |
| 데모그래픽 | 나이, 성별, 지역, 직업 |
| 유사 고객 | 기존 고객 DB 기반 유사 사용자 |

### 광고 포맷
- **피드 광고**: 카카오스토리, 뉴스 피드 노출
- **배너 광고**: 카카오 서비스 전반 배너
- **동영상 광고**: 인스트림/아웃스트림

## 카카오 비즈보드 (2026)

- **노출 위치**: 카카오톡 채팅 목록 상단 (최상위 노출)
- **형식**: 이미지 + 텍스트 배너 (640×200px 권장)
- **특징**: 클릭 시 채널 친구 추가, 쿠폰 다운로드, 웹 연결
- **과금**: CPM (노출) 또는 CPC (클릭)
