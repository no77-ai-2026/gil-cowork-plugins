# 브랜드 컨텍스트 템플릿

랜딩 페이지 제작 전 수집해야 할 브랜드 정보입니다.

## 1. 브랜드 보이스

### 톤 형용사 (3-5개)
예: 신뢰감 있는, 전문적인, 따뜻한

### 톤 예시
**이런 느낌 좋아요 (Do)**:
- "3개월 만에 매출 40% 성장을 경험하세요"
- "10,000개 팀이 선택한 이유가 있습니다"

**이런 건 싫어요 (Don't)**:
- "혁신적인 솔루션으로 비즈니스를 혁신하세요"
- "최첨단 기술로 모든 것이 바뀝니다"

### 금지 표현
- "혁신적인 솔루션", "최첨단 기술", "업계 최고"
- 근거 없는 숫자 ("수천 명이 사용", "엄청난 성과")
- 과도한 감탄부호(!!!)
