# 랜딩 페이지 카피 규칙

## AI 카피 안티패턴 (HARD FAIL)

다음 표현이 발견되면 반드시 수정합니다:

| 안티패턴 | 수정 방향 |
|---------|----------|
| "혁신적인 솔루션" | 구체적 메커니즘으로 대체 ("자동 분류 알고리즘이 수동 작업 80% 감소") |
| "최첨단 기술" | 기술명 + 효과 명시 ("GPT-4o 기반 실시간 번역") |
| "빠르다" (숫자 없이) | 정량화 ("기존 대비 3배 빠른 처리") |
| "모든 사용자가 만족" | 구체 수치 ("4.8/5.0 평균 평점, 2,340개 리뷰") |
| "오늘 바뀝니다!" | 증거 기반 약속 ("14일 무료 체험 후 판단하세요") |
| "업계 최고" | 비교 근거 제시 ("G2 리뷰 기준 카테고리 1위") |

## 필수 카피 요소 체크리스트

- [ ] 구체적인 숫자 포함 (%, 배수, 금액, 시간)
- [ ] 후기에 실명 + 직책 포함 ("김OO, A사 마케팅 팀장")
- [ ] FAQ에서 주요 반대 의견 미리 해소
- [ ] 보증/안전장치 문구 ("14일 무료 체험, 신용카드 불필요")
- [ ] 헤드라인이 10초 내 가치 전달

## 카피 구조 원칙

### 헤드라인
- 10단어 이내
- 핵심 혜택 1가지만 전달
- "어떻게" 아닌 "무엇을 얻는가" 관점

### 서브헤드라인
- 헤드라인 보완 (방법 또는 차별점)
- 20단어 이내

### 본문
- 문장당 20단어 이내
- 단락당 3문장 이내
- 기능(Feature) → 혜택(Benefit) 전환 필수
  - 기능: "AI 자동 분류"
  - 혜택: "하루 2시간 절약"

### CTA 버튼
- 동사로 시작 ("시작하기", "다운로드", "상담 신청")
- 7단어 이내
- 보증 문구 하단 배치 ("30초 만에 가입, 카드 불필요")

## 톤 프로파일 출력 예시

```
톤 분석:
- 주요 감정: 신뢰, 전문성
- 부가 감정: 호기심, 안도감
- 독자가 느낄 것: "이 팀은 내 문제를 이해하고 실제 해결책이 있다"
- 문장 평균 길이: 15단어
- 피해야 할 것: 과도한 감탄사, "꼭!", "반드시!", "지금 당장!"
```
