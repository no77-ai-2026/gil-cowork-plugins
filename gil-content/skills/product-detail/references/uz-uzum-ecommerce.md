## 개요

**Uzum Market**은 우즈베키스탄의 최대 이커머스 플랫폼으로, 한국의 쿠팡과 유사한 위치를 차지합니다. 한국 회사가 UZ 시장 진출 시 Uzum Market 입점이 가장 효과적인 채널 중 하나입니다.

## 3. Uzum Market 상세페이지 규격

### 이미지

| 항목 | 규격 | 비고 |
|------|------|------|
| 메인 이미지 | 1000x1000px | 정사각형 권장, 흰 배경 |
| 추가 이미지 | 8~12장 | 사용 장면·옵션·디테일 |
| 비디오 | 30초~2분, MP4 | 옵션, 전환율 +30% |
| 이미지 용량 | 장당 500KB 이하 | WebP 권장 |

### 카피 (다국어)

상세페이지 카피는 **러시아어 우선, 우즈벡어 보조** 또는 양쪽 병기:

- 제품명: 러시아어 + 우즈벡어 (영어는 옵션)
- 상세 설명: 러시아어 본문 + 우즈벡어 핵심 정보
- 옵션·스펙: 러시아어
- 후기는 자동 다국어 (사용자 언어 자동 표시)

### 가격

- 통화: UZS (우즈벡 솜) 단일 표기
- 천 단위 구분: **공백** (예: `850 000 UZS`, 한국식 콤마 ❌)
- USD 환산 표기: 부가 (1 USD ≈ 12,500 UZS, 변동)
- 할인: 정가 (취소선) + 할인가 + 할인율 (%)
- 할부: 3·6·12개월 옵션 표기 권장 (UZ 일반적)

### 결제 방식 (Uzum Pay 통합)

- Uzum Pay (자체)
- Click (UZ 결제 게이트웨이 1위)
- Payme (UZ 결제 게이트웨이 2위)
- 카드 결제 (Visa, Mastercard, Humo, UzCard)
- 후불 (현금) — 일부 카테고리

## 4. 상세페이지 카피 구조 (Uzum 최적화)

### 표준 8섹션 구조

```
[1] 히어로 — 제품 이미지 + 제품명 + 가격 (UZS) + 별점
[2] 옵션 선택 — 색상·크기·모델 (Uzum 표준 위젯)
[3] 핵심 셀링 포인트 — 3~5개 불릿 (러시아어 + 우즈벡어)
[4] 상세 설명 — 텍스트 + 이미지 4~8개
[5] 스펙 표 — 표 형식 (재료·치수·전압·인증)
[6] 사용법 / 보관법 — 단계별 (필요 시)
[7] 배송·반품 — Uzum 표준 정책 + 셀러 추가 정책
[8] 후기 — 자동 게시 (셀러 통제 X)
```

### 카피 작성 규칙 (러시아어)

```
제품명 공식:
[브랜드] [핵심 기능] [모델/스펙]
예: "Samsung Galaxy A55 5G 256GB Чёрный"

핵심 셀링 포인트:
- ✓ 짧은 동사구로 시작 ("Защищает кожу 24 часа")
- ✓ 숫자·증거 포함 ("В 3 раза эффективнее")
- ✗ 추상적 표현 회피 ("Высокое качество")

상세 설명:
- 단락당 2~4문장
- H2/H3 소제목 활용 (Uzum HTML 일부 지원)
- 이미지 사이 텍스트 단락
```

### 한국 제품 차별화 포인트 (UZ 시장)

UZ 소비자에게 한국 제품의 차별 포인트:

- **K-Beauty**: 한국산 = 프리미엄 인식
- **K-Tech (가전)**: 삼성·LG = 신뢰 1위
- **한국식 디자인**: 미니멀·세련 = 도시 청년층 선호
- **K-Food**: 라면·과자 = 트렌디
- **K-Drama 연계**: 드라마 노출 제품 = 즉시 인기

이를 카피에 명시 ("Сделано в Корее", "Корейский стандарт качества" 등)하면 전환 +20% 효과.

## 5. Uzum 검색 최적화 (Uzum SEO)

### 키워드 전략

- **카테고리 정확 선택**: Uzum은 카테고리 기반 검색이 강함
- **제품명에 핵심 키워드**: 검색 시 가장 우선 매칭
- **상세 설명 키워드**: 자연스럽게 반복 (스팸 X)
- **태그**: 5~10개 (러시아어 + 우즈벡어 혼합)

### 한국 회사 일반 실수

- 한국어 키워드 직역만 (현지 검색어 조사 X)
- 영어만 표기 (UZ 검색 미노출)
- 가격을 KRW로 표기 (UZS 의무)
- 배송지를 한국으로 (UZ 내 창고 또는 빠른 국제배송 명시 필수)

## 6. Uzum Express (배송)

Uzum 자체 물류 시스템 — 한국 쿠팡 로켓배송과 유사:

| 옵션 | 배송 시간 | 비용 |
|------|----------|------|
| Uzum Express | 1~2일 (도시 내) | 무료~$2 |
| 일반 배송 | 3~7일 (지방 포함) | 무료~$1 |
| Pickup Point | 다음 날 (수령 지점) | 무료 |
| 국제 배송 | 7~14일 | $5~$30 |

**한국 회사 운영 모델:**
- A) UZ 내 창고 임대 → Uzum Express (1~2일 배송)
- B) 한국에서 직배 (DHL/FedEx 통해 7~14일)
- C) 한국 → UZ 일괄 입고 → 현지 창고에서 출고

A·C가 전환율 가장 높음.

## 7. Uzum 마케팅 도구

### 7.1 Uzum 광고 (Sponsored)

- 검색 결과 상단 노출
- 카테고리 페이지 노출
- 비용: CPC $0.05~$0.5

### 7.2 Uzum 프로모션

- 시즌 세일 (네이비 후레이드, 우즈벡 명절 등)
- 가격 할인 — Uzum 알고리즘이 노출 우대

### 7.3 Uzum + Telegram 시너지

- Uzum 상품 링크를 Telegram 채널에 공유
- Telegram 봇으로 Uzum 결제 연동 가능
- 인플루언서 → Telegram → Uzum 구매 퍼널이 가장 일반적

## 8. 후기 관리

### 한국과 차이

- Uzum 후기는 셀러가 삭제 불가 (Uzum 통제)
- 부정 후기에 셀러 답변 필수 (이미지 손상 방지)
- 평점 4.5 이상이 노출 우위
- 후기 1,000개 이상이 카테고리 상위 노출

### 후기 유도 전략

- 첫 100명 구매자에게 할인 쿠폰 제공 (후기 작성 조건)
- 사진 후기 별도 보상
- 부정 후기 즉시 답변 + 환불·교환 신속 처리
