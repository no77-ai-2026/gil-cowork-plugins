# 플랫폼별 규격 가이드

## 1. 네이버 스마트스토어

### 이미지 규격

메인 이미지 (대표 이미지):
- 크기: 860x860px (정사각형 필수)
- 포맷: JPG, PNG (WebP 미지원)
- 용량: 5MB 이하/장
- 배경: 순백(#FFFFFF) 권장, 여백 최소 10%
- 해상도: 72dpi 이상

추가 이미지:
- 최대 10장
- 동일 규격 (860x860px)
- 다양한 앵글: 정면, 후면, 측면, 디테일, 사용 장면, 패키지

상세 이미지:
- 폭: 860px 고정 (높이 무제한)
- 포맷: JPG, PNG
- 용량: 각 500KB 이하 권장 (총합 10MB 이하)
- 이미지 기반 상세가 기본 (HTML 텍스트보다 이미지에 텍스트를 넣는 방식)

동영상:
- 30초-60초 제품 소개 영상
- MP4 포맷, 100MB 이하
- 해상도: 720p 이상 권장

### 텍스트 및 레이아웃

에디터 제약사항:
- 스마트에디터 ONE 사용 (HTML 직접 입력 제한적)
- iframe, script 태그 사용 불가
- 외부 CSS/JS 로드 불가
- 인라인 스타일만 허용 (style 속성)
- position: fixed, position: sticky 사용 불가
- 최대 폭: 860px (초과 시 잘림)

텍스트 금지 영역:
- 대표 이미지에 가격, 할인율 텍스트 삽입 금지 (검색 노출 불이익)
- 대표 이미지에 테두리/프레임 삽입 금지
- "최저가", "1위" 등 비교 광고 문구 금지 (근거 없는 경우)

### SEO 규칙 (네이버 쇼핑 검색 최적화)

상품명 작성 규칙:
```
[브랜드] [제품명] [핵심 키워드] [용량/사이즈] [색상]
예: "모아이 초음파 가습기 4.5L 대용량 화이트"
```

상품명 금지 사항:
- 특수문자 남용 (★, ♥, ◆ 등)
- 키워드 반복 ("가습기 가습기 가습기")
- 브랜드와 무관한 키워드 삽입
- 40자 초과 (검색 결과에서 잘림)

카테고리 키워드 전략:
- 네이버 쇼핑 인기 검색어 반영
- 시즌 키워드 활용 ("겨울 가습기", "여름 선풍기")
- 롱테일 키워드 조합 ("원룸 소형 가습기 추천")
- 네이버 데이터랩으로 검색량 확인

태그:
- 최대 10개 태그 입력 가능
- 관련 키워드를 태그로 등록 (검색 노출 보조)
- 브랜드명, 카테고리명, 속성 키워드 포함

## 2. 쿠팡

### 이미지 규격

메인 이미지:
- 크기: 780x780px (정사각형 필수)
- 포맷: JPG, PNG
- 용량: 5MB 이하
- 배경: 순백(#FFFFFF) 필수 (회색, 패턴 배경 금지)
- 제품이 이미지의 85% 이상 차지해야 한다

추가 이미지:
- 최대 9장 (메인 포함)
- 동일 규격 (780x780px)
- 9장 모두 활용 권장 (노출 우선순위 영향)

상세 이미지:
- 폭: 780px
- 높이: 무제한
- 이미지에 텍스트 삽입 가능 (상세 설명용)

### 로켓배송 뱃지 활용법

로켓배송 상품 혜택:
- 검색 결과 상위 노출 가산점
- "로켓배송" 필터 검색 시 포함
- 무료 반품 라벨 자동 적용
- 구매 전환율 평균 1.5-2배 상승

로켓배송 입점 조건:
- 쿠팡 물류센터 입고 필수
- 재고 관리 기준 충족
- 상품 품질 검수 통과

뱃지 위치:
- 제품 기본 정보 영역, 배송 정보 옆에 자동 표시
- 별도 이미지 삽입 불필요 (플랫폼 자동 렌더링)

### 검색 알고리즘 대응

쿠팡 검색 순위 결정 요인 (가중치 추정):
```
1. 판매량 + 판매 속도 (40%)
2. 전환율 (25%)
3. 리뷰 수 + 평점 (15%)
4. 상품명 키워드 일치 (10%)
5. 가격 경쟁력 (10%)
```

상품명 작성 규칙:
```
[브랜드] [핵심 키워드] [부가 키워드] [규격/용량]
예: "모아이 초음파 가습기 대용량 4.5L 무드등 원룸"
```

상품명 최적화:
- 50자 이내 권장
- 핵심 키워드를 앞에 배치
- 검색 의도와 일치하는 키워드 사용
- "추천", "인기", "베스트" 등 주관적 표현 지양

### 쿠팡 상세 에디터 특이사항

- HTML 직접 입력 가능 (쿠팡 파트너스 > 상품등록 > 상세설명)
- 인라인 스타일 지원
- 외부 CSS/JS 불가
- iframe 사용 불가
- 최대 폭: 780px
- 이미지 자동 리사이징 기능 제공

## 3. 카카오 커머스

### 카카오톡 선물하기 규격

대표 이미지:
- 크기: 750x750px (정사각형)
- 포맷: JPG, PNG
- 용량: 2MB 이하
- 배경: 깔끔한 단색 또는 연출 컷 (선물 느낌)

추가 이미지:
- 최대 5장
- 동일 규격 (750x750px)

상세 이미지:
- 폭: 750px 권장 (모바일 최적화)
- 모바일 전용으로 설계한다

선물하기 특수 요소:
- 선물 메시지 카드 미리보기 영역
- 수령인 정보 입력 필드
- 포장 옵션 선택 (유료 포장 가능)

### 모바일 최적화 (375px 기준)

카카오 커머스 특성:
- 90% 이상 모바일 트래픽 (카카오톡 인앱 브라우저)
- 데스크톱 레이아웃 불필요 (모바일만 설계)
- 인앱 브라우저 제약: 일부 JavaScript API 미지원

모바일 설계 규칙:
- 뷰포트: 375px 기준
- 이미지 폭: 100% (vw 사용)
- 폰트 최소: 14px
- 터치 타겟: 최소 44x44px
- 스크롤 깊이: 최소화 (핵심 정보 우선 배치)
- 페이지 로딩: 3초 이내 (인앱 브라우저 성능 제한)

### 카카오 쇼핑 검색 특성

검색 노출 조건:
- 카카오 쇼핑 입점 상품만 검색 노출
- 카카오톡 채널 연동 시 가산점
- 리뷰/평점이 검색 순위에 반영

키워드 전략:
- 카카오 쇼핑 트렌드 키워드 반영
- 선물 관련 키워드 추가 ("생일 선물", "기념일", "감사 선물")
- 시즌 키워드 ("크리스마스", "발렌타인", "추석")

## 4. 공통 규칙

### 이미지 용량 제한

| 유형 | 권장 용량 | 최대 용량 |
|------|---------|---------|
| 메인 이미지 | 300KB | 500KB |
| 상세 이미지 (개별) | 300KB | 500KB |
| 상세 이미지 (전체) | 3MB | 5MB |
| 아이콘/배지 | 30KB | 50KB |

이미지 최적화 방법:
- TinyPNG/TinyJPG로 무손실 압축
- WebP 변환 (자사몰에서 사용, 플랫폼은 미지원 가능)
- 이미지 크기를 표시 해상도의 2배로 설정 (레티나 대응)
- 불필요한 메타데이터 제거 (EXIF)

### 동영상 활용

제품 소개 영상 규칙:
- 길이: 15-30초 (주의력 유지)
- 구성: 문제 제시(5초) + 제품 소개(15초) + CTA(5초)
- 자막 필수 (음소거 시청 대비)
- 배경음악: 저작권 프리 BGM
- 해상도: 1080p (세로형 9:16 또는 정사각형 1:1)
- 용량: 50MB 이하

### 금지 표현 체크리스트

의료법 위반:
- "치료", "치유", "완치" (의약품 아닌 경우)
- "암 예방", "당뇨 개선" (건강기능식품 미인증 시)
- "의사 추천" (실제 추천 근거 없는 경우)
- "부작용 없음" (모든 제품에 부작용 가능성 존재)

식약법 위반:
- "다이어트 효과" (기능성 식품 미인증 시)
- "피부 재생", "주름 개선" (기능성 화장품 미인증 시)
- "100% 천연" (화학적 가공이 포함된 경우)

표시광고법 위반:
- "업계 1위" (공인 기관 인증 없는 경우)
- "최저가 보장" (실제 최저가가 아닌 경우)
- "한정 수량" (실제 한정이 아닌 경우 -- 반복 사용 시)
- 허위 할인율 표시 (원가를 부풀린 경우)
- 타사 상표 무단 사용

전자상거래법 필수 표기:
- 상품명
- 제조사/수입사
- 제조국(원산지)
- 제조일/유통기한 (해당 시)
- 품질보증 기준
- A/S 연락처
- 소비자 상담 전화번호

## 6. 자사몰 (독립 쇼핑몰)

### 규격 자유도

자사몰은 플랫폼 제약이 없으므로 최적의 사용자 경험을 설계한다:
- 이미지: 자유 규격 (반응형 srcset 필수)
- HTML/CSS/JS: 제한 없음
- 동영상: 자동 재생 허용 (음소거 기본)
- 인터랙션: 360도 뷰, AR 미리보기, 실시간 채팅

권장 규격:
- 메인 이미지: 1200x1200px (고해상도 레티나 대응)
- 상세 이미지: 최대 1200px 폭
- 반응형 필수: 375px-1440px
- Schema.org Product 구조화 데이터 필수
- Open Graph 메타태그 필수

### 필수 기능

- 실시간 재고 표시
- 장바구니 / 위시리스트
- 최근 본 상품
- 상품 비교 기능
- 소셜 공유 버튼 (카카오톡, 밴드, 페이스북)
- 실시간 채팅 (카카오 채널, 채널톡 등)
- 네이버페이 / 카카오페이 / 토스페이 간편결제
