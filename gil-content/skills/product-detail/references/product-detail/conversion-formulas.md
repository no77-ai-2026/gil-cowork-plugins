# 전환율 극대화 공식 및 체크리스트

## 1. AIDA 모델 세부 구현

### Attention (주의: 0-3초)

**헤더 텍스트 공식**:
```
[숫자/형용사] [핵심 benefit]
예: "10시간 지속 가습" "자극 없는 화장품" "24시간 배송"
```

**금지 사항**:
- 일반적 표현: "좋은 가습기"
- 회사명 중심: "OOO 가습기"
- 기술 용어: "초음파식 가습 방식"

**추천 표현**:
- "밤 10시간 가습으로 아침 피부 촉촉"
- "직장인 89% 추천, 건조함 해결"

### Interest (관심: 3-15초)

**구성 요소**:
1. 고객 pain point 명시 (공감)
2. 우리 제품 강점 (해결책)
3. 경쟁사와의 차이 (우위성)

**고객 pain point 도출 공식**:
```
타겟 고객 [행동] 중에 [문제]가 발생한다
예: "야근 후 퇴근할 때 사무실이 매우 건조해진다"
    "어제 박박 씻어서 피부가 뒧겨운 채로 잔다"
```

### Desire (욕망: 15-40초)

**5감 시나리오 활용**:

| 감각 | 표현 | 예시 |
|------|------|------|
| 시각 | 색상, 광택 | "화장이 자연스럽게 펴진다" |
| 촉각 | 감촉, 온도 | "피부가 부드럽고 촉촉하다" |
| 청각 | 소음, 음성 | "은은한 미스트음향" |
| 후각 | 냄새 | "무향으로 방 냄새 걱정 없음" |
| 미각 | 맛 | (해당 시 활용) |

**라이프스타일 연결**:
```
아침에 일어나면 → 피부가 촉촉하다 → 화장이 잘 먹힌다
→ 자신감으로 하루를 시작한다
```

### Action (행동: 40-60초)

**CTA 버튼 배치 공식**:
```
상품가격 표시
     ↓
긴급성 표시 (오늘만, 남은 재고 10개)
     ↓
Primary CTA 버튼 (주문하기 - 대, 눈에 띄는 색)
     ↓
Secondary CTA 버튼 (찜하기 - 소, 흐린 색)
```

## 2. FAB 카피라이팅 변환표

### 상품별 FAB 예시

**가습기**:
```
Feature: 초음파식, 4.5L 용량, 10시간 연속 작동
  ↓
Advantage: 전기료 낮음, 자주 채울 필요 없음, 밤새도록 사용 가능
  ↓
Benefit: 월 3,000원 절감, 아침에 피부 촉촉, 숙면 가능
```

**세럼**:
```
Feature: 비타민C 20%, 분자량 100nm (초소형), EGF 함유
  ↓
Advantage: 빠른 흡수, 효과 빠름, 피부 재생 촉진
  ↓
Benefit: 3일 내 피부 개선 감지 가능, 여드름 흉터 개선, 탱탱한 피부
```

**노트북 가방**:
```
Feature: 방수 원단, 안티쇼크 패드, 13인치 맞춤 주머니
  ↓
Advantage: 빗에 젖지 않음, 낙하 충격 흡수, 딱 맞는 수납
  ↓
Benefit: 출근길 비 걱정 없음, 노트북 안전함, 짐 가볍게 느껴짐
```

## 3. 심리 트리거 활용 체크리스트

### 스카시티 (Scarcity) - 한정성

```
강점 구분 (강함 → 약함):
1. 시간 한정: "오늘 자정까지만 50% 할인"
2. 수량 한정: "남은 재고 12개"
3. 공급 한정: "생산량 제한으로 월 500개만 판매"
4. 초대 한정: "VIP 회원 전용 50% 할인"
```

### 사회적 증명 (Social Proof)

```
강점 구분 (강함 → 약함):
1. 판매량: "누적 판매 100,000개" (매우 강함)
2. 평점: "★★★★★ 4.8/5 (리뷰 5,000개)" (강함)
3. 유명인: "연예인 OOO 사용" (중간)
4. 전문가: "피부 전문의 추천" (중간)
5. 일반 고객: "고객 후기: '정말 좋아요!'" (약함)
```

**배치 우선순위**:
1. 상품명/가격 아래: 판매량 + 평점
2. CTA 버튼 위: 최근 리뷰 1-2개
3. FAQ 위: "자주 구매하는 고객 후기" 섹션

### 긴급성 (Urgency)

```
약한 긴급성:
- "지금 주문하면 내일 배송" (배송 속도)
- "한정판 상품입니다" (희귀성)

강한 긴급성:
- "오늘 자정까지 50% 할인! 남은 시간 2시간 47분" (시간 카운트다운)
- "남은 재고 12개! 지금 주문하지 않으면 품절됨" (수량 경고)
```

### 권위성 (Authority)

```
약한 권위:
- "페이스북 좋아요 10,000개"

강한 권위:
- "피부과 의사 OOO 추천"
- "소비자연맹 품질 인증"
- "국제 특허 (미국, EU, 일본)"
```

## 4. 전환율(Conversion Rate) 벤치마크

### 업계별 평균 전환율

| 산업 | 평균 전환율 | 목표 전환율 |
|------|-----------|-----------|
| 뷰티/화장품 | 1.5-2.5% | 3.5%+ |
| 의류/패션 | 1.2-2.0% | 3.0%+ |
| 전자제품 | 0.8-1.5% | 2.5%+ |
| 건강식품 | 2.0-3.0% | 4.0%+ |
| 소비재 | 1.0-2.0% | 3.0%+ |

### 전환율 개선 순서

```
1. 현재 전환율 측정 (GA4)
2. 가장 낮은 세션-구매 단계 파악
3. A/B 테스트로 개선안 검증
4. 가장 효과 높은 변수부터 반영
```

## 5. A/B 테스트 공식

### 테스트 항목 우선순위

| 우선순위 | 항목 | 평균 개선율 | 예상 테스트 기간 |
|---------|------|-----------|-----------------|
| 1 | CTA 버튼 색상 | +5-15% | 1주 |
| 2 | CTA 버튼 텍스트 | +3-10% | 1주 |
| 3 | 가격 표시 방식 | +2-8% | 2주 |
| 4 | 헤더 텍스트 | +3-12% | 2주 |
| 5 | 리뷰 표시 위치 | +1-5% | 1주 |

### 최소 샘플 크기

```
신뢰도 95%, 검증력 80%일 때:

기존 전환율 2% → 2.5% 개선:
필요 샘플 = 약 50,000 방문자

기존 전환율 1% → 1.5% 개선:
필요 샘플 = 약 100,000 방문자
```

## 6. 이미지 최적화 공식

### 이미지 배치 규칙

```
Hero Image: 가로 1200px 이상, JPG, 200KB 이내
Product Image: 가로 600px, JPG, 100KB 이내
Icon/Badge: 가로 100~200px, PNG, 50KB 이내

최적 포맷:
- JPG: 사진 (압축률 높음)
- PNG: 투명 배경 필요 시
- WebP: 현대식 브라우저 (25% 더 작음)
```

### 이미지 Lazy Loading

```html
<!-- 성능 개선: 스크롤 후 이미지 로딩 -->
<img src="placeholder.jpg"
     loading="lazy"
     alt="상품 설명">
```

## 8. 배색 및 대비 가이드

### 배경-텍스트 대비율

```
WCAG AA 기준 (최소): 명도 대비 4.5:1
WCAG AAA 기준 (권장): 명도 대비 7:1

예시:
검은색(#000000) vs 흰색(#FFFFFF) = 21:1 (최고)
검은색(#000000) vs 밝은 회색(#CCCCCC) = 5.5:1 (만족)
검은색(#000000) vs 중간 회색(#999999) = 3:1 (미흡)
```

### CTA 버튼 색상 선택

```
Primary CTA (주문하기):
- 배경: 빨강/주황 (고에너지)
- 텍스트: 흰색

Secondary CTA (찜하기):
- 배경: 회색/흐린색
- 텍스트: 검정
```

## 9. 가격 심리학

### 가격 표시 기법

| 기법 | 예시 | 효과 |
|------|------|------|
| 앵커링 | ~~149,000원~~ → 99,900원 | 할인감 극대화 |
| 부수가 (Decoy) | 기본형 80,000원 / 프리미엄형 120,000원 | 중간 선택 유도 |
| 손실회피 | "지금 사지 않으면 30,000원 손해" | 긴박감 |
| 비교 | "스타벅스 아메리카노 30잔 가격" | 가치 비교 |

## 10. 신뢰도 신호 (Trust Signal) 배치

### 배치 위치

```
페이지 상단 (상품명 아래):
- 평점 4.8/5 (리뷰 5,000개)
- 로켓배송 마크

가격 근처:
- 100% 환불 보증
- SSL 보안 결제

CTA 버튼 아래:
- 고객센터: 1577-XXXX
- 카톡 상담 가능
```
