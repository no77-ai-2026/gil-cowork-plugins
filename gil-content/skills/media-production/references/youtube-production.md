# 유튜브 프로덕션 (youtube-production)

> MoAI-Cowork v1.0.0 하네스 참고자료

## 개요

전략 기획, 스크립팅, 썸네일, SEO 최적화를 거쳐 유튜브 영상 콘텐츠를 협력하여 제작하는 하네스입니다.

## 전문가 역할

- **콘텐츠 전략가**: 주제 분석, 타겟 오디언스 정의, 경쟁 채널 벤치마크, 콘텐츠 포지셔닝, 영상 컨셉 설계 담당
  - 주제 분석: 사용자 입력 주제에서 실행 가능한 영상 각도 도출
  - 타겟 오디언스 정의: 시청자 관심사, 통증점, 검색 의도 분석
  - 경쟁 벤치마크: 웹 검색 통해 동일 주제의 유튜브 영상 조사, 차별화 기회 파악
  - 콘텐츠 컨셉: 영상의 톤, 구조(리스트/스토리/튜토리얼/인터뷰/비교 등), 핵심 훅 결정
  - 키워드 조사: 검색 트래픽 기반 주요 및 관련 키워드 파악

- **프로덕션 검수자**: QA 역할, 전략-스크립트-썸네일-SEO 일관성 검증, 갭과 모순 파악, 개선 피드백 제공
  - 전략-스크립트 정렬: 브리프의 핵심 각도가 스크립트에 충실히 반영되었는가?
  - 스크립트-썸네일 정렬: 스크립트가 썸네일의 호기심을 전달하는가?
  - 스크립트-SEO 정렬: 주요 키워드가 스크립트에 자연스럽게 녹아 있는가?
  - 제목-썸네일-훅 삼각형: 이 세 요소가 시너지를 일으키는가?
  - 품질 체크리스트: 구조, 길이, 톤, CTA, 법률 검토 등

- **스크립트 작가**: 전략 브리프 기반 시청자 유지 최적화 스크립트 작성, 타임코드 기반 스크립트 생성
  - 훅 작성: 처음 5-10초 내 시청자 주의 집중 오프닝 설계
  - 본문 구조: 명확한 진행 구조와 자연스러운 전환
  - 대화 스타일: 회화체, 자연스러운 리듬, 시청자에게 직접 어필
  - 비주얼 큐: `[B-roll: ...]`, `[그래픽: ...]`, `[텍스트 오버레이: ...]` 등 편집 지시
  - CTA 설계: 좋아요, 구독, 댓글 자연스럽게 녹이기

- **SEO 최적화 전문가**: 유튜브 검색 최적화, 메타데이터, 태그 전략, 자막 생성
  - 제목 최적화: 검색 키워드 포함, 클릭 유도, 60자 이내, 주요 키워드는 앞에 배치
  - 설명 작성: 처음 2줄에 핵심 가치 명제와 키워드 배치
  - 태그 전략: 주요 키워드 → 관련 키워드 → 채널명 → 시리즈명 순서
  - 챕터 마커: 스크립트의 타임코드 기반 유튜브 챕터 생성
  - 자막 파일: 스크립트를 SRT 형식으로 변환
  - 해시태그: 영상 제목 위에 표시될 3개 해시태그 선정

- **썸네일 디자이너**: 클릭률 최대화 썸네일 컨셉 설계, Gemini 이미지 생성으로 실제 썸네일 제작
  - 썸네일 컨셉: 전략 브리프의 핵심 각도를 시각적으로 표현하는 방법 결정
  - 텍스트 오버레이: 3-5단어의 임팩트 있는 텍스트 설계
  - 색상 전략: 경쟁사 썸네일과 구별되는 색상 배치
  - 이미지 생성: Gemini 이미지 생성으로 실제 썸네일 제작
  - A/B 변형: 주 썸네일 + 1-2개 대체안 제공

## 워크플로우

### Phase 1: 준비 (오케스트레이터 직접 수행)

1. 사용자 입력에서 추출:
   - **주제/키워드**: 영상이 다룰 주제
   - **채널 정보** (선택): 채널 톤, 구독자 수, 기존 콘텐츠 방향
   - **제약사항** (선택): 영상 길이, 특정 요구사항
   - **기존 파일** (선택): 스크립트, 브리프 등 사용자 제공 자료

2. `_workspace/` 디렉토리 생성
3. 입력을 정리하여 `_workspace/00_input.md`로 저장
4. 기존 파일 제공 시 `_workspace/`로 복사 후 해당 Phase 건너뛰기
5. 요청 범위에 기반해 **실행 모드** 결정

### Phase 2: 팀 편성 및 실행

| 순서 | 작업 | 담당자 | 의존사항 | 산출물 |
|------|------|--------|---------|--------|
| 1 | 콘텐츠 전략 | 전략가 | 없음 | `_workspace/01_strategist_brief.md` |
| 2a | 스크립트 작성 | 작가 | 작업 1 | `_workspace/02_scriptwriter_script.md` |
| 2b | 썸네일 설계 및 생성 | 디자이너 | 작업 1 | `_workspace/03_thumbnail_concept.md` |
| 3 | SEO 패키지 | SEO 전문가 | 작업 1, 2a | `_workspace/04_seo_package.md` |
| 4 | 프로덕션 검수 | 검수자 | 작업 2a, 2b, 3 | `_workspace/05_review_report.md` |

작업 2a(스크립트)와 2b(썸네일)는 **병렬 실행**. 둘 다 작업 1(전략)만 의존하므로 동시 시작 가능.

**에이전트 간 커뮤니케이션:**
- 전략가 완료 → 핵심 각도와 톤 작가에게, 제목 후보와 감정 트리거 디자이너에게, 키워드 맵 SEO 전문가에게 전달
- 작가 완료 → 훅 핵심 메시지 디자이너에게, 스크립트 SEO 전문가에게 전달
- 검수자 모든 산출물 교차 검증, 🔴 반드시 수정할 항목 발견 시 담당자에게 수정 요청 → 재작업 → 재검증 (최대 2라운드)

### Phase 3: 통합 및 최종 산출물

검수 리포트 기반 최종 산출물 컴파일:

1. `_workspace/` 내 모든 파일 검증
2. 검수 리포트의 모든 🔴 반드시 수정할 항목 해결 확인
3. 사용자에게 최종 요약 보고:
   - 전략 브리프 — `01_strategist_brief.md`
   - 영상 스크립트 — `02_scriptwriter_script.md`
   - 썸네일 컨셉 + 이미지 — `03_thumbnail_concept.md`
   - SEO 패키지 — `04_seo_package.md`
   - 검수 리포트 — `05_review_report.md`
   - 자막 파일 — `subtitle.srt`
