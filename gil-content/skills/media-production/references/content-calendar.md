# 콘텐츠 캘린더 (content-calendar)

> MoAI-Cowork v1.0.0 하네스 참고자료

## 개요

마케팅, 소셜 미디어, 커뮤니케이션 팀이 체계적인 편집 계획과 스케줄을 수립하여 모든 채널에서 일관된 오디언스 소통을 유지하는 하네스입니다.

## 전문가 역할

- **콘텐츠 전략 리더**: 비즈니스 목표와 오디언스 요구에 맞춘 포괄적 콘텐츠 테마 개발
  - 오디언스 요구 맵핑: 오디언스의 통증점, 질문, 열망, 콘텐츠 소비 선호 기록
  - 콘텐츠 필러 정의: 3-5개 핵심 콘텐츠 테마 수립 (교육, 영감, 제품, 커뮤니티, 엔터테인먼트)
  - 캠페인 계획: 제품 출시, 계절 이벤트, 전략 우선순위 중심 테마별 콘텐츠 모음 생성
  - 메시징 정렬: 모든 콘텐츠가 핵심 브랜드 메시지와 가치 제안 강화 확인
  - 성공 지표: 콘텐츠 성과 추적 위한 참여, 도달률, 전환율 목표 정의

- **편집 캘린더 관리자**: 채널 전반 발행 스케줄 조율, 일관성과 적시성 보장
  - 타임라인 개발: 3-6개월 선행 콘텐츠 배포 캘린더, 주요 날짜, 휴일, 캠페인 마일스톤 매핑
  - 채널 조율: 각 콘텐츠의 발행 채널(블로그, 소셜, 이메일, 영상, 팟캐스트)과 시간 명시
  - 콘텐츠 배칭: 주제별 관련 콘텐츠 그룹화 및 누적 영향 위한 순서 지정
  - 비상 계획: 계절/트렌드 주제와 신속 대응 기회용 버퍼 콘텐츠 유지
  - 이해관계자 소통: 제품, 고객 성공, 비즈니스 개발 팀과 콘텐츠 스케줄 확인

- **콘텐츠 제작 리더**: 전략과 오디언스 요구에 맞춘 다양한 콘텐츠 형식 제작 감독
  - 형식 전략: 최적 콘텐츠 믹스 결정 (블로그, 영상, 인포그래픽, 케이스 스터디, 웨비나, 팟캐스트)
  - 제작 계획: 각 콘텐츠 유형별 제작 일정과 리소스 요구 예상
  - 크리에이터 배정: 콘텐츠 주제를 주제 전문가에 매칭, 프리랜서/에이전시 조율
  - 품질 기준: 편집 지침, 팩트 체킹, 검수 프로세스 수립
  - 재활용 전략: 핵심 콘텐츠를 채널 및 형식 전반에 걸쳐 재활용하는 방법 계획

- **크로스 플랫폼 조율자**: 모든 고객 터치포인트 전반 일관된 메시징과 최적화된 배포 보장
  - 플랫폼별 조정: 핵심 메시지를 각 채널의 고유 요구사항에 맞춤 (트위터 길이, 링크드인 톤, 유튜브 형식)
  - 오디언스 세분화: 이메일, 소셜, 사이트 개인화를 통해 적절한 오디언스에 콘텐츠 분배
  - 링크 생태계: 채널 간 관련 콘텐츠 연결, 마찰 최소화
  - 댓글 및 참여: 각 플랫폼별 응답 템플릿과 참여 전략 계획
  - 메타데이터 최적화: 제목, 설명, 해시태그, 키워드 발견 가능성 개발

- **캠페인 및 계절 관리자**: 비즈니스 캘린더 맞춘 테마별 콘텐츠 캠페인 계획 및 실행
  - 캠페인 맵핑: 연간 캠페인 기회 파악 (제품 출시, 휴일, 인식의 달, 업계 이벤트)
  - 콘텐츠 내러티브: 각 캠페인 단계별 포괄적 내러티브와 특정 콘텐츠 개발
  - 크로스채널 시퀀싱: 각 채널에 콘텐츠 나타날 시기를 활동 가속화 목적으로 조율 (티저, 공지, 교육, 후속)
  - 홍보 계획: 캠페인 도달력과 참여 증폭 위한 유료 및 유기 홍보 설계
  - 성과 분석: 캠페인 효과 측정, 향후 최적화 위한 학습 기록

## 워크플로우

### Phase 1: 맥락 수집

1. **콘텐츠 목표 정의**: 콘텐츠가 달성해야 할 것 명확히 (인식, 교육, 리드 생성, 유지, 옹호)
2. **오디언스 조사**: 오디언스 인구통계, 통증점, 정보 출처, 선호 콘텐츠 형식 기록
3. **경쟁사 콘텐츠 감시**: 경쟁사 콘텐츠 전략, 갭, 차별화 기회 검토
4. **가용 리소스**: 현재 콘텐츠 제작 역량 목록 (사내팀, 프리랜서, 에이전시, 주제 전문가)
5. **플랫폼 분석**: 타겟 오디언스 활동 채널, 알고리즘 선호도, 참여 패턴 기록

### Phase 2: 분석 및 제작

| 순서 | 작업 | 담당자 | 의존사항 | 산출물 |
|------|------|--------|---------|--------|
| 1 | 콘텐츠 전략 개발 | 전략 리더 | 목표 + 오디언스 | `content-strategy-framework.md` |
| 2 | 편집 캘린더 작성 | 캘린더 관리자 | 전략 | `editorial-calendar.xlsx` |
| 3 | 캠페인 및 테마 계획 | 캠페인 관리자 | 전략 + 캘린더 | `campaign-content-plan.md` |
| 4 | 제작 워크플로우 정의 | 제작 리더 | 전략 | `content-production-guide.md` |
| 5 | 크로스플랫폼 조정 매핑 | 플랫폼 조율자 | 캘린더 | `channel-adaptation-guidelines.md` |
| 6 | 콘텐츠 및 홍보 스케줄 | 캘린더 관리자 | 위 모든 항목 | `content-publishing-schedule.xlsx` |
| 7 | 콘텐츠 템플릿 제작 | 제작 리더 | 제작 가이드 | `content-templates-library.md` |

**에이전트 간 커뮤니케이션**: 전략 리더가 캠페인 테마를 캠페인 관리자와 확인, 캘린더 관리자가 제작 리더와 시간 조율, 플랫폼 조율자가 조정 가이드로 콘텐츠 제작 지원, 캠페인 관리자가 성과 목표를 캘린더 관리자에 제공.

### Phase 3: 검수 및 배포

1. **이해관계자 정렬**: 리더십, 제품, 고객 성공 팀과 캘린더 공유하여 입력과 약속 받기
2. **리소스 검증**: 팀 역량이 계획 콘텐츠 볼륨 전달 가능한지 확인, 필요시 범위 조정
3. **비상 계획**: 트렌딩 주제와 신속 대응 기회용 백업 콘텐츠 옵션 수립
4. **홍보 조율**: 발행 스케줄과 콘텐츠 홍보(유료, 이메일, 소셜 공유) 정렬
5. **모니터링 설정**: 콘텐츠 성과 추적, 최적화, 캘린더 업데이트 프로세스 수립

## 산출물

- **editorial-calendar.xlsx**: 6개월 발행 캘린더, 날짜, 제목, 형식, 채널, 담당자
- **campaign-content-plan.md**: 주요 캠페인의 테마별 콘텐츠 계획, 내러티브 호, 콘텐츠 항목
- **content-production-guide.md**: 형식 기준, 제작 일정, 제작 워크플로우, 품질 체크포인트
- **channel-adaptation-guidelines.md**: 채널별 핵심 콘텐츠 조정 가이드

- **content-templates-library.md**: 블로그 포스트, 소셜 업데이트, 이메일 등 사전 구축 템플릿

## 에러 처리

| 에러 유형 | 대응 전략 |
|---------|---------|
| 불충분한 콘텐츠 역량 | 발행 빈도 감소, 기존 콘텐츠 더 적극 재활용, 프리랜서/에이전시 투입 |
| 낮은 참여율 | 오디언스 선호도 감시, 서로 다른 형식과 주제 테스트, 주제 요청 오디언스 조사 |
| 불일관한 플랫폼 성과 | 최고 성과 채널에 집중, 플랫폼 알고리즘 파악 위해 콘텐츠 변형 테스트 |
| 캠페인 테마 무관성 | 고객 피드백 수집, 전체 캘린더 약속 전 테마 호소성 검증, 유연성 유지해서 방향 전환 가능 |
