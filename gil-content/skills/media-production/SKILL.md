# 미디어 프로덕션 (Media Production)

## 개요

영상·팟캐스트·출판 콘텐츠 기획 및 제작 전문 스킬입니다.
유튜브 채널 전략, 팟캐스트 포맷 설계, Remotion 프로그래매틱 영상,
전자책 기획, 콘텐츠 캘린더 설계를 지원합니다.

지원 영역:
- 유튜브 프로덕션 (채널 기획, 콘텐츠 전략, 알고리즘 최적화)
- 팟캐스트 스튜디오 (기획, 녹음, 편집, 배포 가이드)
- 출판 기획 (원고 기획, 출판 프로세스, 전자책)
- 콘텐츠 캘린더 (월간/주간 콘텐츠 일정 수립)
- Remotion 영상 (React 기반 프로그래매틱 영상 생성)

## 트리거 키워드

영상, 유튜브, 팟캐스트, Remotion, 출판, 전자책, 콘텐츠 캘린더, 미디어 제작, 동영상

## 워크플로우

### 1단계: 영역 및 모듈 판별

사용자 요청을 분석하여 다음 영역 중 하나로 분류합니다.
- 유튜브 프로덕션
- 팟캐스트 스튜디오
- 출판 기획
- 콘텐츠 캘린더
- Remotion 영상

### 2단계: 해당 영역 기획 수행

#### 유튜브 프로덕션

**채널 기획**
```
채널 포지셔닝:
- 타겟 시청자: 직군, 관심사, 고통점
- 채널 카테고리: 교육 / 엔터 / 리뷰 / 브이로그 / B2B 전문
- 경쟁 채널 분석: 상위 3개 채널 포맷·주기·조회수 분석
- 차별화 포인트: "우리만의 고유한 관점"

채널 기본 설정:
- 채널 이름: 검색 가능 + 기억하기 쉬운 이름
- 아이콘·배너: 브랜드 일관성
- 소개 문구: 50자 내외, 구독 이유 명시
- 채널 트레일러: 60초 이내, 채널 가치 제안
```

**콘텐츠 전략**
```
업로드 주기: 주 1~2회 (알고리즘 안정화)
영상 길이: 10~20분 (광고 삽입 가능 + 시청 시간 최적)
콘텐츠 믹스:
- 80%: 핵심 시리즈 (SEO 최적화 주제)
- 15%: 시의성 콘텐츠 (트렌드 반응)
- 5%: 커뮤니티 콘텐츠 (Q&A, 브이로그)
```

**영상 구조 (10분 기준)**
```
0:00~0:30: 후킹 (Hook) — 시청자가 얻을 것 약속
0:30~1:00: 인트로 (채널 브랜딩)
1:00~8:30: 본문 (핵심 내용, 3~5개 섹션)
8:30~9:30: 요약 + 핵심 포인트 반복
9:30~10:00: CTA — "구독/좋아요/댓글 + 다음 영상 예고"
```

#### 팟캐스트 스튜디오

**기획 단계**
```
포맷 선택:
- 솔로: 전문 지식 공유, 스크립트 필요
- 인터뷰: 게스트 초청, 질문 리스트 준비
- 패널: 2~3인 토론, 주제 사전 합의

에피소드 구조 (30분 기준):
- 0~3분: 오프닝 + 이번 화 주제 소개
- 3~25분: 메인 내용 (3개 섹션)
- 25~28분: 핵심 요약
- 28~30분: CTA + 다음 화 예고
```

**배포 플랫폼**
- 팟빵 (국내 1위), 네이버 오디오클립 (국내 검색 연동)
- Spotify, Apple Podcasts (글로벌 확장)
- RSS 피드로 동시 배포 (Anchor/Buzzsprout 활용)

#### 콘텐츠 캘린더 설계 (월간)

```
월간 콘텐츠 캘린더 예시 (유튜브 주 1회 + 블로그 주 2회):

월: 블로그 — 트렌드 큐레이션 (SEO 타겟)
화: 블로그 — How-to 가이드 (SEO 타겟)
목: 유튜브 — 영상 업로드
금: SNS — 영상 클립 + 인사이트 공유

월말: 다음 달 캘린더 계획 + 성과 분석
```

#### Remotion 영상 제작

React 기반 프로그래매틱 영상 생성 프레임워크.
- 데이터 시각화 영상, 모션 그래픽, 자동화 영상 제작
- 참조 가이드: `references/remotion-video/guide.md`

## 주의사항

### AI 생성 콘텐츠 주의

AI가 생성한 미디어 기획안과 스크립트는 사실 확인 후 사용하세요.
특히 업계 통계, 인물 인용, 영상 내 수치는 반드시 원출처를 검증하세요.

### 문제 해결

- **플랫폼 규격 변경**: 유튜브·팟빵·Spotify의 업로드 규격과 알고리즘은 수시로 변경됩니다. 최신 정보는 각 플랫폼 공식 Creator 채널에서 확인하세요.
- **브랜드 가이드 미제공**: 채널 톤앤매너, 로고, 색상 가이드를 제공하면 더 일관된 미디어 기획안을 작성할 수 있습니다.
- **이미지 생성 실패**: 썸네일 디자인은 Canva, Adobe Express를 활용하거나 `card-news` 스킬의 이미지 생성 기능을 사용하세요.
- **Remotion 빌드 오류**: Node.js 버전, @remotion/cli 버전 충돌이 흔합니다. `package.json`의 Remotion 패키지 버전을 일치시키고 `npx remotion check`를 실행하세요.
