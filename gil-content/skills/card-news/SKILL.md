# 카드뉴스 (Card News)

## 개요

한국 SNS 환경(인스타·스레드·카카오 채널)을 위한 4장 카드뉴스 풀스택 기획 스킬입니다. **10가지 구성 패턴** 중 주제에 가장 적합한 것을 자동 선택하고, **본문 카피·디자인 가이드·이미지 프롬프트·SNS 캡션**을 묶어서 출력합니다. 실제 이미지 렌더링은 `gil-media`의 이미지 프롬프트 빌더가 담당합니다.

## 트리거 키워드

카드뉴스, 캐러셀, 인스타그램 카드, 스레드 카드, 슬라이드 콘텐츠, AI 이미지 카드, 인포그래픽 카드, SNS 카드, 4장 카드, 카카오 채널 카드

## 10가지 구성 패턴

주제에서 자동으로 1개 패턴을 선택합니다. 매칭이 모호하면 AskUserQuestion으로 후보 3-4개를 제시해 사용자가 확정합니다.

| # | 패턴 | 핵심 의도 | 자주 매칭되는 표현 |
|---|---|---|---|
| 1 | **A·B 듀얼 선택** | 두 옵션 중 어느 쪽이 맞는가 | "A vs B", "차이", "어느 쪽이" |
| 2 | **순차 빌드** | 차근차근 한 단계씩 | "방법", "순서", "단계", "어떻게" |
| 3 | **체크박스 점검** | 놓치면 안 되는 N가지 | "체크리스트", "준비물", "필수" |
| 4 | **궁금증 해소** | 자주 묻는 질문 묶음 | "FAQ", "궁금", "헷갈리는" |
| 5 | **함정 회피** | 흔히 빠지는 함정과 회피법 | "실수", "주의", "피해야 할" |
| 6 | **첫 발 가이드** | 완전 초보자의 첫 한 걸음 | "처음", "입문", "초보자", "시작" |
| 7 | **개념 사전** | 핵심 용어 한 번에 정리 | "용어", "개념", "뜻", "정리" |
| 8 | **페인 → 솔루션** | 문제 인식 → 원인 → 해결 | "[문제] 어떻게", "해결", "막힐 때" |
| 9 | **실전 사례** | 현장에서 어떻게 적용되는가 | "예시", "사례", "실전" |
| 10 | **즉시 활용 키트** | 복사·붙여넣기 템플릿 | "템플릿", "양식", "복사" |

상세 패턴별 본문 골격과 프롬프트는 [`references/prompt-templates.md`](./references/prompt-templates.md)에 있습니다.

## 워크플로우

### 1단계 — 주제·맥락 파악

사용자 한 줄 입력에서 다음을 자동 추출합니다.

- 주제 (예: "재택근무 생산성 도구 비교")
- 목표 (정보 전달 · 인지 확장 · 리드 수집 · 제품 소개)
- 타겟 (직장인 · 창업자 · 학생 · 일반 대중)
- 채널 (인스타 · 스레드 · 카카오 채널 · 페이스북)
- 길이 (4장 기본 · 7장 확장 · 10장 시리즈)

부족한 정보는 AskUserQuestion으로 1-2 라운드 보완.

### 2단계 — 구성 패턴 자동 선택

위 10가지 표의 키워드 매칭으로 가장 적합한 패턴 1개를 선택. 키워드 점수가 동률이거나 약한 경우 사용자에게 후보 3-4개를 제시합니다.

### 3단계 — 4장 본문 골격 설계

선택된 패턴의 본문 골격을 그대로 채워 4장을 구성합니다. 각 장은 **한 메시지 · 헤드라인 1줄 · 본문 15-30자 · 시각 요소 1개**로 제한됩니다.

```
1장 (표지): 후킹 · 한 줄 핵심 · 대표 비주얼
2장 (전개): 본격 내용 1차 — 패턴별 핵심
3장 (확장): 본격 내용 2차 — 깊이·예시
4장 (마무리): 요약 + 단일 CTA
```

### 4단계 — 디자인 톤 선택

사용자가 명시하지 않으면 다음 5가지 중 자동 추천:

| 톤 이름 | 컬러 | 어조 | 권장 주제 |
|---|---|---|---|
| **Soft Cream** (기본) | 크림 베이지 + 따뜻한 오렌지 포인트 | 친근·따뜻함 | 라이프스타일·일반 정보 |
| **Claude Modern** | Light Beige `#faf9f5` + Anthropic Orange `#d97757` | 신뢰감·세련됨 | 테크·B2B·도구 소개 |
| **Corporate Trust** | 화이트 + 진한 네이비 | 격식·전문성 | 금융·법무·임원 보고 |
| **Playful Pop** | 파스텔 핑크 + 옐로 | 활기·친근함 | 뷰티·아동·이벤트 |
| **Bold Dark** | 차콜 #1a1a1a + 형광 오렌지 | 모던·하이엔드 | 개발자·디자이너·키노트 |

### 5단계 — 통합 프롬프트 생성

본문 골격 + 디자인 톤 + 한국어 가독성 가이드 + 마무리 꿀팁을 하나로 묶어 **이미지 생성 도구에 그대로 붙여 넣을 수 있는 단일 프롬프트**를 출력합니다. 형식은 [`references/prompt-templates.md`](./references/prompt-templates.md)의 통합 템플릿을 따릅니다.

### 6단계 — 이미지 생성 위임

`gil-media` 플러그인의 이미지 프롬프트 빌더에 통합 프롬프트를 전달합니다.

| 위임 스킬 | 모델 | 추천 시점 |
|---|---|---|
| **`gil-media:higgsfield-image`** | **GPT Image 2** (1순위) | **텍스트가 정확히 들어가는 카드뉴스 ★ 권장** — MCP 자동 호출 |
| `gil-media:higgsfield-image` | Nano Banana Pro | 보조·fallback |
| `gil-media:higgsfield-image` | Soul Cinema | 시네마틱 카드 (글자 비중 낮을 때) |
| `gil-media:gpt-image-2-prompt` | OpenAI GPT-image-2 | ChatGPT 외부 도구로 복붙 (MCP 미사용 환경) |
| `gil-media:gemini-3-image-prompt` | Google Gemini 3 Image | Google AI Studio 외부 도구로 복붙 |
| `gil-media:midjourney-v8-prompt` | Midjourney v8.1 | Discord MJ 사용 (비주얼 임팩트) |

해상도·비율 미지정 시 사용자 확인. 인스타는 1:1 (1080×1080) 권장.

### 7단계 — 카피 검수 체이닝

- AI 슬롭 표현 회피: `gil:ai-slop-reviewer` 또는 `gil-content:humanize-korean`
- 사실 확인: 통계·수치·인물 정보는 원출처 검증 필수
- 한국어 가독성: 한 줄 24자 이내·줄간격 1.5배

### 8단계 — 채널별 산출물

- 카드 4장 텍스트 (헤드라인·본문)
- 디자인 가이드 (레이아웃·폰트·컬러·여백)
- AI 이미지 생성용 통합 프롬프트
- 채널별 캡션 (인스타·스레드·카카오 채널 각 톤 다름)
- 해시태그 5-10개 (대·중·소 키워드 혼합)

## 한국 SNS 베스트 프랙티스

### 1초 시선 고정 — 표지 4유형

| 유형 | 예시 카피 | 어울리는 패턴 |
|---|---|---|
| 질문 던지기 | "혹시 이런 경험 있나요?" | 페인 → 솔루션 · 함정 회피 |
| 통계 충격 | "한국인 78%가 모르는" | 첫 발 가이드 · 개념 사전 |
| 약속 미리 보기 | "5분 뒤, [결과]를 손에 쥡니다" | 순차 빌드 · 즉시 활용 키트 |
| 비주얼 한 방 | 색·이미지로 시선 잡기 | 모든 패턴 |

### 흐름의 리듬 (긴장 → 완화 → 임팩트)

```
1장 — 긴장: 문제·페인·질문으로 끌어들임
2-3장 — 완화: 친절한 설명·분석·해결책
4장 — 임팩트: 핵심 요약 + 단일 CTA
```

### 한 카드 한 메시지 — 원칙

- 한 카드 = 한 헤드라인 + 한 단락 (15-30자)
- 이미지와 텍스트를 동시에 강조하면 충돌 → 하나에만 무게
- 글자 크기 ≥ 24pt (모바일 가독)
- 여백 30% 이상 — 꽉 차면 읽지 않음

### 한국어 가독성 핵심

| 항목 | 기준 |
|---|---|
| 한 줄 길이 | 한국어 24자 이내 (모바일) |
| 단어 선택 | 한자어 ↓ · 우리말 ↑ |
| 줄간격 | 1.5배 이상 |
| 색 대비 | WCAG AA 4.5:1 이상 |
| 폰트 | Pretendard · Noto Sans KR (오픈) |
| 강조 우선순위 | Bold > 색 > 크기 |

### 마지막 카드 CTA — 6가지 (한 카드뉴스 = 1 CTA)

| CTA | 어울리는 패턴 | 효과 |
|---|---|---|
| 저장 | 체크박스 점검 · 개념 사전 · 즉시 활용 키트 | 알고리즘 상승 |
| 공유 | 함정 회피 · 실전 사례 · 페인 → 솔루션 | 도달 확장 |
| 팔로우 | 첫 발 가이드 · 시리즈 약속 | 구독 누적 |
| 링크 | 제품·블로그·랜딩 전환 | 클릭 |
| 댓글 | 궁금증 해소 · A·B 듀얼 선택 | 참여 |
| DM | 1:1 상담·견적 문의 | 리드 |

여러 CTA를 동시에 노출하면 사용자가 선택 피로로 아무 행동도 하지 않습니다. **한 카드뉴스 1 CTA** 원칙.

## 사용 예시

- "재택근무 생산성 도구 5가지 비교" → A·B 듀얼 선택 + Claude Modern 톤
- "처음 시작하는 사이드 프로젝트" → 첫 발 가이드 + Soft Cream
- "스타트업 시드 라운드 준비 체크리스트" → 체크박스 점검 + Corporate Trust
- "프롬프트 엔지니어링 자주 묻는 질문" → 궁금증 해소 + Claude Modern
- "인플루언서 마케팅 흔한 함정" → 함정 회피 + Playful Pop
- "AI 디자인 도구 핵심 용어 정리" → 개념 사전 + Bold Dark

## 주의사항

### AI 생성 콘텐츠 사실 확인

- AI가 만든 통계·수치·인물·인용 정보는 **반드시 원출처 검증** 후 게시
- 출처가 불확실한 내용은 "예시", "가설" 등으로 명시
- AI 슬롭 표현 (혁신적인·차세대·재정의하는 등) 회피 → `humanize-korean` 체이닝 권장

### 분량 초과 시

- 한 카드 30자 초과: 핵심 키워드만 남기고 나머지는 SNS 캡션·발표자 노트로 이동
- 4장으로 다 못 담는 주제: 7장 또는 10장 시리즈로 확장 후 시리즈 톤 통일

### 플랫폼 규격

- 인스타 캐러셀: 1:1 (1080×1080) 권장, 최대 10장
- 인스타 스토리: 9:16 (1080×1920)
- 카카오 채널: 1:1 또는 4:5
- 스레드: 1:1 또는 4:5
- 최신 규격은 각 플랫폼 공식 채널에서 확인

### 폰트 라이선스

- Pretendard · Noto Sans KR: 오픈소스 (상업·배포 자유)
- 굴림 · 맑은 고딕: Windows 환경 한정
- 외부 폰트 사용 시 라이선스 확인

### 시리즈 운영

여러 편을 시리즈로 운영할 때:
- 톤앤매너(색·폰트·박스 모양) 동일
- 페이지 번호 위치·크기 통일
- 마스코트 사용 시 모든 카드에 일관 배치
- 시리즈 표지 패턴 정의 (예: 좌상단 시리즈 번호)
