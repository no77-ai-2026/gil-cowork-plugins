# 카드뉴스 본문 골격 · 통합 프롬프트 · 디자인 톤

`card-news` 스킬이 내부적으로 사용하는 본문 골격 모음과 이미지 생성 도구에 그대로 입력할 수 있는 통합 프롬프트 형식을 정리합니다.

## 1. 10가지 본문 골격

각 패턴은 4장 분량을 기본으로 합니다. 7장·10장 시리즈는 같은 골격을 2장씩 확장합니다.

### 1) A·B 듀얼 선택

```
1장 (표지): 두 옵션 동시 노출 — "[A]와 [B], 우리에게 맞는 건?"
2장: A의 강점·약점·어울리는 상황
3장: B의 강점·약점·어울리는 상황
4장: 결정 가이드 — 어느 쪽을 언제 고를지 1-2문장
```

핵심 시각: 좌우 분할 또는 상하 분할 + 비교표 1개

### 2) 순차 빌드

```
1장 (표지): "[주제]를 [N] 단계로" — 전체 흐름 미리보기
2장: 1-3단계 — 각 단계 한 줄 + 핵심 행동
3장: 4-N단계 — 같은 형식
4장: 전체 요약 + 가장 자주 막히는 지점·해결 팁
```

핵심 시각: 화살표 또는 번호 박스 흐름

### 3) 체크박스 점검

```
1장 (표지): "[주제] 점검 N가지" — 놓치지 말 것 어필
2장: 항목 1-5 (각 한 줄)
3장: 항목 6-N (각 한 줄)
4장: 가장 중요한 3가지만 강조 + 보너스 한 줄
```

핵심 시각: 체크박스·아이콘 그리드

### 4) 궁금증 해소

```
1장 (표지): "[주제] 헷갈리는 N가지" — 가장 자주 받는 질문 미리보기
2장: Q1·Q2 — 짧은 답변 (각 2-3줄)
3장: Q3·Q4 — 같은 형식
4장: 핵심 한 줄 결론 + 더 깊은 학습 안내
```

핵심 시각: 질문은 굵게, 답변은 박스 안에

### 5) 함정 회피

```
1장 (표지): "[주제] 흔한 함정" — 이미 빠진 경험 환기
2장: 함정 1-3 (왜 빠지는지 한 줄)
3장: 함정 4-N + 회피하는 구체 행동
4장: 올바른 방향 요약 + 미리 점검할 한 가지
```

핵심 시각: 경고 아이콘 + 회피 아이콘 대비

### 6) 첫 발 가이드

```
1장 (표지): "처음 [주제]를 시작한다면" — 초보자 위치 인정
2장: 무엇인지 한 줄 정의 + 잘못 알려진 오해 1개
3장: 왜·언제·어디에 쓰는지
4장: 오늘 당장 할 수 있는 첫 행동 1가지
```

핵심 시각: 친근한 일러스트·아이콘 + 풍부한 여백

### 7) 개념 사전

```
1장 (표지): "[주제] 핵심 용어 N개"
2장: 용어 1-3 — "쉽게 말하면" 형식 (한 줄 정의)
3장: 용어 4-N — 같은 형식
4장: 용어 간 관계도 + 한 줄 요약
```

핵심 시각: 용어 카드 + 관계도(점선·화살표)

### 8) 페인 → 솔루션

```
1장 (표지): "[페인 상황]" — 공감 한 줄
2장: 왜 이런 문제가 생기는지 — 원인 2-3가지
3장: 어떻게 해결하는지 — 행동 가능한 단계 3-5개
4장: 즉시 적용 팁 + 다음에 같은 문제가 생기지 않으려면
```

핵심 시각: 페인은 어둡게, 솔루션은 밝게 (톤 대비)

### 9) 실전 사례

```
1장 (표지): "[주제] 현장에서는" — 호기심 자극
2장: 기본 개념 한 문단 + 적용 가능한 영역
3장: 실제 사례 1-2개 (수치·결과 강조)
4장: 따라 해볼 수 있는 미니 적용 + 한 줄 요약
```

핵심 시각: Stat callout (큰 숫자) · 실제 결과 차트

### 10) 즉시 활용 키트

```
1장 (표지): "[주제] 복사·붙여넣기 키트" — 즉시 사용 어필
2장: 템플릿 1개 — 그대로 쓸 수 있는 형식
3장: 템플릿 2개 — 다른 상황용
4장: 적용 팁 + 자주 바꿔야 하는 부분 안내
```

핵심 시각: 모노스페이스 폰트 또는 박스 처리 (코드처럼 보이게)

## 2. 디자인 톤 — 5가지 컬러·폰트 토큰

### Soft Cream (기본)

```
배경: 크림 베이지 (#f8f3e8 ~ #faf6ec)
포인트: 따뜻한 오렌지 (#e69150 ~ #d67740)
보조: 부드러운 갈색 (#8a6850)
텍스트: 다크 그레이 (#2a2a2a)
폰트: Pretendard Bold 헤딩 + Pretendard Regular 본문
어조: 친근·따뜻함
권장 주제: 라이프스타일·일반 정보·교육
```

### Claude Modern

```
배경: Light Beige (#faf9f5)
포인트: Anthropic Orange (#d97757)
보조: Anthropic Blue (#6a9bcc)
텍스트: Dark (#141413)
폰트: Pretendard Bold 헤딩 + Pretendard Regular 본문
어조: 신뢰감·세련됨·테크
권장 주제: 테크·B2B·도구·SaaS
```

### Corporate Trust

```
배경: White (#ffffff)
포인트: Deep Navy (#1d3557)
보조: Gold (#c9a961)
텍스트: Dark (#141413)
폰트: 맑은 고딕 또는 Pretendard Bold 헤딩
어조: 격식·전문성·신뢰
권장 주제: 금융·법무·임원 보고·B2B 정형
```

### Playful Pop

```
배경: 파스텔 핑크 (#ffe5ec) 또는 베이지
포인트: 옐로 (#ffd166) · 핑크 (#ef476f)
보조: 청록 (#06d6a0)
텍스트: 다크 그레이 (#2a2a2a)
폰트: Pretendard SemiBold + 캐주얼 굵은 헤딩
어조: 활기·친근함
권장 주제: 뷰티·아동·이벤트·캠페인
```

### Bold Dark

```
배경: 차콜 (#1a1a1a)
포인트: 형광 오렌지 (#ff6b35)
보조: 라이트 그레이 (#888888)
텍스트: Light (#faf9f5)
폰트: Pretendard Bold 헤딩 + JetBrains Mono 또는 D2Coding (코드형 강조 시)
어조: 모던·하이엔드
권장 주제: 개발자·디자이너·키노트·테크 컨퍼런스
```

## 3. 통합 프롬프트 — 이미지 도구에 그대로 입력

다음 형식을 사용자 입력으로 채워 이미지 생성 도구(Nano Banana Pro · DALL-E · Midjourney 등)에 그대로 붙여 넣습니다.

```
[주제] 카드뉴스 4장을 만들어 줘. 인스타그램 1:1 (1080×1080).

장별 구성:
1장 (표지)
- 헤드라인: [한 줄, 24자 이내]
- 한 줄 핵심: [부제]
- 시각: [대표 이미지·아이콘 묘사]

2장 ([패턴별 역할])
- 헤드라인: [한 줄]
- 본문 15-30자: [내용]
- 시각: [이미지·아이콘]

3장 ([패턴별 역할])
- 헤드라인·본문·시각 (같은 형식)

4장 (요약·CTA)
- 헤드라인: [한 줄 요약]
- CTA: [저장 / 공유 / 팔로우 / 링크 / 댓글 / DM 중 1개]
- 시각: [CTA 강조 배지·버튼]

디자인 가이드:
- 컬러: [선택 톤의 배경 + 포인트]
- 폰트: Pretendard 굵은 헤딩 + 본문, 한국어 가독성 우선
- 레이아웃: 둥근 박스, 30% 이상 여백, 페이지 번호 우측 하단 작게
- 외곽선·박스 테두리 굵기 일관
- 모바일 가독성 — 본문 최소 24pt 환산
- 4장 시리즈 톤앤매너 통일

마무리 지시:
- 텍스트는 정확히 렌더링 (오타·왜곡 없이)
- 정보가 많아도 시각적으로 복잡해 보이지 않게
- AI 슬롭 표현 회피 — "혁신적인" "차세대" "재정의하는" 같은 진부 표현 X
- 한 카드 한 메시지 — 여러 정보를 한 카드에 욱여넣지 않음
```

## 4. 마무리 꿀팁 모음

통합 프롬프트 맨 뒤에 1-3개 골라 추가하면 결과 품질이 올라갑니다.

```
- 텍스트는 실제 SNS 카드뉴스처럼 정확히 렌더링해 — 오타·왜곡 없이
- 4장 시리즈 톤앤매너 통일 — 색·폰트·박스 모양 동일
- 정보량이 많아도 시각적으로 복잡해 보이지 않게 정리
- 저장하고 싶은 정보형 카드뉴스 느낌으로
- 한 카드 한 메시지 원칙 — 욱여넣지 않음
- 한국어 가독성 우선 — 한 줄 24자 이내
- 색 대비 4.5:1 이상 (WCAG AA)
- 페이지 번호 위치·크기 모든 카드에 일관
- 외곽선·박스 테두리 두께 일관
- AI 슬롭 표현 회피 — 진부한 한국어·영어 단어 사용 금지
- 마스코트가 있다면 모든 카드에 동일 위치
```

## 5. 채널별 캡션 가이드

같은 카드뉴스라도 채널마다 캡션 톤이 달라야 합니다.

### 인스타그램

```
- 첫 줄에 후킹 (질문·통계·약속)
- 줄바꿈 자주 — 모바일 가독성
- 이모지는 핵심 단어 옆에 1-2개
- 본문 끝에 CTA + 해시태그 5-10개
- "더 보기" 펼침 전에 핵심이 보이게
```

### 스레드

```
- 짧고 직설적 (트위터 톤)
- 첫 문장에서 결론을 미리 노출
- 이모지는 최소
- 후속 reply로 상세 풀어쓰기
- 해시태그 1-2개
```

### 카카오 채널

```
- 격식 살짝 — "~합니다" 종결
- 본문 4-6 문단 (좀 더 긴 호흡)
- CTA는 채널 내 액션 (이벤트·구독)
- 해시태그 없음 또는 1-2개만
```

### 페이스북

```
- 본문 2-3 문단 (중간 길이)
- 이모지 자유롭게 (인스타보다 많이)
- CTA는 외부 링크 가능
- 해시태그 3-5개
```

## 6. 사용 흐름 요약

```
주제 입력
  ↓
구성 패턴 자동 선택 (10가지 중 1)
  ↓
4장 본문 골격 채움
  ↓
디자인 톤 선택 (5가지 중 1)
  ↓
통합 프롬프트 생성
  ↓
gil-media 이미지 프롬프트 빌더로 위임
  ↓
카피 검수 (humanize-korean · ai-slop-reviewer)
  ↓
채널별 캡션 + 해시태그 추가
  ↓
시각 검수 → 게시
```

## 7. 분량별 확장

| 분량 | 적합 시점 | 골격 확장 방식 |
|---|---|---|
| **4장** (기본) | 단일 주제·간단 정보 | 그대로 |
| **7장** | 중간 깊이·예시 추가 | 2-3장과 3장을 각 2장으로 확장 |
| **10장** (시리즈) | 깊은 가이드·튜토리얼 | 각 본문 장을 2장으로 확장 + 인덱스 추가 |

10장 시리즈는 1장에 "시리즈 1/N" 표시. 시리즈 전체 톤앤매너 통일.

## 8. 다른 스킬과의 연계

| 스킬 | 시점 |
|---|---|
| `gil-media:gemini-3-image-prompt` | 통합 프롬프트 → 실제 이미지 생성 (★ 권장) |
| `gil-media:gpt-image-2-prompt` | 일러스트 강조 시 |
| `gil-media:midjourney-v8-prompt` | 비주얼 임팩트 중심 |
| `gil-content:humanize-korean` | 한국어 카피 AI 티 자연화 (검수 단계) |
| `gil:ai-slop-reviewer` | 전반 AI 슬롭 표현 검수 |
| `gil-marketing:sns-content` | 채널 단위 운영 전략 |
| `gil-marketing:campaign-planner` | 캠페인 시리즈 카드뉴스 |
| `gil-design:claude-design-slop-check` | 디자인 카피 슬롭 검수 |
