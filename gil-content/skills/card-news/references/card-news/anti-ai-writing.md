# AI 문체 회피 가이드 — 한국어 카드뉴스용

카드뉴스 텍스트가 "AI가 쓴 티"가 나면 신뢰도가 떨어진다.
한국인 카피라이터·마케터가 직접 쓴 것처럼 들리는 문체를 지향한다.

---

## 1. 한국어 AI 문체의 대표적 특징

AI가 한국어로 글을 쓸 때 반복적으로 나타나는 패턴:

### 어미 단조로움
- "~합니다", "~됩니다"가 연속으로 3회 이상 반복
- "~하세요", "~해보세요"의 과도한 사용
- 모든 문장이 비슷한 길이와 리듬

### AI 습관어 (한국어)
절대 사용하지 말 것:

| AI 습관어 | 대체 표현 |
|----------|----------|
| 확실히 | 분명, 틀림없이 |
| 전반적으로 | 대체로, 전체적으로 보면 |
| 일반적으로 | 보통, 흔히 |
| ~의 세계로 빠져보세요 | (삭제하거나 구체적 행동으로 대체) |
| 혁신적인 | 새로운, 기존에 없던 |
| 완벽한 가이드 | 핵심 정리, 실전 가이드 |
| 꼭 알아야 할 | (삭제 — 독자가 판단할 일) |
| ~에 대해 알아보겠습니다 | (삭제 — 바로 본론으로) |
| 여러분 | (삭제 또는 "당신"으로 대체) |
| ~하는 것이 중요합니다 | (구체적 이유나 결과로 대체) |
| 다양한 | 구체적 숫자나 예시로 대체 |
| 효과적인/효율적인 | 구체적 결과로 대체 |
| 핵심적인/필수적인 | (맥락에 따라 삭제 가능) |
| 궁극적으로 | 결국, 끝내 |
| ~할 수 있습니다 | ~된다, ~가능하다 |

### AI 습관어 (영어→한국어 번역투)
- "~의 잠재력을 열어보세요" (Unlock the potential)
- "~의 최전선에서" (At the forefront)
- "~의 여정을 시작하세요" (Embark on a journey)
- "게임체인저" (Game-changer)
- "~를 극대화하세요" (Maximize)
- "빠르게 변화하는 환경에서" (In the ever-evolving landscape)

### 구조적 패턴
- 서론→본론→결론의 교과서적 3단 구성
- "첫째, 둘째, 셋째" 나열 후 "이처럼" 요약
- 모든 단락이 비슷한 길이 (3-4문장)
- 접속사의 과다 사용: "또한", "그리고", "따라서", "그러므로"

## 2. 사람이 쓴 것처럼 만드는 기법

### 문장 길이 불규칙하게
```
나쁜 예 (AI):
"이 방법은 매우 효과적입니다. 많은 사람들이 이 방법을 활용하고 있습니다.
특히 초보자에게 추천드립니다. 누구나 쉽게 따라할 수 있습니다."

좋은 예 (사람):
"이 방법, 생각보다 잘 먹힌다.
특히 막 시작한 사람한테. 3분이면 된다."
```

### 어미 변화 주기
한 슬라이드 안에서 어미를 섞는다:
- "~한다" (단정)
- "~이다" (설명)
- "~일까?" (질문)
- "~하지 마라" (경고)
- "~뿐이다" (한정)
- 체언 종결: "핵심은 타이밍." (명사로 끝내기)

### 구어체 섞기
- "근데" (그런데 대신)
- "좀" (약간/다소 대신)
- "진짜" (정말/매우 대신)
- "딱" (정확히 대신)
- "한마디로" (요약하면 대신)

### 군더더기 삭제
```
삭제 대상:
- "~인 것 같다" → "~이다"
- "~하게 된다" → "~한다"
- "~할 수 있게 된다" → "~된다"
- "~라고 할 수 있다" → "~이다"
- "의" 남용 → "브랜드의 성장의 과정" → "브랜드 성장 과정"
```

### 감탄사·형용사 절제
```
나쁜 예: "정말 놀라운 결과입니다! 매우 효과적이고 혁신적인 방법이에요!"
좋은 예: "매출이 2배 올랐다. 광고비는 그대로."
```

## 3. 카드뉴스 전용 문체 규칙

### 표지 (1장)
- 10자 이내가 이상적. 길어도 20자 미만.
- 숫자 포함 시 아라비아 숫자 사용 ("세 가지" → "3가지")
- 물음표나 마침표 하나만. 느낌표 금지.

### 본문 슬라이드
- 제목: 명사형 종결 또는 단정형 ("~하는 법" / "~이 답이다")
- 본문: 1문장 = 1줄. 줄바꿈으로 호흡 조절.
- 숫자·데이터는 볼드 또는 대형 폰트로 강조

### 마무리 슬라이드
- "도움이 되셨나요?" 같은 AI식 마무리 금지
- 대신: 체크리스트, 한 줄 요약, 또는 여운 있는 문장

## 4. 자가 점검 체크리스트

텍스트 작성 후 아래를 확인:

- [ ] "~합니다"가 연속 3회 이상 반복되지 않는가?
- [ ] AI 습관어(확실히, 전반적으로, 혁신적인 등)가 포함되지 않았는가?
- [ ] 모든 문장이 비슷한 길이가 아닌가? (짧은 문장과 긴 문장이 섞여 있는가?)
- [ ] 느낌표를 과도하게 사용하지 않았는가? (최대 1개)
- [ ] "~해보세요", "~하세요"가 3회 이상 반복되지 않는가?
- [ ] 구체적인 숫자나 사실이 포함되어 있는가?
- [ ] 번역투 표현("~의 잠재력", "~의 여정")이 없는가?
- [ ] 마무리가 "도움이 되셨나요?" 같은 AI식이 아닌가?
