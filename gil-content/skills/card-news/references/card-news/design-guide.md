# 인스타그램 카드뉴스 디자인 가이드 v2.0

## 색상 이론

### 기본 원칙
카드뉴스에서 색상은 "정보의 계층"을 만든다. 색상이 너무 많으면 독자의 시선이 분산되어 핵심 메시지를 놓친다.

**규칙: 3색 법칙**
- 배경색 (60%): 눈에 편안한 색
- 메인 텍스트/요소색 (30%): 배경과 대비되는 색
- 포인트색 (10%): 강조할 때만 사용

### 주제별 추천 팔레트

| 분야 | 배경 | 메인 | 포인트 | 분위기 |
|------|------|------|--------|--------|

### 대비 규칙
- 흰 배경 + 검정 텍스트: 항상 안전
- 어두운 배경 + 흰 텍스트: 임팩트 있음
- 컬러 배경 + 흰 텍스트: 포인트용으로만
- 절대 금지: 빨강 배경 + 녹색 텍스트 (색맹 고려)

## 폰트 가이드

### 필수 폰트: Pretendard

```html
<link rel="stylesheet" as="style" crossorigin
  href="https://cdn.jsdelivr.net/gh/orioncactus/pretendard@v1.3.9/dist/web/variable/pretendardvariable-dynamic-subset.min.css" />
```

```css
body {
  font-family: 'Pretendard Variable', 'Pretendard', 'Noto Sans KR', -apple-system, sans-serif;
}
```

### 폴백: Google Fonts (Pretendard 로드 실패 시)
```css
/* 기본 정보 전달형 */
@import url('https://fonts.googleapis.com/css2?family=Noto+Sans+KR:wght@400;700;900&display=swap');

/* 감성/매거진형 — 명조체 포인트 */
@import url('https://fonts.googleapis.com/css2?family=Nanum+Myeongjo:wght@400;700&display=swap');
```

### 폰트 크기 체계 (1080px 기준)
```
대제목 (표지 메인): 80-120px, weight 900
소제목 (각 슬라이드 제목): 56-72px, weight 700
본문: 36-44px, weight 400-500
캡션/부연: 28-32px, weight 400
슬라이드 번호: 24px, weight 400
```

### 행간(line-height)
- 제목: 1.2-1.3
- 본문: 1.6-1.8

## 스와이프 심리학 상세

### 오픈 루프 기법
인간의 뇌는 완결되지 않은 것에 집착한다 (자이가르닉 효과).

```
좋은 예: "근데 진짜 이유는 따로 있다 →"
나쁜 예: "이상으로 세 가지 방법을 알아봤습니다."
```

### 진행바 심리학
독자가 "얼마나 남았는지" 알 때 완독률이 40% 이상 증가한다.

```css
.progress-bar {
  width: calc(var(--current) / var(--total) * 100%);
  height: 6px;
  background: var(--accent-color);
  border-radius: 3px;
}
```

### 시각적 연속성
슬라이드 간 연결감:
- 색상 블록이 다음 슬라이드로 "넘어가는" 느낌
- 일관된 레이아웃 그리드 유지
- 슬라이드 번호로 진행감 표현

## 저장률을 높이는 콘텐츠 유형

알고리즘이 선호하는 콘텐츠:
1. **체크리스트형**: "□ 이것도 확인했나요?" — 저장하고 나중에 체크
2. **가이드/튜토리얼형**: 순서가 있는 방법 — 참고용으로 저장
3. **비교표형**: A vs B — 나중에 결정할 때 다시 보기
4. **통계/데이터형**: 숫자가 있는 정보 — 공유하고 싶다
5. **공감형**: "맞아, 이거 나 얘기잖아" — 친구에게 공유

## 해시태그 전략

### 추천 조합 공식
```
브랜드태그 1개 + 대형태그 2개 + 중형태그 6개 + 소형태그 4개 = 총 13개
```

### 주제별 필수 해시태그

**정보/교육:** `#정보 #꿀팁 #알아두면좋은것 #지식공유`
**라이프스타일:** `#일상 #라이프스타일 #오늘의일상 #소소한일상`
**비즈니스/마케팅:** `#마케팅 #비즈니스 #자기계발 #성장`
**건강/음식:** `#건강 #건강한식습관 #오늘의식단 #헬시라이프`
**카드뉴스 공통:** `#카드뉴스 #인포그래픽 #정보그램`
