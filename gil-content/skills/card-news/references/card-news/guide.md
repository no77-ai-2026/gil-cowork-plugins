# 인스타그램 카드뉴스 생성기 (Instagram Card News Creator)

텍스트와 이미지를 조합한 카드뉴스(이미지 기반 뉴스)를 자동 생성하여 인스타그램, 카카오톡 채널에 배포합니다.

## Overview

소상공인/매체/교육업체의 정보를 **카드뉴스** 형식으로 시각화합니다.
한 장의 이미지에 1-2문장의 텍스트 + 주요 정보를 배치하여 **빠른 이해**와 **높은 공유성**을 실현합니다.

> **핵심**: 사용자가 "프리랜서 절세 팁 5가지 카드뉴스 만들어줘"라고 말하면,
> 5장의 고품질 카드뉴스(이미지 파일) + 인스타그램 캡션을 즉시 받을 수 있어야 합니다.

---

## 카드뉴스의 특징

### 구조
```
한 장당 1개 정보 (수직 직사각형)
상단: 카테고리 라벨
중앙: 큰 텍스트 + 핵심 시각
하단: 로고 + 출처
```

### 최적화된 크기 (인스타그램 기준)
| 플랫폼 | 크기 | 비율 | 특징 |
|--------|------|------|------|
| 인스타 피드 | 1080×1350 | 4:5 | 표준, 가로 여백 적음 |
| 인스타 릴스 | 1080×1920 | 9:16 | 풀스크린 스토리 |
| 카카오톡 | 540×810 | 2:3 | 모바일 채팅 최적화 |

## 카드뉴스 유형별 템플릿

### 1. 정보형 (Educational)
```
주제: 팁, 가이드, 상식
예: "프리랜서 절세 5가지", "봄 피부관리 팁"

구조:
- 상단: 번호 (1/5, 2/5, ...)
- 중앙: 제목 1줄 + 설명 1~2줄
- 시각: 아이콘 또는 일러스트
- 색상: 밝은 배경 + 진한 텍스트

레이아웃: 좌측 30% 시각, 우측 70% 텍스트
```

### 2. 뉴스형 (News/Article)
```
주제: 속보, 트렌드, 소식
예: "정부 지원금 신청 기한 연장", "새로운 법규 안내"

구조:
- 상단: 날짜 + 카테고리
- 중앙: 헤드라인 + 요약
- 하단: 더보기 링크 또는 연락처
- 색상: 매체 브랜드 색상 활용

레이아웃: 풀 너비 텍스트
```

### 3. 제품 쇼케이스 (Product)
```
주제: 신상품, 프로모션, 할인
예: "2024 신상 출시!", "봄 한정 세일"

구조:
- 중앙: 제품 사진 (큼)
- 상단: 제품명
- 하단: 가격, 할인율, 구매 링크
- 색상: 제품 색상에 맞춘 그래디언트

레이아웃: 60% 제품 이미지, 40% 정보
```

### 4. 비포/애프터 (Before/After)
```
주제: 변신, 효과, 개선
예: "홈트 30일 후", "시술 전후"

구조:
- 좌측: Before 이미지
- 우측: After 이미지
- 중앙: 텍스트 오버레이 ("→ 변화")
- 색상: 대비 강한 색상

레이아웃: 50:50 분할
```

### 5. 통계/데이터 (Statistics)
```
주제: 조사 결과, 통계, 트렌드
예: "직장인 스트레스 요인 TOP3", "2024 소비 트렌드"

구조:
- 상단: 제목 + 데이터 출처
- 중앙: 차트 (막대, 원형, 꺾은선)
- 하단: 해석 또는 결론
- 색상: 차트 색상 팔레트

레이아웃: 데이터 시각화 중심
```

## 디자인 시스템

### 색상 팔레트
```
기본 팔레트:
- 메인: 브랜드 색상 1개 (Primary)
- 보조: 보색 또는 유사색 (Secondary)
- 배경: 흰색, 연한 회색, 또는 그래디언트
- 텍스트: 검은색 (RGB 0,0,0 또는 어두운 회색)

대비 규칙:
- 텍스트 색상 대비도 4.5:1 이상 (WCAG AA)
- 배경이 밝으면 어두운 텍스트
- 배경이 어두우면 밝은 텍스트
```

### 폰트
```
한글: Pretendard, 배민쌍둥이, 나눔바른고딕
- 제목: Bold 또는 SemiBold (48~64px)
- 본문: Regular (32~40px)
- 보조: Regular (24~28px)

영문: Inter, SF Pro, Roboto
- 수치: 숫자 정렬 (tabular figures)
```

### 이미지
```
스타일: 일관성 유지
- 아이콘: 단색 또는 심플한 그래디언트
- 사진: 고해상도 (RGB, sRGB)
- 일러스트: 스타일 통일 (선화 vs 채색)

최적화:
- 포맷: JPG (사진), PNG (투명), WebP (최적)
- 해상도: 1080×1350px → 300~500KB
- 압축: 손실 압축 (quality 80~90)
```

## 제작 워크플로우

### Step 1: 주제 및 콘텐츠 확인
```
1. 카드뉴스 주제
2. 시리즈 길이 (5장? 10장?)
3. 타겟 고객
4. 브랜드 색상 / 로고
5. 배포 일정
```

### Step 2: 스크립트 작성
```
각 카드당:
- 제목 (한 문장, 16자 이내)
- 본문 (1~2문장, 30자 이내)
- 시각 요소 (아이콘, 사진, 차트)

형식:
[1/5] 제목
본문 텍스트
(아이콘 위치: 좌측)
```

### Step 3: 이미지 생성
```
nano-banana 스킬 활용:
- Imagen 모델로 이미지 자동 생성
- 또는 사용자 제공 이미지 활용

크기: 1080×1350 또는 1080×1920
포맷: PNG (투명도 필요) 또는 JPG
```

### Step 4: 카드뉴스 합성
```
Python/PIL 또는 Figma:
- 배경색 또는 이미지 레이어
- 텍스트 오버레이 (폰트, 위치, 정렬)
- 로고 위치 (우측 하단 또는 중앙)
- 번호 추가 (1/5, 2/5, ...)

출력: PNG 또는 JPG (300~500KB)
```

### Step 5: 인스타그램 캡션 작성
```
[첫 줄] 후킹 문장
.
[본문] 2~3줄 요약
.
[CTA] 액션 유도
.
[해시태그] 최대 30개

예:
"📌 프리랜서 절세 5가지 1/5

소득세 절약의 첫 번째 스텝은 '적격증빙'입니다.
카드영수증, 세금계산서, 현금영수증 챙기는 습관이 연 100만원+ 절세로 이어집니다. 👇

저장하고 공유하세요!
#절세 #프리랜서 #세무팁 #절세전략"
```

## 절대 규칙 (Hard Rules)

### 1. 텍스트 가독성
```
- 최소 폰트 크기: 32px (모바일 보기 기준)
- 최대 한 줄 문자 수: 12자 (개행으로 나눔)
- 줄 간격: 1.5 이상
- 텍스트 그림자: 검은색 텍스트는 흰 테두리 (stroke)
```

### 2. 색상 대비
```
WCAG 기준:
- 일반 텍스트: 최소 4.5:1
- 큰 텍스트 (18px+): 최소 3:1

도구: contrast-ratio.com
```

### 3. 로고 배치
```
위치: 우측 하단 또는 중앙 하단
크기: 전체 너비의 15~20%
투명도: 100% (완전 불투명) 또는 60~80%
패딩: 상하좌우 20px 이상
```

### 4. 시리즈 일관성
```
- 모든 카드 배경색 통일 (또는 일관된 그래디언트)
- 폰트 / 폰트 크기 통일
- 로고 위치 / 크기 통일
- 번호 형식 통일 (1/5, 2/5, ... 또는 ①, ②, ...)
```

### 5. 이미지 최적화
```
파일명: card_news_1.png, card_news_2.png, ...
크기: 1080×1350px (인스타 표준)
포맷: PNG (완전한 색감) 또는 JPG (작은 용량)
용량: 300~500KB (너무 크면 업로드 지연)
```

## 출력 형식

### 이미지 (필수)
```
PNG 또는 JPG 파일:
- 1080×1350px 크기
- sRGB 색상 공간
- 해상도: 72dpi (웹 기준) 또는 150dpi (인쇄 검토용)

zip 패키지:
card_news_series/
├── card_1.png
├── card_2.png
├── card_3.png
├── card_4.png
└── card_5.png
```

### 인스타그램 캡션 (선택)
```
텍스트 파일 (.txt) 또는 마크다운 (.md):
1번 카드 캡션
2번 카드 캡션
...

또는 한 번에 모두 제공 (복사-붙여넣기 가능)
```

### Figma 링크 (고급 옵션)
```
사용자가 추후 수정을 원할 경우, 
Figma 파일 링크 또는 소스 파일 (.figma) 제공 가능
```

## 워크플로우

```
주제 / 타겟 확인
↓
시리즈 길이 결정 (3~10장)
↓
스크립트 작성 (카드별 제목/본문)
↓
이미지 요청 (nano-banana 또는 사용자 제공)
↓
카드뉴스 합성 (배경 + 텍스트 + 로고)
↓
인스타그램 캡션 작성
↓
최종 이미지 + 캡션 제공
```

## 트러블슈팅

| 증상 | 원인 | 해결 |
|------|------|------|
| 텍스트 안 읽힘 | 폰트 크기 너무 작음 | 최소 32px로 확대 |
| 색상 대비 낮음 | 어두운 배경 + 어두운 텍스트 | 색상 반전 또는 밝은 배경 사용 |
| 이미지 깨짐 | 해상도 부족 | 1080×1350px 이상 확인 |
| 인스타 업로드 실패 | 이미지 크기 초과 | 500KB 이하로 압축 |
| 텍스트 짤림 | 폰트 설정 오류 | PNG 임베드 폰트 확인 |
| 로고 어색함 | 배치/크기 불균형 | 로고 크기 15-20% 조정 |
