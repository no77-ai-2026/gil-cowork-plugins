# 카테고리 분기: 여행/체험 (Category — Travel / Experience)

호텔·투어·액티비티·체험 패키지. **무형 경험 상품** — 사진·후기·기대감이 구매 결정.

---

## 1. 여행 상품의 구매 결정 요인

| 요인 | 비중 | 페이지 반영 |
|---|---|---|
| **사진·동영상** | ★★★★★ | 풍부한 비주얼 (상단·갤러리) |
| **후기·별점** | ★★★★★ | 후기 모듈 별도 |
| **가격 (총 비용)** | ★★★★ | 모든 비용 포함 표시 |
| **위치·접근성** | ★★★★ | 지도·교통 정보 |
| **편의시설·옵션** | ★★★ | 체크리스트 |
| **취소·환불 정책** | ★★★ | 명확 표기 |

---

## 2. 여행 페이지 필수 8 섹션

```
[1] 히어로 — 풍경 사진 + 한 줄 약속 + 예약 CTA
[2] 갤러리 — 객실·식사·풍경·체험 10~20장
[3] 일정 / 코스 — 1일·2일·3일별 흐름
[4] 위치·접근성 — 지도 + 교통 옵션
[5] 포함·불포함 — 명확한 체크리스트
[6] 후기 — 별점 + 사진 후기
[7] 가격 + 취소 정책 — 일자별 가격 + 환불 일정
[8] FAQ — 비자·언어·날씨·짐 등
```

---

## 3. 일정·코스 표시 (필수)

투어·패키지는 **시간 단위 일정** 명시.

### 예시 — 1박 2일 부산 자유여행

```
Day 1 (월요일)
14:00  KTX 부산역 도착 → 호텔 체크인
15:30  광안리 해변 + 일몰
17:30  해운대 시장 저녁
20:00  부산 야경 투어
22:00  호텔 휴식

Day 2 (화요일)
08:00  호텔 조식
09:30  감천문화마을
12:00  자갈치 시장 점심
14:00  KTX 부산역 출발
```

---

## 4. 포함·불포함 (투명성)

후속 추가 비용으로 불만 생기지 않도록 명확.

### 포함

- 항공권 (왕복)
- 호텔 (3박)
- 조식 (3회)
- 가이드·통역

### 불포함

- 중·석식 (자유)
- 입장료 (선택 투어)
- 팁
- 개인 비용 (쇼핑·기념품)

---

## 5. 계절·시즌 분기

여행은 **시즌별 가격·체험·후기**가 완전히 다름.

| 시즌 | 카피 어필 | 가격 |
|---|---|---|
| **성수기 (7-8월·12월-1월)** | "최고의 시즌" | 1.5-2배 |
| **준성수기 (4-5월·9-10월)** | "여유롭게 즐기기" | 1.2-1.3배 |
| **비수기 (3·6·11월)** | "가성비 최고" | 기본 |

페이지에 **시즌별 가격 표** 또는 캘린더 권장.

---

## 6. 후기 모듈 (여행 핵심)

여행 후기는 **사진 + 별점 + 정성** 모두 중요.

### 표시 권장

```
[1] 별점 (큰 글씨) — 4.8/5
[2] 별점 분포 그래프 (5/4/3/2/1)
[3] 인기 키워드 태그 — "친절·청결·맛·교통"
[4] 사진 후기 갤러리 (가로 스크롤)
[5] 정성 후기 (긴/짧 균형 5~10건)
[6] 응답률·응답 시간 (호텔·투어 운영자)
```

## 7. 무형 경험 가시화

만질 수 없는 "경험"을 사진·동영상·시나리오로 가시화.

| 요소 | 표현 |
|---|---|
| **감각 경험** | "현지 음식의 알싸한 향" |
| **시간 흐름** | "해질 무렵 5시 30분" |
| **사회적 경험** | "현지 가족이 직접 안내" |
| **개인 변화** | "도시 스트레스 벗어남" |
| **사진 가치** | "인생 사진 보장" |

## 8. 취소·환불 정책 (필수)

여행 취소는 자주 발생 — 명확한 정책 = 신뢰.

### 표준 정책 표

| 시점 | 환불율 |
|---|---|
| 출발 30일 전 | 100% |
| 출발 14일 전 | 80% |
| 출발 7일 전 | 50% |
| 출발 3일 전 | 30% |
| 당일·노쇼 | 0% |

여행 보험 옵션 추천.
