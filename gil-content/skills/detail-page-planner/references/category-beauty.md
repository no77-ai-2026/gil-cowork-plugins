# 카테고리 분기: 뷰티 (Category — Beauty)

뷰티 카테고리 (스킨케어·메이크업·헤어·바디) 상세페이지의 카테고리 특화 가이드.
**한국 화장품법·표시광고법 광고 심의를 가장 엄격히 적용해야 하는 카테고리.**

---

## 1. 뷰티 본문 4단계 (USP 정리 공식)

```
[1] USP 도입 (대표 USP 1~2개)
    → "5가지 식물 추출 비누, 24시간 보습"

[2] 좋은 제품 특징 2~3개
    → "거품 풍부", "사용 후 당김 없음", "민감성 피부 OK"

[3] 신뢰도 정보
    → 인증·임상·후기·전문가 추천

[4] 옵션·용량·가격 정보
    → 100g·200g, 1팩·3팩
```

이 4단계가 뷰티 본문의 표준 뼈대.

---

## 2. 3 미션 체크 (뷰티 특화)

| 미션 | 뷰티 적용 |
|---|---|
| **진짜 차별점** | "다른 비누와 뭐가 다른가" — 성분·공정·효과 1가지 명확히 |
| **갈아타게** | "현재 쓰는 제품 대비 무엇이 좋은가" — 비교 우위 |
| **생생한 간접 경험** | 발색·제형·사용 후 변화 — 사진/gif/후기 |

## 5. 뷰티 카테고리별 세부 가이드

### 스킨케어

- 핵심 USP: 성분·텍스처·효능
- 필수 모듈: 사용 순서 (토너→세럼→크림), 피부 타입별 추천
- 사진 우선: 제형·발색·흡수·전후

### 메이크업

- 핵심 USP: 발색·지속력·발림성
- 필수 모듈: 컬러 차트, 피부 톤별 매칭
- 사진 우선: 발색 (다양한 피부 톤), 사용 장면

### 헤어

- 핵심 USP: 모발 변화 (윤기·볼륨·진정)
- 필수 모듈: 모발 타입별 (건성·지성·손상모), 사용 빈도
- 사진 우선: 사용 전후 (모발 광택)

### 바디

- 핵심 USP: 향·보습·텍스처
- 필수 모듈: 향 노트, 함께 쓰면 좋은 제품
- 사진 우선: 제형·향 시각화 (꽃잎·과일 컷)

---

## 6. 뷰티 후기 모듈 (필수)

뷰티는 **후기·리뷰가 구매 결정의 70% 이상** 영향. 본문에 후기 모듈 별도 배치.

### 후기 표시 권장

```
[1] 별점 분포 그래프 (5/4/3/2/1)
[2] 평균 별점 (큰 글씨)
[3] 키워드 태그 (자주 언급된 단어 5개)
[4] 사진 후기 (가로 스크롤 갤러리)
[5] 텍스트 후기 (긴/짧 균형 5건)
```

⚠️ 후기 조작·과장 금지. 표시광고법 위반.

## 7. 뷰티 가격 표시 전략

| 전략 | 적용 |
|---|---|
| **1ml당 가격** | 같은 카테고리 비교 우위 |
| **정기구독 할인** | 스킨케어 (월 1회 정기 배송) |
| **세트 할인** | 토너+세럼+크림 3종 세트 |
| **샘플 증정** | "본품 구매 시 샘플 5종" |
| **사은품 (파우치·메이크업 도구)** | 객단가 ↑ |
