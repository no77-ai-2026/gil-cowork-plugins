# 카테고리 분기: 정기구독 (Category — Subscription)

정기구독은 **1회 구매와 근본적으로 다른 비즈니스 모델**. 신규 가입 + 이탈 방지 동시 설계.

대상: 구독 박스, 정기 배송, 멤버십, 스트리밍, 정기 결제 서비스.

---

## 1. 1회 구매 vs 정기구독 차이

| 축 | 1회 구매 | 정기구독 |
|---|---|---|
| 가치 강조 | 제품 자체 | **LTV·반복 경험** |
| 가격 표시 | 절대 가격 | **월·연 + 회당 가격** |
| 페이지 목적 | 구매 결제 | **첫 회 구매 + 약정 동의** |
| 핵심 메트릭 | 전환율·객단가 | **MRR·이탈율** |
| 카피 톤 | 즉시 매력 | **장기 관계·예측 가능성** |

---

## 2. 정기구독 페이지 필수 8 섹션

```
[1] 히어로 — "월 1회, 매월 새로운 OO 도착" + 첫 회 할인 CTA
[2] What's in the Box — 매월 받는 내용 시각화 (사진·박스 개봉)
[3] 큐레이션 가치 — "왜 이걸 매월 받아야 하나" (전문가 / AI / 사용자 기반)
[4] 구독 가격 표 — 1개월 / 3개월 / 6개월 / 1년 선택 (장기 할인)
[5] 일시정지·해지 정책 — "언제든 일시정지·해지" 강조 (불안 해소)
[6] 사회적 증명 — 후기 + 별점 + 회원 수
[7] 첫 회 무료 / 할인 — 진입 장벽 해소
[8] 자주 묻는 질문 (FAQ) — 배송 주기·환불·변경
```

---

## 3. 가격 표시 전략

### 핵심 — "회당 가격" 명시

```
월 39,000원 (회당 9,750원)   ← 4회 배송 기준 회당 가격
연 360,000원 (월 30,000원, -23%)   ← 연간 결제 할인
```

회당 가격이 부담 없이 보이면 가입율 ↑.

### Tier 권장

| Tier | 기간 | 할인 |
|---|---|---|
| 월간 | 매월 결제 | 0% (기본) |
| 분기 | 3개월 결제 | -10% |
| 반기 | 6개월 결제 | -15% |
| 연간 | 12개월 결제 | -25% |

장기 약정 할인이 핵심.

---

## 4. 첫 회 할인·무료

신규 가입 진입 장벽을 낮추는 5가지 전략:

| 전략 | 적용 |
|---|---|
| **첫 달 무료** | "1개월 무료 체험" |
| **첫 회 50% 할인** | "첫 박스 9,900원" |
| **사은품 증정** | "첫 가입 시 OO 증정" |
| **2개월 묶음 할인** | "2개월 결제 시 1개월 무료" |
| **추천 코드** | "친구 추천 시 5,000원 적립" |

---

## 5. 이탈 방지 메시지

페이지 곳곳에 "불안 해소" 메시지 배치.

| 불안 | 해소 카피 |
|---|---|
| "약정에 묶일까?" | "언제든 일시정지·해지 가능" |
| "마음에 안 들면?" | "첫 박스 100% 환불 보장" |
| "다음달 자동 결제?" | "결제 3일 전 알림" |
| "내용 변경?" | "매월 선호 변경 가능" |
| "주기 변경?" | "월·격월·분기 선택" |

---

## 6. 큐레이션 가치 어필

정기구독 가입의 핵심 동기는 **"내가 직접 고르지 않아도 되는 편의성"**.

### 큐레이션 방식 4종

| 방식 | 어필 카피 |
|---|---|
| **전문가 큐레이션** | "10년 차 소믈리에가 매월 직접 선정" |
| **AI 추천** | "구독자의 취향 데이터 기반 AI 추천" |
| **신상품 자동 배송** | "매월 출시되는 신상품 자동 도착" |
| **테마 박스** | "이번 달은 봄꽃 테마" |

각각 다른 페르소나 타깃 — 페이지에서 큐레이션 방식 명시 필수.

---

## 7. 사회적 증명 — 회원 수·후기

| 표시 | 효과 |
|---|---|
| **누적 회원 수** | "30,000명이 구독 중" |
| **누적 발송 박스** | "100만 박스 발송" |
| **평균 구독 기간** | "평균 8개월 구독" |
| **재가입율** | "해지 후 30% 재가입" |
| **별점 + 후기** | 4.5+ 별점 강조 |

---

## 8. 환불·해지 정책 (투명성)

페이지 footer 또는 별도 섹션에 명시. 숨기면 불신.

```
[해지 방법]
- 마이페이지에서 클릭 한 번
- 다음 결제 전 언제든
- 채팅 상담사 도움 가능

[환불 정책]
- 첫 박스 100% 환불 (만족도 보장)
- 발송 전 결제 취소
- 부분 환불 (분기·연간 결제 시)
```

---

## 9. 본문 톤 매너

- **편안 + 신뢰 + 친근**
- "당신" "여러분"
- 장기 관계 강조 — "1년 후" "10번째 박스"
- 약정·자동결제 부담을 "편리함"으로 재구성
