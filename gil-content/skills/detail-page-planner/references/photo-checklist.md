# 카테고리별 사진 체크리스트 (Photo Checklist)

상세페이지에 들어갈 사진은 **카테고리 × 거리(가까운/먼)** 두 축으로 나누어 체크리스트화한다. 빠진 사진이 있으면 본문 흐름이 끊긴다.

---

## 사진 없이 상세페이지 기획 5단계 (사진 촬영 전)

촬영 전에 기획을 마쳐야 사진을 효율적으로 찍을 수 있다.

```
[1] 원 메시지 결정
    → 모듈 1 (One Message) 산출
    → 사진의 미적 방향 확정

[2] 특징 3~5개 추출
    → USP 1~3개 + 좋은 제품 특징 2~3개

[3] 각 특징별 사진 종류 결정
    → 가까운 vs 먼
    → 사진 vs 디자인 이미지 (보이는 특징 vs 안 보이는 특징)

[4] 사진 vs gif 결정
    → 정적 사진 (텍스처·발색·디테일)
    → gif (작동·움직임·전후 변화)

[5] 흐름 짜기 (PowerPoint 워크플로우)
    → 슬라이드별 캡션 + 사진 자리만 표시
    → 후속 촬영 가이드 도출
```

## 사진 vs gif 선택 기준

| 사진(정적) | gif(동적) |
|---|---|
| 텍스처·재질·색·디테일 | 작동·움직임·접힘·펼침 |
| 정확한 가늠이 중요 | 시간 흐름이 중요 |
| 인쇄·SNS 다른 채널 재활용 | 모바일 상세 전용 |
| 용량 가벼움 | 용량 무거움 (3MB 이하 권장) |

## 레퍼런스 찾기 5단계

1. **unsplash·pexels** — 무료 컨셉 이미지로 무드 확인
2. **유료 광고 단어 추출** — 카테고리 검색 → 광고 이미지에 쓰인 단어·구도 메모
3. **무료 사이트 재검색** — 추출한 단어로 다시
4. **경쟁사 상세페이지 캡처** — 좋은 흐름·구도 5개
5. **자체 무드보드** — 위 4개를 한 페이지에 정리해 촬영 가이드 작성

## 파워포인트 이미지 편집 팁

- 보정은 PowerPoint 내장 도구로 충분 (밝기·대비·선명도)
- 사진 누끼 따기 → "그림 형식 → 배경 제거"
- gif 분할 → "그림 형식 → 자르기" 후 슬라이드별 분할 캡처
- 일관된 비율 (정사각 1:1 또는 가로 4:3) 유지
