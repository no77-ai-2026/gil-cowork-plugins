# 카테고리 분기: 패션 (Category — Fashion)

패션 카테고리 (의류·가방·신발·잡화) 상세페이지의 카테고리 특화 가이드.

---

## 1. 페르소나 5단계 설정법

패션은 **타깃의 라이프스타일이 구매를 좌우** — 페르소나를 추상적으로 두지 말고 5단계로 구체화.

```
[1] 인구통계  → 성별·나이·지역·직업 (예: "30대 후반 서울 직장인 여성")
[2] 라이프스타일 → 평소 패션 관심도·예산·구매 채널 (예: "월 패션 지출 30만원, 무신사·29CM")
[3] 가치관      → 무엇을 중시하나 (예: "환경 친화·지속가능성 중시")
[4] 스타일 무드  → 선호 스타일 키워드 5개 (예: "미니멀·차분·은은·여성스러움·실용")
[5] 구매 트리거  → 무엇이 구매를 유발하나 (예: "친구 추천·SNS 광고·세일 알림")
```

이 5단계를 Brief의 `target_redefinition`에 매핑.

---

## 2. 컬러 가이드 (3색 원칙)

상세페이지 전체에서 **최대 3색**만 사용. 그 이상은 시각 노이즈.

| 역할 | 비율 | 예시 |
|---|---|---|
| 메인 컬러 | 60% | 브랜드 컬러 또는 제품 컬러 (예: 베이지) |
| 서브 컬러 | 30% | 메인의 보색 또는 명도 변화 (예: 차콜) |
| 액센트 컬러 | 10% | CTA·할인 표시 등 강조 (예: 와인레드) |

조합 권장:
- **모노톤 + 액센트** — 베이지/차콜 + 골드 (고급스러움)
- **컴플리먼트** — 네이비/베이지 + 오렌지 (밸런스)
- **트라이어드** — 보라/그린/오렌지 (개성, 디자이너 브랜드)

---

## 3. 폰트 가이드 (2폰트 원칙 + 3 카테고리)

상세페이지 전체에서 **최대 2폰트** 조합.

### 폰트 카테고리 3종

| 카테고리 | 분위기 | 용도 |
|---|---|---|
| **산세리프** (Pretendard·Noto Sans) | 모던·깔끔·실용 | 본문·CTA |
| **세리프** (Noto Serif·Nanum Myeongjo) | 우아·전통·고급 | 헤드라인·브랜드 명 |
| **손글씨** (KCC·Cafe24) | 친근·감성·캐주얼 | 액센트·인용·자필 후기 |

### 조합 권장

| 브랜드 무드 | 헤드라인 | 본문 |
|---|---|---|
| 모던·미니멀 | 산세리프 굵게 | 산세리프 가늘게 |
| 클래식·럭셔리 | 세리프 | 산세리프 |
| 캐주얼·감성 | 손글씨 | 산세리프 |

**금지**: 손글씨 본문 (가독성 ↓), 3폰트 이상 사용 (혼란).

---

## 4. 톤앤매너 가이드

### 톤 (Tone) — 메시지 전달 방식

```
공식 (Formal)        ←→        캐주얼 (Casual)
정확 (Precise)       ←→        감성 (Emotional)
직접 (Direct)        ←→        은유 (Indirect)
```

브랜드 무드에 따라 위 축에서 위치 결정. 패션은 보통 **캐주얼 + 감성** 우세.

### 매너 (Manner) — 시각·구도 표현 방식

- **여백**: 충분 (제품 부각)
- **사진**: 모델 컷 + 디테일 컷 + 코디 컷 1:1:1
- **타이포 크기 비**: 1.5배 이상 (헤드라인 vs 본문)
- **CTA**: 페이지 하단 + 중간 1회 추가

## 5. 패션 본문 추가 권장 모듈

| 모듈 | 내용 |
|---|---|
| **사이즈 차트** | 실측 (가슴·어깨·허리·총장) + 모델 착용 사이즈 |
| **소재 정보** | 혼방률 + 세탁 방법 |
| **시즌 활용** | "봄·가을 데일리 룩" 또는 "여름 단독·겨울 레이어드" |
| **코디 제안** | 3-5 코디 (하의·상의·신발 매칭) |
| **계절 출고 일정** | 시즌 패션은 출고 시점 명시 |

| 사용자 페인 | 카피 |

| "배송 늦음" | "주문 즉시 출고, 다음 날 도착" |

---

## 7. 패션 광고 심의 주의

- 비교 광고: "최고", "유일한" 같은 단정 표현 주의
- 원산지 표기: 의류는 원단·봉제 원산지 별도 표기
- 알레르기 표시: 가죽·울 등 알레르기 유발 소재 명시
- 친환경 표시: "친환경", "에코", "지속가능" 등 표현은 인증 근거 필요
