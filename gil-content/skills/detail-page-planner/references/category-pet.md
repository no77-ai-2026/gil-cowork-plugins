# 카테고리 분기: 펫 (Category — Pet)

반려동물 사료·간식·용품·서비스. **보호자가 구매하고 반려동물이 사용**하는 이중 페르소나.

대상: 강아지·고양이 / 기타 (햄스터·새·물고기 등).

---

## 1. 이중 페르소나 매핑

| 페르소나 | 관심사 | 카피 어필 |
|---|---|---|
| **구매자 (보호자)** | 안전·성분·반응·신뢰 | "수의사 추천", "100% 자연 성분" |
| **사용자 (반려동물)** | 기호성·소화·건강 | "맛있게 잘 먹어요", "변 상태 좋아짐" |

상세페이지는 **보호자 중심 90% + 반려동물 반응 10%** 비율 권장.

---

## 2. 반려동물 종류·연령 분기

| 종류 | 연령 | 핵심 요구 |
|---|---|---|
| **강아지** | 퍼피 (0-12개월) | 성장기 영양·작은 알갱이 |
| | 어덜트 (1-7세) | 활동량·체중 관리 |
| | 시니어 (7세+) | 관절·소화 보조 |
| **고양이** | 키튼 (0-12개월) | 성장기·치아 발달 |
| | 어덜트 (1-10세) | 헤어볼·요로 건강 |
| | 시니어 (10세+) | 신장·관절 |
| **소형/대형견** | 견종별 | 알갱이 크기·턱 크기 |

페이지에 **연령·크기 추천 표** 또는 필터 권장.

---

## 3. 사료 성분 표시 (가장 중요)

| 항목 | 표시 |
|---|---|
| **단백질 함량** | "조단백질 30% 이상" |
| **주재료 원산지** | "美 농무부 인증 닭고기" |
| **첨가 비타민·미네랄** | 11종 비타민·미네랄 |
| **함유 X 물질** | "방부제·인공색소·옥수수 무첨가" |
| **알레르기 정보** | 닭·소·옥수수 등 알레르기원 |
| **유통기한** | 명확히 |

성분표는 **이미지 + 텍스트** 동시 표시.

---

## 4. 보호자 페인포인트 매핑

| 페인 | 어필 |
|---|---|
| "안전한 성분인가?" | 인증 + 무첨가 + 원산지 |
| "우리 아이가 잘 먹을까?" | "기호성 95%·만족 후기" |
| "변 상태 좋아질까?" | 소화 데이터 + 후기 |
| "알레르기 있을까?" | 알레르기 원인 표시 |
| "가격 부담?" | 1kg당 가격·정기 배송 할인 |
| "수의사 추천?" | 수의사·전문가 추천 |

---

## 5. 펫 본문 추가 권장 모듈

| 모듈 | 내용 |
|---|---|
| **종류·연령 추천** | 강아지/고양이/소형/대형 + 연령 |
| **성분표 시각화** | 표 + 이미지 + 함유 X |
| **수의사 추천** | 동물병원·수의사 명시 |
| **급여 가이드** | 체중별 1일 급여량 표 |
| **반응 후기** | 보호자 정량 후기 + 사진 |
| **정기 배송 할인** | 월·격월 배송 옵션 |

---

## 6. 광고 심의 가드

| 금지 | 사유 |
|---|---|
| **"100% 자연·100% 안전"** | 단정 표현 |
| **"질병 치료·예방"** | 의약품 효능 (사료에 사용 X) |
| **"수명 연장"** | 단정 표현 |
| **수의사 직책 사칭** | 실제 수의사 인증 필요 |

대체: "건강 관리", "장 건강에 도움 (실증)", "급여 후 변 상태 개선 사례".
