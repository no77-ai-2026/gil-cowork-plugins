# 카테고리 분기: 명품/럭셔리 (Category — Luxury)

명품·하이엔드 브랜드의 상세페이지는 **격식·여백·간결**이 핵심. 일반 이커머스와 완전히 다른 톤 매너.

---

## 1. 럭셔리 카피 톤의 4 원칙

| 원칙 | 설명 |
|---|---|
| **격식 (Formal)** | 친근체·이모지·신조어 금지 |
| **여백 (White Space)** | 시각 노이즈 최소 |
| **간결 (Brevity)** | 한 문장 = 한 메시지 |
| **정통성 (Heritage)** | 브랜드 역사·장인 정신 |

---

## 2. 럭셔리 vs 매스 비교

| 축 | 매스 시장 | 럭셔리 |
|---|---|---|
| 가격 강조 | 할인·가성비 | **무가격** 또는 "문의" |
| CTA | "지금 구매" | "스토어 방문" "VIP 컨설팅 예약" |
| 페이지 길이 | 길게 | **짧게·압축적** |
| 색상 | 다채 | 모노톤·골드·검정 |
| 폰트 | 산세리프 | **세리프** (전통) |
| 톤 | 친근·감성 | **격식·세련** |
| 이미지 | 풍부·다양 | **고화질·소수정예** |
| 사회적 증명 | 후기·별점 | **잡지·기관 추천** |

---

## 3. 럭셔리 페이지 필수 6 섹션

```
[1] 히어로 — 제품 한 컷 + 짧은 헤드라인
[2] 브랜드 헤리티지 — 창립 연도·장인·국가
[3] 제품 디테일 — 소재·생산·디자이너 (간결한 텍스트 + 고화질 사진)
[4] 컬렉션 / 세트 — 함께 어울리는 다른 제품
[5] 서비스 — 케어·수리·VIP 멤버십
[6] CTA — 스토어 위치·문의·예약
```

---

## 4. 브랜드 헤리티지 (필수)

럭셔리는 **이야기 = 가치**.

### 표시 요소

- **창립 연도** ("Maison founded in 1854")
- **국가·도시** ("Paris" "Italian craftsmanship")
- **창립자·장인 이야기**
- **시그니처 패턴·디자인 유산**
- **역사적 사용자 (왕족·셀러브리티)**
- **수상·인증** (장인 인증·전통 공예 인증)

---

## 5. 가격 표시 — "문의" 또는 무가격

명품은 **즉시 가격 X**.

| 방식 | 적용 |
|---|---|
| **무가격** | 페이지에 가격 미표시, 매장 문의 |
| **"문의" 표시** | 가격 자리에 "Price upon request" |
| **사진만** | 가격 자체를 표시하지 않음 |
| **공식 가격 (간소)** | 자리 마련은 하되 강조 X |

가격 강조는 럭셔리 가치 훼손.

---

## 6. CTA — 구매가 아닌 "방문·예약"

| CTA | 적용 |
|---|---|
| **"부티크 방문 예약"** | 오프라인 매장 우선 |
| **"VIP 컨설팅 예약"** | 1:1 상담 |
| **"카탈로그 요청"** | 우편·이메일 |
| **"문의하기"** | 가격·재고 |
| **"위시리스트 추가"** | 즉시 구매 X |

---

## 7. 럭셔리 사진 (photo-checklist 보강)

| 추가 사진 | 이유 |
|---|---|
| 고화질 클로즈업 (소재·바느질) | 장인 정신 |
| 제품 단독 (배경 모노톤) | 시각 노이즈 X |
| 모델 컷 (셀러브리티 또는 무명) | 라이프스타일 |
| 박스·포장 (언박싱) | 럭셔리 경험 |
| 브랜드 부티크·작업실 | 헤리티지 |

촬영은 **고화질·고가 장비 필수** (스마트폰 사진 X).

---

## 8. 사회적 증명 — 매체·기관

후기·별점 대신 **권위 있는 매체·기관**.

### 표시 우선순위

```
[1] 유명 매체 인용 (Vogue·Elle·GQ·Wallpaper*)
[2] 영화·드라마 협찬 노출
[3] 박물관·갤러리 전시
[4] 셀러브리티 착장 (셀럽 마케팅)
[5] 디자인 어워드 수상
```

---

## 9. 럭셔리 본문 추가 권장 모듈

| 모듈 | 내용 |
|---|---|
| **헤리티지 스토리** | 브랜드 역사·창립자 |
| **장인 정신** | 생산 공정·소재 (영상 권장) |
| **컬렉션** | 함께 어울리는 제품 |
| **케어 가이드** | 제품 관리·세탁·보관 |
| **수리·리퍼브** | 평생 수리 보장 |
| **VIP 서비스** | 멤버십·맞춤 제작 |

---

## 10. 카피 톤 매너

- **격식 + 세련 + 절제**
- "당신"보다 "Madame·Monsieur"·"고객님"
- 감탄사·이모지·신조어 X
- 한 문단 = 한 메시지
- 영어·프랑스어·이탈리아어 단어 자연 혼용 OK ("Maison" "Atelier" "Savoir-faire")
