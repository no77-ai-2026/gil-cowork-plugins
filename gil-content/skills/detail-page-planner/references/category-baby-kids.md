# 카테고리 분기: 베이비/키즈 (Category — Baby / Kids)

유아·아동용품은 **부모가 구매하고 아이가 사용**하는 이중 페르소나. 안전 인증·연령 분기가 핵심.

---

## 1. 이중 페르소나 매핑

| 페르소나 | 관심사 | 카피 어필 |
|---|---|---|
| **구매자 (부모)** | 안전·품질·교육적 가치·가격 | "엄마·아빠 마음 놓이는" |
| **사용자 (아이)** | 재미·시각·인지 발달 | "아이가 좋아하는·놀이 시간" |

상세페이지는 **구매자 중심 80% + 사용자 중심 20%** 비율 권장.

---

## 2. 연령 분기 (필수)

연령별로 안전 기준·사용 방식·카피가 완전히 다름.

| 연령 | 핵심 요구 | 본문 강조 |
|---|---|---|
| **0-6개월** | 안전·청결·신생아 적합 | "100% 무자극 성분", "삶을 수 있음" |
| **6-12개월** | 발달 단계·이유식·기지개 | "이유식 보조·정밀 미세 운동" |
| **1-3세** | 활동성·인지 발달 | "걷기·말하기·인지 발달 자극" |
| **3-6세** | 학습·창의·사회성 | "유아 교육·역할놀이·사회성" |
| **6-12세** | 학습·취미·자율성 | "초등 학습·취미·자기 표현" |

페이지에 **연령 추천 표** 또는 **연령 필터** 권장.

---

## 3. 안전 인증 (한국 시장 필수)

| 인증 | 의미 | 카테고리 |
|---|---|---|
| **KC 마크** | 한국 안전 인증 | 전 카테고리 필수 |
| **유럽 CE** | EU 안전 | 수입품 |
| **EN 71** | EU 장난감 안전 | 장난감 |
| **OEKO-TEX 100** | 유해물질 검증 | 의류·침구 |
| **친환경 인증** | 무독성 | 식기·장난감 |
| **유기농 인증** | 식품·코튼 | 이유식·의류 |

페이지 상단 또는 별도 섹션에 인증 로고 배치 필수.

---

## 4. 부모 페인포인트 매핑

베이비/키즈 페이지에서 자주 어필되는 부모 고민 7가지:

| 페인 | 카피 어필 |
|---|---|
| "안전한가?" | KC·CE 인증 + 무독성 성분 |
| "오래 쓰나?" | "1년 사용 후기" + 내구성 인증 |
| "발달에 좋나?" | 발달 전문가 추천 + 연령별 효과 |
| "청소 쉬운가?" | "세탁기·식기세척기 OK" |
| "아이가 좋아할까?" | "100% 아이 만족 후기" |
| "환경 친화적인가?" | 친환경 소재·재활용 가능 |
| "가격 대비 가치?" | "1년 사용 시 일당 X원" |

## 5. 베이비/키즈 본문 추가 권장 모듈

| 모듈 | 내용 |
|---|---|
| **연령 추천** | "Best for 6-12개월" 또는 연령 슬라이더 |
| **안전 인증 배지** | KC·CE·OEKO-TEX 등 상단 배치 |
| **성분·소재 표** | 유해 물질 검사 결과 |
| **부모 후기** | "1년 사용" 같은 장기 후기 강조 |
| **연계 추천** | "함께 사용하면 좋은 제품" |
| **선물 포장** | "선물용 포장 무료" |
| **A/S·교환·환불** | 명확히 |

---

## 6. 카피 톤 매너

- **안심 + 신뢰 + 따뜻**
- 의학·발달 전문 용어는 부모가 이해할 수준
- "엄마·아빠" 호명 OK (단, 한부모·조부모 양육도 고려)
- 아이의 미소·웃음 이미지 풍부
- **자극적 표현 금지** ("우리 아이 천재로", "100% 효과" 같은 단정)

---

## 7. 광고 심의 가드

| 금지 | 사유 |
|---|---|
| **"천재"·"영재"·"우등생"** | 발달 단정 |
| **"100% 효과"·"확실히 발달"** | 효능 단정 |
| **"의학적 효과"** | 의약품 효능 (장난감·교육제품) |
| **"안전 100%"** | 사고 가능성 부정 |

대체: "발달에 도움", "재미와 학습 함께", "안심 사용 가능".
