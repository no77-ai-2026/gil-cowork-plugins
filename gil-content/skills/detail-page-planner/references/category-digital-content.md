# 카테고리 분기: 디지털 콘텐츠 (Category — Digital Content)

강의·e북·뉴스레터·소프트웨어 등 **무형 디지털 상품**. 물리 상품과 근본적으로 다른 카피·구조.

---

## 1. 무형 상품의 핵심 어려움

| 어려움 | 해결책 |
|---|---|
| 만질 수 없음 | **미리보기·체험판** 제공 |
| 효과 가시화 어려움 | **수강 전후 변화·정량 데이터** |
| 가치 가늠 어려움 | **시간 환산·동일 비용 대안 비교** |
| 환불 부담 | **환불 보장·만족 보장** |
| 신뢰 어려움 | **수강생 후기·완강율·재구매율** |

---

## 2. 디지털 콘텐츠 페이지 필수 8 섹션

```
[1] 히어로 — 핵심 약속 + 무료 체험 CTA
[2] 강사·저자 소개 — 경력·실적·신뢰
[3] 커리큘럼 / 목차 — 전체 흐름 + 각 챕터 소요 시간
[4] 미리보기 / 샘플 — 1~2개 챕터 무료 공개
[5] 학습 결과 — "이 강의 후 ___을 할 수 있어요" (5~7개)
[6] 수강생 후기 — 정량 결과·완강 후기
[7] 가격 + 환불 보장 — 평생 소장 or 기간 한정 + 환불 정책
[8] FAQ — 수강 환경·기간·자료·자격증
```

---

## 3. 학습 결과 어필 (Outcome 중심)

기능 나열 X, **수강 후 무엇을 할 수 있는가** 명시.

### 결과 카피 공식

```
"이 강의를 마치면, ___을 ___할 수 있어요."
                ↑              ↑
            구체 결과       구체 시간/수준
```

예시:
- ❌ "파이썬 기초를 배웁니다"
- ⭕ "3주 후, 파이썬으로 데이터 100건을 1초에 정리할 수 있어요"

- ❌ "디자인 강의"
- ⭕ "한 달 후, 피그마로 모바일 앱 화면 5개를 직접 설계할 수 있어요"

---

## 4. 강사·저자 신뢰 모듈

디지털 콘텐츠는 **강사 신뢰 = 구매 결정**. 별도 섹션 필수.

### 표시 요소

```
[1] 강사 사진 (전문 촬영)
[2] 한 줄 타이틀 ("10년 차 데이터 사이언티스트")
[3] 주요 경력 3개 (회사·직책·기간)
[4] 성과 정량 (책 판매·구독자·강의 횟수)
[5] 매체 노출 (인터뷰·기고·강연)
[6] SNS 팔로워 수
```

| 챕터 | 소요 시간 | 학습 결과 |
|---|---|---|
| 1. 환경 설정 | 30분 | 개발 환경 갖춤 |
| 2. 기초 문법 | 2시간 | 기본 코드 작성 |
| 3. 데이터 처리 | 3시간 | 100건 데이터 정리 |
| ... | ... | ... |

각 챕터에 **결과 명시** — 단순 제목 X.

| 유형 | 적용 |

| **무료 워크북** | 강의 |

페이지 상단·하단·중간에 체험 CTA 반복.

## 7. 환불 보장 (필수)

디지털 콘텐츠는 환불 부담이 가입 장벽. 명확한 환불 정책으로 해소.

### 표시 위치

- 페이지 하단 별도 섹션
- CTA 버튼 옆 한 줄 ("30일 환불 보장")
- FAQ에 상세

### 표준 환불 정책

```
[1] 30일 100% 환불 보장
[2] 강의 진도 30% 미만 시 환불
[3] 결제 후 7일 내 100% 환불 (소비자보호법 기본)
[4] 만족 안 하면 환불 — 사유 묻지 않음
```

## 8. 수강생 후기 — 정량·정성

| 표시 | 효과 |
|---|---|
| **완강율** | "75% 완강 (업계 평균 30%)" |
| **재구매율** | "수강생 40%가 다른 강의 재구매" |
| **별점 분포** | 4.7/5 (1,200건) |
| **정량 후기** | "이 강의 후 회사 데이터 분석 시간 50% 줄었어요" |
| **정성 후기** | "강사가 친절해서 끝까지 들었어요" |

---

## 9. 가격 표시 — 가치 환산

무형 상품의 가격은 **가치 환산**으로 정당화.

| 환산 방식 | 예시 |
|---|---|
| **시간 환산** | "오프라인 1:1 컨설팅 10시간 = 100만 원, 이 강의는 19,900원" |
| **시급 환산** | "이 강의 후 시급 5만 원 →월 50만 원 추가 수입 가능" |
| **대안 비교** | "MBA 1년 = 5,000만 원, 이 강의는 핵심만 49,000원" |
| **일당 가격** | "한 달 정기 = 일당 990원" |

## 10. 디지털 콘텐츠 카피 톤

- **명확 + 친근 + 전문**
