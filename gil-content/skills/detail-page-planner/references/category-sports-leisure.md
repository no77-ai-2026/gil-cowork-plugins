## 2. 활동 시나리오 시각화

스포츠는 **사용 장면 = 구매 동기**. 시나리오 사진 풍부.

### 시나리오 분류

| 종목 | 시나리오 예시 |
|---|---|
| 캠핑 | 새벽 텐트 안·모닥불 옆·비 오는 타프 아래 |
| 러닝 | 새벽 트레일·도시 야경·마라톤 코스 |
| 자전거 | 시골 도로·산악 트레일·도시 출퇴근 |
| 골프 | 페어웨이·그린·러프·날씨별 |
| 등산 | 능선·바위·강풍·일출 |

각 시나리오마다 제품이 어떻게 도움 되는지 명시.

## 3. 기능 사양 표 (정량)

스포츠 제품은 **숫자 = 신뢰**.

### 예시 — 트레일 러닝화

| 사양 | 값 |
|---|---|
| 중량 (250mm) | 220g |

| 방수 | GORE-TEX |
| 핏 | 미듐 발폭 |
| 적합 거리 | 5-50km |
| 적합 지형 | 트레일·자갈·바위 |

---

## 4. 사이즈·핏 가이드

스포츠 제품은 **사이즈·핏 = 부상 방지**. 정밀 가이드 필수.

### 표시 권장

- 발 길이 / 발폭 측정법 (사진 + 다이어그램)
- 다른 브랜드와 사이즈 비교 표 ("나이키 270 = 우리 275")
- 시즌·양말 두께에 따른 권장 사이즈
- 후기 기반 "정사이즈 vs 큰 사이즈" 통계

---

기존 장비와 함께 쓰는 경우 — 호환 명시 필수.

|---|---|
| **자전거 부품** | 프레임 크기·크랭크·드레일러 호환 |
| **카메라 액세서리** | 마운트·전기 접점 |
| **골프 그립** | 샤프트 굵기·기존 그립 제거 방법 |
| **러닝화 인솔** | 발 모양·아치 높이 |

## 6. 운동선수·전문가 추천 (신뢰)

스포츠는 **신뢰 = 전문가 사용**.

### 표시 우선순위

```
[1] 국가대표·프로선수 사용 (있다면)
[2] 인플루언서·유튜버 추천
[3] 전문 매장 입점
[4] 잡지·미디어 리뷰
[5] 사용자 후기 (정량)
```
