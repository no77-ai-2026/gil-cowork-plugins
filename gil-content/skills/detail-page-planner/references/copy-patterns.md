# 상세페이지 카피 패턴 (Copy Patterns)

상세페이지 본문·CTA·가격 표시에 쓰이는 카피 패턴 모음. detail-page-copy 스킬에 위임 시 입력 가이드로 활용.

---

## 1. 저관여 vs 고관여 언어 분기

### 저관여 제품 (가격 1만 원 이하·즉시 결정)

| 원칙 | 적용 |
|---|---|
| **초등 5학년 가독성** | 한자어·전문용어 제거 |
| **짧게** | 한 문장 25자 이내 |
| **우리말** | 외래어 최소 |
| **1EA 가격** | "박스당" 가격 X, "1개당" |
| **5초 안에 결정** 가능한 정보만 |

예시:
- ❌ "영양 성분이 풍부한 프리미엄 비타민" → ⭕ "하루 1알, 종합 비타민"
- ❌ "탁월한 흡수율과 지속력" → ⭕ "8시간 가는 보습"

### 고관여 제품 (10만 원 이상·고민 시간 김)

| 원칙 | 적용 |
|---|---|
| **신뢰 정보 풍부** | 인증·수상·연구 결과 |
| **타사 비교 OK** | 차별화 명시 |
| **상세 옵션 설명** | 사이즈·옵션별 차이 |
| **사용 후 변화 시나리오** | "1주일 후·1개월 후" |

예시:
- 인증 마크 (FDA·CE·KC)
- 임상 데이터 (n=200, 효과 87%)
- 전문가 추천 (피부과 전문의)

### "고민 시간" 프레임 (3박자 법칙)

저관여/고관여 둘 다 적용 가능한 프레임:

```
[1] 고객의 고민 명시 — "이런 거 있죠?"
[2] 그 고민의 진짜 원인 — "사실 이게 원인이에요"
[3] 우리 제품이 줄여주는 정도 — "이걸 30분에서 5분으로"
```

## 2. 가격 표시 13전략 (사게 만드는 노하우)

| # | 전략 | 적용 예 | 효과 |
|---|---|---|---|
| 1 | **가격 인상 예고** | "내일부터 1만원 인상" | 즉시 구매 유도 |
| 2 | **가격 쪼개기** | "하루 980원" | 부담 ↓ |
| 3 | **앵커링** | ~~정가 50,000원~~ → 29,000원 | 할인 체감 |
| 4 | **단수 가격** | 9,900원 (10,000원 X) | 심리 저항 ↓ |
| 5 | **빨간색 표시** | 할인가만 빨강 | 시선 집중 |
| 6 | **비교 가격** | "스타벅스 커피 2잔 가격" | 가치 환산 |
| 7 | **수량 한정** | "남은 수량 12개" | 희소성 |
| 8 | **시간 한정** | "오늘 자정까지" | 긴급성 |
| 9 | **묶음 할인** | 1+1, 2+1 | 객단가 ↑ |
| 10 | **정기구독 할인** | 정기 -20% | LTV ↑ |
| 11 | **무료배송 임계** | "3만 원 이상 무료배송" | 객단가 ↑ |
| 12 | **사은품 추가** | "지금 주문 시 OO 증정" | 가치 ↑ |
| 13 | **첫 구매 할인** | "회원가입 5천 원 쿠폰" | 진입 장벽 ↓ |

조합 권장:
- **앵커링 (3) + 단수 가격 (4) + 빨간색 표시 (5)** — 가장 흔한 3종 세트
- **수량 한정 (7) + 시간 한정 (8)** — 긴급성 극대
- **가격 쪼개기 (2) + 비교 가격 (6)** — 부담 해소형

## 3. 카피 변환 공식 (FAB → 사용자 체감)

### 3단계 변환

```
[1] Feature (기능)        → "USB-C 충전, 30W 출력"
[2] Advantage (이점)       → "30분에 80% 충전"
[3] Benefit (사용자 혜택) → "회의 5분 전, 잠깐 꽂으면 하루 종일"
```

상세페이지 본문은 항상 **Benefit 중심**으로 작성. Feature 나열 금지.

### 어려운 정보 → 일상 표현

| 어려운 (전문) | 일상 (체감) |
|---|---|
| "히알루론산 1.5%" | "오후 4시에도 촉촉" |
| "20,000mAh 배터리" | "비행기 5번 충전" |
| "OLED 4K HDR" | "영화관 같은 화면" |
| "0.3mm 마이크로 니들" | "통증 없이 흡수" |

## 4. 뷰티 카피 8유형 (각 10-15 패턴)

뷰티 상세페이지에서 가장 잘 작동하는 카피 유형 분류:

1. **즉시 효과 호소** — "오늘 바르고 내일 거울 보세요"
2. **공감·페인** — "마스크 자국, 이제 안녕"
3. **숫자 임팩트** — "3초만에 흡수, 24시간 보습"
4. **비교 우위** — "왜 다들 이걸 쓰는지 알겠어"
5. **권위·전문가** — "피부과 의사가 직접 추천"
6. **자연·성분** — "5가지 식물 추출, 합성 0%"
7. **라이프스타일** — "출근 5분 전 메이크업"
8. **시즌·이벤트** — "봄꽃 피기 전 피부 깨우기"

각 유형마다 페르소나·시즌·가격대에 맞춰 변주 가능.

## 5. CTA 카피 패턴

상세페이지 CTA(Call to Action)는 페이지 하단 + 중간 + 상단 3곳에 배치 권장.

### CTA 카피 5유형

| 유형 | 예시 | 적용 시점 |
|---|---|---|
| **즉시형** | "지금 구매하기" | 가장 무난 |
| **혜택 강조형** | "30% 할인가로 받기" | 가격 매력 시 |
| **결핍 해소형** | "내 피부에 적용하기" | 페인 해결 시 |
| **체험 약속형** | "30일 무료 체험 시작" | 구독·SaaS |
| **수량 한정형** | "마지막 12개 구매" | 희소성 |

CTA 디자인 원칙:
- 색상 대비 (배경 ≠ CTA 색)
- 충분한 여백 (다른 요소와 거리)
- 모바일 thumb-friendly (44px 이상 높이)
- 1페이지 1차 CTA 우선 (혼란 X)

## 6. 본문 카피 톤 매트릭스

```
              | 캐주얼 톤              | 격식 톤
--------------+-----------------------+------------------------
저관여(생활)   | "오늘 저녁 어때요?"    | "일상에 가벼운 도입"
고관여(투자)   | "지금이 그 시점이에요"  | "장기적 가치를 고려해주세요"
```

상세페이지는 보통 **저관여+캐주얼** 또는 **고관여+격식** 조합. 채널이 펀딩이면 캐주얼 우세, 커머스면 격식·중립.

## 활용 — detail-page-copy로 위임

본 references는 detail-page-planner의 Brief에서 **카피 톤·가격 전략·CTA 유형**을 결정하는 데 사용. 결정된 항목은 Brief YAML의 다음 필드로 출력:

```yaml
copy_strategy:
  involvement: "{low|high}"
  tone: "{casual|formal}"
  pricing_tactics: ["앵커링", "단수가격", "수량한정"]
  cta_type: "{즉시형|혜택강조형|결핍해소형|체험약속형|수량한정형}"
```

이걸 `gil-commerce:detail-page-copy`에 인자로 넘기면 13섹션 카피 JSON에 반영됨.
