# 카테고리 분기: 푸드 (Category — Food)

푸드 카테고리 (식품·음료·건강기능식품·농수산물) 상세페이지의 카테고리 특화 가이드.

---

## 1. 목적형 vs 탐색형 소비자 분리

푸드는 구매 동기에 따라 두 가지 사용자 타입이 명확히 갈린다.

### 목적형 (특정 제품 찾는 사용자)

| 행태 | 본문 전략 |
|---|---|
| 검색 → 비교 → 즉시 구매 | **최상단에 USP·가격·옵션 배치** |
| 페이지 체류 짧음 | 군더더기 제거 |
| 가격 민감 | 할인·묶음·정기구독 강조 |
| 신뢰 정보 중시 | 인증·원산지·유통기한 명시 |

### 탐색형 (새로운 식품 발견하는 사용자)

| 행태 | 본문 전략 |
|---|---|
| 인스타·블로그 → 호기심 → 시도 | **스토리·라이프스타일 우선** |
| 페이지 체류 길어도 OK | 인트로 충실 |
| 감성 구매 | 비주얼·체험 후기 강조 |
| 새로운 경험 추구 | 차별화·독특함 강조 |

### 채널 매핑

- 목적형 우세: 쿠팡·SSG·마켓컬리 (검색 기반)
- 탐색형 우세: 와디즈 펀딩·스마트스토어 (광고 기반)
- 양쪽 혼재: 네이버 쇼핑·카카오 커머스

## 2. 푸드만의 키워드 — "먹는 경험"

상세페이지에 다음 4개 키워드를 자연스럽게 녹여야 식욕 자극.

| 경험 | 카피 예시 |
|---|---|
| **시각** | "터지는 노른자, 윤기 흐르는 면발" |
| **후각** | "들어서면 풍기는 깊은 된장 향" |
| **촉각** | "한 입 베어물면 바삭, 안은 촉촉" |
| **청각** | "젓가락 들 때 '톡' 갈라지는 소리" |

미각 직접 표현보다 **다른 감각으로 미각 환기**가 효과적.

## 3. 농수산물 4가지 스토리텔링

농수산물은 산지·생산자 스토리가 차별화.

### 4가지 스토리 패턴

| # | 패턴 | 적용 |
|---|---|---|
| 1 | **산지 이야기** | "전북 무주 해발 700m 무공해 토양" |
| 2 | **생산자 이야기** | "30년 차 농부 김OO, 새벽 4시 출하" |
| 3 | **계절 이야기** | "12월 첫 서리 후 수확, 단맛 최정점" |
| 4 | **전통·역사 이야기** | "조선 왕실 진상품, 200년 명맥" |

조합 권장: **산지 (1) + 생산자 (2)** — 가장 강력 ("어디서 누가" 명확)

## 4. 식품 광고 심의 가드

### 절대 금지 표현 (식품법)

| 금지 | 대체 |
|---|---|
| "치료·완화·예방" | (효능 단정 X) |
| "면역력 향상" | "건강 관리" |
| "다이어트 효과" | "체중 관리에 도움" |
| "암 예방" | (절대 사용 X) |
| "100% 자연" | "주요 성분 자연 유래" |

### 건강기능식품 별도 규제

- 건강기능식품 표시는 **식약처 인증 표현**만 가능
- 일반 식품에 건강기능식품 표현 사용 시 처벌
- 사례: "장 건강에 도움" → 일반 식품에 사용 X (건기식만)

### 안전 표현 가이드

- "맛있는" "신선한" "고소한" → ✅ OK
- "건강한" "면역" "다이어트" → ⚠️ 신중
- "치료" "예방" "완화" → ❌ 절대 X
