# Ghost CMS 퍼블리싱 및 뉴스레터 가이드

> gil-content | blog 참조 가이드 | 2026 기준

## 개요

Ghost는 콘텐츠 퍼블리싱과 뉴스레터를 하나의 플랫폼에서 운영할 수 있는 오픈소스 CMS입니다. WordPress보다 가볍고, Substack보다 자체 데이터 소유권이 강합니다. 유료 멤버십 기능이 내장되어 있어 콘텐츠 수익화에 유리합니다.

---

## 1. Ghost 배포 옵션

| 방식 | 비용 | 특징 |
|------|------|------|
| Ghost(Pro) | 월 $9+ | 관리형 호스팅, 설치 불필요 |
| Self-hosted | 서버 비용만 | 완전한 커스터마이징, 기술 지식 필요 |
| Railway/Render | 월 $5-10 | 자동 배포, 중간 난이도 |

**추천**: 기술 역량이 없다면 Ghost(Pro), 자유도 원한다면 Railway + Ghost

## 2. 퍼블리싱 워크플로우

### 글 작성 에디터 특징

- Markdown 기반 카드 에디터 (직관적 UI)
- 이미지, 갤러리, 북마크, 코드, HTML 카드 지원
- 유료/무료 접근 분기 설정 (멤버십 paywall)
- 예약 발행 및 A/B 테스트 지원

### 발행 설정 체크리스트

- [ ] SEO 제목 및 메타 설명 설정 (각 글마다)
- [ ] OG(Open Graph) 이미지 설정 (소셜 공유용)
- [ ] 태그 및 저자 설정
- [ ] 접근 권한 설정: 전체 공개 / 무료 회원 / 유료 회원
- [ ] 뉴스레터 발송 여부 선택

---

## 3. 뉴스레터 연동

### Ghost 뉴스레터 핵심 기능

- 발행과 동시에 구독자에게 이메일 자동 발송
- 여러 뉴스레터 목록 분리 운영 가능
- Mailgun 또는 자체 이메일 서버 연동
- 무료/유료 구독자 세분화

### 뉴스레터 성장 전략

- [ ] 홈페이지 상단에 구독 폼 배치
- [ ] 무료 콘텐츠 제공으로 초기 구독자 확보
- [ ] 주 1-2회 일정한 발행 주기 유지
- [ ] 유료 전환 유인: 독점 콘텐츠, Q&A 세션, 아카이브 접근

## 4. SEO 설정

### Ghost 내장 SEO 기능

- 자동 XML 사이트맵 생성
- 구조화 데이터(JSON-LD) 자동 삽입
- 캐노니컬 URL 자동 설정
- AMP 지원 (테마 설정에서 활성화)

### 추가 SEO 최적화

- 구글 서치 콘솔 연동 (설정 → 코드 삽입)
- 구글 애널리틱스 4 연동
- 영문 슬러그 사용: `/how-to-start-newsletter` 형식
- 내부 링크: 관련 글 2-3개 연결

---

## 5. 유료 멤버십 설정

- Stripe 연동으로 월/연 구독 결제
- 한국 결제: 토스페이먼츠·아임포트 연동 가능 (커스텀 설정 필요)
- 멤버십 단계 설정: 무료 / 월 구독 / 연 구독
- 무료 체험 기간 설정 권장 (첫 1개월 무료)
