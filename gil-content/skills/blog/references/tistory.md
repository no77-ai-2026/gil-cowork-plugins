# 티스토리 SEO 및 애드센스 최적화 가이드

> gil-content | blog 참조 가이드 | 2026 기준

## 개요

티스토리는 구글 검색 유입 + 애드센스 수익화를 동시에 추구할 수 있는 플랫폼입니다. 네이버 블로그와 달리 구글 SEO 원칙이 그대로 적용됩니다.

---

## 1. 티스토리 SEO 기본 설정

### 필수 설정 체크리스트

- [ ] 카카오 계정 연동 및 블로그 주소 확정 (변경 불가)
- [ ] 스킨 선택: SEO 친화적 HTML 구조 스킨 사용 (커스텀 HTML 지원 확인)
- [ ] 메타 설명(description) 각 글마다 개별 설정
- [ ] robots.txt 확인 (노출 차단 여부)
- [ ] 구글 서치 콘솔 등록 및 사이트맵 제출
- [ ] 카카오 웹마스터 도구 등록

---

## 2. 구글 검색 최적화

### 글 작성 SEO 원칙

| 요소 | 권장 설정 |
|------|-----------|
| URL 구조 | 영문 슬러그 사용 (예: /how-to-get-business-license) |
| 제목 태그 | H1 1개, H2 3-5개, H3 필요시 |
| 이미지 alt | 모든 이미지에 키워드 포함 alt 텍스트 |
| 내부 링크 | 관련 글 3개 이상 연결 |
| 글 길이 | 정보성 키워드: 2,000자+, 상업성: 1,000-1,500자 |

### 구글 E-E-A-T 충족

- Experience(경험): 직접 해본 경험 서술 포함
- Expertise(전문성): 블로그 소개에 전문 분야 명시
- Authoritativeness(권위): 출처 링크, 인용 명시
- Trustworthiness(신뢰): 연락처, 작성자 정보 노출

---

## 3. 애드센스 수익화 전략

### 수익 극대화 광고 배치

- 본문 상단 (제목 아래 첫 문단 이후): CTR 가장 높음
- 본문 중간 (긴 글의 경우 스크롤 50% 지점)
- 본문 하단: 읽기 완료 후 클릭율 높음
- 사이드바: PC 방문자에게 노출

### 고수익 콘텐츠 유형

1. 정보성 긴 글 (리뷰, 비교, 방법 안내)
2. 계절성 콘텐츠 (연말정산, 수능 등 시기별)
3. 상업적 의도 키워드 (가격, 구매, 추천)

---

## 4. 티스토리 특화 팁

- **카테고리 전략**: 카테고리당 20개 이상 글 쌓기 후 구글 인덱싱 확인
- **발행 시간**: 평일 오전 7-9시, 오후 8-10시 발행 시 초기 유입 유리
- **내부 검색**: 티스토리 내부 검색 최적화를 위해 태그 5-10개 설정
- **소셜 공유**: 다음 뷰 노출 활용 (카카오 계정 연동 시 자동 연계)

---

## 5. 콘텐츠 품질 체크리스트

- [ ] 핵심 키워드가 제목·첫 문단·소제목에 자연스럽게 포함
- [ ] 모바일 가독성 확인 (짧은 문단, 충분한 여백)
- [ ] 대표 이미지 설정 (SNS 공유 시 미리보기용)
- [ ] 발행 후 구글 서치 콘솔에서 URL 검사 및 색인 요청
