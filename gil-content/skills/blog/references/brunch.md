# 브런치 스토리 작성 가이드

> gil-content | blog 참조 가이드 | 2026 기준

## 개요

브런치(Brunch)는 카카오가 운영하는 콘텐츠 플랫폼으로, 작가 심사제를 통해 양질의 글쓰기 환경을 제공합니다. 수익화보다 브랜딩·포트폴리오·독자 연결에 강점이 있습니다.

---

## 1. 브런치 작가 신청

### 작가 선정 기준

브런치는 신청 후 검토 기간(1-4주)을 거쳐 작가 자격을 부여합니다.

**선정 가능성을 높이는 요소**:
- [ ] 명확한 전문 분야 또는 고유한 관점
- [ ] 신청서에 글쓰기 이유와 독자 대상 명확히 기술
- [ ] 샘플 글 2-3편 수준 높게 작성 (맞춤법, 구성력)
- [ ] 기존 SNS 또는 다른 플랫폼 활동 이력 첨부

**탈락 요인**:
- 광고·홍보성 목적 명확한 경우
- 단순 정보 나열 수준의 글
- 다른 플랫폼 글 그대로 복사

---

## 2. 브런치 글 특성

### 플랫폼 성격

| 특징 | 설명 |
|------|------|
| 독자층 | 30-40대 직장인, 감성·교양 콘텐츠 선호 |
| 바이럴 구조 | 에디터 추천 > 다음 메인 노출 > 카카오 채널 공유 |
| 수익화 | 직접 광고 없음. 책 출판, 강의, 외부 협업 통로 |
| 분위기 | 에세이, 인터뷰, 심층 분석, 여행기 강세 |

### 에디터 추천을 받는 글의 특징

- 개인적 경험 + 보편적 공감을 연결하는 서사 구조
- 트렌드에 대한 독자적 해석 (정보 단순 나열 지양)
- 제목: 궁금증을 유발하는 문장형 또는 역설적 표현
- 글 길이: 2,000-4,000자 (너무 짧거나 길면 불리)

---

## 3. 브런치 글 작성 원칙

### 구조 권장안

```
1. 도입부 (200~300자)
   - 독자의 경험 또는 감정과 연결되는 장면 묘사
   - 질문형 또는 역설로 시작하면 클릭율 상승

2. 본론 (1,500~2,500자)
   - 개인 경험 → 사회적 맥락 → 핵심 인사이트 순서
   - 사실보다 "의미"를 드러내는 문장 강조
   - 소제목으로 섹션 구분 (선택, 에세이는 자연 흐름 유지)

3. 마무리 (300~500자)
   - 결론보다 여운을 남기는 마무리 선호
   - 독자가 자신의 경험을 떠올리게 유도
```

---

## 4. 브런치 성장 전략

- **시리즈 연재**: 단발 글보다 시리즈가 구독자 확보에 효과적
- **발행 주기**: 주 1-2회 정기 발행으로 독자 습관 형성
- **소셜 연동**: 카카오스토리, 트위터(X), 링크드인과 동시 공유
- **타 작가 교류**: 구독·댓글 활동으로 자연스러운 노출 확대
