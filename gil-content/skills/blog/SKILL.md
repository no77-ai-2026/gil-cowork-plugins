# 블로그 (Blog)

## 개요

6개 주요 블로그 플랫폼에 최적화된 블로그 포스팅을 작성합니다. 2026년 최신 네이버 C-Rank·D.I.A.+ 알고리즘, GEO(생성형 AI 검색 최적화), 카피라이팅 안티패턴을 반영하여 검색 상위 노출과 독자 전환율 모두를 극대화합니다.

## 트리거 키워드

블로그, 블로그 포스팅, 블로그 글쓰기, 네이버 블로그, 티스토리, 브런치, 워드프레스, Ghost, 포스팅, 콘텐츠 작성, SEO 블로그, 검색 상위노출

## 실전 블로그 패턴 (한국 시장)

### 1. 블로그 헤드라인 8유형

| 유형 | 예시 | 적용 |
|---|---|---|
| 즉시 효과 | "오늘 시작하는 ~" | 튜토리얼 |
| 공감·페인 | "~ 안 되는 분 보세요" | 페인 솔루션 |
| 숫자 임팩트 | "30분 만에 ~ 완성하는 5가지" | 리스트형 |
| 비교 우위 | "A vs B, 왜 다들 ~ 쓸까" | 비교 리뷰 |
| 권위·전문가 | "10년 차 ~가 알려주는" | 전문 가이드 |
| 자연·솔직 | "솔직히 ~ 안 좋은 점" | 정직 리뷰 |
| 라이프스타일 | "출근 5분 전 ~" | 일상 활용 |
| 시즌·이벤트 | "봄꽃 피기 전 ~ 준비" | 시기 매칭 |

### 2. 잘 되는 기업 블로그 콘텐츠 사례 10유형

1. 사용자 인터뷰 / 도입 사례
2. 데이터 인사이트 (자체 데이터 시각화)
3. 업계 트렌드 리포트
4. 제품 사용법 / 튜토리얼
5. 팀 문화 / 백오피스 스토리
6. 전문가 인터뷰
7. 시장·경쟁사 분석
8. 실패 회고 / 후일담
9. FAQ 모음 (검색 키워드 기반)
10. 뉴스레터 발췌 / 큐레이션

### 3. 검색 vs 콘텐츠 분기

| 목적 | 분기 |
|---|---|
| **검색 유입 (SEO)** | 키워드 기반 → 정확 매칭 / 제목 키워드 정직 |
| **콘텐츠 인지** | 톤·스타일 → 본인·브랜드 정체성 / 제목 매력 위주 |
| **AI 검색 인용 (GEO)** | 답 구조 명확 → 헤딩·요약 박스 / 질문형 제목 |

### 4. 블로그 시리즈 구성 (재방문 유도)

- **3편**: 입문 → 실전 → 고급
- **5편**: 문제 → 원인 → 해결 → 사례 → 도구
- **10편**: 전체 가이드북 (장기 시리즈, SEO 강화)

각 편 끝에 다음 편 예고 + 시리즈 인덱스 페이지 필수.

### 5. 한국 블로그 톤 매트릭스

```
              | 격식 (-습니다)         | 친근 (-요/해요)
--------------+---------------------+----------------------
정보 전달     | 네이버 비즈니스·기업 블로그 | 개인 브랜드·뉴스레터형
경험 공유     | 전문가 후기·리뷰          | 일상 블로그·브이로그형
```

페르소나에 따라 위 4개 위치 결정. 1 글 = 1 톤 일관.

## 워크플로우

### 1단계: 플랫폼 및 목적 확인
- 발행 플랫폼 선택 (1개 이상)
- 작성 목적: 검색 유입(SEO/GEO), 브랜드 인지도, 전환 유도, AI 검색 인용
- 타겟 독자 정의

### 2단계: 키워드 및 주제 전략

**네이버 C-Rank 키워드 전략 (2026):**
- 단일 주제 집중: 31개 카테고리 중 하나에 전문화
- 경쟁도 분석: 네이버 키워드 도구로 월간 검색량·경쟁률 확인
- 로얄 키워드 회피: 검색량 대비 문서 수 비율이 낮은 키워드 타겟
- 롱테일 키워드: "스타트업 B2B SaaS 영업 전략" 등 구체적 표현

**D.I.A.+ 검색 의도 분석:**
- 정보형 쿼리: "X란 무엇인가", "X 하는 방법" → 가이드·설명 형식
- 비교형 쿼리: "X vs Y 차이" → 비교 표·체크리스트 형식
- 후기형 쿼리: "X 사용 후기", "X 리뷰" → 경험 기반 서술 필수

**GEO 최적화 키워드 전략 (2026 신규):**
- 질문형 헤딩: "Q: ~는 어떻게 하나요? A: ..." 형식
- 정의형 단락: "~란, ... 이다" 명확한 정의 문단
- AI 인용 가능한 통계·수치 포함

### 3단계: 포스팅 구조 작성

**표준 블로그 포스팅 구조:**
```
[제목] (40자 이내, 핵심 키워드 포함, 호기심 유발)

[도입부] (150~200자)
- 독자의 문제 공감 또는 핵심 인사이트 선 제시
- "이 글을 읽으면 ~를 알 수 있습니다" 명확한 가치 제안

[목차] (네이버·워드프레스 필수)

[본문] (각 섹션 H2/H3 소제목)
- 섹션당 300~500자
- 실제 경험·사례·데이터 포함 (D.I.A. 경험 정보)
- 표·리스트·이미지로 시각화

[결론] (100~200자)
- 핵심 정리 3가지
- CTA (다음 액션 유도)

[참고 자료] (GEO 신뢰도 강화)
```

### 4단계: 네이버 블로그 특화 최적화

**C-Rank 향상 전략:**
- 주제 전문성: 같은 카테고리 글 꾸준히 발행 (주 2-3회)
- 체류 시간: 읽기 시간 5분 이상의 충분한 분량
- 상호작용 유도: 댓글 질문, 이웃 추가 CTA

**D.I.A. 고품질 요소:**
- 실제 경험 기반 후기 포함 (본인 사진, 영수증, 스크린샷)
- 최신 정보 반영 (날짜 명시)
- 이미지에 텍스트 설명 추가 (대체 텍스트)
- AI 생성 텍스트 그대로 사용 금지 (패널티 위험)

**2026년 네이버 최신 변화:**
- 스마트블록 통합: VIEW 탭 → 스마트블록으로 완전 전환
- 영상 콘텐츠 우대: 네이버 클립 첨부 시 추가 노출 혜택
- 지역 콘텐츠 우대: 로컬 키워드 포함 시 지역 검색 노출

### 5단계: 카피라이팅 안티패턴 회피

**피해야 할 표현 패턴:**
- "안녕하세요! 오늘은 ~에 대해 알아보겠습니다" → 즉시 핵심으로 시작
- 과도한 이모지 남발 (🔥💯✨) → 1-2개 이내로 제한
- 근거 없는 최상급: "최고", "최대", "No.1" → 수치·출처로 대체
- 클릭베이트 제목: "이것 모르면 손해" → 구체적 혜택 제목으로
- AI 티 나는 문장: "이 글에서는... 살펴보겠습니다. 첫째로..." → 자연스러운 구어체

## 사용 예시

- "네이버 블로그에 '스타트업 초기 자금 조달 5가지 방법' 포스팅 써줘"
- "티스토리에 SEO 최적화된 '파이썬 데이터 분석 입문' 글 작성해줘"
- "브런치 스타일로 '원격 근무 1년 차 회고' 에세이 써줘"
- "GEO 최적화해서 '챗GPT 프롬프트 엔지니어링' 가이드 만들어줘"
- "제품 리뷰 블로그 포스팅 구성 짜줘 (네이버 C-Rank 고려)"

## 출력 형식

- 플랫폼 최적화 블로그 포스팅 초안 (제목·목차·본문·CTA)
- SEO 메타 정보 (메타 제목 60자, 메타 설명 160자)
- 이미지 ALT 텍스트 제안
- 내부 링크 전략 권고
- GEO 최적화 FAQ 섹션

## 주의사항

⚠️ **AI 생성 콘텐츠 주의**: AI가 생성한 블로그 내용은 사실 확인 후 게시하세요. 통계, 날짜, 인물 정보는 반드시 원출처를 검증하세요.

**플랫폼 규격 변경**: 네이버·티스토리 알고리즘은 수시로 변경됩니다. 최신 정보는 각 플랫폼 공식 블로그나 SEO 커뮤니티에서 확인하세요.

**브랜드 가이드 미제공**: 브랜드 톤앤매너, 타겟 독자, 금지어가 있다면 미리 알려주세요. 없으면 일반 독자 친화적 문체로 작성합니다.

**이미지 생성**: 이 스킬은 텍스트 콘텐츠를 생성합니다. 이미지가 필요한 경우 Midjourney, Adobe Firefly 등 이미지 생성 도구를 별도로 활용하거나 `card-news` 스킬을 사용하세요.

## 발행 연동 (선택)

WordPress Cowork 커넥터가 연결되어 있으면 발행까지 자동화합니다.

실행 시 커넥터 연결 여부를 확인합니다:
- **연결됨** → 카피 생성 후 WordPress에 직접 발행 (카테고리, 태그, 특성 이미지 자동 설정)
- **미연결** → "WordPress 커넥터가 연결되어 있지 않습니다. Settings > Connectors에서 연결하세요." 안내 후, 마크다운 카피만 생성

네이버/티스토리/브런치/Ghost 등: 플랫폼에 최적화된 마크다운 카피를 생성합니다 (발행은 수동).
