# 비주얼 스토리텔링 (visual-storytelling)

> MoAI-Cowork v1.0.0 하네스 참고자료

## 개요

스토리 설계, 텍스트 작성, AI 이미지 생성, HTML 레이아웃을 통해 비주얼 스토리텔링을 협력하여 제작하는 하네스입니다.

## 전문가 역할

- **스토리 아키텍트**: 내러티브 구조, 장면 구성, 시각-텍스트 균형 설계
- **에세이 작성자**: 비주얼 스토리텔링의 텍스트 부분 작성, 장면별 이모션 호호흡
- **이미지 프롬프터**: Gemini 이미지 생성, 장면 간 시각 일관성 유지
- **레이아웃 빌더**: HTML/CSS로 텍스트와 이미지 통합, 반응형 페이지 제작
- **검수자**: 스토리-텍스트-이미지-레이아웃 일관성 교차 검증

## 워크플로우

### Phase 1: 준비
- 스토리 주제 및 형식 정의
- 기존 콘텐츠 및 이미지 수집

### Phase 2: 제작 실행
| 순서 | 작업 | 담당자 | 산출물 |
|------|------|--------|--------|
| 1 | 스토리 설계 | 아키텍트 | `story_blueprint.md` |
| 2a | 텍스트 작성 | 작성자 | `essay_text.md` |
| 2b | 이미지 생성 | 프롬프터 | `image_prompts.md` |
| 3 | HTML 레이아웃 | 빌더 | `layout.html` |
| 4 | 검수 | 검수자 | `review_report.md` |

## 산출물
- `story_blueprint.md` — 스토리 설계
- `essay_text.md` — 에세이 텍스트
- `image_prompts.md` — 이미지 프롬프트
- `layout.html` — HTML 레이아웃
- `review_report.md` — 검수 리포트
- `images/` — 생성 이미지 디렉토리

## 에러 처리
| 에러 유형 | 대응 전략 |
|---------|---------|
