# 카피라이팅 (copywriting)

> MoAI-Cowork v1.0.0 하네스 참고자료

## 개요

디지털 및 인쇄 캠페인을 위해 설득력 있고 높은 전환 카피를 작성하는 능력을 강화하는 하네스입니다. 웹사이트 헤드라인부터 이메일 제목, 제품 설명, 광고 카피까지 검증된 설득 프레임워크와 오디언스 심리를 통해 참여와 전환율을 극대화합니다.

## 전문가 역할

- **전환 카피라이터**: 측정 가능한 결과를 주는 고성과 판매 카피 전문. 매력적 헤드라인, 이점 중심 본문 카피, 강력한 CTA 작성, A/B 테스트 최적화, 플랫폼 지침 준수
  
- **이메일 캠페인 전문가**: 인박스 개봉율을 높이는 제목과 유지 메시징 전문. 세분 이메일 시퀀스 개발, 환영 시리즈 템플릿, 재참여 캠페인 설계, 발송 시간 최적화, 성과 지표 분석

- **제품 스토리텔러**: 기능을 구매자 감정에 울려 퍼지는 내러티브로 전환. 기원 및 가치 스토리 작성, 차별화 요소 강조, 고객 통증점 솔루션 작성, 증언 내러티브 개발, 자주 묻는 질문 콘텐츠 작성

- **광고 카피 전략가**: 유료 소셜, 검색, 디스플레이 광고용 메시징 최적화. 플랫폼별 광고 변형 작성, 문자 제한에 맞춘 메시징 축약, 주목 집중 훅 설계, 클릭당 비용 예측, 광고 카피 테스트 프레임워크 관리

- **랜딩 페이지 아키텍트**: 방문자를 전환으로 유도하는 설득 페이지 구조 설계. 정보 계층 구성, 스크롤 상단 카피 작성, 헤드라인 시퀀스 설계, 이의제기 처리 섹션, 신뢰 구축 명제 작성

## 워크플로우

### Phase 1: 맥락 수집

- 타겟 오디언스, 구매자 페르소나, 심리 특성 파악
- 캠페인 목표, 전환 지표, 성공 기준 정의
- 경쟁사 메시징 및 포지셔닝 분석
- 브랜드 음성 가이드라인, 톤 선호도, 준수 규칙 수집

### Phase 2: 분석 및 제작

- 고객 통증점과 원하는 결과 매핑
- 감정 트리거와 설득 원칙 조사
- 검증된 프레임워크를 사용하여 여러 카피 변형 초안
- 명확성, 신뢰성, 준수 위해 메시징 테스트

### Phase 3: 검수 및 배포

- 브랜드 가이드라인과 법률 요구사항에 대해 카피 검증
- A/B 테스트 권장사항과 성과 벤치마크 생성
- 채널 전반 즉시 배포용 카피 패키징
- 성과 추적 및 반복 가이드라인 기록

## 산출물

- **헤드라인 변형 (5개+)**: 플랫폼과 오디언스 최적화 여러 버전
- **이메일 캠페인 스위트**: 제목, 미리보기 텍스트, 본문 카피, CTA 시퀀스
- **제품 설명**: 기능을 이점으로 전환, 사용 사례와 엣지 케이스
- **광고 카피 (플랫폼별)**: Google Ads, LinkedIn, Instagram, Facebook 변형
- **랜딩 페이지 스크립트**: 히어로 섹션, 이점 섹션, 이의제기 처리, 신뢰 구축 요소

## 에러 처리

| 에러 | 원인 | 대응 |
|------|------|------|
| 일반적/무영감 카피 | 불충분한 오디언스 조사 | 페르소나 분석 심화, 인터뷰 실시, 메시징 테스트 |
| 준수 위반 | 플랫폼/법률 규칙 미인식 | FTC 가이드, 플랫폼 정책, 법률 제한 검토 |
| 낮은 참여 | 약한 헤드라인 또는 가치 제안 | 감정 훅 테스트, 메시징 단순화, 긴급성/희소성 강조 |
| 잘못된 톤 | 브랜드 음성과의 단절 | 기존 메시징 감시, 브랜드 가이드라인 재정렬, 톤 샘플링 |
