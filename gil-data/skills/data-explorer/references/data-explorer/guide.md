# 데이터 탐색기 가이드

## 인코딩 처리

CSV 파일의 인코딩을 자동으로 감지합니다.

| 인코딩 | 감지 패턴 | 대응 |
|--------|----------|------|
| UTF-8 | BOM 또는 기본 | 기본값 |
| CP949 | 한글 Windows 파일 | 자동 변환 |
| EUC-KR | 레거시 한글 시스템 | 자동 변환 |

## 이상값 탐지 기준

### IQR 방법 (기본)
- Q1 - 1.5*IQR 미만 또는 Q3 + 1.5*IQR 초과

### Z-score 방법 (대용량)
- |Z| > 3 기준

## 프로파일링 출력 형식

각 컬럼에 대해:
- 타입, 유니크 수, 결측률
- 수치형: min, max, mean, median, std
- 범주형: top 5 values with counts
